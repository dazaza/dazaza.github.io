This will mount the VMware Tools installation package as a virtual CD/DVD in the guest operating system. 3. **Access the mounted drive**: - In the guest operating system, open the file explorer and locate the newly mounted virtual CD/DVD drive where the VMware Tools installer is located. 4. **Run the installer**: - For Windows guests: Double-click the setup file (usually named `setup.exe`) to start the installation process. Follow the on-screen instructions to complete the installation. - For Linux guests: Open a terminal, navigate to the mounted directory (e.g., `/mnt/cdrom`), and run the installation script using the following commands: ```bash sudo ./vmware-install.pl ``` Follow the prompts to configure and install VMware Tools. 5. **Reboot the virtual machine**: - After the installation is complete, it is recommended to reboot the virtual machine to ensure all components of VMware Tools are properly initialized. 6. **Verify the installation**: - Check if VMware Tools is running by going to **VM > Removable Devices > VMware Tools > Connect**. If the option says "Connected," VMware Tools is successfully installed.
