Windows 8.1 RTM/VS is now available for download via MSDN/TechNet Subscription.

by anonymous on 2013-09-11 13:28:03

Today, Microsoft announced that MSDN and TechNet subscribers can already download the Windows 8.1 RTM version, as well as the Windows Server 2012 R2 RTM version. At the same time, developers can also download the Visual Studio 2013 RC version. The Windows 8.1 Enterprise RTM version will be available to MSDN and TechNet subscribers by the end of this month.

Steve Guggenheimer mentioned in his blog that Microsoft acknowledged their initial decision to delay the release of Windows 8.1 RTM across all channels until October was a significant challenge for developers. Now, they have listened and taken action accordingly.

Microsoft stated that the purpose of this pre-release of Windows 8.1 RTM is for testing. Developers and IT professionals can already prepare for the final development of Windows 8.1 applications and system deployment. The Microsoft engineering team is still continuing to refine and update the system in preparation for the official release of Windows 8.1 on October 18th. Starting at 7 PM Beijing time on the 17th, it will be pushed via the Windows Store, and Windows 8 users can update for free.

Windows 8.1, codenamed Blue, will continue to expand Microsoft's vision for Windows 8 regarding device form factors and touch experiences. It improves the functionality and user experience of the Windows 8 operating system, including features like the Start button, booting to the desktop, personalization, search, built-in apps, the new IE 11, Windows Store experience, cloud connectivity, enterprise management, and security (via Windows).

Official download page for Visual Studio 2013 RC: http://www.microsoft.com/visualstudio/chs/2013-downloads

— XAML Editor

— Cloud Business App

— C++

— TypeScript

— CodeLens

— Work Item Charts
