Windows Vista正式发布日期及价格泄露！

by gosman on 2006-08-29 17:44:14

微软寄以重托的新一代操作系统Windows Vista的正式发布日期及价格现在已经被曝光，因为著名的电子商务网站Amazon已经开始进行Windows Vista的预购活动，并且披露了Vista的正式发布日期：2007年1月30日。这也是微软之前一直所承诺的最后发布日期。

　　下面是Amazon（via ZDNet）上面列出的各个Vista版本的定价（左边的价格是完整版，右边是升级版）：

Windows Vista Home Basic, $199/$99.95

Windows Vista Home Premium, $239/$159

Windows Vista Business, $299/$199

Windows Vista Ultimate, $399/$259

　　这样的价格仍然不低。如果你也像我这样很喜欢Vista的透明界面，并且你也刚好有购买新电脑的计划，可以考虑一下购买附送正版Vista的电脑，这样划算得多。当然，我说的是正版。
