ZEAZ CLAW: 25 Mobile RSS Reader Solutions

by zeaz on 2008-02-18 19:28:32

As mobile devices, such as cell phones, continue to evolve, their roles in our lives are becoming increasingly extensive. Today, I will introduce solutions for RSS reading on mobile phones! There are software options as well as WAP-based ones, and I believe there is one that suits you! Most of the applications introduced here are based on Java, so their compatibility may vary across different models. Due to the limited number of phones I own, I am unable to test them all.

1) Viigo: This should be one of the top 5 apps for Blackberry, it's incredibly useful. It supports web and mobile management of RSS feeds, and can be customized according to personal preferences. Blackberry phones even support update alerts! The latest update has added synchronization with Google Reader. The only regret is that this software only supports Blackberry and WM (Windows Mobile) platforms.

2) freerange: This app is great, but its biggest drawback is that the free version only allows you to subscribe to 10 feeds (you can combine multiple RSS feeds through aggregation). It has comprehensive features, good speed, allows setting of various font styles, and has a very beautiful interface. Multi-platform support!

3) piconews: The website seems a bit slow, but it works well on mobile phones. One nice design feature is that if the title is too long, it scrolls to display it. The interface is slightly inferior compared to the previous two, but the functionality is not bad!

4) newsgator: The company behind the famous feeddemon software recently announced that it will be free and has released a mobile version!

1. Register an account on newsgator.com, import Feeds from other readers like Google Reader. OPML format files are supported.

2. Download a software called FeedDemon and install it on your computer. Download link: http://www.newsgator.com/download/FeedDemon/FeedDemonInstall.exe

3. In the Tools menu of FeedDemon, select "Synchronization Options", set the NewsGator account password and folders to be synchronized.

4. Install NewsGator Go! on your mobile phone, and enter the NewsGator account password in the Account section of the Settings menu. Only CMNET access is supported.

5) lifefeed: Multi-platform support, average interface, didn't work successfully on my Blackberry.

6) flurry: A simple but practical software that can also be used as an email program! Recommended! Registration requires filling in your phone number, no SMS will be sent, just need to fill in the phone number when logging in. The default email and password must be consistent to log in.

7) Panshuhu: A good domestic software, supports mobile QR code recognition!

8) Thumb Sky: Supports S60, SP, Java.

9) Plusmo: Although I don't particularly like its interface, I still want to release it for those who need it. It's a bit slow on my Blackberry.

10) Unyverse: A software with a Moto E1 Apple style interface, registration is required to get the Pin Code, which needs to be entered in the software interface. It has five major functions: Contacts, Notebooks, RSS News, Wiki Groups, Wikipedia. The first two can be imported via Google Toolbar, making it convenient for information exchange and storage between mobile phones and computers. The RSS News function perfectly supports Chinese and allows setting three font sizes. Wiki Groups is a community composed of people with similar interests, including sports, literature, mobile phones, etc.
