Linux Boot Process:

BIOS self-check --> MRB --> GRUB --> Kernel Loading --> INIT -->

/etc/inittab --> /etc/rc.d/rc.sysinit --> According to the run level

--> Run all scripts under /etc/rc.d/rcX.d/ (X represents the number of the run level, such as rc5.d)

--> /etc/rc.local --> login

/etc/inittab defines the current system run level:

0 - Shutdown

1 - Root single user mode, no password required

2 - Multi-user mode, without network services

3 - Full multi-user mode, character interface

4 - Reserved

5 - Graphical mode

6 - Restart

runlevel - Displays the current system run level

Control services that run at system startup:

ntsysv [--level n] - System service management (graphical interface)

chkconfig - System service management

--list - Display

--add - Add

--del - Remove

on - Enable

off - Disable

Example: chkconfig vsftpd off --level 234

File Search:

which - Shows the location of a command file (only searches in $PATH)

whereis - Shows the command file and its related files

locate - Finds specified files; for more accurate results, update the system database with updatedb before searching

find - Finds files

Syntax: find search path [options]

-name - Find by filename

-perm - Find by permission

-user - Find by file owner

-group - Find by group

-type - Find by type

-size - Find by size (default is blocks; add 'c' after the number for bytes)

-newer - Newer than a specified file

-exec - Executes a specified command after finding

-ok - Executes with confirmation

Examples:

find ./ -newer ab -ok rm {} \;

find ./ -newer ab -exec rm {} \;

File Times:

atime (access time) - Changes every time the file is read

ctime (change time) - Changes when attributes are changed

mtime (modify time) - Changes when content is modified; also modifies atime and ctime

Default situation:

ls -l - Views mtime

ls -lu - Views atime

ls -lc - Views ctime

stat - Views all three dates

x-window

Server Settings:

xhost - Controls allowed client machines

+ ip - Allows the specified IP

- ip - Removes the IP from the allowed list

Client Usage:

DISPLAY=10.1.1.125:0 xclock

Or:

On the local machine, xhost +

Then: ssh -X 10.1.1.125

xclock

sort - Sorts text files

-n - Numeric sort

-t - Specifies delimiter

-k2 - Sorting key

Example: sort -n -t: -k2 s1.txt

wc - Counts

-l - Number of lines

-w - Number of words

-c - Number of characters

Disk Quota:

1. Mount disk with quota options

vim /etc/fstab

/dev/sda11 /disk11 ext3 defaults,usrquota,grpquota 0 0

2. Generate relevant configuration files

quotacheck -cavug

-a - Scans all mounted disks

-c - Creates quota data

-v - Shows execution process

-u - Generates aquota.user for user quotas

-g - Generates aquota.group for group quotas

3. Set user quotas

edquota -u tom

Filesystem blocks soft hard inodes soft hard

/dev/sda11 0 1000 1200 0 0 0

Space limit File count limit

4. Enable quotas

quotaon /dev/sda11

(To automatically enable quotas at boot, add this command to /etc/rc.local)

5. View quotas

repquota -a

quota -u tom to view a specific user
