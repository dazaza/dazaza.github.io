Software Introduction

Time Fairy (Timefairy) is an elegant Windows platform universal network computer time calibration software. It is easy to use, occupies a small bandwidth, and also has a relatively low occupancy of system resources on the client side.

Time Fairy (Timefairy) was developed by the Chinese National Time Service Center. You can use the free time calibration service provided by the National Time Service Center through the Time Fairy client.

Time Fairy (Timefairy) is a standard server-client software. The National Time Service Center provides a standard time server and offers standard time services, obtaining the standard time from the server end. Users can access and query the current standard Beijing Time or calibrate their personal computer time through the Time Fairy (Timefairy) client.

The Time Fairy (Timefairy) server end is based on Microsoft's latest .NET software platform. Using .NET's advanced network platform services and optimized system program structure, it acquires the standard Beijing Time provided by the dedicated time server of the National Time Service Center. Meanwhile, it sends time information to the client via the HTTP protocol. The National Time Service Center promotes Time Fairy (Timefairy) as a new Internet time service method, aiming to allow Chinese Internet users to conveniently, accurately, and at any time, obtain the current Beijing Time and calibrate the current computer time.

The Time Fairy (Timefairy) client uses Microsoft ActiveX HTML embedding technology, with code written in VB, providing good system compatibility. Due to the time sensitivity of this software, the code has been specially optimized to ensure the accuracy of the time obtained by the user. Time Fairy (Timefairy) uses the HTTP protocol, which is universal and can be used normally on internal local area networks or computers with firewalls set up. Time Fairy (Timefairy) client installation is simple, the program is short, and after digital certification, it can ensure the safe and reliable use of the software.

New Features

1. A simple interface makes the software easier to use;

Installation and Uninstallation

System Requirements

To use Time Fairy (Timefairy), the following basic system needs are required:

- Microsoft Windows 9x, Me, NT, 2000, XP or later versions

- IE4.0 or higher browser

- 100 MHz CPU

- 16 MB RAM

- 5 MB hard disk space for software installation

- 56Kbps Internet access

It is recommended to use Internet access of 512Kbps or higher.

Installation

For Time Fairy (Timefairy), please visit www.time.ac.cn/times/timefairy.htm. If the visitor is accessing this webpage for the first time, it will ask whether to install the digitally signed Time Fairy (Timefairy) software. If the visitor agrees to install and use it, Time Fairy (Timefairy) will be installed on the visitor's operating system as an ActiveX software package. If you find any program without digital certification, do not download, install, or run it, as it could likely result in your computer being attacked by viruses or hackers. If the correct selection still cannot install or run the software, please refer to the "User Manual" "Frequently Asked Questions" or log into the www.time.ac.cn website to contact the administrator to solve the problem.

Note: The normal installation and usage process targets the default settings of the Windows system and browser. If changes in settings cause installation and usage issues, please log into the www.time.ac.cn website to contact the administrator.

Time Fairy Installation Issues under XP SP2

1. Open Internet Options in Control Panel

2. Enter the Security tab

3. Select Trusted Sites

4. Click Sites

5. In the "Add this website to the zone" input www.time.ac.cn

6. Do not select "Require server verification for all sites in this zone"

7. Log in to http://time.ac.cn/times/timefairy.htm

8. When the certificate option appears, click Yes

9. After completion, re-enter Control Panel --> Internet Options --> Security --> Trusted Sites and delete the one just added

10. Complete

If you don't want to temporarily lower the security level or have not completed the installation, click here to download the local installation of Time Fairy.

Upgrade

When you access www.time.ac.cn/times/timefairy.htm through your browser to use Time Fairy, the software will automatically judge the software version and perform upgrades.

Or follow these steps to upgrade:

1. Close the browser accessing www.time.ac.cn/times/timefairy.htm

2. Open the Downloaded Program Files folder under the system folder of the partition where the system is located in My Computer

Example: C:\WINDOWS\Downloaded Program Files

3. Find the client file package in the list, right-click on the file package, and select Upgrade

4. After the upgrade is complete, browse www.time.ac.cn/times/timefairy.htm again to use the new version of the software.

Uninstallation

You can follow the steps below to uninstall Time Fairy (Timefairy):

1. Close the browser accessing www.time.ac.cn/times/timefairy.htm

2. Open the Downloaded Program Files folder under the system folder of the partition where the system is located in My Computer

Example: C:\WINNT\Downloaded Program Files

3. Find the client file package in the list, right-click on the file package, and select Delete

4. Deletion of Time Fairy (Timefairy) is complete.
