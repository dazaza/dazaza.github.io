Microsoft Video: New version of the video website is online

by yinyin on 2008-06-16 14:40:28

Microsoft's Preview website always brings quite a few surprises. A new version of the Microsoft Video website has gone online. It features a style similar to the Microsoft homepage. More importantly, it has made innovations based on the UI and structure of traditional websites so that users can more easily find the video materials they need with its search and classification functions. This site is developed based on Microsoft Silverlight, thus requiring relevant plug-ins.
