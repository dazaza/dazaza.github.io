与 Facebook 的 HE-Double-Hockey-Sticks

已知最早版本的成语“压死骆驼的稻草”是英国哲学家托马斯·霍布斯 (Thomas Hobbes) 于 1677 年写的，尽管它与马和羽毛有关：坏，可能跟随在任何行动上，不完全是整个原因，而是它的最后一部分，但可以说必然产生结果，就像最后一根羽毛可以说是打破马背一样，以前有那么多人躺着，但那个人愿意去做。”在过去的几年里，我们 Damn Interesting 收到了许多长期粉丝的来信，他们错误地认为我们几年前就停止了所有运营。这些粉丝通常很高兴听到 a) 我们仍在写作和播客； b) 自他们上次访问以来，有大量新内容。当我们问他们是什么导致我们死亡的假设时，他们总是提到我们的帖子从他们的 Facebook 新闻提要中消失的事实。这一趋势大致与 Facebook 为页面引入“提升”相吻合；在这种新模式中，根据我们可以看到的统计数据，Facebook 停止向大约 94% 的关注者展示我们的帖子，要求付费将每个帖子“提升”到广告中，这将使更多的观众看到它。我们几乎在一夜之间失去了与成千上万粉丝的联系。我们不介意为有价值的服务付费，但我们绝对不想通过在 Facebook 上购买广告空间来接触我们的受众。哎呀。但是，没有其他选择可以覆盖以前关注我们帖子的许多人，而且他们可能想继续这样做。我们在 2008 年建立了我们的 Facebook 页面，因为这个刚刚起步的社交媒体网站越来越受欢迎，我们继续维护我们的页面，主要是作为事后的想法。在随后的几年里，Facebook 从一个可疑的好奇心演变成一个坚定地为坏主意服务的大公司。在一项感觉早就应该采取的举措中，我们 Damn Interesting 正在放弃与 Facebook 的所有互动和联系。当 Facebook 使用无处不在的嵌入式“在 Facebook 上点赞”按钮在人们不知情或未经他们同意的情况下跟踪人们在网络上的活动时，我们真的应该这样做。我们应该这样做，因为 Facebook 通过在新闻源中向一些人展示更积极的故事，而向其他人展示更多的负面故事，从而真正玩弄人们的情绪，以了解这将如何影响他们的情绪状态。当 Facebook 允许广告商将广告定位到对“仇恨犹太人”等话题感兴趣的人时，我们应该这样做。我们以前应该这样做过很多次。对于在 Facebook 上关注我们的数万粉丝：我们真诚地道歉。无论如何，您可能不会在那里看到我们的大部分帖子。对于仍然看到它们的少数人，我们不再希望在 Facebook 上分享可能导致人们在那里花时间的内容。我们希望您能通过我们的多种非 Facebook 选项之一与您保持联系。

我们离开的原因并不完全抽象。我们相信你们中的许多人，像我们一样，亲身体验过 Facebook 如何让人们成为最坏的自己。它可以将单纯的政治观点分歧提升为愤怒和敌意，将朋友和家人推向极端观点，将所爱的人变成以前的自己的丑陋漫画。也许你甚至对自己在那里的一些帖子感到遗憾； Facebook 界面旨在让善意的分歧变得难以进行。它为错误信息、虚假信息和仇恨提供了不应有的论坛。科学证明使用 Facebook 会导致抑郁。 Facebook 以历史上罕见的幅度从世界的质量中减去，当它消失时，我们都会变得更好。是的，我们知道 Facebook 不是地球上唯一的有害公司，但甜蜜的吉普斯男孩-你好，它是存在电梯中令人毛骨悚然的放屁。有人称其为“必要的邪恶”，但如果是这种情况，让我们尽我们所能让它变得不必要。 Facebook 联合创始人马克·扎克伯格 (Mark Zuckerberg) 和他大学时的一个朋友之间有一个现在臭名昭著的聊天记录，其中扎克伯格提出分享私人用户信息，并将他对使用 Facebook 的人的看法总结为“愚蠢的 ***” s。”我们必须不情愿地承认这一点。等了这么久才摆脱 Facebook 的热垃圾果汁，我们真的是傻了。如果您有任何特别不愉快的 Facebook 体验，请在评论中告诉我们。也许它会证明是宣泄。
