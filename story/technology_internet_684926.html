Features:

Glass protective screen to prevent scratches on the LCD screen

Large LCD display with blue backlight and user-friendly operation interface

Suitable for floor cables, electric heating films, manifolds, as well as radiator heating. It can also be used in sweat rooms, incubation rooms, or other places that require temperature control.

Power failure memory function: after power failure, it can operate according to the parameters set before the power failure when powered on again.

Automatic adjustment of indoor temperature, temperature hysteresis can be selected by yourself.

Single/dual sensors, anti-freeze function, temperature hysteresis, temperature calibration can be selected by users in menu mode.

High-performance imported microcontroller chip, strong anti-interference ability, FLASH program memory.

Transformer-powered, stable and reliable.

Relay used as output control device, high reliability, strong practicability.

Clock display function, 5+2 programming function.

Low-temperature and high-temperature protection functions (optional).
