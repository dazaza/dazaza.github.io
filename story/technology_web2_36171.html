12 priceless sites for web design and design inspiration

by maker on 2007-11-22 23:36:03

Whether you're a professional or amateur web designer, or just want to learn more about web design, these are great resources to browse and add to your collection. This collection includes design styles, wizards and tools as well as font photo backups and vector images. These are organized into three complimentary catalogs: Inspiration, Guides, and Resources. Enjoy it!

Web design inspiration:

CSS Zen Garden A based on CSS Demonstration of visual operation of design. You can select any style menu from the list to load into this page. The goal is to inspire, inspire and encourage participation.

Web Design Inspiration has a large collection of successful web designs of many types, 400 variety.

Web Creme provides the most up-to-date, elegant but indirect, blog-like screenshots in common use. For web designers to browse.

Unmatched Style is a growing community dedicated to the use of cascading style forms and web standards for those who make gorgeous web sites.

CSS Beauty provides its viewers with a database of well-designed CSS web pages from around the world.

Web Design Guides:

Web design wizard:

Web Design from Scratch was created for anyone interested in designing web pages - novice or expert. It will help you understand what causes a web page to succeed or fail, and what increases the chances of success.

Web 2.0 how-to Design Style Guide: In this guide, the author describes a large number of web design components in the Web 2.0 era. It explains how it works and explains when and how you can use these components.

Top 50 Logo Design Tutorials Provide a good logo design guide and useful screen shots that will help you navigate multiple links.

51 Photoshop Text Effect Tutorials. The text and the way the text is viewed is the main work of any design. If the web page looks wrong, a good design will be compromised. Any logo is almost entirely text. This 51 guide will show you how to generate the text you need.

Web Design Resources:

Web design resources:

Open Source Web Design is a site where you can freely download web design templates and share yours.

Blue Vertigo showcases an amazing collection of free resources, from backup images to vector screenshots and more.

Browser Shots allows you to quickly and easily test your designs in the browser. Screenshots can be taken in different browsers.
