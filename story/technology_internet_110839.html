Top 10 statistical chart generation plugins

by fwdear on 2008-12-11 23:00:23

Flot - Flot provides javascript code library for jQuery. Easy to use, featured charts that provide interactive features (the ability to zoom in and out of data areas, etc.).

Open Flash Chart - Open Flash Chart is a Flash charting component that is easy to install and provides apis in the following languages: PHP, Perl, Python, Java, Ruby on Rails, and.NET to control the chart.

AmCharts - AmCharts is an animated interactive Flash chart. Support: Pie Charts, Line Charts, Scatter/Bubble Charts, Bar/Column Charts, and even stock charts. (The website seems to be down now. Somehow)

Emprise JavaScript Charts - Emprise is a 100% pure JavaScript charting solution that does not require any JavaScript framework.

PlotKit - PlotKit is a Javascript library for charts and images. PlotKit works with the MochiKit javascript library to support HTML Canvas and Adobe SVG display as well as native browser support. In particular, the documentation is very complete.

Flotr - Flotr is a javascript plugin library for the Prototype 1.6.0.2 framework. Flotr helps you create charts in the browser, supports mouse event response, zooming in and out, CSS type support, and more.

PHP/SWF Charts - PHP/SWF is a powerful tool that simply supports web charting. You can use PHP to collect data and send it to flash. It supports many chart types: Line, Column, Stacked column, Floating column, 3D column, Stacked 3D column, Parallel 3D column, Pie, 3D Pie, Bar, Stacked bar, Floating bar, Area, Stacked area, Candlestick, Scatter, Polar, Mixed.

Visifire - Visifire is an open source visual data component made with Silverlight. With Visifire you can create animated Silverlight charts, Supports ASP, ASP.Net, PHP, JSP, ColdFusion, Ruby on Rails or simple HTML calls.

FusionCharts - FusionCharts is a cross-browser and cross-platform flash charting component. Can be invoked by ASP.NET, ASP, PHP, JSP, ColdFusion, Ruby on Rails, simple HTML pages and even PPT. It supports almost any language you know.

JFreeChart - JFreeChart is an open source java diagram library that makes it easy for developers to display high-quality diagrams in their applications. The JFreeChart project began seven years ago, and since January 2000, more than 40,000 developers have used JFreeChart. JFreeChart supports many graphics file formats: JPEG, GIF, PDF, EPS and SVG. This is a great resource for java developers.
