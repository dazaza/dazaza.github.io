Develop Android applications using Android app development

by geekzhang on 2013-05-10 21:02:05

Dear Android developers, what platforms and tools do you usually use for development? Windows? MacOS? Linux? Eclipse ADT? IntelliJ IDEA? Actually, the platform and tools don't matter much because Android itself is cross-platform. So, have you ever thought about developing Android applications using the Android platform itself (i.e., a smartphone or tablet)? Regardless of whether this approach has practical significance, the idea is absolutely feasible! Such tools already exist, and this is where AIDE comes in – an Android-based Java IDE that runs on Android devices.

AIDE - Android Java IDE

AIDE is an integrated development environment (IDE) for Java application development that allows direct app development on Android devices. AIDE supports the entire workflow of writing code, compiling, and running programs. It features a rich text editor, code auto-completion, real-time syntax checking, refactoring, intelligent code navigation, and more. To run your program, you simply need to press the "Run" button.

If your Android device has an external keyboard, AIDE can turn it into a powerful development tool. For example, you could use an ASUS Transformer Prime notebook-tablet hybrid running Android 4.0 (code-named Ice Cream Sandwich) for app development. AIDE also transforms Android smartphones into mini development computers, allowing coding via touch input.

AIDE supports two modes of development: Android SDK (Java/XML) and Android NDK (C/C++).

AIDE fully supports Eclipse projects. You can directly copy the source code of a project created with Eclipse ADT onto your Android device and continue developing with AIDE. Additionally, you can upload Eclipse project source code to Dropbox for cloud storage, as AIDE integrates with Dropbox to allow version control management through the service.

Of course, AIDE wouldn’t be complete without Git support!

AIDE currently offers two versions: free and pro. The free version provides basic functionality so users can experience the unique features of this IDE. However, if you're serious about using AIDE for app development, it's best to purchase the pro version, which includes the following features:

- Allows more than 5 Java files per project

- Supports Git push/commit/branch protocols

- APK publishing functionality

- Offline SDK documentation lookup

- Customizable user options

After reviewing these features, how do Android developer fellows feel? If you often have a lot of coding to do and spend over 2 hours commuting each day, might it be worth trying out?
