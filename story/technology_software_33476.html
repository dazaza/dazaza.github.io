Shanghai University develops Shanghainese input method, to be launched in 2008

by netbeta on 2007-11-06 18:47:54

"Recently, we have completed the alignment work of 15,000 Shanghai dialect words with pinyin. In a few months, citizens who like to use the Shanghai dialect will be able to download the 'Shanghai Dialect Input Method' online for typing." Professor Qian Nairong from the Chinese Department of Shanghai University, who is in charge of the development of the Shanghai dialect input method, stated that at that time, the Shanghai dialect input method would also come with a user manual and a list of correct Chinese character spellings for the most commonly used words, allowing more netizens to become familiar with and use the Shanghai dialect.

Should "Wubi" or "Pinyin" input method be used?

"For the development of the Shanghai dialect input method, there are two schools of thought among scholars and enthusiasts online: one is to use an input method similar to Wubi, and the other is the phonetic input method currently adopted," Qian Nairong told reporters that the reason they ultimately chose the phonetic input method was because over 95% of netizens currently use the pinyin input method.

According to him, the "Shanghai dialect pinyin scheme" adopted now is close to the Chinese pinyin scheme, mainly to make it as easy to learn and type as possible. The same consonants and vowels in Shanghai dialect that are identical to Mandarin are represented by the same letters as in Mandarin. Qian Nairong said that finalizing the "Shanghai dialect pinyin scheme" is the foundation for developing the Shanghai dialect input method. Starting this summer, Zheng Xiaojun, a graduate of the Physics Department of Shanghai University, who is specifically responsible for implementing the "Shanghai dialect input method," began aligning the pinyin with Chinese characters in the input method, and this work has now been fully completed. "This is the most critical part of the input method. If the alignment succeeds, Shanghai dialect can be typed directly on the internet," Qian Nairong noted.

Some commonly used Mandarin terms are also included in the alignment scope

Professor Qian indicated that since many Shanghai residents mix Mandarin while speaking the Shanghai dialect, this input method must also allow Mandarin terms used in the Shanghai dialect to be typed using Shanghai dialect pinyin. This is exactly what Zheng Xiaojun is currently doing. "Aligning common Mandarin entries such as 'wenti' (problem), 'shehui' (society), 'hexie' (harmony), 'renzhen' (serious) with Shanghai dialect pinyin is the second phase of the alignment work. Thus, when typing Shanghai dialect sentences in the future, there won't be a need to switch between two input methods," Zheng Xiaojun said, adding that this work is expected to be completed by early December.

According to Zheng Xiaojun, after completing the alignment of pinyin with entries, it will be necessary to create a Shanghai dialect character library and conduct technical adjustments, including compiling a list of correct Chinese character spellings for commonly used Shanghai dialect words in the user manual for the convenience of users.

Is the Shanghai dialect input method practical?

The reporter learned that before the "Shanghai input method," the internet already had the "Wu dialect input method" created by Zheng Xiaojun. Although this input method was once popular among enthusiasts, its popularity did not last long. Will the Shanghai dialect, as a branch of Wu dialect, continue the fate of the "Wu dialect input method"? Zheng Xiaojun told reporters that the previous Wu dialect input method could be considered a prelude to the "Shanghai dialect input method." Last year, after Professor Qian Nairong's application for the "Digitalization of Shanghai Dialect Vocabulary" as part of the Shanghai Philosophy and Social Sciences Planning Project was approved, they began discussing specific implementation plans and secured funding. Aligning pinyin with the expert-approved Chinese characters from the "Comprehensive Dictionary of the Shanghai Dialect" differs significantly from the initial Wu dialect input experiments which were more ambiguous but geographically broader. There are also many new innovations and improvements in the software design of the input method. Zheng Xiaojun believes that once the entire Shanghai dialect input method is completed, citizens will be able to directly type Shanghai dialect characters based on their pronunciation. For example, entering "shang he hhe hho" will produce "Shanghai Xianhua."
