Large collection of free CSS layouts and templates!

by cacard on 2007-11-09 18:19:47

There are many templates on the Internet based on (X)HTML/CSS tags. If you are A Web developer, you don't want to waste time again and again to repeat the code design above, here are A list, provided based on CSS free template, Almost all resources are accompanied by images and a complete layout structure. Some of these resources have already appeared in diglog, which is a good collage.

Open Source Templates is A free CSS and XHTML Source template design community.

OSWD is a community of 2080 designers who download and share Web design templates for free.

The OSWT project is designed to create pages quickly for users without any design skills.

MaxDesign provides A set of CSS page layout, including 2, and 3 column layout.

CSSplay offers many free layout very professional level. Most do not require authorization and can be used freely, but some do require email authorization!

Mollio is a simple collection of html /CSS templates.

Six Shooter Media provides 15 template ( - _____ - ` ` it also get you anything? However, still very much in the spirit of sharing!) .

CSS Design Templates provided by the template, you can use any modification, and however, have to indicate the website links of the provider.

CSS Intensivstation contains 15 layout. All templates are based on XHTML 1.0 and CSS2.

CSSFILL CSS/ XHTML-based layout and templates by Michael Strand, free to download.

CSS Frame Layouts template, based on CSS page template, used to quickly build your page architecture.

CSS Template Gallery provides search functionality, but only has 32 templates (-_____-).

the Ex - DesignZ provides more than 300 based on the (X) HTML/CSS template.

Jimmy Lin provides completely (fully 100%!) CSS/XHTML based and available for IE5.0+, FireFox 1.5, Opera8.5 and Safari 2. No Hacks are used and can be easily modified. There are no copyright restrictions, but the source link must be noted.

Layout Gala provides more than 40 professional Layout. Using standard markup and CSS, it can be tested in current major browsers.

Little Boxes contains 16 layout, classic!

Solucija  A completely free XHTML/CSS based site template download site. Users can use commercial or non-commercial websites, but keep a link to the website.

The Layout Reservoir to provide The Layout of The two columns and three columns.

Linknosie is A tool, you can customize the layout type, width and other parameters to generate the layout.

Code Sucks provides about 100 CSS layouts for download. All layouts are designed in 1024x768 dimensions.

SSI Develper provided based on the XHTML 1.0 and CSS2 CSS template.
