Usage instructions:

After decompressing, execute AVP.bat (use AVP98.BAT for Win98), and a graphical selection menu will appear. Use the "↑" and "↓" keys on your keyboard to switch options.

1. Menu 1 - Hard drive virus removal:

(1) Executes virus removal directly under Windows DOS mode.

(2) When exiting, you can choose whether to create the original compressed package of AVP_DOS software.

(3) After exiting, you can run it via the command line (AVPDOS32 X:) with options available by checking /?. Common usage includes *:/* for all disk letters and files.

2. Menu 2 - Floppy disk virus removal: After execution, it will prompt you to insert a floppy disk, which will be made into a bootable antivirus floppy disk that automatically enters the virus removal process upon booting.

3. Menu 3 - USB flash drive virus removal: After execution, it will prompt you to insert a USB drive and enter the USB drive letter, creating an antivirus USB drive. Boot from the USB drive to start the virus removal process.

4. Menu 4 - CD virus removal: After execution, generates an AVP.img image file in the IMG directory, which can be used in CD burning software to create an antivirus CD.

5. Menu 5 - Virtual floppy drive virus removal: After execution, when the computer restarts, select "Boot from virtual floppy disk" to enter the virus removal interface.

6. Menu 6 - Virtual CD drive virus removal:

(1) After execution, when the computer restarts, select "Boot ISO CD image" to enter the virus removal interface.

(2) After execution, the AVP.ISO file generated on C: can also be directly used to burn a bootable antivirus CD.

7. Menu 7 - Run antivirus CD:

(1) Copy the various DOS antivirus IMG files generated by menu 4 into the IMG directory as needed.

(2) After execution, an antivirus CD DOSAV.ISO file will automatically be created on C:\, which can be used to burn an antivirus CD.

(3) When the computer restarts, select "Boot ISO CD image" to enter the CD menu interface.

8. Menu 8 - Update virus database:

(1) After execution, the IE download tool will automatically open to download the updated virus pack. Please save it in the software directory.

(2) After closing the download tool, the software will automatically update all virus database files (the system needs to have WinImage installed).

Precautions:

1. Before making an antivirus USB drive, please confirm your USB drive letter so you can correctly input the USB drive letter.

2. When running the virtual floppy drive startup antivirus, the computer will restart after 30 seconds. Please save any unsaved data before starting the antivirus process.

3. When running the virtual CD drive startup antivirus, the computer will restart after 30 seconds. Please save any unsaved data before starting the antivirus process.

4. If there are multiple lines in the startup menu saying "Start from virtual floppy disk" or "Start ISO CD image," you can use the BootEdit feature in the software to edit and delete them. Be cautious not to delete useful boot items in c:\boot.ini, which could cause the computer to fail to boot.

5. Due to different computer configurations, if the program restarts, try disabling NTFS support, which may solve the problem.

Evaluation of methods:

1. It is recommended to use methods 3 and 4 (USB flash drive and CD) for pure DOS operations, as they provide more thorough virus removal.

2. For users without a USB drive or CD drive, method 2 (floppy disk) can be used. Since the antivirus program exists in the hard drive in an encrypted compressed image form, it is relatively safe and only temporarily unpacked when needed.

3. For users without a floppy (or CD) drive, virtual floppy (or CD) drive antivirus is a supplementary choice. It is faster but less thorough than pure DOS or floppy disk startup virus removal.

4. The hard drive DOS antivirus method is the least thorough. However, due to the improved safety of saving the antivirus program on the hard drive and its convenience, it serves as a supplementary option for systems with only one antivirus software installed, still having practical value.

5. The most recommended DOS antivirus solution is: Download various antivirus image files (IMG), burn a personalized antivirus maintenance CD using RW, and frequently re-burn it to update the virus database (only need to replace the IMG file in the ISO).
