◆ Detailed Instructions for Ghost Usage:

A. Partition Cloning and Backup

===========================================================

　　Using Ghost for system backup, there are two ways: entire hard disk (Disk) and partition hard disk (Partition). In the menu, click the Local (Local) item, and in the menu that pops up on the right, there are three sub-items. Disk indicates backing up the entire hard disk (i.e., cloning), Partition indicates backing up a single partition of the hard disk, Check indicates checking the hard disk or backup files to see if they may fail due to partition or hard disk damage. Partition backup has practical value for personal users to preserve system data, especially when restoring and copying system partitions.

　　Select Local→Partition→To Image from the menu, a hard disk selection window will pop up to start the partition backup operation. Click the white hard disk information bar in the window to select the hard disk, enter the window, and select the partition to operate (if you don't have a mouse, you can use the keyboard: TAB key switches, Enter key confirms, direction keys select). In the pop-up window, select the directory path where the backup will be stored and input the backup file name, note that the backup file name has a GHO extension. Next, the program will ask whether to compress the backup data and give three choices: No means no compression, Fast means low compression ratio and fast execution of backup, High means high compression ratio but very slow execution of backup. Finally, select the Yes button to start the partition hard disk backup. Ghost backup speed is quite fast, it doesn't take long to complete, and the backup file is stored in the set directory with a GHO extension.

B. Hard Disk Cloning and Backup

===========================================================

　　Hard disk cloning refers to backing up and restoring the entire hard disk. Select the menu Local→Disk→To Disk, in the pop-up window select the source hard disk (the first hard disk), then select the target hard disk to copy to (the second hard disk). Note that you can set the size of each partition on the target hard disk, Ghost can automatically partition and format the target hard disk according to the set partition values. Select Yes to start execution.

　　Ghost can clone the target hard disk almost exactly like the source hard disk and achieve partitioning, formatting, copying the system and files in one step.

Just need to pay attention that the target hard disk cannot be too small, it must be able to hold the data content of the source hard disk.

　　Ghost also provides a hard disk backup function, which backs up all the data on the hard disk into a file saved on the hard disk

( Menu Local→Disk→To Image), then it can be restored to other hard disks or the source hard disk at any time, which is very convenient for installing multiple systems. The usage method is similar to partition backup.

C. Backup Restore

===========================================================

　　If the backed-up partition data on the hard disk is damaged and cannot be repaired by general data repair methods, and the system is destroyed and cannot start, the backup data can be used to completely restore without reinstalling programs or systems. It can also restore backups to another hard disk.

　　To restore the backed-up partition, select the menu Local→Partition→From Image in the interface, select the backup file to restore in the pop-up window, then select the hard disk and partition to restore, and click the Yes button.

　　This system integrates a one-key GHOST 11.2 080723 Olympic Edition, which can very easily back up and restore any disk,

fully automatic, no human intervention required, just press once and it's OK.
