AVG Antivirus 2012 is here.

by wujizhui on 2011-09-28 05:00:55

The full-featured AVG 2012 software has recently been officially launched. Today, I will bring you an in-depth review of the AVG Full Software 2012 edition to introduce this antivirus software from Europe.

1. Installation Experience

Firstly, click to install the downloaded installation package. The installation process takes about 5-10 minutes. After clicking "Complete," we can start using the AVG Full Software 2012 edition. On the AVG 2012 homepage, several major functional modules can be seen: Anti-Virus for virus detection and removal, LinkScanner mainly provides online protection functions, email protection checks for viruses in emails. Firewall controls communication transmitted through each network port to prevent attacks. The system tools module allows viewing processes, browser plug-ins, and network connections, etc.

The Anti-Rootkit function component scans for Rootkits hidden within applications, drivers, or DLL libraries. The PC Analyzer helps users scan their computers and report potential computer errors; users need to download the AVG PC Optimization software to fix them. Identity Protection is a behavior monitoring module that uses advanced artificial intelligence technology.

2. Cloud Antivirus - Comprehensive Protection

What users value most in antivirus software is its antivirus performance. After installation, I started a full-disk scan on my computer, which only took 1 hour and 10 minutes.

In AVG 2012, cloud protection technology simultaneously uses multiple scanning engines and behavioral detection to identify brewing and previously unknown threats. Once the cloud protection technology identifies a threat, it develops a solution and updates AVG clients worldwide almost in real-time.

In the advanced settings of AVG's virus scan, we see that AVG provides multiple scanning options, including cookie tracking, registry reference files, etc. The cookie tracking feature prevents malicious programs from invading user IE cookies. Cookies that are not regularly cleaned may contain user IDs or bank account passwords and other private information.

3. Link Rating - Rejecting Fake Websites

LinkScanner is a feature of AVG's real-time protection. Currently, social networking sites and microblogs based on Web 2.0 have a large number of users. Applications based on Web 2.0 on these websites are very popular with users but are also easily exploited by hackers. The online protection function provided by LinkScanner can block web attacks, allowing users to browse social networking sites with confidence.

In the search results, the checkmark indicates that AVG has already applied security settings to the search links. According to my tests, this function can be displayed in Baidu, Google, Bing, and Yahoo search results in the IE browser.

AVG Linkscanner can display evaluations of website safety on your search engine results, providing the safety level of websites. It works by collaborating with Google, Yahoo, Bing, or other search service providers to place a safety rating. Through placing easily recognizable color codes on links, it lets you know which websites can be trusted and which are dangerous links.

4. Email Protection - Blocking Viruses

Another powerful feature provided by AVG is email protection, which can check incoming and outgoing emails, scan email attachments, and report the results to the user. It can also check unsolicited group emails (spam) in incoming emails and block viruses, phishing attacks, or other threats.

5. Rootkit Search - Rejecting Malicious Programs

A Rootkit is typically a set of malicious software used by attackers to gain administrator-level access to a computer (or even an entire network). Hackers use Rootkits to intrude and attack others' computer systems. Computer viruses, spyware, etc., often use Rootkits to hide traces. Rootkits are frequently used to conceal the tracks of Trojans or keyloggers, and ordinary antivirus software finds it difficult to detect Trojans using Rootkits as shells. Linux, Windows, Mac OS, and other operating systems can all potentially become targets of Rootkit attacks.

The figure shows AVG scanning for Rootkits. AVG offers four scanning speeds: user-sensitive, low priority, medium priority, and high priority. The speed of user-sensitive level scanning changes according to resource usage, so it is recommended to use this option.

If the installed applications or drivers contain Rootkits, by clicking "Search Rootkits," AVG can check if the user's computer has already been controlled by hackers. As early as 2007, AVG launched a Rootkit detection and removal tool called AVG Anti-Rootkit.

6. Real-Time Protection - Safeguarding Financial Security

Identity Protection (referred to as IDP for short) can simply be understood as common behavior monitoring. However, AVG's IDP employs artificial intelligence technology, enabling efficient and low-false-positive detection of unknown viruses, preventing users from being harmed by unknown viruses, thus avoiding financial losses and privacy leaks. Notably, 2011 was the year of the "Fake Antivirus Software" outbreak, and AVG considered this comprehensively in version 2012. In the IDP, it integrated scanning technology based on behavioral detection of fake antivirus software. That is to say, even if this fake antivirus software is not in AVG's virus database, IDP can still "detect" it. This technology is currently the first in the industry and can also be considered one of the highlights of AVG 2012.

7. Firewall - Preventing Malicious Attacks

The firewall function of AVG 2012 controls all communication content through each network port, meaning all receiving behaviors while browsing the internet go through AVG's firewall control to determine whether they contain malicious attacks. If there is a malicious attack, the firewall automatically intercepts it. AVG 2012 sets three firewall modes and five firewall profiles, allowing users to configure according to different needs.
