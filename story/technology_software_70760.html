Nightfire: gh0st RAT is an excellent remote control software developed by the Red Wolf team. Since the last time Red Wolf published the source code of gh0st version 2.5, it has not been open-sourced again. This update of gh0st 3.6 is open-sourced once again.

**Gh0st RAT**

**C.Rufus Security Team**

**http://www.wolfexp.net**

- The controller uses the IOCP model, and data transmission uses zlib compression.

- Stable and fast, no limit on the number of online connections; it can simultaneously control tens of thousands of hosts.

- The controller automatically detects CPU usage to adjust its working threads for stable and efficient performance.

- The host runs as svchost as a system service with a remote guardian thread, and the connection interval is two minutes.

- Heartbeat packet mechanism to prevent accidental disconnections.

- Supports two methods of going online: HTTP and DNS.

- Automatically restores SSDT (everyone knows what this function does; you can make it undetectable yourself). Administrator privileges are required to install this program.

- The controller is 279K in size with a simple interface. The generated server is unobfuscated, 106K in size, and the resources inside the EXE are compressed using UPX. Multiple servers can be installed.

- Other detailed features are left for you to discover.

**Features:**

- **File Management**: Fully emulates Radmin, allowing batch upload, deletion, download, creation, and renaming of files and folders.

- **Screen Monitoring**: The scanning algorithm can reach up to 120 frames per second. The difference algorithm is suitable for poor network conditions, providing fast transmission speed. It also allows controlling the screen, sending Ctrl+Alt+Del, clipboard operations, and seven color display modes, etc.

- **Keyboard Logging**: Logs both Chinese and English information with offline logging capability (up to 50M).

- **Remote Terminal**: A simple shell.

- **System Management**: Process management, window management, and retrieval of dial-up internet passwords.

- **Video Viewing**: View remote cameras, take snapshots, record videos, compress video, etc.

- **Voice Monitoring**: Listen to remote audio and transmit local voice to the remote side for voice chatting using GSM610 compression for smooth transmission.

- **Session Management**: Log off, restart, shut down, or uninstall the server.

- **Other Functions**: Download and execute programs from a specified URL, hide or show access to specific websites, clear system logs.

- **Geolocation**: Place the QQWry.Dat IP database file in the same directory as the program to display geographical location.

- **Cluster Control**: Simultaneously control multiple hosts and open video surveillance and other management functions at the same time.
