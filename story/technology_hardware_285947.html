Learning Experience and Insights: Experiences with the Wubi Input Method

by henguai on 2009-11-29 23:42:49

The typing principle of Wubi is: If you can type one key, never type two! Hehe :) This is about the number of key presses. It means that if you can type a character with one letter, don't use two letters or the full code; every press saved counts. This is the secret to speed, which relates to the different versions of Wubi you'll use in the future and the continuous accumulation of experience. But as a foundation for learning Wubi, I personally suggest understanding this. You don't have to use Wubi input method, but you must know what the full code is! For example, to type "我" normally, just press 'Q' and a space, but you must know that the full code for "我" is '丿T + 扌R + 乚N + T丿 = TRNT'.

Three. Basic typing conditions —— How to type using Wubi (First, download Wubi! It's recommended to download the 86 version as the basis, see [one]):

Wubi is divided into four levels: To type a word or a phrase with Wubi, you need up to four letters.

Level One Simple Code: One letter per word.

Level Two Simple Code: Two letters per word.

Level Three Simple Code: Three letters per word.

Level Four? That's the full code! Four letters per word.

Typing Method:

Methods for typing horizontal, vertical, left-falling, right-falling, and turning strokes: The encoding methods for these strokes are fixed, no need to ask why.

Basic input rules:

Horizontal stroke: GGLL

Vertical stroke: HHLL

Left-falling stroke: TTLL

Right-falling stroke: YYLL

Turning stroke: NNLL

Key names: These refer to representative characters on each key, known as key names. These characters can be seen as the name of the key, though not all key names are complete Chinese characters. For example, the key name for the X key is "纟".

Method for typing key names: Press the key where the key name is located four times.

Example: The method for typing "金": QQQQ.

Another example: The method for typing "人": WWWW.

Note: Although some words do not require pressing the same letter four times to be typed, the standard full code method for key names involves pressing the same key four times. Our suggestion is to have the concept of the full code, and on this basis, simplify the number of key presses to increase your speed.

For words within a key (excluding key names), including some radicals and components on the key: Report the location (press the letter corresponding to the location of the word) + the first stroke of the key name + the second stroke of the key name + the last stroke of the key name.

Note: Reporting the location does not count as the first stroke of the word.

Example: The method for typing "早": Early J + Vertical H + Turning N + Vertical H.

Why is it typed this way? Look at the breakdown of "早": Early J (report location) + the first stroke of "日" which is vertical "|" + the second stroke of "日" which is turning "乙" + the last stroke of "早" which is vertical "|".

The method for typing a word: The first stroke of the Chinese character + the second stroke + the third stroke + the last stroke, one stroke per letter.

If the character has more than four strokes, how is it calculated? Answer: 1st, 2nd, 3rd stroke + the last stroke, all intermediate strokes are discarded.

Example: The method for typing "幕": Grass A + Sun J + Big D + Vertical H = AJDH. Why is the last character "巾" only typed as vertical? The reason is simple, because vertical is its last stroke.

The method for typing two-character words: The first stroke of the first character + the second stroke of the first character + the first stroke of the second character + the second stroke of the second character.

Example: The method for typing "百科": Hundred DJ + Science TU = DJTU.

The method for typing three-character words: The first stroke of the first character + the first stroke of the second character + the first stroke of the third character + the second stroke of the third character.

Example: The method for typing "龙井茶": Dragon D + Well F + Tea A + Person W = DFAW.

The method for typing four-character words: Add together the first stroke letters of each of the four characters.

It’s a bit troublesome, but it’s the first stroke of the first character + the first stroke of the second character + the first stroke of the third character + the first stroke of the fourth character.

Example: The method for typing "春暖花开": Spring D + Warm J + Flower A + Open G = DJAG.

The method for typing multi-character phrases: The first stroke of the first character + the first stroke of the second character + the first stroke of the third character + the first stroke of the last character.

Example: The method for typing "中华人民共和国": China K + Huá W + Person W + Country L = KWWL.

This multi-character phrase is just a rare example contained in the Wubi lexicon. There aren't many such multi-character phrases that can be typed this way, and the method isn't very practical. If we were to nitpick and find such multi-character phrases to type, it would actually be useless. What we use most often are two-character phrases.
