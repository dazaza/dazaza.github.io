CodePlex: Microsoft Open Source website goes live

by jgbell on 2006-06-28 18:38:18

Microsoft is finally launching a new web site for open source developers --www.CodePlex.com -- with a wide range of Microsoft code licenses available for viewing and downloading. CodePlex.com is still in beta, and Microsoft has set up the site to change its "anti-open source" image and engage more with the open source community. Open source code already available from Microsoft includes the Atlas AJAX development tools suite and Iran Python Beta 1. CodePlex.com is just one step in Microsoft's open source initiative to promote open source content to make developers more familiar with Microsoft's software architecture and boost potential development/purchasing power. CodePlex was built using Microsoft Visual Studio 2005 Team Foundation Server.
