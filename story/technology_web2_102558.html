Show Document: A free live file sharing session

by xjyzhenai on 2008-11-11 20:19:27

This title looks weird. But an attractive one, right? So what's going on? Let's move on to the following:

image Show Document is a foreign file sharing service. Such services seem to be plentiful in China as well. But the difference is that Show Document supports online meetings where multiple people can browse any document together in full sync. This meeting tool, no need to download or install, just upload your files and run a free online web meeting process. All participants in the meeting can read this document, and instantly see each other's drawings, annotations, and so on. Let's elaborate:

This is the main screen of the Show Document, where you can start a new meeting or join one that is already in progress. For example, now that we have a word document that we need to discuss remotely together, let's hold a meeting. Select the file, then click the Upload button. Once the upload is complete, click the Start button.

image

A new meeting begins. In the welcome screen, enter your name and invite your team members to attend. We can see the ID number of the meeting room, tell the ID number to your team members, or send them the address of the meeting room via IM and EMAIL.

image

When everyone is here, the meeting can begin.

image

Everyone can see the contents of the word document and annotate it instantly using the pencil tool, highlighter work tool, and Type tool. There is also a chat window in the upper right corner. You can speak here. After the discussion, you can also save the annotation results as PDF documents as meeting minutes.

How's that？? Simple, right? If it helps you, then come and use it!
