AWS CodeArtiject：完全托管的软件工件存储库服务

2020-06-11 09:58:49

AWS CodeArtiact是一项完全托管的软件工件存储库服务，使任何规模的组织都可以轻松安全地存储、发布和共享软件开发过程中使用的包。CodeArtiact使您不再需要设置、操作和扩展工件管理所需的基础设施，这样您就可以专注于软件开发。使用CodeArtiact，您只需为您使用的内容付费，而无需支付许可费或前期承诺。

AWS CodeArtiact可与常用的包管理器和构建工具配合使用，例如Maven和Gradle(Java)、NPM和YAR(JavaScript)、pip和twine(Python)，从而可以轻松地将CodeArtiact集成到您现有的开发工作流中。可以将CodeArtiact配置为自动从公共构件存储库(如NPM public Registry、Maven Central和Python package Index(PyPI))获取软件包，从而确保团队能够可靠地访问最新的软件包。

IT领导者可以使用AWS CodeArtiact创建集中式存储库，用于共享其开发团队批准使用的软件包。CodeArtiact与AWS Identity and Access Management(IAM)的集成为他们提供了控制谁有权访问包的能力。此外，CodeArtiact对AWS CloudTrail的支持使领导者能够清楚地了解哪些软件包正在使用以及在哪里使用，从而可以轻松识别需要更新或删除的软件包。CodeArtiact还支持使用AWS密钥管理服务进行加密，因此客户可以控制用于加密其包的密钥。

要了解有关AWS CodeArtiact的更多信息，请查看博客，访问CodeArtiact产品页面或参考CodeArtiact文档。
