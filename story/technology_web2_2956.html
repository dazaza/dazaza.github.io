Jike Web2.0 New Army

by donyenator on 2006-06-08 10:20:45

A highly recommended online storage space, with an elegant interface and simple operations. Basic Account: Large capacity: Free users get 1GB of storage space with a 10MB limit on individual file size. Usability: Utilizes AJAX technology, has a neat and attractive appearance, very user-friendly operations, almost all functions can be completed with a single click. Multi-functionality: Supports tag function, supports multiple file upload and download methods, sharing can be set. There are also some web2.0-based features such as RSS output function, support for Netvibs modules, and synchronization with files on your computer (this feature is still under development and currently not perfect). Aggregated invitation: Has group and group file sharing functions, share the treasures you've collected with everyone! PREMIUM Account: If upgraded to a PREMIUM account, the storage space increases to 5G, no limit on individual file size, supports BLOG publishing. Sub-accounts can also be set up, which is very useful if friends need to temporarily send files to you. To upgrade to the more powerful PREMIUM account, besides paying, there's a simpler method - recommend 10 users and you'll get a free upgrade, it should be a market occupation strategy.
