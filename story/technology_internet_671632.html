SEO Optimization: Commonly Used Commands for Search Engines

by seo17514 on 2012-02-22 11:13:24

1. **site:** Indicates the inclusion status of a specific website. The format is: site:URL.

2. **cache:** Shows the last time a search engine indexed a particular website. By entering a page, you can find the most recent date and version that the crawler indexed.

3. **info:** Provides information about the inclusion of a specific website, including the latest cache, similar webpages, site links, internal links, and pages containing the domain.

4. **link:** Displays external links to a specific website, which is a major factor in obtaining rankings and weight. The format is link:URL.

5. **related:** Finds webpages particularly related to a given webpage.

6. **allinurl:** or **inurl:** Searches for specific content included in the URL title of webpages.

7. **allintext:** or **intext:** Shows all webpages that contain the specified text within the document. In SEO, it's used to discover related sites and potential links, and is useful for competitive research.

8. **allinanchor:** or **inanchor:** Finds all webpages that include the specified anchor text in their links.

9. **allintitle:** or **intitle:** Finds all webpages whose titles contain the specified text.

10. **define:** This advanced feature is rarely used by SEO professionals but is still valuable, functioning as a dictionary.

To query external links to a website, we typically use the 'link' and 'domain' commands. However, sometimes these commands don't yield the expected results. Let’s analyze what 'link' and 'domain' mean in different search engines:

**First: link:**

- **Baidu:** 'link' is not a command; it's just a regular keyword. The search result for 'link:URL' includes all webpages with this keyword phrase, just like any other normal keyword search. It does not find backlinks.

- **Google:** Finds backlinks but only shows a small portion of all backlinks to the site. We recommend using Google Webmaster Tools to check all backlinks to your site.

- **Yahoo:** Finds backlinks, but you need to add "http://" before the domain. Note that adding "www" or not can lead to different results.

**Second: domain:**

- **Baidu:** Checks webpages that include a certain keyword in their content. The search result for 'domain:sdfnhh457ker' includes all webpages with this keyword phrase. Similarly, for 'domain:seo', it refers to all webpages containing 'seo', similar to directly searching for 'SEO'. If you want to know how many webpages contain your URL, 'domain' can be helpful.

- **Google:** 'domain' is not a command; it's just a regular keyword.

- **Yahoo:** Functions similarly to 'site'. It finds all indexed webpages under the specified domain.
