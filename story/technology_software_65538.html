Usage:

1. Select the working network card in the parameter settings;

2. Check if the gateway information and local machine information are correct. If not, manually input them without saving;

3. Click on traffic detection, manual addition, or automatic scanning to fill in the target list;

4. Set up a proxy, preferably different from the target. After setting up the proxy, only the proxy host will receive fake traffic;

5. Select the host whose internet speed you want to limit;

6. Start control.
