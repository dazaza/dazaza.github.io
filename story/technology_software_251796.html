5 Powerful Tools You Can't Miss for Website Maintenance

by dang000 on 2009-10-25 11:56:40

Dreamweaver, Photoshop, Fireworks, and Flash will not be reintroduced here. Instead, I will recommend a few less commonly used but very useful and efficient website maintenance software options.

1. **FlashFXP**: If you need to upload files via FTP, FlashFXP is the way to go.

FlashFXP is a powerful FXP/FTP software that incorporates some of the best features from other top FTP programs. For example, it can compare folders like CuteFTP, supports color-coded text display; it allows multi-folder file selection similar to BpFTP and caches folders; its interface resembles LeapFTP, and even its design concept is somewhat similar. It supports folder (including subfolders) file transfer and deletion, uploads, downloads, and third-party file resuming. You can skip specific file types and only transfer necessary files, customize different file type display colors, cache remote folder lists, support FTP proxies and Socks 3&4, prevent idle timeouts to avoid being kicked out by sites, show or hide "hidden" attribute files/folders, and use passive mode for each site.

**Green Version Download Link for FlashFXP**: [http://www.greendown.cn/soft/478.html](http://www.greendown.cn/soft/478.html)

Additionally, you can also choose an online FTP tool: Online FTP Management Tool: Manage your FTP online.

2. **Webmaster Toolbox**

The Webmaster Toolbox is a comprehensive site information query tool created with Flash. Its main functions include checking a site's Google PageRank, Alexa global ranking, Sogou index, Chinese website rankings, as well as backlink inclusion in major search engines. Later versions added practical tools frequently used by webmasters, such as domain queries.

Key Features: Easy to use, open the tool and start querying immediately. It directly accesses various target interfaces without needing to read data from third-party sites (except Google, Alexa, Sogou, and major search engines). The query speed is fast. Additionally, it can store query conditions, providing convenience for users who frequently check information on the same site.

**Webmaster Toolbox**: [http://www.flashplayer.cn/webmaster-toolbox/](http://www.flashplayer.cn/webmaster-toolbox/)

3. **Online HTML Editor**:

If you want to edit HTML but don't have Dreamweaver handy, an online HTML editor can help you make simple edits. Although it lacks the full functionality of Dreamweaver, it doesn’t require downloading and is portable, which is incomparable.

**Online HTML Editor**: [HTML Online Editor](html在线编辑器)

4. **Free Website Statistics Tools**

With website statistics tools, we can better understand our site's traffic, visitor demographics, referral sources, search engine keywords, etc.

Currently available free website statistics tools include: cnzz.com, 51.la, and Google Analytics.

If you'd like to add a visible counter so everyone can see the visit count, you can use Demi Counter, which offers 60 stylish skins that can be easily changed directly from the backend.

5. **Website Broken Link Checker**

Sometimes after modifying pages, deleting them, or renaming web pages, we need a tool to detect broken links on the website.
