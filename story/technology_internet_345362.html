Factors Affecting the Image Quality of Files in PCB Copying

by famang5911 on 2010-04-14 14:12:19

I. Scanning Process

In the PCB copy board process, there is no doubt that PCB scanning is the first step in all procedures. After obtaining a complete PCB board, it must first go through computer scanning to back up relevant parameters and the original PCB layout.

After depaneling, with the separated bare PCB board in hand, the formal copying stage begins, starting with scanning to store and record the PCB image. It’s worth mentioning that to ensure the clarity of related parameters on the PCB after scanning, any dirt or residual tin on the surface of the PCB should be removed before scanning.

Since PCB copying involves the issue of copying accuracy, for circuit boards like mobile phone boards which require high precision, achieving a high-precision PCB layout during the scanning process requires accurate selection and setting of the scanner values, ensuring the accuracy of the original scanned image. The accuracy of the copied board mainly depends on the original scanning accuracy.

Here, it's necessary to introduce the concept of DPI. DPI refers to the number of dots per inch. This means the distance between two points on the scanned image is 1000/DPI, measured in mils. In the case of copying mobile phone boards, setting the DPI to 1000 during PCB scanning results in a distance of 1000/1000 = 1mil between two points on the image, meaning the accuracy at this point is 1mil for the PCB board.

It's important to note that the higher the precision of the scanned image, the larger the image file size, requiring more advanced hardware. Therefore, when setting the DPI, it should be adjusted according to the specific conditions of the original board to ensure the best possible effect for the subsequent steps in the copying process.

II. Software Technology

After the original PCB board has been scanned, based on the original image, it is necessary to use PCB copying software to complete the file diagram.

In terms of software types, we have fully functional copying software, which is very important because the choice of software can be reflected in the final exported PCB electronic layout and the derived schematic diagrams.

After selecting well-functioned software, to ensure perfect results, especially for double-layer and multi-layer board copying, one needs rich experience and proficient skills. Since in the same double-layer or multi-layer PCB board, the holes are in the same position but the line connections differ, when depicting the original board's wiring rules in the copying software, the already copied top layer PCB file of the double-layer board can be overlaid on the scan image of the other layer. With the vias overlapping, set the top layer traces and silkscreen to not display, then trace out the lines of the other layer according to the via positions. Thus, the exported PCB file will contain the data from both sides of the double-sided panel.

For multi-layer boards, the principle is similar, except after drawing the outer layer PCB file, the outer layer needs to be sanded off with sandpaper to expose the inner layer wiring rules, and then using the same skillful method within the copying software, the internal layers can be copied accordingly.

With advanced scanning techniques combined with mature software operation skills, strictly following the copying process, you will find that the exported PCB file diagram or PCB electronic layout will maintain consistency with the original PCB board in terms of wiring rules, via positions, line directions, etc. Conversely, any mistake in any intermediate step, whether it's the accuracy setting in the scanning process or the judgment and depiction of the routing rules and function modules by the copying software, will affect the final effect of the PCB file diagram.

III. Testing the File Diagram

For the completed file diagram, to ensure compliance, the last step should involve testing. One method of testing double-sided panel file diagrams is to print the surface two-layer file diagrams onto transparent film using a laser printer, then compare the film with the original board to check for consistency. Testing should also include evaluating the electronic technical performance of the PCB board to ensure its functionality matches the original board.
