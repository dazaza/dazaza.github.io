Official version: Fineplus 1.0 QQ2007 Auxiliary Software

1. Four types of IP display, with built-in powerful IP format customization.

2. Optional personalized signature display, giving you a clean friend list.

3. Selectable camera policy to protect your privacy.

4. Optional side panel closing, which can be set separately for friends and groups.

5. Powerful boss key settings, optimized the original QQ hotkey part so that multiple QQ accounts can share one hotkey.

6. For most friends who only chat, you can choose to prohibit other component programs from running simultaneously.

7. Can close the display of service icons.

8. A series of filtering settings, providing you with a relatively clean chatting environment.

9. Flexible menu components, allowing you to add your commonly used external links, programs, or even documents at will.

10. Selectable QQ Show display settings, saving your precious network bandwidth and memory.

11. Customizable file cleanup.

12. Automatically identifies and supports multiple IP library formats.

13. Two loading methods, allowing you to choose as needed.

IV. File List

fineplus.dll - Core file of finelus, used to implement auxiliary functions.

fphelper.dll - Auxiliary dll, used to call internal QQ functions, responsible for reading nicknames and other information.

fineplus.ini - Settings save file.

version.ini - Version list for QQ/TM etc.

menu.rc - Menu customization file.

qqwry.dat - IP address database.

(or coralwry.dat)

doc directory - Various instructions.

fineplus.exe - Fineplus guide form load library, used to start QQ and load fineplus.dll.

FPset.exe - Small tool to copy the fineplus load library (only required for forwarding form).

msimg32.dll - Fineplus forwarding form load library, used to load fineplus.dll.
