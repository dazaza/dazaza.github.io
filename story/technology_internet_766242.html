Arkoon will take over from Microsoft to continue supporting Windows XP.

by anonymous on 2013-09-08 20:04:46

According to a report by the American tech blog ZDNet, after Microsoft ended support for the old operating system Windows XP on April 8, 2014, thousands of enterprise users seemed determined to stick with this platform. As a result, their IT departments would have to seek non-Microsoft support for Windows XP. Arkoon, a company usually close to Microsoft, has explicitly stated that it will take over and continue supporting the operating system after Microsoft ends its support.

Arkoon is based in Paris, France, and falls under Cassidian Security Company, part of the European Aeronautic Defense and Space Group EADS. It offers ExtendedXP (abbreviated as EXP) products to enterprise users who are unable or unwilling to leave Windows XP after its end-of-life, continuing to support their usage of Windows XP.

This new product will undergo public testing in October this year. Bob Foley, CEO of Matrix Global Partners, Arkoon's U.S. distributor, likened the product to a "shelter." "Our solution addresses issues that haven't yet emerged. However, we hope to let people know that this can buy them time," he said.

Foley mentioned that many enterprise users were troubled by the final deadline set by Microsoft for supporting Windows XP because they needed several more years to upgrade all desktops and laptops. "Unfortunately, the deadline set by Microsoft requires them to update at a faster pace, but this exceeds their budget. If they knew about this option, then they could readjust their budget," he said.

### Sticking with Windows XP

There is a category of enterprise users in the retail industry using Windows XP who are determined to stick with it.

"They say, 'We have Windows XP installed in our stores. We don't intend to change it, nor do we want to change it. Installing cash registers running Windows 7 or Windows 8 in the store brings us no benefit. We will never install it.' Therefore, they have no intention of upgrading and will continue sticking with Windows XP," Foley said.

Arkoon's product uses its StormShield endpoint protection technology, installing an 8MB security agent on Windows XP computers.

The IT department also has a management platform with an SQL database that can track nodes and agents in real-time and access external monitoring services to identify vulnerabilities in the Windows XP operating system, which was first released to manufacturers in August 2001.

This monitoring service will call upon various resources to discover new vulnerabilities and upgrade the XP security templates stored on the company's servers. "It will automatically upgrade all protected security agents and continuously provide protection while enterprise users use Windows XP. The security agent and monitoring team are the guarantees," Foley said.

### Future Vulnerabilities in Windows XP

Foley mentioned that Arkoon has always had a close relationship with Microsoft, but he is not sure whether this will help them obtain information about Windows XP vulnerabilities from Microsoft in the future.

"In this particular area, we're not sure how much information Microsoft will share with others. Our assumption is that Microsoft will not share information because they want users to quickly migrate to newer operating systems. The more information we collect from Microsoft, the better, of course," he said.

However, even if we cannot get information about Windows XP threats and vulnerabilities from Microsoft, we should be able to find clues elsewhere.

"If we pay attention to the Tuesday patches Microsoft provides for Windows 7 in March 2014, we will see that potential vulnerabilities in Windows XP can be traced," Foley said. "Therefore, even if Microsoft does not wish to share information about Windows 7 vulnerabilities, I think we can still find some clues in Microsoft's updates to newer operating systems."

### Alerts and Entry Points

Arkoon will assist users in configuring and installing the software. Activities from the agent to the management platform are all under the supervision of the software. If there is an entry point violation or a specific event occurs, the software will issue an alert.

"It is not application software. We don't plan to promote it to PC users, but you can run it in a virtual system. If there is a violation of your set policies, it will pop up warning messages," Foley said.

He expects that EXP will work together with a series of other security measures to protect Windows XP.

"We hope all Windows XP users have a multi-layered security strategy. But without something like Microsoft's Tuesday patch, all these layers combined cannot fully protect users. These security measures help somewhat, but our software is the ultimate guarantee, the last line of defense," he said.

Companies starting to trial EXP from October may evaluate its ease of use and its impact on Windows XP performance. They might also test the software using Microsoft's previous Tuesday patches and simulate hacker attacks.

He believes that EXP will help those enterprise users sticking with Windows XP. However, he also worries that EXP might affect their compliance certification. "The final test will be conducted by customers hiring auditors after April 2014 to ensure that EXP becomes a compensating control. We've consulted many experts in this field, and they all consider EXP as a compensating control, so we feel optimistic about its application," Foley said.

For contracts covering hundreds of PCs, the annual service fee for EXP agents is approximately $15 per computer. Larger numbers of PCs can offer some discounts.

The software comes in three different versions: the first includes the Windows XP operating system, IE browser, and Microsoft Office suite; the second provides Adobe Flash, Mozilla Firefox, and Google Chrome browsers; the third is specifically designed for Adobe Reader and Java. Additionally, it can provide custom versions.

"If you purchase 100 copies, the service price is $15 per computer. If you purchase 50,000, then the price will be lower," Foley said.

He clearly stated that his company would not oppose Windows 7 or Windows 8 or discourage people from migrating to these platforms. "But, we've seen many people unwilling to migrate. After April 2014, there will still be 500 million to 600 million people using Windows XP. If they have a secure way to use it, then it's not necessarily a bad thing," Foley said.
