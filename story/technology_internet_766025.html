Microsoft officially provides Windows Server 2012 R2 to hardware partners

by anonymous on 2013-09-04 14:10:59

Author: Brad Anderson, Corporate Vice President of Microsoft Server and Tools Division

As mentioned earlier, we are eagerly looking forward to delivering Windows Server 2012 R2 to our partners and customers. I am proud to announce that Microsoft has officially released Windows Server 2012 R2 to hardware partners!

This means we have handed over the software to hardware partners for them to complete final system validation. This is also the last step before we deliver the next generation of Windows Server into your hands.

While every product launch is worth celebrating (believe me, there will definitely be a party in Redmond to celebrate), this release is particularly exciting because we've accomplished these achievements in a very short time. Windows Server 2012 R2 includes countless incredible new features spanning virtualization, storage, networking, management, access, and information protection.

No matter how you look at it, achieving so much innovation within just one year after the release of Windows Server 2012 is no small feat!

Many readers have noticed that this release differs significantly from those in previous years. In the past, Microsoft allowed access to the software via MSDN and TechNet subscriptions shortly after RTM was released. However, this time we are doing things differently - partly because the development and release of this product were much faster than previous ones; partly because we want to provide users with the highest quality product possible. Therefore, we decided to conduct final verification first before releasing the new version. This way, users can use R2 with greater confidence to build their private cloud and hybrid cloud architectures while enjoying the best user experience.

On behalf of the Microsoft server engineering team, I am proud to recommend this version to you. The extensive, powerful, and interoperable nature of the R2 product once again strongly demonstrates the importance of universal engineering standards (Common Engineering Criteria).

Additionally, it's worth mentioning that the next upgrade for Windows Intune will be released when Windows Server 2012 becomes officially available, and we are also actively preparing for the release of System Center 2012 R2.

Sincere thanks to everyone who provided feedback during the preview period. Without you, we wouldn't have been able to accomplish all of this!
