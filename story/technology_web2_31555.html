webchat- A newly launched webchat software that supports smart-mouthed dolls and second-degree relationships

by lola on 2007-10-08 23:51:53

WEBCHAT is a new type of instant messaging software running in the web browser, users do not need to download any program can be used, WEBCHAT supports sending messages to offline users, fully supports sending html (hypertext) messages, supports gif, smart mouth doll and other multimedia expressions.

WEBCHAT is a typical social network software, which has extremely powerful GROUP functions, including whiteboard and expert functions. It also allows users to see contacts of contacts, and users can meet friends of friends, which is a living application of "six degrees of relationship theory" in WEBCHAT.

Features of WEBCHAT

+ No need to download any client controls or software, completely executed in the browser, WEBCHAT based on Ajax technology development, less memory consumption, high communication efficiency

+ Full support for sending html messages, WEBCHAT is the world's rare WEBIM that fully supports HTML

+WEBCHAT can not only send gif emojis, but also send smart mouth dolls and magic emojis

+WEBCHAT group function is extremely powerful, the implementation of whiteboard and expert functions, all users have the right to create groups

+ Chat records are saved on the server and can be permanently retained, chat records can be cleared at any time or saved to the local computer

+WEBCHAT supports contact groups. Users can create, modify, and delete contact groups, and can easily transfer contacts to other groups

+ Users can view the open profile of a contact, modify the remarks of a contact, and view the contact list of a contact, and users can meet friends of friends
