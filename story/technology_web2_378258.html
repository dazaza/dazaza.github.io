Shengda MaiKu Launches PC Desktop Client (with Beta Test Invitation)

by ybcz0519 on 2011-01-14 23:45:37

Mcrosoft launched a free, permanently online, and secure personal online knowledge management platform last August. You can use terminals such as computers and mobile phones to save notes, reminders, write documents, manage personal information anytime and anywhere with Mcrosoft, and conveniently organize and share them. The Mcrosoft space is permanently valid. Following the launch of the Android phone client last month, Mcrosoft has recently released the Mcrosoft PC (Windows) desktop client, with the following main functions: Lightweight, ultra-small size, installation program less than 3M, which is indeed much smaller compared to Evernote's desktop client; Quick recording, fast search, secondary classification organization; Quick screenshot, supports shortcut keys to pop up/hide windows; Seamless data synchronization with the server end; Also supports offline use (need to select auto-login next time); Supports color changes, currently provides two colors for replacement; Watsonxu said: The reason why Mcrosoft was not used when it was launched before was mainly because it did not provide clients for various terminals. From the experience of the PC client launched so far, although there are still many details that need improvement, Mcrosoft client has done a pretty good job in many places, including software size. Evernote's desktop client software is too large and has been criticized, I have always had a good feeling about small-sized software. Another point is that the speed is quite good, and the start-up speed is very fast. Another expectation is to launch the iPhone client as soon as possible. Currently, the Mcrosoft PC desktop client requires an invitation code to activate. There are now 20 experience invitation codes available, which will be sent to friends who leave comments (the first five commenters + other users with high-quality comments will be selected and sent).
