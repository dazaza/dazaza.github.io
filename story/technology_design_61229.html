Fashionable -- Make the countdown display for the 2008 Beijing Olympic Games

by eeyye on 2008-04-09 00:10:20

In order to always welcome the arrival of the Beijing 2008 Olympics, you can DIY a countdown display on your computer when it is turned on. It will remind you at any time and surely add a unique charm. Below, let's use Yongzhong Office 2004 Enhanced Edition together to create a countdown display for the 2008 Beijing Olympics.

1. Input Content

Start Yongzhong Office 2004, create a new file, and select "Spreadsheet" under "Worksheet 1". In cell A1, input "距2008年北京奥运会还有" (which translates as "Until the 2008 Beijing Olympics remains"), in cell C1 input "天" (days), and in cell B1 input the formula "=DATE(2008,8,8)-TODAY()".

Here, the format of cell B1 must be "General", starting with "=", parentheses and date separators must be half-width characters; otherwise, it will display as text rather than days. The meaning of the two functions used in cell B1: DATE(year.month.day) function represents returning the serial number of a specified date, here it is August 8, 2008; TODAY() represents returning the current date from the system clock, which is a parameter that changes over time, updating every day when the computer is turned on; the difference between the parameters on both sides of the "-" sign is the number of days we need for the time interval.

After completing the input of the countdown display content, you can customize according to your preferences, such as changing the font, size, color of the text, and the position of the content within the cells.

Finally, "Save" the file named "2008年北京奥运会倒计时.eio" and specify the storage path.

Modify Win.ini File

2. Startup Settings

The key lies in how the countdown display starts with each boot. If you are using the Windows 98 system, please open the Windows folder in the root directory of drive C, find the Win.ini file, and enter the full path of the "2008年北京奥运会倒计时.eio" file after "load=" under [windows] (as shown in Figure 2). If there is already content after "load=" under [windows] in your computer's Win.ini file, leave a space before entering the full path of this file. If you are using Windows XP or Windows Server 2003 systems, please place "2008年北京奥运会倒计时.eio" in "C:\Documents and Settings\Administrator\「开始」菜单\程序\启动" (assuming C is the system drive).

Countdown Display

After completing the above operations and restarting the computer, the "2008年北京奥运会倒计时" display we made will automatically appear on the screen (as shown in Figure 3), and the number of days will change automatically. Don't you feel a great sense of accomplishment after seeing it!

Tip: By referring to this method, we can also design reminder signs for any commemorative day, such as the birthdays of elders, spouses, children, or even important event times.
