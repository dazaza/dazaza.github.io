Lispе…Ҙй—Ё

2020-12-21 19:06:26

еңЁHackerNewsпјҢ2019е№ҙе’ҢLispзҡ„ж•…дәӢзҒ«зҲҶгҖӮиҷҪз„¶еӯҳеңЁеӨҡз§ҚLispж–№иЁҖпјҢдҪҶCommon LispжҳҜжңҖеҸӨиҖҒе’ҢжңҖеҸ—е…іжіЁзҡ„ж–№иЁҖгҖӮеӣ жӯӨпјҢеҰӮжһңжӮЁеёҢжңӣејҖе§ӢдҪҝз”ЁCommon LispиҝӣиЎҢзј–зЁӢпјҢйӮЈд№ҲжңүдёӨз§ҚйҖүжӢ©пјҡ

дҪҝз”Ёroswellе®үиЈ…еӨҡдёӘLispе®һзҺ°+жһ„е»әе№¶жҚҶз»‘Lispеә”з”ЁзЁӢеәҸгҖӮ Roswellй»ҳи®Өжғ…еҶөдёӢдјҡе®үиЈ…SBCLгҖӮ

еҜ№дәҺжҲ‘жқҘиҜҙпјҢжҲ‘дёҚжғід»…д»…дёәдәҶејҖе§ӢдҪҝз”ЁLispиҖҢеӯҰд№ е…Ёж–°зҡ„ж–Үжң¬зј–иҫ‘еҷЁпјҢжүҖд»ҘжҲ‘йҖүжӢ©дәҶзҪ—ж–ҜйҹҰе°”гҖӮиҜ·йҒөеҫӘжңүе…ідёәжӮЁзҡ„е№іеҸ°е®үиЈ…Roswellзҡ„иҜҙжҳҺгҖӮе°ұжҲ‘иҖҢиЁҖпјҢжҲ‘еҶіе®ҡйҖҡиҝҮд»ҘдёӢж–№ејҸе°Ҷе…¶е®үиЈ…еңЁLinuxи®Ўз®—жңәдёҠзҡ„$ HOME / binдёӢпјҡ

$ sudo apt install rlwrap $ mkdir $ HOME / bin $ git clone -bзүҲжң¬https://github.com/roswell/roswell.git$ cd roswell / $ sh bootstrap $ ./configure --prefix = $ HOME / bin $ make $е®үиЈ…

зЎ®дҝқе°Ҷroswellи·Ҝеҫ„ж·»еҠ еҲ°жӮЁзҡ„гҖң/ .bashrcдёӯпјҲиҜ·жіЁж„ҸпјҢroswellдјҡе°Ҷе…¶иҮӘиә«е®үиЈ…еңЁгҖң/ bin / binдёӢпјҢеҰӮжһңжӮЁи§үеҫ—еҫҲеҘҮжҖӘпјҢиҜ·е°Ҷе…¶е®үиЈ…еңЁ$ HOME / appsжҲ–зұ»дјјзҡ„ж–Үд»¶дёӢпјүпјҡ

зҺ°еңЁпјҢжҲ‘д»¬е°Ҷи®ҫзҪ®VSCodeжқҘиҝҗиЎҢlispгҖӮйҰ–е…Ҳе®үиЈ…vscode-lispжү©еұ•гҖӮжү“ејҖдёҖдёӘж–°ж–Үд»¶пјҢе№¶еңЁе…¶дёӯй”®е…Ҙд»ҘдёӢеҶ…е®№пјҡ

з„¶еҗҺеңЁVSCodeеҶ…йғЁеҗҜеҠЁдёҖдёӘз»Ҳз«ҜгҖӮеңЁз»Ҳз«ҜиҝҗиЎҢrlwrap ros runд»ҘеҗҜеҠЁREPLпјҲиҜ»еҸ–/иҜ„дј°/жү“еҚ°еҫӘзҺҜпјүгҖӮйҖүжӢ©дёҠйқўе®ҡд№үзҡ„еҠҹиғҪпјҢз„¶еҗҺдҪҝз”ЁвҖңеңЁжҙ»еҠЁз»Ҳз«ҜдёӯиҝҗиЎҢжүҖйҖүж–Үжң¬вҖқгҖӮд»Һе‘Ҫд»ӨйқўжқҝпјҲF1пјүиҝҗиЎҢжӮЁзҡ„д»Јз ҒпјҒиҜ·жіЁж„ҸпјҢжӮЁеҸҜд»ҘйҖҡиҝҮд»ҘдёӢж–№ејҸйҖҖеҮәREPLпјҡпјҲSB-EXTпјҡEXITпјүгҖӮ rlwrapе®һз”ЁзЁӢеәҸдјҡи®°дҪҸд»ҘеүҚй”®е…Ҙзҡ„е‘Ҫд»ӨпјҢиҝҷдјҡдҪҝREPLдҪ“йӘҢжӣҙеҘҪгҖӮ

Roswellзҡ„жғҠдәәд№ӢеӨ„еңЁдәҺе®ғе…·жңүи„ҡжң¬/жһ„е»әеҠҹиғҪпјҢеҸҜи®©жӮЁиҪ»жқҫеҲҶеҸ‘еә”з”ЁзЁӢеәҸгҖӮиҰҒжҹҘзңӢе®һйҷ…жғ…еҶөпјҢиҜ·йҰ–е…ҲйҖҡиҝҮros init hello-worldеҲӣе»әroswellи„ҡжң¬гҖӮз„¶еҗҺж·»еҠ д»ҘдёӢд»Јз ҒпјҢд»ҘдҪҝжӮЁзҡ„и„ҡжң¬еҰӮдёӢжүҖзӨәпјҡ

пјғпјҒ/ bin / shпјғ|-*-modeпјҡlisp-*-| ## | exec ros -Q-$ 0пјҶпјғ34; $ @пјҶпјғ34; |пјғпјҲprogn ;; init formпјҲrosпјҡensure -asdfпјү;;пјғ+ quicklispпјҲqlпјҡquickloadпјҶпјғ39;пјҲпјүпјҡsilent tпјүпјүпјҲdefpackageпјҡros.script.hello-world.3774807541пјҲпјҡuseпјҡclпјүпјүпјҲеҢ…иЈ…еҶ…пјҡros.script .hello-world.3774807541пјүпјҲdefun helloWorldпјҲпјүпјҲж јејҸtпјҶпјғ34; Hello worldпјҶпјғ34;пјүпјүпјҲdefun mainпјҲпјҶamp; rest argvпјүпјҲdeclareпјҲignorable argvпјүпјүпјҲhelloWorldпјүпјү;;; vimпјҡи®ҫзҪ®ft = lisp lispпјҡ

зҺ°еңЁпјҢжҲ‘д»¬еҸҜд»Ҙз®ҖеҚ•ең°йҖҡиҝҮros hello-world.rosиҝҗиЎҢжӯӨи„ҡжң¬пјҢдҪҶжӣҙжңүи¶Јзҡ„жҳҜпјҢжҲ‘д»¬е®һйҷ…дёҠеҸҜд»ҘйҖҡиҝҮros build hello-world.rosпјҶamp;пјҶamp;зј–иҜ‘дәҢиҝӣеҲ¶ж–Үд»¶гҖӮ гҖӮ/дҪ еҘҪпјҢдё–з•ҢгҖӮ

жңүи¶Јзҡ„жҳҜпјҢжҲ‘дёҚжҳҜCommon Lispзҡ„е®Ңе…ЁиҸңйёҹпјҢе®һйҷ…дёҠжҲ‘жҳҜеңЁ15е№ҙеүҚзҡ„жң¬з§‘иҜҫзЁӢдёӯеҜ№е…¶иҝӣиЎҢзј–зЁӢзҡ„гҖӮжҲ‘е’ҢдёҖдҪҚеҗҢдәӢеңЁеҜ»и·Ҝй—®йўҳдёҠе°Ҷе®һйӘҢйҒ—дј з®—жі•дёҺиҡҒзҫӨдјҳеҢ–з®—жі•иҝӣиЎҢдәҶжҜ”иҫғгҖӮдёҖж—ҰиҝҗиЎҢдәҶLispпјҢжҲ‘е°ұжү“ејҖдәҶйЎ№зӣ®пјҢе№¶еңЁREPLдёӯеҹәжң¬жү§иЎҢдәҶants.lispпјҢе…ҲиҝҗиЎҢдәҶпјҲINITIALIZE-ANT-WORLDпјүпјҢз„¶еҗҺиҝҗиЎҢдәҶпјҲDISPATCH-ANTSпјүе’ҢvoilaпјҢжҲ‘зҡ„иҡӮиҡҒеҸҜд»ҘжүҫеҲ°д»–д»¬зҡ„йЈҹзү©дәҶпјҒ

жІЎжңүиҝӣдёҖжӯҘйҳ…иҜ»зҡ„жё…еҚ•пјҢжІЎжңүд»»дҪ•ж–°зҡ„Lispе…Ҙй—ЁжҢҮеҚ—е°ҶдёҚе®Ңж•ҙпјҢиҝҷе°ҶдҪҝжӮЁеңЁжҺҘдёӢжқҘзҡ„100е№ҙйҮҢйғҪеҫҲеҝҷпјҢеӣ жӯӨпјҢжҲ‘д»¬ејҖе§Ӣпјҡ

йҖҡеҗ‘Lispзҡ„йҒ“и·Ҝ-Steve Loshзҡ„е…Ҙй—ЁжҢҮеҚ—гҖӮд»ҘдёӢжҳҜиҜҘеё–еӯҗдёӯжңүз”Ёзҡ„дҝЎжҒҜзҡ„дёҖе°Ҹж®өпјҡзі»з»ҹжҳҜд»Јз Ғзҡ„йӣҶеҗҲпјҢжңүе…іеҰӮдҪ•еҠ иҪҪиҜҘд»Јз Ғзҡ„иҜҙжҳҺпјҢдҫқиө–йЎ№еҲ—иЎЁе’Ңе…ғж•°жҚ®гҖӮе®ғд»¬дёҺеҢ…иЈ…жӯЈдәӨгҖӮ

йЎ№зӣ®жҳҜпјҶпјғ34; stuffпјҶпјғ34;зҡ„й«ҳзә§йӣҶеҗҲгҖӮдҫӢеҰӮд»Јз ҒпјҢж–ҮжЎЈпјҢд№ҹи®ёиҝҳжңүдёҖдәӣеӣҫеғҸиө„дә§зӯүгҖӮе®ғд»¬пјҲеӨ§йғЁеҲҶпјүдёҺзі»з»ҹжӯЈдәӨгҖӮ

иҜҘжҢҮеҚ—иҝҳеҜ№еә“иҝӣиЎҢдәҶеҫҲеҘҪзҡ„еӣһйЎҫпјҢз»қеҜ№еҖјеҫ—дёҖиҜ»гҖӮ жҺЁиҚҗзҪ—ж–ҜеЁҒе°”зҡ„еҸҰдёҖзҜҮж–Үз« пјҢе№¶жҸҗдҫӣжңүе…іAtomйӣҶжҲҗд»ҘеҸҠйЎ№зӣ®е’Ңеә“з®ЎзҗҶзҡ„иҜҙжҳҺгҖӮ еҘҪеҗ§пјҢе°ұжҳҜиҝҷж ·-LispеҫҲжјӮдә®пјҢжҲ‘жғіжҠҠжҲ‘зҡ„еӨҙзј еңЁеӨ§зәҰ15е№ҙзҡ„зңӢиө·жқҘдёҚеӨӘзіҹзі•зҡ„д»Јз ҒдёҠпјҢйӮЈе°ұеҺ»еҮҪж•°ејҸзј–зЁӢеҗ§пјҒ
