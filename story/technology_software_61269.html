FlashGet 2.0 official version is now live!

by yinyin on 2008-04-09 09:20:45

After more than half a year, the official version of FlashGet 2 is finally here! During this period, six BETA versions and one preview version were released. The update logs also show that our product and technical staff have indeed made a lot of efforts.

We would like to express our gratitude to those who provided feedback on bugs and suggestions. We hope both new and old users of FlashGet will give us more attention and support. Thank you all!

The official version 2.0 is the truly stable version since the release of version 2.0, so we recommend everyone to update as soon as possible to enjoy the extraordinary experience brought by the official version.

Update details for Official Version 2.0 (1177) (April 8, 2008)

========================

+ Optimized startup speed

+ Optimized CPU and memory usage during downloads

+ Security Center now supports Jiangmin, Norton Anti-Virus Enterprise Edition, Norton 360, Trend PC-Cillin, AntiVir, etc.

+ Added prompt for "integrity check" for BT tasks

+ Added prompt when deleting categories

+ Added resource recommendation prompt window

+ In Options -> General, the option "Prevent computer from entering sleep mode while downloading" is unchecked by default

* Fixed the issue where 1.x folders cannot be fully imported

* Fixed the problem with saving paths when importing 1.x tasks

* Fixed the issue where the number of main site threads could not be saved

* Fixed the problem where the last downloaded category could not be saved, and left-click activation could not be displayed

* Fixed the issue where log, download list, and category window size adjustments could not be saved, and the window would revert to its original state upon restarting FlashGet

* Modified the left-side icon representing the "Events" page in "Options"

* Fixed the issue where re-downloading BT tasks might cause crashes

* Fixed the problem where download progress might sometimes be negative

* Fixed the interface drawing issue for floating windows under certain circumstances

* Fixed the issue where the number of main site threads could not be saved

* Fixed the black background issue for floating windows upon startup under certain conditions

+ Added global ranking to client user information

+ Improved the user experience for the bookmarking function

+ Corrected the points rules, improved the detailed rules, and set an upper limit for points

+ Added the "My Toolbox" feature, which can check the relevant versions of installed software on the computer
