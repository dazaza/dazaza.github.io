Microsoft: Would you still stick to WinXP after seeing these data?

by anonymous on 2013-11-16 17:38:14

Windows XP will stop receiving support services on April 8th next year, after which the system will face serious security risks. Microsoft has already reminded users more than once to abandon this operating system that is over 12 years old as soon as possible, but the effect has not been ideal. Now, Microsoft reminds everyone again: it's time to give up this old system - the latest version of Microsoft's security report shows that Windows XP is still a hotspot for malicious software.

As shown in the figure, the data on the left shows the probability of various versions of Microsoft's operating systems encountering malicious programs. From the chart, we can see that the rate of encountering malware on Windows XP is lower than on Vista/7 and slightly higher than on Windows 8.

However, the data on the right is quite different. The right side shows the probability of each system being ultimately infected after encountering malicious programs. Among every 1000 computers using Windows XP, an average of 9.1 are detected as being infected with malicious programs. This is significantly higher than the 5.5‰ for Windows Vista and 4.9‰ for Windows 7, and six times higher than the 1.6‰ for Windows 8.

Holly Stewart, senior program manager at Microsoft's Malware Protection Center, said that currently, Windows XP is still installed on 21% of computer devices worldwide, meaning two out of every ten computers are running the Windows XP system. Even in the United States, this number reaches 13%, which is clearly too high for an operating system that has reached 12 years of life.

Recently, Google and Mozilla both announced that their browser products will continue to support the Windows XP system after April 8th next year. When asked about her opinion on this matter, Stewart stated that these companies can certainly continue to support Windows XP, but after April 8th next year, it will become more vulnerable to attacks.

In addition, Stewart also released some data showing the infection rates of malicious software on the Windows XP SP2 system and the SP3 system. The SP2 infection rate is 66% higher than SP3, and the former is precisely the version no longer supported by Microsoft.

Given all the above data, it might really be time for current users and businesses still using Windows XP to consider upgrading.
