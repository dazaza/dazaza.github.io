The fonts above are needed for localization, please download them in advance. Both must be downloaded.

Below is a detailed tutorial that will guide you step by step:

Copy the downloaded files to the C:\Windows\Fonts directory.

Below is the detailed tutorial:

1. Please update your client to the latest version. Win7 users should consider carefully before using it as some Win7 users may encounter issues with character display. Ensure the client is updated to the latest version and then carefully follow the steps below.

2. Update the game to its latest version. The version number as of March 18th is: 39.

3. Open the Dragon Nest folder, use Notepad to open the "version.cfg" file, change the value to 34, save and exit! (Note: Remember the original value.)

4. Update the game. Open the game installation directory and when the file "Patch00000036.pak" appears, close the game update. (Note: Close the update once this file appears.)

5. Open the "hosts" file located in C:\windows\system32\drivers\etc using Notepad. After opening the hosts file, add the following line at the end: "119.145.138.36 dragonnest.nefficient.co.kr", save and exit. (Note: This step is related to Dragon Nest free external plugins, StarCraft STL, etc., visit www.lzgfz.com for more information.)

6. Use Notepad to open the "version.cfg" file in the game installation directory, change the number from 35 to 20, save and exit. Then update the game again; the update size will be approximately 9.8MB. Do not close the client after the update is complete.

7. Reopen the "hosts" file and add a "#" before "119.145.138.36 dragonnest.nefficient.co.kr", save and exit.

8. Reopen the "version.cfg" file and change the value back to 39, save and exit.

9. Delete the "Patch00000036.pak" file in the game installation directory.

10. Launch the game directly.
