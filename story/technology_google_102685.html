Google Search Server Virtual Release - Run the search engine on your own computer

by yinyin on 2008-11-12 13:04:50

Google's mission is to integrate the world's information and make it accessible and useful to everyone. For the information behind the firewall that is out of reach of Google's Search crawlers, we offer the Google Search Appliance (GSA) software and hardware solution. GSA is able to conduct efficient and secure general search for web pages, documents, databases and other related information owned by enterprises, universities, etc., and supports more than 220 file formats. With the release of the latest version 5.2 of GSA, we are simultaneously releasing a software-only version of GSA running in virtual machines worldwide, Google Search Appliance virtual edition (GSAve). Today, Tech enthusiasts around the world can be free from here download GSAve, Run your own search engine on your computer and use it to crawl and index your own websites, blogs, databases, and web folders. GSAve inherits GSA's powerful search capabilities and programming interface. Google encourages programmers to use the interface provided by GSAve to develop interesting search experiences tailored to specific Intranet environments. - Using the connector management tool, you can let GSAve crawl and index network resources in non-Web form, such as documents stored in Sharepoint. - Using the Feed API and metadata search capabilities, you can set up search capabilities for your own web forums (BBS). - You can use the Onebox programming interface to provide real-time data in the search results page. -GSA's powerful secure search function supports multiple authentication methods, allowing users to see only the documents they have permission to access in search results. - We have designed a special web sorting algorithm for small document sets. - Users can customize the GSA search results interface, even in XML format, to integrate into their own applications. we set up on the Google online BBS Sub-forum , where GSA users from all over the world share their experiences together. If you have any suggestions or feedback about Google Search Server Virtual Edition, you can email us directly at enterprise-gsa-virtual@google.com. GSA provides us Googler with a powerful Intranet search, and we want to make this enjoyable search experience available to more users, so that everyone can access and benefit from it.
