by ofree on 2007-07-18 16:22:06

The webmaster has always been a loyal reader of Cfan. Every time the bound edition comes out, he will buy a set to collect. Today, he found the download address for the accompanying CD of the 2007 first-half bound edition online. This saves money, so let's just download and burn the disc. The only downside is that the download speed is a bit slow.

"Computer Enthusiast First Half 2007 Bound Edition: Main Text Volume" Disc A Download Address

Main Content:

1. "Glorious Anti-Virus Software" CD boot-up anti-virus system. This system uses the Windows PE operating system, allowing the CD to automatically guide and perform automatic virus removal. It can help users repair their computers even when they are completely paralyzed. It is a very practical tool.

2. Dreamweaver 8 Beginner and Practical Multimedia Tutorial, an extremely valuable multimedia tutorial that is easy to understand and get started with. It is a rare and excellent tutorial.

"Computer Enthusiast First Half 2007 Bound Edition: Appendix Volume" Disc B Download Address

Main Content:

1. The "Elite Software" section includes nearly 20 five-star software programs, all of which can be used for free.

2. The "Enthusiast Network Selected Multimedia Tutorials" are multimedia tutorials made by CFAN editors themselves. They cover almost 60 amazing computer tricks used by the editors.

3. The "E-books" section includes PDF electronic versions of four books published by the Computer Enthusiast Magazine: "2005 Interactive Software Special Issue - Secret Skill Supplement", "2005 Computer Enthusiast Collector's Edition", "Local Area Network Solutions and Troubleshooting Handbook", and "From Zero to 200% Anti-Hacker Expert".

4. "Mini Games" and "Fantastic Materials".
