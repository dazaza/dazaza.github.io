This is a list of common Linux commands.

Entries marked with ? can be directly copied to the terminal without any modification, so it's best to open a terminal while reading and use cut-and-paste.

Command Description

? apropos word Displays commands related to "word". See thread-safe.

which command Shows the full pathname of a command.

time command Calculates the time taken by a command to run.

? time cat Starts timing. Ctrl-d stops it. See sw.

? nice info Runs a low-priority command (here "info").

? renice 19 -p $$ Makes the script run at low priority. Used for non-interactive tasks.

? look prefix Fast lookup in an (ordered) dictionary.

? grep --color expr...ion /usr/share/dict/words Highlights matching regular expressions in the dictionary.

gpg -c file Encrypts a file.

gpg file.gpg Decrypts a file.

? alias hd=’od -Ax -tx1z -v’ Convenient hexadecimal output. (Usage example: ? hd /proc/self/cmdline | less)

? alias realpath=’readlink -f’ Shows the real path that a symbolic link points to. (Usage example: ? realpath ~/../$USER)

? set | grep $USER Finds $USER in the current environment.

? ls /usr/bin | pr -T9 -W$COLUMNS Prints nine-column output within the current terminal width.

touch -c -t 0304050607 file Changes the file timestamp (YYMMDDhhmm).

Directory operations

? cd - Goes back to the previous directory.

? cd Goes to the user's home directory.

(cd dir && command) Goes to directory "dir", executes the command, then returns to the current directory.

? pushd . Pushes the current directory onto the stack so you can return to it later using popd.

CD/DVD

gzip cdrom.iso.gz Saves a copy of the CD/DVD.

mkisofs -V NAME -r dir | gzip > cdrom.iso.gz Creates a CD image of the directory "dir".

mount -o loop cdrom.iso /mnt/dir Mounts the CD image to /mnt/dir (read-only).

cdrecord -v dev=/dev/cdrom blank=fast Erases a CDRW.

gzip -dc cdrom.iso.gz | cdrecord -v dev=/dev/cdrom - Burns the CD image (use dev=ATAPI -scanbus to confirm the device).

cdparanoia -B Rips CD audio tracks into wav files in the current directory.

cdrecord -v dev=/dev/cdrom -audio *.wav Burns all wav files in the current directory into an audio CD (see cdrdao).

oggenc --tracknum=’track’ track.cdda.wav -o ’track.ogg’ Converts a wav file into ogg format.

Archiving

tar c dir/ | bzip2 > dir.tar.bz2 Compresses and archives the directory "dir/".

bzip2 -dc dir.tar.bz2 | tar x Extracts the compressed archive (use gzip instead of bzip2 for tar.gz files).

tar c dir/ | gzip | gpg -c | ssh user@remote 'dd of=dir.tar.gz.gpg' Compresses and archives the directory "dir/" and places it on a remote machine.

find dir/ -name '*.txt' | tar c --files-from=- | bzip2 > dir_txt.tar.bz2 Archives all .txt files in the directory "dir/" and its subdirectories.

find dir/ -name '*.txt' | xargs cp -a --target-directory=dir_txt/ --parents Copies all .txt files from the directory "dir/" and its subdirectories while preserving the directory structure to "dir_txt/".

( tar c /dir/to/copy ) | ( cd /where/to/ && tar x -p ) Copies the directory "copy/" to "/where/to/" while preserving file attributes.

( cd /dir/to/copy && tar c . ) | ( cd /where/to/ && tar x -p ) Copies all files in the directory "copy/" to "/where/to/" while preserving file attributes.

( tar c /dir/to/copy ) | ssh -C user@remote 'cd /where/to/ && tar x -p' Copies the directory "copy/" to the remote directory "/where/to/" while preserving file attributes.

dd bs=1M if=/dev/hda | gzip | ssh user@remote 'dd of=hda.gz' Backs up an entire hard drive to a remote machine.

rsync (use the --dry-run option to test)

rsync -P rsync://rsync.server.com/path/to/file file Only gets diffs. Can be done multiple times when downloads are interrupted.

rsync --bwlimit=1000 fromfile tofile Local copy with bandwidth limit (beneficial for I/O).

rsync -az -e ssh --delete ~/public_html/ remote.com:'~/public_html' Mirrors a website (using compression and encryption).

rsync -auz -e ssh remote:/dir/ . && rsync -auz -e ssh . remote:/dir/ Synchronizes the current directory with a remote directory.

File search

? alias l='ls -l --color=auto' A one-character file listing command.

? ls -lrt Lists files by date. See newest.

find -name '*.[ch]' | xargs grep -E 'expr' Searches for 'expr' in all .c and .h files in the current directory and its subdirectories. See findrepo.

find -type f -print0 | xargs -r0 grep -F 'string' Searches for the string 'string' in regular files in the current directory and its subdirectories.

find -maxdepth 1 -type f | xargs grep -F 'string' Searches for the string 'string' in the current directory only.

find -maxdepth 1 -type d | while read dir; do echo $dir; echo cmd2; done Executes multiple commands for each found file (using a while loop).

? find -type f ! -perm -444 Finds all unreadable files (useful for websites).

? find -type d ! -perm -111 Finds inaccessible directories (useful for websites).

? locate -r 'file[^/]*\.txt' Uses locate to find all files matching *file*.txt.

Networking (ifconfig, route, mii-tool, nslookup commands are deprecated)

? ip link show Shows the interface list.

ethtool interface Shows the status of the interface.

ip link set dev eth0 name wan Renames eth0 to wan.

ip addr add 1.2.3.4/24 brd + dev eth0 Adds IP and netmask (255.255.255.0).

ip link set dev interface up Starts the interface (or shuts it down).

ip route add default via 1.2.3.254 Sets the default gateway to 1.2.3.254.

? tc qdisc add dev lo root handle 1:0 netem delay 20msec Adds 20ms transmission delay to the loopback device (for debugging).

? tc qdisc del dev lo root Removes the added transmission delay.

? host pixelbeat.org Finds the IP address for a given name, or vice versa.

? hostname -i Finds the local IP address (equivalent to host `hostname`).

? netstat -tupl Lists internet services in the system.

? netstat -tup Lists active connections.

wget (multi-purpose download tool)

? (cd cmdline && wget -nd -pHEKk http://www.pixelbeat.org/cmdline.html) Downloads the specified webpage and related files into the current directory for full browsing.

wget -c http://www.example.com/large.file Resumes a previously interrupted download.

wget -r -nd -np -l1 -A '*.jpg' http://www.example.com/ Bulk downloads files into the current directory.

wget ftp://remote/file[1-9].iso/ Downloads an entire directory from an FTP site.

? wget -q -O- http://www.pixelbeat.org/timeline.html | grep 'a href' | head Processes the output directly.

echo 'wget url' | at 01:00 Downloads the specified file into the current directory at 1 PM.

wget --limit-rate=20k url Limits the download speed (to 20KB/s here).

wget -nv --spider --force-html -i bookmarks.html Checks if links in the file exist.

wget --mirror http://www.example.com/ Updates a local copy of a website (can easily be used with cron).

Windows (Samba provides all Windows-related networking support)

? smbtree Finds a Windows host. See findsmb.

nmblookup -A 1.2.3.4 Finds the Windows (NetBIOS) name for a specific IP.

smbclient -L windows_box Shows all shares on a Windows host or Samba server.

mount -t smbfs -o fmask=666,guest //windows_box/share /mnt/share Mounts a Windows share.

echo 'message' | smbclient -M windows_box Sends a popup message to a Windows host (disabled by default on XP SP2).

Mathematics

? echo '(1 + sqrt(5))/2' | bc -l Convenient calculator (calculates φ).

? echo 'obase=16; ibase=10; 123' | bc Base conversion (decimal to hexadecimal).

? echo $((0x2dec)) Base conversion (hexadecimal to decimal) ((shell math extension)).

? echo 'pad=20; min=64; (100*10^6)/((pad+min)*8)' | bc More complex calculation, calculating the maximum FastE packet rate here.

? echo 'pad=20; plot [64:1518] (100*10**6)/((pad+x)*8)' | gnuplot -persist Shows a graph of FastE packet rates relative to packet size.

Text manipulation (sed uses standard input and output; to edit files, use newfile)

sed 's/string1/string2/g' Replaces string1 with string2.

sed 's/\(.*\)1/\12/g' Replaces any string ending with 1 with a string ending with 2.

sed '/ *#/d; /^ *$/d' Deletes comments and blank lines.

sed ':a; /\$/N; s/\\\n//; ta' Joins lines ending with \ with the next line.

sed 's/[ \t]*$//' Deletes trailing whitespace from each line.

sed 's/\([\\`\"$\]\)/\\\1/g' Adds escape characters before all escape characters.

sed -n '1000p;1000q' Outputs the thousandth line.

sed -n '10,20p;20q' Outputs lines 10 through 20.

sed -n 's/.*\(.*\).*/\1/p;T;q' Outputs the content inside the field of an HTML file.

sort -t. -k1,1n -k2,2n -k3,3n -k4,4n Sorts IPv4 addresses.

? echo 'Test' | tr '[:lower:]' '[:upper:]' Converts to uppercase.

? tr -dc '[:print:]' /dev/null Summarizes/profiling system calls made by a command.

? strace -f -e open ls >/dev/null Shows system calls made by a command.

? ltrace -f -e getenv ls >/dev/null Shows library functions called by a command.

? lsof -p $$ Shows files opened by the current process.

? lsof ~ Shows processes opening the user's home directory.

? tcpdump not port 22 Shows network traffic except SSH. See tcpdump_not_me.

? ps -e -o pid,args --forest Shows processes in a tree-like structure.

? ps -e -o pcpu,cpu,nice,state,cputime,args --sort pcpu | sed '/^ 0.0 /d' Shows processes ordered by CPU usage.

? ps -e -orss=,args= | sort -b -k1,1n | pr -TW$COLUMNS Shows processes ordered by memory usage. See ps_mem.py.

? ps -C firefox-bin -L -o pid,tid,pcpu,state Shows all threads of a specified process.

? ps -p 1,2 Shows information about specified process IDs.

? last reboot Shows system reboot records.

? free -m Shows total (remaining) memory (-m displays in MB).

? watch -n1 'cat /proc/interrupts' Monitors changes in the file /proc/interrupts.

System information (See sysinfo)

hdparm -i /dev/hda Shows information about disk hda.

hdparm -tT /dev/hda Tests the read speed of disk hda.

badblocks -s /dev/hda Tests all bad sectors on disk hda.

? mount | column -t Shows all mounted filesystems with aligned output.

? cat /proc/partitions Shows all registered partitions in the system.

? grep MemTotal /proc/meminfo Shows the total visible memory in the system.

? grep "model name" /proc/cpuinfo Shows CPU information.

? lspci -tv Shows PCI information.

? lsusb -tv Shows USB information.

Recode (iconv, dos2unix, unix2dos are deprecated)

? recode -l | less Shows all valid character sets and their aliases.

recode windows-1252.. file_to_change.txt Converts a Windows ANSI file to the current character set (automatically handles CR/LF conversion).

recode utf-8/CRLF.. file_to_change.txt Converts a Windows ANSI file to the current character set.

recode iso-8859-15..utf8 file_to_change.txt Converts a Latin9 (Western Europe) character set file to UTF-8.

recode ../b64 file.b64 Base64 encoding.

recode /qp.. file.qp Quoted-printable decoding.

recode ..HTML file.html Converts a text file to HTML.

? recode -lf windows-1252 | grep euro Finds the euro symbol in the character table.

? echo -n 0x80 | recode latin-9/x1..dump Shows the character mapping in latin-9.

? echo -n 0x20AC | recode ucs-2/x2..latin-9/x Shows the latin-9 encoding.

? echo -n 0x20AC | recode ucs-2/x2..utf-8/x Shows the UTF-8 encoding.

Interactive

? mc Powerful file manager that can browse rpm, tar, ftp, ssh, ...

? screen Multi-window virtual terminal, ...

? links Web browser.

? gnuplot Interactive and scriptable plotting tool.

? octave Tool similar to MATLAB.
