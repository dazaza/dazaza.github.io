瀚宇彩晶EMC-3412复古x86单板电脑

2021-01-17 06:10:21

自1976年基于英特尔C8080A CPU推出dyna-micro以来，单板计算机已不再是新事物。从那时起，它们已用于从数字标牌到卫星和航空航天应用的许多嵌入式应用中。2012年，Raspberry Pi基金会发布了Raspberry Pi Model B，该模型重新定义了SBC标准，许多制造商都基于Raspberry Pi外形尺寸和GPIO生产电路板此后的标准是什么，但是以前的标准是什么？

早在1987年，一家名为Ampro的公司发布了SBC，该SBC后来在1992年定义了称为PC / 104（或PC104）的标准。1997年，开发了包含PCI的PC / 104-plus规格，2008年开发了PC / 104。 -Express是包括PCI-Express总线在内的最终版本。 PC-104基于IBM XT总线（包括64个引脚）和IBM AT总线（包括40个引脚），因此命名为PC / 104。

以上所有标准在电气和物理上都相互兼容，尽管安装孔和端口对齐，但唯一的区别是物理板尺寸。有关PC / 104的更多历史记录，请参见Wikipedia。让我们看一下PC / 104兼容板。

HannStar EMC-3412是与PC104兼容的SBC，在2001年至2005年之间生产，该主板具有PC133 SODIMM插槽，USB接头连接器，PS / 2连接器，RS232公头和RS232母头，VGA视频，CF和IDE。 NSC Geode GX1S @ 300Mhz和单个Realtek 100Mb NIC。当我购买具有128MB RAM的全新主板时，我一直在努力寻找有关该设备的大量信息。我决定像其他任何SBC一样对其进行审核。我使用Virtualbox和BENFEI 4合1 USB USB-C到SD Micro SD MS CF卡读卡器将Debian 8安装到4GB微型驱动器上，因为该板无法从USB引导，并且手头没有IDE光盘驱动器。 Debian 8是支持i586处理器的Debian的最终版本。安装过程顺利进行，并且从CF卡插槽中安装的MicroDrive顺利启动了主板。该板使用标准的Molex 4针连接器供电，尽管看起来仅使用5v针。我创建了一个临时适配器以通过USB运行该设备。

CPU运行在300MHz，功能很少，它是x86兼容的i586处理器。

功耗并不像我预期的那样糟糕，在空闲时仅拉1w，在满负载时拉6w。我用科威斯USB功率计和定制电缆测量了功耗。

我运行UnixBench将此与我的其他SBC进行了比较，它总共获得了16分！甚至比Raspberry Pi Zero（零）还慢4倍。

Hannstar EMC-3412玩起来很有趣，并且可能会成为一台出色的Retro Gaming游戏机，尽管我想不到2021年该板有任何真正有用的用例。我在ebay上购买了这款新产品，卖家仍然有一个合理的选择。 很少有要价的要约出售，价格为30英镑，我的出价很低，因此被接受。 如果您想自己尝试一下，可以在这里购买。 PC / 104标准仍被广泛用于嵌入式计算中，许多制造商正在生产带有现代CPU的新板，但是它们非常昂贵，并且针对特定的应用而设计。 如果您想了解复古SBC的更多内容，请在下面发表评论。
