Hello, if you encounter a situation where you cannot send or receive emails, you can change the server address set in your client to the server's IP address. The free email POP3 server IP address is: 202.106.182.168, and the SMTP server IP address is: 202.106.187.158. Then, check if my server requires identity verification; if not, please enable it. By switching to the IP address, you can skip the DNS resolution step. If you still cannot normally send and receive emails at a proper speed, we suggest you perform the following test:

Please use the ping network command to check the connection status with Sina Mail servers. The specific method is as follows:

If your computer operating system is: win2000, XP

Click the Start button at the bottom left, choose "Run" from the menu that pops up. In the dialog box that appears, input "cmd". A black DOS-style dialog box will pop up. After the prompt, input "ping" followed by a space and then "pop3.sina.com.cn" (for receiving server) or "smtp.sina.com.cn" (for sending server). Please provide the detection results shown below and confirm the issues, thank you!

If your computer operating system is: win98, winme

Click the Start button at the bottom left, choose "Programs" from the menu that pops up. A new dialog box will appear. In the additional menu bar, select "MS-DOS mode". A black DOS-style dialog box will pop up. After the prompt, input "ping" followed by a space and then "pop3.sina.com.cn" (for receiving server) or "smtp.sina.com.cn" (for sending server). Please provide the detection results shown below and confirm the issues, thank you!

In the results, if the value for "time" is less than 100, the link speed is normal. If it is greater than 150, your internet speed is slow, please contact your network provider.

If you have more questions, please consult Sina Customer Service Center.

National unified customer service hotline: 95105670 Consultation email: webcn@staff.sina.com.cn

Firstly, make sure your network has no issues! If you can access the internet, using webmail means there should be no problem in receiving emails. Then, check if the email address you are sending to is correct. If none of these are the issue, it could be that the recipient's mailbox is full, which prevents them from receiving your email!
