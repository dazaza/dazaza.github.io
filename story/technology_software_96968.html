III. Installation Instructions

1. CD Installation:

Use a burning software, select "Image Burn" mode to burn the ISO file. Before burning, please verify the accuracy of the file. Recommended burning speed is 24X!

Set the BIOS to boot from the CD-ROM. Select the first option to automatically install to the first partition of the hard drive. For branded computers with hidden partitions, it is recommended to install manually.

2. Hard Disk Installation:

If the CD has been burned, double-click the auto-play program in WINDOWS and select "Install WINDOWS".

If the CD has not been burned, use a virtual CD-ROM to load the ISO CD image in WINDOWS, double-click the auto-play program, and select "Install WINDOWS".

Note: Please back up important data from the original system before installation. It is recommended to transfer personal data and back up hardware drivers in advance.

V. Common Issues

1. GHOST installation interruption, with the following prompt:

Output error file to the following location: A:ghosterr.txt

The cause is usually a read or write data error, which may be due to poor quality of the CD or poor CD-ROM reading ability, or an error writing to the hard disk.

2. Realteck HD Audio sound card front microphone and headphone jack cannot be used

This is caused by a new feature in the latest official Realteck sound card driver, and is not a system issue. If this happens, go to "Control Panel" - "Sound, Speech, and Audio Devices" - find "Realtek HD Audio Manager--->Audio I/O--->Jack Settings--->Check 'Disable Front Panel Jack Detection'".

3. After installation, a message appears saying the system has not passed genuine validation, with a star icon appearing in the lower right corner

This situation is mostly caused by incorrect system time. Correct the computer time and restart the computer. Do not use so-called cracking tools, as they cannot solve the problem.

4. What should be done if the system blue screens or constantly reboots during installation?

Such situations are generally caused by errors in windows recognizing drivers during installation. Re-GHOST restore the system, and when the driver selection screen appears, press the [ESC] key to cancel all integrated drivers to resolve the issue.

5. After GHOST, the entire hard drive becomes one partition

This is generally due to manual GHOST operation errors, restoring the partition image to the entire hard drive. Immediately shut down the computer and do not write data to the hard drive.

It is recommended to connect the hard drive to another computer as a slave drive and use relevant software to recover the partition table. If the data is important, seek help from a professional data recovery company.
