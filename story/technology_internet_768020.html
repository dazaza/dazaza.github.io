Can Nokia's Lumia 2520 tablet save Windows RT?

by anonymous on 2013-11-16 16:50:44

Nokia's yet-to-be-released Lumia 2520 is a Windows RT tablet equipped with a 10.1-inch display. After Nokia’s mobile hardware business is incorporated into Microsoft, the Lumia 2520 will become Microsoft's second Windows RT tablet. Despite this, Windows RT still faces serious challenges. Most of Microsoft's hardware partners, including Samsung, have distanced themselves from Windows RT.

Significant differences between Windows RT and Windows 8

From an external perspective, Windows RT and Windows 8 appear identical; however, there are significant internal differences. Windows 8 is a fully functional PC operating system capable of running almost all PC software, whereas Windows RT has many restrictions and can only run mobile applications specifically written for it along with a streamlined version of Office.

Windows RT targets tablets and is an operating system released by Microsoft to compete with Apple's iOS and Google's Android in the tablet market. However, unlike iOS and Android, Windows RT lacks a robust ecosystem chain. Microsoft's app store still lacks some of the most popular apps among users, including LinkedIn, Pinterest, and HBO Go. The Facebook app only launched on the Windows RT platform earlier this month.

Some may think that these apps are not essential, and users can access these services through browsers. However, according to market research company Flurry, 80% of the time users spend on mobile devices is spent running apps. An interface optimized for touch screens is crucial.

This could be the reason why Microsoft's efforts to sell Surface tablets have been ineffective. Microsoft hasn't disclosed Surface sales data, but tech blog GeekWire estimated in July that from October last year to June this year, Microsoft sold only 1.7 million Surfaces. Although Microsoft reported in its latest earnings call that Surface sales doubled, revenue was only $400 million, far below Apple's iPad revenue of $6 billion in the third fiscal quarter.

Besides better cameras and shell colors, there are no significant differences between the Lumia 2520 and Microsoft's Surface 2. Their size, weight, and display resolution are completely identical, both come with protective covers that double as keyboards, and their battery life is almost the same. The Lumia 2520 features a better camera on its back and an appearance similar to the Lumia series phones which is more appealing, but it costs $50 more than the Surface 2. The Lumia 2520 supports LTE (Long Term Evolution) connectivity technology. Nokia has stated that it will sell the Lumia 2520 through mobile carriers. However, this is unlikely to be a major selling point for the Lumia 2520 because the vast majority of tablets sold in the U.S. support only Wi-Fi connectivity technology.

Microsoft's hardware partners have already distanced themselves

The most noteworthy aspect of the Lumia 2520 is that it was made by Nokia: all of Microsoft's hardware partners have distanced themselves from Windows RT. Asus and Dell once launched Windows RT devices, but both discontinued these products due to weak sales.

Samsung once launched a Windows RT tablet in the European market but later abandoned the product and never introduced a Windows RT device to the U.S. market. Samsung even blamed the failure of Windows RT on Microsoft, stating that Microsoft's strategy confused consumers, forcing Samsung to expend great effort explaining Windows RT to consumers.

Despite criticizing Microsoft harshly, Samsung still launched tablets running Windows 8. Samsung's sale of Windows 8 tablets highlights a bigger issue facing Windows RT: at the same price as the Lumia 2520, consumers can purchase Samsung's ATIV Smart PC 500T. Since it runs Windows 8, Samsung's tablet won't suffer from the limitations of Windows RT.

The Future of Windows RT

Even with Nokia launching another product, Windows RT still faces serious challenges. Sales of Windows RT devices have consistently been weak, scaring away all of Microsoft's hardware partners. Due to the limited number of apps, and with Microsoft's hardware partners like Samsung offering Windows 8 tablets at competitive prices, the prospects for Windows RT are not bright. Investors should not place overly high expectations on Lumia 2520 sales.
