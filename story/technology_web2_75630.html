Web 2.0 Mini Review (11) - Online Bookmarking (International Edition) | Internet Odyssey

by cpxxpc on 2008-06-26 09:31:00

Delicious (del.icio.us) [Brief Review]: The world's most famous online bookmarking portal, with a simple interface but powerful features. [Submission]: Online submission, browser toolbar submission (automatically fetches webpage titles, options to set tags and sharing permissions), browser plugin submission, import from local bookmarks file. [Management]: Provides a short domain personal collection page, an independent settings center with complete functions, clear layout, supports tags (can be edited in bulk, managed through bundled categories), friends, subscriptions, collections. [Features]: Rich blog and Facebook display plugins (support setting individually as bookmarks or tags, can set title, tags, number of entries, refer to "DIY a 'Delicious' Blog Link Bar" for specific applications), supports exporting bookmarks. [Drawbacks]: Does not support bulk editing of bookmarks (such as changing permissions, deletion), only allows opening URLs in the current window, no Chinese version available yet (but supports normal display of Chinese). [Region]: USA, English.

Yahoo Favorites+ [Brief Review]: A network bookmarking site under Yahoo China, comprehensive and rich in functionality, slogan - Collect, Share, Easy-to-use. [Submission]: Online submission, right-click or toolbar submission in the browser (automatically fetches webpage titles, selected content automatically added as a summary, options to set tags and sharing permissions), submission and management via Yahoo Assistant, blog or webpage button submission, import from local favorites and Delicious bookmark files (refer to this page for specifics). [Management]: Provides a dedicated domain personal collection page (domain format http://myweb.cn.yahoo.com/userurls.html?ou=username), a complete management center with a clear and reasonable layout (supports batch moving, deletion, sending, changing permissions, adding tags), supports folders, tags (can be edited in bulk), subscriptions, collections, snapshots. [Features]: Blog display plugins (can set date, number of entries, tags), export bookmarks. [Drawbacks]: Few user-related extension functions (such as avatars, friends), cannot choose folder during submission, personal homepage domain is hard to remember. [Region]: USA, Simplified Chinese.

Mister Wong [Brief Review]: Europe's largest network bookmarking portal, authoritative and feature-rich. [Submission]: Online submission, browser plugin submission (supports multiple browsers, automatically fetches webpage titles, options to set tags, sharing permissions, groups), blog or webpage button submission, widget submission (personalized home page modules like Pigeon, VISTA sidebar widgets, WP exclusive bookmark plugins, Firefox search engine plugins, refer to this page for details), import from local favorites and Delicious bookmark files (refer to this page for details). [Management]: Provides a short domain personal collection page, a complete management center with a clear and reasonable layout (supports batch deletion, changing permissions), supports tags (can be edited in bulk), friends, subscriptions, collections, groups, recommendations, thumbnails. [Features]: Rich and distinctive related plugins, supports exporting bookmarks. [Drawbacks]: Lacks green and convenient submission methods. [Region]: Europe, Simplified Chinese.

Windows Live Favorites [Brief Review]: A network bookmarking site under Windows Live, basically lacks sharing capabilities. [Submission]: Online submission, import from local favorites file, import from other network bookmarking sites (supports Delicious, TianTian Net Digest, etc.). [Management]: Complete management center with a concise and reasonable layout (supports batch deletion, moving), supports folders, tags (can be edited in bulk), etc. [Features]: Supports bookmark pinning and preview, supports right-click operations, export bookmarks. [Drawbacks]: Does not support a personal shared collection homepage (basically private nature), single and inconvenient submission method, does not support bulk editing of tags. [Region]: USA, Simplified Chinese.

Google Bookmarks [Brief Review]: A network bookmarking site under Google, lacks sharing capabilities (must log in to view). [Submission]: Online submission, Google Toolbar plugin submission. [Management]: Simple management center, supports tags (can be edited in bulk), etc. [Drawbacks]: Does not support a personal shared collection homepage or shared bookmarks (completely private nature), does not support bulk editing of bookmarks (such as deletion, setting tags), overly single and inconvenient submission method, does not support importing or exporting bookmarks. [Region]: USA, Simplified Chinese.
