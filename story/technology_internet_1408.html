Amazon starts offering online storage services.

by eustacia on 2006-03-15 21:57:42

According to BetaNews, Amazon launched a scalable and reliable online storage service for software developers on Tuesday. The service is named Simple Storage Service, abbreviated as S3, with a charge of $0.15 per GB for storage capacity each month, and $0.20 per GB for traffic each month. It should be noted that this service is not aimed at ordinary users, but it can be used for any purpose. Almost any type of file can be stored on the server through this service, and each file object will have its own unique Key. You can set whether it is public or private data, and you can even operate these file objects through REST and SOAP interfaces.
