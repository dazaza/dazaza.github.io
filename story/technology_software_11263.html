28 types of data recovery software

by jssz on 2007-01-07 16:22:22

Test reason:

The day before yesterday, a friend asked me to help him recover the data of an IBM notebook, which belonged to his elder brother and contained important data, mainly Word documents. The disk size is 20 GB, and disk D is 8 GB. This partition was formatted due to misoperation and stopped immediately upon discovery. The current symptom is that drive D can be seen in Windows, but cannot be accessed. The friend has scanned the partition with recovery software, but has not found any data. As his brother will attend a meeting in the afternoon, we only have half a day to recover the data, so it is impossible to completely rely on manual solution, so we have to use software to recover as quickly as possible. After analysis, it was found that the partition table was disordered, so after repairing the partition table, several commonly used software were used for recovery, but the effect was very poor, so I had to go to the Internet to find a new version of the software, and finally recovered most of the data.

Analyzing the recovery process this time, I think it is very important to choose the software in different situations. Choosing the appropriate recovery software is not only very important for the recovery of data, but also can save a lot of time. Therefore, it is decided to conduct a simple test on the current popular data recovery software on the Internet to make a reference for friends who have this need.

Test process:

① A partition is formatted and filled with files, and data recovery is performed after all deletion.

② Format the partition again before restoring it.

③ Delete this partition for data recovery.

PS: My hard drive finally has a hidden 150M or so partition, is usually used to work under DOS. In order to save testing time and facilitate operation, this partition is used for testing.

Test environment:

Motherboard ASUS P4P800-X

CPU C4D 2.4

Memory 512M DDR333

Disk Maxtor 120 GB

Test results:

① Almost all software can recover deleted files, but some software has problems with recovered data.

② Only some software supports data recovery on formatted disks.

PS: Due to time, I did not conduct a comprehensive test, only a simple test on whether the file can be effectively recovered. According to the test results, these software are divided into three categories, and only the software that can be formatted and recovered is compared in detail. The other two categories are not compared and are therefore not described.
