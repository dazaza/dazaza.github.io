[Download] Android SDK 1.0

by yinyin on 2008-09-24 22:57:24

On September 23, Google released the final version of Android SDK 1.0 for download. The main changes include updates to org.apache.http.impl and Wi-Fi related features. Additionally, there have been several modifications to Android permissions, and the entire developer documentation has been improved. This version is very close to the one on the T-Mobile G1, which was released on October 23, but it does not include components like GMail or YouTube. For more detailed changes, you can refer to the article "Porting Android 0.9 to Android 1.0."
