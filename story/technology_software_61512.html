The Eight Classic Troubleshooting Write-ups for IE Browser

by wszdw142 on 2008-04-09 20:04:29

When you're riding your beloved IE across the Internet, suddenly a mysterious warning message box pops up on your desktop, and then all IE windows close; or no matter how you click the link, IE has no response... These malfunctions can be very frustrating and leave you feeling helpless. However, there's no need to panic. Today, we will work together to identify common IE errors and completely eliminate them!

1. Sending Error Reports

[Problem Symptoms]

While using IE to browse web pages, the message "Microsoft Internet Explorer encountered a problem and needs to close..." appears. At this point, if you click the "Send Error Report" button, an error report will be created, and clicking the "Close" button will cause the current IE window to close; if you click the "Don't Send" button, all IE windows will close.

[Problem Review]

This is a small program designed by IE to understand user errors during use. However, I don't want to be Microsoft's "free tester," especially since it faces thousands of reports every day. Who knows if they even care about my report?

[Problem Resolution]

Depending on the situation, you can use the following methods to disable the IE error reporting function:

① For IE 5.x users: Go to "Control Panel → Add or Remove Programs," select "Internet Explorer Error Reporting" in the list, then click the "Change/Remove" button to uninstall it from the system.

② For IE 6.0 users under Windows 9x/Me/NT/2000: Open the "Registry Editor," find [HKEY_LOCAL_MACHINE\Software\Microsoft\Internet Explorer\Main], create a DWORD value named IEWatsonEnabled in the right pane, and set its value to 0.

③ For IE 6.0 users under Windows XP: Go to "Control Panel → System," switch to the "Advanced" tab, click the "Error Reporting" button, select the "Disable Error Reporting" option, and select "But notify me when serious errors occur," then click the "OK" button.

2. Internal Errors Close IE Windows

[Problem Symptoms]

When using IE to browse some web pages, an error message dialog appears: "The program performed an illegal operation and will close...", after clicking the "OK" button, another dialog box pops up, prompting "An internal error occurred...". After clicking the "OK" button, all open IE windows are closed.

[Problem Review]

There are various reasons for this error, such as excessive memory resource usage, mismatch between IE security level settings and the website being browsed, conflicts with other software, or the website itself containing erroneous code... All these situations could potentially lead to the issue and require patient resolution.

[Problem Resolution]

① Close too many IE windows. If running programs that consume large amounts of memory, it's recommended not to have more than 5 IE windows open.

② Lower the IE security level. Execute the "Tools → Internet Options" menu, select the "Security" tab, click the "Default Level" button, and drag the slider to lower the default security level.

③ Upgrade IE to the latest version. IE 6.0 download address: http://download.sina.com.cn/cgi-bin/detail.cgi?s_id=6041 . IE 6.0 SP1 download address: download.microsoft.com.

You can use browsers based on IE, such as MyIE2. It consumes relatively fewer system resources, and when the browser crashes, the next time you start it, it will prompt you with "Do you want to reopen the page that had an error last time?" trying to minimize your losses. Download address: http://down.vipcn.com/SoftView/SoftView_5024.html

3. Runtime Errors Appear

[Problem Symptoms]

A dialog box pops up while browsing web pages with IE, saying "A runtime error occurred, do you want to correct the error?" After clicking the "No" button, you can continue browsing the internet.

[Problem Review]

It might be due to problems with the website being browsed or because IE does not support certain scripts.

[Problem Resolution]

① Start IE, execute the "Tools → Internet Options" menu, select the "Advanced" tab, check the "Disable Script Debugging" checkbox, and finally click the "OK" button.

② Upgrade the IE browser to the latest version.

4. IE Window Always Minimized Issue

[Problem Symptoms]

Every new window opened is minimized, even if you click the "Maximize" button, the next time you start IE, the new window will still be minimized.

[Problem Review]

IE has an "auto-memory function" that can save the state parameters of the last closed window. IE does not provide relevant setting options, but this can be achieved by modifying the registry.

[Problem Resolution]

① Open the "Registry Editor," find [HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Desktop\Old WorkAreas], then select "OldWorkAreaRects" on the right side of the window and delete it.

② Similarly, in the "Registry Editor," find [HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main], select "Window_Placement" on the right side of the window and delete it.

③ Exit the "Registry Editor," restart your computer, then open IE, maximize its window, click the "Restore Down" button to restore the window, then click the "Maximize" button again, and finally close the IE window. From now on, when you reopen IE, the window will be normal!

5. IE Cannot Open New Windows

[Problem Symptoms]

During the process of browsing web pages, clicking on hyperlinks has no reaction.

[Problem Review]

Mostly due to the IE new window module being damaged.

[Problem Resolution]

Click "Start → Run," run "regsvr32 actxprxy.dll" and "regsvr32 shdocvw.dll" one by one to register these two DLL files, then restart the system. If it still doesn't work, you can also register mshtml.dll, urlmon.dll, msjava.dll, browseui.dll, oleaut32.dll, shell32.dll.

6. Offline But Unable to Browse Local Web Pages

[Problem Symptoms]

Through the "Offline Browsing" feature of IE, we can almost browse all the web content that has been downloaded to the local hard drive, which is a great way to save money for dial-up internet users. But sometimes, the target web page is on the hard drive, yet it prompts "Unable to browse."

[Problem Review]

This is mostly due to you modifying the system time, causing confusion in IE's history records.

[Problem Resolution]

① You can activate it by searching directly in the "Temporary Folder." Press Win+F, input some remembered keywords in the "Containing Text" section, press the "Browse" button in the "Search" section to choose the address of the IE temporary folder, such as "C:\WINDOWS\Temporary Internet Files," click "Start Search," and double-click the target page in the result list to open it.

② You can try using browsers like Tencent TE for offline browsing.

7. Browser Cannot Open Certain Sites While Online

[Problem Symptoms]

After going online, encounter various connection errors while browsing certain sites.

[Problem Review]

These errors are generally caused by website failures or lack of browsing permissions.

[Problem Resolution]

For different connection errors, IE will give different error messages. The more common ones include:

① Prompt Information: 404 NOT FOUND This is the most common IE error information. Mainly because IE cannot find the webpage file you requested, the file may not exist at all or has been moved elsewhere.

② Prompt Information: 403 FORBIDDEN Common on websites that require registration. Generally, this issue can be resolved by registering online instantly, but some completely "closed" websites still cannot be accessed.

③ Prompt Information: 500 SERVER ERROR Usually caused by design errors in the web page program or database errors, you can only wait for the opposing web page to correct the errors before browsing again.

Xiaoyu has collected a comprehensive list of IE error codes. When you encounter an IE error prompt but are unclear, you can refer to this site: http://www.cfan.com.cn/09net/19ie.htm.

8. IE Cannot Reinstall

[Problem Symptoms]

IE cannot be used normally, and during reinstallation, it prompts "The same version of IE was found in the system" and refuses installation; there is no uninstall option in "Add or Remove Programs."

[Problem Review]

"Reinstallation" is the ultimate solution to IE problems and also a treasure for beginners.

[Problem Resolution]

① For reinstallation of IE 5.0, follow these steps:

Step 1: Open the "Registry Editor," find [HKEY_LOCAL_MACHINE\Software\Microsoft\Internet Explorer], and click the Version Vector key under it.

Step 2: Double-click the IE subkey in the right pane, change the original "5.0002" to "4.0," click "OK," and exit the "Registry Editor."

Step 3: Restart, and then you can reinstall IE 5.0.

② There are two methods for reinstalling IE 6.0:

Method 1: Open the "Registry Editor," find [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Active Setup\Installed Components\{89820200-ECBD-11cf-8B85-00AA005B4383}], and change the IsInstalled DWORD value to 0.

Method 2: Insert the Windows XP installation disc, type "rundll32.exe setupapi,InstallHinfSection DefaultInstall 132 %windir%\Inf\ie.inf" in the "Start → Run" window.
