Microsoft may follow iOS to merge the Windows systems of手机 and tablet

by anonymous on 2013-10-01 20:39:12

After the release of Windows 8 operating system, Microsoft once proclaimed the slogan "There is only one Windows 8." However, it was discovered by the outside world that behind the slogan, the fragmentation of Microsoft's Windows system became even more severe and the versions more disordered. Among these, for smartphones and tablets with ARM chips, Microsoft has two operating systems: Windows Phone and Windows RT.

Recently, foreign media have analyzed based on statements from Microsoft executives that with the increasing screen size of phones and the gradual convergence of tablets and phones, Microsoft will merge the phone and tablet systems for ARM architecture.

The unification of operating systems is also one of the goals of Ballmer's reorganization of Microsoft. In the newly formed Windows division, Terry Myerson, responsible for Windows Phone 8, has become the head. At Thursday's annual analyst meeting of Microsoft, Myerson also attended and discussed the strategy of merging operating systems.

Myerson stated that in the future, across various different hardware, Windows should have a unified set of development interfaces API, allowing third-party developed applications to run on all Windows hardware. Additionally, future Windows devices will rely more on network cloud services.

Previously, for tablets with ARM chips, Microsoft launched Windows RT. Since this system cannot run traditional Windows software (based on x86 architecture), this system as well as Microsoft's Surface-RT products met with a Waterloo in the market. Samsung, Acer, and other device manufacturers previously gave poor evaluations of the Windows RT system. Many analysts believe that Microsoft should completely abandon Windows RT.

However, judging from Myerson's remarks, Microsoft still has full confidence in ARM tablets and operating systems.

Myerson stated that ARM chip devices, especially phones, have gained enormous market share due to their excellent battery life and communication capabilities. As phones gradually converge towards tablets, consumers will see more ARM tablets in the future.

The media interpreted this as meaning that after the increase in phone screen sizes, Microsoft also saw the potential use of Windows RT in phones. Microsoft may also merge the Windows Phone and Windows RT operating systems. However, how and when Microsoft will integrate them remains unclear at present.

In today's Microsoft, under Ballmer's transformation "baton," it has embarked on the path of copying Apple. Comparing with Apple, Apple has only one unified operating system iOS on mobile devices like phones and tablets (and another one for computers), whereas Microsoft has three systems on mobile devices: Windows Phone, Windows RT, and Windows Pro, clearly showing redundancy.
