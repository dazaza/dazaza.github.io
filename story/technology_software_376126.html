Massive added the function of aggregating original comments from third-party Weibo.

by xdash on 2010-12-27 21:06:54

Masar (http://www.masar.cn) is a new social aggregation and synchronization platform that allows users to simultaneously update messages across multiple linked social media sites and aggregates all messages from the connected sites into one interface. Currently supported third-party services include Sina Weibo, Tencent Weibo, Sohu Weibo, NetEase Weibo, Fanfou, Douban, ZuoSha, 9911, Diguo, Renjian, Tongxue, and Follow5, among others.

On December 26th, Masar underwent another upgrade and now supports the aggregation of original comments from third-party linked microblogs on "My Homepage."

(Pictured above: Messages and comments from Sina Weibo displayed on "My Homepage.")

Previous microblog synchronization tools only supported one-way message sending. To see feedback comments from different microblogging platforms, users still had to log in to other microblogs. This innovation by Masar enables users to send and receive information on the same screen, achieving true two-way aggregation of social networking sites.

The launch of this feature helps reduce the operational cost of frequently switching between different platforms, allowing for more efficient information retrieval. However, users should exercise caution to avoid further information overload.

Masar: http://www.masar.cn/

Appendix: Update content for this version of Masar ——

(1) Newly supported platforms: Follow5, Tencent, Fanfou. The total number of supported platforms has now reached 14.

(2) New features list:

1. Added image, video, and music publishing functions;

2. Displayed comment lists in "My Homepage" (third-party platforms supporting this feature: Sina, NetEase, Sohu, Douban, Follow5, Tencent, Fanfou), enabling one-stop review of comments from multiple platforms and support for replying to comments;

3. Added function to share original content from "My Homepage" to other platforms, eliminating the frustration of not being able to share great content across different platforms;

4. Added local storage function for original microblog content in "My Homepage";

5. Added personal profile editing function in settings;

6. Added personal avatar editing function in settings;

7. Password reset function;

8. Added a concise version of "Weibo Revelation," with the full version of "Weibo Revelation Issue One."

(3) Fixed or optimized the following issues:

1. Fixed the issue where links in some microblog lists on "My Homepage" could not be clicked;

2. Modified registration login password format to support letters, numbers, and underscores;

3. Optimized Oauth protocol account binding process and user experience;

4. Improved the display effect of error prompt messages;

5. Fixed the issue where clicking logout resulted in no response;

6. Fixed the issue of blank page headers in some browsers.
