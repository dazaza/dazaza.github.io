Fao Electronics teaches you how to identify optical cable models, recommended! It's practical.

by fuaokeji on 2011-08-23 09:45:07

Due to the large variety of optical cables, many people are unclear about the model numbers of optical cables. In fact, these model numbers can be identified. Below, Guangzhou Fuo'ao will provide a simple explanation regarding the identification of optical cable model numbers.

The format of an optical cable model: The format consists of five parts, each represented by a code as shown in the figure below. Among these, the structural characteristics refer to the cable core structure and the derived structural characteristics of the optical cable.

ⅠⅡⅢⅣ　Ⅴ

1. Classification code

GY —— Outdoor (field) communication optical cable

2. Strengthening component code

The strengthening component refers to the element used to enhance the tensile strength of the optical cable, which is either expanded inside or embedded in the sheath. If both metal and non-metal strengthening components are present, only the metal component structure is indicated.

(No symbol) —— Metal strengthening component

F —— Non-metal strengthening component

3. Code for the optical cable core and the derived structural characteristics of the optical cable

The structural characteristics of the optical cable should represent the main type of the cable core and the derived structure of the optical cable. When several structural features need to be noted in the optical cable format, a combined code can be used, arranged in order from top to bottom according to the corresponding codes listed below.

D —— Optical fiber ribbon structure S —— Optical fiber loose tube coated structure

J —— Optical fiber tight tube coated structure (No symbol) —— Layer stranded structure

X —— Cable center tube (coated) structure T —— Filled structure

C —— Self-supporting structure E —— Elliptical shape

Z —— Flame-retardant structure

4. Sheath code

Y —— Polyethylene sheath V —— Polyvinyl chloride sheath

A —— Aluminum-polyethylene bonded sheath (shortened to A sheath) S —— Steel-polyethylene bonded sheath (shortened to S sheath)

W —— Steel wire-reinforced steel-polyethylene bonded sheath (shortened to W sheath)

5. Outer protective layer code

When there is an outer protective layer, it may include some or all parts of the padding layer, armor layer, and outer jacket, with its code represented by two groups of numbers (the padding layer does not need to be indicated). The first group represents the armor layer, which can be a one-digit or two-digit number; see Table 1. The second group represents the outer jacket or outer sheath, which should be a one-digit number; see Table 3.

Composition of the optical fiber specification: The specification of the optical fiber is composed of the number of optical fibers and the category of optical fibers.

Code for the number of optical fibers: Represented by the actual effective number of optical fibers of the same category in the optical cable.

Code for the category of optical fibers: The category of optical fibers should be represented by the classification code of the optical fiber product. Specifically, use capital A to indicate multimode fiber and capital B to indicate single-mode fiber, then use numbers and lowercase letters to indicate different types of fibers. For multimode fibers, see Table 3.
