Appearance Introduction:

The terminal has five buttons in total. The green button at the top left is the power on/off key. The blue button at the bottom left is the mode (interface) switch key. The upper button on the right is the emergency dialing (SOS) button. The middle button on the right is the dialing button, and the lower button on the right is also a dialing button.

Detailed Introduction:

Basic functions:

1. Precise positioning, supports positioning under conditions with or without a positioning server: Without a positioning server, you can query real-time latitude/longitude/time/speed using public map software and a safety protection network by manually entering latitude and longitude. Under conditions with a positioning server, GPRS positioning is performed directly.

2. If the target being queried enters an underground parking lot or other GPS blind spot, the tracker will report the last known location of the target.

3. Out-of-area alarm

4. Timed tracking function.

5. Authorized number addition/deletion function.

6. Alarm function.

7. Electronic fence: A fixed area can be set, and an alarm will sound if the fixed area is exceeded.

8. Displacement alarm: When the tracker stops moving, it can be armed, and if moved, it will alert the authorized number.

9. Overspeed alarm: Set the speed at which the monitored object travels, and if this speed is exceeded, an alarm will sound.

10. Low battery alarm function.

Specific Usage Method:

1. Button and port introduction:

- Top left - Button 1: Power on/off key

- Bottom left - Button 2: MODE key

- Port 3: Headphone / Charging port

- Top right - Button 4: SOS alarm key

- Middle right - Button 5: Phone key 1

- Bottom right - Button 6: Phone key 2

Press and hold [Button 4] to perform an emergency alarm operation. The interface will prompt as shown in the figure when the watch will automatically call the three phone numbers set inside one by one. If the first phone number is answered, the watch will enter the call interface as shown in the figure. After the call ends, the watch will not call the remaining two numbers. If the first phone number is not answered or is off, the terminal will start calling the second number, and so on.

(5) Terminal active dialing function: Press and hold [Button 5], and the interface will display "Call 2...", indicating that the watch is calling the second phone number set inside. After the call is connected, the watch will enter the call interface.

Press and hold [Button 6], and the interface will display "Call 3...", indicating that the watch is calling the third phone number set inside. After the call is connected, the watch will enter the call interface.

(6) Center switching method: Switch the center through instructions. Detailed instructions are in the appendix.

(7) Call and automatic answer function: If there's a call to the terminal, the incoming call interface will appear as shown in the figure. At this point, pressing the SOS button will connect the call. If the call isn't actively answered, it will automatically answer after 30 seconds. You can also hang up by pressing the power on/off button.

Note: In Server 1 mode (for details see the appendix), if there's an incoming call, the watch will first display the incoming call interface for 15 seconds before ringing. In Server 2 mode (for details see the appendix), if there's an incoming call, it will ring directly.

(8) Point-to-point query: In Server 1 mode (settings for server mode are in the appendix):

Send a text message "123" from any of the three pre-set relative numbers to the SIM card installed in the watch or directly call the SIM card number in the watch (the call does not need to be answered). Afterward, the relative's phone will immediately receive a URL link SMS. Use this phone to connect to the URL (specific connection methods vary depending on each individual's phone, requiring correct GPRS internet settings, please refer to your phone's user manual). After a successful connection, the exact position of the watch will be displayed on the querier's phone. If the GPS signal is good, the relative's phone will receive a second location information within three minutes after the query. The location obtained from this query will be more accurate than the first one. It is recommended that if you receive two connection information SMS messages, you can directly use the second one for location queries. [As shown in the figure]

Note: When using Server 1 mode, the SIM card in the querying phone must have GPRS functionality enabled. The second information will return no later than three minutes. If it hasn't returned after three minutes, it means the second location information won't return, and you don't need to wait anymore.

In Server 2 mode (settings for server mode are in the appendix):

Send a text message "123" from any of the three pre-set relative numbers to the SIM card installed in the watch. Within three minutes, the relative's phone will receive a location information SMS, which indicates the current geographical location of the terminal. [As shown in the figure]

Additionally, send a text message "456" from any of the three pre-set relative numbers to the SIM card installed in the watch. Within one minute, the relative's number will receive a latitude and longitude information SMS. [As shown in the figure]:

Note: When using Server 2 mode, the SIM card installed in the watch must have GPRS functionality enabled (it is recommended to use China Mobile's card).

If you query the watch's location in an area without GPS signals, the querying relative's number will receive the most recent valid location, and the time of this location will be given at the end of the SMS.
