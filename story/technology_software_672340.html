I. Introduction: Sanhe is based on the traditional Longhu gameplay, broadcasting real person videos for customers to guess the game results, ensuring fairness and justice. It also provides a platform for customers to compete with each other. Customers can take turns being the banker, playing against others. The game features pleasant music, making it enjoyable for customers, while the venue owner earns a steady income by taking a cut from the winners.

II. Installation: A set of Sanhe consists of one main scoring machine and ten sub-machines connected via a betting line. A display signal line is split through a distributor to the sub-machines as shown in the diagram. The speaker connects to the main machine. The connection line is a 34-pin wire, and the display signal line is a 15-pin wire that goes through the distributor to the sub-machines.

III. Main Machine Operation:

1. Enter menu: Unlock and click the top left corner "777". If you click start, you cannot enter the menu. As shown in the figure.

2. Input password: After clicking, input password "123456". As shown in the figure.

3. Game settings menu, as shown in the figure:

1. Machine number: The identification number of the machine.

2. Banker's retention: The percentage taken from the banker when a customer becomes the banker, recommended 2-3%.

3. Player's retention: The percentage taken from players when a customer becomes the banker, recommended 2-3%.

4. Dragon payout rate: 1:95%, usually 95. If both sides have a cut, suggest 97-98.

5. Tie payout rate: 1:800%, usually 800.

6. Tiger payout rate: 1:100%, usually 100. If both sides have a cut, suggest 97-98.

7. Tie eats all: Whether the tie eats all dragon and tiger bets. 1 eats all, 0 does not eat. Usually does not eat.

8. Betting time per round: Adjust betting time according to the venue.

9. Maximum player score: The maximum total score for players, exceeding this will cause the machine to explode after refunding scores.

10. Total red limit: The difference in bet amounts between dragon and tiger.

11. Minimum bet: The minimum bet amount.

12. Maximum tie bet: The total bet amount for tie across the entire venue.

13. Minimum score for becoming banker: The minimum total score required for a customer to become the banker, usually the red limit multiplied by the number of times they can be the banker.

14. Number of consecutive banker rounds: The number of consecutive rounds a customer can be the banker after obtaining it.

15. Background music: Whether there is singing, 0 means no, 1 means yes.

16. Game difficulty: 1-3, effective when not showing single cards, suggested 2.

17. Chip 1: The value of small chips.

18. Chip 2: The value of large chips.

19. Score increment: The step size for adding scores.

20. Whether to show single cards: Whether to follow difficulty or show single cards. Restart after saving.

21. Select increment: As shown in the figure.

22. Change password: As shown in the figure, modify the game parameter password, input old password "123456", then input a six-digit password twice.

23. Clear base score: Click query profit to clear, input password "111111", click clear to confirm clearing base score, click cancel to cancel clearing base score, click change password to modify zero-clearing password.

24. Clear records: Click change password to clear, input password "111111", click clear to confirm clearing records, click confirm to clear all records. Click cancel to cancel clearing records, click change password to modify zero-clearing password.

IV. Check accounts: As shown in the figure.

a. Unlock and right-click the blank space between the field sum and time, input password "123456", as shown in the figure.

b. Click print to print out the bill.

V. Adding/Subtracting Scores Operations: As shown in the figure.

Unlock, left-click to 1000 to change the score increment, click to 1680 to display the score to add, click OK to confirm adding scores. The first grid on the right of OK shows the last added score. Subtracting scores as shown in the figure: Unlock, right-click to 1000 to change the score decrement, click to -2000 to display the score to subtract, click OK to confirm subtracting scores. The second grid on the right shows the last subtracted score.

VI. Edit Single Card Display:

Unlock, right-click, input zero-clearing password to edit single card display. Click result to make direct changes.
