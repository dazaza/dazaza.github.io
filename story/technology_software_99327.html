FAQ:

1. Kaspersky prompts a risk software warning during installation.

※ This is because the installation process calls other software to run (e.g., setting themes), which Kaspersky naturally lists as risky software. This is normal; Kaspersky lists countless software programs as risky, including industrial software like PROE with millions of users that also triggers alarms.

2. After installation, the menu bars on My Computer and IE disappeared, and there's no title bar either?

※ Normally, this should not happen. This is due to Styler becoming ineffective. It happens because you installed Styler or other software before this style package, causing program path conflicts and failure. Solution: Click Start Menu -> "Vista Style Package" -> "Set Styler", then input N and press Enter.

3. Web fonts appear too large and uncoordinated. How can I cancel the display of web pages in the YaHei font?

※ Some websites haven't considered size issues with YaHei font display. In this case, the solution is: Open Internet Options, go to the "Fonts" settings under the "General" tab, set the webpage font to SimSun; open the "Accessibility" settings next to "Fonts", and remove all checkmarks from the boxes in front of "Do not use fonts specified on webpages". Now check if all web pages can display the YaHei font.

4. After installation, the icons did not change.

※ This is due to the system icon cache. During installation, clearing failed due to other software. To fix this, click Start Menu -> All Programs -> Vista Style Package -> Rebuild Icons, and it will take effect.

5. The theme hasn't changed, and I can't select the installed theme. There's no transparent glass effect.

※ This is because your XP system hasn't cracked the theme limitation, preventing the use of third-party themes. Choose the component "Remove XP restrictions on using other themes" during installation. You must install the "WB Transparent Effect" component to get the glass effect.

6. After upgrading patches, the icons reverted to XP's. How to solve this?

※ Click Start Menu -> "Vista Style Package" -> "Repair Icons", and wait until it completes.

7. I installed the sidebar and effects, but there's no effect. How to enable them?

※ Open Start Menu -> "Vista Style Package" -> "Set Effects", choose the effects you want to enable based on your needs, click OK, and you won't need to set it again afterward.

8. How to change the region of the weather on the sidebar?

※ Right-click the icon at the bottom right corner, go to the "System Settings" -> "Network Settings" in the sidebar to make changes.

9. After uninstalling the style package, some files still remain in the installation directory.

※ After uninstalling the style package, if you installed the STYLER and WB components, restart and rename the installation directory, then delete the installation directory to completely uninstall.
