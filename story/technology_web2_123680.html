Today, I received an invitation from Mickey Net, and I was invited to visit.

In fact, it was officially put into operation on August 8, 2006, and is the world's largest online software factory (present location: http://fac.misuland.com). Perhaps it is to see the signs of web2.0 warming, but also can not help but create a social network: Mickey community, and there is a trend of change.

image

Since its creation, Micmouse has been making bold attempts in the design of products and functions, meeting the fundamental needs and potential needs of Internet users with innovative means, constantly pursuing a more perfect user experience, and striving to make Micmouse become a convenient and practical large-scale community platform. At present, the Mickey community has the following three modules:

Software Factory: Micmouse Online Software Factory is a software component collaborative development and outsourcing service platform, committed to building China's largest software component library and China's largest software component collaborative development and outsourcing service platform.

For more information, please visit http://fac.misuland.com

Mouse community: Mouse community includes web games, chess and card games, etc. All games produced by Mouse online software factory can run under various versions of the linux operating system. Mouse network will continue to make its own efforts for the development of China's open source cause and the popularity of linux.

For more information, please visit http://www.misuland.com

Instant Messaging: Micky Instant Messaging (MSC) is one of the most popular instant messaging on the market at present, its excellent performance under various versions of linux and various versions of windows at home and abroad is an international leader, Micky instant messaging (MSC) with its cross-platform and other excellent performance, Won the International Prize at the 2008 International Open Source Software Design Competition held in October this year.

For more information, please visit http://msc.misuland.com

The Mickey community is now in a testing period. It is a complete SNS service. Contains the discovery of resources, sharing, some web2.0 small games, such as buying and selling friends, fighting for parking Spaces and so on. Interested friends can ask JarryJarry for an invitation code.

Jarry, however, disagrees with the way Mousenet operates. It seems that the Mickey net is going to transform, but the transformation intention is not clear. It is actively making some attempts, but it does not need to run its own foundation for years to make bets. The main domain name of the Mickey network has been directly ceded to the Mickey community. Jarry believes that if there is a new attempt, even a transformation, it should be a new domain name dedicated to this service. If it's just a test, use a second-level domain name. The pattern of the mouse web is a little confusing.

But I don't know what Mickey Net is up to. We'll have to wait and see how this plays out for the Mickey.
