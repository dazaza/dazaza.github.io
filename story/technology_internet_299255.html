Sogou Cloud Pinyin - the latest conceptual product of Sogou Pinyin

by ofree on 2009-12-08 17:20:53

Sogou Input Method has recently released a conceptual product - Sogou Cloud Input Method. The standout feature of this input method is that it operates entirely on servers, requiring no downloads or installations, and can be used across different platforms and operating systems, making it very convenient.

The Sogou Cloud Input Method is based on JavaScript technology, utilizing the Ajax communication principle and adopting a B/S architecture. It is compatible with various operating systems such as Windows, Linux, and Mac, as well as mainstream browsers like IE and Firefox. It is currently the first input method to be fully reliant on cloud computing technology.

The usage of the Sogou Cloud Input Method is very simple. Users do not need to install any files; they just need to "add to favorites" a JavaScript script button. Then, while browsing other web pages, users can click the "Enable Sogou Cloud Input Method" button in the browser's favorites bar to run the Sogou Cloud Input Method. After successful operation, a Sogou Cloud Input Method icon box will appear in the top-left corner of the current page.

Compared to the traditional version of the Sogou Pinyin Input Method, the latest Sogou Cloud Input Method operates on cloud servers with enormous memory and powerful computational capabilities. Theoretically, the size of the stored vocabulary and language model libraries can be infinite, thus achieving higher typing accuracy. According to official data, the language model size of the Sogou Cloud Input Method is 4GB, and the vocabulary scale is 2 million, far exceeding the Sogou Pinyin Input Method (12MB, 400,000). Therefore, in terms of input accuracy for both short and long sentences, the Sogou Cloud Input Method reaches an impressive 84% and 94%, respectively.

The site administrator tested the Sogou Cloud Input Method immediately upon its release and found it to be excellent. In terms of typing speed and accuracy, it is no less effective than traditional input methods. However, since this product was just launched, there are still many areas for improvement, such as:

1. When using the Sogou Cloud Input Method, the client-side input method needs to remain in English keyboard mode.

2. Since it is a Web-based input method, every time a new page is opened, the user must click the "Enable Sogou Cloud Input Method" button, and it can only be used within the browser.

3. Its compatibility with some browsers is not optimal. While the site administrator could use the Sogou Cloud Input Method normally under "WIN XP + TheWorld Browser," it failed to work under "WIN7 + TheWorld Browser." The exact reason remains unclear.
