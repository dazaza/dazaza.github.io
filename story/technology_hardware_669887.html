Essential Skills for Understanding and Using the Android Operating System (Part II)

by dream2012 on 2012-02-20 13:12:31

The Android operating system may seem complex and confusing to many novice users who are just getting started. Here, we have compiled some essential tips for understanding and using the Android operating system to hopefully be of help. Please note that these tips are mainly targeted at the most mainstream Android 2.1 system, but most of them are also applicable on older or newer Android systems.

11. Quickly Switch Home Screens

The Android system usually provides five home screens. You might find it troublesome to switch by swiping your finger each time. Simply press the home button and you can choose to enter a specific home screen. HTC Sense phones even support multi-touch to bring up all home screens for selection.

12. Add a Quick Navigation Key

The Android 2.1 system supports users creating shortcuts for map navigation. You can set up a shortcut menu on the desktop for any selected navigation route, allowing you to preset navigation routes before traveling, saving time by simply clicking the shortcut when needed.

13. Set Double-Tap Screen Zoom Level

Most Android phones that do not support multi-touch allow you to zoom in on web content by double-tapping the screen while browsing. Additionally, you can select three different zoom levels in the browser settings under the "default zoom" option, changing the effect each time you use the double-tap zoom.

14. Change Android Browser Font

Android devices come in various shapes and sizes, and the optimal browsing font size varies with different screens and resolutions. By setting an appropriate font size in the browser settings, reading efficiency can be improved.

15. Search Specific Web Content

In the browser settings > more > find option, you can search for specific fields within web content, helping you quickly locate relevant information. This is one of the standard features of PC browsers.

16. Voice Input

Starting with Android 2.1, voice input functionality is supported. Although there may be some delay in processing, with practice, you can achieve basically correct input. It's also a way for students practicing English oral skills.

17. Android Dictionary Word Creation Function

When using the Android keyboard to input a specific word, you can long-press it and choose to add it to the suggested words area, thereby adding this word to the Android dictionary. The next time you type, entering the first few letters will bring up this word.

18. Sync Phone Photos to Picasa Album

Although Google's Picasa online album is not as popular as Flickr, starting with Android 2.1, the 3D photo album began supporting automatic syncing of phone photos to a Picasa account. Without third-party software, many users now have a reason to use Picasa. However, tragically, Picasa cannot be opened normally in China due to known reasons.

19. Download Webpage Images

When browsing and wanting to save an image, simply long-press the image to save it individually. Saved images can be viewed in the photo gallery application.

20. Manage Android Call Logs

Managing Android call logs is very convenient. Long-press any outgoing or incoming call log to delete it, making it very easy.
