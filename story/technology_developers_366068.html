```plaintext
RA(Config)#isdn switch-type basic-5ess

RB(Config)#usename RA password ra

RB(Config)#enable pass 123

RB(Config-if)#encapsulation ppp

RB(Config-if)#ppp authentication pap

RB(Config-if)#isdn spid1 111001 5555555

RB(Config-if)#ppp pap sent-username RB pass rb

RB(Config-if)#snapshot client 5 10 suppress-statechange-update dialer

Note: The "suppress-statechange-updates" command prevents the router from sending routing updates during user-initiated data exchanges. The "Dialer" keyword allows the client router to call the server router even when no specified traffic is present.

RB(Config-if)#exit

RB(Config)#router igrp 100

RB(Config-router)#network 192.168.2.0

RB(Config-router)#network 192.168.4.0

RB(Config-router)#exit

RB(Config)#int e0

RB(Config-if)#ip address 192.168.4.254 255.255.255.0

RB(Config-if)#no shutdown

```

### Explanation:

- **ISDN Switch-Type**: Configures the ISDN switch type.

- **Username/Password**: Sets up usernames and passwords for authentication between routers.

- **Enable Pass**: Sets an enable password for privileged mode.

- **Dialer-List**: Defines access lists for dial-on-demand routing.

- **Interfaces (BRI/Ethernet)**: Configures IP addresses and encapsulation types for interfaces.

- **PPP Authentication**: Enables PAP (Password Authentication Protocol) for PPP links.

- **IGRP Routing**: Configures IGRP routing protocol with specific networks.

- **Snapshot Commands**: Used for optimizing dialer operations and suppressing unnecessary updates.

This configuration ensures secure communication and efficient routing between two routers (RA and RB) using ISDN and PPP protocols.
