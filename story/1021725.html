NeoAxisжёёжҲҸеј•ж“Һ2020.5

2020-09-02 16:25:07

NeoAxisе…¬еҸёеҸ‘еёғж–°зүҲNeoAxis EngineпјҢиҝҷжҳҜдёҖдёӘеҶ…зҪ®3Dе’Ң2DжёёжҲҸеј•ж“Һзҡ„йӣҶжҲҗејҖеҸ‘зҺҜеўғгҖӮиҜҘзҺҜеўғеҸҜз”ЁдәҺејҖеҸ‘еҗ„з§Қ3Dе’Ң2DйЎ№зӣ®пјҢеҰӮи®Ўз®—жңәжёёжҲҸгҖҒеҸҜи§ҶеҢ–и®ӯз»ғеҷЁгҖҒVRзі»з»ҹпјҢд»ҘеҸҜи§ҶеҢ–иҝҮзЁӢе’ҢејҖеҸ‘зӘ—еҸЈеә”з”ЁзЁӢеәҸгҖӮ

еңЁжӯӨзүҲжң¬дёӯпјҢеј•ж“Һе·Ід»Һ.NET FrameworkиҪ¬жҚўеҲ°.NET Core 3.1пјҢе·Іж·»еҠ дәҶеҜ№Visual Studioд»Јз ҒгҖҒRiderе’Ңе…¶д»–C#IDEзҡ„ж”ҜжҢҒпјҢзј–иҫ‘еҷЁе·Іеҫ—еҲ°ж”№иҝӣе’ҢдјҳеҢ–пјҢеңЁеҗҜеҠЁеҷЁдёӯжӮЁзҺ°еңЁеҸҜд»ҘеҲӣе»әWindows AppйЎ№зӣ®пјҢиҝҳеҗ‘Assets\Samples\addonsж–Үд»¶еӨ№ж·»еҠ дәҶеҲӣе»әеҠ иҪҪйЎ№зҡ„зӨәдҫӢгҖӮ

еңЁжӯӨзүҲжң¬дёӯпјҢеј•ж“ҺеҚҮзә§еҲ°.NET Core 3.1пјҢд»ҘеүҚдҪҝз”Ёзҡ„жҳҜ.NET Framework 4.6гҖӮиҝҷжҸҗй«ҳдәҶжҖ§иғҪгҖӮеј•ж“Һе’Ңзј–иҫ‘еҷЁжҳҺжҳҫжӣҙеҝ«гҖӮзҺ°еңЁеҲӣе»әзҡ„еә”з”ЁзЁӢеәҸдёҚйңҖиҰҒе®үиЈ….NETпјҢе®ғеҢ…еҗ«еңЁйЎ№зӣ®дёӯгҖӮзҺ°еңЁиҝҳеҸҜд»ҘиҪ»жқҫең°е°ҶйЎ№зӣ®жӣҙж–°еҲ°иҫғж–°зүҲжң¬зҡ„.NETпјҢеҰӮ.NET5гҖӮ

зҺ°еңЁеҸҜд»ҘдҪҝз”Ёд»»дҪ•ж”ҜжҢҒC#е’Ң.NET Core 3.1зҡ„IDEжқҘж”ҜжҢҒйЎ№зӣ®зј–иҜ‘гҖӮ

еҲӣе»әзӘ—еҸЈеә”з”ЁзЁӢеәҸзҡ„иғҪеҠӣе·Іеҫ—еҲ°ж”№иҝӣгҖӮж–°е»әйЎ№зӣ®ж—¶пјҢжӮЁеҸҜд»ҘйҖүжӢ©Window AppжЁЎжқҝпјҢдҪҝз”ЁиҜҘжЁЎжқҝеҸҜд»Ҙи®ҫзҪ®еҲқе§Ӣи®ҫзҪ®гҖӮиҜҘеј•ж“ҺеҢ…жӢ¬иҮӘе·ұзҡ„GUIзі»з»ҹе’ҢGUIи®ҫи®ЎеҷЁгҖӮиҝҷе…Ғи®ёжӮЁеҲӣе»әжҲҗзҶҹзҡ„зӘ—еҸЈеә”з”ЁзЁӢеәҸпјҢдҪҶжҳҜпјҢеҲӣе»әзӘ—еҸЈеә”з”ЁзЁӢеәҸд»Қз„¶жҳҜNeoAxisзҡ„дёҖйЎ№ж–°еҠҹиғҪгҖӮ

зј–иҫ‘еҷЁж”№иҝӣдәҶWindowsзҡ„е·ҘдҪңпјҢжҸҗй«ҳдәҶжҖ§иғҪгҖӮжҖ»иҖҢиЁҖд№ӢпјҢзј–иҫ‘еҷЁзҺ°еңЁжҳҜдёҖдёӘз”ЁжҲ·еҸӢеҘҪзҡ„гҖҒзІҫиҮҙзҡ„еә”з”ЁзЁӢеәҸгҖӮ

д»Һ.NET Framework 4.6еҲҮжҚўеҲ°.NET Core 3.1гҖӮе®ғзҡ„е·ҘдҪңйҖҹеәҰжӣҙеҝ«гҖӮзҺ°еңЁжӮЁеҸҜд»ҘиҪ»жқҫең°е°ҶйЎ№зӣ®еҚҮзә§еҲ°жңҖж–°зҡ„.NET(дҫӢеҰӮ.NET5.0)гҖӮ

еңЁеҠ иҪҪзј–иҫ‘еҷЁзҡ„еҗҢж—¶иҝҗиЎҢNeoAxis playerж—¶пјҢеҜ№еҶ…зҪ®зј–иҫ‘еҷЁProject.csprojзҡ„зј–иҜ‘ж”ҜжҢҒжӣҙеҘҪгҖӮ

з»„д»¶зұ»е·Іжӣҙж–°пјҢд»ҘдҫҝжӣҙеҘҪең°ж”ҜжҢҒи·Ёе№іеҸ°гҖӮйЎ№зӣ®DLL(NeoAxis.CoreExtension.dllгҖҒProject.dllгҖҒAdd-ons)зҺ°еңЁдёҚдҪҝз”ЁSystem.Windows.Form.dllгҖҒSystem.Drawing.dllгҖӮ

NeoAxis.Core.dllзҺ°еңЁжІЎжңүеҜ№е…¶д»–DLLзҡ„еј•з”ЁгҖӮиҝҷдҪҝеҫ—з®ЎзҗҶеә“еј•з”ЁеҸҳеҫ—жӣҙе®№жҳ“пјҢеӣ дёәеҸӘйңҖиҰҒдёҖдёӘеҜ№ж ёеҝғDLLзҡ„еј•з”ЁгҖӮ
