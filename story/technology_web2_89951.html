Firefox必备插件推荐汇总

by onehedge on 2008-09-07 12:11:50

#

Adblock Plus:Firefox必备广告拦截插件，让飞来飞去的广告消失。

#

Add to Search Bar:浏览器右上角的搜索引擎目录太少？把你想要的搜索引擎诸如VeryCD，豆瓣，Google全套，百度全套，全放上去。

#

Auto Context : 每首反白一段文字后都需要右击搜索才能进行后续操作，这个插件让你瞬间完成任务。自动调出右键菜单，可自定义。

#

Autofill Forms：每次去好友博客添加评论都要重复那一次姓名，邮箱，网址？有了它以后一切都是自动的。帮你自动填完所有常用表单。

#

Better GReader :让我订阅的种子显示Favicon图标，多种Reader皮肤，直接在阅读器页面阅读原文（对付那些不输出全文
