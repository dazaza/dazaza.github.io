Mobile phone Want Want official beta version launched

by yinyin on 2010-03-30 10:09:10

After a long period of no official version of mobile phone Want Want, in order to cooperate with the promotion of mobile phone Taobao, Taobao finally launched the mobile phone Want Want public beta version, written in Java. It has added an important member to the Taobao family, which is of great significance to Taobao. Although the phone is still in the public beta stage, the UI is already very beautiful, using purple-black glass interface. However, the carrier's CMWAP or UNIWAP connectivity is not currently supported. I hope Taobao can launch a stable official version as soon as possible.
