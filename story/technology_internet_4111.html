Currently, web-based instant messaging services are emerging one after another. Among them, the most famous one is undoubtedly Meebo, but there are also big players like imhaha (the first to launch a web-based version of QQ, which was a great blessing for domestic users). Here's a small summary of the web-based instant chat sites I've come across:

1. **Meebo**

- Website: [http://www.meebo.com/](http://www.meebo.com/)

- Meebo.com is a web-based multi-protocol IM service similar to eMessenger. It currently supports five instant messaging services: GTalk, MSN, AOL or ICQ, and Yahoo! Like integrated instant messaging software such as MYIM, Gaim, and Trillian, it provides text-based chat communication, but it does not support voice or video calls, group chats, or email notifications. The difference is that Meebo.com uses AJAX, so there's no need to install any software on the client side. You can simply log in to meebo.com to use it, providing a full Windows client user experience. It is highly recommended for occasional use on computers where GTalk, MSN, AOL or ICQ, and Yahoo! are not installed.

2. **Ebuddy**

- Website: [http://www.ebuddy.com/](http://www.ebuddy.com/)

- Ebuddy is another web-based integrated IM tool that also supports mobile IM communication. Currently, Ebuddy supports mainstream international IM tools such as MSN, Yahoo!, and AIM. Although Meebo, introduced earlier, also offers similar services, what makes Ebuddy stand out is its support for MOBILE terminals. In the currently popular mobile application field, Ebuddy is leading the way.

3. **Kool IM**

- Website: [http://www.koolim.com/](http://www.koolim.com/)

- Koolim is another online instant messaging server. It is basically similar to the two previously introduced services, except that as a newer service, its speed is still acceptable. It currently supports mainstream IM communication protocols such as AIM, MSN, Y! IM, Gtalk, ICQ, and Jabber.

4. **I Love IM**

- Website: [http://www.iloveim.com/](http://www.iloveim.com/)

- There isn't much to introduce here; it's basically the same as the ones mentioned earlier, currently supporting only MSN, Y! IM, and AOL.

5. **Imhaha**

- Website: [http://www.imhaha.com](http://www.imhaha.com)

- Recently, the hottest has been Imhaha, not because of any major breakthrough in functionality, but because Imhaha supports QQ! Imhaha was founded by a company called "Imhaha from Silicon Valley." Its layout is very similar to Meebo, and its biggest advantage is that it supports QQ! There's nothing else particularly noteworthy. If you're a fan of QQ, you should give it a try.

6. **Goowy**

- Website: [http://www.goowy.com/](http://www.goowy.com/)

- Goowy itself is not a web chat tool; rather, it's a web-based desktop system (similar to WebOS Desktop). However, it has built-in IM services for MSN, Y! IM, and AOL. With a pure Flash-based interface, it's very comfortable to use.
