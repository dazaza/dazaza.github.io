Microsoft: Enterprise users should start deploying Win8.1 now

by anonymous on 2013-08-02 10:13:20

Today, the public release of the Windows 8.1 Enterprise Preview has been announced, and Microsoft recommends that enterprise users download and experience Windows 8.1. Microsoft stated that the full version of Windows 8.1 Enterprise will be released later this year, and IT professionals should start deploying Windows 8.1 now.

Microsoft will also release the official versions of deployment tools simultaneously with the official release of Windows 8.1 Enterprise, including the Assessment and Deployment Kit for Windows 8.1, Microsoft Deployment Toolkit 2013, and System Center 2012 R2 Configuration Manager.

For users who are currently using Windows 8 Enterprise, upgrading to Windows 8.1 Enterprise is very simple: just enable automatic updates on your system, and the entire upgrade process will take approximately 15 minutes.

For users running older versions of Windows, you can use free tools such as the Microsoft Deployment Toolkit 2013 or System Center 2012 R2 Configuration Manager. Using these tools, you can achieve a fully automated upgrade and deployment process.

The Windows 8.1 Enterprise Preview includes both 32-bit and 64-bit versions, and the ISO image provides the following language editions: Arabic (Saudi Arabia), Simplified Chinese, Traditional Chinese, English, French, German, Japanese, Korean, Portuguese (Brazil), Russian, Spanish, Swedish, and Turkish.

System requirements for Windows 8.1 Enterprise Preview:

— Processor: 1 GHz or faster

— Memory: 1 GB (32-bit) or 2 GB (64-bit)

— Available hard disk space: 16 GB (32-bit) or 20 GB (64-bit)

— Graphics card: Microsoft DirectX 9 graphics device with WDDM driver

To use certain specific features, the following additional requirements must be met:

— For 64-bit installation, your CPU must also support CMPXCHG16b, PrefetchW, and LAHF/SAHF.

— To use touch functionality, you need a tablet or monitor that supports multi-touch.

— To access the Windows Store and download and run apps, you need a valid Internet connection, at least 1024 x 768 screen resolution, and a Microsoft account.

— To Snap applications side by side, you need a minimum screen resolution of 1366 x 768.

— Internet access (possible ISP fees)

— Some features (such as touch and client Hyper-V) may require advanced hardware or hardware certified for Windows 8.
