Making red lanterns with 3D Studio Max

by lk0pcdd on 2009-07-22 16:53:16

By Wang Weiguang

"In the first month, on the 15th day, we celebrate with lanterns ablaze; lantern flowers, red lanterns, snowfall on the lanterns foretells a good year." As children, we would sing this nursery rhyme and carry lanterns to welcome the 15th day of the first lunar month every year. With these beautiful childhood memories, I will now demonstrate how to create a large red lantern using 3D Studio Max.

**Production Tool:** 3D Studio Max 5.0 Chinese Edition

**Key Points:** Use commands such as Freeform Deformation, Array, Rotate, and Scale

### I. Modeling the Lantern

1. Click the "Create" button to enter the creation command panel. Then select the "Shapes" button to access the shape creation panel. Click the "Line" button and draw a 2D line in the "Front" view. In the "Selection Set," click the "Vertex" button, right-click, and choose "Bezier" to adjust the shape of the 2D line. Then click the "Curve" button under "Selection Set," select the 2D line, and choose "Offset" from the "Geometry" menu with an offset value of 0.1 cm.

2. Click the "Modify" button to enter the modification command panel. From the drop-down list, select the "Rotate" command and set the parameters to "Minimum" with 36 segments.

3. Click the "Geometry" button to enter the geometry creation panel. Click the "Sphere" button and create a sphere in the "Top" view. Set the radius parameter to 60 cm. Click the "Non-uniform Scale" button in the toolbar, then click the "Y" button to scale the sphere.

4. Click the "Modify" button to enter the modification command panel. From the drop-down list, select the FFD4×4×4 command to access the freeform deformation modification panel. Click the "Control Points" option to activate the sub-object editing state, and adjust the control points to form a circular arc. Click the "Array" button in the toolbar to open the array dialog box. Set the rotation Z-axis to 10 degrees, quantity to 18, and click "OK."

5. Click the "Geometry" button to enter the geometry creation panel. Click the "Cylinder" button and create two cylinders in the "Top" view. Set the parameters to a radius of 15 cm and height of 12 cm.
