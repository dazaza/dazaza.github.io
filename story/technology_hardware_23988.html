从1976到2005！30年掌机发展全盘回顾

by cacard on 2007-07-04 02:58:44

你是否会在坐车的时候感觉无所事事？你是否会在旅游的间歇找点小乐子呢？或者是在上WC的时候……，嘿嘿！这些时候，年轻的一代会想到NDSL或PSP，当然还有手机了！便携的掌上娱乐设备使我们的生活更加充实有趣，而为游戏而生的掌机都经历了那些历程呢？下面我们将通过历史上出现的掌机，带您回顾掌机的发展历程……

Mattel公司掌机系列 1976-78年

Mattel Electronics Handheld Games

　　尽管真正意义上的“可更换卡带式掌机”的出现还是十几年以后的事，但Mattel公司早在70年代就开始尝试开发一些既有别于投币式街机，也不同于家用电视主机的新型游戏装置，这就是一系列基于LED技术的单人便携式掌机。尽管今天人们能够记住的大概也只有这一部游戏掌机，但事实上Mattel公司还曾推出过其它一些极具创意的体育游戏，如《Baseball》与《Basketball》；以及一些非体育作品，比如《Missle Attack》、《Armor Battle》、《Sub Chase》等等。近年来，Mattel又掀起了一阵怀旧风，于2000年重版了《Football》与《Baseball》。

　　Mattel Electronics Handheld Games的诞生，标志着游戏机硬件平台类型的最终完善，电脑，家用机，街机和便携机四大门类从此齐全了。这个掌机只有极其简单的LED显示屏幕，游戏也都是随主机固化的。这个东西至今在美国还可以买到。它的贡献在于提出了平台便携这样的概念，对游戏消费群体的扩张和游戏方式的扩张起到了积极作用。至于它本身的游戏性，则是基本不值一提的。

●Microvision - Milton Bradley公司 1979年

Microvision

　　Milton Bradley公司，最早是因“Hungry Hungry Hippos”（一种四人玩的鳄鱼吃豆子的儿童玩具）而非电子游戏而出名。不过，它的最大声望却是创造了世界上第一台可更换卡带的掌上游戏主机Microvision。该主机的游戏很少，而且从一开始就备受各种问题困扰，包括那个很容易被腐蚀的“16×16象素”LCD屏，以及极易受到静电冲击而损害的卡带。如果你硬要问这款掌机的实际价值，只能说它是一个不错的收藏品。

● Game & Watch系列 - 任天堂 1980-91年

　　决意像Mattel那样赌一把的任天堂，于1980年推出了它们的首部“Game & Watch”掌机，并在随后十年里推出了十数种系列产品。就像它的名称所暗示的那样，该掌机装有一个时钟和报时器，不过它真正吸引人之处当然在于游戏（是不是很像NDS？）。

● Game Pocket Computer - Epoch公司 1984年

　　在Milton Bradley公司推出Microvision五年之后，才有另一家公司正式尝试开发可更换卡带的便携掌机，可是这一次比上一次败得还惨。Epoch公司于1984年推出的Game Pocket Computer仅在日本地区上市，游戏则只有区区五款。虽然75*64的LCD屏比Microvision进步了不少，但它却从未流行开来，因为任何真正接触过它的人,都不会对它存有任何好感。

● Game Boy - 任天堂 1989年

　　任何语言都无法准确阐释Game Boy的影响力。Game Boy，一部有着多种变种机型的掌机，毫无疑问是有史以来最成功的一部主机——不管是就掌机领域，还是整个游戏机领域而言。GB的成功，部分是因为其合理的价格(日版:12600日元,美版109美元)，但更大的成功因素却在于它同捆了一些可玩性很高的游戏，和同期竞争对手，只有一个模糊绿色屏幕和简陋图象的GB显得毫无优势，但它却成功地证明了“游戏决定主机,而非机能决定”的定理。

　　为了延续GB的生命力，任天堂又先后对它做了多次重大改进。1996年推出了更加轻薄的Game Boy Pocket(它采用了一个色调更舒服的崭新绿色屏幕)，以及采用了背光式屏幕的Game Boy Light(可惜的是，它仅在日本发行)。

● Lynx / Lynx II - Atari公司 1989年

　　Game Boy的第一个挑战者是Atari与Epyx联合开发的Lynx(1989年发行)。这部主机的画面效果远远强过GB，甚至不输于当时的家用主机。但它的体积过于庞大，价格也比实惠的GB贵得多。尽管Atari于1991年重新设计了该主机，但Atari糟糕的市场策略又远远不及任天堂，这从反面促进了老任在掌机市场的霸者地位。

● Turbo Express - NEC公司 1990年

　　野心勃勃的NEC试图打造一部技术最先进的掌机,这就是Turbo Express。实际上，它就是Turbgraphx 16主机(PC-Engine)的便携版本。Turbo Express不但与Game Boy有着同样的体积，而且有着更为先进的活动矩阵彩色显示屏，甚至还能变成一部移动电视(加装一个选择调谐器)。但它的不利之处显然在于价格，299,99美元的高价格看来是对准了重度掌机迷市场——一个实际上并不存在的市场。

● Game Gear - 世嘉 1990年

　　Game Boy众多挑战者中比较成功的一位，和Lynx与Turbo Express一样，世嘉的Game Gear也采用了彩色屏幕，但其价格却大大压低到一个相对合理的价位：149美元。Game Gear借助世嘉的优势，当然获得了比Atari/NEC掌机更大的成功，并有着更为丰富的软件选择，但在任天堂面前，GG的成功就显得黯然失色了。

● Nomad - 世嘉 1995年

　　20世纪90年代大部分时间里，任天堂都占据着掌机市场的统治地位，其它厂商在经历了无数失败的尝试后，也都放弃了对掌机市场的觊觎。90年代周期，世嘉成了第一个重新介入该市场的厂商，它发布了Genesis的便携版Nomad.这似乎是一个好主意——毕竟Genesis有着丰富的软件库，市场占有量也相当庞大——但是可怜的电池寿命与体积过大的设计最终毁了它。就算它的价格最好降至79.99美元，也仍然无法挽救Nomad被埋葬的命运。

● game.com - Tiger Electronics公司 1997年

　　你不能质疑Tiger Electronics的野心。他们的game.com掌机——就像其名称所显示的那样——整合了互联网访问与PDA等功能。但不幸的是，它却没有一个方面做好/它的游戏本就令人失望。再加上那过时的屏幕显示。就更糟糕了。它所谓的“互联网访问”功能其实就只能查阅电子邮件，或是浏览一些文本网页而已，而没有任何网络游戏功能。一些死忠玩家们只有在网上才能找到一丝的慰籍，他们为这台不幸的掌机开发自制软件，用一种特别的爱护方式延续着它的生命力。

● Neo-Geo Pocket / Pocket Color - 1998-99

　　提到Neo-Geo这个名字，那些25岁以上的玩家一定会发出会心的微笑。尽管只有少数幸运儿拥有过这部绝对强过SNES/Genesis的家用主机，但Neo-Geo最为人熟知的当然还是它的街机游戏——尤其是与等格斗系列。

　　为了建立自己的名望，Neo-Geo也将触手伸向了掌机市场，于1998年推出了Neo-Geo Pocket.但这部主机的开始阶段很不顺利，这主要是因为它只有一个简陋的黑白屏幕，直到一年之后，SNK才修复了这个错误，推出了彩版的Neo-Geo Pocket Color(简称NGPC)。虽然这部掌机有着非常不错的游戏，但它却从未得到过第三方的有效支持，也就无法吸引足够的玩家来挑战任天堂的权威。

● Game Boy Color-1998

　　1998年，任天堂推出了Game Boy的第一个重要改进版本：Game Boy Color，毫不意外的是，它提供了一个彩色屏幕，其画面效果当然因此而大大加强，并且完全兼容GB软件。虽然这套主机获得了成功，但它更像是任天堂的一个权益之计，因为老任还酝酿着更大的计划。

● WonderSwan/WonderSwan Color/WonderSwan Crystal - Bandai 1999-2000

　　Bandai的WonderSwan(一年后被WonderSwan Color取代)，最引人注目之处在于独占了一部(一代)移植作。它也因此在日本获得了一定的成功，但在美国却没有任何作为。对Bandai来说更糟糕的是：Square又与任天堂修好了关系，开始为GBA开发游戏，这让WonderSwan这款设计新颖独特的掌机彻底没了转机。

● Game Boy Advance / Advance SP / Micro - 2001 / 2003 / 2005

　　2001年推出的Game Boy Advance，是打10年前Game Boy发售以来对掌机市场的最大一次震动。从技术层面来说，GBA与SFC相差不多，因此很容易将后者的作品移植过来，但是GBA也不乏Advance Wars这样高品质的原创游戏。

　　2003年，任天堂又推出了经过彻底重新设计的Game Boy Advance SP，最重要的是，它增加了一个前发光屏幕，解决了GBA原版用户所遭遇的一个大问题。

　　最近，任天堂又发布了另一个GBA的变种机型—Game Boy Micro，这台更小巧，更圆滑的掌机，显然将消费群对准了年龄层次更广泛的iPod观众。

● N-Gage / N-Gage QD - 诺基亚 2003-04

　　在N-Gage问世之前，谁也没有将诺基亚和游戏厂商联系在一起——即使在现在，即N-Gage上市几年后，诺基亚仍然在为吸引玩家们的注意力做着努力。尽管N-Gage/QD是一款能力相当出众的游戏设备，但现在大多数人显然还是更愿意将手机和游戏机区别对待。尽管如此，诺基亚仍有希望在未来数年内改变一切，当前它正打算将N-Gage的游戏功能整合到未来的智能电话中——这可绝对不是什么游戏手机。

● DS / DS Lite - 任天堂 2004 / 2005

　　甚至早从第一张概念图在网上公布起，NDS就受到了广泛的质疑。对很多人来说，它的外形设计和雅致的GBA SP比起来，似乎是一种倒退。它的双屏和触控笔输入也更像是一种噱头，而非任天堂所承诺的革新。但是随着越来越多的大作的推出(包括一些极具创意的作品)，玩家们开始慢慢爱上了这部主机。它的内置WiFI功能更是将掌上游戏的网络功能提升到了一个的层次。而随着更轻薄，更漂亮的NDS Lite的推出，任天堂对掌机市场的统治无疑还将继续下去。

● PlayStation Portable - 索尼 2004

　　作为目前最先进的一部掌机，索尼的PSP已稳稳地在掌机市场打下了一片属于自己的天地，哪怕它面临的是任天堂的强势统治，哪怕它的价格较高，好游戏较少。索尼还希望通过PSP来推动UMD格式电影，但现在看来已没有成功的可能，但更多玩家显然更愿意用它来看MP4等格式的电影。如果你能选择正确的固件版本,甚至还能使用许多好用的自制软件。

● Gizmondo - Tiger Telematics 2005

　　Tiger Telematics公司与Gizmondo的前景显然并不美妙，这主要是因为它是一部“样样通，但样样不精”的掌机。这部主机相当强大，甚至还拥有GPS与内置摄像头这样的一流功能。但由于没有引人注目的游戏和合理的价格，它从诞生起就要面临着艰难的挣扎。Tiger公司甚至从一开始就没有一个正确的指导方向，它试图通过一种名为“Smart Adds”的广告系统，来降低主机的价格(只要玩家同意观看该系统中播发的广告，就能获得打折优惠).但这样的主意显然不会被用户理睬，Tiger公司最终在欧洲申请了破产。

● GP32/GP2X － GamePark

　　对很多人来说，GP2X(包括之前的GP32,与之后的XGP/XGP Mini)系列掌机就仿佛降临掌机业的一尊圣杯。它拥有记忆卡，还支持多种模拟器，你几乎可以在它上面玩任何平台的任何游戏，当然也包括数以千计的街机与家用机游戏。这足以吸引那些怀旧派玩家的目光，当然也不会被其它公司的版权律师放过。如果你手上拥有模拟游戏的原版，那最好，如果没有，试试用它来怀旧也不错。

● PDA模拟器

　　PDA也许并不适合游戏(尽管它拥有良好的点击式操作特性)，但这并不妨碍开发商们为它开发原创游戏以及模拟器。确实，它不是GP2x，但它却拥有更多其它掌机所没有的优势，比如偷玩功能。你可以一面假装工作，一面痛痛快快地打。
