Function Introduction: The features of this program are as follows:

1. Resetting does not change the difficulty level;

2. Power loss does not trigger alarms or damage accounts;

3. Stable performance without losing money;

4. Independent score-clearing record and real-time profit record;

5. Score memory upon shutdown;

6. No battery required and zero soft faults for the 6116 chip;

7. No vulnerabilities when shutting down that could cause losses;

8. Company machine difficulty set by a coding device;

9. Dual password protection to prevent others from decoding;

10. Real-time calendar with timekeeping even when powered off;

11. Adjustable number of days the machine can be in operation, segment profit scores, and bonus scores for the gaming area!;

12. After reaching segment profits, it will automatically set the bonus score for the gaming area.;

13. Resistant to infrared scythes and manual pull scorers for rapid theft prevention.

Light show patterns: Includes Little Four Joys, Little Three Yuan, Big Three Yuan, Train Opening, Grand Slam, JP Award, Lucky Star (up to 3 times), Traveling Far and Wide, Cat Transformation, Fairy Scattering Flowers, High Multiplier Spotlight (any light up to 100x), and High Multiplier Train, etc.

Comparison and multiplier patterns: Can guess big/small and odd/even, JP Award, Open Card (revealing 7 doubles the score), etc.

Two, Game Description:

Upper middle x100: Spotlight 100x.

Lower middle Yuanbao: All bets rewarded, scoring a dragon gives all placed points multiplied by 1~13x, up to 3 consecutive times.

Left/right opportunity gates: Scoring a dragon 0000~2222: Train opens with 2~9 cars.

3333: Inner circle high-multiplier train with 1~3 cars,

4444: Traveling Far and Wide, six lights in all directions hit simultaneously, Guangzhou gift machine.

5555: Little Four Joys, three apples hit simultaneously.

6666: Little Three Yuan, oranges, papayas, and bells hit simultaneously.

7777: Big Three Yuan, watermelons, stars, and sevens hit simultaneously.

8888: Jackpot draw, placed points multiplied by 5~15x.

9999: Grand Slam, all 24 lights hit simultaneously.

Four different numbers: Big spotlight (10x or 20x).

When guessing odd/even for multipliers: If two out of three numbers are odd, it's considered "odd", if two are even, it's considered "even", not referring to the point value being odd or even.

Three identical numbers double 2~9x.

Three, Account Checking Method:

Press and hold the K0 key or the add-score key while turning on the machine. Press the start key to switch pages, page numbers displayed in the multiplier section, account details shown in the upper-right total score bar.

0 item: Total profit (Yuan), JP light indicates negative.

1 item: Total coins inserted + total added scores (Yuan).

2 item: Total coins returned + total deducted scores (Yuan).

3 item: Current actual profit = 0 item - 4 item (Yuan).

4 item: Previous total profit.

5 item: Number of score-clearing coins. Note: Coins returned = 2 item - 5 item.

6 item: Machine running time (machine stops after 5000 minutes).

7 item: Number of startups (number of barcode inputs).

8 item: Machine number.

9 item: Total play score for light shows.

10 item: Total win (profit) score for light shows.

11 item: Total play score for multipliers.

12 item: Total win (profit) score for multipliers.

13 item: Total remaining score on the machine.

14 item: Bonus score left on the machine.

15 item: Account checksum (used to prevent incorrect accounting).

16 item: Minimum thousandth ratio for light shows; smaller data means more excitement, larger fluctuations. Adjusted in K3 item 0!

17 item: Minimum thousandth ratio for multipliers, game machine supplier, smaller data means more excitement, larger fluctuations. Adjusted in K3 item 1!

18 item: Profitable score played in the current period, equals (9 item - 10 item + 11 item - 12 item).

19 item: Maximum thousandth ratio for light shows, smaller data means harder, do not lower than item 16 otherwise no fluctuations. Adjusted in K3 item 2!

20 item: Maximum thousandth ratio for multipliers, smaller data means harder, do not lower than item 17 otherwise no fluctuations. Adjusted in K3 item 3!

Note: On page 0, the data to the left of the bet strip is the actual thousandth ratio for multipliers, the data to the right is the actual thousandth ratio for light shows, the middle '0' indicates a buy/sell machine, '1' indicates D-type company machine, '2' indicates E-type company machine, '3' indicates G-type company machine.

Four, Delay Method:

A) After the company machine exceeds the set number of days, it will stop with code 09, first check accounts and report items 0, 3, 7, 8, 15, then use the barcode scanner to calculate the delay password.

B) Press and hold K0+K3 keys or add-score key+deduct-score key while starting the machine, if 8 zeros appear, input the correct delay password using the corresponding bet key, then press the start key below to display 99, after displaying 99, it will not clear the field nor return eaten scores.

C) Input 0 page + 3 page passwords + user-defined password (see K2 instructions) then press the start key to display 99 to complete!

Five, Zeroing Method:

After delay, item 3 in the account check becomes zero. At this point, if you repeat the delay once more, it will reset and output 66; if item 3 is not zero, encoding still outputs 99 as a delay.

Six, Parameter Adjustment:

(This adjustment must be done after resetting, otherwise alarm 42 will sound!)

Press and hold K3 key while starting the machine, input the adjustment password (wrong password will trigger alarm 41) then press the start key to enter. Press the start key to switch pages, page numbers displayed in the multiplier section, data displayed in the total score section. Press the corresponding bet key to adjust each digit of the data:

Page 0: Light show fluctuation: Minimum winning rate in thousandths for light shows, smaller data means greater fluctuation.

Page 1: Multiplier fluctuation: Minimum winning rate in thousandths for multipliers, smaller data means greater fluctuation.

Page 2: Light show difficulty: Maximum winning rate in thousandths for light shows, smaller data means greater difficulty.

Page 3: Multiplier difficulty: Maximum winning rate in thousandths for multipliers, smaller data means greater difficulty.

Note: Games randomly deal cards between the minimum and maximum thousandths ratios. There must be a difference between high and low for there to be fluctuations!

Suggest adjusting to 900%% when 1 Yuan = 5 coins, suggest adjusting to 930%% when 1 Yuan = 1 coin.

Page 4: Segment profit score: Indicates the score needed to reach before the game area clears once.

Page 5: Segment clearing score: Indicates the score value needed to reach before the game area clears once after reaching the segment profit score! Set to 0 to turn off!

For example: If segment profit score is set to 1000 points, and segment clearing score is set to 500 points:

Every time the light show reaches 1000 points, it automatically clears 500 points.

Every time the multiplier reaches 1000 points, it automatically clears 500 points.

Adjust according to need, but after adjusting the company machine, it must be reset to take effect. Otherwise, alarm 21 will sound.

Nine, System Adjustment:

Press and hold K2 key while starting the machine, input the adjustment password then press the start key to enter. Press the start key to switch pages, page numbers displayed in the multiplier section, data displayed in the total score section. Press the corresponding bet key to adjust each digit of the data:

Page 0: Machine number: Random code at position 8, positions 7~6 are the regional code, positions 5~1 are the machine number.

Page 1: Days until machine stops: Used for company machine control. Invalid for buy/sell machines!

Page 2: User password: Used for delays and resets, factory default is seven 0s.

Page 3: Adjustment password: Used next time pressing K3, factory default is seven 0s.

Page 4: System password: Used next time pressing K2, factory default is seven 0s.

Note: After adjusting passwords, please make sure to record them in a notebook, do not forget! Otherwise, decryption will require sending to the manufacturer and incur a decryption fee.
