Facebook 详细介绍了实验性混合现实和直通 API

2021-07-24 10:19:14

Facebook 分享了有关其实验性 Passthrough API 的一些细节，以便为 Oculus Quest 2 启用新型混合现实应用程序。该功能还可以作为该公司在增强现实领域的长期努力的基础，有效地将 Quest 2 转变为价值 299 美元的 AR 开发人员成套工具。当被问及该功能是否会出现在最初的 Oculus Quest 中时，一位 Facebook 代表回答说：“今天，这仅适用于 Quest 2。”新功能将在即将发布的软件开发工具包中提供给 Unity 开发人员，“未来将支持其他开发平台。” Facebook 表示，使用 API 的应用程序“无法访问、查看或存储来自 Oculus Quest 2 传感器的物理环境的图像或视频”，来自四个车载摄像头的原始图像“在设备上进行处理”。合成：您可以通过现有的混合技术（例如打孔和 alpha 混合）将 Passthrough 图层与其他 VR 图层合成。样式：您将能够将样式和色调应用于预定义列表中的图层，包括将颜色叠加应用于提要、渲染边缘、自定义不透明度和海报。自定义几何体：您可以将 Passthrough 图像渲染到自定义网格，而不是依赖默认样式网格 — 例如，将 Passthrough 投影到平面表面上。

Facebook 认为 Passthrough API 可以实现更强大的生产力应用程序示例——远程工作解决方案 Spatial 是最早访问 API 的应用程序之一——以及与 VR 内容同时与物理房间中的人或宠物互动的能力，它可以开辟新类型的游戏，可以融入更多的物理环境。
