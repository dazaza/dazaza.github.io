Internet software bundled installation test: installing 4 software may become 12

by tuoqing1188 on 2011-08-04 09:38:26

Did I really agree to let you install it?

The Ministry of Industry and Information Technology (MIIT) released the "Regulations on the Administration of Internet Information Services (Draft for Comments)" last week. The regulations once again clearly require that there should be no unfair competition among internet information service providers, and software must not be bundled and installed without user selection. Violators could face a maximum fine of 30,000 yuan. The reporter tested the bundling installation situation of four of the most commonly used software programs in the Chinese internet, and found that under default installation conditions, installing these four software programs could potentially result in up to twelve additional software programs being installed on the computer.

"Default selected" software galore

The reporter selected four of the most commonly used internet software programs currently on the Chinese internet for testing: Tencent QQ, 360 Security Guard, Sogou Pinyin Input Method, and Xunlei download software. A laptop with newly installed Windows XP was used.

According to Article 9 of the Regulations, internet information service providers must not engage in the following behaviors: (1) Without prompt and active user consent, arbitrarily install, run, upgrade, or uninstall software on user terminals, or install, run, upgrade, or uninstall software by means of deception, misguidance, or coercion.

The key point of this regulation lies in "active user consent." Do the above four software programs comply with the regulations when installing their bundled software?

When installing Tencent QQ 2011 (Q+) version, four software programs (plugins) including QQ Toolbar and Chinese SouSou are pre-selected by default in the custom installation options. If users follow the conventional "just click next all the way" installation method, these four software programs will all be installed on the computer.

When installing 360 Security Guard, a window will appear with "Install 360 Default Browser" pre-selected by default. If this option is manually deselected at this step, immediately after the Security Guard is installed and runs for the first time, a window recommending "the lightest and fastest dual-core browser" will pop up. This window has only a "Try Now" button apart from the close button. At the same time, upon its first run, the Security Guard will also pop up windows recommending products like 360 Compression and 360 Secure Desktop. Each product page has only a "Try Now" button apart from the Windows default close window button, with no "Cancel Installation" buttons.

No-prompt installation still exists

The reporter also found that despite there being no prompts during the installation process, after installation, an independent 360 Software Manager is installed along with the 360 Security Guard.

When installing Sogou Pinyin Input Method, "the world's first true dual-core browser" Sogou High-Speed Browser is pre-selected by default.

Unlike the aforementioned three software programs, Xunlei's bundled software installation appears on the last page of the installation process, where Baidu Search霸 is pre-selected by default.

■ Response

"User Active Choice" ambiguous

In summary of the most popular software domestically, all the software provide varying numbers of bundled software, and these bundled software are not essential for the original software to function. Although all bundled software can be canceled via manual deselection, under the mindset of many novice users who think "I don't know what is useful and what isn't, so the safest way is to install according to the default settings," the pre-selected software is usually installed on the computer.

The reporter contacted the developers of the aforementioned software regarding the above issues.

Tencent responded that during the software installation process, users can independently cancel the installation of bundled software, so there is no phenomenon of installation without user consent.

360 responded that after the new MIIT regulations take effect, the company will strictly abide by them.

Sogou replied that the key point of the regulation is whether users are given the choice during software installation. "Currently, all products are set to default, and Sogou is consistent with the industry." Sohu CEO Charles Zhang also commented on this regulation during a financial report release this Monday, stating, "Regardless of any regulation, it should benefit industry development, conform to international practices, and align with industry norms."

By the time of deadline, Xunlei Company did not respond.

From the responses of various manufacturers, it is clear that the understanding of relevant provisions in the MIIT Regulations tends to be lenient. The regulations state that software must not be installed "without prompt and active user consent," but the manufacturers believe that if users do not deselect the default bundled software, it can be considered as "active consent."
