Optimal configuration for Windows 2003

by zzjiaodaij1i on 2012-02-27 15:28:31

1. Activate Common Services

Windows 2003, by default, disables many services that are not useful for servers. In addition to "Windows Audio" services related to sound cards and "Themes" services for desktop themes, there are also:

① Image Capture Service

If you need to use imaging devices such as webcams, digital cameras, or scanners, you must enable the image capture service. Go to "Start → Administrative Tools → Services", find the "Windows Image Acquisition" item in the "Services" window, double-click it to enter the settings form, change the "Startup Type" to "Automatic", and click the "Start" button.

② CD Burning Service

By default in Windows 2003, you cannot burn CDs. The solution is: go to the "Services" window, find the "IMAPI CD-Burning COM Service" item, double-click it to enter the settings form, change the "Startup Type" to "Automatic", and click the "Start" button.

2. Log in Without Pressing "Ctrl+Alt+Del"

Every time Windows 2003 starts, you need to press the "Ctrl+Alt+Del" key combination before entering your password to log in, which can be quite troublesome. To disable this requirement, go to "Start → Administrative Tools → Local Security Policy". In the left pane, find "Local Policies → Security Options", and on the right side, find the "Interactive logon: Do not require Ctrl+Alt+Del" option, double-click it, and set it to "Enabled".

3. Move "My Documents"

Open File Explorer, right-click on "My Documents", select "Properties", and in the "Target Folder" tab, click the "Move" button. Choose the target drive and click "OK". In Windows 2003, "My Documents" is hard to find; it's no longer visible on the desktop or in the Start menu. For frequent users, it's recommended to create a shortcut on the desktop.

4. Move IE Temporary Files

Go to "Start → Control Panel → Internet Options". In the "General" tab under the "Internet Files" section, click the "Settings" button. In the pop-up window, click the "Move Folder" button, choose the target folder, and click "OK". Select "Yes" in the confirmation dialog box, and the system will automatically re-login.

5. Move Virtual Memory

Go to "Start → Control Panel → System". In the "System Properties" window, select the "Advanced" tab, click the "Settings" button under the "Performance" section, and select the "Advanced" tab. In the "Virtual Memory" section, click the "Change" button, select the system partition, then choose the "No paging file" option, and click "Set". Confirm with "Yes" in the pop-up window. Then select a non-system partition, choose "Custom size", input the data, click "Set", and finally click "OK". Restart the computer after completing these steps.

6. Move System Environment Variables

Go to "Start → Control Panel → System", select "Advanced", and click the "Environment Variables" button. In the "User variables for Administrator" section, select the Temp item, click the "Edit" button, and change its value (e.g., to "D:\Temp"), then click "OK". Use the same method to change the second item Tmp to the same address. Next, in the "System variables" section, find the Temp and Tmp items and modify them to the same address.

7. Delete Hardware Driver Backup

The hardware driver backup file "driver.cab" for Windows 2003 is located in the "Windows\Driver Cache\i386\" directory of the system partition. After hardware installation is complete, this file can be deleted or moved, freeing up approximately 70MB of space.
