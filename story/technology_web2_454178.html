Analysis Method of Faults in Haier Air-conditioner Remote Control Signal Receiving Circuit

1. Determine whether the remote control signal transmitter is normal. The method to determine whether the remote control signal transmitter is normal is: use a remote control of the same model to start up for verification. If there is no remote control of the same model available, you can also place the remote control near a working AM radio (adjust the radio frequency to the lowest end) (the distance can be within a few centimeters), then press any button on the remote control. If the AM radio speaker emits a brief clicking sound, it indicates that the oscillation encoding circuit of the remote control is intact, and as long as the infrared emitting diode is positive, it will emit infrared light outward. There are also several other methods to check if the remote control transmitter is working normally.

Method One: Visual Inspection Method. The so-called visual inspection method means to first look at the remote control casing with your eyes to see if there are any cracks, shake it with your hands to listen for any abnormal sounds inside, thus judging whether the remote control has been dropped and damaged.

Method Two: Waveform Inspection Method. The so-called waveform inspection method refers to using an oscilloscope (meter) to check the output waveform of the remote control, thereby determining the condition of the remote control. The specific method is: open the rear cover of the remote control, connect the probe of the oscilloscope (meter) to the signal input end of the infrared emission diode, appropriately adjust the scanning speed of the oscilloscope (meter), then press any key on the remote control keyboard, and observe whether there is a pulse signal output on the oscilloscope (meter). If there is a signal output, it indicates that the remote control is working normally; otherwise, it indicates that there is a fault with the remote control.

Method Three: Multimeter Inspection Method. The so-called multimeter inspection method refers to using a multimeter to measure the static/dynamic voltage value of the remote control, thereby determining the condition of the remote control.

(1) Remove the remote control cover, turn on the power of the remote control, and use a multimeter to monitor the voltage across the two ends of the infrared emission diode. Depending on different models of the remote control, you may measure a certain voltage value or 0V, but all are stable.

(2) Press any function control key on the remote control. At this moment, if the voltage value drops to 0V or rises to a certain voltage value, and both are slightly fluctuating, it indicates that the remote control is good. If there is no change in voltage between static and dynamic states, or if the voltage fluctuates when static, it indicates that there is a fault with the remote control.

2. Use the air conditioner's own forced or automatic key to turn on/off. Use the air conditioner's own forced or automatic key to turn on/off. If the air conditioner can run, it is generally due to a receiver fault. If it does not start, it indicates that the fault is on the main control circuit board.

3. Judgment of the remote control signal receiving circuit. To judge whether the remote control signal receiving circuit is normal, press the remote control transmitter and use a multimeter to measure the output terminal voltage of the remote control signal receiver, which should have changes. If there is no change in the output voltage, after confirming that the remote control is normal, the fault is mostly caused by an abnormality in the remote control signal receiving circuit.
