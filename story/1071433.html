AMD 锐龙 5000G 系列以超值价格推出集成显卡

2021-08-05 21:28:06

AMD 正式揭开了其最新的 APU，Ryzen 7 5700G 和 Ryzen 5 5600G 的面纱。正如他们的品牌所暗示的那样，这些新产品基于与锐龙 5000 系列处理器相同的技术，即 AMD 出色的 Zen 3 核心架构，但也具有集成图形功能，因此被称为“G”。然而，这些特定的处理器标志着与类似的上一代 AMD 产品有所不同。虽然 Ryzen 3000G 系列 APU 被降级为最多 4 个内核且 CPU 和 GPU 时钟速度有些慢的入门级系统，但 Ryzen 5000G APU 通过提供多达 8 个内核，最大 CPU 加速时钟超过 4.4 GHz，更快的内存支持，以及显着更高的 GPU 时钟速度。然而，片上图形仍然基于 AMD 的 Vega GPU 架构。当然，AMD 在技术上确实发布了具有更高时钟和核心数的 Ryzen 和 Ryzen PRO 4000G 系列处理器，但这些都是 OEM 专用处理器，并不打算以零售方式出售。与标准的 Ryzen 5000 系列处理器相比，AMD 通过 Ryzen 7 5700G 和 Ryzen 5 5600G 瞄准更多主流应用。两款芯片均具有 65W TDP 和插槽，可直接插入 AMD 的插槽 AM4 平台。这些处理器基本上适用于任何基于 400 或 500 系列芯片组的主板，前提是制造商已使用必要的 AGESA 更新了主板的 BIOS。 POST/启动这些 APU 需要 AGESA 1.1.8.0，但建议使用 AGESA 1.2.0.3b（或更新版本）以获得最佳性能。锐龙7 5700G显然是这两款CPU的高端机型。它是一个 8 核/16 线程处理器，具有 4MB 的 L2 缓存和 16MB 的 L3。这些 CPU 内核与 8 CU Radeon Vega 图形引擎配合使用，并具有 24 条 PCIe Gen 3 连接通道。 5700G 的基本 CPU 时钟为 3.8GHz，最大升压时钟为 4.6GHz。处理器上的 GPU 可提升至 2GHz，与上一代 3000 系列 APU 的 1.4GHz 相比有了大幅提升。 Ryzen 5 5600G 具有 6 个 CPU 内核（12 个线程）和相应较小的 3MB L2 缓存，将性能降低了一个档次；但是，L3 缓存大小保持不变。 iGPU 也略有缩小，只有 7 个 CU。在 3.9GHz 时，5600G 的基本时钟比 5700G 高 100MHz，但其最大提升频率为 4.4GHz。 iGPU 的最大时钟速度也略低于 1.9GHz。 AMD 将这两款 APU 定位于 Intel Core i7-11700 和 Core i6-11600。 Ryzen 7 5700G 的建议价格为 359 美元，Ryzen 5 5600G 的建议价格为 259 美元，整整少了 100 美元。这两个价格都与英特尔的竞争产品完全一致。然而，从 Ryzen 7 5800X 和 Ryzen 5 5600X 的当前定价来看，这些新的 APU 的定价要低得多，考虑到它们的 CPU 内核数量相似，而且基本上是“免费”的 GPU，这很好。 Ryzen 7 5700G 和 Ryzen 5 5600G 可能是 AMD 的 Infinity Fabric。 Infinity Fabric 可以灵活地链接各种芯片上的 IP，结合 AMD 当前的 CPU 架构，基本上允许 AMD 将 Zen 3 放入设计中，作为上一代 Zen 2 架构的替代品。使用这些芯片，最多 8 核/16 线程 Zen 3 CCD 可连接到最多 8 个计算单元 Radeon Vega iGPU 和相邻的多媒体引擎。这些芯片是在台积电的 7nm 节点上生产的，处理器直接进入 AMD 现有的插槽 AM4 生态系统——同样，假设主板已经更新了必要的 AGESA（可能已经更新）。从外观上看，锐龙 7 5700G 和锐龙 5 5600G 与 AMD 目前所有的主流处理器没什么两样。包装、引脚、集成散热器和品牌都是一样的。只有印在顶部的型号发生了变化。启动最新版本的 CPU-Z 可以一瞥这些新的 Ryzen 5000G 系列处理器的配置。每个内核有 64KB（32KB 数据 + 32KB 指令）的 L1 缓存，每个内核有 512KB 的 L2 缓存，板载每个 CCD 有一个 16MB 的 L3 池。时钟和电压会根据负载而变化（稍后将在超频部分详细介绍），并且 5700G 和 5600G 之间的系列、型号、步进和版本是相同的。抛开所有这些，让我们看看这些最新的 APU 表现如何，好吗？
