The 8 Most Popular Java Experts

by anonymous on 2013-07-29 15:30:08

Translator: Tang Youhua

Below is a list of 8 influential Java personalities who have significantly impacted the Java community by creating frameworks, products, tools, or writing books that changed the way Java programming is done.

P.S. The rankings below are purely based on personal preference.

1. Tomcat & Ant Founder

James Duncan Davidson created the Java-based web server Tomcat while he was a software engineer at Sun Microsystems (1997–2001). Tomcat continues to be widely used in many Java web projects today. Additionally, he developed Ant, a build tool that uses XML to describe the build process and dependency management, which became the de facto standard for building Java web applications.

2. Test-Driven Development and JUnit Founder

Kent Beck pioneered Extreme Programming and the test-driven software development methodology. Moreover, he co-authored JUnit with Erich Gamma, a simple testing framework that became the de facto standard for testing Java web applications. The combination of JUnit and test-driven development transformed traditional Java programming practices, although many Java developers remain skeptical.

3. Java Collections Framework

Joshua Bloch led the design and implementation of numerous Java platform features, including the highly regarded Java Collections Framework and language improvements in JDK 5.0. In June 2004, he left Sun Microsystems to become Google's Chief Java Architect. His book "Effective Java" won the prestigious Jolt Award and remains a controversial yet essential read for Java programmers.

4. JBoss Founder

Marc Fleury founded the open-source Java application server JBoss in 2001. JBoss became a controversial but widely adopted standard for Java web applications. Later, he sold JBoss to Red Hat and joined the company to continue its development. On February 9, 2007, he decided to leave Red Hat to pursue personal interests such as teaching, researching biology, music, and spending time with his family.

Craig McClanahan created the popular MVC framework Struts. Controversially, every Java developer seems to know how to write Struts code. Due to its early success, nearly every early Java web application utilized Struts for implementation.

6. Spring Founder

Rod Johnson is the founder of the open-source Java application framework Spring. He is both the creator of Spring and the CEO of SpringSource. Furthermore, Rod's book *Expert One-on-One J2EE Design and Development* (2002) is one of the most influential J2EE books.

7. Hibernate Founder

Gavin King is the founder of Hibernate, a popular object/relational persistence solution for Java, and also the author of the Java EE 5 application framework Seam. Additionally, he made significant contributions to the design of EJB 3.0 and JPA.

8. Father of the Java Language

James Gosling invented Java in 1994. He completed the original design, compiler, and virtual machine for Java. Due to his contributions, he was elected to the National Academy of Engineering. On April 2, 2010, he left Sun Microsystems, which had recently been acquired by Oracle. Regarding his departure, Gosling wrote in his blog: "More precisely, my leaving was more meaningful than staying."
