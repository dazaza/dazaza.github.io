Before installation, you need to prepare:

Step 1 Prepare

Use a burning software such as Nero to burn a DVD image to a single layer DVD disc.

Format the USB flash drive with the disk mark set to "Patcher" (without quotes).

Extract the Zip file and put it in the USB flash drive.

Your USB flash drive should now contain a folder named "files".

Step 2 Install Leopard

Now that you have the cracked DVD in your hand, boot from the disc and start installing!

After booting from the CD, the installation wizard will load, it will take some time, please wait patiently.

Select a language, make sure you select "Customize" and deselect all displayed Packages.

OK, Leopard is ready to install, it will take a while, you can have a coffee break.

will ask you to restart at the end, so restart. Before restarting, make sure the USB flash drive is connected to your PC.

Step 3 Crack Leopard

Now that you've installed Leopard, it's time to crack (patch) it. Before cracking, we need to get it from Leopard  again; DVD boot.

Wait for the Darwin Bootloader to load. Once it's loaded, press F8. You will see a pop-up box. Type -s and press Enter. The DVD will be loaded in Verbose mode. Watch for any error messages. There shouldn't be any problem.

After loading, select the language. Then you will see the welcome screen. At this point, you go to a tool terminal.

After the terminal is loaded, type the following command: cd... cd .. cd Volumes cd Patcher cd files

Now it's time to crack. Type the following command on the terminal: ./9a581PostPatch.sh

The cracking program will run.

When the cracking program is allowed to finish, restart can be.
