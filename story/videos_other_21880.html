Freely customize the toolbar

by shooter on 2007-06-07 22:19:34

1. Relocate temporary files

Similar to Internet Explorer, Firefox Latest also has a temporary folder that defaults to the system drive, which can easily cause disk fragmentation on the system drive. In Firefox's address bar, type "about:config" to open its configuration page. Right-click in the page and select "New → String". Create a new preference called browser.cache.disk.parent_directory and set its value to the new folder where you want to store the files.

2. Specify default download folder

In the latest version of Firefox, downloaded files are saved by default on the desktop, which is not an ideal place for storing downloads. You can change the default file download save path by clicking on the menu "Tools → Options", selecting "Downloads", choosing "Save all files to this folder" in the right-side window under "Download Folder", then clicking "Other" in the drop-down list to open the "Browse Folders" window and select your desired folder.

3. Customize toolbar freely

When first running the latest version of Firefox, there are only a few button icons on the toolbar. Below, we will reveal the hidden buttons. Right-click on the toolbar and choose "Customize" to open the "Customize Toolbar" window. Drag the needed icons to the toolbar. Additionally, clicking "Add New Toolbar" can create a new toolbar.

4. Backup extensions

Extensions make Firefox more powerful. To prevent losing extensions during system reinstallation, it’s best to back them up. By default, Firefox installs extensions in C:\Documents and Settings\User Name\Application Data\Mozilla\Firefox\Profiles (for Windows XP). Back up the contents of this folder. When needed, simply copy them back into this folder to restore the extensions.

5. Freely specify scroll speed

Using the mouse wheel to scroll through web pages is very convenient, but if you're dissatisfied with Firefox's scroll speed, you can adjust it at will. Open the configuration page mentioned above, enter "mousewheel" in the filter, find the preferences mousewheel.withnokey.sysnumlines and mousewheel.withshiftkey.numlines, change the value of the former to False, and input the number of lines to scroll each time in the value of the latter.

6. No warning when closing tabs

When closing multiple tabs simultaneously, Firefox prompts with a "Confirm Close" message. We can disable this prompt by going to "Tools → Options", selecting "Advanced" on the left, and unchecking "Warn me when closing multiple tabs" on the right.

7. Number of homepages as you wish

In the latest version of Firefox, you can set multiple homepages instead of just one like in IE. First, open the multiple web pages you want to set as homepages in different Firefox tabs, click "Tools → Options", in the opened window, select "Basic Information" from the left-side list, click the "Use current multiple pages" button under "Homepage", and the addresses of the currently opened pages will appear in the text box. If you've already added the URLs you want to set as homepages to your bookmarks, you can also click the "Use Bookmarks" button, select the corresponding bookmarks in the "Set Homepage" window, and set the URLs in the bookmarks as your homepage.

8. Adjust webpage text size at will

Reading small text can be uncomfortable for the eyes. In the latest version of Firefox, the text on webpages can be enlarged or reduced at will. Press the "Ctrl++" key combination to enlarge and "Ctrl+-" to reduce the text size on webpages. Press "Ctrl+0" to reset the text to normal size.

9. Backup bookmarked webpages

All the great websites you've found over time are stored in your bookmarks, so it's important not to lose them. It's better to have a backup. In the "Bookmark Manager" mentioned above, click "File → Export Bookmarks" to save the collected URLs to a specified Html file. After reinstalling the system, simply click "File → Import Bookmarks" in the "Bookmark Manager" to import the backed-up bookmarks back into Firefox from the file. Here, you can also import the URLs you've collected in IE by selecting "Microsoft Internet Explorer" during import.

10. Search bookmarked webpages

Everyone's bookmark folder contains many URLs. If they're not well organized, it can be troublesome to find a specific URL when needed. In Firefox, you can search your bookmarks. Click on the menu "Bookmarks → Manage Bookmarks", enter the keyword you want to search for in the "Bookmark Manager" window that pops up, press Enter, and you can search for related website names and addresses in the bookmark folder.
