Several new engineering projectors all appeared in the application solutions, providing the media, audience, and merchants with a chance to get close to high-end application solutions. Interactive application solution Engineering projector, engineering application, high-end solution, these terms are familiar to many friends, but the specific implementation and their uses are quite unfamiliar, especially for non-projector industry professionals who are shrouded in mystery. Large conference engineering application In fact, the seemingly mysterious engineering application can be understood as a combination of hardware equipment including engineering projectors, display, fusion, etc., forming a method that can meet special needs such as film and television, large screen, 3D, interaction, command, monitoring, or called a solution. Large arc splicing Micro-projector art design Short focus education application We will simply analyze the seven mainstream application solutions demonstrated by Optoma for netizens. This article does not involve technology; in reality, these solutions can evolve into more detailed, professional, and special demand solutions. Page 2: First Category - Welcoming Passage Application The first category is the welcoming passage application. Features of the solution: 1. 270-degree panoramic image seamless stitching and fusion, stronger on-site feeling; 2. Support for real-time interactive technology; 3. Back-projection method, not affected by external environmental light. Application analysis: Wedding banquet venue entrance, underwater tunnel scene, cultural center, amusement park, shop window advertising, large conference welcoming scene, etc. Top back projection layout This set of solutions is at the entrance of the venue and is the first content to catch the eye when entering and leaving. Seven Optoma projectors EX785 + three high-definition hardware fusion systems M8 + four body sensing devices jointly build a relatively complex back-projection system. These four cameras may look familiar, and upon closer inspection, they turn out to be Microsoft Xbox's Kinect body sensing device. Long-distance camera view Camera close-up This set of system patterns can be made very dazzling and transform as the audience moves, making the venue entrance full of mysterious colors. It is hoped that similar projection solutions can be seen in Disneyland, Carnival, and other amusement parks this year. Page 3: Second Category - Curved Projection Application The second category is the curved projection application. Solution features: 1. Can achieve ultra-wide, super-large images; 2. Seamless stitching; 3. When used together, can achieve special dynamic effects. Application analysis: Museums, aquariums, large conference centers, concerts, meteorological bureaus, and various large indoor and outdoor displays. Front view effect Although the scale of this demonstration meeting is not large, watching a 60-square-meter image up close is still quite breathtaking. Side view effect This system uses a total of six Optoma projectors EX785 + twelve fusion processors M8, plus that super-large 60-square-meter screen. It is reported that during Optoma's exhibition preparation period, this application solution really broadened the horizons of a director, and it is estimated that it will gradually become popular in movie shooting in the future. Suspended EX785 This type of extra-large irregular large curved projection system has very wide practical applications and many classic cases. Large water curtain movies, building projections, curved panoramic systems, etc., are all extended applications of this type of solution. Cambridge University's 800th anniversary architectural projection If we say that in some places, LCD splicing, plasma splicing, LED splicing can rival projection splicing, but in this type of extra-large, extra-long case, projection splicing technology stands alone. Page 4: Third Category - Simulation and Emulation Application The third category is the simulation and emulation application. Solution features: 1. Wide field of vision; 2. More realistic simulation and emulation effects. Application analysis: Ship systems, military simulation, driving simulation, amusement parks, large PC games, etc. The first impression of this application solution for viewers is "a very advanced game console." Distant view effect Three Optoma projectors EX785 combined with simulation game operation equipment greatly increase the playability of this application solution. Suspended projector Overall effect On the surface, it seems like a super-luxury gaming device, but it is precisely the best platform for future simulation operations; aviation, military, driving schools, and other specialized teaching and tutoring fields will be the main application objects. Page 5: Fourth Category - High Definition 3D Cinema Application The fourth category is the high definition 3D cinema application. Solution features: 1. Large-sized cinema; 2. Realize multiple 3D technologies. Application analysis: 3D cinemas, large meeting rooms, museums, zoos, science museums, meteorological bureaus, etc. Watching movies is everyone's common hobby, and watching 3D movies is also everyone's favorite. On-site actual shooting effect On-site layout Three Optoma EX785 + two 3D fusion processors M9 form this luxurious 3D cinema system, using shutter-based active 3D technology. If the number of projectors doubles, polarized 3D technology can also be supported. Suspended projector This cinema system is somewhat luxurious for home users and is more suitable for large venues like cinemas. Page 6: Fifth Category - Dynamic Sandbox Simulation Application The fifth category is the dynamic sandbox simulation application. Case features: 1. Supports full HD presentation; 2. Dynamic presentation; 3. More convenient and intuitive operation. Application analysis: Building presentations, military command, large-scale engineering demonstrations, and other dynamic sandbox applications. For those unfamiliar with Optoma's past classic projection products, this sandbox demonstration system is currently demonstrating to the audience. Behind the horizontal screen are nine real machines, and one important hardware component of this system, EH7500, is among them, Guangzhou Panyu game machine manufacturer. Dynamic sandbox simulation system Side view effect Front view effect One EH7500 + one EX850 jointly form this system, and the content displayed this time is Optoma's classic projection products over the past decade. Top projector This dynamic sandbox simulation system has a relatively narrow application range and has already been adopted by some countries' military forces. Page 7: Sixth Category - Large Screen Stacking Application The sixth category is the large screen stacking application. Solution features: 1. Super large image; 2. Full HD image; 3. High brightness effectively overcoming environmental light influence; 4. Single-machine dual-lamp, ensuring system stability. Application analysis: Military exercises, satellite cloud maps, meeting presentations, security command, traffic monitoring, etc. The main venue of this Optoma demonstration meeting was this 200-inch screen. This application solution uses a floor-standing screen, and through comparison with on-site personnel, one can feel the visual impact of the large screen. Main venue display effect Projector installation effect Two EH7500 + two high-definition fusion processors, the imaging images of the two projectors are completely stacked, forming this giant screen system, which can be said to have the widest application range, relatively simple operation, and key investment cost lower than single-machine having strong market competitiveness. Suspended projector Interaction application with iPad2 This application solution has high scalability, cooperating with more interactive systems, it can easily realize interaction application with iPad2; cooperating with multiple cameras arranged within the venue, the display supports picture-in-picture function. Picture-in-picture monitoring system For industries requiring high precision such as satellite cloud maps, security command, monitoring systems, etc., the practical value of this large-screen HD solution is very high. Page 8: Seventh Category - Spherical Screen Application Solution features: 1. Through the combination of high-brightness projector + fish-eye lens + special housing, it realizes a 360-degree panoramic viewing with a single projector; 2. No additional fusion technology required; 3. Effectively overcome the influence of external environmental light. Application analysis: Science museum, school teaching, advertising, 360-degree panoramic viewing of large events, etc. This time, Optoma demonstrated two sets of spherical multimedia projection systems to the media, all using internal projection. Although compared with other application solutions, the display area is the smallest, in the venue, it appears particularly prominent. 0.8-meter spherical screen system A 0.8-meter diameter spherical screen adopts one projector EX850 + one fish-eye lens. 1.5-meter spherical screen system A 1.5-meter diameter spherical screen adopts one projector EH7500 + one fish-eye lens. Beijing Olympics The grand globe at the opening ceremony of the 2008 Beijing Olympics was a large spherical screen projection display system, with very shocking effects, instantly capturing the eyes of friends around the world. For teaching purposes, the use value of this internal projection spherical screen system is higher, and interactive content can also be added. Summary: There are many engineering solutions, these solutions are the accumulation of hardware, the choice of matching methods, and more importantly, the crystallization of the designer's wisdom. Perhaps the charm of engineering projection lies here.
