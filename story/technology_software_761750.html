NY Times: Windows 8 Disappoints PC Makers

by geekzhang on 2013-05-08 22:59:47

According to a report by the New York Times on May 8 Beijing time, Microsoft has made a sweeping reform of its Windows 8 software system. Now, Microsoft admits that these changes are a bit too radical for some users. The company will release an updated version of Windows called "Windows Blue" at the end of the year to help users better adapt to the new software.The report said that Microsoft announced on Wednesday that since its release six months ago, sales of the new flagship software system have exceeded 100 million copies. Sales performance in the same period is on par with the previous Windows 7.Despite strong sales figures, some users have voiced complaints about the new system. In the Windows 8 system, Microsoft replaced the traditional interface with full-screen themed squares, which frustrated many users. Although users can switch between the new and old versions, many people don't know how to do it.In an interview, Tami Reller, the Chief Financial Officer and Marketing Director of Microsoft's Windows Division, said that an updated version of the system, codenamed "Windows Blue," would be released at the end of the year. It includes some modifications to make user operations more convenient, especially aiming to leverage the advantages of the new system on computers without touchscreens.Another issue with the new system is that the square-themed appearance was designed specifically for touchscreen device users, but not many such devices actually run Windows 8. Market research firm IDC estimates that Microsoft sold only 900,000 Surface tablets in the first fiscal quarter this year, accounting for 1.8% of the total market share. Other Windows tablet manufacturers, like Acer, contributed additional sales. In contrast, Apple's iPad sold 19.5 million units, capturing 39.6% of the market share.Dell and HP, PC manufacturers, were disappointed that Windows 8 did not help them fend off competition from iPads. IDC reported that global PC shipments dropped by 13.9% in the first fiscal quarter compared to last year, reducing to 76.3 million units; this is the worst performance in nearly 20 years. Meanwhile, tablet shipments surged by 142.4%, reaching 49.2 million units.The original intention of Windows 8 was to bridge the gap between tablets and traditional personal computers using software. Its software is suitable for touchscreens and can switch to a traditional interface when users need to create slideshows or Excel spreadsheets, making it easy to use keyboards and mice. Microsoft also anticipated a surge in Windows 8 touch devices, including laptops with touchscreens.But none of this happened, as most PCs in stores are still old-style keyboard machines. If you don't provide users with the right environment to help them understand how to use the new system, you may lose your loyal users, said an IDC analyst.Reller stated that Microsoft would reveal its update plans for the Windows 8 system in the coming weeks. However, she refused to disclose whether the current tile-based start interface would be abandoned. Reller added that Microsoft has trained retail partners to remind consumers that the old interface is still available within the Windows 8 system. We discuss the desktop as an application, she said, but in reality, the desktop is very important for PC users.Microsoft conducted its own surveys on Windows 8 usage habits. The survey found that if users are using tablets or other touch devices, their satisfaction level is on par with Windows 7. But if users are using traditional PCs, the results aren't as positive. We need to help them quickly learn the new system, she said.Joshua Blood, an audio engineer from Houston, uninstalled Windows 8 from his computer just a few days after installing it. He believes that Windows 8 only makes sense on touch devices. I can do everything on Windows 7, and the interface is also beautiful, Blood said.The sale of 100 million copies of the Windows 8 system sounds impressive, but the numbers do not indicate how many actual users there are. This is because most of Microsoft's Windows systems are sold through long-term cooperation contracts with enterprises, and enterprise customers have the right to choose which system their computers use.In fact, many enterprise users have downgraded to the Windows 7 system. An IDC analyst also indicated that 40% of Microsoft's customers have the right to downgrade. Typically, enterprise users are cautious when accepting new software, especially when a new version of Windows is released. They do not easily change their usage habits.
