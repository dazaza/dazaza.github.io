Security for the "cloud-based" IDC virtual hosting rental business needs to be strengthened.

by xwkj0983 on 2012-02-21 12:01:13

In the view of Tan Yi, the chief technology expert of the telecommunications industry, cloud computing brings opportunities for large enterprises, including telecom operators themselves, in terms of resource integration and utilization as well as new business expansion. Large enterprises have their own enterprise IT support systems and also provide data services externally, involving IT support clouds, business clouds, IDC clouds, etc., all of which have corresponding cloud security needs. At the same time, from the three-layer cloud application model of IaaS, PaaS, SaaS, virtualization security, data transmission security, sensitive data storage security, interface security, security auditing, identity recognition and management, application security, system disaster recovery, etc., and cloud platform security management all have corresponding needs.

Currently, domestic operators are actively promoting IDC virtual hosting leasing services. Compared with traditional IDC hosting leasing services, the security protection boundary has been virtualized, so it faces different security threats such as virtualization security, traffic analysis management, and the continuity of cloud service SLA.

In response to this, Hillstone has proposed technical solutions in China Telecom's "Seed Cloud" and "Cloud IDC" by combining the characteristics of its own products. For example, the SG-6000-X6150 data center firewall launched by Hillstone provides the following capabilities for cloud computing data centers: up to 100Gbit/s, 1 million new connections, and 50 million concurrent secure protection capabilities; performance elastic expansion, by increasing the number of security service modules (SSM), gradually improve the firewall performance from 20Gbit/s to 100Gbit/s, and the expansion process is plug-and-play without manual configuration; comprehensive reliability assurance, providing full redundancy capability from CPU, security service module (SSM), system control module (SCM) to fans and power supply, and achieving system-level session synchronization to meet the continuity requirements of data center services to the greatest extent; providing independent security service planes for different businesses or tenants, supporting up to 250 virtual firewalls; multi-business isolation, using a side-hanging deployment mode to ensure secure and controllable mutual visits for multiple businesses on the virtual switch layer; visualization, based on deep application identification technology, traffic logs, and intelligent security log analysis, providing strong guarantees for data center security operations, etc.

Currently, Hillstone actively participates in multiple cloud construction projects of operators and provides some cloud security solutions, such as cloud transmission security solutions, cloud data center virtual network layer security protection solutions. Successful cases include the "Seed Cloud" node construction project of China Telecom, which is mainly used for the use of telecom business platforms and serves as the cloud resource pool of telecom business platforms.
