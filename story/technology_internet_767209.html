Microsoft releases a new wave of enterprise cloud solutions to perfect its cloud strategy

by anonymous on 2013-10-10 18:39:41

Microsoft has announced the launch of a new wave of enterprise products and services. These new offerings align with Microsoft's enterprise cloud strategy and serve as complements to Office 365 and other services.

To assist customers in building IT infrastructures that provide continuous service and applications, Microsoft will release Windows Server 2012 R2 and System Center 2012 R2 on October 18th.

Microsoft stated that customers can use Visual Studio 2013 and the new .NET 4.5.1 on these hybrid cloud platforms to develop modern applications for various devices and services.

Additionally, starting November 1st, Microsoft will offer Enterprise Agreement (EA) customers the opportunity to purchase Windows Azure cloud services at discounted rates, regardless of whether they previously purchased the service or faced penalties for overuse. Payment options will also be flexible.

To lower the barrier for adopting cloud services, Microsoft also announced a strategic partnership with Equinix. Prior to this, Microsoft had announced partnerships with AT&T and other companies. The collaboration with Equinix will provide additional options for customers seeking private cloud solutions. Customers can connect their networks to Windows Azure at Equinix exchange sites to achieve greater throughput, uptime, and security.

To help U.S. federal, state, and local governments realize the benefits of public cloud computing services, Microsoft will introduce its Windows Azure US Government Cloud service tailored specifically for government entities. This service will provide a dedicated community cloud for U.S. government departments to handle data, applications, and infrastructure.

Next week, Microsoft will release the second preview version of SQL Server 2014. This new version will include in-memory technology but will be priced the same as previous versions. As a result, customers can boost performance by 10 to 30 times without rewriting applications or purchasing new hardware. SQL Server 2014 is also compatible with Windows Azure, helping users build cloud backup and disaster recovery solutions.

Regarding big data analytics, Microsoft will release the Windows Azure HDInsight Service later this month. This service, based on Apache Hadoop, is compatible with SQL Server and various commonly used business intelligence tools such as Excel and Power BI for Office 365. Users can combine private and public data using Power BI for visualization and global analysis.

Furthermore, Microsoft will release a new version of Windows Intune on October 18th, which works alongside System Center Configuration Manager. This helps IT departments provide secure access to applications and necessary data for mobile employees, enhancing the data protection capabilities of Windows Server 2012 R2.

In addition, Microsoft will introduce an application called Microsoft Remote Desktop within Windows Server 2012 R2, available for download later this month. This allows customers to easily access PCs and virtual desktops from any device or platform.

Lastly, Microsoft will release the next major version of its CRM solution, Microsoft Dynamics CRM Online Fall 13, later this month. This update will enable more personalized interactions through contextual information. An on-premises version of the CRM solution is expected to be released later in the fall.
