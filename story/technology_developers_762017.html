Eclipse 4.3 正式版发布（Kepler）

by @geekzhang on 2013-06-28 11:47:39

代号为 Kepler 的 Eclipse 4.3 正式版的下载已经出现在 Eclipse 官方的首页上。该版本的亮点是：

Eclipse Web Tools Project (WTP) 3.5 支持开发 Java EE 7 应用

Stardust 1.0 提供完整的商业过程管理 BPM 套件工具和运行环境

Orion 3.0 包含显著的可用性提升和更轻松的发布到 Java 应用服务器

Business Intelligence and Reporting Tools (BIRT) 支持大数据报表，支持 MongoDB 和 Cassandra

Mylyn 通过导航视图和 Gerrit 集成更好的支持代码评审

Maven Integration for WTP 简化在 Eclipse 中使用 Maven 支持 Java EE 项目

(via.http://www.oschina.net/news/41781/eclipse-4-3-final)
