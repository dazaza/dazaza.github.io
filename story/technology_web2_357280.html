File Destructor：在线伪造“被损坏”文件 | ZFreet . CHeung

by zfreet on 2010-06-01 10:51:42

File Destructor 2.0 这个网站的功能很简单直接：可在线的人为伪造一个“被损坏”的文件，支持自定义文件大小和22种文件后缀。

现在的2.0 版本有以下特点：

More file extensions added 支持22种后缀

Unlimited file size 不限文件大小

100% Mac and PC compatible 全面支持PC和Mac

Improved destruction and saving time 提升了生成和下载速度

No plug-in needed 无需插件

No bundled software or virus of any kind 无捆绑和病毒

Free to use 免费

对于这个功能，我能想到的用途就是，当领导向你要东西，或老师要你交作业的时候，可以拿个无法打开的文件暂时拖延一段时间来救急。当然理由有千万种，就看自己怎么忽悠了；这种小伎俩对“张总”那样的BOSS 最为有效。

另外，伪造的文件大小和下载时间是成正比的。如果你想造一个G级的大家伙，还不如直接拿个视频文件改后缀，只是文件大小不好精确控制。
