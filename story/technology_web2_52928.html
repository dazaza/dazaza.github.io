西行资讯: Windows Live FolderShare Beta 更新了

by legendnic on 2008-03-12 09:45:40

入口：https://www.foldershare.com/相关服务：- Windows Live FolderShare：通过浏览器访问文件，与Messenger共享文件夹类似的同步功能。- Windows Live Folders：真正的存储服务，可以讲文件存储在微软数据中心，可以离线访问文件。- Windows Home Server：是家庭服务器操作系统，可以作为家庭存储器，也可以远程通过浏览器访问其中的文件。
