Microsoft Launches Companion Network Program, Integrating IE with Mobile Experience

by anonymous on 2013-07-24 17:07:23

Beijing Time, July 18th news, Microsoft published a blog post on Wednesday morning stating that the company has initiated a new program called "Companion Web" which will assist developers in creating web applications capable of running on displays of any size.

In other words, Microsoft hopes this set of developer tools will allow users to obtain a network experience that integrates IE with mobile devices. In fact, Microsoft had conceived such an idea long ago. Last year, Microsoft announced the SmartGlass project, which can unify the content on Xbox 360 and any smartphone or tablet installed with the SmartGlass app.

Microsoft plans to install SmartGlass on future Xbox One gaming consoles, with more games and applications supporting second-screen experiences.

Bryan Saftler, Senior Product Director of Microsoft IE, stated: "We are doing this because no matter what device you use, this network is omnipresent. We know that 86% of smartphone users use their smartphones while watching TV, so why don't we create some companion web apps to enhance those experiences?"

To demonstrate what excellent companion web experiences look like, Microsoft has collaborated with polling startup Polar to help it develop an application that works between connected TVs or PCs and smartphones or tablets.

Users will be able to use their smartphones to create surveys and participate in them, with the entire survey feedback and other questions displayed on the main screen. Users can search for content related to the programs they are watching on their phones, while seeing the search results pop up next to the program. Polar created a video to show the entire process.

Luke Wroblewski, founder of Polar, said he founded Polar to address a fundamental issue he observed in surveys, namely that people dislike providing feedback information, especially to various companies.

Wroblewski stated: "Think about the last time you participated in a survey, you hated lengthy surveys, numerous buttons, seemingly endless pages and questions, we want to turn surveys into something people enjoy doing and an entertaining experience."

Therefore, he decided to combine surveys with television. Wroblewski mentioned that he could see extensive usage scenarios for live TV programs.

It should be noted that surveys never interfere with the TV experience, allowing you to focus on the program. Now, the application mainly focuses on short links or simple searches to discover content related to what the user is watching.

Microsoft stated in the blog post that the Polar app is just the first step, and in the future, it will release more new companion web experiences.
