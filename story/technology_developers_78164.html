十大变化！Windows 7比Vista多什么

by @maker on 2008-07-18 00:20:19

【IT168 软件评测】作为微软最新一代操作系统，Windows 7在很多朋友眼中，一直显得十分神秘。然而就在前两天，如今我们终于可以近距离与Windows 7 M1接触，并揭开Windows 7 的神秘面纱。那么，和Vista相比，这款Windows 7 M1到底会为我们带来哪些不一样的体验呢?下面，就让我们一同来看一看，两款操作系统之间的异同吧。

　　变化一. 启动画面的变化

图1 Vista启动画面

Windows 7 M1启动界面

　　相比Vista那黑乎乎的启动窗口，Windows 7的启动显然“漂亮”了很多。除了将梦幻般的极光作为背景之外，它还特意将Win9X时代的滚动条引入其中。而且，当这些滚动条“碰”到极光之后，还会“迸发”出漂亮的星芒效果，煞是好看。

　变化二. 窗口最大化的变化

Vista窗口最大化效果

Windows 7 M1窗口最大化效果

　　为了防止漂亮的半透明特效，分散用户的注意力。Vista每次都会将最大化后的窗口边框，调成黑色。不过，这样的设计，同时也导致了其与系统整体风格，不相匹配。而在新版Windows 7 M1中，最大化窗口依然保持了漂亮的半透明效果，只不过透明度被大大降低。这样，不仅整个系统在风格上更为统一，而且同样可以达到令用户专心工作的目的了。

　　变化三. 窗口搜索框的变化

Vista的窗口搜索栏

 

Windows 7 M1的窗口搜索栏

　　在Vista中，窗口搜索栏的长度是固定的。如果待搜索关键字较长，就无法将其全部显示到搜索栏中。而在新版Windows 7 M1中，我们不仅可以利用鼠标，直接调整搜索栏的大小。而且，当用鼠标点击搜索栏与地址栏之间的分隔线时，搜索框还会自动放大，大大方便了用户日常的搜索操作。

变化四. 侧边栏的变化

Vista的侧边栏 Windows 7 M1的侧边栏

　　和Vista一样，Windows 7 M1同样为我们预设了实用的侧边栏(Sidebar)功能。不过，为了美观起见，新版侧边栏，取消了老版本中的侧边框线，使之与系统更加协调。同时，Widget的添加，也不再局限于Sidebar的范围之内。如今，您只要在桌面的任意位置右击鼠标，就可以通过菜单，直接找到Widget的添加选项了。

　变化五. WMP11的变化

Vista版Windows Media Player

Windows 7 M1版Windows Media Player 11

　　虽然，Windows 7 M1并没有在系统中，加入WMP 12之类的全新播放软件。但新版WMP 11仍然在老版本的基础上，有了不小改进。除了新增加的“Player”模式之外，新版WMP 11还在界面右下方，加入了“视图选择”和“播放模式选择”两个快捷按钮，大大方便了我们的日常播放操作。

变化六. 计算器的变化

Vista版计算器

Windows 7 M1版计算器

新版计算器的“Unit Conversion(单位换算)”模块

　　在经历了N个版本之后，经典的计算器工具，终于有了更新。除了新增加的一个中间结果显示区之外，新版计算器还特意为我们提供了“History”(历史保存区)和“Unit Conversion”(单位换算模块)两项新功能，从而大大方便了用户的日常计算操作。

变化七. 系统还原区的变化

Vista的系统还原设置

Windows 7 M1的系统还原设置

Windows 7 M1的系统还原详细设置

　　在Vista中，系统还原一直都是一项非常实用的功能，可由于没有详细的设置选项。很多朋友在不满意系统的默认设置之后，往往只能将其一关了之。而在最新版Windows 7 M1中，我们终于见到了盼望已久的系统还原设置选项。点击之后，除了可以修改系统还原的默认保护范围之外，还可以自由调整还原所占用的磁盘空间，大大提高了该项功能的实际可用性。

变化八. 开始菜单的变化

Vista开始菜单

Windows 7 M1开始菜单

Vista开始菜单快捷搜索

Windows 7 M1开始菜单快捷搜索

　　开始菜单，是Windows 7 M1中变化较大的一个组件。首先，当我们将鼠标悬停于某个菜单项时，菜单右端会立刻显示出一个“置顶”按钮。点击之后，该程序便会被马上放置到菜单顶端。此外，菜单搜索界面也较之以前，有了很大改进。当我们使用新版菜单，进行快捷搜索时。结果会自动覆盖到整个菜单面板，大大增加了结果显示空间。

　变化九. 资源监视器的变化

Vista版资源监视器

Windows 7 M1版资源监视器

　　和开始菜单一样，资源监视器也是Windows 7 M1中，变化较大的一个组件。虽然，新版监视器，并没有在功能方面有所提升。但重新设计的界面布局，仍然显得十分实用。更为重要的是，由于“CPU”和“Memory”，这两项最常用的指标，都已经以标签形式单独列出。因此，操作系统便能以更大的屏幕空间，详尽展示这些参数了。

变化十. 控制面板的变化

Vista 版控制面板

Windows 7 M1版控制面板

　　为了节省宝贵的屏幕空间，新版Windows 7 M1对所有系统窗口，全都进行了一番改进。以往常驻于窗口左侧的“常见任务”区，已经被设置为自动隐藏模式。凡是没有包含常见任务的窗口，都会自动隐藏该区域，以便腾出更多的屏幕空间。只有当本窗口中包含相关任务时，“常见任务”才会自动弹出，并提供给用户相关选项。

变化十一. 视图选项的变化

Vista系统视图选项

Windows 7 M1系统视图选项

　　除了上述这些大的变化之外，新版Windows 7 M1还新增了一项，名为“Smart Details”(智能细节)的视图选项。不过，也许是新版本的一个Bug，或者是一个尚未完成的功能。笔者并没有在试用过程中，发现它与传统的视图模式，有何不同。因此，关于这项更新，目前就只能是一谜了。

　　写在最后

　　看到这里，相信您已经对新版Windows 7 M1，有了一个大体了解。其实，除了上面这些大的改进之外，新版Windows 7 M1还新增了PowerShell(工作于服务器操作系统上的命令提示符)和XPSView(XPS文件查看器)等一些实用功能。不过，正所谓百闻不如一见。如果您真的对这款Windows 7 M1感兴趣，还是自己亲手来尝试一番吧。
