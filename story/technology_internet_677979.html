Tips for running HTC Sense mobile applications

by dream2012 on 2012-02-27 11:02:38

As the currently most successful Android phone manufacturer, the HTC Sense user interface developed by HTC for Android phones has become a major feature of HTC's Android phones. After the pre-installed Sense user interface on HTC Hero phones initially released in Europe became highly popular, all subsequent HTC Android phones were equipped with the HTC Sense user interface, with the only exception being the Nexus One phone manufactured for Google. Despite the large number of Droid and Galaxy series phone users now, it is certain that domestically and globally, the number of Android phone users utilizing HTC Sense interface is definitely the highest. For HTC Sense users, there are some functions and tips you should be aware of. Some of these tips you may already be very familiar with, while others are rarely known. Here we introduce a few common HTC Sense operation tricks to help you better use your phone. Please note that some of these also apply to other Android phones.

Entering *#*#4636#*#* in the dialing interface will bring up a hidden window not usually seen. In this window, you can not only view phone information but also detailed battery information and battery usage history records. You can use this information to check the status and usage of the battery, easily identifying the programs that consume the most power. It also provides a function to view the usage of all programs, helping you determine which application you use the most. As an HTC Sense user, this feature will definitely surprise you, so check it out quickly. Long pressing the notification bar at the top of the phone will display the system date in the upper left corner of the screen. This function seems to be available on non-HTC Sense phones as well.

When using the screen keyboard to input text, a predictive input bar will appear suggesting words or phrases you might want to input. However, sometimes you may find the position of this bar inconvenient. You can long press this bar and drag it to any position on the screen. When your phone receives a call at an inconvenient time, you can repeatedly press the volume down button to reduce the volume or enter silent mode. You can also choose to flip the phone over, and the ringtone will automatically stop, which is particularly useful during meetings. When your phone is in silent mode, you can press the volume up button to turn on the ringtone. When your phone is in vibration mode, pressing the volume up button once will enter silent mode.

To search for a song in a long music list or look for a contact in the address book, you can slide the screen and hold the gray scrollbar that appears on the right side to directly jump to entries starting with specific letters. This function is mainly used for English content, and if your music playlist contains many English songs, this function is quite useful when looking for songs. The fastest way to end a sentence when using the phone's screen keyboard to input text is to double-tap the Spacebar. The keyboard will automatically stop inputting, add a space after the previously entered content, and then you can start entering the next word. If you find it difficult to accurately position and modify a part of the previously entered text with your finger, you can use the trackball or optical navigation key to precisely position to the place that needs modification. The trackball or optical navigation key of the phone can also be used to switch between several screens of the phone, sliding left or right.

When a particular item is selected, long pressing the phone's trackball or optical navigation key has the same effect as long pressing the screen. When in the all applications interface of the phone, you can press the menu key, then choose grid or list to switch between two program display modes, making it easier for you to find relevant programs. If your HTC Sense runs on the Android 2.2 system, pressing the menu key can also use HTC's newly introduced application sharing tool. In SMS and other applications supporting phone contacts, pressing the contact avatar will bring up a shortcut menu bar supporting operations such as making a call, opening the address book, sending an email, sending a text message, chatting via Google Talk, and finding the contact's address on the map. Depending on what information is included in the contacts on your phone, the options in the popped-up shortcut menu bar will vary. This shortcut menu also supports many third-party applications, provided you add related information when editing contacts.

In the mobile browser, long pressing the back key can bring up a list of recently visited web pages' history records. To rename a folder on the phone screen, click to open the folder, then long press the folder name field, and a dialog box to modify the folder name will pop up; make modifications and save. To forward a short message, just long press the message, select forward in the popped-up dialog box, then choose the contact to send. Many mobile applications providing search functions hide an Easter egg. Long pressing the search button will bring up related operation options.
