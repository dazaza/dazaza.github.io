Let's discuss running IE in Firefox.

by shooter on 2007-06-01 20:07:10

Some sites only display properly in IE, including Microsoft's Windows Update site. At this time, you can use the free IE Tab extension to run IE in Firefox. When you view a page that requires IE to display properly, this extension will enable Microsoft's rendering engine, allowing you to browse the page without any hindrance. If IE Tab does not work at this moment, you can right-click on the page and select "View Page" in IE Tab. You can even set certain websites to open using IE in Firefox: choose "Tools > IE Tab > Options", click on the "Site Filter" tab, enter the URL address, then click "Add" and "OK".

1. Change your "Throbber" page: In the Firefox browser, there is a small circular icon in the top-right corner that spins and flashes when your browser is working. This icon is called a "throbber." When you click it, it takes you to the "Firefox Start" page. But you can also associate other pages with the "throbber." Enter "about:config" in the address bar and press Enter. Input "browser.throbber.url" in the "Filter," then double-click the entry, and input the URL you want to associate in the "Enter String Value" area and press Enter. From now on, the "throbber" will take you to the site you have set.

2. Restart Firefox in Safe Mode: Sometimes, Firefox downloads may cause the entire browser to freeze due to an extension or theme. Like Windows XP, Firefox has a safe mode that loads a "no-add-ons" version of the program, without any extensions or custom settings. First, completely close Firefox. Then click "Start > Run," input "firefox.exe -safe-mode," and press Enter. The resulting dialog box allows you to reset toolbars, controls, and user customizations. After completing the settings, click "Continue" in safe mode.

3. Use Google search in the address bar: Firefox has a search box to the right of its address bar for Google searches. However, if you still need to move your mouse to this search box, it might be a bit cumbersome. You can also directly input in the address bar to conduct a Google search. First, select "Bookmarks > Manage Bookmarks," then click the "Bookmark" button to create a new bookmark. Input "Google Search" in the "Name," input "http://google.com/search?q=%s" in the "Address," input "g" in the "Custom Keyword," and then press Enter. Now, you can use the address bar for Google searches by simply typing "g," pressing the spacebar, entering the item you wish to search, and pressing Enter to see the Google search results.

4. Recover from unresponsive Firefox: When the Firefox browser becomes unresponsive, you must restart Firefox and reopen the tabs of the web pages you were browsing, which is indeed troublesome. A free extension called "Tab Mix Plus" provides many useful tab-related settings. After installing this extension, when Firefox becomes unresponsive, you just need to restart the browser and select "Tools > Tab Mix Plus Options," then check "Enable tab recovery function" under "Events > Tab Functions." The Session Manager feature of this program can even remember all the tabs you previously closed, so if you accidentally close a tab's page, you can quickly reopen it.
