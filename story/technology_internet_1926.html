The first blog in China to support web standards.

by xavier on 2006-04-08 13:33:44

After a period of development, the 1.0 version of 5DBlog is about to be released. Here's a preview of the main features in version 1.0:

1. Pages and blog posts will be generated as static web pages, which is designed for future multi-user development.

2. Custom templates can be defined for individual categories, allowing users who like to create personal homepages to make their blogs more colorful.

3. Template backup functionality will greatly benefit users who enjoy DIY customization.

4. UTF-8 encoding will be adopted to align with international standards, solving Wallop's RSS issues.

5. Trackback functionality has been added, connecting bloggers more effectively.

6. Templates adhere to XHTML standards and use DIV-based layouts, separating style and data to improve system performance. This will be discussed in detail later.

7. Interfaces can be created with various user data systems, meaning that 5DBlog will be available for other specialized websites.

8. Some content utilizes AJAX technology, offering a unique user experience.

9. A VIP purchasing function has been added, along with the ability to transfer blog coins.

10. Image blogging has been enhanced, differing from other systems where all uploaded images are treated as image blogs. This feature uses thumbnails for listings, essentially functioning as an article blog with an image list.

An important upgrade is that the new system will increase free user storage space from 1MB to 5MB. We hope everyone continues to support the development of 5DBlog.
