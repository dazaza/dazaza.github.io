Microsoft is reportedly planning to merge the WP and Windows app stores, and has already issued an internal announcement.

by anonymous on 2013-10-02 10:46:48

Milson previously publicly stated that he envisioned a future for the Windows RT system running on phones and tablets.

The Verge learned from an insider source that Microsoft is preparing to merge the Windows Phone app store with the Windows app store. Terry Myerson, head of the operating systems division, has already announced this news at a conference attended by over a thousand employees within the company today.

Sources revealed that Myerson did not explain how a unified app store would work, nor did he clarify whether WP apps would be compatible with the new Windows system. However, Myerson had previously publicly expressed his vision of seeing the Windows RT system run on phones and tablets in the future.

"Windows RT tablets were our first ARM-based tablets. As the boundaries between phones and tablets become increasingly blurred, we expect to see more ARM-based tablets (and phones) emerge in the future," Myerson recently mentioned during a financial analysts meeting.

Regarding the operation of a unified Microsoft app store, The Verge speculated that Microsoft might take inspiration from Apple's App Store. The operational model of Apple’s App Store is such that tablet apps cannot run on phones, but phone apps can run on tablets. Microsoft has made commitments more than once to launch a "shared app platform," but it remains unclear if this "shared app platform" will come to fruition in the era of Windows 8.1.
