Ajiu AspWebServer (Version 1.6) Server

by pyj2004 on 2007-06-23 21:35:42

As the name suggests, this software is a small and lightweight Asp Web server. It provides nearly perfect support for ASP.

If you are tired of IIS's bulkiness, if you don't want to reinstall IIS for the Nth time, if you are interested in alternatives, please give Ajiu AspWebServer a try...

Usage Instructions:

'1' Place your webpages in the same directory as this software and double-click to run!

'2' Double-click the taskbar icon at the bottom right corner to start your website!

'3' Default homepage: default.asp; default.html; index.asp, etc.
