Google Algorithm in September 2008

by perdie on 2008-09-30 15:31:26

Note: This article is reprinted from http://www.yueseo.com/SEO/10.html. The authenticity of this article will not be discussed in this blog.

Some parts of the content can be trusted, but others seem off. At least, some parts don't translate smoothly, though it's still possible to understand them勉强. Whether this article was personally written or released internally by Google remains unknown.

For reference only. Don't over-pursue SEO!

I. Website Quality - 39.9 Points

1. Popularity and breadth of external links for the website. - 4.4 Points

2. Domain age (calculated from when indexed by search engines). - 4.1 Points

3. Relevance between the content of external link pages and keywords. - 3.9 Points

4. Link popularity of the site within a community of topic-related websites. - 3.9 Points

5. Rate at which new external links are generated. - 3.5 Points

6. Number of pages indexed. - 3.2 Points

7. Relevance between user query keywords and the website theme. - 3.1 Points

8. User behavior. - 2.8 Points

9. Artificial weight granted by Google. - 2.6 Points

10. Special nature of the domain (.edu, .gov, etc.). - 2.6 Points

11. Rate at which new pages are generated. - 2.5 Points

12. Frequency with which users search for the site. - 2 Points

13. Whether the site has been confirmed via Google Webmaster Central. - 1.3 Points

II. External Links - 31.5 Points

1. Anchor text of external links. - 4.4 Points

2. Link popularity of the page where the external link resides. - 3.6 Points

3. Topic relevance of the external link page. - 3.5 Points

4. Link popularity of the external link page within related-topic website communities. - 3.5 Points

5. Age of the link. - 3.2 Points

6. Surrounding text of the link. - 3.1 Points

7. Link popularity of external link pages under the same domain. - 2.8 Points

8. Creation and update time of external links. - 2.5 Points

9. Special nature of the domain of the external link website. - 2.5 Points

10. PR value of the external link website. - 2.4 Points

III. Keywords - 29.7 Points

1. Use of keywords in the website TITLE. - 4.9 Points

2. Application of keywords in web page content. - 3.7 Points

3. Relevance between page content and keywords (semantic analysis). - 3.4 Points

4. Use of keywords in H1 tags. - 3.3 Points

5. Use of keywords in the website domain. - 3 Points

6. Use of keywords in the page URL. - 2.8 Points

7. Use of keywords in H2, H3, etc., headline tags. - 2.8 Points

8. Optimization of image keywords. - 2.6 Points

9. Use of keywords in Meta Description. - 2 Points

10. Use of keywords in Meta Keywords. - 1.2 Points

IV. Page Quality - 21.8 Points

1. Internal linking structure of the website. - 4 Points

2. Quality and relevance of outgoing links. - 3.5 Points

3. Age of the page. - 3.4 Points

4. Quality of page content. - 3 Points

5. Structural hierarchy of the website. - 2.8 Points

6. Number of times the “/” symbol appears in the URL. - 1.9 Points

7. Correctness of spelling and grammar. - 1.8 Points

8. Whether the HTML code passes W3C certification. - 1.4 Points
