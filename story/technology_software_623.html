WordPress 2.0.1 发布

by geekzhang on 2006-02-01 15:32:59

WordPress 2.0.1 包含了许多重要的更新，现在它可以下载了。这次更新一共关闭了 114 个 bug。大致总结如下：

你现在可以指定上传目录，并且可以选择是否按日期保存。

修正了某些特定的 PHP 环境下缓存的问题。

永久链接在特殊环境下也可以很好的工作了。

实现了 XML-RPC 上传。

兼容更老的 PHP 版本。

修正了一些所见即所得编辑器的问题。

现在导入功能需要更少的内存。

现在可以在 MySQL 5.0 严格模式下工作了。
