How to Root HTC Desire and Flash Tutorial

by dream2012 on 2012-02-21 18:08:09

European users of the HTC Desire have already received the official Android 2.2 OTA update, but the timing for the Asian version of the Desire remains unknown. Many Desire users are dissatisfied with this differential treatment. If you don't want to wait aimlessly, you can follow our tutorial to conveniently obtain root access on your phone and then install any third-party HTC Desire Android 2.2 ROM. Please note that this method only applies to Desire phones running Android 2.1; if you've already upgraded to Android 2.2, do not attempt this method.

### Obtain Root Access and Install Recovery Program

1. Download the Windows installer file from the UnrEVOked team website.

2. Run the downloaded reflash_package.exe file and extract it to a specific system directory.

3. Install the HBOOT driver according to the method in the hboot driver.htm file (it includes detailed steps and screenshots).

4. Install the HTC Sync software (download link here).

5. Restart your computer.

6. On your Desire phone, go to the settings menu, select "Connect to PC," set the default connection type to disk drive, and uncheck "Ask me" in the upper-level settings.

7. In the phone settings > Applications, check the "Unknown sources" option. Then, in settings > Applications > Development options, check USB debugging and confirm.

8. Download a recovery program, we recommend Amon Ra Recovery.

9. Run the reflash.exe program. If you're using Vista or Windows 7, right-click the program and select "Run as administrator." After opening, choose Custom Recovery from the File menu, then select the previously downloaded Amon Ra Recovery file.

10. Connect your Desire phone to the PC.

11. Wait for 1-2 minutes.

12. The Desire phone will reboot into recovery mode. Click reboot to restart. At this point, you have successfully rooted your Desire and installed the recovery program, and you can proceed to the next step of flashing.

**Note:** The above steps have been tested multiple times and are very reliable. If during the process, after connecting your phone to the PC, the UnrEVOked program stays stuck at the "Waiting for Device" interface, it is usually because the HBOOT driver was not successfully installed. Go back to step three and reinstall it.

### Install a Third-Party ROM

1. Download a third-party ROM file. You can choose to install any Desire ROM, but we recommend installing ROMs provided by XDA forums or MoDaCo (MoDaCo's ROM includes the HTC Sense user interface and is the closest to the official version).

- [MoDaCo ROM](http://modaco.com/)

2. Transfer the downloaded ROM file to the root directory of your phone's SD card (do not place it in any subfolder).

3. Download the Titanium Backup app from the Android Market, open the program, and click the 'Problems' button to install BusyBox.

4. In the Titanium Backup program interface, press the Menu key and select > Batch > Backup All Apps + System Data to back up all installed applications and system data (this step ensures that after installing other ROMs, you can restore previously installed applications and system settings).

5. Turn off the phone, press and hold the power button and the volume down button simultaneously, then use the volume buttons to select the Recovery menu and press the power button to confirm.

6. When the phone enters the recovery interface, use the Desire's optical control key to select Nandroid, then press the optical control key again to back up your system (this step prevents you from being unable to restore to the previous system if the flash fails).

7. After completing the Nandroid backup, press the back key to return to the main recovery interface and perform a Wipe Data/Factory Reset operation once.

8. After backing up, you can enter the most important ROM installation phase. Use the Flash zip from SD Card option, then select the ROM file previously placed in the SD card's root directory and confirm.

9. After installation, select Reboot to restart the phone, and you will enter the newly installed ROM. Get ready for some surprises!

10. If you want to install another ROM, just repeat the above steps.
