Visa 以 9.63 亿美元的价格收购了 Currencycloud，后者为汇款和货币转账提供 API

2021-07-22 22:45:17

与上一轮融资相比，这个价格是一个非常可观的飞跃，当时消息人士告诉我们这家初创公司的估值约为 5 亿美元。 （由于 Visa 已经拥有该公司的股权，其实际支付的金额将减少该金额。） CurrencyCloud 在 180 个国家/地区拥有约 500 名客户，这些客户使用其 API 为多币种钱包、货币兑换服务和账户管理提供支持，包括一些最大的初创公司，如 Monzo、Moneze、Starling、Revolut 和 Dwolla。这些将继续进行，除此之外，Visa 将利用这家初创公司的技术来加强自己的货币兑换渠道，为自己的客户提供更广泛的服务，包括金融机构、金融科技等，并建立新服务对于消费者也是如此。 “在 Currencycloud，我们一直致力于为所有人创造更美好的明天，从最小的初创公司到全球跨国公司。随着我们加入 Visa，重新想象资金如何围绕全球经济流动变得更加令人兴奋，” Currencycloud 首席执行官 Mike Laven 在一份声明中表示。 “Currencycloud 的金融科技专业知识和 Visa 的网络相结合，将使我们能够为跨境资金转移的企业提供更大的客户价值。”汇款和货币转账将成为金融服务领域的大生意，而且这种机会正在增长。推动这一趋势的两个因素是电子商务已经远远超出了我们的国界，尤其是在过去的 18 个月中，供应链也是如此。 （Visa 指出，2020 年全球约有 43% 的小企业进行了某种形式的国际贸易。）随着基于云的移动服务的兴起以促进交易，消费者的前景也变得更加全球化。与此同时，汇款和货币转账是两个可能被颠覆的领域，现有服务往往成本高昂且效率低下。所有这些都为像 CurrencyCloud 这样的公司奠定了基础，该公司已经构建了一种新的货币转移实现，可以嵌入到其他金融服务中，帮助它们更顺畅地运行。退出也是一个典型的例子，说明较大的、现有的金融巨头通常发现创新和跳入新服务更难，所以他们转而利用规模较小、更灵活的初创公司，这些初创公司在技术上押下大赌注，并取得成功，以帮助推动自己进入下一代金融服务。 Visa 是否能够成功整合和使用 CurrencyCloud 的技术并与其团队合作，这两件事已经过测试：在这笔交易之前，两人是战略合作伙伴。

“收购 Currencycloud 是 Visa 执行我们的网络战略以促进全球资金流动的另一个例子，”Visa 全球财务主管科琳·奥斯特罗夫斯基 (Colleen Ostrowski) 在一份声明中说。 “消费者和企业在进行或接收国际支付时越来越期望透明度、速度和简单性。通过收购 Currencycloud，我们可以支持我们的客户和合作伙伴进一步减少跨境支付的痛点，并为他们的客户开发出色的用户体验。”
