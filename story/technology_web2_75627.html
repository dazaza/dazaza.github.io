Fanfou [Brief Review]: A well-known domestic microblogging site, slogan -- mini-blog, send messages anytime and anywhere. [Submission]: Online submission, browser quick toolbar and right-click submission, IM submission (QQ, MSN, Gtalk, supports signature update and publication), mobile text message submission (China Mobile, China Unicom), MMS photo submission, blog sharing button submission (details click), mobile WAP client submission and management. [Management]: Provides a short domain personal homepage, the management center functions are complete, layout is clear, interface is fresh, supports friends, follow, share (favorite), photos, subscription, private messages, etc. [Features]: Diverse submission methods; rich blog integration extension plug-ins. [Shortcomings]: Does not support receiving friend information via IM. (For specific reviews, please see "Did you 'Fanfou' today?")

Jiwaide [Brief Review]: A new domestic microblogging site. [Submission]: Online submission, IM submission (QQ, MSN, Gtalk, Skype, Tsinghua University community, supports signature update and publication), mobile text message submission (China Mobile, China Unicom, details click). [Management]: Provides a short domain personal homepage (supports custom color scheme), the management center functions are complete, layout is clear, supports friends, fans (follow), favorites, subscriptions, whispers (private messages), etc. [Features]: Relatively rich blog integration extension plug-ins; sending supports OpenID; receive friend information via mobile phone. [Shortcomings]: Not enough variety in information submission methods, does not support photo submission.

Weike [Brief Review]: A new domestic microblogging site, slogan -- record every moment of life. [Submission]: Online submission, IM submission (QQ, MSN, Gtalk, supports signature update and publication), mobile text message submission (China Mobile, China Unicom, PHS), mobile WAP submission and management. [Management]: Provides a short domain personal homepage (supports custom background images), the management center functions are complete, layout is clear, supports friends, follow, favorites, photos, subscriptions, notes (private messages), etc. [Features]: Online photo sharing (submit to album), rich blog integration extension plug-ins, supports PHS users, receive friend information via IM.

Ai Laodao [Brief Review]: A domestic microblogging site, slogan -- what are you doing? One-sentence blog, pay attention to friends, use chatter to record life! [Submission]: Online submission, IM submission (QQ, MSN, Gtalk), mobile text message submission (only supports China Mobile users), mobile WAP submission and management. [Management]: Provides a custom domain personal homepage, the management center functions are relatively complete, supports friends, follow, favorites, subscriptions, private messages, etc. [Features]: Relatively rich blog integration extension plug-ins; categorized robot friends (news, sports, finance); receive friend information via mobile phone and IM. [Shortcomings]: Does not support photo submission.

Komoo [Brief Review]: A domestic microblogging site, interface strives to imitate Twitter, slogan -- what are you doing? [Submission]: Online submission, browser quick toolbar submission (share), IM submission (MSN), mobile text message submission (China Mobile, China Unicom), MMS photo submission. [Management]: Provides a short domain personal homepage (supports custom style), the management center functions are relatively complete, supports friends, follow, favorites, photos, subscriptions, private messages, etc. [Features]: Blog integration extension plug-ins. [Shortcomings]: Few extension plug-ins, supported IM software is too singular; does not support receiving friend information via IM.
