Wang Kaka: Analysis of 13 reasons for search engines not indexing or reducing indexing

by kakaseo0813 on 2010-08-21 11:07:05

It is well known that the inclusion of websites by search engines is very important. If there is no inclusion or the inclusion decreases, then our optimization, promotion, and operation work will encounter very significant difficulties. Recently, many friends in the group have been asking about this issue, so today I will share with you the reasons I know for why search engines may not include or reduce the inclusion of websites.

1. New Websites

Generally, newly uploaded websites will not be included immediately. Even if they are included by search engines, they won't be released right away. When my blog "Wang Kaka" went online, Google included it on the same day, but Baidu, although it was also included on the same day, only released it a week later. Here, I suggest that after uploading a new website, you can create some high-quality external links to attract spiders. The content of the website should be rich, and it would be best to place some original articles. Finally, upload the website one or two days before the major update of the search engine and submit it to the search engine (this cannot be done frequently). This way, you can quickly get the search engine to include your website.

2. Over-Optimization of Website

There is nothing wrong with optimizing a website, but over-optimization instead loses the favor of search engines. In the past, methods such as keyword stuffing, bolding keywords, and high-density keywords could rank higher, but now they cannot. Search engines have clearly stated that they dislike these behaviors. Therefore, if our website still has over-optimized areas, we should modify them quickly. Either it won't be included, or the website will be penalized, which is not worth it.

3. Website Contains Unharmonious Content

This is something that cannot be helped; unharmonious content will definitely get your website harmonized. So the content of our website must comply with national laws and various regulations. Another point is to respect search engines and not include any language or content that attacks search engines negatively. This is relatively serious, and generally, when a site is penalized, there is no negotiation.

4. Use Advertising Alliances Cautiously

Nowadays, many websites have advertising alliance content, which is also not favored by search engines. One reason is that search engines prefer their own products, another is the large amount of repeated content, and the third is that it affects user reading. I once had a website that was penalized by another search engine because it hung an advertisement from a different search engine. This problem is not necessarily going to happen, but it is worth noting!

5: Excessive Use of Baidu Products for Optimization and Promotion

I believe many webmasters have already noticed this issue. Overusing Baidu's products to post a large number of links not only fails to produce good results but also risks having the site penalized. Not just Baidu, any search engine is the same. Doing this on its platform is clearly provoking it. Therefore, we should reasonably use the products of search engines and not let them find out that you are cheating!

6: Penalized Friend Links

Many small and medium-sized enterprise sites, due to the lack of professional technical personnel or not taking friend links seriously, do not have the habit of checking friend links. If a friend link site is penalized by a search engine, then your site will also be affected, and in severe cases, it will be penalized! Or if there are a large number of private server sites or other things that search engines do not like among the external links of the site, it will cause the reduction of the site's weight.

7: Being Considered a Spam Site by Search Engines

When the content of the site is all collected or the text content basically has no readability or value, users don't like it, and neither do search engines. Not being included, reduced inclusion, or being penalized is inevitable! So we need to organize our article content well!

8. Adjustment of Search Engine

The search engine algorithm is being adjusted, temporarily stopping or reducing the inclusion of websites, and your website happens to be within the scope of adjustment. At this time, we should promptly communicate with friends in the webmaster circle to see if everyone is experiencing similar problems, and then make timely adjustments!

9. Large-Scale Renovation of Website

This is generally more common. A large-scale renovation causes the search engine to consider it a new site and thus re-examine it, leading to a decrease in the site's weight, which reduces the site's inclusion. 10. Unreasonable Website Structure

A large number of dead links appear on the website, frame calls, web pages appear garbled, or a large number of pictures, flash, etc., are used to compose content that cannot be normally crawled and included, all of which prevent search engine spiders from visiting normally.

11. Website Infected with Malware

This problem boils down to the security issues of the website. Websites infected with malware inevitably affect users' reading and safety, so being penalized and resulting in reduced inclusion is inevitable!

12. Affected by Other Websites on the Same Server

If a website on the same IP address of the server is penalized, it leads to the server's IP being blocked by the search engine, and the website is similarly affected. The IP of the blocked server is stopped from being crawled and included by the search engine.

13. Errors in Writing robots.txt

Mistakes in the website's robots.txt file prevent search engines from crawling the website, ultimately resulting in the website not being included.
