Virtually Human Studio 是数字赛马服务 Zed Run 背后的初创公司，已经筹集了 2000 万美元的融资，突显了投资者对探索娱乐、游戏和所谓的不可替代代币行业交叉领域的公司的兴趣。专注于媒体和技术的投资公司 TCG Capital Management 领投了 A 轮融资，Andreessen Horowitz 和 Red Beard Ventures 也参与了这笔交易。 Zed Run 于 2019 年推出，使用户能够购买、出售和繁殖虚拟马匹，并在电子游戏风格的环境中与其他马匹比赛。这些马用数字货币购买并充当 NFT，本质上是具有价值的数字收藏品。 Zed Run 从用户支付的费用中收取费用，用于在网站上虚拟繁殖马匹，以及比赛报名费和这些 NFT 的销售。当他们的马匹价值上涨时，个人虚拟马主就会赚钱，从而在 Zed Run 的马匹繁殖市场和马匹转售市场上吸引更高的价格。每场比赛都会在以太坊（一种加密货币）中获得第一到第三名的奖金。 Zed 数字赛马的价格从 130 美元到 45,000 美元不等。该公司表示，包括多匹马在内的整个马厩售价超过 25 万美元，业主可以转售马匹。 《华尔街日报》将向您收取 $ + 税款（如果适用）。您可以随时在客户中心更改您的计费偏好或致电客户服务。如果费率或条款发生任何变化，您将提前收到通知。您可以随时致电客户服务部取消订阅。
