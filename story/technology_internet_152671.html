Common Problems and Solutions for Windows Vista

(Note: The following content is for reference only when needed. Some functions in the system brought by this disc have already been set up, please do not set them again.)

1. How to turn off UAC?

Control Panel → User Accounts → Turn User Account Control on or off → Uncheck Use User Account Control (UAC) to help protect your computer.

2. How to turn off system backup and restore?

Control Panel → Backup and Restore Center → Create a restore point or change settings → System Protection → Automatic Restore Points → Uncheck the restore points on the selected disk.

3. Why does the hard drive read intensively at a fixed time?

This is because Vista automatically executes the disk defragmentation schedule. The method to turn it off is through Start Menu → Accessories → System Tools → Disk Defragmenter → Cancel "Run on schedule (recommended)"

4. How to stop the computer from going into hibernation?

Desktop right-click → Personalization → Screen Saver → Change Power Settings → Change when the computer sleeps → Set computer to sleep to never.

5. The default setting of the power button in the Start menu is "Hibernate," how to change it to "Shut Down"?

Control Panel → Power Options → Selected "Preferred Plan" → Change plan settings → Change advanced power settings, in the pop-up settings window, find "Power buttons and lid" → "Start" menu power button → Setting → Shutdown, confirm to complete the setting.

6. Every time I start my computer, a prompt appears in the Security Center, how to turn off this prompt?

Control Panel → Security Center → Change the way Security Center notifies me → Select Not notify me, but show icon.

7. How to display the Run command in a non-traditional Start menu?

Taskbar and Start Menu Properties → Start Menu → Customize → Check "Run command."

8. How to cancel the default sharing every time I start my computer?

Use Vista Optimization Master, or import the following content into the registry, restart (premise is to turn off UAC).

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Lanmanserver\Parameters]

AutoShareWks=dword:00000000

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Lsa]

restrictanonymous=dword:00000001

9. How to achieve automatic login?

Start → Run → Enter "rundll32 netplwiz.dll,UsersRunDll" command to open the account window, first select the account to automatically log in, deselect "To use this computer, users must enter username and password" checkbox, input the password for that account (premise is to turn off UAC), using Vista Optimization Master makes it easier to achieve.

10. How to change media files to play with players other than WMP?

Select media file → Right-click menu → Open with → Choose default program → Select playback program → Check "Always use the selected program to open this kind of file."

11. Why can't I visit each other on the local area network?

Try the following approaches:

Local Connection → Right-click Properties → Uncheck Internet Protocol version 6, shorten validation time.

Check if the machine's workgroup name is consistent.

Check if the machine's IP is in the same subnet, such as 192.168.1.X.

Set the Network and Sharing Center → Network Connections → Customize → Location Type to Private, lower protection.

12. How to visit each other on the local network after enabling ICF?

Windows Firewall cancels blocking all incoming connections and selects "File and Printer Sharing" in "Exceptions."

13. How to log in with the Administrator account?

Computer → Manage → Local Users and Groups → Users → Administrator account → Properties → Uncheck Disabled account option, for the currently used account → Properties → Check Disabled account option.

14. An error occurs when opening the Control Panel, how to fix it?

Hardware driver conflict, possibly related to sound card drivers, try reinstalling the sound card driver.

15. How to quickly copy file, folder paths?

Hold down the Shift key, right-click the file, folder, or shortcut you want to copy the path of, in the pop-up menu, there is an item "Copy as Path (A)," click it to copy the file or folder path.

16. How to view system license information?

Run slmgr.vbs -dlv.

17. What is the shortcut key for reverse selection of files?

ALT+E pops up the menu then press I.

18. How to change the size of icons on the desktop?

Ctrl+Mouse Wheel or Desktop right-click → View → Classic Icons.

19. How to switch programs in 3D?

Under Aero appearance, use Win (Window Key) + Tab to switch programs in 3D, Win (Window Key) + Ctrl + Tab keeps the 3D window temporarily on the desktop.

20. How to set proxy for IE browser?

Open the Internet Options toolbar in IE → Connections → Dial-up and Virtual Private Network Settings, select your dial-up connection → Settings, in the "Settings" page, check "Use a proxy server for this connection" then enter the proxy server address and port in the "Address" field, click "OK."

21. How to view the PID value of a process?

Task Manager → View → Select Columns can be chosen.

22. How to clear the history of recently opened documents when exiting the system?

Run gpedit.msc, User Configuration → Administrative Templates → 'Start' Menu and Taskbar → Clear recent document history when exiting the system, set to Enabled.

23. Screen capture shows garbled screen

You will often find that Vista itself displays normally, but whether using the Print Screen key or other screenshot tools, part of the captured image will be garbled, sometimes a line, sometimes a block. This is mainly due to the imperfect built-in graphics driver of Vista. The solution is to download and install the latest version of the graphics driver, which usually resolves the issue easily.

24. Recover the missing "Run" dialog box

By default, the familiar "Run" command is not present at the bottom of the Vista Start menu, which can be inconvenient for some advanced operations. You can bring up the "Run" dialog box through the following two methods.

(1) Press the "Win+R" key combination to open the "Run" dialog box, which we already used in question four.

(2) Right-click on the Start menu, select "Properties," go to the Start menu properties settings interface, click the "Customize" button at the bottom, find "Show Run" in the list box and check it, then click "Confirm" to exit, and you will find the "Run" command reappears at the bottom of the Start menu.

25. Unable to boot

Unable to boot is a relatively common and serious fault. Previously, under XP, this kind of fault had to be manually repaired by users. Under Vista, you just need to boot the machine from the disc into the recovery console, then enter the fixboot command, and Vista will automatically detect boot faults and repair them using the BCDMD command. This is a great help for ordinary users, worthy of praise.

26. Why does QQ always disconnect under Vista?

QQ always disconnects under Vista but quickly returns to normal. This is because the network card defaults to working in energy-saving mode under Vista. If you want it to continue saving energy, you can ignore this phenomenon, as it does not affect the normal use of QQ. If you need to handle it, simply remove the checkmark before "Allow the computer to turn off this device to save power" in Device Manager → Network Adapter → Properties → Power Management!

27. Why does my system automatically exit during automatic updates?

During automatic updates, the latest upgrade program needs to be downloaded and installed. This process involves closing and restarting the automatic update. Just wait a moment and everything will return to normal.
