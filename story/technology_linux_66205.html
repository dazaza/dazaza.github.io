The Tsinghua-Peking Linux CD Project

Founded in 2002 by students from Tsinghua and Peking Universities, the Tsinghua-Peking DIY Club is committed to providing high-quality computer products and related services. To promote the popularization and development of open-source software in China, Tsinghua-Peking started distributing free Linux system CDs to universities in Beijing in 2006. This distribution of Ubuntu 8.04 system CDs follows the previous large-scale distributions of Ubuntu 6.06 and Ubuntu 7.10.

This time, the Ubuntu 8.04 system CD distribution will be available nationwide for the first time. Users in the Beijing area can apply for one set of Ubuntu 8.04 system CDs (1 DVD + 1 CD). After applying on the Tsinghua-Peking service website and receiving confirmation, they can collect the CDs from the Tsinghua-Peking E-World storefront. For users in other regions of China, we will select 1 to 3 volunteers from each region's applicants to assist with the distribution of the CDs. Volunteers will be responsible for the courier fees from Beijing to their local area, and Tsinghua-Peking will provide 1.5 times the number of free CDs based on actual application numbers.

Ubuntu's success worldwide is undeniable. We hope that through the Tsinghua-Peking Linux CD project, more volunteers will join this activity to promote the vigorous development of open-source software in China.

If there are other providers offering free Ubuntu 8.04 system CDs, they can post on the event topic page of the Tsinghua-Peking service website. We will collect and organize this information so more users can benefit.

Wishing everyone a pleasant journey with Ubuntu 8.04!

Introduction to Ubuntu 8.04

As the world's most popular Linux desktop operating system, Ubuntu has a friendly interface and is easy to operate. The goal of the Ubuntu distribution is to provide the latest and most stable operating system for ordinary non-professional computer users, while also being suitable for professional computer users, including developers. Sponsored by Canonical, founded by Mark Shuttleworth, Ubuntu is based on the well-known Debian GNU/Linux distribution and includes both desktop and server versions. The desktop version of Ubuntu has multiple derivative versions such as Kubuntu, Xubuntu, Edubuntu, Ubuntu JeOS, UbuntuStudio, and Mythbuntu, which can meet different user needs and habits.

Ubuntu 8.04, codenamed Hardy Heron, was released on April 24, 2008. Ubuntu 8.04 is the second long-term support version (LTS, Long-Term Support) after Ubuntu 6.06. Its desktop version will receive two years of support, and the server version will receive five years of support, fully meeting enterprise users' requirements for stability.

New features in Ubuntu 8.04 include:

* Linux kernel 2.6.24

* GNOME 2.22.1

* Xorg 7.3

* KVM virtualization technology

* Additional memory protection against rootkits and other malicious attack codes

* Firefox 3 Beta 5 Web browser

* Pidgin 2.4.1 instant messaging tool

* OpenOffice.org 2.4.0 office software

* GIMP 2.4.5 image processing tool

* Totem 2.22.1 movie player

* Rhythmbox 0.11.5 music player

* Brasero 0.7.1 disc burning tool

* Tomboy 0.10.1 note-taking software

* Useful desktop applets such as a world clock.

You can also easily obtain thousands of popular applications like QTerm and FileZilla from free software sources.

Ubuntu is a completely open-source operating system built on the Linux kernel. The Ubuntu community is built around the ideals of the Ubuntu manifesto: software should be freely available, software tools should be usable in all language environments without any functional differences; people should have the freedom to customize and modify software unrestricted, as long as they find it suitable. Based on these reasons:

* Ubuntu will always be free, and there will never be additional charges for an "enterprise edition." We will do our best work for everyone under the same spirit of freedom.

* Ubuntu includes top translations and accessibility architectures provided by the free software community to make Ubuntu accessible to as many people as possible.

* Ubuntu will be released regularly and planned consistently; a new version will be released every six months. You can use either the current stable version or the development version, and we will provide at least 18 months of support for each version.

* Ubuntu fully adheres to the principles of open-source software development; we encourage people to use, improve, and spread open-source software.
