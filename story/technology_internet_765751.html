Domestic open-source Java CMS, Jspxcms 3.0 Released

by anonymous on 2013-08-20 15:48:17

Jspxcms-3.0.0-release is officially released today.

This release marks further maturity of Jspxcms, with rich features and flexible settings, achieving an all-encompassing effect in website creation. It truly realizes non-intrusive secondary development, unified permissions, menus, tags, and internationalization. All added functionalities do not require modification of existing code.

New Features: Member Center (registration, login, member center, password recovery, email verification, personal information modification, password change), watermark, message board, advertising, friend links, rating groups, sensitive words, user-defined fields, organization management, workflow, function permissions (control to the button level), node data permissions, information data permissions, galleries, videos, products, downloads.

Nearly perfect performance:

Test results summary for http://demo.jspxcms.com/: 5000 requests, 500 concurrent, all successful, total time consumed 31.124 seconds, handling 160.65 requests per second, each request taking 6.225 milliseconds.

Test results summary for http://demo.jspxcms.com/node/40.jspx: 5000 requests, 500 concurrent, all successful, total time consumed 11.969 seconds, handling 417.73 requests per second, each request taking 2.394 milliseconds.

Detailed test report: http://bbs.jspxcms.com/thread-35-1-1.html

Download address: http://www.jspxcms.com/download/

Demo site: http://demo.jspxcms.com Backend: http://demo.jspxcms.com/cmscp/index.do Username: test, Password: 123

Main technologies: SpringMVC3.2, Spring3.2, JPA2.0, JSP2.0, Freemarker2.3, Spring Data JPA, QueryDSL, Shiro, Lucene, etc.

Technical highlights: An incredibly concise and efficient persistence technology composed of JPA, Spring Data JPA, and QueryDSL; Shiro security framework; Near real-time search with Lucene; Freemarker template technology; Gmail-style captcha, etc.

Product design philosophy: Quickly build websites with minimal effort; Pursue maximum flexibility to adapt to various usage scenarios; Powerful functions to meet the needs of various types of websites; Superior performance, system security, stable operation, minimizing system maintenance time as much as possible.

Function list:

1. Information Management (news, galleries, downloads, videos, works, etc.)

2. Node Management (full-text search, page staticization)

3. File Management (zip upload self-decompression, zip package download, templates, images, js, css)

4. Information-related (special category management, special topic management, TAG management, comment management, sensitive word management, rating group management)

5. Extension Modules (friend link type management, friend link management, message board type management, message board management, ad position management, ad management, voting management)

5. User Management (administrator management, role management, member management, member group management, organization management)

6. System Configuration (website settings, system settings, model management, information attribute management, workflow management, workflow group management)
