Index Ventures 关闭了一只 9 亿美元的早期基金、一只 2B 美元的成长阶段基金和一只 2 亿美元的种子阶段基金，为 Index 提供了 3B 美元的新资本，这是其迄今为止最大的一笔

2021-07-22 22:21:03

Index 的新资本是在它关闭最后一只基金后一年多一点的时间里出现的，这些基金分别是 12 亿美元的成长阶段投资和 8 亿美元的早期投资。在更广泛的风险投资似乎以前所未有的速度增长的同时，它还宣布了这一消息——最近几周，Accel 宣布它也在三个基金中关闭了 30 亿美元，而 Andreessen Horowitz 获得了一个 22 亿美元的基金，专门用于加密初创公司。资金的涌入意味着整个生态系统的支票规模和估值都在增长。例如，Index 表示已将其 A 系列支票规模从 1000 万美元增加到 1500 万美元，同时将其增长支票从 3500 万美元增加到 5000 万美元。其在美国的支票规模比在欧洲的支票规模高出约 20%，这意味着前者占公司总投资金额的 55% 至 60%。虽然 Index 的支票规模为了跟上竞争而略有增长，但合伙人 Mark Goldberg 表示，该公司保持纪律，因此不会成为 FOMO 轮次或嗡嗡声估值的牺牲品。他说：“在某些情况下，我会在两年前参与，”参加后期回合的比赛。 “现在我说我们可能不会竞争……因为在问题的核心，我认为资本是一种商品。”这位合伙人没有提到 Accel 30 亿美元的收盘价，而是指出“Tiger Global 和其他交叉基金”如何创造了一种新的资本产品，这意味着 Index 的竞争能力必须与仅仅提供大量资金不同。 “如果你只是在寻找资金，我们可能不会成为最适合你的产品，”戈德伯格补充道。 Index 合伙人 Martin Mignot 认为筹款市场“确实发生了分歧”，一些创始人能够轻松筹集资金，但如果他们没有“在正确的时间、正确的地点进入正确的领域，许多创始人将继续挣扎” 。” Mignot 一直在花时间将非洲视为资源仍然不足的潜在投资领域。他暗示，一项针对非洲初创企业的官方计划可能正在进行中。该公司表示，它“已采取多项与多元化相关的举措，包括增加来自代表性不足群体的投资人数，并使我们的投资渠道多样化”，但表示它“尚未”在投资组合公司层面跟踪多元化统计数据。 Index 表示，其投资团队由 38% 的女性个人和 14% 来自代表性不足的群体的个人组成，但不清楚他们是否有支票书写权。

在为投资组合公司提供的服务方面，Index 似乎正在经历一段试验期。虽然该公司拒绝就即将推出的多元化或更广泛的举措提供太多细节，但它确实指出其不断增长的 TikTok 存在是其战略的一部分。 Index Ventures TikTok 拥有超过 28,000 名粉丝。其观看次数最多的视频是关于 Costco 的商业模式，观看次数为 480 万次，其次是 Scale AI 创始人 Alexandr Wang 的采访剪辑，观看次数为 315,000 次。 “我认为下一笔交易不是来自 TikTok 帐户，而是其中一项举措……我认为它会带来好处”，帮助 Index 接触在科技 Twitter 或 Clubhouse 上不活跃的年轻观众。由 Rex Woodbury 领导的 Index 的 TikTok 让人想起一些新兴基金经理如何处理创造性的交易流程。该公司还在尝试在内部孵化初创公司，这在历史上从未成为主要关注点。一条推文表明，Index 正在积极招募企业家加入孵化器，这可以按主题进行划分。戈德堡拒绝就孵化计划的结构和条款提供更多细节，称指数孵化器还不是官方产品。也就是说，人们只能想象一家拥有数十亿美元资本的新公司可能有一些闲钱——从孵化项目到那些需要积极投资者的成长阶段公司。
