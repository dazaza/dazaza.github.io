Windows终端1.0

2020-05-20 00:44:27

去年在Build 2019上，我们首次发布了Windows终端。从那时起，我们一直在与社区合作，在仍是预览产品的情况下创造精彩的终端体验。

我们现在是在Build 2020大会上，我们非常高兴能与您分享我们的最新公告！

我们非常自豪地宣布Windows终端1.0的发布！自Windows终端在微软Build 2019上发布以来，它已经走过了很长的路。您可以从Microsoft应用商店或GitHub版本页面下载Windows终端。Windows终端将于2020年7月开始每月更新。

我们还将推出Windows终端的预览频道。如果您喜欢参与Windows终端的开发，并且一开发就使用最新的功能，这里就是您的频道！您可以从Microsoft应用商店或GitHub版本页面下载Windows终端预览版。Windows终端预览版将从2020年6月开始每月更新。

安装Windows终端后，您可能希望了解如何充分利用新的开发工具。我们刚刚启动了Windows终端文档站点，该站点提供了有关终端必须提供的所有设置和功能的详细信息，以及一些帮助您开始自定义终端的教程。您可以在aka.ms/Terminal-docs上找到所有的Windows终端文档。

Windows终端功能丰富，可以改进您的工作流程，并为您提供多种自定义选项，为您提供最佳体验。下面是我们粉丝最喜欢的一些功能的概述。

Windows终端允许您在选项卡和窗格内运行任何命令行应用程序。您可以为每个命令行应用程序创建配置文件，然后并排打开它们以实现无缝工作流。您的每个配置文件都可以根据您的喜好进行独特的定制。此外，如果您的计算机上安装了Windows Subsystem for Linux发行版或其他PowerShell版本，终端将自动为您创建配置文件。

Windows终端利用GPU呈现其文本。这在使用命令行时提供了更快的体验。此渲染器还支持Unicode和UTF-8字符。这让你有机会以各种语言使用终端，同时还可以显示所有你最喜欢的表情符号。😉最后，我们在Windows终端包中包含了我们最新的字体，Cascadia代码。默认字体设置为Cascadia Mono，这是不包括编程连字的字体变体。有关Cascadia Code字体的其他变体，请访问Cascadia Code GitHub Repo。

Windows终端充满了提供无限量定制的设置。您可以拥有具有独特配色方案和自定义字体的丙烯酸背景和背景图像，以获得您自己的个性化体验。您还可以添加自定义密钥绑定，让您更有宾至如归的感觉。此外，每个配置文件都可以自定义以匹配您需要的工作流，无论它是在Windows、WSL上，甚至是在SSH上！

Windows终端中一些最酷的功能是由GitHub上的社区成员贡献的。首先我们要强调的是背景图像支持。Summon528为Windows终端编写了支持文本窗口背景中的gif和图像的功能。到目前为止，这是我们最常用的功能之一。

另一个粉丝最喜欢的是复古终端效果设置。Ironyman增加了对文本缓冲区内发光文本和扫描线的支持，从而提供了使用经典CRT机器的感觉。团队从未预料到GitHub会实现这一功能，但它太棒了，以至于我们不得不将其包含在终端中。

如果没有我们社区的帮助，Windows终端就不会有今天的成就。我们要感谢那些在将Windows终端升级到1.0的过程中做出巨大贡献的人。

我们有成百上千的贡献者，我们希望我们能把他们都列在上面！我们要特别感谢他们所有的帮助，没有他们我们做不到！

我们正在积极开发新功能，这些功能将出现在6月份的Windows终端预览版中。如果您想要加入其中并为Windows终端做出贡献，在我们的GitHub回购上有许多标有“需要帮助”的问题！我们很快就会在GitHub上发布我们的Windows终端2.0路线图，所以一定要密切关注。如果您对我们正在积极进行的工作感到好奇，我们的里程碑将让您对我们的发展方向有一个很好的了解。😊。

我们希望您喜欢Windows终端1.0，以及我们新的Windows终端预览和aka.ms/Terminal-docs站点。如果您有任何问题或反馈，请随时在Twitter(@cinnamon_msft)上联系Kayla Cinnamon。如果您发现任何错误或想要提交功能请求，请在GitHub上提交新的问题。如果您想了解更多关于Build 2020上发布的开发人员工具的信息，请访问Kevin Gallo的博客文章。我们迫不及待地继续与社区合作并开发新功能，以继续使Windows终端成为开发人员的伟大工具！
