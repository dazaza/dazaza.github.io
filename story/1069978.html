随着 Matrix 的爆发，Element 筹集了 3000 万美元

2021-07-27 16:42:41

世界已经真正觉醒到了去中心化通信的时代。美国国会正在发布 ACCESS 法案，以加强平台之间的互操作性和数据可移植性；欧盟正在为此推进《数字市场法案》——德国国家医疗保健系统已公布其计划，将 Matrix 上的 150,000 个医疗保健组织标准化。正是在这种背景下，我们非常高兴地宣布 Element 已经筹集了 3000 万美元的 B 系列资金，以彻底改变应用程序的可用性，构建主要的新功能，扩展企业市场并使 Matrix 完全成为主流！这轮融资由我们在 Protocol Labs 的老朋友（libp2p、IPFS、Filecoin 等的创造者）和 Metaplanet 领投，Metaplanet 是由 Skype 联合创始人 Jaan Tallinn 设立的投资基金。 Automattic 和 Notion 也参与了这一轮，将他们对 Element 的现有投资加倍。我们想不出比 Protocol Labs 和 Metaplanet 更好的新投资者来支持 Element 的使命。 Protocol Labs 毫无疑问是去中心化运动的领导者：只要我们一直致力于将 Matrix 作为去中心化通信层，他们就一直忙于构建 IPFS 作为互联网的去中心化存储层。在 Element 诞生之前，他们就一直在 Matrix 上（Vector 是最早完全托管在 IPFS 上的应用程序之一！），我们迫不及待地想在未来与他们更密切地合作，尽管技术只会在显然是对我们来说正确的技术决策——例如，Element（和 Matrix）不会很快使用加密货币激励措施。与此同时，Metaplanet 是最具前瞻性的投资基金之一——当他们说他们的投资主题是“支持可以为人类带来巨大回报的新颖和循证创新”时，他们并不是在开玩笑。换句话说，他们投资于那些为了让世界变得更美好而做出疯狂破坏性长期事情的人——在我们的例子中，是通过民主化交流使私人交流成为所有人的基本人权。除此之外，Jaan 构建 Skype（他负责聊天！）的经历与我们对 Matrix 的追求之间也存在明显的大量重叠。许多人忘记了 Skype 最初是一个庞大且成功的去中心化通信应用程序——实际上是完全点对点的，建立在与 Kazaa 相同的技术上。随着 P2P Matrix 工作的进展，从构建 Skype 的经验中吸取教训会很棒。这笔投资也非常具体地反映了 Matrix 和 Element 的进步和成熟，以及越来越注重隐私的市场。仅在过去的六个月中，我们就在 EMS 上为 Slack 和 Teams（以及 Telegram 和 Discord）推出了桥梁，在企业市场上获得了真正的吸引力 - 很快就会有更多。与此同时，事实证明，Element Home 在想要拥有自己的服务器的朋友和家人中很受欢迎。与此同时，在更广阔的 Matrix 世界中，我们看到世界上最大的开源会议 FOSDEM 通过 Matrix 实现虚拟化（在 660 多个演讲中有超过 30,000 名与会者），激发了许多其他开源项目的加入。最后，Spaces 的 Beta 版发布了一场风暴——终于让用户可以将他们的房间分成组甚至层次结构，很快就会退出 Beta 版！

在公共部门，越来越多的政府和非政府组织（希望很快宣布！）正在加入英国、美国、法国和德国的行列，建立 Matrix。就在上周，我们看到了德国国家卫生机构 gematik，它宣布德国的所有医疗保健部门都将使用 Matrix 来发送消息。私营部门也在迎头赶上，意识到将所有未加密的对话交给微软和 Salesforce (Slack) 可能不是最好的举措。在其他地方，我们看到越来越多的用户意识到 Element 是 Discord 的绝佳安全替代品——或者是 WhatsApp 的去中心化替代品。坦率地说，作为一家公司，如果我们愿意，Element 在这一点上可以自给自足，通过来自 EMS 托管、EMS 服务和法国和德国等国家级部署的支持合同的收入为基本的 Matrix 和 Element 开发提供资金。公共网络现在拥有超过 3500 万可寻址用户，分布在约 7 万个部署中，我们正在与其他 10 多个国家/地区合作，以将他们也加入进来。矩阵就在这里。然而，在大流行、对 WhatsApp 隐私政策变化的反应，甚至监管机构现在都意识到这些平台垄断是多么不健康的事实之间：过去 18 个月很明显，安全的分散通信已成为一项生存要求，我们应该这样做尽我们所能加快我们剩余的工作。很简单，世界现在需要Matrix。运行 Matrix 服务器应该像运行 Web 服务器一样简单明了。 Element 使用起来应该非常令人愉快，以至于它推动了 Matrix 的采用以获得临界质量并迅速爆发。因此，我们有意识地决定主动筹集这笔新资金，以加速 Element 的增长，并尽快将数据主权带给大众。实际上，这意味着我们要在核心工作上加倍努力：使 Element 尽可能地精致和流畅——引入更多的设计师和产品可用性专家来帮助我们挖掘应用程序中剩余的漏洞；添加剩余的核心缺失功能：空格、线程、固定消息、加星标的消息、自定义表情符号等。在 Matrix 方面，让 Synapse 服务器成为运行的乐趣，并在下一代服务器 Dendrite 上继续开发。我们还组建了专门的团队，专注于完善我们的端到端加密、VoIP/VR、信任与安全以及 P2P 计划。简而言之，对于 Element 和 Matrix，我们正在全面招聘 - 因此，如果您想构建 Web 缺少的通信层，请与我们联系！看到自 2017 年 7 月成立 Element 以来 Matrix 已经走了多远，这有点令人眼花缭乱，但几乎感觉我们仍处于起步阶段。我们正处于 Element 不受控制地在我们的计算机和手机上爆炸的风口浪尖，其方式与 Netscape 在 Web 的早期所做的或电子邮件在早期的 Internet 上所做的几乎相同。想象一个所有服务器都默认带有 Matrix 主服务器的世界，就像今天它们带有邮件传输代理一样——或者是一个点对点世界，其中 Element 的每个副本都嵌入了自己的服务器：用户对他们的对话拥有完全的数字主权。

随着我们向这个美丽的新世界转变，接下来的几个月将证明是至关重要的。新资金将用于将 Element 的抛光提升到一个新的水平——同时全额资助 P2P Matrix 工作、去中心化 VoIP/视频会议和去中心化声誉。这应该是 Matrix 和 Element 的一个步骤变化，我们要感谢 Protocol Labs、Metaplanet、Automattic 和 Notion 为其提供燃料！
