框架上的 NixOS

2021-07-30 06:09:59

在我购买自己的笔记本电脑之前几个月检查一下 Framework 笔记本电脑是一种享受。 Framework 有望成为一款功能强大的高端超薄笔记本电脑，具有易于维修和升级的突出特点。我认为他们已经做到了。您需要使用 Linux 5.13 或更高版本的安装映像才能拥有可用的 Wi-Fi 卡。如果您愿意在没有 GUI 的情况下安装 NixOS，您可以使用 Hydra 的最新内核获取最低 21.05 ISO。构建您自己的自定义安装媒体非常简单。你需要一台装有 Nix 的 Linux 机器。 $ nix-shell -p nixos-generators nix-shell$ nixos-generate -I nixpkgs =channel:nixos-unstable --format iso --configuration ./custom-media.nix 解包 'https://nixos.org/channels /nixos-unstable/nixexprs.tar.xz'...这些派生将被构建：...snip.../nix/store/gnnbjvd916yh1f4svbgrssq94550pbxl-nixos-21.11pre304626.8ecc61x8c91a/iso -21.11pre304626.8ecc61c91a5-x86_64-linux.iso 注意：安全启动将阻止未签名的软件运行。 NixOS 目前不支持安全启动。

他们寄给我的评测单元是预发布硬件，但规格齐全且可爱：它的性能似乎相当不错，足以满足我的日常工作。驱动我的 38 英寸超宽显示器没有问题。除了编译的时候，机器是安静的，从来没有感觉到热。经过长时间的编译，铰链附近的一些区域明显温暖，但手托仍然凉爽。对我来说，键盘和触控板感觉很完美。按键感觉清脆但不重。触控板感觉大而精确。也就是说，点击有点太点击了。音频和摄像头的硬件开关很好，但有点难以切换。开关非常小且光滑，因此难以移动。开关是可更换挡板的一部分，这很容易固定。电源砖经过 UL 能源验证，仅比戴尔 XPS 9300 砖稍大。 Intel Wi-Fi 6 AX210 与 Linux 存在集成问题。有时wifi固件在启动时会崩溃，但似乎不会比50/50更频繁。一旦启动，wifi就坚如磐石。

一切都有一个二维码作为指令的链接。我的意思是一切：直到将墙壁连接到电源砖的电缆。然而，他们中的许多人实际上并没有领导任何地方。看到电源线的二维码是 404ing，我并不太惊讶，但看到 RAM 替换 404ing 的链接有点惊讶。我确信这是一项正在进行的工作，这些二维码都可以正常工作。我想要我能买到的最小、最轻的笔记本电脑，而 Framework 是这里的有力竞争者。这台笔记本电脑的宽度和厚度基本相同：在一两毫米之内。该框架仅比 XPS 重 60 克（1330g 与 1270g）。与 9300 相比的一项重大改进：指纹传感器由开箱即用的最新版本 fprintd 支持，并且不需要 TOD 补丁。传感器快速可靠，而戴尔 XPS 9300 的 TOD Goodix 驱动程序在 Linux 上几乎一文不值。从砖块到墙壁的电缆看起来像台式 PC 的电缆。厚而有光泽的塑料。与戴尔电缆的更抛光、抛光、品牌外观相比。总的来说，我对Framework 非常满意。团队看起来很棒，他们的价值观似乎与我自己的相符。今年晚些时候，当我现在的笔记本电脑不是那么新时，我会掏出我的钱包。我认为他们已经实现了他们的承诺，并创造了一种令人耳目一新的机器：一种可维修的超轻机器。
