OOR云对象发现进入外太阳系统

2021-06-26 06:30:47

天文学家通过来自黑暗能量调查（DES）的数据来筛选了一个大型oort云对象，接近太阳系的外部区域。

该发现引起了行星科学界兴奋的涟漪，因为该物体异常大的尺寸 - 初始估计表明它可能与跨越130-160公里，大大大于一些最大的彗星。研究该对象还可以使研究人员在太阳系中的神秘过程中洞察。

作为C / 2014 UN271编目的，游荡访客被DES项目捕获的档案数据中发现，通过拍摄遥远的星系来调查暗能的宇宙学神秘。

“对象出现在[大约] 80,000中出现了大约30张图片，该调查六年以上，”宾夕法尼亚大学的Gary Bernstein“，他们共同发现C / 2014 UN271以及博士学生佩尔尼纳迪尼尔，告诉物理世界。 “Pedro立即将其作为特殊利益被认为是特别兴趣的，因为它的近似逃逸轨道和首次看到的远距离，”他补充道。

C / 2014 UN271似乎已经来自一个地区，从阳光远远超过地球。这是在oort云中 - 一个巨大的冰冷物壳，认为包围整个太阳系。在其当前的轨迹上，该物体需要近550万年来完成其轨道的一个循环。

物体可能像彗星一样富有冰，目前距离太阳约有30亿公里。它将达到其最接近的点，称为Perihelion，在2031年。当时，它将位于太阳系的平面下方，靠近土星轨道的轨道。

在C / 2014 UN271中的部分兴趣是它可能是一个过渡对象的东西。天文学家认为，许多长期彗星，偶尔出现在明亮的尾巴中，实际上来自oort云。恒星在阳光附近徘徊可以从他们的位置和千禧年向内推动这些物体，他们向内工作，巨人行星的重力在每次访问中调整他们的路径，直到他们居住在我们今天看到的地方。

“[C / 2014 UN271]距离太阳距离太阳有一个渐晕可能会告诉我们，这是几次这样做，但仍然是我们所知道和爱的那些长期彗星的过程。 ，“Meg Schwamb解释了Queen大学贝尔法斯特的Kuiper Belt和Oort云专家。

科学家们将热衷于看看C / 2014 UN271如何在漫长的旅程中表现如何。例如，它会在活动中展示爆发，甚至崩溃？ Vera C Rubin Observatory由于开始于2023年从智利开展地面调查，应该是在冠军位置来记录发生的事情。 “随着鲁宾天文台的每三个晚上，我们将获得一部有效的电影，这对该物品在十年内发生了有效的电影，”Schwamb说。

最初，C / 2014 UN271似乎没有覆盖云，称为昏迷状态。然而，使用LAS Cumbres天文台网络的天文学家在6月22日在Astronomer的电报公告中宣布，他们已经在南非的大望远镜中检测到这样的功能。昏迷的出现可能是从身体表面升华的冰切的结果。

“涉及一个复杂的过程，还有很多尚未理解的过程，这是从oort云上的轨道上的最大的小世界，我们看到我们经历了这项活动的初创活动，”米歇尔·班尼斯特说，基于坎特伯雷大学的外太阳系统专家。

一些研究人员建议C / 2014 UN271可以成为样品返回任务的目的地。然而，肯特大学的彗星专家斯蒂芬·洛瑞说：“没有足够的时间。有几十年来发展这样的航天器使命。“他补充说，C / 2014 UN271也可能与欧洲即将到来的彗星拦截机关抵达，这是距离地球轨道的距离。

尽管如此，天文学家可能会因为C / 2014 UN271悲伤而靠近太阳的蠕动很大。 “更多的热量应该变得更多的活动，但它的白天仍然远离太阳，超越土星的轨道，”潘坎特说。 “看看未来几年的价格会令人兴奋。”
