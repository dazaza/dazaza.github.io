Websnapr - Generate website screenshots online

by yava555 on 2008-04-06 08:59:48

Websnapr is a tool that generates website screenshots online, and it also allows you to add the generated website screenshots to your own site. By using the plugin provided by Websnapr, you can achieve the effect of automatically displaying the corresponding website screenshot when you hover over a link with your mouse. For this purpose, Websnapr provides plugins for some mainstream CMS platforms (such as WordPress, Drupal, PostNuke...), so you don't need to perform complex operations to implement this functionality.
