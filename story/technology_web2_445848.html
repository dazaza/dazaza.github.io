Linux Commands:

ls View file directories

Syntax: ls [options/parameters] to view path

-a View all, including hidden files

-l Long format display

- Regular file

d Directory

c Character device

b Block device

s Socket

p Pipe file

-rw------- 1 root root 1310 01-11 11:03 anaconda-ks.cfg

File type Permissions Hard link Owner Group

touch Create an empty file or update file date

rm Delete a file

-r Delete directory

-f Force delete without prompt

mv Move files

-b Backup before overwrite

-f Force overwrite without prompt

-i Prompt before overwrite

-u Do not overwrite if destination file is newer than source

Path Representation:

./ Current directory

~ User's home directory

~username Specified user's home directory

scp Network copy

scp Source address Destination address

Example: Copy /soft/xxx.rpm from machine 10.1.1.128 to local /soft/

scp 10.1.1.128:/soft/xxx.rpm /soft/

scp /soft/rhel-5.3-server-i386-dvd.iso 10.1.1.128:/soft/

cp Local copy

cp Source address Destination address

ifconfig Check IP

Install virtual machine:

rpm -ivh Virtual machine software filename

Get help: man info --help

cat View file content

cd Directory switch

Syntax: cd Target path

cd Switch to your home directory

cd - Return to the previous directory

cd .. Return to the parent directory

mkdir Create directory

Syntax: mkdir Directory name

-p Automatically create parent directories if they do not exist

-m Specify permissions when creating directory (e.g., mkdir -m 755 /root/test2)

vi/vim Usage

Modes of use: Command mode (bottom left blank), Insert mode (bottom left shows "-- Insert --"), End-line mode (bottom left shows ":")

ESC Switch to command mode

I(i) O(O) Switch from command mode to insert mode

: Switch from command mode to end-line mode

:w Save

:wq Save and exit

:q Exit

:q! Force exit without saving

yy Copy line

Nyy Copy N lines (N represents a number)

p Paste

u Undo

:e! Discard all changes and re-edit

useradd Create user

Syntax: useradd [options] Username

-c Add user notes information

-d Force change user home directory location

-g Specify main group for user

-G Specify secondary group for user

-u Specify user ID

-s Specify shell

passwd Set user password

Syntax: passwd [options] [Username]

-l Lock user

-u Unlock

-S View password status

-d Delete password

Actions during user creation:

1. Add one entry in /etc/passwd

2. Add one entry in /etc/shadow

3. Add one entry in /etc/group

4. Add one entry in /etc/gshadow

5. Create a home directory for the user under /home and copy all files from /etc/skel/

6. Create a mail directory for the user under /var/spool/mail/

usermod Modify user information

Syntax: usermod [options] Username

-c

-d

-g

-G

-L Lock user

-U Unlock user

-s Modify user shell

-u Modify UID

su Switch users within current shell

Syntax: su [-] Username

- Switch user while changing user environment

Exercise: Add a user bean, set password expiration cycle to 30 days, warn 5 days before expiration,

allow login for 3 days after expiration, permanently lock if password is not changed within these 3 days.

userdel Delete user

Syntax: userdel [options] Username

-r Delete user along with their home directory

groupadd Create new group

Syntax: groupadd [options] Group name

-g Specify group ID

-r Specify group ID less than 500

gpasswd Manage groups

-a Add user to group

-d Remove group member

No option sets group password

newgrp Login to group

groups View current user’s groups

groupdel Delete group

id Show user and group information

users, who, w View currently logged-in users

write Send messages to specified online users

wall Send message to all users

alias Show alias settings

unalias Delete alias

last View login information

lastb View failed login information

lastlog View last login time of users

Permission Management

Permission Classification

View Permission

chmod Set permission

Object representation:

u Owner

g Group

o Other users

a All users (u+g+o)

Action representation:

+ Add permission

- Remove permission

= Assign permission

Permission representation:

r(4) Read

w(2) Write

x(1) Execute

-R Apply to directory and its subdirectories

Sticky bit (Privileged bit):

u's s(suid) bit Execute as owner's identity

g's s(sgid) bit Execute as group's identity

o's t File can only be deleted by owner

chown Modify file/directory ownership

Syntax: chown Username:Groupname Filename/DirectoryName

Exercise:

As root, create a directory test2 in /, copy files starting with p from etc to it.

Allow other users (non-root) to create and delete files, but only delete their own files.

Create a directory test3 in /, allow tom read/write/execute, smith read/write but no execute,

and mary only read. Verify by whether they can create/delete files.

chattr Set file attributes

Syntax: chattr [options] Filename

i No modification allowed on file

a Only append allowed

Use + before options to set, - to cancel

lsattr View file attributes

System Management

1. hostname View host name

Can also temporarily modify; for permanent modification, write to configuration file

/etc/sysconfig/network

2. uname Display system information

-a All information

-r Kernel version

-s Kernel name

3. cal Calendar

4. file View file type

5. ssh Secure remote login

Syntax: ssh username@ip

-X Enable X11 forwarding (x-window)

-x Disable X11 forwarding

Also possible to remotely invoke programs on specified machines

ssh username@ip command

6. init 6 Restart

init 0 Shutdown

shutdown -h now Shutdown

shutdown -r now Restart

7. date View/set time

Set display format: date +%Y-%m-%d

Network Management Commands

1. ifconfig View/modify IP

For permanent changes, modify configuration file

/etc/sysconfig/network-scripts/ifcfg-eth0

After modifying, restart network service with

service network restart

Disable network card: ifconfig eth0 down

Enable network card: ifconfig eth0 up

2. route View routing table

Set default gateway:

route add default gw 10.1.1.1

3. Set DNS

Modify configuration file /etc/resolv.conf

4. Network Service Management

service network start Start service

service network stop Stop service

service network restart Restart service

service network status View service status
