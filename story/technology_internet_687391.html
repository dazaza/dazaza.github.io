Detailed Tutorial for Moving a DEDE Website

by qingian007 on 2012-03-02 15:31:20

When building a website with DEDE, I purchased a hosting space that was very slow, so I decided to switch to a new space. However, transferring the data proved to be quite challenging and required some technical expertise. With the help of an expert, I finally managed to successfully transfer everything. Below are the detailed steps:

1. Log in to your DEDECMS backend. In the [System] section, find [Database Backup/Restore] and perform a [Backup] of the database.

2. Download all files from the old hosting space to your local computer.

3. Upload these files to the new hosting space.

4. Install the DEDECMS system on the new hosting space.

**Method for installing DEDECMS on the new hosting space**: Download a fresh installation package from the official Dedecms website. Upload the contents of the [install] folder from the installation package to the new hosting space, overwriting any existing files. Then, run http://yourdomain/install to start the installation process. Note: The database table prefix must be the same as the original database table prefix (you can use the default settings).

**Note**: Do not select the option to "Install sample data" (this sample data includes examples of most DedeCMS functionalities, but it is unnecessary for this migration).

5. After the installation is complete, log in to the backend. Click on [System Management] —— [Database Backup/Restore] —— [Data Restore]. Select all discovered backup files. (Note: Do not select the option to "Restore structure information" in the additional parameters.)

6. Click "Start restoring data". Wait a moment, and the data will be restored.

7. Enter the system —— [System Basic Parameters], and reconfigure the site settings.

8. Finally, regenerate the entire site, and you're done.
