疑似Android 4.4 screenshot exposure, rumored to be released in mid-October.

by anonymous on 2013-10-01 20:42:54

The new Android 4.4 KitKat system has recently been a hot topic, but so far, not much information about the features of the new system version has been disclosed. Now, a screenshot leaked online by Google Korea brings us more information about the Android 4.4 KitKat system, including changes in the color scheme of the status bar and debugging icons, which may imply that there will be significant changes to the UI interface in the new system version.

Screenshot Exposure

This exposed screenshot reportedly came from Google Korea when they were showcasing their latest Korean input method, accidentally leaking the new interface of Android 4.4. From the newly exposed interface, the main color scheme of the Android 4.4 system's status bar is gray, which differs significantly from the blue of the previous Jelly Bean (Android 4.3 and other versions) system.

However, it is worth noting that the debugging icon of the new system has also changed. Surprisingly, the engineering mode icon in the top-left corner of the screenshot has changed from the original Jelly Bean to an icon resembling a lime pie cake. Therefore, it seems that the version number of this screenshot may need further confirmation, whether it is Android 4.4 or even the real Android 5.0 version.

In Line with Some Rumors

Despite this, the exposed screenshot still confirms some rumors about the new version. According to previously disclosed information, Android 4.4 will make certain modifications to functions such as system management, notification bar, and gallery. The system will address file fragmentation issues, come with a new notification bar and default color scheme, as well as visualized gallery functionality. Clearly, changes like those in the notification bar and debugging icons have already confirmed the authenticity of the above claims.

Previously, Google Senior Vice President Sundar Pichai officially announced through Google+ and Twitter the codename "KitKat" for the next mobile system, Android 4.4. However, no more new features have been disclosed yet. According to Google's statement: "Use KitKat to create an excellent Android experience for everyone."

Release in Mid-October

According to reports from foreign media, the Android 4.4 KitKat system will be officially released on October 14th alongside the Nexus 5 manufactured by LG. Meanwhile, an employee of Google's partner Nestle revealed on Twitter that the Android 4.4 system will be launched in October, so it is expected that more features of the new version will be gradually exposed in the coming days.

According to South Korean media reports, the Google Nexus 5 running Android 4.4 will be a mid-range model, with specifications almost identical to the LG G2, featuring a 5-inch touchscreen, quad-core processor, and support for LTE networks.
