Usage Tips for Android 2.2 System (Part One)

by dream2012 on 2012-02-10 11:41:56

Google has been rolling out Android 2.2 for a while now, and major phone manufacturers have been releasing updates to Android 2.2 for their phones or even launching new phones preloaded with Android 2.2. An increasing number of users are experiencing the new features and smoother performance of the latest version of Android. As Android 2.2 becomes more widespread, it's time to educate users about the new features and usage tips of Android 2.2. We've compiled 20 of the most common and useful tips for using Android 2.2 to share with everyone, hoping to help Android 2.2 users better utilize their phones.

1. **Powerful Search Functionality and Detailed Search-Related Settings**

The search program in Android 2.2 not only supports cool features like voice search but also greatly increases the options that can be searched. There is a dedicated search setting in the phone settings where you can enable or disable web suggestions and search history for Google searches. The searchable items have expanded to cover the entire phone, including various files stored on the phone, as well as applications, calendar, contacts, email, music, and even data from individual applications.

2. **Multiple Search Widgets on the Screen**

If you use the search function frequently and have enough space on your screen, you can add multiple Google search widgets to your screen, such as one for web search, one for application search, and another for contact search.

3. **A More Convenient and Thoughtful Keyboard Experience**

Input functionality is something every phone user uses often, but few users notice the improvements in the keyboard in Android 2.2. In previous versions of Android, entering numbers or symbols required switching to another screen, but in Android 2.2, numbers and related symbols are placed next to letters on the keyboard. For example, the number 1 is next to the letter Q, and the @ symbol is next to the letter S. When entering characters, if you need to input a number or symbol, just long-press the corresponding letter. For instance, when entering an email address and needing the @ symbol, just long-press the letter S, which is very convenient and quick.

4. **Use the Power Button to End Calls**

In the Android 2.2 settings under accessibility, there is an option to set the power button to end calls, which is very convenient. However, this feature is not supported by all Android 2.2 phones; some manufacturers, like HTC, have removed this function from their phones.

5. **Enable Flash in the Phone Browser**

One of the selling points of Android 2.2 is its support for Adobe Flash 10.1 player. Phones like the Nexus One need to separately download and install Flash from the Android Market to use it, whereas manufacturers like HTC have already pre-installed Flash on their phones.
