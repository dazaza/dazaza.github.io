A small software that arranges the chat window in a tabbed manner like Maxthon. It no longer occupies your valuable taskbar space. It can be minimized to the tray area at the lower right corner of the screen; when a new message is detected, the tab will highlight, and you can choose to display it in a bubble; extremely low memory usage. Under Vista, it occupies about 2M of memory, and under XP, about 7M; supports the boss key; supports closing tabs with double-click/middle button/right button of the mouse; supports hotkeys and mouse wheel switching between chat tabs.

Update content for version 1.6:

New features:

Supports Feixin 2008

Supports TM2005

Supports tab dragging (dragging out of the window will separate the tab from the window)

Can set the text color of the tab

Can set the maximum length of the tab text

Can set whether to prompt the tab number when pressing Ctrl

Can set to display the chat window icon on the tab

Can set to start on boot

If the window has already been included in the tab, double-clicking the buddy list will automatically switch to the corresponding tab

Locked tabs will display an icon prompt

Hotkeys can use the Win key (not supported on Windows 2000)

When closing windows that are transferring files or emoticons, there will be a prompt (only for QQ)

When the mouse is in the chat window, the right button/Ctrl+wheel switches tabs. When the mouse is on the tab, the wheel directly switches tabs

When there are no chat windows, pressing the right button on the main interface will pop up a menu

Adjusted the skin of the tabs

No more flashing when closing chat windows

To accommodate older machines, there will be a 0.2-second delay when detecting new windows before including them in the tab
