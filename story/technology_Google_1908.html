Subscribe to restricted blogs with Google Reader

by kukusky on 2006-04-07 15:16:10

As we all know, Google's blogger blog site is restricted in China due to one reason or another. Although we can open the main page of blogger, we can't access the generated blog site, while Google's official blog is basically placed on blogger. So domestic users can basically only access it through proxies, which is really very troublesome. Now there is a way that you can easily view the articles on these blogs, in fact, the principle is very simple, because blogs have RSS output, and basically Google blogs are full text output, so as long as you know their FEED address, you can subscribe through RSS reading software. Of course, which RSS reading software to use is also very important, because if you use a Reader installed in the local, due to limitations, still can not access, so we can use Google Reader, because Google Reader is a web-based RSS reader, in the update time is through Google's server to carry out. It is equivalent to playing a transit role, and with the help of Google's powerful search function, we can easily find what we want to see.
