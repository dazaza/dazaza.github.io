八年后，Microsoft 还是领头羊和垄断者吗？

Microsoft 仍然是众多反托拉斯诉讼的对象，但这个软件巨人已经变得如此虚弱，以至于不能继续在技术市场叱诧！

八年前的本周，华盛顿D.C的法官 Thomas Penfield Jackson 职务上宣告微软是一个垄断企业。哪些日子，微软仍在斗争，但在欧洲的主要案件中输掉。当 Jackson 宣告时，它几乎不再说是全能巨人了。

因此，微软仍是一个使用其到处被使用的操作系统而挤走竞争的垄断者吗？或者是一个无齿的巨人，面对其竞争者 Google 和 Apple 的激烈竞争？

首先，我必须说我不是律师，我也没有和律师谈论过这个话题。对于这个国家Redmond的力量来说，我对微软是否违反了反托拉斯法的狭义法律案例更感兴趣索然。

PC市场

你可能当然会辩论说在一个地区，反托拉斯诉讼所称的——微软声称 Windows 操作系统的需求的使用去影响PC卖主在其机子中包含的系统——微软仍然起着巨大的作用。当然，Dell 销售带有 Linux 的 PC，惠普，联想和其他一些厂商也这样做。但那些非 Windows 系统的销售还是微不足道的。虽然苹果带有 Mac 笔记本和 PC 在上季度的销售比去年同期增长了34％，但这并不意味着惠普，Dell 或其他厂商也会有同样结果。他们的生计还是和销售 Windows 是捆绑在一起的。

不管是因为反托拉斯诉讼，还是因为其他市场因素，微软不在能继续使用其力量来把其竞争对手逐出市场。恰当的事实是 Dell 正销售 Linux 系统的机子，这是一个根本的例子。近来我买了一个台式电脑以供家用，当这台电脑启动时，开始加载来自于微软的最强大的两个对手的软件——Google 的桌面搜索服务和 Mozilla 的 Firefox。

Windows 正在衰落？

一些趋势表明 Windows 的力量在接下来的几年里会下降。苹果的最新美洲虎操作系统的多数评论家表示其优越于 Windows Vista。当虚拟产品如 Parallels 开始在 Mac 上运行 Windows 软件，至少，从个体来说，很可能已经不那么依赖 Windows 平台了。

长期来看，更强的对 Windows 的威胁来自网络本身。我们的大量工作——通过e-mail发送文件，查询信息，组织生活，在线产生文档、电子表格和其他文件。网络成为影响每个人的操作系统，任何特殊的东西在你坐在奔跑面前的PC面前变得越来越不重要了。
