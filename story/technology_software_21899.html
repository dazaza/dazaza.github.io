Using Windows Mobile Device Center, you can establish new partnerships, synchronize music, pictures, and videos with Windows Mobile devices (Windows Mobile 2003 or later versions), and manage them. Windows Mobile Device Center is closely integrated with an efficient business data synchronization platform, providing a refreshing user experience. Windows Mobile Device Center helps you quickly establish new partnerships, synchronize important business information (such as emails, contacts, and calendar appointments), easily manage synchronization settings, and transfer business documents between the device and the PC.

This new version of Windows Mobile Device Center includes many major improvements and new features that support Windows Mobile 6 devices. Only Windows Vista supports Windows Mobile Device Center 6.1.

The main updates in Windows Mobile Device Center 6.1 include:

- Windows Mobile 6 feature support

- Information Rights Management Activation – Automatically configure Windows Mobile 6 devices to open IRM-protected documents and files.

- HTML Email – Set up Windows Mobile 6 devices to synchronize emails in HTML format.

- Certificate Enrollment – Obtain certificates via the PC currently connected to the Windows Mobile 6 device.

- Allow data connections on Windows Mobile 6 devices when connected to a PC.

- File Synchronization for Smartphones – Synchronize files with Windows Mobile 6 devices, including touch screen and non-touch screen devices.

- Automatic Device Authentication – Connect Windows Mobile devices to PCs without entering the device lock PIN code each time.

- Product Registration – Register Windows Mobile devices and obtain available information and services for the device.

Windows Mobile Device Center 6.1 is fully compatible with Windows Mobile 2003 devices and higher versions.
