Register Total Commander, don't think too much!

by zouxiz on 2007-06-10 18:35:04

I haven't been using Total Commander for long, but I can't live without it now.

By chance, I found the download for Total Commander 6.56 online. The comments underneath were endless, all praising how amazing TC was. Was it really as great as they claimed? With a speculative mindset, I downloaded TC 6.56, which was the trial version. The startup screen was quite unique, and the English interface initially left me unable to find the program's entry point. After reading carefully, I realized that I had to click the indicated button to start the trial. Each time it started, one of the buttons among the prompted 1, 2, 3 would be different. I couldn't help but admire how thoughtful the author was.

When using someone else's work, we should show at least basic respect. My way wasn't buying a registration code. For those who love 0day, visiting the software (or author's) official website to understand who created the software is a sign of respect for the original creator—just like citing an article requires mentioning the source or linking it.

I discovered that TC version 7 had already reached RC4 stage. It seems I've missed out on a lot of good things in my daily life.

A few days later, the 0day community released a cracked version of RC4. I downloaded it to try it out, but found that the included registration key file could not be imported into the system registry. I thought to myself, "Is this even a real crack?"

A few more days passed, and RC5 was also out. The same cracked key still couldn't be imported into the registry. I began to wonder why cracks didn't use a keygen and instead relied on these keys. What principle were they based on?

The next day, I saw the announcement that Total Commander 7 Final had been released. The power of 0day was truly impressive; the next day, there was already a cracked version available online. Besides the original installation files, it was still the familiar key-cracked file. With a glimmer of hope, I double-clicked the key, only to be disappointed again when the system prompted, "…unable to import…."

At this point, I thought, "Never mind! It’s just one extra click during startup." I downloaded the simplified Chinese language pack and continued using it without issue. However, fans of 0day probably couldn't tolerate the "NOT REGISTER" reminder in the title bar, right? Unfortunately…

The new version of TC has a more aesthetically pleasing interface, with icon designs that feel more generous compared to version 6. It's said improvements have been made in built-in compression programs like RAR. TC is indeed very useful, disrupting Explorer habits in some ways and surpassing Explorer in speed!

Today, while browsing the school BBS, the software section is one of my personal favorite sections. Suddenly, I saw someone asking about the registration method for Total Commander, also mentioning that the key file couldn’t be imported. The response below surprised me: the key file doesn't need to be executed; it just needs to be copied directly into the TC installation directory.

Like finding a treasure.

I tried it, and sure enough...

OMG, I’ve been fooled!
