Windows System Services: An Extremely Detailed Guide

by xloong on 2007-10-22 10:45:28

For the Windows 2000/XP that we often use, there are many services in it. So what do these services do? Which ones do we need? And which ones don't we need? Here, I would like to give everyone an introduction...

Win32 services consist of three parts: service applications, service control programs, and service control managers. The service control manager maintains the service data in the registry, while the service control program is a module that controls the service application, acting as a bridge between the service application and the service manager. The service application is the main program of the service, which is the executable code for one or more services. We can find them in Control Panel - Administrative Tools - Services. You can modify their current status and startup method. There are three startup methods: "Automatic" means it starts when the computer boots up or when needed. "Manual" means you can open it with the "net start" command and close it with the "net stop" command at the command prompt. "Disabled" means it will not start before changing the startup method.

Among the numerous service programs, many depend on each other, so we cannot arbitrarily stop any particular service; otherwise, it may cause abnormal system behavior. However, some services are indeed useless to us and also consume our valuable system resources. In fact, many programs are unnecessary and can be turned off to save resources.

**Alert (Notifier)**

Notifies selected users and computer system administrators of alerts. If this service is stopped, programs using system management alerts will not receive notifications. All services dependent on it will fail to start. Generally, home computers do not need to send or receive computer system management alerts (Administrative Alerts), unless your computer is used in a local area network.

Dependency: Workstation

Suggestion: Disable

**Application Layer Gateway Service**

Provides support for third-party communication protocol plug-ins for Internet Connection Sharing and Internet Connection Firewall. If you do not use Internet Connection Sharing (ICS) to provide Internet access for multiple computers and Internet Connection Firewall (ICF) software, you can turn it off.

Dependency: Internet Connection Firewall (ICF) / Internet Connection Sharing (ICS)

Suggestion: Disable

**Application Management (Application Management)**

Starting from Win2000, a new and effective software management solution based on the msi file format was introduced -- the Application Management Component Service. Not only can it manage software installation and deletion, but it can also use this service to modify, repair existing applications, monitor file recovery, and troubleshoot basic failures through recovery.

Suggestion: Manual

**Automatic Updates**

Enables the download and installation of important Windows updates. If this service is disabled, you can manually update the operating system from the Windows Update website. Allows Windows to automatically connect in the background and check and download update patches from Microsoft Servers.

Suggestion: Disable

**Background Intelligent Transfer Service**

Transfers data using idle network bandwidth. Transmits data in the background via HTTP1.1, such as Windows Update working with this service.

Dependency: Remote Procedure Call (RPC) and Workstation

Suggestion: Disable

**ClipBook (Clipboard)**

Enables the clipboard viewer to store information and share it with remote computers. If this service is stopped, the clipboard viewer will not be able to share information with remote computers. Any service explicitly dependent on it will fail to start. Shares the information in the clipboard with other computers, which is generally unnecessary for home computers.

Dependency: Network DDE

Suggestion: Disable

**COM+ Event System (COM+ Event System)**

Supports "System Event Notification Service (SENS)", which allows events to be automatically distributed to COM components. If this service is stopped, SENS will close and will not provide login and logout notifications. Any service explicitly dependent on it will fail to start. Some programs may use COM+ components, such as BootVis's optimize system application, if the event check shows that DCOM is not enabled.

Dependency: Remote Procedure Call (RPC) and System Event Notification

Suggestion: Manual

**COM+ System Application**

Manages the configuration and tracking of COM+ components. If this service is stopped, most COM+ components will not run smoothly. Any service explicitly dependent on it will fail to start. If the event check shows that DCOM is not enabled.

Dependency: Remote Procedure Call (RPC)

Suggestion: Manual

**Computer Browser (Computer Browser)**

Maintains an updated list of computers on the network and provides this list to computers acting as browsers. If this service is stopped, this list will not be updated or maintained. If this service is disabled, all services dependent on it will fail to start. Home computers generally do not need it, unless your computer is used in a local area network. Is it necessary to open it on large local area networks?

Dependency: Server and Workstation

Suggestion: Disable

**Cryptographic Services**

Provides three management services: confirming Windows file certification; adding and removing trusted root certificates from this computer; and assisting in registering this computer to obtain certificates. If this service is stopped, these management services will not work correctly. Any service explicitly dependent on it will fail to start. Simply put, it's the WHQL (Windows Hardware Quality Lab) certification by Microsoft. If you use Automatic Updates, you may need this.

Dependency: Remote Procedure Call (RPC)

Suggestion: Manual

**DHCP Client (DHCP Client)**

Manages network settings by registering and updating IP addresses and DNS names. People using DSL/Cable, ICS, and IPSEC need this to specify dynamic IPs. If the system is not on any network or there is no DHCP service on the network, it can be set to disabled.

Dependency: AFD Network Support Environment, NetBT, SYMTDI, TCP/IP Protocol Driver, and NetBios over TCP/IP

Suggestion: Manual

**Distributed Link Tracking Client (Distributed Link Tracking Client)**

Maintains links between NTFS files on the computer or different computers within the network. For most users, it is virtually useless and can be turned off, except for special users.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Distributed Transaction Coordinator (Distributed Transaction Coordinator)**

Coordinates information transactions across multiple resource managers, such as databases, message queues, and file systems. If this service is stopped, these transactions will not occur. Any service explicitly dependent on it will fail to start. As mentioned above, it is generally not needed for home computers unless you enable Message Queuing.

Dependency: Remote Procedure Call (RPC) and Security Accounts Manager

Suggestion: Disable

**DNS Client (DNS Client)**

Resolves and quickly retrieves the domain name system (DNS) name of this computer. If this service is stopped, this computer will not be able to resolve DNS names and locate Active Directory domain controllers. All services dependent on it will fail to start. Additionally, IPSEC requires it.

Dependency: TCP/IP Protocol Driver

Suggestion: Manual

**Error Reporting Service**

Allows error reporting for services and applications running in non-standard environments. Microsoft's application error reporting service is not very useful for most users, especially for pirated users, so it is recommended to disable it.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Event Log (Event Log)**

Enables event messages issued by Windows-based programs and components to be viewed in the Event Viewer. This service cannot be stopped. It allows event messages to be displayed in the Event Viewer.

Dependency: Windows Management Instrumentation

Suggestion: Automatic

**Fast User Switching Compatibility**

Provides application management in a multi-user environment. Features like the switch user function in the logoff screen are generally not recommended to be stopped, otherwise many functions will not be realized.

Dependency: Terminal Services

Suggestion: Manual

**Help and Support**

A service provided by Microsoft for support documentation and help files. If this service is stopped, you will not be able to use Help and Support Center. All its dependent services will fail to start. If you do not use it, just turn it off. In reality, not many people need it unless there is a special requirement, so it is recommended to disable it.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Human Interface Device Access**

Enables generic input access to Human Interface Devices (HIDs). HID devices initiate and maintain predefined shortcut buttons on keyboards, remote controls, and other multimedia devices. If this service is stopped, the shortcut buttons controlled by this service will no longer function. Any service explicitly dependent on it will fail to start. If there are no HID devices, it can be disabled.

Dependency: Remote Procedure Call (RPC)

Suggestion: Manual

**IMAPI CD-Burning COM Service**

Manages CD burning using the Image Mastering Applications Programming Interface (IMAPI). If this service is stopped, this computer will not be able to burn CDs. Any service explicitly dependent on it will fail to start. XP's integrated drag-and-drop burn feature on CD-R and CD-RW drives is not as good as dedicated burning software. Turning it off can speed up Nero's launch time. If you are not accustomed to using third-party software, keep it.

Suggestion: Disable

**Indexing Service (Indexing Service)**

Indexes content and file attributes on local and remote computers; provides fast file access through a flexible query language. Simply put, it can speed up search times, but I think few people search on remote computers unless they have special needs.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Internet Connection Firewall (ICF) / Internet Connection Sharing (ICS)**

Provides network address translation, addressing, and name resolution services for your home network or small office network, as well as protection against interference. If you do not use Internet Connection Sharing (ICS) or the built-in Internet Connection Firewall (ICF) in XP, you can turn it off.

Dependency: Application Layer Gateway Service, Network Connections, Network Location Awareness (NLA), Remote Access Connection Manager

Suggestion: Disable

**IPSEC Services (IP Security Services)**

Manages IP security policies and initiates ISAKMP/Oakley (IKE) and IP security drivers. Assists in protecting data transmitted over the network. IPSec is an important component that provides security in virtual private networks (VPNs), allowing organizations to securely transmit data over the Internet. It may be required in certain domains, but most general users do not need it.

Dependency: IPSEC driver, Remote Procedure Call (RPC), TCP/IP Protocol Driver

Suggestion: Manual

**Logical Disk Manager (Logical Disk Manager)**

Detects and monitors new hard disk disks and sends partition information to the logical disk management service for configuration. If this service is stopped, dynamic disk status and configuration information may become outdated. Any service explicitly dependent on it will fail to start. The disk manager is used to dynamically manage disks, such as displaying available disk space and using Microsoft Management Console (MMC) console functions.

Dependency: Plug and Play, Remote Procedure Call (RPC), Logical Disk Manager Administrative Service

Suggestion: Automatic

**Logical Disk Manager Administrative Service (Logical Disk Manager System Management Service)**

Configures hard disk disks and partitions. The service only performs configuration procedures and then stops. It is only used when using Microsoft Management Console (MMC) console functions.

Dependency: Plug and Play, Remote Procedure Call (RPC), Logical Disk Manager

Suggestion: Manual

**Messenger (Messenger Service)**

Transmits network transmissions and "Alerter" service short messages between clients and servers. This service is unrelated to Windows Messenger. If this service is stopped, Alerter messages will not be transmitted. All services dependent on it will fail to start. For example, the function of sending prompt information between networks, net send function, can be turned off if you do not want to be disturbed.

Dependency: NetBIOS Interface, Plug and Play, Remote Procedure Call (RPC), Workstation

Suggestion: Disable

**MS Software Shadow Copy Provider**

Manages software-based shadow copies obtained by the Volume Shadow Copy Service. If this service is stopped, software-based shadow copies cannot be managed. Any service explicitly dependent on it will fail to start. As mentioned above, it is used for backup purposes, such as MS Backup program, which requires this service, but most people do not need this feature.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Net Logon**

Supports pass-through authentication for account login events on computers on the network. General home computers are unlikely to use this service for logging into network reviews.

Dependency: Workstation

Suggestion: Disable

**NetMeeting Remote Desktop Sharing (NetMeeting Remote Desktop Sharing)**

Allows authorized users to remotely access this computer through NetMeeting over the company's internal LAN. If this service is stopped, the remote desktop sharing feature will not be usable. Any service dependent on it will fail to start. As mentioned above, it allows users to share control of their computer with other users on the network or the Internet. If you value security and do not want to open additional backdoors, turn it off.

Suggestion: Disable

**Network Connections (Network Connections)**

Manages objects in the Network and Dial-up Connections folder, where you can view local area network and remote connections. Controls your network connections.

Dependency: Remote Procedure Call (RPC), Internet Connection Firewall (ICF) / Internet Connection Sharing (ICS)

Suggestion: Manual

**Network DDE (Network DDE)**

Provides network transmission and security for Dynamic Data Exchange (DDE) between programs running on the same or different computers. If this service is stopped, DDE transmission and security will not be available. Any service explicitly dependent on it will fail to start. Most people seem not to need it.

Dependency: Network DDE DSDM, ClipBook

Suggestion: Disable

**Network DDE DSDM (Network DDE DSDM)**

Informs about Dynamic Data Exchange (DDE) network sharing. If this service is stopped, DDE network sharing will not be available. Any service explicitly dependent on it will fail to start. Most people seem not to need it.

Dependency: Network DDE

Suggestion: Disable

**Network Location Awareness (NLA)**

Collects and stores network configuration and location information, and notifies applications when this information changes. If you do not use ICF and ICS, you can turn it off.

Dependency: AFD Network Support Environment, TCP/IP Procotol Driver, Internet Connection Firewall (ICF) / Internet Connection Sharing (ICS)

Suggestion: Disable

**NT LM Security Support Provider (NTLM Security Support Provider)**

Provides security for remote procedure call (RPC) programs that do not use named pipe transmission. If you do not use Message Queuing or Telnet Server, you can turn it off. General users do not need it either.

Dependency: Telnet

Suggestion: Disable

**Performance Logs and Alerts (Performance Logs and Alerts)**

Collects performance data from local or remote computers based on pre-set parameters, and writes the data to logs. If this service is stopped, performance information will not be collected. Any service explicitly dependent on it will fail to start. This service has little value.

Suggestion: Disable

**Plug and Play (Plug and Play)**

Enables the computer to identify and adapt to hardware changes with little or no user input. Stopping or disabling this service will lead to system instability. It is the PNP environment, which is generally needed in computers, so do not turn it off.

Dependency: Logical Disk Manager, Logical Disk Manager Administrative Service, Messenger, Smart Card, Telephony, Windows Audio

Suggestion: Automatic

**Portable Media Serial Number**

Retrieves the serial number of any portable music player connected to your computer. This service has little value, so consider turning it off.

Suggestion: Disable

**Print Spooler (Print Spooler)**

Loads files into memory for later printing. It optimizes printing and helps with printing functions. If you do not have a printer, you can turn it off.

Dependency: Remote Procedure Call (RPC)

Suggestion: Disable

**Protected Storage (Protected Storage)**

Provides a protected storage area to store important data, preventing unauthorized services, processes, or users from accessing it. It is used to store passwords on your computer, such as Outlook, dial-up programs, and other applications.

Dependency: Remote Procedure Call (RPC)

Suggestion: Automatic

**QoS RSVP (QoS Admission Control, RSVP)**

Provides network signaling and zone traffic control installation features for QoS-aware programs and control applets. It reserves 20% bandwidth. If your network card does not support 802.1p or there is no ACS server on your computer's network, turn it off.

Dependency: AFD Network Support Environment, TCP/IP Procotol Driver, Remote Procedure Call (RPC)

Suggestion: Disable
