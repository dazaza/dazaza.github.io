1. What is BlackBerry?

The BlackBerry is a mobile email system terminal introduced by the Canadian company RIM (Research In Motion). Its distinguishing feature is support for push email, mobile telephony, text messaging, internet faxing, web browsing, and other wireless information services.

"BlackBerry" technically refers to a mobile email system that operates in a two-way pager mode, compatible with existing wireless data links. It was introduced in 1998. RIM's brand strategy consultants named it BlackBerry because the small standard English keyboard keys, which are black, resemble the seeds on the surface of a strawberry.

BlackBerry syncs seamlessly with desktop PCs. It can automatically forward Outlook emails to the BlackBerry device. However, when sending emails via BlackBerry, it automatically adds the phrase "Sent from my BlackBerry" at the end of the email.

BlackBerry Enterprise Solution is a leading wireless solution enabling mobile professionals to access information needed for interacting with clients, colleagues, and business operations. This proven platform provides secure wireless connections to a wide range of business information and communications for mobile users worldwide.

Email - With BlackBerry Secure Wireless Extension, mobile business users can easily manage their corporate email accounts even when they're outside the office. Users can send, receive, archive, delete emails, and read email attachments while traveling. Supported formats include Microsoft Word, Microsoft Excel, Microsoft PowerPoint, Adobe PDF, Corel WordPerfect, HTML, and ASCII. The "always-on" push technology of the BlackBerry solution automatically delivers emails without requiring any user action.

Enterprise Data - BlackBerry uses proven technology identical to the email push delivery architecture to provide immediate access to application and system data such as customer details, pricing data, order information, and inventory updates for mobile business users.

Phone - BlackBerry wireless handheld devices include built-in phones supporting voice services and features like call waiting, call forwarding, and conference calls. To make or receive calls, users simply bring the device close to their ear or use an optional headset to take notes or record important information during the call.

Personal Assistant - BlackBerry allows users wireless access to various personal assistant applications and synchronizes PIM (Personal Information Management) information wirelessly on the handheld device. Using BlackBerry, mobile users can access updated calendar, address book, task, and notepad information throughout the day.

Internet and Intranet Access - BlackBerry allows users wireless access to the internet and intranets, giving them more time for promotions and customer service.

Text Messaging - BlackBerry supports SMS (Short Message Service), allowing communication with other SMS-enabled devices and phones.

3. What Are Hole-Punched Machines, Flying Wire Machines, Hard Unlocked Machines, Soft Unlocked Machines, 14-Day Machines, and Original Machines?

After understanding BlackBerry's origin and functions, you're probably ready to make a purchase. However, you must be cautious. Since our country hasn't officially introduced BlackBerry yet, the current market is exceptionally chaotic, filled with various types of products. Therefore, it's crucial to understand their meanings to avoid being deceived. Below are the common types of machines listed by Smartphones.net for your careful identification.

Hole-Punched Machine

There are two explanations. One involves defective motherboards. Most watered-down goods in the market are international carrier-customized machines. Carriers buy in bulk at prices much lower than the market price, and manufacturers don't bear after-sales responsibilities, which are handled by carriers. Due to high labor costs abroad, repairing some machines costs nearly as much as buying a new one, so they are mostly scrapped, which is one kind of hole-punching.

Another type is customs hole-punching, usually done in inconsequential places, similar to some hole-punched records.

Flying Wire Machine

Due to certain reasons causing partial circuit problems on the motherboard, vendors manually connect broken circuits using wires or metal threads rather than printed circuits on PCB boards, hence the name flying wire machine.

Hard Unlocked Machine

Since machines flowing back from abroad have network locks and cannot be used domestically, skilled technicians replace the original motherboard font library by heating it with a blowtorch.

Soft Unlocked Machine

Unlocking the network lock by entering a software code.

14-Day Machine

In foreign countries, network operators provide a service where you activate a number within the network, sign a usage agreement, pay a fixed monthly fee, fulfill a certain number of years, and you can own the phone, with a fourteen-day trial period. If you're dissatisfied, you can cancel the agreement and return the phone to the network operator. Such phones are called 14-day machines.

Original Machine

As the name suggests, it's a brand-new original product.

4. What is Three Code Unification?

In the early days, three-code unification meant the IMEI codes on the outer box, inside the phone, and inside the battery compartment were the same, all being the same number.

Outer Box IMEI Code

This is straightforward and visible; no further explanation is needed. Many boxes now lack this number.

Inner Phone IMEI Code

Retrieval Method: First method: Input *#06# in the dialing state, and the phone will display a code, which is the inner IMEI code. Second method: Under "Settings," there's a "Status," and opening "Status" will reveal this IMEI code.

Battery Compartment IMEI Code

First, turn off the phone, open the battery cover, remove the battery, and find an IMEI code on a white sticker inside the battery compartment.

Now, due to the complexity of equipment and BlackBerry's unique PIN code, the current three-code unification consists of IMEI code, PIN code, and BT-MAC code (Bluetooth code), which must match between the battery compartment and inside the phone.

IMEI code retrieval methods are as described above.

PIN code: Under "Settings," there's a "Status," and opening "Status" will reveal this PIN code.

BT-MAC code: Enable Bluetooth connection, and under "Options," there's a "Device Address," which is the Bluetooth code.
