20 Excellent Web Apps as Desktop Software Alternatives

by geekzhang on 2011-01-06 09:30:40

In this article, I will introduce 20 excellent web applications that can replace desktop software. I hope you like them!

1. Google Docs

Google Docs is a free online Office suite provided by Google that can perform most of the functions of desktop word processors, Excel, and PowerPoint.

Can Replace:

• Microsoft Office

• Open Office

2. Meebo

Meebo is an Ajax-based online instant messaging software that is compatible with major instant messaging platforms such as AOL, ICQ, Yahoo! Messenger, MSN Messenger, Jabber, and Google Talk. It allows users to stay connected without needing to install any instant messaging software.

Can Replace:

• Facebook Chat

• Yahoo Messenger

• AIM

• Windows Live Messenger

3. Slide Rocket

Slide Rocket is a web application focused on slide presentation documents. Unlike other Office 2.0 tools like PreZentit and Preezo that use AJAX, Slide Rocket's biggest feature is its abandonment of AJAX in favor of Adobe Flex + ASP.NET technology, which fully structures this excellent app. The stunning user interface and comprehensive feature settings leave people in awe.

Can Replace:

• Microsoft PowerPoint

• OpenOffice Impress

4. Aviary

Aviary is a website that integrates a very convenient set of online applications. Its main applications include image and audio processing features.

Can Replace:

• Adobe Photoshop

• Adobe Illustrator

• Soundbooth

5. Mint

It is a free online personal finance management tool. It will help you manage your financial accounts and set up your budget needs.

Can Replace:

• Quicken

6. JayCut

JayCut is an online media mixing and video editing tool similar to JumpCut. It offers unlimited upload space for users to upload images, videos, and audio content.

Can Replace:

• Windows Movie Maker

• Ulead

• Other Video Editing Software

7. WobZip

WobZip is a free online decompression tool. It has been around for quite some time and is well-known abroad.

Can Replace:

• Unzipping Softwares

8. Zamzar

It can help you convert various music, video, image, and document formats. The conversion process is completed online, and the results will be sent to the email address specified by the user, eliminating the need for local computation.

Can Replace:

• File Conversion Software

9. Hulu

Hulu presents an endless treasure trove of TV shows through its website. The quality of its online playback is also pretty good. It’s an easy-to-use video player that operates much like a VCR. The powerful search engine on the site can help you find specific actors or programs.

Can Replace:

• TV Tuner Software

10. TokBox

TokBox is a video chat software based on the Web platform. It provides a video chat widget tool that can be embedded in blogs. Like other widget services, it can be easily installed through code calls. TokBox not only offers real-time chat functionality but also provides video messaging capabilities.

Can Replace:

• Video Chat

11. Moof

Moof is an online network streaming media. It's like gathering all your favorite music in one place. One of its biggest features is the ability to export your iTunes library and place it on the web, so you never miss your favorite songs.

Can Replace:

• iTunes

• Music Streaming Software
