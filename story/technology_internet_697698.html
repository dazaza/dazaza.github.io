Facebook Releases Windows Version of Messenger Desktop Chat Client

Tech Sina, March 6 - Facebook officially launched the Windows version of the Facebook Messenger desktop chat client on Monday. Facebook Messenger is an instant messaging software, and the download link for the Windows client was leaked back in December.

Facebook Messenger allows users to chat continuously without interruption, receive notifications, and read desktop information streams. The Windows 7 version of Facebook Messenger can be downloaded here (http://www.facebook.com/about/messenger), and it will be rolled out to other Windows users over the next few weeks. The launch of this product signifies that Facebook's current focus will not overlook any engagement activities by existing users.

For Mac users, Facebook stated: "Mac users, we are busy developing a Mac version of Messenger, stay tuned."

Facebook's off-site Facebook chat application is also very popular, as the Microsoft Messenger Live website allows users to chat on Facebook, read updates from friends, and share. According to statistics from market research company AppData, Facebook chat is technically speaking the most popular Facebook application, with 22.4 million monthly active users and 19.1 million daily active users, achieving a user retention rate of 85.5%. Meanwhile, Messenger for Windows provides users with this chat functionality while avoiding distractions from other Microsoft features such as Hotmail and SkyDrive.

Facebook likes to test products with a small group of users before officially launching them globally, using this to observe user behavior patterns and adoption rates. They have even specifically designed "gatekeeper" software to filter administrators who view the code. When testing browser-based products, Facebook exercises full control, displaying and hiding functions at will for different users.
