Methods to make search engines quickly index specified web pages

by chinaaoyun080808 on 2008-06-03 23:59:35

An important method of acquiring traffic in website promotion is to create search sources to achieve high traffic. Here, I have summarized many methods from my predecessors and written this article, hoping to discuss with friends who are learning SEO like me.

The overall steps are:

1. Establish a keyword, preferably a person's name that currently has no search results on the network, such as "Liang Jiji" or similar names.

2. This keyword should be in the title position.

3. Get this ad page indexed by search engines.

4. Post some popular articles on large forums containing this keyword, highlighting the keyword to arouse others' curiosity, thereby prompting them to search for this keyword and ultimately direct traffic to our target ad page.

Throughout the process, an important link is how to get our keyword webpage indexed by search engines.

Currently, most people use blogs. The main reasons why keywords are not being indexed include:

1. People are too eager for quick results, hoping for indexing the day after they post.

2. Lack of outbound backlinks.

3. Using their own websites which were previously penalized by search engines.

So, how can we quickly create a large number of keywords?

1. Prepare 30 keywords at once, all of which cannot be found in search engine results.

2. Apply for 30 accounts each on Sina, Sohu, and Baidu Spaces.

3. Each keyword should simultaneously be used on Sohu's blog, Sina's, and Baidu Space's.

4. Use the same method to prepare these 30 keywords well.

5. Then put all these blog addresses on one webpage, and post this webpage address as a backlink on forums.

The purpose is to let the search engine index this list page. Once it is indexed, when we open a new blog, just posting the address on this list page will ensure it gets indexed the next day. Since this page is indexed, search engines will revisit it daily. Why use three platforms?

Because Google doesn't like Baidu Spaces, and Baidu doesn't like Sina and Sohu. Therefore, when we create three blogs for the same keyword, we can often occupy this keyword across various search engines simultaneously.

Beginner friends might not be very familiar with creating backlinks yet.

So, how do you create backlinks?

1. Register a username on bbs.163.com.

2. When you have posting permissions, click reply and quote, then behind the quote, there will be a “:”. Create a backlink for this “:”.

3. Initially, you can create more backlinks, for example, create 10, and every day persistently create a few more. This ensures that you always have new keywords available.

This is the first knowledge point we explain, about how to quickly get search engines to index our keyword pages.

Skill one: Choose multiple blogs to work on a single keyword simultaneously.

Skill two: Create backlinks on large forums.

Skill three: Use social bookmarking to create backlinks.

Among these three skills, which one allows our keyword pages to be indexed faster by search engines? Next, we will conduct an experiment. Here, I will simultaneously create four webpages:
