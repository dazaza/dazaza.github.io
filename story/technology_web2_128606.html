What does Chi-MP do and what does it do?

Chi-mp is a social network aggregation service website, which can aggregate some Web2.0 website services, equivalent to a Web2.0 application service platform website.

The biggest feature of Chi-mp is that it provides a top-level domain name based on the.mp suffix, and this domain name is free.

So, which region is the.mp domain?

The.mp domain name is the domain name of the Northern Mariana Islands country and Territory top-level domain (ccTLD), and the registration service of this domain name has not been carried out in China. Since it is a new domain name for a small area, there is still a lot of registration space based on the.mp suffix. This is how Chi-MP attracts a large number of users to its website services.

What services does Chi-MP support with its free domain name?

Chi-mp provides free top-level domain names based on the.mp suffix, which supports OpenID, which is the most basic. At the same time, Chi-MP aggregates some of the main web2.0 services, such as Gmail; Hotmail; Yahoo; Feed; Flickr; Twitter; Facebook, etc.

There are also some other easy-to-use features, such as getting contacts from Twitter and FaceBook, uploading your own website Icon (favicon), using Google Analytics and Google Webmaster tools, with a variety of themes to choose from. For the website export function.

How do I register for Chi-MP service?

Previously, the Chi-MP service was open for registration, but after the recent revision, Chi-MP closed the green channel entrance, and all users using the Chi-MP service must first get an invitation registration code (Need a beta code).

So how to get the invitation registration code of Chi-MP?

1, the most basic of course is to already have the Chi-MP invitation registration code of the user to ask for, generally speaking, Chi-MP users have 5 Chi-MP invitation registration code share.

2, go to the official website register, the address is Chi-mp /interested, maybe you will get one.

3, you can use the twitter search address (search.twitter.com) to search for Chi-mp, you may meet a kind person, get Chi-MP invitation registration code.

4. You can use Google Blog search (//blogsearch.google.com) to see who provides the invitation registration code for Chi-MP.
