#### **III. Short Answer Questions (10 points per question, total 50 points)**

1. **How is the dynamic IP domain name resolution system applied in the security field?**

Answer: What is dynamic IP? An increasing number of security systems connect to the Internet via various broadband lines to support remote monitoring over the Internet. To allow the system to be accessed by the monitoring end, the traditional method is to use a fixed IP address dedicated line connected to the Internet. However, fixed IP addresses and dedicated lines are very expensive, leading to high operating costs. With the widespread adoption of ADSL and other broadband networks, to reduce operating costs, systems supporting remote network access have switched to cheaper broadband networks. These networks often provide dynamic IP addresses. A dynamic IP address means that the IP address provided by the network service provider (such as telecommunications companies) changes periodically, and each time the user dials into the Internet, the assigned IP address is different from the previous one. To access a system with a changing IP address, a dynamic IP resolution domain name system can be used. The remote end only needs to know the Internet domain name of the accessed end, such as www.google.com, and can automatically find the current IP address of the accessed end through the Internet's domain name resolution mechanism.

2. **What is "false alarm"? How to overcome the impact of false alarms?**

Answer: During the installation of infrared alarms and fire alarm detectors, there may be situations where the detector triggers an alarm without any actual event occurring. This phenomenon is called a false alarm. To address this, we need to identify the cause of the false alarm—whether it is due to equipment quality, installation methods, environmental factors, or other issues—and solve them accordingly.

3. **What is Signal-to-Noise Ratio (SNR)?**

Answer: The ratio of the amplifier's output signal voltage to the simultaneously output noise voltage is called the signal-to-noise ratio of the amplifier, abbreviated as SNR. It is usually represented by the English symbol S/N, where S represents the image signal value of the camera assuming no noise, and N represents the noise value generated by the camera itself (such as thermal noise). The higher the SNR, the better. A higher SNR indicates fewer disturbances mixed in the signal, resulting in higher video quality. Conversely, lower quality results.

Answer: The commonly used "three defenses" refer to human defense, physical defense, and technical defense. The five major security systems in residential areas are: intercom, home alarm, perimeter defense, patrol, and monitoring.
