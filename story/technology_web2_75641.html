Wikipedia [Brief Review]: The most famous international Wiki website, positioned as a freely editable encyclopedia for everyone. [Operation]: It relies on numerous contributors to add and modify its entries. Since its creation in 2001, the English version has continuously expanded its content and now includes 480,000 entries, with many other language versions available (including a Chinese version, but the official Chinese site at http://zh.wikipedia.org is currently inaccessible).

Interactive Wiki [Brief Review]: A well-known domestic Wiki website, slogan--you and I jointly create a great encyclopedia. [Operation]: When creating an entry, you can choose categories, preset templates, tags (customizable), etc., with relatively complete page editing functions; for existing entries, you can collaborate (re-edit paragraph by paragraph or the entire article), collect, comment, export to Word documents, etc. Provides personalized Wiki space with custom domain names (supports customizable homepage content); the management backend has clear and comprehensive functions (provides note-taking, friend management, application for expert status, etc.). [Features]: Special sections like "Wei Bar" and groups; supports user points, level-based incentive mechanisms; Digg functionality supported on entry content pages.

Tianxia Wiki [Brief Review]: A domestic Wiki website built using standard Wiki software, slogan--a web knowledge base you can modify. [Operation]: You can create new entries by using the search function or observing empty links in existing entries; for existing entries, you can edit them or add them to a watchlist. The user settings page layout is simple and clear, with comprehensive and detailed functions. [Features]: Content involving website help, personal settings, etc., are presented in an editable Wiki format; materials related to Wiki technology are relatively abundant (click here to view). [Shortcomings]: The entire site's page display is abnormal, and many functional options are not fully localized into Chinese, giving a sense of disorder; the method for creating new entries is not convenient enough.

Network Heavenly Book [Brief Review]: A domestic Wiki website built using standard Wiki software, slogan--limitless online wisdom, jointly create a free heavenly book. [Operation]: You can use the search function or observe empty links in existing entries to create new ones; for existing entries, you can edit them or add them to a watchlist. The user settings page layout is simple and clear, with comprehensive and detailed functions. [Features]: Content involving website help, personal settings, etc., are presented in an editable Wiki format; the editing page provides style and template codes. [Shortcomings]: Some functional options on the site are not fully localized into Chinese; the method for creating new entries is not convenient.

Chinese Wiki [Brief Review]: A domestic Wiki website built using standard Wiki software. [Operation]: Similar functions to the first two sites, but new user registration has been suspended.

Baidu Baike [Brief Review]: The Wiki channel under Baidu, claiming to be--the world's largest Chinese encyclopedia. [Operation]: You can use the search function to create new entries (can set open categories, add images, reference materials, etc., requires approval); for existing entries, you can edit them (must specify the reason for modification), add them to Baidu Collections, comment, etc. The management backend allows you to view personal contributions and points. [Features]: Special sections like Baike Tasks, Baike Stars, Contribution Rankings, etc.; high degree of freedom in open categorization.

iAsk Open Dictionary [Brief Review]: The Wiki channel under Sina iAsk, claiming to be--the world's largest Chinese encyclopedia. [Operation]: You can conveniently create new entries using guided links on the home page (can set tags, etc.), with relatively complete page editing functions; for existing entries, you can supplement information (cannot modify the original editor's content), collect to ViVi, report, etc. You can view and display entries through the iAsk personal center homepage. [Features]: Flexible classification via tags. [Shortcomings]: Does not support editing of existing entries.
