Adobe to Launch Photoshop Software Based on PPC Platform

by yinyin on 2008-09-01 10:57:27

The renowned image processing software vendor, Adobe, the owner of Photoshop, will launch a version of Photoshop compatible with Windows Mobile. As internet content and search service providers like Yahoo and Google extend their services to the mobile phone sector, it is surprising that Adobe has taken the lead in image processing.

As we all know, Adobe's famous image processing software, Photoshop, has a large number of users worldwide. These users will surely be astonished when they use Photoshop on their Windows Mobile smartphones.

It is reported that the development plan for the mobile version of Photoshop will be released in October, with a beta version likely to be available next month. The first batch of supported phones includes the Motorola Q 9h, Q Music 9m, Samsung Blackjack I and II, and the Palm Treo 750w. The mobile version of Photoshop will support common editing operations such as rotation, cutting, and automatic correction. This software will be provided along with the storage space offered by Photoshop.com.
