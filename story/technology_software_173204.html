sShutdown is a system beautification software that replaces the computer shutdown dialog box. Windows XP is a great operating system, and for the system interface, boot screen, landing interface, we can beautify. Only look at the unchanging Windows XP system shutdown dialog box for a few years, always the same old look. sShutdown does that for us:

image

sShutdown has the following features:

* Blocks Windows shutdown messages, so the shutdown dialog box for sShutdown is displayed in any program under any circumstances

* Skin support (dialogs and buttons are drawn by you)

* Allow any image to be used as background (defined in skin)

* Supports the use of multiple effects on the background

* The shutdown dialog can be replaced by a specified third-party program

* Absolutely safe to use - no need to hack system files

* Support PNG image, support transparency effect

* Can be displayed in full screen (defined in skin)

The official website of sShutdown provides many beautiful skins to download, please click here to enter.

If you want to edit the skin yourself, you can use the official software sShutdown ThemeEditor.
