How to Convert Paragraph Text to Curves in Coreldraw? (5 Methods)

by lantonad on 2009-03-26 20:16:07

This issue mainly pertains to CorelDraw version 9. In CorelDraw version 11 and later, there are convenient shortcut keys available, so this problem basically does not exist.

Method One: For versions of CorelDraw prior to version 10, the "freeze" function in the filter can be used to solve the problem. Draw a completely black rectangle over the paragraph text, open the filter - transparency - 100%, and check the freeze and remove surface options. Once applied, you can then remove the text layer behind it, leaving only the curves.

Method Two: The above method may sometimes result in errors for texts with superscripts or subscripts in paragraphs, such as shrinking superscripts or subscripts. Comparatively, using the "output WMF file" to convert text to curves is more reliable.

Click on the menu "File - Output to WMF or AI format", then create a new blank file and import this WMF document, which will be in full curve format.

Method Three: In CorelDraw version 11 and later, Ctrl+Q can directly convert text to curves.

Note: If you need to print or open the same file on another computer, the text must be converted to curves; otherwise, either the text will not be visible, or it will appear as garbled code.

Also note that after converting text to curves and saving, the text cannot be edited anymore. So many beginners often ask if there's a way to re-edit text after it has been converted to curves. Let me tell you, it's impossible. Although OCR software can be used for text recognition, it is labor-intensive and time-consuming, and not recommended. The best method is to develop the habit of saving two file types while working: one in text format, and one in curve format, for future use. Or simply copy a portion of the text before converting it to curves and place it at the edge of the page. If any changes are needed, it's much easier to edit and re-convert than using OCR.

Of course, now that external hard drives are common, you can also use the pre-output preparation option in the file menu to package all fonts together with the file, saving a lot of time and effort. Alternatively, outputting as PDF can also avoid this trouble (details on the next page).

Below, we introduce some other methods collected online.

Method 1:

(1) Select the paragraph text you want to convert to curves, color is also possible.

(2) Choose menu - output, select EPS as the file type.

(3) Re-input this EPS file (PS file type with subtitle prn).

Method 2:

(1) Select the paragraph text you want to convert to curves.

(2) Use the text tool to select the text you want to output (so you can output the desired text).

(3) Output in any of the following formats: ai, wmf, etc. (AI has better color and no color bias).

(4) Re-output the file just output.

※ Small tip: When outputting curved text, double-click on the rectangle tool in the left toolbar, and the page will automatically generate a rectangle the same size as the page.

Output this rectangle along with the paragraph text in curve format, and when re-inputting, simply choose the correct file and press Enter to automatically align the page. Then delete the rectangle.

Method 3: Use printing to PS file to convert text to curves.

(1) Select the paragraph text you want to convert to curves.

(2) Choose print, (select standalone PostScript file as the printer).

(3) Print range selection 'Selected Area'.

(4) Click print to print the currently selected paragraph text into a PS file, then re-input this PS file.

Method 4: Use the filter freeze method to convert text to curves.

(1) Press Alt+F3 to open the 'Filter Dock Window'.

(2) Use the rectangle tool to draw a rectangle slightly larger than the paragraph text box, and fill it with the same color as the text.

(3) Select this rectangle, and choose 'Transparency' from the filter effects.

(4) Set the scale to 100%, check the freeze and remove surface options, and apply it.

(5) The converted text will automatically overlay on top of the paragraph text box.

※ This style of converted text will produce an invisible rectangle that needs to be manually deleted.

If there is a background behind the paragraph text box, the filter effect will also produce a background.

If it is a CMYK mode color block, the filter effect will also produce a CMYK color block.

If it is a CMYK bitmap, the filter effect will become an RGB mode bitmap.

If it is a black and white draft, you can use the PDF publishing method to convert to curves, but this has not been tested for colored drafts, out of concern for color distortion.

Method 5: Use the PDF publishing method to convert text to curves.

(1) Select the menu "File - PDF Publishing".

(2) Choose settings, and in the object section, select 'Output all files as curves', then confirm and save.

(3) Actually, in CorelDRAW 9, Chinese characters output as PDF will automatically convert to curves, but English characters will not. If you want English characters to also convert to curves, you need to select the option to output all text as curves.
