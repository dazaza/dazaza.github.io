Maxthon 2.1 Final Free Download

by xloong on 2008-05-08 19:36:01

After going through dozens of 2.0.x versions, Maxthon has ushered in a new round of upgrades - the first 2.1 version was born. The 2.1 series will significantly improve the usability of Maxthon's user interface and the stability of its use.

In Maxthon 2.1, the web toolbar has been redesigned. Now the web toolbar includes functions such as forward, back, font size, and web page encoding by default. To facilitate user usage, commonly used tools such as content control, smart form filling, and screen capture are placed on the right side of the web toolbar by default, accompanied by appropriate text descriptions.

The smart form filling is not displayed in the standard toolbar by default. Users who need it can right-click on the standard toolbar, select the smart form filling button in the "Add or Remove Buttons" of the pop-up menu.

Additionally, at the request of many users, the last opened list has been changed to an append mode. That is, if the last opened list page is not actively closed, the unopened links in the list will be retained for the next time, and will not disappear due to closing.

In addition to interface adjustments, this upgrade also fixes several interface bugs and resolves some issues that could cause crashes.
