CudaText：Sublime Text 的替代开源版本

2021-07-24 10:57:50

CudaText 是一个跨平台的文本编辑器，用 Object Pascal 编写。它是开源项目，可以免费使用，甚至可以用于商业用途。它启动得非常快：在 CPU Intel Core i3 3Hz 上的 Linux 上，使用 ~30 个插件的 ~0.3 秒。它可以通过 Python 附加组件进行扩展：插件、linter、代码树解析器、外部工具。语法解析器功能丰富，基于 EControl 引擎（虽然不如某些竞争对手快）。常见问题是“它与 Sublime 相比有什么优势？”维基回答它。 Wiki 包含几乎完整的程序文档。
