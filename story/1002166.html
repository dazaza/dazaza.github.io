宣布Google Cloud VMware Engine

2020-05-15 06:46:55

VMware技术构成了许多客户企业IT环境的基石，这些企业渴望在云中运行其VMware环境，以便快速扩展并从云服务中获益。

去年夏天，我们宣布支持客户在Google Cloud上运行VMware工作负载，此后我们取得了重大进展。今年秋天，我们收购了CloudSimple，为客户提供完全集成的基于VMware的解决方案，今天我们很自豪地宣布了另一个重要里程碑-Google Cloud VMware Engine，这是一个集成的第一方产品，提供端到端支持，可以在Google Cloud中迁移和运行您的VMware环境。这项完全托管的服务预计本季度将在美国两个地区全面推出，下半年将扩展到全球更多的Google Cloud地区。

该服务在Google Cloud高性能且可靠的基础设施上的专用环境中提供完全托管的VMware Cloud Foundation堆栈-VMware vSphere、vCenter、VSAN、NSX-T和HCX，以支持企业生产工作负载。借助这项服务，您可以通过Google Cloud Console直接连接到专用VMware环境，从而在几分钟内将您的本地工作负载迁移或扩展到Google Cloud。这使您可以无缝迁移到云，而无需花费重构应用程序的成本或复杂性，并可根据您的内部部署环境一致地运行和管理工作负载。通过在Google Cloud上运行您的VMware工作负载，您可以在受益于可扩展性和敏捷性的同时减轻运营负担，并保持现有工具、策略和流程的连续性。

重要的是，您可以在几分钟内在Google Cloud上创建VMware SDDC环境，从而快速满足您的业务需求，从而使您能够按需扩展业务关键型应用程序。该服务经过VMware云验证，是对基于VMware的云服务的最高级别验证，可帮助实现跨内部部署和云环境的兼容性和运营连续性。

VMware高级副总裁兼云提供商软件业务部总经理Ajay Patel表示：“VMware和Google Cloud正在合作，帮助客户推动多云战略，新的Google Cloud VMware引擎将使我们的共同客户能够使用目前在其数据中心运行的同一个VMware Cloud Foundation来推动数字转型和业务弹性，”VMware云提供商软件业务部高级副总裁兼总经理Ajay Patel表示。Google Cloud VMware Engine使组织能够在Google Cloud中快速部署其VMware环境，提供可扩展性、敏捷性和对云本地服务的访问，同时利用对VMware工具和培训的熟悉程度和投资。

Google Cloud VMware Engine构建在Google Cloud的高性能、可扩展的基础架构之上，具有完全冗余的专用100Gbps网络，可提供99.99%的可用性来满足您要求最苛刻的企业工作负载的需求。互连和VPN等云网络服务简化了从您的内部部署环境到云的访问，云服务的高带宽连接优化了性能和灵活性，同时最大限度地降低了成本和运营开销。集成了端到端、一站式支持，以便在此服务和Google Cloud的其余部分之间提供无缝体验。

Google Cloud VMware Engine旨在最大限度地减轻您的运营负担，让您可以专注于您的业务。我们负责VMware软件堆栈的生命周期，并管理所有相关的基础架构和升级。客户可以继续利用与其内部部署环境一致的IT管理工具和第三方服务。我们与领先的存储、备份和灾难恢复提供商(如NetApp、Actifio、Veeam、Zerto、Coheity和戴尔技术)密切合作，以确保支持第三方解决方案、简化迁移过程并实现业务连续性。

除了易于迁移之外，您还可以完全访问创新的Google Cloud服务，如BigQuery、云运营、云存储、Anthos和Cloud AI。计费、身份管理和访问控制也完全集成到Google Cloud中，以统一与其他Google Cloud产品和服务的体验。随着时间的推移，您希望迁移工作负载并使其现代化，这些云本地服务使您能够简化管理、展现新的数据洞察力，并为客户提供创新的新服务。

在过去的几个月里，我们通过我们的Early Access计划与众多客户进行了接触。客户在寻求将工作负载扩展或迁移到云中时，亲身体验了Google Cloud VMware Engine支持的快速而简单的迁移。资本市场基础设施提供商德意志交易所集团(Deutsche Börse Group)对将VMware工作负载迁移到Google Cloud的简单易行印象深刻。

德意志交易所集团执行董事会成员兼首席信息官Christoph Böhm博士说：“作为世界上最大的市场基础设施提供商之一，为金融市场实施创新和有弹性的解决方案对于维持高效、稳定和最重要的安全运营至关重要。作为VMware的长期客户，我们热衷于将我们的大环境扩展到超大规模选项，保持现有控制平面和生命周期管理的稳定性。Google Cloud VMware Engine现在使我们能够快速将我们的VMware环境扩展到德意志交易所的公共云合作伙伴之一Google Cloud，从而提高我们的业务敏捷性，并构建更高级别的弹性。到目前为止，我们共同经历的步骤非常鼓舞人心，为我们提供了运行混合云场景的创新和灵活的方式。“。

领先的ERP软件提供商QAD也对在谷歌云上运行VMware的好处感到兴奋。QAD IT架构总监Scott Lawson表示：“有了Google Cloud VMware Engine，我们能够迅速将我们基于VMware的平台扩展到Google Cloud，以实现快速、敏捷和高效的目标。”作为一家领先的ERP软件提供商，与谷歌云和VMware的合作使我们能够减轻运营负担，提高灾难恢复能力以确保客户的一致可用性，并受益于原生的谷歌云服务以不断创新。“。

我们很自豪能与地区和全球系统集成商密切合作，简化并实现我们共同客户的云迁移之旅的成功。我们的合作伙伴，如德勤、ATOS和WWT，致力于构建云服务，帮助客户采用Google Cloud VMware Engine，并通过原生Google Cloud服务加速其数字化转型。

合作伙伴可以在加快迁移并帮助您更快实现价值方面发挥重要作用。德勤咨询有限责任公司戴尔技术全球领导联盟负责人Bob Black表示：“当客户希望简化他们的云迁移之旅时，我们致力于构建云服务，以帮助客户受益于在Google Cloud上运行VMware工作负载的更高灵活性和效率。”通过将谷歌云的技术与德勤的业务转型经验相结合，我们可以使我们的共同客户能够加快他们的云迁移，统一运营，并在他们寻求实现应用程序现代化的同时受益于创新的谷歌云服务。“。

合作伙伴还将Google Cloud VMware Engine视为帮助客户加快云之旅的关键产品。Atos Cloud Enterprise Solutions高级副总裁Peter Cutts表示：“对于许多企业客户来说，在Google Cloud上运行VMware工作负载是他们的首要任务，因为他们希望从云的规模和敏捷性中获益，同时保持混合云和多云环境的一致性，”Atos Cloud Enterprise Solutions高级副总裁Peter Cutts表示。“我们很高兴有机会加强我们与谷歌云的合作关系，将Atos给VMware和谷歌带来的所有价值结合起来，提供差异化的体验，同时使客户能够受益于交钥匙服务，包括BigQuery、AI&；机器学习等云原生服务。”

万维网技术公司首席技术官迈克尔·泰勒说：“作为谷歌云的主要合作伙伴，我们对谷歌云VMware Engine加入到已经为我们共同客户带来价值的不断增长的服务清单中感到兴奋。混合云战略仍然是我们客户关注的焦点，这项服务大大加快了组织将其工作负载转移到云并使其基础设施现代化的时间框架。

Google Cloud VMware Engine预计本季度将在北弗吉尼亚州(US-East4)和洛杉矶(US-West 2)地区向客户全面开放。我们计划在今年下半年在另外八个地区-伦敦、法兰克福、东京、悉尼、蒙特雷亚尔、圣保罗、新加坡和荷兰-提供这项服务。

我们为这一里程碑感到兴奋，并致力于提供一个最佳平台，与谷歌云服务一起运行您的VMware工作负载，以解决业务问题并在新领域进行创新。
