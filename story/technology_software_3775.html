Skype 向 OSX 用户提供工具包

by Wangtam on 2006-07-26 03:45:04

Skype 将于本周内针对 Mac OS X 用户发布一款 Skype 工具包，其中 Mac X 1.5 Skype 软件将包括中文、法文、西班牙文、德文、意大利文、葡萄牙文及荷兰文等版本。改版之后后的 Mac Skype 软件包括 Windows 系统中增强的通话功能，重新调整了聊天图标，并增强了可存储不同国家代码的数据库。此外 SkypeWeb 用户还可借助在线状态检测器来显示在线状态。除 Skype 软件外，该工具包还包含了一个 USB 立体声耳机。IDC 高级分析师 Rebecca Swensen 认为，出于争抢网络电话用户目的，Skype 希望能进一步开拓 OSX 及企业用户市场。此前 Skype 与 Polycom 签定协议，以方便 Skype 用户开通电话会议功能。目前 Skype 用户之间可免费通话，用户免费拨叫传统电话将持续到今年年底。...
