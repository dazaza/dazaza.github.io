立即被Service收购的Element.ai

2020-12-01 04:43:27

ServiceNow周一宣布了这笔交易，并指出已签署协议收购Element AI，并有望在2021年初完成。

尽管被视为加拿大AI领域的领导者，Element AI仍难以将产品推向市场。

收购条款尚未披露。但是，据《环球邮报》报道，Element AI的售价不到5亿美元。这将使购买价格远低于蒙特利尔初创公司在2019年9月进行的2亿加元B轮融资后的估计估值。

Element AI由首席执行官Jean-FrancoisGagné，Anne Martel，Nicolas Chapados，Jean-Sebastien Cournoyer，Philippe Beaudoin和AI Yoshua Bengio的教父于2016年创立。迄今为止，它已经筹集了总计3.4亿加元（2.57亿美元）的资金，吸引了投资者的关注，例如魁北克政府，退休基金，魁北克养老金（CDPQ），麦肯锡公司，DCVC（数据收集） ，韩华资产管理。

2017年，这家初创公司从包括英特尔，微软，加拿大国家银行，加拿大开发银行（BDC），NVIDIA和Real Ventures在内的一群著名投资者那里筹集了当时的历史性1.375亿美元资金。 2018年12月，Element AI通过魁北克经济发展计划从联邦政府获得了500万美元的贷款。

其2亿美元的融资标志着加拿大历史上最大的一轮风险投资。 Element AI鼎盛时期拥有500多名员工，其中包括100名博士学位。

Element AI将自己定位为AI的全球领导者，这有助于进一步发展魁北克的行业，并与联邦政府和全球组织合作，在国际舞台上推广道德AI政策。这家初创公司帮助实现了2018年5月的《多伦多宣言》，为道德AI设定了国际标准。

尽管被视为加拿大AI领域的领导者和后起之秀，但Element AI仍难以将产品推向市场。 《环球报》在2019年报道说，这家初创公司面临高昂的运营成本，最低限度的收入以及其两款旗舰产品开发的挫折，导致其裁员。

去年，Gagné在与BetaKit交谈时，讨论了Element AI在商业化方面遇到的困难，与未能有效利用其数据的合作伙伴合作，缺乏现有的基础架构来扩展AI模型以及IP问题。

“当时我们必须建立大量的东西，因此要花费大量的精力和时间，”该首席执行官当时告诉BetaKit。 “这是一个关键的学习，这是我们尚未解决的问题。”

去年，Element AI看到了一些成功的签约客户并推出了一些自己的独立产品。据报道，在COVID-19大流行的前几个月中，尽管COVID-19延迟了某些合同，但Element AI达到了其所有目标，并且“犀利”地执行。

但是，2020年也带来了重组。据报道，Element在5月份解雇了15％的员工，同时任命了其首任首席财务官和首席收入官来支持其进入市场的战略。当时，这家蒙特利尔创业公司拥有约350名员工。

据TechCrunch称，ServiceNow将全面收购Element AI，并将保留其“绝大部分”技术人才，尽管在与ServiceNow集成后将“缩减”其现有业务。

ServiceNow还宣布了此次收购，将使其能够在加拿大建立AI创新中心，以加速其Now Platform的以客户为中心的AI创新。 Bengio还准备担任ServiceNow的技术顾问。

ServiceNow首席AI官Vijay Narayanan表示：“凭借Element AI强大的功能和世界一流的人才，ServiceNow将使员工和客户能够专注于只有人类才能脱颖而出的领域-创意思维，客户互动和不可预测的工作。”

ServiceNow将其对Element AI的收购称为其最新战略投资，以加速其Now Platform中的AI创新。 3月，ServiceNow雇用了Narayanan，并推出了具有一组AI功能的Now Intelligence。 Element AI是ServiceNow在2020年第四次收购AI，仅次于Loom Systems，Passage AI和Sweagle。

Gagné说：“ Element AI的愿景一直是重新定义公司如何使用AI来帮助人们更智能地工作。” “ ServiceNow是我们将人才和技术应用于当今企业面临的最重大挑战的明确合作伙伴。”
