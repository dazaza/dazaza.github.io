**From**: Dulaodu.com

**Function**: Extract the actual address of Tudou video FLV files from Tudou.com.

**Demo**: [http://www.51du.cn/tudou.php](http://www.51du.cn/tudou.php)

**PHP Source Code**:

```php

Dig Tudou - Dulaodu.com

Tudou playback page URL: " name="q" />

(.*?) - 视频/i', $html, $t);

preg_match('/iid=(\d+)/', $html, $m);

if (!empty($m[1])) {

$player_html = file_get_contents('http://www.tudou.com/player/v.php?id=' . $m[1]);

preg_match('/http:(.*?)\.flv/i', $player_html, $flv_match);

if (empty($flv_match[0])) {

echo "Could not extract. If the input URL is correct, try again!";

} else {

echo "" . htmlspecialchars($t[1] ?? '') . " has been extracted. The FLV address is:" . htmlspecialchars($flv_match[0]);

}

} else {

echo "Could not extract. If the input URL is correct, try again!";

}

} else {

echo 'Please enter the URL of the playback page, e.g., "http://www.tudou.com/programs/view/vZriVsqk-xc/"';

}

?>

```

### Explanation:

- **Title**: "Dig Tudou - Dulaodu.com"

- **Form**: A form that accepts the URL of a Tudou video page.

- **Logic**:

- It decodes the input URL and retrieves the webpage content.

- Uses regular expressions to extract the video title and ID.

- Fetches the player page using the extracted ID.

- Extracts the `.flv` file URL from the player page.

- Outputs either the `.flv` file URL or an error message if extraction fails.

Note: This script may no longer work due to changes in Tudou's website structure or security measures. Always ensure compliance with website terms of service when extracting content.
