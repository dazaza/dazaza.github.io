One key restore wizard, is a fool type system backup and restore tools. It has the characteristics of security, fast, strong confidentiality, high compression rate, good compatibility, etc., especially suitable for computer novices and people who are worried about operation trouble.

Software Features:

★ Installation fool, clear and simple

To achieve a key installation, silly operation. Users who do not have a floppy drive or optical drive can also install it.

★ Simple operation, reliable confidentiality

No boot disk is required, just press the F11 key at boot time to restore the system, and can be set up two level password protection.

★ Safe and fast, stable performance

Developed on the basis of GHOST 8.3, it has the characteristics of security, stability, and fast, and never destroys hard disk data.

★ Space saving, easy maintenance

You can manually select the backup partition size and automatically divide the backup partition capacity based on the data size of disk C. In addition, all partitions to be backed up are backed up with high compression to save disk space to the greatest extent. And can be updated at any time backup, uninstall convenient and safe.

★ Independent and efficient, good compatibility

Run independently under DOS system, never occupy system resources and affect the use of the operating system; Compatible with all partitioning tools; Supports FAT, FAT32 and NTFS partition formats; Support for WIN 98/ME and WIN2000/XP/2003/SERVER systems. Unique GHOST and PQDI dual kernel function, can automatically or manually select different cores. Supports backup and restoration of multiple hard disks and partitions.

★ Quick restore, yesterday again

If the computer is infected by the Trojan horse or the system crashes or even the C disk is formatted, press the F11 key after starting the computer, the operating system can be quickly restored to the health state. At the same time, the backup files are located in the hidden partition and are not infected by viruses.

The difference between personal version and installed version:

Personal version: no partition does not modify the partition table and MBR, installation is very safe, backup files are automatically hidden, not suitable for WIN98 system.

Installed version: need to use PQMAGIC partition as a hidden backup partition, there is a certain risk in the installation process, and it is very safe after installation.

The backup file of the installed version is in the hidden partition, and the backup file will not be damaged. If it is an installer or a new computer, using the installed version is a good choice.
