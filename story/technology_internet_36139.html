Preview of the New Version of DEDECMS

by lengpengcheng on 2007-11-22 18:59:24

Taking into account the various needs of users, the following improvements or new features have been finalized in the development and evaluation process for the official release of DedeCms 2007. Currently, the interactive module has undergone significant adjustments, and it is expected to be released by the end of this month. If there are delays due to design integration, the official team will announce it immediately and provide a Demo.

Final Feature List for DedeCms 2007

I. New Models or Plugins

1. Product publishing model;

2. Novel model;

3. Simple classified information model;

4. Custom form system;

5. Q&A module (similar to Baidu Zhidao);

II. New or Improved Functions

1. Static resource markers can use resource attribute IDs for caching, reducing the number of database reads;

2. Added custom small category options for sections, such as supply, demand, rental, shared rental, sale, etc., with the program indexing these different attributes;

3. Added custom search options function;

4. Added TAG function;

5. Section management in the backend can link to external links;

6. Added site statistics, such as the number of articles in a certain section, supporting template calls;

7. Friendship link template tags;

8. Download system added error reporting function;

9. Collection system automatically detects and filters out empty titles and content;

10. Membership system:

a. All open channels allow submissions, including guest submissions;

b. Added the ability to customize user registration agreements;

c. Internal SMS functionality;

d. In-site friend functionality;

e. Circle functionality;

f. Corporate space functionality, allowing corporate members to achieve an automatic website building function, including modules such as company introduction, contact us, online messages, online orders, product/goods display, supply-demand information, and company business cards.

11. Re-adjusted the management of attachment uploads; members will no longer use file-based attachment management but instead switch to database-based management;

12. Fixed possible timezone issues in PHP versions above 5.1;

13. Backend editor integrated with FLV player;

14. Added one-click full site generation function in the backend;

15. Administrators can manage multiple sections simultaneously;

III. Unique Features

1. Added the function to update HTML at scheduled times through Shell interface;

2. Added SQL tags, allowing any specified SQL query statement to realize data reading functions;

3. Dig-like voting function;
