PartitionMagic v8.05 Chinese Edition

by impark on 2007-05-13 22:02:07

After Symantec acquired PowerQuest last year, PartitionMagic 8.05 has finally been released under a new name! PowerQuest PartitionMagic is an excellent hard disk partition management tool. This tool can re-partition, format, copy, move, hide/reveal partitions, boot the system from any partition, and convert partition structures (such as FATFAT32) without losing existing data on the hard drive. It's powerful and can be considered the best-performing tool in this area at present. Simplified Chinese version instructions: 1. This is the "simplified Chinese version", no need to install the original software, it can be used under 9x/Me/NT/2000/XP with a common guide program that automatically recognizes your operating system, but it cannot be used for Server operating systems. 2. This is the full version, whether or not you enter the registration code, it can still be used indefinitely. After installation, the registration information will be automatically written into the about dialog box. If there isn't any, please double-click the pm805.reg file in the localization package to import the registry. 3. This localization patch does not include other utility software in PartitionMagic, nor does it include the localization of PowerQuest BootMagic. 4. The original software is for English operating systems, the version you are using is localized based on the English version. Localization is only for the interface and cannot change the program core, so sometimes it may not support Chinese well. When , especially NTFS -> FAT32, it cannot truly support Chinese, please use with caution!! Solution: Temporarily change all volume labels, Chinese directory names, and file names within the partition to English names, which allows correct conversion, then change them back to the original Chinese names. Other functions work fine on Chinese systems. 5. Operating on hard drives is very dangerous, beginners should not attempt it easily. In case of problems during use, such as power outages, system crashes, forced shutdowns, etc., please try the small tool "diskgen.exe" attached in the localization package. In most cases, this software can help you recover the damaged partition table. 6. To better create high-quality localizations, this version has bundled Alibaba Business Opportunity Express, which serves as a subsidy for amateur localizations. Alibaba Business Opportunity Express is a professional search tool custom-made for merchants. This bundling is calculated by installation quantity, after installation, it will add an icon to the IE toolbar and add a Business Opportunity Express link in the IE View-Browser Bar menu, that's all. Installation is optional, after installation, it can be completely uninstalled in Add/Remove Programs.
