Weather forecast

by 8o8o on 2006-10-04 10:23:20

Been a while since I last updated the main camp of Geeks | Newbies. Here's an introduction to a website that allows you to add weather services on Blogger: http://netwx.accuweather.com/

It can help you create real-time weather forecasts on your Blogger (the only highlight), but there are some issues:

It does not support Chinese;

Celsius/Fahrenheit display has problems (when 8O registered and previewed, Celsius displayed correctly, but after placing it on Blogger, it showed Fahrenheit);

It can only display the weather for specified cities and cannot automatically determine based on the visitor's IP.

Friends who are interested can give it a try. The registration process is quite simple, and the code is auto-generated (you just need to place it where you want).
