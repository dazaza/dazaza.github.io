FaWave: A Chrome extension that supports multiple microblogs

by ybcz0519 on 2011-01-06 17:11:07

FaWave is a Weibo plugin for Chrome that supports multiple microblogs, making it easy to switch between different accounts and allowing simultaneous posts to multiple microblogs. I tried it today and found it to be quite excellent, so I recommend it to everyone.

Key features include:

- Currently supports the following microblogs: Sina Weibo, Twitter, Sohu Weibo, Fanfou, ZuoSha, Digu, Renjian Wang, LeiHou, Douban, Google Buzz, NetEase Weibo (163).

- Supports OAuth authentication with both Basic and OAuth methods.

- Can set intervals for fetching information.

- Allows selective posting of microblog updates to one or multiple accounts.

- Supports unread message notifications.

- Supports insertion of browser geolocation information.

- Users can choose URL shortening services.

- Can reset Sina Weibo's unread notification count to zero simultaneously.
