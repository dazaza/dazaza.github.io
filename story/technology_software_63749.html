How does the QQ chat on mobile phone charge? Do you really understand? I want IP traffic network.

by 51ipw on 2008-04-18 04:51:05

What is the most popular communication tool in daily life? Of course, it's the mobile phone; what is the most popular communication tool for surfing the Internet? Of course, it's QQ. Then do you know what the most popular way of communication is now? It's the mobile QQ chat.

When it comes to chatting on QQ via mobile, the question that gets asked the most is how it is charged. This is one of the issues that people are most concerned about during use. To talk about the charging issue of mobile QQ, we must first discuss how users use mobile QQ. Some users use the QQ software (Bao Bao Xiang) that comes pre-installed on their phones, such as Nokia 3100 and other models. Other users download the mobile QQ software and log in through a website before logging into QQ. These two different login methods will result in different charging standards. The first one is generally called Mobile QQ, while the second one is called Mobile QQ (JAVA).

MM sends text messages when she has time

The first type involves paying Tencent 10 yuan per month, then each message costs an additional one jiao. This method can be used by any mobile phone with pre-installed Mobile QQ software (Bao Bao Xiang). However, the cost is relatively high, much higher than sending text messages.

The second type requires downloading the software, similar to using QQ on a computer. This has no relation to Tencent, and only requires paying the data fee to the mobile company. This requires a higher-performance mobile phone, and if you choose a monthly data plan, it can save you money. After checking, the author found out the current main data package monthly calculation and charge method.

Mobile QQ software chat traffic calculation method:

1KB = 0.03 yuan

Text: 7 ordinary text messages (about 70 characters/message) = 512 Chinese characters = 1KB = 0.03 yuan

Voice: 1 second = 0.6kb, 5 seconds = 3kb = 0.09 yuan

Picture: Determined by the actual size of the picture, a common MMS picture is approximately 20kb = 0.6 yuan

Video: 1-4KB per second

Preview songs: MP3 format, each song is approximately 100-200KB

MV appreciation: MV uses 3GP format, size is approximately 200-300KB

Mobile QQ software idle without chatting traffic calculation method:

Even if you only let the mobile QQ software run in the background without doing any operations, it will still generate data traffic.

Traffic standard: Approximately 100-200K/hour.

With the continuous development of technology, mobile phones have more and more functions, and different functions have different charging standards. The more standards there are, the more confused the users become. Different functions have different packages, do you know which package offers better value for money? Today, I just want to give everyone a simple introduction to the charges I am aware of, hoping that netizens will actively participate in discussions so that more netizens can clearly understand the charging standards and make informed decisions, striving to ensure that everyone consumes clearly and understands.
