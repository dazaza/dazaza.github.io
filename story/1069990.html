科学家创造出可在室温下工作的单原子薄磁铁

2021-07-27 18:19:18

伯克利实验室和加州大学伯克利分校开发的单原子薄二维磁铁可以推动计算和电子领域的新应用。在室温下运行的超薄磁铁的开发可能会导致计算和电子领域的新应用——例如高密度、紧凑的自旋电子存储设备——以及用于研究量子物理学的新工具。最近在《自然通讯》杂志上报道的超薄磁铁可以在下一代存储设备、计算、自旋电子学和量子物理学方面取得重大进展。它是由能源部劳伦斯伯克利国家实验室（伯克利实验室）和加州大学伯克利分校的科学家发现的。 “我们是第一个制造在环境条件下化学稳定的室温二维磁体，”资深作者、伯克利实验室材料科学部的教师科学家、加州大学伯克利分校材料科学与工程副教授姚杰说。 “这一发现令人兴奋，因为它不仅使室温下的 2D 磁性成为可能，而且还揭示了实现 2D 磁性材料的新机制，”加州大学伯克利分校姚研究组研究生、该研究的第一作者 Rui Chen 补充道。学习。当今存储设备的磁性组件通常由磁性薄膜制成。但在原子水平上，这些材料仍然是三维的——数百或数千个原子厚。几十年来，研究人员一直在寻找制造更薄更小的二维磁体的方法，从而使数据能够以更高的密度存储。此前在二维磁性材料领域取得的成果带来了可喜的成果。但是这些早期的 2D 磁铁会失去磁性并在室温下变得化学不稳定。

“最先进的二维磁铁需要非常低的温度才能发挥作用。但出于实际原因，数据中心需要在室温下运行，”姚说。 “我们的 2D 磁铁不仅是第一个在室温或更高温度下运行的磁铁，而且还是第一个达到真正 2D 极限的磁铁：它像单个原子一样薄！”研究人员表示，他们的发现也将为研究量子物理学提供新的机会。 “它打开了每一个原子进行检查，这可能会揭示量子物理学如何控制每一个磁原子以及它们之间的相互作用，”姚说。研究人员从氧化石墨烯、锌和钴的溶液中合成了新的二维磁体——称为钴掺杂范德华氧化锌磁体。在传统的实验室烤箱中烘烤几个小时，混合物就变成了氧化锌的单原子层，少量的钴原子夹在石墨烯层之间。在最后一步，石墨烯被烧掉，只留下一个掺杂钴的氧化锌原子层。 “对于我们的材料，行业采用我们基于解决方案的方法没有重大障碍，”姚说。 “它有可能以较低的成本进行大规模生产。”为了确认最终得到的 2D 薄膜只有一个原子厚，姚和他的团队在伯克利实验室的分子铸造厂进行了扫描电子显微镜实验，以确定材料的形态，并通过透射电子显微镜 (TEM) 成像逐个原子地探测材料。

伯克利实验室高级光源的 X 射线实验表征了高温下二维材料的磁参数。在 SLAC 国家加速器实验室的斯坦福同步辐射光源进行的额外 X 射线实验验证了合成二维磁体的电子和晶体结构。在阿贡国家实验室的纳米材料中心，研究人员使用 TEM 对二维材料的晶体结构和化学成分进行成像。研究人员发现，石墨烯-氧化锌系统在钴原子浓度为 5-6% 的情况下变得具有弱磁性。将钴原子的浓度增加到约 12% 会产生非常强的磁铁。令他们惊讶的是，超过 15% 的钴原子浓度将 2D 磁体转变为奇异的“挫折”量子态，从而使 2D 系统内的不同磁态相互竞争。与之前的 2D 磁铁在室温或更高温度下失去磁性不同，研究人员发现，新的 2D 磁铁不仅可以在室温下工作，而且可以在 100 摄氏度（212 华氏度）下工作。 “与之前的 2D 磁体相比，我们的 2D 磁系统显示出一种独特的机制，”陈说。 “而且我们认为这种独特的机制是由于氧化锌中的自由电子。”当您命令计算机保存文件时，该信息以一系列 1 和 0 的形式存储在计算机的磁存储器中，例如磁性硬盘驱动器或闪存。

与所有磁体一样，磁存储设备包含具有两个磁极的微型磁体——北极和南极，它们的方向遵循外部磁场的方向。当这些微小的磁铁被翻转到所需的方向时，数据就会被写入或编码。据陈说，氧化锌的自由电子可以作为中间体，确保新 2D 器件中的磁性钴原子继续指向同一方向——从而保持磁性——即使在这种情况下，半导体氧化锌是一种非磁性材料。 “自由电子是电流的组成部分。它们向同一方向移动以导电，”姚补充道，将金属和半导体中的自由电子的移动与水流中的水分子流动进行了比较。这种新材料——几乎可以弯曲成任何形状而不会断裂，比一张纸薄一百万倍——可以帮助推进自旋电子学或自旋电子学的应用，这是一种利用电子自旋方向而不是自旋方向的新技术。比它对数据进行编码的费用。 “我们的二维磁铁可以形成超紧凑的自旋电子器件，以设计电子的自旋，”陈说。 “我相信在室温下发现这种新的、坚固的、真正的二维磁体是一个真正的突破，”共同作者、伯克利实验室材料科学部的高级科学家、加州大学伯克利分校的物理学教授罗伯特·比格诺说。谁共同领导了这项研究。 “我们的结果甚至比我们预期的还要好，这真的很令人兴奋。在科学中的大部分时间，实验可能非常具有挑战性，”姚说。 “但是当你终于意识到一些新的东西时，它总是很充实。”参考文献：Rui Chen、Fuchuan Luo、Yuzi Liu、Yu Song、Yu Dong、Shan Wu、Jinhua Cao、Fuyi Yang、Alpha N'Diaye，“共掺杂二维范德华 ZnO 中的可调室温铁磁性”， Padraic Shafer、Yin Liu、Shuai Lou、Junwei Huang、Xiang Chen、Zixuan Fang、Qingjun Wang、Dafei Jin、Ran Cheng、Hongtao Yuan、Robert J.
