CISCO Router, Switch Password Recovery

by fdisk54 on 2010-09-29 20:59:52

### One, Common CISCO Commands:

- **show version**: Check the configuration register value, hardware configuration, and IOS version.

- **show flash**: Check the IOS in Flash or the size of Flash, usage (how much is used, how much is left).

- **show startup-config**: Check the startup configuration file in NVRAM (saved configuration).

- **show running-config**: Check the file in RAM (currently running configuration).

### Two, Router Password Recovery:

#### Steps for 2600 Series Routers:

Using the console to control the router:

1. Start the router, press **Ctrl+Break** within 60 seconds.

2. **rommon>confreg 0x2142**

3. **rommon>reset**

4. **router#copy startup-config running-config**

5. **router(config)#no enable secret** // Delete the password or change it, here we delete it.

6. **router(config-line)#no enable password**

7. **router#copy running-config startup**

8. **router(config)#config-register 0x2102**

9. **router#reload** // Optional command

#### Steps for 2500 Series Routers:

1. Start the router, press **Ctrl+Break** within 60 seconds.

2. **>o/r 0x2142**

3. **>i**

The remaining steps are the same as for the 2610.

### Three, Switch Password Recovery (Example with CISCO 2950 Switch):

Using the console to control the switch:

Restart the switch: Press the **MODE** key to enter the **switch:** mode as follows:

```

C2950 Boot Loader (C2950-HBOOT-M) Version 12.1(11r)EA1, RELEASE SOFTWARE (fc1)

Compiled Mon 22-Jul-02 17:18 by antonino

WS-C2950-24 starting...

Base ethernet MAC Address: 00:13:1a:9a:2b:80

Xmodem file system is available.

The system has been interrupted prior to initializing the

flash filesystem. The following commands will initialize

the flash filesystem, and finish loading the operating

system software:

flash_init

load_helper

boot

```

#### Steps:

1. **switch: flash_init** // Initialize Flash:

```

Initializing Flash...

flashfs[0]: 4 files, 1 directories

flashfs[0]: 0 orphaned files, 0 orphaned directories

flashfs[0]: Total bytes: 7741440

flashfs[0]: Bytes used: 3090944

flashfs[0]: Bytes available: 4650496

flashfs[0]: flashfs fsck took 6 seconds.

...done initializing flash.

Boot Sector Filesystem (bs:) installed, fsid: 3

Parameter Block Filesystem (pb:) installed, fsid: 4

```

2. **switch: load_helper** // Load helper.

3. **switch: dir flash:** // View contents of Flash:

```

Directory of flash:/

2 -rwx 736 vlan.dat

3 -rwx 3086336 c2950-i6q4l2-mz.121-22.EA2.bin

5 -rwx 1558 config.text // Configuration applied at switch startup

6 -rwx 5 private-config.text

4650496 bytes available (3090944 bytes used)

```

4. **switch: rename flash:config.text flash:config-old.txt** // Rename config.text.

5. **switch: reset** // Restart the switch.

```

Are you sure you want to reset the system (y/n)? y

System resetting...

```

After restarting, the switch will no longer apply the configuration file since the name of the configuration file was changed. The switch enters the configuration dialog mode.

```

--- System Configuration Dialog ---

Would you like to enter the initial configuration dialog? [yes/no]: n

Switch#dir

Directory of flash:/

2 -rwx 736 Mar 01 1993 00:19:14 +00:00 vlan.dat

3 -rwx 3086336 Jan 01 1970 01:12:26 +00:00 c2950-i6q4l2-mz.121-22.EA2.bin

5 -rwx 1558 Mar 01 1993 02:36:44 +00:00 config-old.txt

6 -rwx 5 Mar 01 1993 02:36:44 +00:00 private-config.text

7741440 bytes total (4650496 bytes free)

```

6. **Switch#copy config-old.txt running-config**

```

Destination filename [running-config]?

1558 bytes copied in 1.152 secs (1352 bytes/sec)

```

7. Enter configuration mode:

```

sw#config t

Enter configuration commands, one per line. End with CNTL/Z.

sw(config)#line console 0

sw(config-line)#no pass cisco // Clear the console port password

sw(config)#no enable secret // Clear the enable password

sw(config)#no enable password // Clear the enable password

sw#write // Save

```

Note: If the passwords are successfully deleted, in general, you can choose not to delete them but instead directly change them to your desired passwords.
