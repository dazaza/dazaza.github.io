Make your C drive cleaner | System Speed-up | Computer Expertise | A way to make your system drive quieter (which can help ensure your computer runs faster to a certain extent):

1. Go to: Control Panel -- System -- Advanced -- Environment Variables

2. Click on TEMP in User Variables -- Edit -- Change the long string content to D:\TEMP (you can change it to another drive if you prefer, just not the system drive).

3. Similarly, change TMP to D:\TEMP as well. These items are major factors causing rapid expansion of your system drive and they are hidden, so some users might have trouble finding them.

4. In "System Variables", also change TEMP and TMP to the same as above.

5. Right-click on "My Documents" on the desktop -- Properties -- Target Folder -- Set "Target Folder Location" to D:\MyDocuments or D:\我的文档 (choose wherever you like!). Then you can optionally move it.

This is very important. In reality, when our computer crashes, there's usually nothing critical left on the system drive; everything useful is in MyDocuments. So by moving this folder, you don't need to worry about anything else. Also, quite a few people don't know how to back up the contents of MyDocuments in DOS mode (especially if the folder name is long and you're using Chinese characters, which makes it even more troublesome!).

6. Right-click on Internet Explorer -- Properties -- General -- Internet Temporary Files -- Settings -- Move Folder -- Select D:\TemporaryInternetFiles\ (if it doesn’t exist, you can create it first, or use D:\TEMP or something else – it’s your choice!) -- Confirm!

7. System -- Advanced -- Performance -- Settings -- Advanced -- Virtual Memory -- Change -- Choose D or any non-system partition -- Custom Size -- Enter size in M -- Set; choose C or the system partition -- No paging file -- Set -- Confirm. This will move the virtual memory to another hard drive.

Now, the world inside your system drive has truly become a peaceful utopia!
