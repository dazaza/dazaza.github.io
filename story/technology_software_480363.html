Fetch.io – A Powerful Tool for Extracting and Downloading Overseas Cloud Storage Resources

by xdash on 2011-08-21 12:43:57

Fetch.io is not only a cloud storage service but also an online extraction, storage, and download tool for foreign mainstream disk resources. It provides users with 2GB of free cloud storage space and 4GB of monthly bandwidth. Fetch.io supports RapidShare, MegaUpload, FileServe, Netload.in, FileSonic, Easy-Share, MegaShares, Hotfile, Despositfiles, and more. (Source: Sync Control)

As stated on its homepage: "You like to download things and you don't like waiting a long time for them…" The value of Fetch.io lies in solving the pain points users experience when using cloud storage services, especially foreign ones – slow speed. The root cause of slow speeds may come from various aspects, such as fewer seeds, server location distance, limited bandwidth, ad wait times imposed by cloud storage services themselves, restrictions on download tools and thread numbers, or even being blocked by a firewall.

Fetch.io solves this problem through a "transit station" style approach: online stream (stream) extraction -> stored in cloud space -> available for users to download at any time. Users no longer need to wait on their local computers for downloads to complete; they just need to set up file extraction tasks on Fetch.io. After some time, they will find that the files are quietly waiting in their cloud storage space for further action.

### Usage Method:

1. First, switch to Fetch under the options File/Fetch/Transfer/Account (the functions are self-explanatory). Start a new extraction task. In Step1: Choose Location, select the location where the files should be stored in your cloud storage (you can create folders for better organization). Then, in Step2: Paste Links, paste the supported links into the designated area. Batch tasks are allowed (one link per line), and automatic decompression after extraction is supported. Click Next to proceed.

2. The system automatically identifies the links. If it's okay, it will display the file name, size, and a checkmark. At this point, click Fetch! to proceed.

3. Extraction begins. During my personal testing, the speed was very ideal, reaching several MB/s, far exceeding the direct download speed of the original link (the visual effect is also excellent). Extraction tasks are performed on the server side, meaning you can close your browser and go out for a while. When you return, the progress will still be running smoothly or even completed.

Once the extraction is complete and the files are stored in the cloud storage space, you can download them locally (with very good speed) or share them with others (the recipient must also have a Fetch.io account, as the shared files are saved in the recipient’s Fetch cloud storage space).

It should be emphasized that Fetch.io is fundamentally a cloud storage service, so basic file upload functionality is naturally included.

For more information, refer to the evaluation by Douban users: http://www.douban.com/note/156268311/
