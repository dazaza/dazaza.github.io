Wubi input method, input method, another main battlefield of the Internet?

by henguai on 2009-12-07 03:26:32

What are the most basic applications of a network, including the Internet and mobile phone networks? Not only networks, but also terminals, desktops, laptops, and mobile phones, etc., should all possess one fundamental function. It's not the browser, not IM, not antivirus software, nor office software - it is the input method. Whether or not your computer is connected to the internet, whether your phone is just used for texting or surfing the web, you will need an input method. The input method is the most basic application of this network era, but it has also been consistently overlooked. Many hardware and software products, as well as content services based on the network era, have created many successful enterprises, but it seems that only the input method has always been a supporting role, witnessing the changes in the network world without ever dominating it. Among the Chinese input methods we've encountered, Wubi was probably the earliest - it was really difficult. In high school, I memorized the character root table, but fortunately, I eventually came into contact with phonetic input methods. The earliest phonetic input method was probably Microsoft Pinyin, followed by Quanpin, and then Smart ABC. Chinese phonetic input methods have been striving to improve in intelligence and user experience. In my memory, a relatively distinctive phonetic input method was Ziguang, but there might have been issues with the technical team because Ziguang often had bugs. Subsequently, Sogou Input quietly took up residence in the bottom right corner of most people's computers. If earlier input methods were either bundled products like Microsoft's, or the result of small teams, then Sogou Input marked the beginning of serious cultivation of this market by major players in the Chinese internet industry. Later, Google, Tencent, and even Baidu, which recently acquired a mobile input method company, all began entering the input method market - perhaps this will become another fiercely contested battlefield for internet giants in the future? Did Google's input method lose completely against Sogou Input? And recently, the lawsuit between Sogou and Tencent over the input method made waves. For the internet world, resolving disputes through legal means may become the mainstream in the future, and this may also be the best way to resolve competition. Sohu's early layout can be seen in its current success from the numbers: According to a survey report by iResearch, 43.7% of users who use Sogou Input do so based on recommendations from friends. This is much higher than the proportion of users gained through promotion on new media or traditional media. At the same time, more than 100 million users use Sogou Input every day. Among these, nearly 80% of users used Sogou Input in 2009, making it the most commonly used input method software by users, with a user ratio exceeding 50%, far ahead of other input method products ranking first. Data also shows that 61.4% of users are willing to recommend Sogou Input to their friends, a proportion significantly higher than other input method software. In the field of input methods, which requires technical accumulation, direct competition relies on technological leadership and more on product service. I believe Sogou Input's leading position is the result of multiple interactions. If we set aside technical accumulation and Sogou's continuous R&D innovation, I think it is interaction that has made Sogou Input successful. Only in the internet era could such a successful Chinese phonetic input method as Sogou Input appear. In the future, Sogou Input and its competitors will engage in fierce competition, but there is no doubt that Sogou Input has already taken the lead in market competition - interaction, intelligence, internetization, cloud comfort, and mobile input may become key terms in the competition among input methods in the next few years.
