Preview of the new version of McKew's floating window tool

by xdash on 2011-08-12 18:22:28

MaiKu (http://mknote.com) is a permanently free personal cloud-based notepad developed by Shanda Innovation Institute. You can use it to store various notes and documents without worrying about losing them. Currently, MaiKu has released clients for Windows desktop, iPhone/iTouch, and Android.

Recently, @XDash got hold of the new version of MaiKu's Windows client which is still under development. Personally, I believe the most striking and practical feature of the new version is the newly added floating window in the upper right corner. Here's an exclusive preview for everyone:

(Yes, that's right, showing off my desktop — I'm not Guo Meimei, and I don't own a Maserati.)

Zooming in, the floating window (which I personally think should be called a toolbar) looks like this:

(At first glance, does it remind you of a certain Weibo desktop client? ^^ The functions are quite straightforward.)

Clicking on the first pencil icon will directly pop up an editing area similar to a sticky note, where you can quickly jot down or sketch ideas. You can also set it to stick on the desktop or launch the full editor for more detailed editing. As shown below:

Clicking on the screenshot icon enters the screenshot mode, similar to QQ's screenshot function. The captured image is automatically saved into a new note in MaiKu. As follows:

Clicking on the third list icon will display the last 10 notes and their previews in a drop-down list format, allowing you to enter the full editor for further editing. As shown:

The function of the fourth synchronization button is self-explanatory. The last magnifying glass icon, when clicked, initiates a quick search (the search speed is very fast, almost instant). As shown:

Personally, I feel that the design of the floating window in the new version of MaiKu greatly improves the efficiency of note-taking and searching. It allows users to focus more on capturing and recording information rather than learning how to use the tool. After using it for a while, I've become quite reliant on this floating window. In contrast, the MaiKu icon hidden in the system tray and the full editor window have been somewhat neglected. Actually, the floating window is better suited for collecting and acquiring information, while the full editor with rich text editing capabilities is more suitable for post-processing and organizing.
