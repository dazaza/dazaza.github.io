Streamline system settings! Make your Vista more secure.

by cqe on 2009-08-03 17:29:06

A brand-new computer may bring you more than excitement and thrill. A new machine with pre-installed Vista system will make you eager to try it out. However, it's not perfect - you should at least make some necessary adjustments before using it and create a more personalized setting so that your system can show its maximum performance, highest security, and best usability.

1. First Boot

After you've decided how to use your computer and have done the connection work (power supply, monitor, network cable, keyboard, and mouse), Vista will prompt you to set basic information such as language, time zone, and clock, and also ask you to create a username and password. If you skip this step, unless you're 100% sure no one else will use this computer, you will definitely encounter a lot of trouble.

Create a new account for the system.

2. Slim Down Your System

Big-name system manufacturers usually install software on computers in the factory. And these softwares often come with some synonyms: "bloated", "bundled", "redundant". Manufacturers bundle these softwares under the guise of considering users but actually for their own benefit. Only a few manufacturers like Sony and Dell provide an option-free installation, and this is only for small business users. Companies like Velocity Micro offer a relatively "clean" system to users.

So how do you streamline your new computer? Download and run the free PC Decrap-ifier, which will delete those redundant software tools: from AOL Installation Assistant to Yahoo Toolbar. If you think uninstalling isn't thorough enough, you can try RevoUninstaller (free uninstall tool), which can replace the system's Add/Remove Programs.

Now, you can also delete unnecessary components in Vista. Open Programs and Features in Control Panel, click "Turn Windows features on or off", at this point UAC will pop up a warning, click OK. Uncheck the features you don't need, such as games, optional PC components, etc. If you don't know the meaning of a certain item, hover your mouse over the item name, and you'll see its description.

Note that you shouldn't confuse trial software with redundant software. You might want to use these software during the trial period, so it's wise to keep the trial software, especially those free trial security software.

3. Activate Security System

If you want to spend money on security products for strong protection, then I recommend Norton Internet Security 2009. Its anti-spyware and antivirus functions are very powerful, and it has a minimal impact on system performance.

If you don't want to spend money on services, it's fine too. You can build a two-in-one comprehensive antivirus system with AVG Anti-Virus Free 8.0 and ThreatFire 3.5. The former uses virus signature codes to detect viruses, while the latter uses behavior analysis. Therefore, even if these two anti-malware programs run simultaneously on the same computer, they won't conflict with each other (note that ThreatFire can be overly effective, you need to turn it off when installing new software, otherwise it will consider the new software as a security threat).

ThreatFire

Broadband users need to install a firewall to control which applications can access the network. Having just a router as a firewall is far from enough. If you're talking about free firewalls, you can consider Comodo Firewall Pro.

4. Download Updates

Sometimes, your computer will tell you that there are available updates for Windows Vista. It will display an icon in the lower-right corner of the system tray to indicate that updates are available for download, of course, you can also choose Windows Update from the Vista system (find it in all programs). Depending on your system version and installation time, the available updates will vary. After all the updates are downloaded, run Windows Update again, it will start searching for available updates again, repeat this process three times, after that, your system will be relatively secure.

5. Backup System

After a system crash, some technicians only think of reinstalling the system with the original CD, but this is not only troublesome, but also means needing to reinstall those system updates. On the contrary, backing up the system is a wiser approach.

The most commonly used backup system is probably the famous Ghost tool. If you are using Vista Business Edition or Advanced Edition, you can also use the built-in Complete PC Backup to complete the system backup. Then, save the backup to an external drive or burn it onto a DVD.

Backup your system and files

6. Transfer Files

Compared to Vista, using the Windows Easy Transfer tool would be simpler. This tool can also cooperate with what we will discuss (CDs, networks, and transfer cables), not only transferring data and folders, but also transferring settings from the old system to the new system. It can even recreate your user account according to your needs without adversely affecting existing applications. To fully utilize Easy Transfer, your old computer must be running Windows 2000, XP (price, parameters, pictures) or Vista.

You can also use the old method - copying files from the old computer to CDs, DVDs or mobile storage devices, and then copying them to the new machine. However, if you have a lot of files to move, this method can be time-consuming. You can consider buying a USB interface SATA/IDE adapter, which can help turn your old hard drive into an external hard drive for the new PC.

In fact, you can also achieve the same purpose by using a home network. Go to the system control panel, click on advanced system settings, then go to the system properties under the computer name option tab, and click the change button. Ensure that the computer name is unique within the home network, and the workgroup is exactly the same; otherwise, they cannot access each other. Next, check the firewall and ensure that it is open for computers on the home network. Find the folder you want to share, right-click properties, and specify the folder and files to be shared, so you can access this file on other computers on the home network.

7. Browser Plugins

Believe it or not, Firefox is many users' best browser. If you have installed a few important extension tools, wouldn't you want to know how they work on IE or Google's Chrome?

Here are some plugins worth trying (in the Firefox menu bar, go to Tools, then select Add-ons and find the corresponding plugin in the plugin tab).

Cooliris: A free plugin that brings 3D experience to your browser

DownThemAll: A download manager that can handle simultaneous downloads of multiple batches

FireFTP: If you need an FTP client, you can use this plugin for the Firefox browser

Foxmarks: You can use it to synchronize bookmarks across multiple computers

Greasemonkey: The function of Greasemonkey is to add some custom scripts, adding many practical functions to the page

IE Tab: The IE Tab extension for Firefox allows you to browse web pages incompatible with Firefox directly in Firefox

ScribeFire: ScribeFire is a full-featured blog editor that helps you easily write blog posts

Tab Mix Plus: It's an enhanced version of the original Tab Mix (price, parameters, pictures), improving Firefox's tab browsing capabilities

Update Notifier: As the name suggests, it displays update lists and lets you know the latest update requests

If you use Firefox on your old computer, you might want to use the same settings, bookmarks, and add-ons on your new computer. You can use the free software MozBackup (step-by-step guide to backing up Firefox information with MozBackup) to back up this data, then save and restore it to the new computer.

Of course, you can install other components according to your preference, but no matter what browser you use, we recommend you install these tools: QuickTime, Adobe Flash, Microsoft Silverlight, Windows Media Player, and PDF reader (the author recommends Foxit Reader - a smaller and faster reading software than Adobe Reader).

8. Optimize System

Possibly every user will face choices between performance and speed. Below, I will introduce some methods to optimize the system and increase system speed rather than for impractical flashy appearance.

Set the desktop wallpaper to a background image with a single color and simple pattern. Large topic photos will affect the system loading speed.

If you can't enter the desktop tool at the screen edge or prefer to get tool options from Google, etc., please close the Windows sidebar because it occupies desktop space. Find the Windows Sidebar Properties control panel and uncheck start with the system.

As everyone knows, Vista provides a very beautiful Aero transparency window interface, but it comes at the cost of sacrificing system performance. In the Personalization control panel, select Windows Color and Appearance. In the pop-up window, click to open the classic appearance properties and change the corresponding color theme, such as Windows Standard, and click to close effects like menu shadows, etc.

Enter the system control panel, click System Protection, and click the Performance button on the Advanced tab. If you turn off every visual effect (such as animation effects, shadow menus, and shadow cursors), the system performance will improve significantly.

The network icon in the system tray will animate to show your connection status. Right-click and select "Disable animated activity," unless you don't mind being constantly reminded that you're online, otherwise turning it off is definitely a wise choice.

If you have a high-speed USB drive, plug it into your computer and activate Windows ReadyBoost. This way, when the drive is plugged in, the cache will play a role in enhancing performance.

Adjust power settings, especially when you're using a laptop and haven't unplugged the power, you should do this.
