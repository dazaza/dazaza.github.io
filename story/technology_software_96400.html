Perfect Decode is an all-in-one media codec package that enables playback and encoding of various popular video formats and HDTV. It comes bundled with three popular media players: Media Player Classic, KMPlayer, and BSPlayer. It supports installation under simplified Chinese and English language platforms and can play a wide variety of audio/video file formats such as AVI, VCD, DVD, MPG, MP4, RMVB, TS, TP, EVO, M2TS, MKV, OGM, MOV, SCM, CSF, FLV, and more. The recommended installation environment includes Windows XP, DirectX 9.0C, and Windows Media Player 10/11. It does not support Windows 9x. If you intend to use it on the Vista system, please disable the UAC function before installation. To use it alongside Realplayer, do not select the Real decoding components during installation; QuickTime works similarly.

### Installation Notes and Parameters:

1. **Before installing**, please uninstall any other similar decoding packages or players (including older versions of Perfect Decode). It is recommended to restart the Windows system after uninstallation before proceeding with the new installation.

2. **To support HD-DVD/BD VC1 Remux(TS) playback**, you need to upgrade to Windows Media Player 11 for VC-1 video decoding (you can also install its runtime separately, by running wmfdist11.exe extracted from the WMP11 installer).

3. **Perfect Decode** is released with the aim of sharing and never bundles any rogue/advertising plugins. However, some auxiliary tools are written in NSIS scripts, which might be mistakenly flagged as advertising software by certain monitoring programs. Please handle this situation with the mindset of "trust those you employ, and don't employ those you distrust." This is hereby stated to avoid repeated inquiries.

4. **Silent installation command-line parameters** are as follows:

- `/S` - Silent installation (no user interface will be displayed. If no other parameters are specified, the default components will be installed using the system's default language settings.)

- `/D=Path` - Specifies the path for the installation directory.

Using the above parameters, you can perform a simple customized installation. For example, to achieve a fully silent installation with the installation path set to `E:\Pure Codec` (assuming the filename of Perfect Decode is `PureCodec.exe`), use the following format:

```

PureCodec.exe /S /D=E:\Pure Codec

```
