卡巴斯基警告假 Windows 11 安装程序正在传播恶意软件

2021-07-25 05:07:20

微软尚未发布 Windows 11，但新的操作系统已经可供下载和预览。当然，网络犯罪分子正在利用这一点，将恶意软件发送给认为自己正在下载 Microsoft 新操作系统的用户。微软实际上声称 Windows 10 将是 Windows 的最后一个版本，该公司只会从那里发布更新。然而，在今年 6 月 24 日，该公司推出了 Windows 11。尽管在引擎盖下基本上是 Windows 10，但 Windows 11 仍然是该操作系统六年来最大的更新，具有数量可观的新功能和界面修改。正式地，Windows 11 将于 2021 年向公众开放，但很多人正在通过安装预发布版本提前试用它。一些爱好者想尝试新系统的新颖性；其他人，例如技术记者，通知用户有关新功能的信息。对于软件开发人员，他们需要操作系统与自己的产品进行兼容性测试，并在发布日之前修复任何问题。尽管微软已经使从其官方网站下载和安装 Windows 11 的过程变得相当简单，但许多人仍然访问其他来源下载该软件，其中通常包含来自网络犯罪分子的未经宣传的好东西（根本不一定是 Windows 11）。网络犯罪分子欺骗用户的最直接方式是插入一些额外的东西（或其他东西）。一个示例涉及一个名为 86307_windows 11 build 21996.1 x64 + activator.exe 的可执行文件。文件大小高达 1.75GB，看起来确实合理。但事实上，该空间的大部分由一个包含大量无用信息的 DLL 文件组成。打开可执行文件启动安装程序，它看起来像一个普通的 Windows 安装向导。它的主要目的是下载并运行另一个更有趣的可执行文件。第二个可执行文件也是一个安装程序，它甚至带有许可协议（很少有人阅读），称其为“86307_windows 11 build 21996.1 x64 + 激活器的下载管理器”，并指出它还会安装一些赞助软件。如果您接受该协议，则您的机器上将安装各种恶意程序。

卡巴斯基产品已经击败了数百次使用类似 Windows 11 相关方案的感染尝试。该恶意软件的很大一部分由下载器组成，其任务是下载和运行其他程序。这些其他程序可能非常广泛——从相对无害的广告软件（我们的解决方案将其归类为非病毒），到成熟的特洛伊木马、密码窃取程序、漏洞利用程序和其他令人讨厌的东西。按照 Microsoft 的建议，仅从官方来源下载 Windows 11。到目前为止，Windows 11 仅对 Windows Insider 计划的参与者正式开放——也就是说，您必须注册。您还需要一台已安装 Windows 10 的设备。要将您的 Windows 10 计算机升级到 Windows 11，请转到“设置”，单击“更新和安全”，然后选择“Windows 预览体验计划”并激活“开发者频道”以获取更新。我们不建议在您的主计算机上运行更新；预构建可能不稳定。我们还建议您使用可靠的安全解决方案，永远不要禁用它，这样网络犯罪分子就无法通过社会工程或非黄金时段系统中的漏洞访问您的计算机。
