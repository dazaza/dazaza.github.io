Sogou Pinyin input method 3.0 public test version 2 officially launched

by yinyin on 2007-07-02 13:06:38

Dog Pinyin input method 3.0 public test version 2 officially launched! This is the 11th version of Sogou Pinyin input method since its birth. Sogou Pinyin input method has been massively improved in these 4 months. Mainly completed the cell thesaurus, Unicode version, core structure optimization, Vista compatibility and other new functions or improvements.

During the private beta, the new version of the cell thesaurus function was warmly welcomed by thesaurus makers, and within three days of the launch of the cell thesaurus page, users uploaded more than 200 cell thesaurus. In addition, some thesaurus have been edited more than 20 times, and the update function of the cell thesaurus allows these modified cells to be automatically synchronized to the input method.

The new version of the cell thesaurus defaults to adding "300 Tang Poems", "Famous sentences of ancient poetry", "all stock names" and other thesaurus that we may often use. Almost all the famous ancient poems can be typed smoothly.

Unicode encoding support allows the input method to run on traditional, Japanese, Korean, and English operating systems, and Vista compatibility improvements make the input method perfectly compatible with IE7 in Vista.

At the same time, as the most important part of the input method, the input core has been significantly improved. Due to the optimization of the data structure and algorithm, the reaction speed is further improved. The new input method is more responsive and consumes fewer system resources.

The new version of the download address is: http://ime.sogou.com/dl/sogou_pinyin_3b2.exe

1. Fixed incompatibility with IE7 in Vista;

2. Fixed the bug that the input method could not be upgraded under Vista;

3. Fixed the bug that crashed when inserting special symbols in Word2003.
