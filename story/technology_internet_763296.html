Samsung Develops World's Fastest Embedded Memory Product

by anonymous on 2013-07-29 18:07:04

Seoul, July 28 (Xinhua) -- According to foreign media reports, Samsung Electronics announced on Friday that it has begun mass production of the world's fastest embedded memory products, which are also the industry's first eMMC 5.0 memory devices. These memory products come in three capacities: 16GB, 32GB, and 64GB, and can be applied to next-generation smartphones and tablets.

The interface speed of the eMMC PRO memory card reaches up to 400MB/s, enabling extremely fast application startup and loading. These chips can more quickly handle multitasking, web browsing, app downloading, file transferring, HD video recording and playback, and can support ultra-large file games and productivity applications.

KyongMoo Mang, Vice President of Memory Product Marketing at Samsung, said: "The performance of our newly developed ultra-high-speed eMMC PRO product is ten times faster than that of external memory cards. With the start of mass production, Samsung will accelerate the promotion of high-end mobile devices to meet market demands for large-screen and multimedia devices. We will continue to provide advanced mobile memory solutions so that users can enjoy smooth content experiences with high clarity and large capacity, while strengthening technical cooperation with mobile device manufacturers."

Samsung's eMMC PRO memory chips are built on Samsung's 64GB 10-nanometer-class NAND flash technology.

Samsung's new chips support the eMMC 5.0 standard, which the JEDEC, the standards organization for the microelectronics industry, is about to finalize.

At densities of 32GB and 64GB, the random read data speed of the new memory solution is 7000IOPS, and the random write data speed is 7000IOPS. In addition, the sequential read data speed of these chips is 250MB/s, and the sequential write data speed is 90MB/s.

Since the fastest eMMC devices access data ten times faster than external memory cards, the new mobile memory products can greatly increase the speed of data movement between one application and another.

Samsung's 16GB, 32GB, and 64GB eMMC 5.0 devices have a form factor of 11.5x13mm, making them very suitable for mobile devices where available space on the printed circuit board is limited. The eMMC PRO 5.0 chip is equipped with Samsung's own intelligent NAND controller and firmware, providing a vertically integrated mobile DRAM solution.
