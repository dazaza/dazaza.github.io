First, the method of transferring PPT to WORD form

one Open the powerpoint presentation using the "Outline" view, click "Outline", click the "Outline" TAB in the left "Slides/Outline" task pane, press "Ctrl+A" to combine the full selection, then use the "Ctrl+C" key combination or right click to select "Copy" command from the shortcut menu, and paste it into Word.

Tip: This method will copy the lines and symbols in the original slide.

2. Open the PPT slide you want to convert using the "Send" function and click the "File "→" Send "→"MicrosoftWord" menu command. Then select the "Use outline only" radio button and click the "OK" button, and wait a moment to find the entire PPT document in a Word document opened.

Tip: You will find that Word has a lot of empty lines after conversion. To delete all empty lines in Word with the Replace function, press "Ctrl+H" to open the "Replace" dialog box, enter "^p^p" in "Find Content", enter "^p" in "Replace to", and click the "Replace All" button several times. ("^" can be entered using "Shift+6" in English.) 3. Open the slide you want to convert using the Save As direct conversion, click File → Save As, and select Save as rtf in the Save Type list box. Now use Word to open the just saved rtf file, and then make appropriate editing to achieve conversion.

4. PPTConverttoDOC software conversion PPTConverttoDOC is green soft, decompress and run directly, before running, please close both Word and PPT programs. Select the PPT file you want to convert and drag it directly into the "PPTConverttoDOC" program. Click the "Start" button in the tool software to convert, and the program will exit automatically after the conversion.

Tip: If you check "Convert with separator flag", the converted word document will show which page of the original slide the current content is on. After the conversion is completed, you can automatically create a new Word document, showing all the text in the PPT file.

Second, the method of converting PDF to Word

Adobe Acrobat 7.0 Professional is the software for editing PDFS.

Open it with Adobe Acrobat 7.0 Professional and save it as WORD.

Or with ScanSoft PDF Converte, no setup is required after installation, it will automatically integrate into Word. When we click on the "Open" menu in Word, we can see the "PDF" option in the "File Type" drop-down menu of the "Open" dialog box, which means that we can use Word to open PDF documents directly!

The working principle of ScanSoft PDF Converter is actually very simple, it captures the information in the PDF document, separates the text, pictures, tables and volumes, and then unifies them into Word format. Because Word converts PDF format to DOC format when opening PDF documents, the opening speed will be slower than ordinary files. When opened, the PDF Converter conversion progress is displayed. After the conversion, you can see that the text format and layout design in the document have maintained the original flavor without any changes, and the tables and pictures have been completely saved and can be easily edited.

in addition to being able to Open the PDF document directly in Word, right-click the PDF document and choose "Open PDF in Word" command from the pop-up menu to open the file. Sometimes we search for a PDF file on the Internet, and we can also open it directly in Word through the relevant commands of the right-click menu.

Three, Word into the method of PPT

We usually use Word to input, edit and print materials, and sometimes we need to make PowerPoint presentations that have been edited and printed for presentations and lectures. If you re-enter it in PowerPoint, it is a hassle and a waste of time. If in between, by copying and pasting pieces, one by one to make a slide, it is also more difficult. In fact, we can use PowerPoint outline view to quickly complete the conversion.

First, open the Word document, select all of them, and execute the "copy" command. Then, launch PowerPoint, if it is Word 2002, select the "Normal" view and click the "Outline" TAB (Figure 1); If there are no "Outline" and "Slide" tabs, the method of display is to click "Normal (Recovery pane)" on the "View" menu or in the lower left corner of the window, click the "Normal View (Recovery pane)" button; If it is Word 97/2000, you can directly select the outline view, position the cursor to the first slide, and execute the "paste" command to insert the entire content of the Word document into the first slide. Then, the text format can be set according to the need, including font, size, font, color and alignment of the word; Then move the cursor to the next slide and press Enter to create a new slide. If you need to insert a blank line, press Shift+Enter. After adjustment, you can quickly complete the production of multiple slides (Figure 2). Finally, you can also use the Outline toolbar to make further adjustments by using the "Upgrade", "downgrade", "Move up", "move down" and other buttons.

On the other hand, if you are converting a PowerPoint presentation into a Word document, you can also use the Outline view to do it quickly. The method is to position the cursor at the beginning of the other slides except the first slide, press BackSpace (backspace), repeat several times, combine all slides into one, and then select them all, by copying and pasting them into Word.

5. PDF to BMP

Method 1: Use PDFTOOLS

PDF is a universal format developed by Adobe as a portable electronic document in the world, which can correctly save the font, format, color and image of the source file, so that the exchange of files can easily cross the restrictions of the application and system platform. However, some document-specific operations (such as extracting a piece of text or an image from a PDF document) are not so easy for the average user to use. But now with the amazing tool PDFTools, you can easily convert the contents of PDF documents to HTM, TXT, and BMP three commonly used file formats.

In order for PDFTools to serve us better, we first need to set it up. In the main interface, click the menu "File → Settings" or directly press "Ctrl+D" to bring up the Settings dialog box. In the "General Settings" TAB, set the conversion range of the page, the default is all conversion, you can click the drop-down button, select "select specified page", enter the start page and stop page of conversion. The unregistered version can only convert the first five pages of a PDF document.

The content of the "Output Settings" box is to set the pixel and JPEG image quality of the input picture. Check the "Browse after conversion" check box, then when the document conversion is complete, the program will bring up the corresponding program to open the converted file. The "PDF2HTM" TAB has only one "Use full Processing mode" check box, depending on the situation.

6. PDF to HTM method

HTM is a common format for web pages, but after PDFTools converts HTM files, the formed HTM files do not contain text in PDF documents, but convert the contents of PDF documents into JPG image files, and then use HTML language to embed the pictures into the web page. The resulting HTM file is a frame page, that is, the left side is the link window of each page, and the right side is the JPG image. If you are using Internet Explorer 6, when you move the mouse over the right window, a row of shortcut toolbars will appear.

First select the menu "Tools →PDF to HTM" to make it into HTM mode. Then open the file has a variety of operations: through the menu "File → open" directly press the "Ctrl+F" key to open the dialog box to select the file, or directly drag the PDF document to be converted to the PDFTools main interface, the software immediately began to convert. The converted file is saved in the source file, and there are subfolders with corresponding names, which are some JPG images and other resources. Finally pop up a prompt window, click "OK" can be.
