LumaQQ for Mac 20070601 Children's Day version update

by yinyin on 2007-06-01 19:06:09

Miss your childhood? Come on! LumaQQ Mac OS X Children's Day Edition!! ... At least this will not let you relive the feeling of childhood.

1. Fixed the issue of not receiving messages from the mobile QQ client; mobile QQ refers to using the QQ client on a phone, not replying to QQ text messages via phone. Oh, I always thought you were talking about something else.

2. Disabled the automatic hiding of the main window when a sheet window is open.

3. Corrected the problem of incorrect ID display in group information windows.

4. Fixed the issue where clicking the temporary group icon in the temporary group chat window would bring up a non-temporary group information window.

5. Completed iTunes QBar. The iTunes QBar can display the music information you're currently playing and provides simple controls with four buttons for iTunes. One feature that QQ doesn't have is: it can instantly append the music you're currently listening to in your personal signature, just like MSN does. Of course, MSN’s feature is stronger as it can also be clickable.

6. Corrected the logic error of the "Clear" menu item in the search friends menu.

7. Updated the Pure IP database to the 2007.05.20 version.

8. Added a display bar at the bottom of the friend chat window to show the friend's personal signature.

9. Fixed some bugs related to proxies; now Socks proxy should work normally.

10. Fixed a potential crash bug in the network layer, which may help improve stability.

11. Corrected the issue where the status menu in the English version was in Chinese.

12. Fixed the problem of abnormal display in the third column of the QBar selection window in the English version.

13. Added IPSeeker QBar, providing a simple tool to query IP locations.

14. In tabbed chat windows, pressing Cmd+W will close the current tab instead of the entire window.

15. Fixed some bugs in displaying levels in the level bar, such as not showing one sun for level 16.

16. Fixed the issue of not prompting when the other party refuses to add you as a friend.

17. Fixed the issue of not prompting when the other party refuses to add you to a group.

18. Added sound theme settings in preference sound settings, increasing the types of sound prompts to 11.

19. Added the default QQ sound theme.

20. Added the Super Mario sound theme.

21. Fixed some small bugs in sound playback.

22. Updated the default QQ sounds, as the previous ones had issues.

23. Added the TM sound theme.

24. Added the iChat sound theme.

25. Set the minimum tab width for tabbed chat windows to prevent inconvenience when selecting tabs due to too short friend nicknames.

26. Added an option in preferences to "Not display floating user list tip windows."

27. Improved the custom emoticon import function, fixing the issue of being unable to import super-large emoticon packages.

28. Fixed the issue where operations continued even after canceling during custom emoticon import.

29. Fixed the bug where deleting custom emoticon groups sometimes failed.

30. Fixed the bug where table content didn’t refresh when creating new custom emoticon groups.

31. Completed the floating tip window.

32. When using tabbed chat, Shift+Cmd+W can close the entire window, and Cmd+W can close the current tab.

33. Added Growl support.

34. Added send buttons in normal chat, group chat, and mobile chat windows.

35. Fixed the issue of multiple online notifications after login.
