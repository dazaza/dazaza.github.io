Web 2.0 Domain Generator!

by xjyzhenai on 2008-09-16 10:53:25

Web 2.0 services have become commonplace. The domain name of a Web 2.0 service is so important that it directly affects the user experience. Owning a good domain name that corresponds to a Web 2.0 service should be the first thing to do before launching a project. But now, with so many existing Web 2.0 services, you might rack your brains and still not come up with a good domain name. Let's just wait for a stroke of genius :)

Now, there is a tool that can help you generate a large number of Web 2.0 domain names. Let's take a look at what it can do.

[Image]

The functionality is very simple, with just two buttons. One generates names, and the other checks the domain status. First, let's generate a domain name suitable for yourself. For example, Aimbu in the image above. The name of a Chinese service could be phonetically translated as "Ai Bu" (爱布).

Then, by clicking the "Check Availability!" button, you can check whether domain names related to Aimbu are available for registration.
