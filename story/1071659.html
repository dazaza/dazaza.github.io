化石燃料 Facebook 广告在一年内被浏览了 4.31 亿次

2021-08-06 06:54:11

根据一项新研究，纽约（CNN 商业）大型石油公司战略性地利用 Facebook 向美国人发出源源不断的信息，旨在延缓化石燃料使用的灭绝。 InfluenceMap 周四发布的一份报告发现，仅 2020 年，在 Facebook（FB）美国平台上，化石燃料广告的浏览量就超过了 4.31 亿次。根据专注于能源和气候变化的智囊团 InfluenceMap 的数据，尽管覆盖面很广，但石油和天然气行业在广告上仅花费了 960 万美元。报告发现：“石油和天然气行业正在使用更复杂的剧本来破坏气候行动，这涉及使用更微妙和微妙的信息传递策略。”在 InfluenceMap 研究的 25 个组织中，Facebook 美国平台上付费广告的最大用户是埃克森美孚 (XOM) 和该行业强大的贸易组织美国石油协会。在研究人员分析的广告中，埃克森美孚和 API 占了惊人的 62%。埃克森美孚在给 CNN Business 的一份声明中说：“这些断言是故意误导，完全没有根据。” “我们相信，当各种知情的声音参与政治进程时，就会实现健全的公共政策。出于这些原因，埃克森美孚行使其支持和参与政策讨论的权利。” API 在给 CNN Business 的一份声明中表示，其“为辩论提供信息的工作”以政府和独立研究为基础并得到验证。

“我们在社交媒体上发布的能源素养帖子只是我们公司每天对突破性技术研究进行的大力投资的一小部分，以塑造低碳未来——诸如甲烷捕获、氢气、碳封存等努力，”高级工程师梅根·布卢姆格伦写道。 API副总裁。该报告发布之际，Facebook 仍因其在传播错误信息（包括有关 2020 年大选的谎言）中的作用而面临压力。就在一周前，埃克森美孚的一位高级说客似乎在一段卧底视频中透露了这家石油公司如何利用其政治力量削弱气候行动。埃克森美孚首席执行官达伦伍兹谴责并为录音中的评论道歉。 InfluenceMap 发现，2020 年 7 月 15 日，化石燃料集团在美国 Facebook 广告上的支出急剧增加——当时的候选人乔拜登宣布了他的 2 万亿美元气候计划。拜登的提议呼吁到 2035 年结束发电厂排放，并大幅减少化石燃料消耗以应对气候危机。 “该行业正在战略性地使用社交媒体，并在关键的政治时刻部署其广告，”InfluenceMap 报告称。这些广告避免了直接否认气候。研究人员发现，这些信息试图通过更细微的信息传递策略来“延长石油和天然气的使用”。广告中采用的一种策略是将石油和天然气的使用与维持高品质生活联系起来。另一种策略是提请注意该行业针对气候变化采取的自愿措施。

该报告将 6,782 条宣传天然气的广告确定为“清洁或绿色”——并指出，Facebook 没有将这些广告作为错误信息或误导性内容删除。针对该报道，Facebook 发言人指出，化石燃料广告并非社交媒体独有。 “虽然像这样的广告在许多平台上运行，包括在 CNN 等电视网络上，但 Facebook 提供了额外的透明度，要求它们在发布后最多 7 年内在我们的广告库中向公众提供，”Facebook 发言人说在一份声明中说。 Facebook 补充说，当独立的事实核查合作伙伴将广告评为虚假或具有误导性时，该公司确实会拒绝广告。 Facebook 发言人表示，该公司拒绝了多个支持化石燃料的广告，因为这些广告是在没有适当的政治广告授权的情况下投放的，并补充说，负责人因此面临限制。研究人员表示，将天然气作为低碳燃料来源“掩盖”了一个事实，即它主要由甲烷组成——一种温室气体，在吸收大气热量方面比二氧化碳强得多。这就是为什么国际能源署在 5 月份表示，世界需要立即停止石油和天然气的钻探，以防止气候灾难。

报告称：“将化石气体作为低碳燃料越来越多地成为监管机构的目标，因为它具有误导性，”该报告指出，2019 年英国广告监管机构如何基于同样的论点向 Equinor 发出警告。然而，值得注意的是，不久前天然气被许多人视为一种过渡燃料，可以缓解煤炭的过渡。多年来，奥巴马-拜登政府一直支持将天然气作为更清洁的替代品。 API 在给 CNN Business 的声明中指出，美国生产、运输和分销天然气，同时满足世界上一些最高的环境标准。 API 的布卢姆格伦说：“全球依赖煤炭的经济体一直将天然气作为更清洁的燃料来源，使美国能够出口气候进展，同时加强我们的国家安全和支持我们的经济。”埃克森美孚将天然气描述为“多功能、丰富的燃料”，自 2000 年以来有助于降低美国的排放量。该公司指出，许多国家和州已将转向天然气作为其碳减排计划的一部分。众议院监督与改革委员会环境小组委员会主席、民主党众议员 Ro Khanna 就报告中的调查结果猛烈抨击石油和天然气行业。 “几十年来，化石燃料公司在其产品构成的真正危险方面误导了公众、监管机构和国会，”卡纳在给 CNN Business 的一份声明中说。 “这份报告证明了我们的知识，即该行业的虚假宣传活动还很活跃。”

卡纳重申，他计划在未来几个月内要求埃克森美孚、雪佛龙 (CVX) 和其他化石燃料公司的首席执行官在他的小组委员会作证。 InfluenceMap 指出，尽管社交媒体公司已公开承诺支持应对气候变化的努力，但 Facebook ( FB ) 继续从石油和天然气行业获得广告收入。尽管 Facebook 从报告中涵盖的化石燃料广告中获得了 960 万美元的收入，但研究人员表示，真实数字可能更高。
