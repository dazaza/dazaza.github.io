五款Win7免费优化应用

by @geekzhang on 2010-12-28 12:51:15

据国外媒体报道，Windows 7作为微软最新的主打产品，经过一年多的努力已经取得了不错的成绩，超越Vista成为业界排名第二的操作系统产品。

　　从分析师和用户的反馈信息来看，较大比例的Win7用户对其是比较满意的，但这并不意味着Win7就是完美的，对于希望打造自主的系统和优化设置要求的用户来说，Win7的自带功能并不完善。下面是科技博客Lifehacker为大家推荐的五款免费系统辅助工具，仅供参考：

　　God Mode

　　支持平台：Windows自带

　　资费情况：免费

　　微软Windows 7系统中隐藏了一个秘密的“God Mode”小功能，也可以称为“上帝模式”。

　　实际上，God Mode就是一个简单的文件夹窗口，但其中包含了几乎所有Windows 7系统的设置，如控制面板的功能、界面个性化、辅助功能选项等方方面面的控制设置，用户只需通过这一个窗口就能实现所有的操控，而不必再去为调整一个小小的系统设置细想半天究竟该在什么地方去打开设置窗口。

　　您可以在任何地方创建一个新文件夹，然后最重要的是要将这个个新文件夹重命名为“GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}”，最后回车就会发现惊喜。双击打开God Mode窗口，您就可以看到这里包括了方方面面的系统设置选项和工具。

　　Regedit

　　支持平台：Windows自带

　　资费情况：免费

　　Windows的注册表是一个宝库，他实质上是庞大的数据库，它存储的内容有：软、硬件的有关配置和状态信息，应用程序和资源管理器外壳的初始条件、首选项和卸载数据；计算机的整个系统的设置和各种许可，文件扩展名与应用程序的关联， 硬件的描述、状态和属性；计算机性能纪录和底层的系统状态信息，以及各类其他数据。

　　运用注册表可以对系统进行相当深度的优化，但不推荐新手使用。

　　Windows 7 Manager

　　支持平台：Windows

　　资费情况：40美元高级版

　　Windows 7 Manager是一个专门为windows 7设计的系统优化工具，为windows 7系统提供完整的解决方案，包括系统优化，调整和清理等等，集众多功能于一身的实用工具，它包括了30种不同的小工具，以帮助使您的系统提高速度，增加稳定性和安全性。

　　Rainmeter

　　支持平台：Windows

　　资费情况：免费

　　Rainmeter原本是一个系统状态监视软件，但现在它的功能已经得到了极大的扩展，rss feeder、邮件监视器等常用功能都能够实现，而低资源占用和简易的设计方式为它赢得了大批用户。Rainmeter的皮肤大都走抽象、简洁的设计路线，非常适合作为一款桌面美化软件。本来占用的资源不多，不过由于有一些ACG的饭给它做了皮肤，现在就比较耗资源了。

　　Ultimate Windows Tweaker

　　支持平台：Windows

　　资费情况：免费

　　很多人对于“终极Windows优化工具”并不陌生，它的体积只有几百K却能提供上百项个性化设置和优化辅助功能。新版本配合Windows7特别设计，支持32/64位系统。
