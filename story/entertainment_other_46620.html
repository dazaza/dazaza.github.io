PSP 3.90 M33 Release?

by gogole111 on 2008-02-02 12:02:55

Dark Alex officially released the custom firmware 3.90m33, with the following improvements in the new version:

- 3.80 -> 3.90 - m33 no-disc: Fixed the issue of getting stuck when exiting using 'home'.

---

### Installation:

- Requires a PSP with firmware 3.52 m33 or higher.

- Copy the "update" folder to /psp/game/.

- Download Sony's official 3.90 update file, rename it to E390.pbp, and place it in the same folder.

- Alternatively, you can download the installer via Wi-Fi.

- Run the program. After installation is complete, press X or O to restart your PSP.

### Updates:

- 3.80 -> 3.90 - m33 no-disc: Fixed the issue of getting stuck when exiting using 'home'.

- Other updates:

- Automatically checks the IDStorage key, and if there's an issue, it automatically repairs it.

- Added the ability to download the 3.90 PBP file via Wi-Fi.

- Note: This feature is not 100% stable yet. When selecting AP (Access Point), it might crash. If this happens, manually restart and reinstall.

- L + △ can skip battery checks.

- If you already have E390.pbp, hold the button until you see "Verifying E390.pbp".

- If E390.pbp was downloaded via Wi-Fi, hold the buttons when the download finishes.
