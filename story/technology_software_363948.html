Mini Feixin Beta version is officially released

by yinyin on 2010-09-06 20:01:37

It is an instant chat tool developed by China Mobile for "light users" of Fexin to meet their simple needs such as sending and receiving messages. This product is suitable for Feisin users who prefer simple pages and only need short messages. It is based on the simplified development of Feishin PC client SDK, retains the core communication capabilities, has the characteristics of lightweight (flexible), agile (fast), green (pure), and allows users to feel the core functions of Feishin "convenient SMS" anytime and anywhere.

Light, agile, green program small size, fast running, very low occupancy of system resources, green free installation, decompression can be used.

The main interface can be hidden by sticking to the edge of the screen, which is quiet and uninterrupted.

Message session mode Simple and efficient message session mode, easy to send and receive.

Send SMS to non-friends Enter mobile phone number to send SMS to non-friends, easily.

The global top supports the main interface to be displayed at the front end of all Windows for easy viewing.

# Support shortcut keys Use shortcut keys to complete all conventional operations, free the mouse.

Add friends Enter mobile phone number to add friends, simple and refreshing.

Quick Search You can quickly search your friends by initials, pinyin, nickname, or mobile number.

Disadvantages: No support for chat record saving
