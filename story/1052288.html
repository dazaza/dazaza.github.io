跳转到导航跳转以搜索相对于难以访问的地理标准的杆子，标志着根据该标准达到最挑战的位置。通常它是指来自海岸线的最遥远的点，暗示了最大程度的大陆或海洋。在这些情况下，可以将无法访问的极点定义为可以在不遇到海岸的情况下在感兴趣区域内绘制的最大圆的中心。在不精确地定义海岸的情况下，极点将类似地不精确。

北极极难以接近的，有时称为北极极，或者只是北极杆，位于北冰洋包装冰上，远离任何土地群众。它位于85°48'n 176°9'°9'W / 85.800°N 176.150°W / 85.800; -176.150（真正的北极端子不可接受），1,008公里（626英里）来自三个最近的土地：Ellesmere Island，Komsomolets Island和Henrietta Island。 [2]由于包装冰的恒定运动，该杆上没有永久性结构可以存在。

直到2013年卫星制图审查，北极极难以被认为位于84°03'N 174°51'W / 84.050°N 174.850°W / 84.050; -174.850（北极难以接近的杆），[引用所需]，距离Ellesmere，Franz-Josef Land和新西伯利亚群岛的1,094公里（680英里），[所需的引用] 1,453公里（903英里）北UTQIAġVIK，阿拉斯加，[所需的引用]和214公里（133英里）远离实际的北极极难以访问。 [3]

第一次达到这种虚假北极领域的无法进入的北极是由627年飞机飞行的哈伯特威尔金斯制造的。

1941年，第一个苏联探险使用飞机落地350公里（217英里）的假北极，因此实际上成为达到真正的杆的第一次探险。

忽视苏联结果，威尔伯特·赫伯特爵士被声称是第一个达到的东西被认为是徒步的北极领域，由1968年被守卫抵达。普罗兰吉姆麦克奈尔在2006年宣称，赫伯特·赫伯特并没有达到杆子，而且推出了自己的不成功尝试达到它。 [4]

1986年，由Dmitry Shparo领导的苏联极性科学家的探险队在极地夜晚达到了徒步脚的止痛点。 [所需的引用]

截至2010年2月，麦克尼尔表示，没有人达到虚假的北极领杆即将到来。 [5]

南极难以进入的是南极大陆最远的南洋大陆。该杆已经给出了各种坐标位置。差异是由于＆＃34;海岸＆＃34的问题被测量到接地线或冰架的边缘，难以确定＆＃34的位置;坚实的＆＃34;海岸线，冰盖的运动和多年来调查数据的准确性以及可能的地形错误。

难以进入的杆通常是指下面提到的苏联研究站的场地，其在82°06的54°58'e / 82.100°S 54.967°E / -82.100中构造; 54.967（难以访问（WMO））[6]（虽然有些源给出83°06的54°58'e / 83.100°S 54.967°E / -83.100; 54.967（南极难以访问）（ iphc））[7]）。这距离南极878公里（546英里），高度为3,718米（12,198英尺）。使用不同的标准，Scott Polar研究机构在85°50的65°47'e / 85.833°S 65.783°E / -85.833; 65.783（南极难以访问（SPRI））。 [8]

根据ThePoles.com的说法，距离海洋占南极地表的重点是82°53'14的55°4'30“E / 82.88722°S 55.07500°E / -82.88722; 55.07500（远离海边的点），冰盖被考虑到最远的点是83°50'37“S 65°43'30”E / 83.84361°S 65.72500°E / -83.84361; 65.72500（距离海计数冰最远的点）。由英国南极调查计算的这些要点被引用为＆＃34;最准确的措施可用＆＃34; （截至2005年）。 [9]

南极难以进入远远远远甚至难以到达地理南极。 1958年12月14日，由Yevgeny Tolstikov领导的国际地球物理年度研究工作的第3苏联南极探险，在82°06的54°58'e / 82.100°S成立了临时止痛部（Polyus Nedostupnosti）的临时杆54.967°E / -82.100; 54.967（难以访问的站点）。第二次俄罗斯队于1967年回到那里。今天，这座建筑仍然仍然在这个地点，这是一个面向莫斯科的弗拉基米尔列宁的萧条，并被视为历史地点。

2006年12月4日，N2I团队，由亨利·康森，鲁珀特龙龙，罗兰甜蜜和保罗兰德队组成，走上探险，成为第一个在没有直接机械援助的情况下达到历史悠久的地区的历史终端，而没有传统的男人的组合拖运和风筝滑雪。该队于2007年1月19日到达了旧废弃的车站，重新发现了苏联留下了遗弃的列宁的被遗忘的雕像，在此前48年以前留下了苏维埃。 [10]球队发现，只有建筑物顶部的胸围仍然可见;其余的被埋在雪地下面。 [11]从Vostok Base的飞机从现场拿起探险家，飞往俄罗斯极地研究船只Akademik Fyodorov的开普敦。 [11]

2005年12月11日，7:57 UTC，RamónHernandodeLarramendi，Juan Manuel VIU和Ignacio，西班牙Transantarctic Expectition的成员，历史上第一次达到82°53'14的南极止痛“55°04'30”E / 82.88722°S 55.07500°E / -82.88722; 55.07500（英国南极调查 - 认可的守人不可思议），在英国南极调查中那一年更新。该团队继续走向第二个南极的难以进入的途径，这是一个占冰货架以及大陆土地的人，并且他们是2005年12月14日在83°50'37的第一次达到它的远征“65°43'30”E / 83.84361°S 65.72500°E / -83.84361; 65.72500（英国南极调查 - 认可的杆子不可接受）。两项成就都在东南南极高原的一个雄心勃勃的先驱越过，诺沃佐拉西斯卡亚基地开始，在超过4,500公里（2,800英里）之后以进步基地结束。这是没有机械辅助工具实现的最快的极地旅程，平均速度为每天约90公里（56英里），每天最多311公里（193英里），使用风筝作为其电源。 [9] [12] [13] [14]

2011年12月27日，Sebastian Copeland和合作伙伴Eric McNair-Laindry也达到了82°06的54°58'e / 82.100°S 54.967°E / -82.100; 54.967（守人不可接受站）南极难以实现。他们是第一个没有补给或机械支撑的那样，从诺沃佐拉涅斯卡在前往南极的路上离开，以完成南极洲的第一个/西路穿过两极，超过4,000公里（2,485英里）。 [15]

难以接近的海洋杆（48°52.5的123°23.6'w / 48.8750°S 123.3933°W / -48.8750; -123.3933（海洋杆的难以访问））[17]是海洋中的地方离土地最远。它位于南太平洋，距离最近的土地2,688公里（1,670英里）：Ducie Island（Pitcairn Islands的一部分）到北部，Motu Nui（复活节岛的一部分）到东北部，靠近较大的Siel岛，距离南部的Marie Byrd土地的海岸，南极洲）到南方。该地区是如此偏僻，与居住地区有超过400公里的位置 - 有时，最接近的人类是宇航员在它通过开销时乘坐国际空间站。 [18] [19]

它周围的观点和地区吸引了文学和文化的关注。它被称为点Nemo，Nemo是拉丁语和＃34;没有人和＃34;而且还参考Jules Verne＆＃39;船长Nemo。 [18]综合领域在1928年短篇小说中发挥着重要作用，惠普Livecraft在Cthulhu的召唤中，作为持有虚构的R＆＃39; Lyeh的位置，虽然这个故事在官方发现前写了66年点Nemo。 [18]

该地区也被称为A＆＃34;航天器公墓＆＃34;因为在重新进入气氛时已经存放了数百个退役的卫星，空间站和其他航天器，以减少占用居住地点[20]或海上交通的风险。点Nemo相对没有生命;它位于南太平洋的地理位置，阻止了营养成分从到达该地区，到目前为止，从沿海水域营养出来的营养不良。 [18]

在欧亚大陆，难以接近的欧洲欧洲（欧亚人难以接近的欧洲敌人），是距离海洋最远的土地的地方，它位于中国西北部，如果海洋海洋忽视了哈萨克斯坦边境。

在北美，难以接近的欧洲钢管杆在西南南达科他州的松树岭预订，大约11公里（7英里）距离艾伦镇以北约11公里（7英里），距离最近的海岸线（1,030英里）距离最近的海岸线43°22'n 101°58'W / 43.36°N 101.97°W / 43.36; -101.97（北美难以接近的杆）。 [1]据称，加拿大钢管难以接近的恐龙，艾伯塔省59°02'n 112°49'w / 59.03°N 112.82°W / 59.03; -112.82（加拿大难以进入的极点），距离海湾河（该名称的六个加拿大河流之一）流过它的和平河流。 [22]

在南美洲，难以接近的欧陆杆位于14°03的56°51'W / 14.05°S 56.85°W / -14.05; -56.85（南美洲的欧洲欧洲围球），附近的Arenápolis，Mato Grosso，[1]距离最近的海岸线1,504公里（935英里）。 2017年，特纳双胞胎成为第一个徒劳到南美难以移植的冒险者。 [23]

在澳大利亚，难以进入的欧陆杆位于23°10的132°16'e / 23.17°S 132.27°E / -23.17; 132.27（澳大利亚难以访问的欧式杆）[1]或在23°2的132°10'e / 23.033°S 132.167°E / -23.033; 132.167（澳大利亚难以接近的杆子），[24]距离最近的海岸线920公里（570英里），大约是161公里（100英里）的西北西北斯普林斯。最近的城镇是Papunya，北领地，约有30公里（19英里）到两个地点的西南部。

在非洲，难以进入的杆处于5°39'n 26°10'e / 5.65°N 26.17°E / 5.65;来自海岸的26.17，1,814公里（1,127英里），[1]靠近中非共和国的Obo镇，靠近国家＆＃39;南苏丹和刚果民主共和国。

对于特定的海岸线数据集来计算极点。常用的数据集是全局自我一致，分层，高分辨率的高分辨率地理数据库[25]以及OpenStreetMap行星转储。

接下来，必须确定距离功能以计算海岸线和潜在极之间的距离。早期的工作倾向于将数据投入到平面上或进行球面计算;最近，改进的算法和高性能计算允许使用全分辨率数据集和高精度椭圆形计算。 [26]

最后，必须开发优化算法。几个工作[1] [27]使用自适应网格方法。在该方法中，创建例如21x21点的网格。每个点距离海岸线的距离和距离海岸线的距离和离海岸最远的点识别。然后，该网格在这一点上被缩小了一些因素。此过程迭代，直到网格变得非常小。自适应网格方法是有问题的，因为它不保证找到来自海岸线的最远点。更新的方法B9-Hillclimbing，使用随机重启山攀爬，模拟退火和K-D树，以找到具有更大概率的海岸线最远的点。 [26]
