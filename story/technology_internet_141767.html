In fact, there are many debugging tools for testing the loading speed of the website, with different functions, basically need to be used together, and I prefer to use Firebug, although I am not a professional developer.

So today's introduction of Webslug provides us with the following services, and other debugging tools are still a little different;

1.Webslug is working to set up a performance layer experimental base, and make these data public, we can see the fastest and slowest lists on the website, this may be only the first step of Webslug experimental base, I am looking forward to the wonderful future.

2.Webslug has a very popular feature, that is, it does not need to install any software, just need you to provide the URL, everything is done in the browser.

3. When we use other tools, we need to record data to make comparison. Webslug does the comparison work for us, and we don't have to spend any effort to record data.

4. To optimize the website, Webslug can identify the potential problems of our website, but how to improve it? I guess that's the scope of Webslug's business, and that's what they do for a living.
