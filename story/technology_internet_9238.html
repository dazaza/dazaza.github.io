The NetEase Cloud Disk has been updated, with a more optimized interface.

by ivan on 2006-11-18 09:16:52

Today when I logged into my 126 email account, I found that the attached NetEase cloud disk has made some updates in the interface part, and it is indeed more convenient than before.

First of all, the "upload" button was directly added to the top of the page. In the past, you could only find the "upload" button at the bottom of the page. And now the number of files in each folder will be displayed at the end of the folder, which is quite convenient.

The "space information" has been added on the right side of the cloud disk interface, so that you can intuitively understand the space usage and use the search function in the current directory.

This update also made a lot of improvements in file upload. Before, when uploading multiple files, you need to click the "more" button to add the entrance for uploading files. Now, you only need to use one "add file" to deal with the problems of uploading single or multiple files.

In fact, this update is only about some details. Although there is no functional enhancement, in terms of user experience, it has improved a lot. However, compared with websites dedicated to online storage such as Box.net, NetEase Cloud Disk still has a long way to go.
