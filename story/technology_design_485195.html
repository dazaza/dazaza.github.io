1. Method for Entering/Exiting Factory Mode

1) Aging Mode

A. User Remote Control

1) Press the "MENU" key to go to the channel search menu;

2) Press the "CH-" key to select the standby channel number;

3) Enter aging mode by pressing the numeric keys "2 4 8 3", "6483", or "6568"; the screen will display "factory".

B. Using the Factory Debugging Remote Control:

1) Press the "D-mode" (factory key) directly to enter aging mode; the screen will display "factory".

Note:

In factory mode or aging mode, press "SYS", "Sound Effect Mode", "sleep" to directly adjust data; the screen will display the corresponding menu + factory. Press "MENU" to display the main menu for adjusting various data items. All analog quantities are stepped in units of 50, and the channel search speed doubles. To exit data debugging, press "DISP" to exit; the screen will only display "factory".

2) Entering Factory Mode

A. User Remote Control:

1) First enter aging mode, then in aging mode, press "DISP" to exit "factory" and enter white balance mode.

2) In white balance mode, press numeric keys 1, 2, 3, 4, 5, 6, 7, DISP to enter factory mode.

B. Using the Factory Debugging Remote Control:

1) Press the "D mode" key three times (factory key) to enter the factory debugging menu.

3) Exiting Factory Mode

1) Press "DISP" to exit the factory menu;

2) Press the power switch to exit factory mode. However, upon rebooting, the "factory" mode is retained.

2. White Balance Adjustment

A. Method for Entering White Balance State:

1) In aging mode, press the "Disp" key to exit "factory" and enter white balance mode.

2) Press the "Factory" key twice to enter white balance mode.

3) In factory debugging mode, press the numeric key "0" to return to white balance mode.

B. Accelerator Adjustment: In white balance mode, press the "MUTE" or "-/--" key to display a horizontal bright line. Adjust the accelerator so that the bright line just becomes visible, then press "MUTE" or "-/--" again to exit;

C. Manual White Balance Adjustment: Enter the white balance state, use P+/P- to select settings, V+/V- keys to adjust setting values.

Shortcut keys for manual white balance adjustment under the horizontal bright line:

"Numeric keys 1, 4" Adjust red bias voltage

"Numeric keys 2, 5" Adjust green bias voltage

"Numeric keys 3, 6" Adjust blue bias voltage

D. Automatic White Balance Instrument Adjustment:

Debugging Interface: XS102-WB (5V, GND, SCL, SDA)

Bus Release: After entering factory mode by pressing the "Factory" key, press the "AV" key to release the bus control to the automatic white balance instrument. The screen will display "BUS OFF". After adjustment, press "AV" to return to factory mode, then press the "Factory" key once more to return to normal mode.

Note: Shortcut Key Explanation for Factory Debugging

After entering factory mode, some button functions on the remote control will change to accommodate production debugging needs.

1) Press the "Mute" key to flip through debug menus forward, press the "Return" key to flip through debug menus backward.

2) Use the "CH+" and "CH-" buttons on the remote control to select menu items, and use "VOL+" and "VOL-" to change set values.

3) Press "SLEEP" to display six internal test signals from the chip;

PAL System:

Black Field: Check dark balance.

White Field: Check bright balance.

Grid: Adjust PAL linearity, field amplitude. The grid's edges should barely disappear for optimal results.

Crosshair: Adjust PAL field center value.

NTSC System:

Crosshair: Adjust NTSC field center value.

Grid: Adjust NTSC linearity, field amplitude. The grid's edges should barely disappear for optimal results.

4. LOGO Editing Method

Under AV, press numeric keys "2 4 8 3", "6483", "6568" to enter the LOGO setup menu.

Select each item sequentially using the MENU key (ON/OFF 1 SIZE COL MOVE ON/OFF 2 SIZE COL MOVE).

[ON/OFF 1 SIZE COL MOVE] Adjust LOGO1.

[ON/OFF 2 SIZE COL MOVE] Adjust LOGO2.

1) Press the MENU key, when ON turns red, you can adjust LOGO1 or LOGO2; when OFF turns red, you cannot adjust LOGO1 or LOGO2;

2) Press the MENU key, when 1 or 2 turns red, use P+, P- keys to set LOGO parameters, and use V+, V- keys to set LOGO parameter positions.

3) Press the MENU key, when SIZE turns red, use V+, V- keys to set LOGO size;

4) Press the MENU key, when COL turns red, use V+, V- keys to set LOGO color.

5) Press the MENU key, when MOL turns red, use P+, P- keys to set LOGO up/down position, and use V+, V- keys to set LOGO horizontal position. After adjustment, press the SLEEP key to save memory. Press the DISP key to exit;

5. Child Lock Function:

Long press the DISP key for 5 seconds, a lock will appear in the lower left corner of the screen, locking all button operations. After turning off the main power, the panel cannot be turned on, achieving the function of restricting children from watching TV. Only by long pressing the DISP key for 5 seconds again, when a lock appears in the lower left corner of the screen, continue pressing the DISP key for 5 seconds, the lock in the lower left corner of the screen will disappear, representing the cancellation of the child lock function.

6. Single Key Search: Long press the "Menu" button on the panel for five seconds, and it will automatically search channels.

7. Favorite Channels:

1) You can use the setting key (SELECT) and the selection key (SCAN) to set and choose favorite channels.

2) Select your favorite channel and then press the setting key (SELECT); at this point, the station number will be displayed in yellow, indicating successful setting.

3) Press the selection key (SCAN) to switch back and forth between favorite channels.

4) Press the setting key to turn the channel lock on/off; long press the setting key for four seconds to clear all channels. Press the selection key to switch back and forth between favorite channels. There are 8 favorite channels that can be set; following the FIFO principle, after setting eight channels, the newly set channel will cancel the first channel.

(☆ Do not enter the following settings during whole machine debugging ☆)

In the first 7 pages of factory mode, press numeric keys "2 4 8 3" to enter page F8. Simultaneously open pages F8~F15. Press the mute key to flip through the menu forward, and press the return key to flip through the menu backward.
