将 Web 引入世界 30 年

2021-08-07 06:12:51

1991 年 8 月 6 日，蒂姆·伯纳斯·李 (Tim Berners-Lee) 向公众发布了有关他的万维网项目的信息，并向全世界介绍了万维网。 “尝试一下”，蒂姆在他的信息中指出——从那时起，数十亿人已经这样做了。想想您使用网络的频率。您是否对 Web 对您的生活如此重要感到惊讶？世界上有 60% 的人在线。我们每天在通讯、工作、与家人联系、分享知识甚至我们的民主中都使用网络。但是，我们认为 Web 的哪些方面是理所当然的？一方面，它是多么无处不在。 2019 年皮尤研究中心指出：“总体而言，81% 的美国人表示他们每天都会上网。” Statistica 发现：在 2009 年，欧洲互联网接入率约为 63%，在十年内扩大到近 80%。 2009 年，生活在非洲的人口中只有不到 5% 可以访问互联网，而现在整个非洲大陆几乎有四分之一的人可以访问互联网。”亚洲的在线用户数量也在快速增长。截至 2021 年，亚洲约有 26 亿人在线。据 CNBC 报道，2020 年东南亚有多达 4000 万人首次上网，这使互联网用户总数增加到东南亚人口的近 70%。这个纪念日在中国特别有意义，根据孔子的诗句，三十岁生日意味着脚踏实地（三十而立）。自 1990 年代以来，中国一直在互联网上永久存在。自 2000 年中国只有 2250 万互联网用户以来，这个数字在 20 年内翻了近 38 倍。 CNNIC报告称，中国在线用户数量现已达到近10亿。网络技术在人们的日常生活和中国数字经济的快速发展中发挥着不可或缺的作用。万维网联盟一直在网络发展的一些最重要领域开发技术，包括我们在全球大流行危机期间的生活方式；以及我们未来将如何生活。要了解它的重要性，请参见 2020 年的“互联网一分钟内发生了什么”，例如：

Web 的丰富性、健康性和未来潜力与 Web Consortium 自 1994 年由 Tim Berners-Lee 成立以来所做的大量工作有关，以确保它是开放的、可互操作的并且适合所有人。如果很多大公司在新闻中被提及到 Web（对于某些人来说，有些甚至已经成为它的同义词），那是因为他们的工程师团结在我们的餐桌旁。由于 Web Consortium 完成的工作，您可以在任何设备上使用 Web；您可以在任何浏览器上访问网页；您几乎可以使用任何语言在网络上阅读和写作；您在台式机、笔记本电脑、手机、电视（不是单独的孤岛）上体验相同的网络；无论您是盲人还是有移动性或可访问性问题，您都可以使用网络；并且您对网络的使用越来越安全。请阅读我们最新的工作文件状态。网络是历史上最强大的工具之一。尽管许多用户担心网络在社会、政治和技术上被滥用，但网络社区正在网络联盟中努力确保网络确实可以成为一种促进善和连接的力量，我们继续将人类的需求转化为网络技术。网络应该是一个可以帮助人们并提供积极的社会效益的平台……网络应该赋予一个公平、知情和相互关联的社会。它已经并且应该继续旨在为每个人提供交流和知识共享。在 Web 开始发展的 30 年里，很明显，Web 平台的使用方式经常会破坏这一使命。此外，网络技术可以用来造成伤害，这不符合这种社会使命的精神。网络应该是一个帮助人们并提供净积极社会效益的平台。随着我们继续发展网络平台，我们必须考虑我们工作的道德影响。网络必须是好的。我们的愿景是打造一个更具包容性、更尊重用户的万维网：一个支持真实而不是虚假、人胜于利润、人性胜于仇恨的网络。我们将提高 Web 平台的基本完整性。网络不仅在我们生活中的范围和重要性都会增加；它将在对用户的尊重中成长，在用户的信任中成长，在将全人类纳入其用户中成长。网络是为全人类服务的。网络是为了用户的利益而设计的。 Web 必须对其用户是安全的。有一个可互操作的万维网。在我们庆祝 Web 向公众发布时，我们庆祝 Web 和 W3C 的所有成员，他们帮助塑造和发展了 Web 技术，为我们提供了我们今天使用的可访问、国际化、安全的 Web。 30 年来，我们的顾问委员会、技术架构小组、成员和社区致力于确保蒂姆·伯纳斯·李 (Tim Berners-Lee) 带给世界并改变历史和我们生活的 Web 将继续造福于人性。
