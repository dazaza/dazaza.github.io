Linux supports running Direct3D 9 games with performance exceeding Wine.

by anonymous on 2013-07-24 15:05:42

With the help of the new Gallium3D state tracker, Linux desktop systems can now support Microsoft's Direct3D 9 API. Unlike earlier implementations of Direct3D 10/11, the new code is indeed capable of running Direct3D 9 games on Linux, and the gaming performance surpasses that of Wine.

Gallium3D is a free software and open-source OpenGL implementation. In 2010, Direct3D 10/11 was implemented natively on Linux as a Gallium3D state tracker, but due to lack of interest from the upstream Wine development community, the project eventually stalled.

This week, Christoph Bumiller, a developer of the Nouveau open-source graphics driver, announced a Direct3D 9 state tracker. Although support for Direct3D 10/11 was short-lived, Bumiller believes the situation for Direct3D 9 will be much better.

Through Gallium3D, Direct3D 9 support has been added to open-source Linux drivers, allowing games such as *The Elder Scrolls V: Skyrim*, *Civilization V*, *Anno 1404*, and *StarCraft II* to run using open-source Radeon and Nouveau drivers without translating Direct3D commands into OpenGL. Testing has shown that game frame rates (fps) can be up to twice as high as those achieved with the current Wine codebase.
