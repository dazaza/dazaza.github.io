Twitter. Fanfou. What are you doing?

by ofree on 2007-08-05 00:47:25

Twitter has been popular in the blogging community for quite some time now. It offers a new mode of communication called "microblogging," which allows users to post updates directly via instant messaging software like Google Talk or through mobile text messages. This effectively addresses the questions, "What are my friends doing?" and "What am I doing?"

Fanfou, on the other hand, is known as the Chinese version of Twitter. The site administrator only recently started using it. Fanfou's biggest feature is that it allows users to post information via mobile text message, QQ, Gtalk, MSN, etc., while also providing a user-friendly interface and practical sharing functions. It serves as both a microblog and a web bookmarking service, and you can easily add Fanfou to your blog, website, or forum signature.

Zuosa is similar to Fanfou, but with more emphasis on the web bookmarking service. You can use the plugin provided by Zuosa to quickly bookmark favorite web pages. Its features include automatically fetching the title and address of the bookmarked page, saving users time. Zuosa also provides IM and mobile text message functions similar to Fanfou.

For these three services, their functions are largely similar and indeed facilitate online interactions. However, they all face the same issue: spam. Due to the binding of IM software, updates from friends are shared via IM instant messages, inevitably generating spam. If you have too many friends added, a large amount of IM messages can be overwhelming. Fortunately, Fanfou offers a function to disable IM notifications.

Another problem is: information chaos. Taking Fanfou as an example, it does not provide user categorization functions or information search functions; you can only search for friends by username. This makes it very difficult to pick out useful information from a sea of data. Compared to Digg's user-driven model, there are still many areas for improvement.
