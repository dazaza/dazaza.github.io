What Can Save the PC? Five Things Microsoft Could Try

by geekzhang on 2013-05-03 17:00:17

Tencent Technology (Dien Ying) - Beijing time, April 29th message: The sudden emergence of smartphones and tablets has put the Wintel alliance in a difficult situation. To make matters worse, Windows 8, which was released late last year, did not perform outstandingly, casting a shadow over the future of Microsoft and PCs. How can Microsoft turn the tide to save Windows 8 and the PC? ReadWrite website writer Mark Hachman made a summary on this issue.Here is the full text of the article:If all the rumors are not baseless, in order to calm down users' incessant complaints and grievances, Microsoft indeed needs to bring back the Start button and add an option for direct booting to the desktop. Can this really cure the ailing Windows 8?Perhaps, the situation may improve somewhat, but such a move still treats the symptoms rather than the root cause. The problem lies in the PC itself, and the operating system it runs can only helplessly be dragged down. This is exactly the current situation that Microsoft and its Wintel alliance partner Intel urgently need to change.This seems clear enough now: the tile interface, touch-supported start screen, and lack of guidance options to the traditional desktop have scared off many consumers. If it weren't for this terrifying interface, they might actually have upgraded to Windows 8. However, reality does not go as planned; PC users familiar with the Windows 7 and Windows XP interfaces remain steadfast; or they simply turned off their computers in anger and switched to smartphones or tablets.Panic of the Wintel AllianceAll these harsh realities have plunged the once dominant Wintel duopoly into an unfamiliar panic. Microsoft needs a consumer-pleasing operating system, and users need hand-holding guidance. Now, Wintel needs to re-engineer Windows 8 for them. Similarly, Microsoft needs a robust application ecosystem and is actively trying to attract developers to the Windows Store. However, such efforts have also proven unpopular.In the meantime, Intel continues to lower the cost of microprocessor chips to support Windows tablets using its own chips in the battle against Android and iOS manufacturers. Intel executives stated that notebooks using Core during the holiday shopping season will be priced between $499 and $599, while the new more powerful Atom products will be around $200.When you add all these factors together, we seem to have found the cure for PCs and Windows:1. Tone down the Start ScreenIf Microsoft reintroduces the direct boot-to-desktop option, it will undoubtedly fall into an awkward marketing dilemma: should it continue promoting the tiled design of the Start interface and risk annoying some customers? The answer is no. This does not mean that Microsoft needs to change the interface design of Windows 8's Start screen; the added tablet interface should be maintained. But Microsoft should reserve this interface specifically for the Surface tablet; and in advertisements targeting PC users, use the Windows desktop.2. Gently guide new users to the desktopClearly, some of Microsoft's customers have been psychologically scarred by their first impression of Windows 8. True dedication lies in the fact that these users may never return to the Windows camp.However, gently and kindly bringing back the boot-to-desktop option may ease the situation. This option should be placed in the Windows installation options, perhaps accompanied by this statement: Do you want Windows 8 to directly boot to the desktop? The Windows desktop provides a familiar environment for Windows 7 and Windows XP users.From there, let them start exploring and do what they want. If the Start screen is as great as Microsoft thinks it is, some users will eventually choose it.3. Solve the Windows app problemSo far, the problem with Microsoft's Start screen seems unsolvable, with most apps being quite weak (FreshPaint being an exception). When Windows XP and Windows 7 were popular, these mediocre apps could hide behind the Start button, often ignored. But with the arrival of the Windows 8 Start screen, suddenly they were exposed, revealing their true nature instantly. It is unclear whether booting to the desktop will significantly improve the situation.However, by making the Windows 8 desktop the focus, Microsoft can at least promote the rich Windows applications indirectly. Mix and match! Learn from Apple's pages. Highlight stunning apps, whether they come from the Windows 8 world or predecessor Windows 7. Supporting older operating systems is actually a feature selling point. Free marketing promotion for Adobe, EA, and other developers can win Microsoft some goodwill.4. Let Windows shine in the tablet market inexpensivelyMicrosoft desperately needs a successful mobile strategy. And the only effective way is to offer better value at a lower price.In other words, if Microsoft wants to apply Windows to the mobile field, it needs to truly use Windows well. The failure of the Windows RT version of Surface can partly be attributed to it being just a truncated version of Windows 8; it's time for it to retire. If the price is low enough, the Windows Pro version of Surface can achieve much in the market. Microsoft needs to convince buyers to experience the full version of the Windows tablet and give them the feeling that the Surface can do much more than other tablets.Microsoft needs to prove that the Surface tablet using the new quad-core Bay Trail chip can provide PC-level services and experiences at the price of a tablet. As everyone knows, tablets are portable. The Microsoft Store needs to arm Windows tablets with applications, showcase the coolest apps, and label them with prominent price tags. Convey this message: All this for just $299? I actually thought about buying an Android tablet?5. Find the Tiger Team for Mobile AppsEvery day, stuffing an Android or iOS phone into our pockets has become a habit of ours. As more and more game developers choose to design games for iOS or Android, the Windows platform appears deserted. iOS and Android platforms have another crucial advantage: you have the chance to play the same game on your iPad and iPhone, or on Android phones and tablets. But you can't do that on Windows Phone and Surface.If users cannot share applications, files, and other content between PCs, laptops, tablets, and smartphones, they will quickly look elsewhere. Microsoft clearly has already realized this, as Office and Xbox address this disadvantage. Netflix also broke through Microsoft's platform boundaries, but all efforts stopped there.It takes more than one day for ice to freeze three feet deep. If Microsoft cannot develop the applications it needs, it will inevitably fade out and have to invest money to purchase them. Obviously, this is the current predicament facing Nintendo. Without AAA third-party software, Microsoft will end up walking alone.Delay is inevitableIDC research data is correct: PCs are dying, and it is unavoidable. What Microsoft is doing is merely delaying, buying time for the entire industry, but the moment will ultimately come ruthlessly, like rearranging the tableware on the sinking Titanic. But under these circumstances, the big ship of PCs still has a chance to return to the harbor before sinking.No matter whether the tablet comes with a keyboard or not, the notebook will eventually make way for this upstart. Microsoft won the desktop market, then monopolized the notebook. Now, it must compete for tablets too. If Microsoft shows signs of fatigue now, it will eventually decline and be buried by the times.Can Microsoft throw enough money at these problems to save the PC? There is no other choice for Microsoft. It's not too late to refine Windows 8 now, and Intel can continue to cut costs. However, the app and mobile issues require deeper solutions. Now is the time for action.
