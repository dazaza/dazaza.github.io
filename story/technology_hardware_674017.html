by z54689 on 2012-02-24 09:47:17

POS Machine, also known as IC Card POS Machine or IC Card Fee Collection Machine. The IC Card Meal Fee Collection POS Machine is a new generation of intelligent card equipment developed to meet the requirements of electronic, institutionalized, and convenient consumption. It can be widely applied in government, enterprise, factory canteens, restaurants for IC card fee collection, campus one-card system, enterprise one-card system, and member consumption one-card system.

The meal machine replaces the traditional cash consumption method, using the advanced MIFARE I type contactless IC card consumption method. Contactless IC cards have advantages such as safety, stability, and convenience in carrying. Users only need to swipe their card within the effective distance of the POS machine's induction area for the POS machine to recognize it and complete the consumption; there's no need for cash transactions, avoiding the inconvenience of giving change and hygiene issues caused by cross-infection from cash transactions. This not only makes it convenient for users to consume but also facilitates the statistics and management of consumption situations by the administrators, achieving the automation of data collection statistics and information processes. The IC card system is safe, stable, and reliable with excellent product performance.

System Usage Introduction

Step 1: Input the charge machine number and group it (or run the charge machine test to automatically complete the number addition).

Step 2: Change the administrator password to ensure software security.

Step 3: Run the charge machine registration, change the system password, register all charge machines. The password is a 12-digit hexadecimal number, write it down and keep it properly, never lose it.

Step 4: Initialize all consumption cards so that the card password matches the system password.

Step 5: Register the consumption card, input the cardholder's information, etc.

Step 6: If multiple people operate the software, add administrators and authorize them according to their respective responsibilities.

Step 7: Carefully check the time of all charge machines, which can be automatically completed by the software.

The smart IC card meal fee collection management system consists of the following parts:

2. System Operating Environment

The system adopts a bus-type + star-type network topology structure, with the client using windows9x/xp/2000. The server end uses Windows2000Server, SQL_SERVER2000 central database integrated way with common interconnected software technology.

3. Communication Protocol of IC Card Meal System

The system is based on the Ethernet client/server architecture, using Ethernet and TCP/IP network communication protocols, adhering to standard TCP/IP protocols for network data transmission. It interacts with other systems at the same network layer, where the data collection adopts the standard TCP/IP protocol, fully utilizing the wiring system for high-level, reliable data interaction and communication. Structurally, it ensures reliable and convenient data interfaces, system scalability, and the realization and stable operation of redundant systems.

2. Implementation Plan Introduction

This system plan is designed according to the characteristics of the user unit and to meet management requirements, consisting of a system management center and a canteen restaurant induction card cash register system connected via RS485 network cables. The system composition is concise and clear, reasonably matched, easy to operate, economical, and practical.

1. System Management Center

● Task

Mainly responsible for selling cards, recharging, reporting lost cards externally, and managing system operations internally, including data collection, parameter download, information inquiry, consumption statistics, account summaries, and report printing.

● Equipment Composition

The system management center consists of a computer (with dual hard drives) equipped with management software, a printer, and a card issuance machine. One of the hard drives serves as a backup.

● Location Setting and Operation

Set up the system management center (commonly known as the "computer management center") in the canteen restaurant management office (or a dedicated room), managed by a dedicated system operator. Multiple operators can be established (logging in through individual passwords) and permissions can be set to restrict certain functions. To prevent computer viruses and software conflicts, no software unrelated to this system should be installed on the computer, otherwise, consequences will be borne by the user!

2. Induction Card Cash Register System

● Task

The induction card cash register primarily handles accepting meal card swipes and can also query the remaining balance on the card.

● Equipment Composition

Composed of several wall-mounted induction card cash registers and desktop induction card cash registers (can be used offline and moved around).

● Installation

Install the switch power supply in a dry, bright, and easily accessible location in the canteen restaurant selling area. Wall-mounted induction card cash registers are fixedly installed on the selling windows in the canteen restaurant. Desktop induction card cash registers are installed on the selling counters, not fixedly installed, facilitating temporary movement, but the selling counter must be equipped with a network cable connection port and a power socket.

● Operation

The induction card cash register is operated by the selling staff in the canteen restaurant. Operators must strictly follow the operating instructions and avoid using sharp objects or rough handling. In case of machine problems, notify the manager immediately and do not handle it yourself.
