基于 FreeBSD 的类似 macOS 的操作系统

2021-08-07 00:08:48

Airyx 是一个新的开源 OS 项目，旨在在 x86-64 系统上提供类似的体验和与 macOS 的一些兼容性。它建立在 FreeBSD 的坚实基础、同一空间中现有的开源包以及填补空白的新代码之上。与 macOS 应用程序的源代码兼容性（即您可以在 Airyx 上编译一个 Mac 应用程序并运行它）类似的 GUI 隐喻和熟悉的 UX（文件管理器、应用程序启动器、反映打开应用程序的顶部菜单栏等）与 macOS 文件系统（HFS+ 和APFS) 和文件夹布局（/Library、/System、/Users、/Volumes 等）在文件夹或单个文件中的自包含应用程序和（主要）无安装程序的 /Applications 体验主要保持与 FreeBSD 基本系统的兼容性和 X11 - 引擎盖下的标准 Unix 环境 如果没有 Cirrus CI 对 CI/CD 和 Cloudsmith 对包托管的慷慨支持，这个项目是不可能实现的！感谢这些很棒的组织支持开源软件！
