And today, an epoch-making domestic WebOS - building blocks, aroused the Chinese people's strong interest in WebOS.

image

Building Blocks online computer is a new generation of open online software application platform for individual users. Building Blocks Online computer to improve the user experience, provide more interactive, personalized services and products as the core purpose, to provide users with on-demand "online personal computer" services. At the same time, the system also provides an open API to bring users a new network experience. Let users according to their own needs, to design and build their own "high-rise buildings".

The interface of WebOS usually gives users the most direct experience. The interface of the building blocks is very similar to our common WINXP platform, and multiple sets of backgrounds and themes are built in for users to choose from.

image

What users are most concerned about is what can the building blocks do? We know WebOS as a Web-based operating system. So it still needs real operating system and network support.

The building blocks are built with a variety of applications and common tools to help you with everyday tasks such as word processing, email, online photo albums, listening to music and watching movies, especially if you only have a bare system. Moreover, the building blocks also support the custom addition of many third-party applications. This is like our operating system, the system simply provides a platform with an interface, and users can install third-party software themselves.
