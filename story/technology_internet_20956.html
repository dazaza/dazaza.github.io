An incomplete list of network hard disk services

by yinyin on 2007-05-31 16:18:30

ilesfly- online transfer, share, backup file website, a single file maximum 300M, no format restrictions, do not need to register, speed is good. Supports email notification.

The feature of localhostr is that you can share files in a wide range of ways, with only three steps (select, upload, get the URL).

Upload speed can also be, file size 20MB, any file can be uploaded.

MediaFire: Upload files in any format, size, or download volume, does not provide direct file links, and provides download pages like Live Share and rapidshare.de. Test upload speeds were OK, slightly slower than Box.net and slightly faster than localhostr.

Online file storage service provider Live-Share: For unregistered users, a single uploaded file cannot be larger than 300M. If no file is downloaded for 4 consecutive weeks, the file is deleted; For registered users, a single uploaded file cannot be larger than 500M. If no file is downloaded for 8 consecutive weeks, the file will be deleted. Upload speed :40KB/s.

4Shared: The free version of 4Shared offers 1 gigabyte of storage, allows uploads up to 50MB of a single file, unlimited access and bandwidth, and provides local upload tools. Chinese file names are supported. In addition, it also provides multiple file upload methods, file details page and preview function. Its sharing function is also quite powerful, not only provides static folder sharing address, but also can set more permissions for the folder, such as adding access password, allowing upload, rename or delete, display thumbnail and so on, more like an FTP folder. Upload speed is basically maintained between 10-20K, while download speed can reach about 60K.

OmniDrive: The interface of OmniDrive is similar to the style of XP. Omnidrive provides three types of file viewing modes, namely icon mode, detail mode and thumbnail mode. OmniDrive also offers a feature called the Live Folder, a way to synchronize updates to directories via RSS feeds, such as images from Flickr, into a newly created Live Folder. OmniDrive also provides a native client program, available in both Windows and MAC versions, through which you can easily upload and manage files. When the installation is complete, it will generate a new disk called OmniDrive in My Computer, from which you can very easily do the corresponding operations.

Xdrive: Xdrive, AOL's long-rumored web disk service, has begun offering free sign-ups. Prior to the launch, users paid $10 a month for 50GB of XDrive service. This time, the free account offers 5 gigabytes of space. Xdrive also has a file management tool named Xdrive Desktop, download and install to the local system, through this software to back up your local files to Xdrive, and it will also generate a local disk with drive letter, you can directly drag files into the disk to upload to Xdrive, very convenient.

gigasize: huge file movement, transfer, backup website, do not register can send 300M size files, keep 45 days; Registered free account maximum support 1.5G per file, storage for 90 days, no download limit; Paid users ($3 per month) Max. 2.5G, no save time limit. All support automatic generation of blogg and other support for external code and Email sharing references.

Box: Nice, clean and simple interface. Using Ajax, the operation feels very smooth. Upload files can use the drag and drop mode, as long as the file to be uploaded is selected, drag directly to the window can be uploaded operation. Free users have 1 gigabyte of free storage space, 10 MB per file, and do not support sharing. You can upgrade to PREMIUM for free by inviting 5 friends, there is no single file size limit, 5GB of space, and there is a sharing function, and 20GB of monthly data. Recently launched Box/lite, you can no registration, sharing has become easier, upload support for most file types, including audio, video, pictures and documents, a single file size of no more than 10MB. I download the average speed in the education network 100k/s, the fastest 200k/s. This is the main web storage site I use at the moment.

MediaMax: A 25GB super-sized website that provides categories for uploaded content, such as pictures, music, etc., because the site will automatically classify according to the files you upload. And music files are available online. But I tried it and it was a little slow.

450M: Simple interface, fast, suitable for small file storage. After registration, a folder of your own is generated. One interesting feature is the ability to create html pages online.

zupload.com, maximum upload 500M, online upload, no tools.

The interface is simple and the speed is OK.

paid4load.de, the maximum upload can be 100M, there is an upload tool, after the file upload, through the Create Linklist (html) can get the link of the file, you can get a certain income according to the number of times the file is downloaded.

filexoom.com, up to 100M files, unlimited file formats, capacity 2G, very easy to use.

rapidshare.de, known as the biggest one-click storage, 1-Click Webhoster, select the file, click upload on it, and then you will get a download address. You can register with Collector's Zone to manage your own files.

MyBloop, with upload speeds of around 70k/s (Education.net) and download speeds of 40k/s, feels a bit slower than box. You can upload all kinds of files including imgage, audio, video, etc.
