QQ Zone - QQ Tools [Basic Introduction]

1. All data in this software comes from "QQ City Genius."

2. The main function of this software is to search for QQ users, but it's different from the built-in search feature of QQ itself. This software allows you to obtain more information and can conveniently help you make ideal friends.

3. This software also applies to some online promotion groups, but it cannot be used directly for promotion purposes. It only provides a number collection service. (This function is not available in the free shared version.)

[Usage Method]

1. After entering the main interface of the software, choose filtering conditions on the left side. These include location, age, gender, status, and page numbers. Page numbers indicate how many pages of data to collect (all data comes from Tencent's official website), with each page generally containing ten people.

2. Set up the collection conditions, such as limiting by city. You can set a specific city, for example, only collecting data from Suzhou, Jiangsu. Alternatively, you can limit by province without specifying the city. Other limitation settings are similar.

3. After setting up, click the "Start" button at the bottom of the software. Data collection will begin.

4. After the collection is complete, a list will appear displaying various details for each person, including their number, nickname, gender, age, personality, education level, etc.

5. Clicking each record will show the corresponding person's photo, which can also be viewed by navigating with the keyboard's up and down keys.

6. Double-clicking allows you to chat with them. There are two other methods: copying the number via the right-click menu to the box in the upper-right corner and clicking the button, or simply pressing the Enter key.

7. Through the right-click shortcut menu, you can easily and quickly access their homepage or space.

8. Another unique feature is that the right-click menu provides an option to view their QQ photo album. Even for spaces with passwords, you can still access the photo albums (though if the album is encrypted, it won't be accessible).

9. Additionally, data can be exported. This function can be found in the upper-right section of the software and in the right-click menu. The usage method is simple and thus not detailed here. (This function is not available in the free shared version.)

Three, Relevant Notes

1. All collected numbers come from QQ City Genius, and the corresponding information comes from the original data filled out by the users. Therefore, this software cannot guarantee 100% accuracy of the data. However, the authenticity of the data on QQ City Genius is relatively high, with over 80% being real and valid. Photos are mostly taken by live webcams.

2. Some number owners have disabled the temporary conversation function, so it may not be possible to chat with them. Thus, the sixth point above has limitations and only works when the owner hasn't disabled the temporary conversation function.
