Today, I will introduce to you all a very refreshing and environmentally friendly green WordPress blog theme - MRuu. Its grand image illustrations and background layout are impressive. You can download and experience it.

Theme Name: [MRuuSkin 1.0] Green Edition

Creation Time: 2011.06.30

Release Time: 2011.07.25

Author: MinRui

Usage Instructions:

1. Suitable resolution (1024*768 \ 1280*1024 \ widescreen or higher).

2. Suitable browsers (IE6 \ IE7 \ Firefox \ Chrome). Note: The comment input box may be misaligned in IE due to the use of CSS3 for rounded corners when replying.

3. For the top logo section, please modify according to the PSD source file.

4. Download MRuuSkin1.0.zip. After decompressing, there are two files: MRuu2011 (theme) and avatar (avatar cache folder). Upload them respectively to the themes directory and the corresponding root directory of your blog. Upload the avatar cache folder to the root directory of the WordPress program, which is at the same level as the wp-content folder. Otherwise, the comment section will prompt an error.

5. The logo.psd file under the theme folder directory is the source file for the logo position of the corresponding theme. Please keep the size unchanged, and the final file name should be logo.png.

6. Friend link page invocation: Please create four categories in your link classification directory named: Image Link, Text Link, Home Page Image Link, Home Page Text Link. Select the category where you add a link to display it. Those who are interested can modify the desired effect in link.php and sidebar.php by themselves.

7. For the microblog display on the right, please paste your microblog code in the theme settings. Note that the width should not exceed 230px.
