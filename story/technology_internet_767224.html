Microsoft SQL Server 2012 makes data more reliable and transformation simpler.

by anonymous on 2013-10-10 19:18:50

The evolution of IT technology has brought about entirely new working models and gradually reduced hardware costs, making the generation of data more frequent and simpler. Especially in rapidly developing China, the data explosion has made markets and businesses more active. At the same time, the challenges of data insight, mining, and security threats are also increasing, putting many enterprise users under dual pressure of budget and resources.

For IT managers specifically, big data is absolutely not just about piling up numbers; it's about handling and analyzing these data appropriately. A common question here might be: Can big data really be fully trusted?

The answer is a resounding yes, but this largely depends on how well the tools, platforms, IT personnel, and business operations fit together. Outstanding platform tools make business decisions more accurate and intelligent, bringing modernization to business intelligence and helping IT personnel manage more easily, accelerating corporate development and transformation.

Take Guotai Junan as an example. As one of the largest, widest-ranging, and most widely distributed securities companies in China today, the company faces the need for business transformation, requiring deeper customer data analysis and precise marketing. Based on previous positive experiences with SQL Server products, Guotai Junan used Microsoft SQL Server 2012 to build a new retail customer BI analysis system. SQL Server 2012 provided ultra-fast data queries and rapid data discovery for Guotai Junan, offering a reliable basic data platform that provides profound internal insights into overall corporate management and operations, enabling real-time access to customer trading conditions, and analyzing the management, operations, and financial status of various subordinate exchanges from multiple dimensions.

As a key product in the SQL Server family, SQL Server 2012 features AlwaysOn and end-user business intelligence capabilities, providing higher reliability and performance. Enterprise users can use SQL Server 2012's rapid data exploration and data visualization to conduct in-depth research on large-scale data. With PowerPivot developed based on Excel and PowerView, they can easily obtain breakthrough insights from billions of rows of data, making the value of big data credible.

SQL Server 2012 offers comprehensive support and smooth transitions for traditional applications, private clouds, and public clouds, meeting all the needs for database expansion in enterprises. Meanwhile, SQL Server 2022 integrates big data into enterprises, providing the most optimized data environment, helping them gain breakthrough insights, and truly turning challenges into opportunities.

Perfect China Company, which has over 5,600 service outlets and authorized specialty stores, sought new information technology methods to enhance the entire company's management level, operational efficiency, and business insight to address pressures from internal development and external competition. After choosing to use SQL Server 2012, Perfect Company achieved all branches and subsidiaries working on a unified integrated platform, avoiding issues of non-unified data sources. Additionally, the self-service business intelligence function provided by SQL Server effectively assisted the senior management team of Perfect Company to conveniently obtain operational conditions of each branch structure from various dimensions, gaining intrinsic business insights. More critically, the current cloud computing and data scalability characteristics of Perfect Company allow it to arbitrarily diffuse any data to any platform, supporting the company to achieve broader future expansions and maintaining its centralization and competitive advantage in information support.

Zhang Lijun, Chief Architect of Kingdee K/3 Product Department, stated that after adopting the new Microsoft SQL Server 2012, the performance of Kingdee K/3 in critical mission applications significantly improved. With the help of Microsoft SQL Server 2012 Always On, better support was obtained, greatly enhancing the overall availability of the system and improving the efficiency of user data management through database features. Simultaneously, the cloud-ready platform of SQL Server 2012 provided a solid technical foundation for Kingdee K/3 Cloud Computing Solutions.

Li Shaojun, Assistant President of Beijing SuperMap Software, expressed that after adopting the new Microsoft SQL Server 2012, SuperMap GIS users significantly reduced their TCO (Total Cost of Ownership) in data management. In critical mission applications, Microsoft SQL Server 2012 provided better support. AlwaysOn, as a new integrated high-availability disaster recovery solution, can prevent potential planned or unplanned downtime events from negatively affecting critical missions, significantly enhancing the overall availability of the system.

Currently, within China, including Perfect, Kingdee, Yonyou, SuperMap, Guangdong Provincial Higher People's Court, Ctrip.com, China Southern Airlines, China Guangdong Nuclear Power, Super 8 Hotels, Ruijin Hospital, and other solution partners, government agencies, and renowned enterprises have successfully deployed SQL Server 2012. Regardless of the type or scale of the data, SQL Server 2012 provides platforms and familiar tools for users to manage data, achieving actionable insights and assisting in corporate business transformation.
