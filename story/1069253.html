谷歌发布了 Chrome 中隐私沙盒推出的详细时间表，重申其早先宣布的将 cookie 淘汰推迟到 2023 年下半年的计划

2021-07-24 03:24:07

上个月，谷歌宣布将推迟 Chrome 逐步停止支持第三方 cookie 的计划。作为其中的一部分，该公司提供了有关隐私沙盒的更多详细信息，并于今天发布了推出时间表。正如承诺的那样，该公司现在有一个详细的时间表。谷歌最初计划在 2020 年发布公告后的两年内逐步淘汰第三方 cookie。现在将在三个月内进行，从 2023 年年中开始，到 2023 年末结束。谷歌将隐私沙盒时间线分解为四个部分，下面列出了相关的 API：大多数目前在“讨论”中阶段，除了打击垃圾邮件/欺诈。接下来的步骤是“测试”和“准备采用”，当浏览器中的各个 API 的 Origin Trials (OT) 开始时，Google 会进行标记。谷歌希望在过渡期之前将属于这四个领域的 API 部署到位，并且在这些技术“准备好大规模采用”之前不会开始广泛部署。如果一切按计划进行，过渡的第一阶段将从 2022 年第四季度开始，一直持续到明年第二季度。在此期间，随着发布商和广告行业的迁移，Google 将监控采用情况。 Chrome 中第三方 cookie 的逐步淘汰将在 2023 年第三季度的第二阶段进行。隐私沙盒也旨在解决隐蔽跟踪技术，但这不在四个类别中。 SameSite Cookie、User-Agent Client Hints API 和 HTTP Cache Partitioning 都已大规模启动。 DNS-over-HTTPS 的规模化可用性计划（最早）在今年下半年开始，而 2022 年下半年将开始出现 WebID API、User-Agent Reduction、Gnatcatcher 和 Origin-Bound Cookies。最后，隐私预算的目标是 2023 年。隐私沙盒时间表将由 Google 每月更新一次，而本页底部还有一个常见问题解答。同时，Chrome 92 正在向现有的隐私沙盒首选项页面推出 FLoC 设置。
