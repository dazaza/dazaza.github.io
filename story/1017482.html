Parallels Desktop 16增加了Big Sur支持、3D Metal支持等

2020-08-11 15:09:39

今天在Mac上发布了Parallels Desktop 16。它是开发人员和其他人用来在MacOS下运行Windows、Linux和MacOS应用程序和虚拟机的软件的最新主要版本。它最引人注目的产品是对MacOS Big Sur的全面支持。

据Ars采访的Parallels代表称，支持Big Sur并非易事：Big Sur终止了对Parallels构建的第三方内核扩展的支持。他们声称，这意味着需要大量的工作才能与Big Sur合作-25年的工程工作。

除了同时支持主机和虚拟机的Big Sur外，Parallels Desktop 16的外观略有不同，以适应苹果在Big Sur中的不同外观。

虽然支持Big Sur是这里的旗舰功能，但在此版本中有一系列小改进。例如，当在MacOS Big Sur主机上运行MacOS Big Sur虚拟机时，Parallels Desktop 16支持3D in Metal应用程序。

打印机可以跨操作系统在主机和虚拟机之间共享，并且为具有缩放/旋转功能的Windows应用程序添加了对多点触摸板上的缩放和旋转手势的支持。Parallels Desktop 16还承诺比前一版本更快的性能；它声称启动速度是以前版本的两倍，DirectX性能提高了20%，Linux虚拟机的GIT状态也提高了75%。对更新版本的OpenGL的支持扩展了哪些Windows应用程序可以通过Parallels在虚拟机上运行。

Pro Edition用户现在可以命名他们的自定义网络，他们可以以压缩格式导出虚拟机，Parallels声称这种压缩格式是其压缩前大小的一小部分。此外，Parallels还推出了Microsoft Visual Studio插件，以简化在不同操作系统上的测试。

我们询问了在大苏尔的Apple Silicon上支持Windows的任何计划，但Parallels代表拒绝谈论这一问题，称他们将在晚些时候讨论这个问题。

Parallels Desktop 16将从今天开始上市。标准版是一次性购买，价格为99.99美元，而专业版和商业版需要每年订阅99.99美元。从Parallels Desktop 14或15升级到新版本标准版一次需要49.99美元。

该软件包包括Parallels Toolbox，最近还更新了新工具和功能。其中包括一个屏幕中断跟踪器，一个单位转换器，以及显示Mac的桌面和窗口管理器选项。
