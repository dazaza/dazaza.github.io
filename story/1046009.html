微软当天Xbox Live Gold逆转价格翻倍

2021-01-24 01:02:26

今天早些时候，微软宣布提高Xbox Live Gold的价格，六个月的订阅价格为60美元。这引起了广泛的抗议，包括我们自己的Xbox领导杰兹·科登（Jez Corden）的严厉评论，称其为“游戏中最糟糕的交易”。

微软刚刚宣布涨价已被取消。此外，免费游戏（例如Fortnite或《使命召唤：战地》）不再需要Xbox Live Gold。后面的更改正在进行中，但将在未来几个月内尽快到来。

Xbox Game Pass和Xbox Game Pass Ultimate保持不变，这意味着它们将提供与以前相同的好处。

这项更改是突然的，使Xbox Live Gold与PlayStation Plus更加接近，PlayStation Plus并不需要免费玩在线游戏。 Halo Infinite将于2021年秋季发布，届时，多人游戏将是免费的。现在，玩家不再需要购买Xbox Live Gold即可享受它，从而真正免费。

Xbox Game Pass可让您以一个月的费用访问超过200种游戏。 Xbox Game Pass Ultimate还向包装中添加了Xbox Live Gold，因此您可以与朋友在线玩。您还可以访问EA Play。

微软总裁布拉德·史密斯（Brad Smith）在最近与公司员工的会晤中为微软对有争议的政治人物的捐款辩护。史密斯（Smith）说，向政客捐款是“政治程序的运作方式”。
