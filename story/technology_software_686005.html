Download addresses for various language versions of the Win8 Consumer Preview

by geekzhang on 2012-03-01 19:35:53

February 29th, Beijing time - Microsoft today launched the Windows 8 Consumer Preview at the Mobile World Congress in Barcelona. The event will last for two hours, from 3 PM to 5 PM local time (10 PM to 12 AM Beijing time).

The Consumer Preview is also known as the public beta test version. Microsoft plans to gradually release the Developer Preview (released in September last year), the Beta Test Version (Consumer Preview), the Final Release Candidate, and the Windows 8 RTM (Release to Manufacturing) version.

So far, Microsoft has officially released download links for five versions including Chinese.
