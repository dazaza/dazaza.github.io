YouTube-dl 事件后，GitHub 的 DMCA 流程现在包括免费法律帮助

2021-08-02 18:45:26

GitHub 已宣布与斯坦福大学法学院建立合作伙伴关系，以支持面临与数字千年版权法案 (DMCA) 相关的删除请求的开发人员。虽然 DMCA 可能以保护电影和音乐等受版权保护作品的法律而广为人知，但它也有规定 (17 USC 1201) 将规避版权保护控制的企图定为刑事犯罪——这包括任何可能帮助任何人违反 DMCA 规定的软件.然而，与向在线内容创建者发送的无数虚假删除通知一样，开源编码人员也经常发现自己处于 DMCA 的火线中，别无选择，只能遵守请求，即使他们没有做错任何事。最终的问题是，自由编码员或小型开发团队通常没有资源来应对 DMCA 请求，这将权力平衡置于可能希望使用 DMCA 扼杀创新或竞争的财大气粗的公司手中.因此，GitHub 的新开发者权利奖学金——与斯坦福大学法学院的 Juelsgaard 知识产权和创新诊所联合——旨在通过向开发人员提供免费法律支持来帮助他们进入这样的境地。该举措是在 GitHub 宣布在美国唱片业协会 (RIAA) 提出删除请求后，对其第 1201 条索赔审查程序进行全面改革之后大约八个月发起的，该请求被广泛批评为滥用 DMCA。与此同时，GitHub 还宣布了 100 万美元的开发者防御基金，目前正在用于开发者权利奖学金。因此，向前推进，每当 GitHub 通知开发人员“有效的删除声明”时，它都会向他们提供请求免费独立法律顾问的选项。 GitHub 的开发者政策负责人 Mike Linksvayer 在一篇博客文章中表示，该奖学金还将负责“研究、教育和倡导 DMCA 和其他对软件创新很重要的法律问题”，以及其他相关项目。 “该研究员还将培训诊所的学生和其他律师如何与开发人员合作并代表开源社区进行倡导，”他说。 “总的来说，这些活动将有助于塑造一个对开发人员友好的法律环境，并平衡对开源开发人员很重要的法律问题的规模。”
