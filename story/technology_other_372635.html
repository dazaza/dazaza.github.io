Product Overview

FineReport reporting software is a self-developed, pure Java enterprise-level web reporting software by FineReport Software (China) Co., Ltd. This reporting software fully supports mainstream B/S architecture as well as traditional C/S architecture, with simple and flexible deployment methods. Product Introduction

FineReport reporting software provides an easy-to-use and efficient report design scheme, using mainstream data bi-directional expansion for truly code-free design of report templates; it has powerful report display functions and offers comprehensive report permission management, report scheduling management; it also has complete report filling capabilities, supporting multi-level aggregated reports….
