EC2 Image Builder现在支持跨AWS帐户分发AMI

2020-11-04 05:27:42

客户现在可以使用AWS命令行界面(CLI)与多个AWS区域的AWS账户共享Image Builder管道中的AMI。

客户可以将使用EC2 Image Builder构建的AMI与AWS帐户共享，方式是授予接收方帐户修改AMI的所有权限。这样就不需要跨多个帐户和AWS区域手动复制AMI。

您可以在此处开始使用AWS CLI。有关分布配置的详细信息，请参阅EC2 Image Builder API Reference中的CreateDistributionConfiguration。
