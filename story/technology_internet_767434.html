Microsoft Details New Features of WP8 GDR3: Larger Screen, Quad-core...

by anonymous on 2013-10-15 21:20:57

The third system update for Windows Phone 8, known as GDR3, will be rolled out to existing phones over the next few months. Microsoft has today detailed the contents of Windows Phone 8 GDR3: support for larger screens with higher resolutions; support for Qualcomm quad-core processors; a new driving mode feature; new accessibility features; improved internet sharing capabilities… along with hundreds of internal performance tweaks and improvements. Additionally, Microsoft will launch the "Windows Phone Developer Preview Program" tomorrow.

GDR3 supports 5-inch and 6-inch 1080p HD touchscreen devices, featuring Qualcomm Snapdragon 8974 quad-core processors. The all-new "Driving Mode" function allows you to stay more focused while driving. With the help of Bluetooth devices, Driving Mode can limit notifications on the lock screen, including text messages, calls, and quick status updates, and it can also automatically reply to callers or senders, informing them that you will get back to them later.

GDR3 introduces a range of accessibility features, such as a screen reader, which facilitates phone and contact management, sending SMS and emails, and listening to various notifications for visually impaired and blind users.

GDR3 enhances the internet sharing functionality, making it easier to use your phone as a data hotspot for Windows 8.1 devices. Simply pair your phone with a Windows 8.1 PC or tablet via Bluetooth to connect to the internet.

GDR3 also brings some highly requested features, including: more ringtones, customizing SMS notification tones for contacts; orientation lock options (locking landscape or portrait); new storage settings; and the ability to quickly close apps using the app switcher.

Microsoft will release a new Windows Phone 8 Update 3 emulator within the next three weeks. There are no new APIs in this update, so apps will run directly on Update 3 without modification. However, developers will need to include high-resolution image resources for 1080p displays.

Microsoft also announced a new Windows Phone Developer Preview Program, launching later today. This program allows installation of Windows Phone 8 Update 3 before official carrier testing is completed, provided one of the following conditions is met: 1) Your phone is developer unlocked; 2) You are a registered Windows Phone Store developer; 3) You have registered as a Windows Phone App Studio developer.

It's important to note that installing the preview version of WP8 firmware may void your phone warranty. Furthermore, the preview version does not include hardware manufacturer optimizations and customizations, and once installed, you cannot revert to an earlier system version.
