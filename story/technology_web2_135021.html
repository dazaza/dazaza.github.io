Several Indispensable Important Technical Points of SEO

1. Is website optimization the same as SEO?

SEO refers to search engine optimization, which involves using technical means to make a website's pages closer to search engines to obtain good rankings on search engines. Website optimization, however, is not just SEO optimization. It includes various factors: making the website closer to visitors and emphasizing user experience more prominently. On this level, SEO optimization techniques can improve the ranking of keywords contained in the website in search results. Therefore, we can understand it this way: SEO technology is just one of the means to make a website more optimized and reasonable, and this means is also an important prerequisite for the website's future product marketing.

2. Website-related planning

A very important premise in SEO: the website must have detailed planning, including the structure of the website, the distribution of directories, how related terms are allocated to these pages. The link structure design should be reasonable, with a good network layout, allowing access to the entire website from any point. After the website is built, operational planning includes changes in each marketing plan based on market research feedback, generating many new hot words for website promotion, and how to highlight these words.

3. Establishment of the site map sitemap

A site map is necessary, and it is recommended to create three different site maps. One is for the visitors to the website, which needs to be made into a directory navigation consistent with the website style. The other two are for search engine spiders to "view," including a sitemap.xml and a sitemap.htm page. Creating a sitemap suitable for Google is very important, and it can be submitted through the Google Webmaster Tools.

Why create a separate sitemap.htm normal page? Because many websites' homepages use a lot of images, flash, and relative addresses, which can cause spiders to fail to obtain a large number of links on these pages. We can create a separate sitemap page, putting all the website links inside, then placing a main keyword anchor text linking back to the homepage. The homepage should also have a link pointing to this site map.

Why create a separate news.xml? Create a page that conforms to the "Baidu News Open Protocol" specifically for the Baidu search engine. Many webmasters overlook the Baidu news page, but the traffic obtained from this page cannot be ignored, and it also includes many potential customers.

4. Reasonable meta tag settings

Commonly used meta tags in SEO include keywords, title, and description. Although some SEO experts value these tags highly, I personally recommend focusing on the title more. As for keywords and description, they do not show good effects in many search engines. However, the title is the first indicator displayed on each search engine's search result page, and as part of the long-term SEO strategy, the title will be retained for a long time, which determines its weight and status in the search engine algorithm.

5. Use of h tags

Text with h tags is like giving people eye-catching clothes. H tags range from h1 to h6, with values decreasing from large to small. The effect it shows is not just the size of the text; it also allows search engines to know that the text on your page annotated with h tags is more important than the text without h tags, and h1 tagged text is more important than h6 tagged text. It highlights which information on a page is more important. Remember not to overuse h tags; ensure one h1 per page, no more than two h2s, and allow a few h3s.

Why shouldn't you use too many h tags? Imagine a document with 1000 characters, and you circle all 1000 characters to indicate importance. When you look at the article again, can you distinguish which words in the article are the central focus?

6. Reasonable internal linking

A good internal linking environment can guide searchers and search engine spider programs through a link entry, allowing single-directional visits to traverse the entire website.

7. Obtain a few high-quality external links

If you want to do SEO well, you must have a few high-quality external links. You may think creating links is a tedious task, but don't forget: links are an important criterion for search engines to judge the value of a page, including Google's PR value.

8. Website content is excellent and updated consistently

Regarding content, many SEOers say: originality is key. I don't quite agree with that. I believe that as long as there is attractive textual content, it can still attract spiders to visit regularly. If you modify these reprinted texts and reasonably incorporate some keywords about your website into these articles, your website will receive more favor from search engines.

About relevant keyword links, I will not comment too much for now. My suggestion is: if you want to achieve the sitelink phenomenon mentioned in Google Webmaster Tools, consider first bringing certain values (such as PR values, where PR values demonstrate their true value in this aspect) to each subdirectory.
