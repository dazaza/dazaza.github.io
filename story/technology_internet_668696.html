Happy Sheep User Manual

I. Account Checking; Press K0 to check accounts.

Page 0: Total profit (Unit: Yuan)

Page 10: Total score of running lights

Page 1: Total coins inserted + total score added

Page 11: Total play score of betting games

Page 2: Total coins returned + total score deducted

Page 12: Total score given out in betting games

Page 3: Actual profit of this session

Page 13: Total play score of betting on Banker, Player, and Tie

Page 4: Total profit of last session

Page 14: Total play score of betting on Banker, Player, and Tie

Page 5: Blank

Page 15: Account verification bit

Page 6: Time (Machine stops after 5500 minutes)

Page 16: Difficulty level of running lights

Page 7: Total number of times the barcode was lost

Page 17: Difficulty level of betting games

Page 8: Machine ID number

Page 18: Difficulty level of Banker, Player, and Tie bets

Page 9: Total play score of running lights

II. Difficulty and Parameter Adjustment; Press K3 then start button to enter adjustment mode.

Page 0: Difficulty level of running lights, type of Mary machine. Use Apple key to adjust units digit, Orange key for tens digit, Mango key for hundreds digit.

Page 1: Difficulty level of betting games. Use Apple key to adjust units digit, Orange key for tens digit, Mango key for hundreds digit.

Page 2: Difficulty level of Banker, Player, and Tie bets. Use Apple key to adjust units digit, Orange key for tens digit, Mango key for hundreds digit.

Page 3: Coin insertion ratio. Use coin return key to select: 1:1, 1:2, 1:5 (for claw machines), 1:10, 1:20, 1:50, 1:100.

Page 4: Betting ratio. Use coin return key to select: 1:1, 1:2, 1:5 (for pinball machine price), 1:10.

Page 5: Maximum bet per door for running lights. Use coin return key to select: 50, 100, 150, 250, 300.

Page 6: Maximum bet for Banker and Player. Use coin return key to select (pinball machine repair method): Adjust between 1000-9000 points.

Page 7: Maximum bet for Tie. Use coin return key to select: Adjust between 1000-5000 points.

This machine has the latest official upgrade program that can adjust difficulty levels. The effective difficulty range is 920-970. If lower than 920, it will be set to 920; if higher than 970, it will be set to 970. To avoid the issue of previous machines resetting to zero when adjusting difficulty and releasing water, this machine does not need to reset to zero. Simply press K3 directly to adjust the difficulty.

III. Reset Method:

For company-owned machines: Hold down K0 on the computer board to check accounts, then input pages 0, 3, 7, 8, and 15 on the encoder.

For privately-owned machines: Hold down K0 + K3 to turn off and on the machine, then input pages 0 and 3 (twice). The first output will be 99 for fruit machines, and the second output will be 66.

IV. Testing:

Hold down K0 + K1 keys while turning on the machine to test the digital tubes and LED lights.
