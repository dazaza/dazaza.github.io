Microsoft Windows 8.1 is available for download, and users of Win8 can update for free.

by anonymous on 2013-11-16 10:50:57

Microsoft Windows 8.1 is now available for download, and Win8 users can update for free.

On October 17th, at exactly 7 PM, Microsoft has already made Windows 8.1 available for download on their servers. All users with Windows 8 installed can download the 8.1 version for free through the Windows Store.

In addition, non-Windows 8 users, including those using Windows XP, Windows Vista, and Windows 7, can pay either $119.99 or $199.99 at Windows.com to obtain the standard edition and Pro edition upgrade downloads. Users who are using the evaluation version of Windows 8.1 will still be in evaluation mode after upgrading their system.

Starting this Friday, retailers worldwide will begin offering DVD versions of Windows 8.1.

Please note:

According to a previous announcement from Microsoft China, the price for the standard edition of Windows 8.1 in the Chinese market is RMB 988, which is RMB 254 more expensive than in the U.S. market; the price for the Professional edition of Windows 8.1 in the Chinese market is RMB 1988, which is RMB 764 more expensive than in the U.S. market.

In addition, currently, Chinese consumers have only two ways to obtain Windows 8.1: purchasing a new device pre-installed with Windows 8.1 from a computer manufacturer, or downloading Windows 8.1 from the Microsoft Windows App Store.
