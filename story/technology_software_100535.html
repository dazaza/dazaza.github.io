Folder Universal Unlocker v0.05 Green Free Version

by ankty on 2008-11-03 10:42:46

Many people like to use folder encryption software to hide their secrets, but due to various reasons (such as the expiration of shared software), they forget the password and need to retrieve their precious data. This software can help you solve this urgent problem.

This program supports netlock, Folder Lock Master, Folder Encryption & Hider, Folder Watchdog, Mobile Encryption, Top Secret Disk, High-Strength Folder Encryption Master, Invisible Folder, and other similar software. In theory, as long as the encryption method uses ..\ and disguised folders, this program can decrypt it. Please do not use it for illegal purposes.

【Usage Method】: First, in the Folder Options, display all files to make the encrypted folders visible. Drag the folder you want to unlock onto the program window and release it. A folder named "Unlocked" will be generated. If some encryption software has multiple layers of protection, you may need to decrypt several times.

【v0.05 Update Notes】: (abc represents normal characters, Aux represents Windows reserved filenames)

1. Supports decryption of hidden attribute folders with a dot.

2. Supported types: abc., abc de., abc.{21ec2020-3aea-1069-a2dd-08002b30309d}, Aux.{21ec2020-3aea-1069-a2dd-08002b30309d}, Aux, ..., 1., Aux.avi.{00022602-0000-0000-C000-000000000046}, abc (1), Aux., .., $Extend (the usage method for this type is temporarily undisclosed).

3. When the Decrypt setting in the configuration file Unlock.ini is set to 0, the program function switches from "Universal Folder Unlocker" to "Drag-and-Drop Folder Deletion."

Compared with similar file shredding/deletion software, this program has the following features:

1. Easy to use, file deletion is done with a simple drag.

2. Programs that are currently running can still be deleted without issue.

3. Files/folders can be deleted even if the administrator does not have the necessary permissions.

4. Deleted files cannot be recovered by anti-deletion software.

5. Can unlock specially constructed folders.

6. Can delete specially constructed folders and supports batch deletion functions.

PS: The deletion effect is controlled by the Unlock.ini file. Files/folders that cannot be deleted immediately will be added to the deletion list and automatically deleted during the next reboot process. Remember to refresh the settings in the program after changing parameters. Do not test the AllFile=1 and IncludeFolder=1 parameters with root directory files unless you want to completely erase everything—remember this!
