Verizon被迫撤下广告，声称消防员需要Verizon 5G

2020-10-16 02:19:40

Verizon在2018年因在一场野火中限制消防部门的无限制数据计划而引发争议，但这并没有阻止该公司推出大量广告，声称Verizon服务是消防员和其他紧急响应人员的必备服务。但其中几个广告显然做得太过分了，在T-Mobile向广告业自律机构提出投诉后，Verizon同意停止播放这些广告。

美国国家广告部(NAD)今天在一份投诉结果公告中表示，Verizon承诺永久停止其面向消防员的5G Build Right和面向First Responders的5G Build Right广告以及其中提出的受到质疑的主张。(注：威瑞森5G Build Right for First Responders广告和其中提出质疑的主张)今天在一份投诉结果公告中表示，Verizon承诺永久停止其针对消防员的5G Build Right广告和针对First Responders广告的5G Build Right广告。

NAD表示，它实际上没有审查消防员和急救人员的投诉，因为Verizon同意在调查之前撤回他们。但NAD调查了T-Mobile的其他指控，并建议Verizon停止或修改其他几个做出不支持声明的广告。Verizon同意遵守NAD的调查结果。

Verizon仍在大肆宣扬其5G网络能够提升消防员拯救生命的能力。但是，为消防员打造的5G广告和为First Responders打造的5G广告不再出现在Verizon的YouTube频道上。

一家商业跟踪网站提供了这则为消防员打造的5G广告的文字记录，这则广告现已停产，显然是在1月份Verizon推出以5G为重点的超级碗(Super Bowl)广告活动时首次亮相的。

当我着火的时候，通常当你打开门的时候，浓烟都落到了地上，一片漆黑。在建筑火灾中，时间真的是至关重要的，因为它可能意味着生死。有了5G，我们将能够将(消防员)穿过面具的视频流回指挥中心，并将这些来自多个来源的视频拼接在一起，以提供更好的态势感知。5G超宽带是所有这些应用程序工作所必需的。

但是，5G超宽带以任何实质性方式帮助消防员的可能性很小。5G超宽带是Verizon所说的毫米波服务，它提供极高的速度，但只在大城市的小范围内提供。这是因为毫米波频率传播不远，很容易被墙壁和其他障碍物阻挡。
