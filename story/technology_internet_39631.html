Beam It Up Scotty Free file upload to mobile

by misterwong on 2007-12-17 19:36:53

BeamItUpScotty is a new intelligent mobile application solution recently released by the Mister Wong development team. It allows users to directly upload files to their handheld devices via a web browser. The usage method is quite simple: select the multimedia files (pictures, music, videos, Word/Excel...) you want to upload from your local device to BeamItUpScotty.com, choose whether to compress the files, and input the recipient's phone number (Chinese users need to enter 0086 + phone number). All the remaining work will be fully handled by BeamItUpScotty. A few seconds later, the phone with that number will receive an SMS with a download link, allowing you to download these files at any time via mobile internet—simple and fast! Some crazy users have even thought of using BeamItUpScotty's service to send free text messages to others.
