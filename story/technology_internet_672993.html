Methods to free up memory space on Android phones

by dream2012 on 2012-02-23 12:41:11

Just like PC users, smartphone users also encounter the problem of insufficient phone storage. For Android phones, due to the system not supporting the installation of programs to the SD card and the limited storage capacity configured on the phone, many users have encountered the issue of insufficient system memory caused by installing too many applications. Although Android 2.2 has supported the Apps2SD function since its release, many users found that under Android 2.2, a lot of mobile applications still could not be installed on the SD card. Even if the program supports being moved to the SD card, some system files and privacy files remain stored in the phone's internal memory. Additionally, some programs like widgets, live wallpapers, and certain system applications will not function properly if installed on the SD card. Insufficient memory continues to be a thorny issue for many users.

However, as an excellent mobile operating system, Android provides several ways to clean up and free up memory space. We have summarized the following points to hopefully help users solve their urgent problems:

1. Clear cache data of large applications

By clearing the cache files of the application, the released memory space may pleasantly surprise you. Go into your phone settings options, select Settings > Applications > Manage Applications. Press the Menu key and choose the "Sort by size" option (if using an Android 2.2 phone, first select the downloaded label), then all installed applications on the phone can be sorted by size.

Click an application from the list, if the program has cache files with visible sizes, simply click the "Clear Cache" option to release the space occupied by these cache files. Some applications' cache files might be several MBs in size, such as Google Maps, Market, browser, and photo gallery programs. Clearing the cache files of these applications can free up quite a bit of space.

Many phone manufacturers preinstall their own UI programs on their phones, such as HTC Sense and MOTOBLUR. If you plan to replace HTC Sense with launcher programs like LaunherPro or ADW, you can clear or even delete the data files of HTC Sense, which can free up dozens of MBs of space on your phone.

Additionally, the Android Market offers some automatic cache-clearing applications, such as the paid software Quick App Clean Cache. For users who have obtained root access, they can download CacheCleaner, CacheMate, and MoveCache from the Market to quickly and conveniently clear program cache files.

2. Delete those applications you never use or rarely use

This is a problem many users face. They silently endure the warning of insufficient phone memory, even sacrificing system performance, just because they are reluctant to delete those applications they never use or rarely use. This might be a form of obsessive-compulsive disorder. Deleting these applications will make you realize how much more space you have gained, and you won't lose anything.

3. Move all movable application data to the SD card

Phones running Android 2.2 support installing applications to the SD card. Make sure to check all the applications you have installed; if they support the apps to SD card feature, move them to the SD card. For some large software, especially game programs, the saved capacity is quite significant. However, be sure to pay attention to widget programs, bundled widget programs, live wallpaper programs, and applications that need to run in the background and interact with the system. Do not move these to SD card storage, otherwise, the programs may not function properly.

If you find it troublesome to set each program individually, you can use programs like Apps 2 SD and SDMove to simplify the process. For users with stronger hands-on skills, you can follow online tutorials to set the default installation location of applications to the SD card, solving this issue once and for all.
