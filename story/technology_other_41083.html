Stuck at 99% while downloading? Here's how to fix it

by luliangpin on 2008-01-01 08:46:51

**Title: Download loses response at 99% - Here's how to fix it**

**Symptoms**

When using Internet Explorer (IE) to download a file, the progress reaches 99% and suddenly stops.

**Cause**

The file has actually been downloaded completely and can be found in the temporary directory of the Internet. The issue occurs when moving the downloaded file from the temporary directory of the Internet to the specified download directory.

**Solution and Checkpoints**

You can try the following steps to check and repair:

1. **Uninstall IE Plugins**: If your system has many IE plugins installed, such as 3721 Online Assistant, Baidu Search Companion, Thunder Download Accelerator, Dudu Download Accelerator, etc., please uninstall them. It is recommended to log in to zs.kingsoft.com and install Kingsoft Antivirus Security Assistant online. This tool has built-in online virus scanning and strong IE repair functionality.

2. **Check Disk Space**: If there isn't enough space in the designated storage location, try specifying another download folder for the files.

3. **Clear Temporary Files and Cookies**: Try clearing the temporary files folder and cookies.

4. **Edit Registry**: Run "regedit", open the registry editor, and find [HKEY_LOCAL_MACHINE] \ SOFTWARE \ Microsoft \ Active Setup \ Installed Components \ {89820200-ECBD-11cf-8B85-00AA005B4383}. Change the key value from 1 to 0. After restarting, Windows XP will automatically begin repairing and reinstalling IE.

5. **Network Disconnection Issues**: If there are occasional network disconnections, you can refresh the webpage and click on the download link again. It will automatically continue the download. In the future, it’s better to use FLASHGET or similar software, as they support resuming interrupted downloads.

6. **Insufficient Disk Space**: It might be due to insufficient disk space. When downloading software, it is first downloaded to the temporary folder and then moved to the specified directory. If the disk space in the specified directory is not sufficient or if there are insufficient write permissions, this could cause the download to stop at 99%.

7. **Clear IE Cache**: Try clearing the IE cache by emptying the contents of the Temp folder and then retrying the download. You can also try disabling the antivirus software's file monitoring feature temporarily.
