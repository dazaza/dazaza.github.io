How to do a good job of designing the hierarchy of your own website?

by betty2012 on 2012-02-04 17:34:10

### I. Construction of Text Links (Precautions)

1. > Try to use text links (mainly for search engine recognition rather than users), and avoid using image links unless necessary (such as logos, advertisements). Image links are secondary to text links. Text links are equivalent to votes, making them easier to calculate.

2. > Every webpage on the website must have at least one text link that can be reached (e.g., product sites, wedding photography, tourist attractions, beauty pictures, avatar images). Important pages should have text links in prominent positions.

1. Ensure there is a search function under relevant conditions.

2. Not only should there be a search function, but it should also be appropriately integrated with the website content.

Considerations for target keyword users:

- First, meet the click demand of users entering the homepage, including users searching for target keywords and those directly entering the URL.

- Second, the text of the set text link should be searchable.

### II. Principles and Ideas for Directory Design

- How to classify directories? When classifying directories, we need to comprehensively consider all elements and strive to maintain a balance, which is the key to determining directory classification.

1. Consider users, make classifications as simple as possible so that even beginner (novice) users can understand the meaning of your classification names and quickly find what they need.

2. Consider the participation of directory keywords in rankings.

3. Consider the relevance of the entire site.

For small and medium-sized websites, the number of directories should not exceed 8.

### III. Design of Breadcrumbs Paths

1. The setting of breadcrumb paths must not include hidden directories.

2. Breadcrumb paths should not be overly细分 (if a directory page can already be fully filled with products, do not further subdivide types, as this will overwhelm users).

3. It’s best not to use uniform calls. Example: http://www.xasq.cn/

### IV. Control of the Number of Page Links

1. Be more user-friendly.

### V. Tree-like Rules for Internal Links

It is not recommended to interlink internal pages between different directories, but internal pages can create links to different directory pages.
