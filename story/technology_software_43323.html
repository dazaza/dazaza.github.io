A fast browser that uses the Gecko engine. Its interface mimics the style of IE and can use IE bookmarks. Its main feature is that the speed of downloading pages is quite fast, which is still advantageous for Chinese users whose internet speed is not very high.

K-MeleonCCF is a browser that uses the Gecko engine and has built-in support for RSS news browsing and web page translation functions (Google engine) and other practical features. You will surely fall in love with this software after using it.

Features:

* Gecko engine (i.e., Mozilla/Firefox engine)

* Mouse gestures

* Mouse drag and drop (right click)

* Pop-up ad blocking

* Web ad blocking

* Web password setup/editing

* Multi-proxy support

* Shortcut settings

* Can use multiple bookmarks from Mozilla/Netscape/IE/Opera

* Built-in RSS news browsing support

* Built-in weather forecast function

* Built-in Bookmarklets function

* Web page translation function (Google engine)

* Shortcut group/startup group function
