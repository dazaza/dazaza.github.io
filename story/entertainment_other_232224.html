Tutorial for Adding QQ Zone FLASH Animation Module (FD Module)

by lwf225 on 2009-09-02 15:58:18

Tags: QQ Zone Tutorial, How to Add a FLASH Module, Adding an FD Module, QQ Zone FLASH Animation Module (FD Module) Addition Tutorial

Many friends don't understand how to add a FLASH animation module, so I will explain it in detail here:

1. The tutorial for using QQ Zone's flash module starts by opening your own space. For example, I will use my space! After opening your space, click Customize → New Module (as shown in the figure below). You will see a dialog box for creating a new module. If you want to create a picture module, click the picture module option; if you want to create a flash module, click the flash module option. This means that whichever type of module you want to create, just select that type. We will use the flash module as an example. The process for other modules is the same as for the flash module. Since we are using the flash module as an example, we will select the flash module! Click on the Flash Module Animation!

2. After clicking on the flash module, the following dialog box appears. Add a title, and give it any name you like. For the flash animation address, we will use http://flash.qqpub.cn/gchs4.swf as an example. Enter this address, then input the verification code, and click Confirm!

3. After clicking Confirm, the following dialog box appears. Do you see the symbol at the bottom right corner? Hold the mouse over it and drag it to adjust the size to your desired effect (you can make it longer or taller)! Once you have adjusted it to your desired effect, click Save (see the figure below). This is the most important step. Some friends may overlook this step, resulting in the space not being saved, and thus no effect. This is also why many friends ask me why there is no effect!!

4. Alright, the tutorial is done. Let's take a look at the effect! Not bad, right?! Hmm!! Some people might say it's nice, but others might say, "It's nice, but the border is too ugly. If only we could remove it!" That's a good question. Next, I will teach you how to remove the module's border!

5. How do we remove the module's border?? Follow along with me! Look at the figure below. Do you see the "More Settings" option? Yes, click on it with your mouse!

6. After clicking "More Settings," the following figure appears. Leave the title blank. Below, there are three options. For the playback options, check both boxes. You don't need to worry about the flash size. For the module background border, if you want a border, select "Yes"; if you don't want one, select "No"! Of course, we will select "No." After selecting "No," fill in the verification code, click Confirm, and then click Save! Alright! We're all done!
