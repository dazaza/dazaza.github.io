Cleverly setting a QQ password can frustrate arrogant malware.

by mylovehisa on 2008-02-18 10:17:31

For safety reasons, the input and storage of passwords in computers are displayed as "*" symbols. This way, others are unable to view the content of the password. Many users assume that this provides complete security, but the reality is harsh. This method of password protection only guards against honest individuals and not malicious ones. Some people with ill intentions often use remote screen monitoring methods and combine them with password viewing software to easily reconstruct the password. Reality forces us to adopt more rigorous measures.

We can use fonts that cannot be displayed by default in the system as the source characters for our passwords. For example, in Windows 2000/XP, the default text display is "SimSun" with a character set of "Simplified Chinese GB2312." If you choose characters that are not included in GB2312 (such as SimSun - Founder Extended Character Set installed with Office XP), the system will be unable to display these password characters correctly. Instead, it will show them as "□," "space," or "?" etc., as garbled text. Even if password viewing software is used, it will display the same garbled text, thus achieving the goal of preventing hackers from stealing passwords. The principle is not complicated, but practical implementation involves knowledge from various areas.
