Uploadjockey - Upload files to 6 network drives at the same time

by yoshio on 2008-05-18 10:50:06

Uploadjockey, like massmirror introduced before, can simultaneously distribute the files you upload to several mainstream file hosting services for storage. Massmirror supports 8 online storage platforms, while Uploadjockey supports 6. However, in terms of speed, Uploadjockey is evidently much faster. But massmirror supports uploading network files (that is, direct URL reference uploading), whereas Uploadjockey only supports local file uploads. In file management, both services support deleting unnecessary files. Uploadjockey can be used either by registering or without registration, but without registration, better file management functions cannot be accessed. Additionally, Uploadjockey has a monetization model: every time your file is downloaded, you earn $0.001, and when the total amount reaches $10, you will receive cash payment. Of course, earning $10 might not be easy.
