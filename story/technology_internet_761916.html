Patch time again: Microsoft releases 10 security updates for May

by geekzhang on 2013-05-15 11:15:33

Microsoft today pushed out the May security patches to Windows users, a total of 10, among which MS13-037 and MS13-038 are the critical patches with the highest security level. Both patches target Internet Explorer (IE). The first one is an accumulated security update for IE, while the other is a specific IE security update, primarily addressing the 0-day vulnerability in IE8 that was publicly disclosed.Details of Microsoft's May security patches:1、Cumulative Security Update for Internet Explorer (2829530)Security Bulletin MS13-038Level: CriticalSummary: This security update resolves 11 privately reported vulnerabilities in Internet Explorer. The most severe of these vulnerabilities could allow remote code execution if a user views a specially crafted webpage using Internet Explorer. An attacker who successfully exploited these vulnerabilities could gain the same user rights as the current user. Users whose accounts are configured to have fewer user rights on the system would be less impacted than those who operate with administrative user rights.Affected Software: For Internet Explorer 6, Internet Explorer 7, Internet Explorer 8, Internet Explorer 9, and Internet Explorer 10 on Windows clients, this security update is rated Critical; for Internet Explorer 6, Internet Explorer 7, Internet Explorer 8, Internet Explorer 9, and Internet Explorer 10 on Windows servers, this security update is rated Moderate.2、Internet Explorer Security Update (2847204)Security Bulletin MS13-038Level: CriticalSummary: This security update resolves one publicly disclosed vulnerability in Internet Explorer. If a user views a specially crafted webpage using Internet Explorer, the vulnerability could allow remote code execution. An attacker who successfully exploited this vulnerability could gain the same user rights as the current user. Users whose accounts are configured to have fewer user rights on the system would be less impacted than those who operate with administrative user rights.Affected Software: For Internet Explorer 8 on Windows clients, this security update is rated Critical; for Internet Explorer 8 on Windows servers, this security update is rated Moderate.3、Vulnerability in HTTP.sys Could Allow Denial of Service (2829254)Security Bulletin MS13-039Level: ImportantSummary: This security update resolves one privately reported vulnerability in Microsoft Windows. If an attacker sends specially crafted HTTP packets to an affected Windows server or client, the vulnerability could allow denial of service.Affected Systems: For all supported versions of Windows 8 and Windows Server 2012, this security update is rated Important.4、Vulnerabilities in .NET Framework Could Allow Spoofing (2836440)Security Bulletin MS13-040Level: ImportantSummary: This security update resolves one privately reported vulnerability and one publicly disclosed vulnerability in .NET Framework. The most severe of these vulnerabilities could allow spoofing if a .NET application receives a specially crafted XML file. An attacker who successfully exploited these vulnerabilities could modify the contents of the XML file without invalidating the file's signature and could gain access to endpoint functionality if they become authenticated users.Affected Software: For Microsoft .NET Framework 2.0 Service Pack 2, Microsoft .NET Framework 3.5, Microsoft .NET Framework 3.5.1, Microsoft .NET Framework 4, and Microsoft .NET Framework 4.5 on all affected versions of Microsoft Windows, this security update is rated Important.5、Vulnerability in DjVuLibre Could Allow Remote Code Execution (2834695)Security Bulletin MS13-041Level: ImportantSummary: This security update resolves one privately reported vulnerability in Microsoft Lync. If an attacker shares specially crafted content (such as a file or program) during a presentation in Lync or Communicator, then entices a user to accept an invitation to view or share the presentable content, the vulnerability could allow remote code execution. In all cases, an attacker cannot force users to view or share files or programs controlled by the attacker. Instead, an attacker must convince users to perform actions, typically by enticing them to accept an invitation in Lync or Communicator to view or share presentable content.Affected Software: For supported versions of Microsoft Communicator 2007 R2, Microsoft Lync 2010, Microsoft Lync 2010 Attendee, and Microsoft Lync Server 2013, this security update is rated Important.6、Vulnerabilities in Microsoft Publisher Could Allow Remote Code Execution (2830397)Security Bulletin MS13-042Level: ImportantSummary: This security update resolves eleven privately reported vulnerabilities in Microsoft Office. If a user opens a specially crafted Publisher file using an affected version of Microsoft Publisher, these vulnerabilities could allow remote code execution. An attacker who successfully exploited these vulnerabilities could gain the same user rights as the current user. Users whose accounts are configured to have fewer user rights on the system would be less impacted than those who operate with administrative user rights.Affected Software: For supported versions of Microsoft Publisher 2003, Microsoft Publisher 2007, and Microsoft Publisher 2010, this security update is rated Important.7、Vulnerability in Microsoft Word Could Allow Remote Code Execution (2830399)Security Bulletin MS13-043Level: ImportantSummary: This security update resolves one privately reported vulnerability in Microsoft Office. If a user opens a specially crafted file or previews a specially crafted email in an affected version of Microsoft Office software, the vulnerability could allow remote code execution. An attacker who successfully exploited this vulnerability could gain the same user rights as the current user. Users whose accounts are configured to have fewer user rights on the system would be less impacted than those who operate with administrative user rights.Affected Software: For all supported versions of Microsoft Word 2003 and Microsoft Word Viewer, this security update is rated Important.8、Vulnerability in Microsoft Visio Could Allow Information Disclosure (2834692)Security Bulletin MS13-044Level: ImportantSummary: This security update resolves one privately reported vulnerability in Microsoft Office. If a user opens a specially crafted Visio file, the vulnerability could allow information disclosure. Note that although an attacker cannot use this vulnerability to execute code or directly elevate their user privileges, this vulnerability could be used to produce information that might be useful in attempting to further compromise the security of the affected system.Affected Software: For all supported versions of Microsoft Visio 2003, Microsoft Visio 2007, and Microsoft Visio 2010, this security update is rated Important.9、Vulnerability in Windows Package Could Lead to Information Disclosure (2813707)Security Bulletin MS13-045Level: ImportantSummary: This security update resolves one privately reported vulnerability in the Windows package. If a user opens Windows Writer with a specially crafted URL, the vulnerability could allow information disclosure. A successful attacker could substitute Windows Writer proxy settings and overwrite files accessible by users on the target system. In a web-based attack scenario, websites may contain specially crafted links intended to exploit this vulnerability. An attacker must persuade users to visit the website and open specially crafted links.Affected Software: For Windows Writer installed on all supported versions of Microsoft Windows, this security update is rated Important.10、Vulnerabilities in Kernel-Mode Drivers Could Allow Elevation of Privilege (2840221)Security Bulletin MS13-046Level: ImportantSummary: This security update resolves three privately reported vulnerabilities in Microsoft Windows. These vulnerabilities could allow elevation of privilege if an attacker logs on to a system and runs a specially crafted application. An attacker must possess valid logon credentials and be able to log on locally to exploit these vulnerabilities.Affected Software: For all supported versions of Windows XP, Windows Vista, Windows Server 2008, Windows 7, Windows Server 2008 R2, Windows 8, Windows Server 2012, and Windows RT, this security update is rated Important.
