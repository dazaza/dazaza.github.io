微软将提供其自己的CLI包管理器，可选择第三方回购

2020-05-20 03:17:47

微软在Windows10中增加了一个内置的软件包管理器。该公司今天宣布Windows Package Manager将在Build 2020中发布，这是一个新的本机软件包管理器，将被内置到Windows10中。

对于那些不熟悉的人来说，软件包管理器允许用户在他们的机器上快速安装软件，而不必手动转到每个应用程序的网站或通过Microsoft Store。

Windows软件包管理器是一个新工具，用户可以使用它来实现这一点。使用新的“Winget”命令，Windows软件包管理器会自动获取您要下载的应用程序的最新版本，验证其真实性，并将其安装在您的计算机上。例如，要安装最新版本的Windows终端(顺便说一下，它刚刚升级到1.0)，您只需运行“Winget install Terminal”即可安装最新版本的Windows终端，而无需前往微软商店。您也可以使用“Winget Search”来搜索应用程序。以下是其他可用命令：

微软将提供自己的第一方存储库，其中包括流行的Windows10应用程序，你也可以提交一个拉取请求来添加新的应用程序(带有清单文件)。该公司还将允许用户从第三方存储库下载应用程序，因此你也可以从第三方来源下载应用程序。

微软表示，该公司制作自己的软件包管理器而不是为现有的软件包管理器做出贡献的主要原因是安全。“我们关心的一个关键问题是如何建立一个受信任的应用程序存储库。我们会自动检查每一份舱单。我们利用SmartScreen、静态分析、SHA256散列验证和一些其他过程来降低恶意软件进入存储库和进入您的机器的可能性，“微软高级程序经理Demitrius Nelon说。

“如果您对当前的包管理器感到满意，请继续使用它。我们的目标是让在Windows上安装软件对每个人都更好，“Nelon补充说。

Windows Package Manager支持Windows10版本1709及更新版本，一旦版本达到1.0，它将预装Windows10的桌面应用安装程序。目前，用户可以使用Windows软件包管理器，方法是通过Windows Insider程序从此处的GitHub存储库下载Windows软件包管理器，或者通过将其Microsoft帐户电子邮件地址发送到[受保护的电子邮件]来加入Windows软件包管理器Insider计划。
