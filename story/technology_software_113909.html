Sogou browser 1.0 public beta version officially released

by yinyin on 2008-12-22 11:56:22

After 6 months of intense research and development, the Sogou team launched another enamored work after Pinyin input method - Sogou browser. Sogou browser will inherit the tradition of input method, trying to gradually create a higher performance, faster response, better experience, more innovative browser products.

Sogou version 1.0 introduced the concept of "multi-task asynchronous browsing" for the first time, enabling each page to be read asynchronously and independently, effectively avoiding the impact of the performance degradation of a single page on the entire browser. So that Sogou browser running speed, running fluency are beyond similar browsers.

Sogou browser will not be unable to operate due to loading large pages; Will not be unable to close due to malicious pop-up dialog box; It won't be slowed down by having multiple pages open at once.

Sogou browser provides a breakthrough video browsing mode for friends who like to watch micro video - Web video player. Allows you to easily browse other pages while watching web videos.

Sogou browser is also the first browser to achieve automatic proxy acceleration function. Sogou browser users in Education network can access public network websites and foreign websites faster and free of charge, overcoming the bandwidth bottleneck between education network and public network.

Sogou browser integrates the technical resource advantages of Sogou search engine, and the first one realizes the function of directly returning multiple search results to the address bar drop-down list from the search engine, optimizing the experience of the address bar.

Sogou browser also provides the most frequently visited bar, privacy protection browsing mode and other features, as well as TAB browsing, search bar, mouse gestures and other common functions, Sogou browser can meet the browsing needs of most browser users.
