Long boot time for Windows system

by jimwang on 2010-09-21 17:32:10

I believe many friends have encountered such an annoyance when starting their computers: the progress bar on the startup screen keeps rolling over and over again, sometimes for more than ten rounds, which can be very frustrating! In fact, this is caused by a long system boot time. By following these five methods outlined below, you can ensure your computer starts up much faster.

1. **Optimize Startup Programs**

Click “Start” → “Run”, input “msconfig” and select the “Startup” tab. Alternatively, open “360 Security Guard” and choose “Manage Applications”, then select “Startup Software”. Here, only keep CTFMON and your antivirus protection software enabled; all others can be disabled.

2. **Optimize BIOS Settings**

Generally, during system startup, press the “Delete” key to enter the BIOS. In the BIOS, set the first boot device as the hard drive and disable floppy disk scanning.

3. **Reduce Startup Progress Bar Time**

Every time Windows starts, the blue progress bar rolls multiple times. However, you can reduce its rolling time to speed up the startup process. Open the Registry Editor and navigate to `HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters`. On the right, find the `EnablePrefetcher` key and change its default value of 3 to 1. This will reduce the time the progress bar rolls.

4. **Speed Up Shutdown Time (which indirectly speeds up next startup)**

When shutting down Windows, the system sends messages to running programs and remote servers, informing them that the system is shutting down, and waits for a response before proceeding with the shutdown. To speed up the shutdown process, first set tasks to end automatically. Find `HKEY_CURRENT_USER\Control Panel\Desktop`, and set the `AutoEndTasks` value to 1. Then, under the same branch, locate “HungAppTimeout” and change its value to “4000” (or less); the default is 50000. Finally, find `HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control` and set `WaitToKillServiceTimeout` to “4000”. With these settings, the shutdown speed will noticeably improve.

5. **Disable Unused Interfaces**

In “My Computer” → “Properties” → “Device Manager”, disable any unused ports and IDE interfaces. Change the “Device Type” for both the primary and secondary hard drives to “None”.
