User Experience: Fedora Core 6 Test 2

by gosman on 2006-08-19 12:34:52

Fedora Core 6 Test 2 has been released for a while now. Although there hasn't been much significant action from the official side, I still want to talk about it here, especially its new features.

I have been using the Fedora Core development version, Rawhide, so I upgrade it daily, and I always do this through network installation. However, during Test 2, the official team provided an installation CD, which you might want to try.

The migration from FC5 to FC6 feels much better than from FC4 to FC5. For example:

1. Installation and boot options have been enhanced, supporting iSCSI, motherboard-based RAID; the system can be installed on extended USB, FireWire-connected disks, and software can be installed from additional package repositories during the first installation. Network installation now supports IPv6.

2. Better support for laptops, such as sleep and standby functions that work on most laptops. The network management software has been further improved, though it is turned off by default.

3. Some software has been updated, like SELinux, the X system, and Xen virtualization software.

4. The desktop environment still isn't perfect and needs improvement.

Overall, FC6 looks more like a more stable and refined enhanced version of FC5 rather than a completely new version built from scratch. I look forward to seeing the final version this fall.
