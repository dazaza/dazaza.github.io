Maxthon 2.0.6 Final Released

by lengpengcheng on 2007-12-04 18:16:41

2.0.6 Official Version

This version focuses on fixing various detail issues since version 2.0.5 and improving the stability of online services.

In addition, this version has added some practical functions such as spell checking.

The highlights of this update (2.0.6 Official Version) are as follows:

Solving multiple problems that could cause the browser to freeze, thereby improving execution efficiency.

Spell Check - Spell check is very helpful for those who frequently use English. It may be even more useful for international users.

Usage method: After typing text in a text box, right-click, and in the pop-up menu select "spell check" to see if there are any spelling errors.

More stable online service - The stability of various network-connected services such as online favorites, automatic updates, and upload/download of data from the collection panel have been significantly improved.

Besides the above-mentioned functions, there are quite a few other detailed improvements, such as the stability of security monitoring and tab messages that plugin authors care about. For the full list of updates, please refer to the appendix.
