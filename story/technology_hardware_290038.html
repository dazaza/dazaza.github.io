BlackBerry Phone: BlackBerry Beginner's Tutorial (Part 2) - Basic Settings for First-Time Use

by cheng543 on 2009-12-03 10:59:09

BlackBerry; http://shop.shichenghy.com The powerful function options of BlackBerry phones, how to handle the basic settings of BlackBerry phones? Here are a few common methods for setting up BlackBerry phones summarized. 1. BlackBerry phone won't turn on: Since batteries need to be discharged during transportation, you just need to insert the battery into the phone, plug in the charger, and charge it for about 30 minutes until the phone turns on automatically. If the battery cannot be charged, please contact the seller directly. 2. How to change the system of a BlackBerry phone to Chinese (assuming it has already been flashed with Chinese): Open the phone, select options (setting or options, if you can't understand, look for an icon that looks like a wrench and press it) -> language or location -> change to Simplified Chinese. 3. When making calls on a BlackBerry phone, the handset feedbacks that you haven't activated services for Hong Kong, Macao, and Taiwan: Dial 10086 (but don't actually call out, meaning don't press the dial key). For devices with a scroll wheel, press the scroll wheel to select options; for devices with a trackball, there is a BB menu key to the left of the trackball, press it to select options -> smart dialing -> country/region code. Use the scroll wheel or trackball to select and change, set it to unknown. 4. When sending text messages on a BlackBerry phone, they display as ?????: You just need to find options -> sms and change the data encoding to ucs2 (with the sim card inserted, select options -> sms, find data encoding and use the scroll wheel to select and change it to ucs2). 5. When using QQ or Opera and other third-party software on a BlackBerry phone, it shows unable to connect to the server: You just need to go to options -> advanced options -> tcp and set all apn fields to cmwap. Restart the device afterward. 6. To check the ROM version of a BlackBerry phone: Go to options -> about. 7. How to switch input methods on a BlackBerry phone: In the normal interface, press alt+enter to switch (the alt key is at the bottom left of the keyboard, and enter I don't need to explain). 8. How to compose text messages on a BlackBerry phone: Just go to the text message interface, press the scroll wheel or the menu key next to the trackball, select compose sms (short message) -> enter the phone number -> enter the content and press the scroll wheel or trackball to select send. To enter numbers, hold down alt+number keys. 9. How to set message reports on a BlackBerry phone: Select options -> sms (short message) -> change send report to on. 10. How to change SMS and call ringtones on a BlackBerry phone: Find profile mode (if you can't find it, look for a loudspeaker icon on the desktop), enter and select advanced options. See various profiles, for example, select edit under the loud profile, which will show the loud profile's interface. Select the options you want to change (such as vibration, ringtone, etc.) and select save. 11. How to set shortcuts and one-touch dialing on a BlackBerry phone: In the dialing interface (dial 10086 but don't call out), directly set screen dialing to no, then you can use shortcuts (only works in English environment for 4.5 rom). For example, pressing M on the desktop enters messaging. Also, holding a letter can set one-touch dialing. (Thanks to netizens for the supplement~) 12. Why can't my BlackBerry phone receive things via Bluetooth, how to set Bluetooth: First pair the Bluetooth, then in the media section, press the menu and select receive via Bluetooth so files sent by the other party can be received normally, up to 2mb in size (except for 7290). 13. How to change the position of desktop icons on an 8-series BlackBerry phone with a scroll wheel: Press alt+scroll wheel simultaneously, and a menu will appear with the option to move icons. 14. How to switch applications (view background running applications) on a BlackBerry phone: Press alt+back key, or press the BlackBerry key in the main interface and select switch applications (home screen, short message, phone interface, browser, these four are default and cannot be turned off, att machines may have push-to-talk which can be deleted with optimization software). 15. What to do if the battery power of a BlackBerry phone is used up quickly: A BlackBerry phone is a smartphone, if programs are opened and not closed, they will keep running in the background. First, consider whether too many programs are open. Specifically, see item 14 on how to switch applications. To close applications, generally press the BlackBerry key and select close. 16. My BlackBerry phone keeps showing app error 523: There's nothing that can be done, only wiping and reflashing the phone will work. For detailed instructions, refer to Little C's flashing tutorial. 17. What to do if my BlackBerry phone shows SIM card invalid: This is because it hasn't been unlocked. Generally, the seller will provide an unlock code. You just need to go to options -> advanced options -> SIM card and enter MEPD to check the status. Before unlocking, the network lock is usually in an active state. Just enter MEP2 when a dialog box appears, input the code, and the unlocking is complete. 18. How to activate large capacity mode (storage card) function on a BlackBerry phone: After inserting the storage card, find the media card in the options, find "automatically allow storage card mode when connected via USB", change it to "ask every time". After plugging in the USB, there will be a prompt on the phone asking if you want to open large capacity storage mode, select yes. At this point, opening the drive on the computer will show the storage card. 19. For installing software and synchronizing contacts on a BlackBerry phone, please refer to the desktop manager usage tutorial. 20. Why did my network suddenly change from GSM to GPRS/EDGE: GSM indicates your phone's network environment lacks data communication signals or the signal is too weak, or your card doesn't have internet access enabled. In short, it means you can't access the internet. GPRS is a data communication protocol, EDGE is double GPRS 2.75G, theoretically twice the speed of GPRS. The network provided by China Mobile is basically 2.75G EDGE second-generation network. 21. I don't know how to place the SIM card in my BlackBerry phone: Just refer to the following image. 22. How to switch between uppercase and lowercase English letters: Hold the caps key at the lower left for uppercase, release for lowercase (when the input method is in English input state). 23. How to continuously input numbers: Hold the left caps (lower left)+alt to enter continuous number input mode, or you can hold alt+number keys to continuously input numbers. 24. How to continuously input capital letters: Hold the left alt+right caps (lower right) to enter capital letter input mode (when the input method is in English input state). http://shop.shichenghy.com
