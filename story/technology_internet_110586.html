1. Reinstalling WindowsXP does not require reactivation

If you need to reinstall WindowsXP, you usually have to reactivate it. In fact, as long as you back up the Wpa.dbl file in the Windows\System32 directory when you activate it for the first time, you don't need to activate it again. After reinstalling WindowsXP, you just need to copy the file to the above directory.

2. How do I know if my WindowsXP is activated

Open Start → Run, enter oobe/msoobe/a in the pop-up dialog box, the system will pop up to tell you whether the system is activated.

3. Disable the zip folder function

Do you find the zip folder feature in WindowsXP slow and unattractive? If so, you can go to Start → Run, type regsvr32/uzipfldr.dll in the dialog box that pops up, and press enter to turn off the ZIP folder feature.

4. Let WindowsXP can also carve ISO files

WindowsXP does not provide the ability to burn ISO files directly, but you can download a third-party plug-in to add this capability to your system. The plug-in download address is: http://members.home.net/alexfein/is... rderSetup.msi[/url].

5. The login interface background changes color

Open the registry editor, find [HKEY-USERS\.default \ControlPanel\Colors], and change the background value to "000" (without quotes) so that the login background is black.

6. Uninstall XP completely

Some friends in the installation of WindowsXP found that they do not like the none system, then you can use Windows98 installation CD to start to DOS state, and then type:

format-ur

This will delete all XP files and reinstall Windows98/Me.

7. Systems save lives

When it is suspected that changes made to the computer before the system reboot may cause a failure, you can restore the changes made to the system by pressing the F8 key while booting the system and selecting "Last Correct Configuration".

8. Restore the previous hardware driver

When you find that the system is unstable or the hardware does not work after installing a new hardware driver, simply select the "Driver Recovery" button in the "Device Manager" to return to the previous normal system state. But the printer driver cannot be restored.

9. Automatic login

Click Start → Run, type "rundll32netplwiz.dll, UsersRunDll" (without quotes), Then cancel in the UserAccounts "Usersmustenterausernameandpasswordtousethiscomputer", click "OK", then in the pop-up dialog box input user name and password you want to log in.
