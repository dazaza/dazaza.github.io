Windows 8.1 Countdown: 10 Days!

by anonymous on 2013-10-08 18:37:08

Users who have been eagerly awaiting Windows 8.1 can now start the countdown, as of today there are only 10 days until the arrival of Windows 8.1!

Existing Windows 8 users will be able to download the Windows 8.1 update for free from the Windows Store starting at 7 PM Beijing Time on October 17th. On October 18th, Windows 8.1 and new devices pre-installed with Windows 8.1 will be available globally.

Microsoft strongly recommends that existing Windows 8 device users upgrade to Windows 8.1 in order to enjoy a variety of new features and enhanced functionalities, such as improved personalization options, smarter Bing-powered Super Search integration, and over 20 new or optimized Microsoft apps and services.

For devices running older versions of Windows, Microsoft has provided the following methods to upgrade to Windows 8.1:

1. Purchase a new device pre-installed with Windows 8.1 from a computer manufacturer or Microsoft Surface.

2. Obtain Windows 8.1 via electronic software download from the Microsoft Windows Store.

(Since the launch of Windows 8, Microsoft has simplified the distribution model for Windows in China and no longer provides physical copies of the product.)

Regarding the pricing of different versions of Windows 8.1 and their services, the following points should be noted:

1. Windows 8 can be upgraded for free.

2. The price of Windows 8.1 (Standard Edition) in China is RMB 988.

3. The price of Windows 8.1 Professional Edition is RMB 1988.

4. Users of Windows 8.1 (Standard Edition) can also purchase the Windows 8.1 Professional Upgrade Pack to access all the features of the Professional Edition and Windows Media Center, priced at RMB 1088.

5. Windows 8.1 Professional Edition users can also purchase Windows Media Center for RMB 88.

It is worth noting that all copies of Windows 8 currently available on the market are "upgrade versions," meaning users can only upgrade from an older version of Windows. After listening to user feedback, Microsoft has changed the release of Windows 8.1 to include a "full installation version" during the download process, allowing direct installation without requiring an older version of Windows to be installed on the device.

Precautions for upgrading XP/Vista/Win7 to Windows 8.1:

1. Windows 7: Users can upgrade their Windows 7 PCs to Windows 8.1 while retaining all files. However, users will need to reinstall desktop applications, including Microsoft Office suites.

2. Windows XP and Windows Vista: For Windows XP or Windows Vista users, it is recommended to download the Windows 8 Upgrade Assistant from the Microsoft website (http://windows.microsoft.com/zh-cn/windows-8/upgrade-to-windows-8) before upgrading to Windows 8.1. It can scan your current computer to determine if it is ready for an upgrade to Windows 8 and provide a compatibility report along with optional steps for purchasing, downloading, and installing Windows 8. After upgrading to Windows 8, you can visit the Windows Store and directly upgrade to Windows 8.1. Microsoft suggests that users whose systems are deemed unsuitable for upgrade consider purchasing a new Windows 8.1 device instead of attempting a direct upgrade.
