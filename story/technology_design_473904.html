Citrix Provides Shanghai Mitsubishi Elevator with Seamless Application System Access Solution

by xiaoling4430 on 2011-08-10 14:24:19

The original intention of using application virtualization technology was to optimize the IT management of Shanghai Mitsubishi Elevator and extend the life of desktop PCs, Fan Bingxun, so that our employees and IT personnel do not have to face endless client installations, Fan Bingxun. After sufficient technical evaluation and cost assessment, it was further decided to adopt Citrix application virtualization as the only standard for the seamless access solution of Shanghai Mitsubishi Elevator's application system in the future.

-- Shanghai Mitsubishi Elevator

Shanghai Mitsubishi Elevator Co., Ltd. was jointly established by Shanghai Electric Machinery Co., Ltd. and Japan's Mitsubishi Electric and other four parties. In the 22 years since its establishment, it has absorbed international advanced technology, adhered to independent innovation, pursued excellent management, and maintained a leading market position in the industry for 16 consecutive years. It has now become an enterprise with the highest annual production and sales volume in the world, creating miracle after miracle and brilliance after brilliance for joint ventures, and creating a more harmonious living space for society.

Challenges

Currently, Shanghai Mitsubishi Elevator Co., Ltd. has dozens of branches (subsidiaries) across the country. Despite a large amount of funds being allocated each year for renting high-quality dedicated data lines between branches and headquarters, the response to application requests from branches by the headquarters' application system is still slow.

At the same time, the lifespan of a total of thousands of desktop PCs within the company is only 3-5 years, and the fixed terminal equipment procurement costs each year remain high. Moreover, the workload for client deployment, maintenance, and upgrades in dozens of branches (subsidiaries) is also very large. The IT management department urgently needs to find an excellent solution to comprehensively address these issues.

Problems to be solved

1. Provide remote seamless access to the application system for dozens of branches across various locations.

2. Extend the lifecycle of desktop PCs and reduce the company's procurement costs for desktop PCs.

3. Reduce the application maintenance workload in various branches and simplify IT management in the branches.

Solution Selection

During the technical evaluation process of Citrix application virtualization solutions, Shanghai Shangxin Information System Engineering Co., Ltd. held multiple face-to-face exchanges with Shanghai Mitsubishi Elevator and meticulously built a test environment that met the actual needs of Shanghai Mitsubishi Elevator. Additionally, they conducted multiple rounds of solution improvements and verifications according to the requirements.

The entire solution achieved stable bandwidth usage between headquarters and branches, reduced client performance requirements, extended the lifecycle of client PCs, lowered client deployment and maintenance costs, and simplified the workload for IT deployment and upgrades in branches. Furthermore, issues such as terminal maintenance efficiency, remote access speed, and security were all resolved due to the centralized deployment of applications. Ultimately, Shanghai Mitsubishi Elevator selected Citrix application virtualization technology as the sole standard for seamless access to the company's application systems.

Leadership in IT management at Shanghai Mitsubishi Elevator stated that the reason for choosing Citrix application virtualization technology was because it truly achieved architecture openness, seamless access, rapid deployment, and extensive terminal device compatibility, as well as good application compatibility during the technical evaluation process. At the same time, Citrix application virtualization technology is highly mature, and successful deployments in Fortune 500 companies gave them great confidence.

Technical Advantages of the Solution

1. Centralized application deployment, optimizing IT management

Citrix XenApp collects all application clients on the server side and then distributes and manages them uniformly via Citrix XenApp servers. Custom templates can be created for each user, and no plugins need to be installed when accessing terminals. All clients are maintenance-free, significantly reducing the heavy maintenance workload of the IT department. New applications can be deployed quickly, and upgrading application software requires operations on only a few servers.

2. Diverse client support

Once Citrix XenApp is deployed, users can directly access all backend applications by installing the Citrix Receiver software on their mobile devices without professional assistance, Fan Bingxun. Citrix Receiver can support almost all desktop operating system platforms and various mobile platforms. Together with Citrix XenApp, Citrix Receiver creates a new generation of mobile application solutions, enabling access to backend business and office systems anytime, anywhere, through any terminal device.

3. Fast speed

A notable feature of Citrix XenApp is its low requirement for network bandwidth, Fan Bingxun. Under normal application conditions, each user occupies an average of only about 20Kb of bandwidth. Satisfactory speed and performance can be achieved through the Internet, saving a large amount of dedicated line costs.

4. High availability

Since all applications and data are completely managed in the backend, even if there is a network issue, the system process will not be interrupted, and the operation status of the front-end desktop will be automatically saved in the backend, Fan Bingxun. Once the network link is restored, all operations can immediately continue. Load balancing can be done between Citrix servers, and the system will automatically distribute users to different machines based on the strategies set by administrators or the load of each machine. Except for the License Server, the failure of any server does not affect the use of other servers.

5. High security

Applications and data run entirely on the server side, and the client only sees the changes in the server's computational screen, without actual data computation, ensuring the concentration, uniformity, and security of data. Even when using the public Internet, strict security can still be guaranteed.

Client usage permissions can be set according to policies, including whether they can see the desktop, whether they can print, and whether they can save to the local device. This ensures that information usage permissions are controllable.

6. Low project implementation risk

Since the deployment of the Citrix desktop virtualization solution does not require modifying or rebuilding the existing system, and the existing system does not need any downtime to switch to the new system, a new system can be built by simply adding a new server layer to the network. The original operation interface and usage methods are fully retained, eliminating the need for additional user operation training.

Main Benefits

1. IT maintenance in branches is greatly reduced, requiring only the unified deployment of Citrix clients. The remote login speed and availability of clients have also significantly improved, ensuring business continuity.

2. The actual lifespan of PC terminals has been extended by twice. In practical testing, laptops with PIII CPUs from several years ago in the customer's warehouse can quickly access all applications through the Citrix client.

3. The deployment of the new solution has no impact on the existing system architecture. The original operation interface and usage methods of terminal users are fully retained, eliminating the need for additional user operation training.
