FB 用户购买 Oculus VR 耳机以获得客户服务优先级

2021-08-05 20:42:13

安吉拉·麦克纳马拉 (Angela McNamara) 第一次暗示她的 Facebook 帐户已被黑客入侵是清晨的一封电子邮件，警告说有人试图登录她的帐户。 “如果这不是你，别担心，我们会保护你的账户安全，”她回忆起 Facebook 的电子邮件说。但她的缓解只持续了一分钟，当另一封电子邮件到达时，说她的密码已被更改。然后是另一个，通知她已为她的帐户设置了双重身份验证——额外的安全层。住在多伦多郊外的麦克纳马拉说：“然后我就想，'好吧，它不见了，'”。她尝试了 Facebook 的自动流程来恢复她的帐户：获取备份代码、重置密码。但没有任何效果。最近很多人都遇到了这种情况，这种体验让许多用户对社交网络感到几乎和对黑客一样感到沮丧。 7 月，NPR 收到了 19 封来自听众的电子邮件，抱怨他们的 Facebook 帐户被黑客入侵或禁用。人们每天都在 Reddit 论坛和 Twitter 上分享类似的悲惨故事。有些人变得如此绝望，以至于他们斥资数百美元购买虚拟现实耳机，试图让 Facebook 恢复他们的帐户。在走向极端之前，许多黑客受害者尝试通过通常的途径获得客户服务，但很快发现似乎无法联系 Facebook 的某个人来帮助解决问题。 “Facebook 没有电话号码可以拨打。没有电子邮件可以发送电子邮件，”住在芝加哥郊外的杰西·马尔萨拉 (Jessie Marsala) 说，她在 7 月初通过电子邮件向 NPR 介绍了她的情况。

当 Marsala 遭到黑客攻击时，她尝试拨打 Facebook 在硅谷的总部。但是这个数字会产生一段录音，上面写着：“很遗憾，我们目前不提供电话支持。”相反，Facebook 告诉用户通过其网站报告被黑账户。该网站指示他们上传驾照或护照的副本以证明他们的身份。但 NPR 采访的人说，他们在这个自动化过程的每一步都遇到了麻烦，希望 Facebook 能够提供一种接触真人的方法。 “我每天早上、中午和晚上多次发送这些表格，”Marsala 说。 “没有人回复我，一次都没有。”新泽西州泽西市的维多利亚·弗洛里亚尼 (Victoria Floriani) 在用便利贴掩盖了除姓名和照片之外的所有内容后，才让 Facebook 的系统接受了她的驾照——这是她在 Reddit 上看到的一个提示。经过两周的尝试，这是她恢复帐户所需的突破。 Facebook 表示，由于冠状病毒大流行，它可以查看 ID 的人数较少。它也使用人工智能，但其帮助中心警告说，评论“可能需要比平时更长的时间”。 Facebook 发言人 Gabby Curtis 在一份声明中告诉 NPR，该公司的帮助中心每天 24 小时可用，以帮助有问题的人并报告问题。但柯蒂斯承认，“我们也知道我们需要在这方面不断改进，并计划在未来进行更多投资。”加利福尼亚州内华达市的布兰登·谢尔曼 (Brandon Sherman) 遵循他在 Reddit 上发现的一条提示，以找回他被黑的帐户。

“我最终崩溃了，买了 300 美元的 Oculus Quest 2，”他说。 Oculus 是 Facebook 旗下的一家虚拟现实公司，但拥有自己的客户支持系统。 Sherman 用他的耳机序列号联系了 Oculus，并立即得到了回复。他计划归还未开封的设备，虽然他很高兴这个策略奏效，但他认为这不公平。 “获得任何客户服务的唯一方法是证明你确实从他们那里购买过东西，”他说。当加拿大的 Facebook 用户麦克纳马拉第一次听说 Oculus 技巧时，她认为这是一个笑话。但她说，“一旦我开始思考它，我所有的记忆，我真的意识到我想尽一切可能把它找回来。”所以她也订购了一个她从未打算使用的昂贵小工具，并在她回到她的 Facebook 帐户后立即退货。 （警告任何想尝试的人 - 其他 Reddit 用户表示他们尝试联系 Oculus 支持，但无法恢复他们的 Facebook 帐户。此外，上周，Facebook 表示暂时停止销售 Oculus Quest 2，该零售起价 299 美元，因为它的泡沫衬里对一些顾客造成皮肤刺激。）“在意识到我被黑客入侵后，第一个担忧是这些人可能能够访问我公司的银行账户，”本科尔曼说”在马萨诸塞州福尔里弗。“那将是一场灾难。

Coleman 的日常工作是从 12 年级向幼儿园儿童教授数学和技术，但他也用无人机拍摄视频，并撰写有关如何折叠折纸盆景树的书籍。对于这两个企业，他都依靠 Facebook 来接触客户。 Coleman 设法在黑客获得控制之前锁定了他的 Facebook 帐户。但他无法解锁——所以他无法访问所有东西。对于密歇根州谢泼德市的 Jon Morgan 来说，情况变得更糟了。黑客使用他的 Facebook 帐户破坏了他帮助管理的小镇枫糖浆节的页面。 Facebook 禁用了摩根的帐户——因此现在他无法访问许多家庭照片，包括他今年去世的儿子的照片。摩根说，这一集让他意识到 Facebook 是如何融入他的生活的。他说：“我们认为它有点像玩具或有趣的东西，但人们会在上面分享新闻，人们从中获取新闻，这是一本相册。” “我认为我从我的经验中学到的是，我真的需要考虑我如何使用它......以及失去它对我意味着什么。”在 NPR 与 Facebook 取得联系后，它发送了 Coleman 和 Marsala 链接来解锁他们的帐户，并为摩根提供了另一个机会就其帐户被禁用提出上诉。 Facebook 表示，它最近没有看到黑客活动的增加，目前还不清楚人们联系 NPR 的黑客活动的幕后黑手是谁。

网络安全公司 Trend Micro 威胁情报副总裁 Jon Clay 表示，许多入侵社交媒体帐户的尝试都是出于经济动机。他说，黑客可能会试图欺骗用户的朋友和联系人，给他们钱，或者在黑市上出售账户。克莱说，其他黑客想窃取 Facebook 帐户以传播虚假信息，例如 2020 年大选或 COVID-19 等话题。 “社交媒体现在是每个人生活的重要组成部分，这意味着它是一个主要目标，”他说。
