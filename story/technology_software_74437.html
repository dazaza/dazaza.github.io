Ghost has vulnerabilities, be careful of traps when restoring images

by xloong on 2008-06-16 15:10:38

Nowadays, various universal Ghost installation CDs and files for the Windows XP system are popular both on the market and online. They are indeed convenient to use; installing a system only requires restoring the Ghost image file, installing some drivers, and the whole process takes just a short ten-odd minutes. However, among these various universal Ghost version system CDs, there are both high-quality ones and junk ones, and even some with hidden backdoor traps!

### 1. Analysis of Windows XP Universal Ghost Systems

When creating a universal Ghost system, after a successful installation, the creator deletes unnecessary files that come with Windows by default and removes hardware information before encapsulating the system. If, before the installation, the creator intentionally replaces a certain system file with a Trojan backdoor, opens some ports, enables dangerous services, or leaves accounts with blank passwords, then the resulting Ghost system will have various security vulnerabilities. When such Ghost systems are distributed and used, the users may be controlled by the creator as "meat chickens" (compromised computers).

### 2. Common Vulnerabilities in Ghost System Versions

1. **Blank Password Remote Desktop Vulnerability**: Allows remote login via port 3389 with a blank password, enabling remote system task operations. This vulnerability can be exploited to steal Q-coin, ADSL passwords/accounts, etc.

2. **Hidden Share Vulnerability**: Any user can access shared resources, non-default IPC$ shares with Everyone Full Control permissions. This can lead to severe consequences, such as Guest group users being able to format your hard drive.

3. **Administrator User Password Vulnerability**: Not much needs to be said about this well-known issue.

4. **Enabled Dangerous Services**: In the service tools, many dangerous services are found to be enabled, and the remote tab allows users to remotely connect to this computer.

5. **Tampered Firewall Settings**: In the system firewall, all unchecked default items allowing traffic through are now checked.

6. **Malware and Backdoors**: The system installs many rogue software programs without permission. Even more terrifyingly, it replaces system files with Homing Pigeon (Gray Pigeon) Trojan horses! (There are also software tools that can clone system file version information, making the Trojan files appear identical to legitimate system files, including icons, sizes, labels, etc.)

### 3. Detection of Dangerous Ghost XP System Versions

The following is a list of known problematic versions:

1. **Tomato Garden Series**: Tomato Garden Windows XP Pro SP2 No Activation Required v2.8, v2.9, and newer versions.

2. **Rainforest Wind Series**: Rainforest Wind Ghost WinXP2v2.0 Installation Edition Pure Member Edition y1.7 v1.85 and newer versions.

3. **Donghai Computer Company Edition**: Ghost XP_SP2 Computer Company Special Edition v4.0, v4.1, v5.0, v5.1, v5.5, and newer versions.

You can check your system version in the system properties dialog box to determine if it has any issues. Other Ghost system versions circulating online also have varying degrees of the aforementioned security problems! Please use them with caution!
