SkyDrive客户端软件

by xjyzhenai on 2009-05-09 14:39:13

SkyDrive是微软Windows Live系列服务中的其中一项，在 Windows Live 服务器上免费存储、管理和下载文件、照片和收藏夹 (表示收藏的网站，其网址保存在计算机或联机服务器上，以便您方便快捷地进行访问。)SkyDrive为用户提供了25 GB 的照片和文件存储空间。

　　但微软的SkyDrive服务却没有提供什么客户端软件。虽然SkyDrive服务还不错，空间也算大，但没有客户端软件的支持，文件管理非常不方便，比如删除照片，只能一张一张地删。难道MS的产品设计师没脑子？有消息称微软注册了两个与SkyDrive相关的域名：

*Skydrivesync.com

*Syncskydrive.com

　　大家可以YY一下这两个域名的意思，是用来做什么的。不过迟迟不见微软有什么动作。于是，大家只好使用第三方的软件来作为SkyDrive客户端了。比如Gladinet Cloud Desktop。

　　Gladinet Cloud Desktop可以在这里下载

　　下载安装后，进入到设置界面：

image

　　你可以选择一个驱动器盘符比如L，再设置一个临时文件目录，就行了。现在双击打开L盘：

image

　　从上图我们可以看出来Gladinet Cloud Desktop支持多家的在线存储服务。今天我们只看微软的。双击挂载Windows Live SkyDrive，弹出一个登陆SkyDrive的界面。

image

　　输入你的Live ID和密码，便连接到你的SkyDrive了。现在整个SkyDrive就成了你的L盘下的一个文件夹了。可以跟操作本地硬盘一样的操作了。而数据的变更，则是保存于SkyDrive的服务器之上。

　　虽然不是微软自家的东东，但Gladinet Cloud Desktop已经很好的实现了SkyDrive本地化的功能，作为第三方产品，与原服务无缝结合是不可能的事。所以，我们仍希望微软继续完善自身服务，多以市场需求为导向，并尊重用户的意愿，推出更好的产品。
