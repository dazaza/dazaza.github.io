Portal-level website construction and management software

by idpx on 2008-01-04 12:02:54

SiteEngine (Website Engine) is a portal-level website construction management software (Web Portal System). After 5 years of development, it has been successfully applied tens of thousands of times across dozens of industries.

SiteEngine has been fully upgraded to the 6.0 series, including personal free version, standard version, professional version, advanced version, business information portal version, multi-domain version, cluster version, etc., providing application solutions that integrate website construction, management, and operation for government, enterprises, e-commerce, education, and various industries.

SiteEngine is developed based on PHP + MySQL, with a B/S structure, spanning multiple operating systems such as Windows, FreeBSD, Unix, Linux, Solaris, etc. It comprehensively adopts the UTF-8 global standard encoding, making it an internationalized and standardized software.

SiteEngine 6 features stronger industry applicability, flexible customization and expansion mechanisms, personalized template technology, diverse user permission management, clear content management modes, multiple web interface methods, excellent operability, and comprehensive search engine optimization techniques, making it an extraordinary portal-level website construction management software.

Main Features:

Internationalization, Standardization: UTF-8 international standard encoding, the system is fully built using universal functional modules and can be customized for expansion.

Flexible system extension: Free combination of functions and channels, arbitrary field expansion, customizable templates, unlimited content classification.

Excellent template mechanism: Separation of program and template, separation of template and content, whole-site personalization, partial personalization, style inheritance, user-added templates, custom template tags.

ERP user management model: Distributed, transcendent, diversified, personalized user management.

Clear content management mode: Independent systems for information content and channel content, collaborative and synchronous management.

Human-oriented design: New user management wizard, common operation settings, operation process synchronized prompts, F1 help activation.

WEB interface technology: Single Sign-On (SSO), batch data import/export, remote data push.

Comprehensive search engine optimization: Whole-site content SEO customization, true/false whole-site static pages, whole-site Sitemaps submission.

Dreamweaver plugin: Supports DW CS3, UTF-8 encoding supports any language, synchronized retrieval of system classification data, synchronized multi-task template processing.

WAP website: Synchronously builds WAP websites, automatic recognition and matching of user access terminals, simultaneous construction of multiple language versions, enables user online interaction.

Secure deployment: Boka Security Engine, triple COOKIE protection.

Others: Seamless integration of SMS in business processes, Chinese word segmentation technology for content reorganization, provision of fifteen payment interfaces.
