Facebook reveals details of global connectivity "1000X"

by anonymous on 2013-09-24 20:59:46

On September 17th, Beijing time, last month Facebook joined hands with Opera, Nokia, Ericsson, MediaTek, Qualcomm, Samsung and other companies to establish the Internet.org alliance organization. They hope to bring the internet to 5 billion users globally, breaking regional restrictions so that everyone has the right to access the internet. Today, the organization released a detailed 70-page white paper explaining the project's details.

Facebook: "Efficiency" first

Before founding Internet.org, Facebook had already been attempting to improve the efficiency of internet access.

Facebook started with an $85 host. Since its establishment, Facebook has been exploring ways to enhance the efficiency of its services. Previously, Facebook launched the HipHop project, which consisted of a PHP-to-C++ conversion program, a rewritten PHP runtime library, and many rewritten versions of common PHP extensions. The aim was to accelerate and optimize PHP.

The HipHop project could increase the speed of each host by 50%, but this was not enough. Subsequently, the company introduced the open-source HipHop Virtual Machine, which increased the entire host service speed by five times.

In addition, Facebook also launched the Open Compute Project (OCP), helping everyone build more efficient hosts and data centers. This project focused on cooling, energy conversion, and simple DIY host design. The OCP project has made Facebook's data center in Lulea, Sweden, one of the most efficient data centers globally. Facebook currently stores 250 billion images on its hosts, exceeding 250 petabytes of data.

Today, Facebook outlined the future of "data efficiency" in the white paper.

To more easily test global data efficiency and stability, Facebook created the "Air Traffic Control" system. This is a system specifically designed to help engineers simulate different network environments within Facebook's offices. The system can control bandwidth, latency, packet loss, damaged packets, and message procedures.

Facebook explained that the "Air Traffic Control" system can test various mobile wireless technologies using WiFi, such as G, EDGE, 3G, and 4G, simulating how these wireless technologies run when using Facebook applications under different national network conditions. It also tests how different network capacities and speeds affect user experience and data connections.

Moreover, Facebook is transitioning image formats to Google's WebP digital image format. Images are Facebook's largest data source. Reducing image space usage would generate significant profits for Facebook. Facebook stated: "Most images on the Android platform will be converted to the WebP format, and our goal is to apply the WebP format to other platforms. When all images are converted to the WebP format, it will save over 20% of network traffic without compromising image quality."

Facebook Home as a testing platform

Additionally, Facebook used the new Android interface, Home, launched last year, as a testing platform. Although Home did not gain much attention after its release, Mark Zuckerberg was disappointed, but now at least Home can still help Facebook conduct tests in the 1000X project.

For instance, since Home can keep the image information on the device unlock screen updated, Facebook can save data charges for users. Home can also decide whether to load high-definition or lower-resolution images by monitoring whether the user is in a WiFi environment or using mobile connections.

Furthermore, Facebook Home uses intelligent caching to avoid excessive image downloads and supports importing cached content into removable SD cards, thus freeing up device memory space. This technology is especially useful in developing countries where phones generally have low memory.

Additionally, Home sets virtual limits on total device data usage to prevent users from unknowingly consuming too much data. These data limits ensure that users in developing countries concentrate their data usage on the most important services (such as communication) rather than downloading images.

Due to the lack of convenient, reliable charging systems in some countries and regions around the world, TV lifespan is another issue faced by the 1000X project. WiFi consumes less power compared to mobile data connections, so it can help extend battery life for mobile phone users worldwide.

Currently, avoiding excessive use of device "backlighting" is also one of the goals of Facebook launching Home. Facebook attempts to automatically refresh the information stream in WiFi environments to reduce data loading via mobile data connections. Facebook explained that even if the device runs Ice Cream Sandwich or the latest operating system, enabling mobile data connections usually consumes 0.02% - 0.1% of the device's power, which is why Facebook collaborated with HTC to embed Home in the HTC First.

Another hardware innovation is Facebook's collaboration with GPU suppliers to adjust the phone's workload, allowing the application processor to operate in an ideal power-saving mode without affecting the user experience. Additionally, Facebook has a device power evaluation system to test power consumption under different environments.

Through Home and battery consumption optimization, Facebook is creating a mobile ecosystem for Internet.org that allows everyone to enjoy affordable data and more power, regardless of location or wealth.

Qualcomm: Spectrum allocation

Qualcomm proposed the "1000X Challenge" project in this white paper. If data usage doubles every year, we will need 1000 times more data traffic in ten years. Qualcomm is currently challenging itself to develop technology to achieve this goal. Below are several innovations proposed by Qualcomm:

1) Aggregating separate spectrums through "Carrier Aggregation" and "Supplemental Downlink" to provide users with more data capacity and faster transmission speeds;

2) LTE-Broadcast technology for video and data multicast;

3) LTE-Direct technology (users can establish device-to-device communications when the mobile network crashes);

4) Next-generation Wi-Fi standard "802.11ac" and other unlicensed applications;

5) DSRC (allowing vehicles to "communicate" with other vehicles to avoid collisions) and next-generation systems providing broadband for airplane passengers;

Another key technology authorization is called "small base points" or low-energy wireless access points (within a 10-meter range). These technologies will be integrated into wireless networks, establishing a "heterogeneous network (HetNet)" of varying sizes inside users' homes.

The biggest obstacle facing Qualcomm's "1000X Challenge" project is the allocation of wireless spectrum, requiring tons of spectrum to cover the globe.

Ericsson expressed that they will incorporate continuous internet connection, faster speeds, and more stable network applications into their services, even if people know nothing about cloud services.

The mission of Internet.org

Although the technology required for Facebook and Qualcomm's "global network connectivity" blueprint may take years to achieve, it is crucial that they have begun implementing the project. If their efforts ultimately make data traffic fees unaffordable for everyone, their business will cease. However, Internet.org acknowledges that they are working hard to create more profitable mobile companies.

"Network connectivity" is a human right, which is the belief supporting Internet.org. Although Internet.org partners have not revealed how internet connectivity affects the global economy, their idea is that internet connectivity brings efficiency and increases everyone's output.

If tech giants focus on global network connectivity, we will eventually achieve this goal, something our ancestors could only dream of.
