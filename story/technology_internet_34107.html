20 Energy-Saving Special Execution Commands in Windows

by xloong on 2007-11-12 19:29:38

1. **systeminfo**: Make XP list more useful information

Windows XP always boasts about how long it can run stably! To get detailed information, you can access the "Start Menu," then open the "Command Prompt" from the "Accessories Menu," and type the command "systeminfo." The computer will display a lot of useful information, including the initial installation time of the system and the duration of the current uptime. If you want to save this information, you can type "systeminfo > info.txt," which will create a text file named "info.txt" that you can later open with Notepad to review. (Limited to Windows XP Professional version)

2. **gpedit.msc**: Set direct file deletion

You can configure the Windows XP operating system to directly delete files without moving them to the Recycle Bin first. Access the "Start Menu," select "Run...", and input "gpedit.msc"; in the dialog box that appears, choose User Settings, Administrative Templates, Windows Components, Windows Explorer, and find the option "Do not move deleted files to the Recycle Bin." Enable this option.

By adjusting the relevant settings in the gpedit dialog box, you can make many adjustments to the system's operation and options, but be careful — some changes you make may cause issues or instability in the system’s performance. (Limited to Windows XP Professional version)

3. **rundll32.exe**: Lock your computer with a mouse click

You can lock your Windows XP system with just two clicks of the mouse. First, right-click on the desktop to create a new shortcut, and when filling in the location, input "rundll32.exe user32.dll,LockWorkStation."

Then give this shortcut a name you like, such as "Lock Computer." It's that simple — now, you just need to double-click this shortcut with your mouse, and your computer will lock immediately. If you think this is still too complicated, holding down the "Windows key (the one with the XP logo)" and "L" key simultaneously on the keyboard will have the same effect.

4. **sysoc.inf**: Bring hidden files back

Windows XP hides some system software that you might want to delete or uninstall, such as Windows Messenger, but you can make these programs reappear. Use the built-in Notepad or WordPad program in Windows to edit the text file "/windows/inf/sysoc.inf," search for and delete the keyword "hide." Then go to the "Control Panel," "Add/Remove Programs," and select "Add or Remove Windows Components." You will see all system software displayed, and now you can uninstall them.

5. **/?**: The allure of the command line

For users skilled in DOS batch commands, Windows XP offers many interesting new commands. These include the "eventcreate" and "eventtriggers" commands used to create and monitor system events, the "typeperf" command used to monitor the operational status of various subsystems, and the "schtasks" command used to manage scheduled tasks.

As before, typing a command followed by "/?" will provide a series of options related to that command.

6. **ipv6 install**: XP also supports IPv6

Windows XP supports IPv6 — the next-generation IP protocol for the Internet. Unfortunately, this feature exceeds the services supported by your network provider (ISP), so you can only experiment with it on your local network. Input the command "ipv6 install" in "Run..." (don't worry, this command won't disrupt your existing network settings), then type "ipv6/?" in the Command Prompt to get more related information. If you don't know what IPv6 is, then please ignore this section.

7. **tskill**: Try terminating tasks with commands

You can terminate a task by typing the command "taskkill/pid" and the task number, or simply "tskill" and the process number, in the Command Prompt. Want to find all running tasks and their numbers? Type "tasklist," which can also help you understand more about the current system operational state.

8. **Fast or Slow**: Browse ZIP files

Windows XP treats ZIP compressed files like regular folders, which is very convenient on fast-running machines, but it can slow down the overall efficiency of slower-running machines. If your computer's configuration isn't high, you can disable this feature by typing the command "regsvr32 /u zipfldr.dll" in the Command Prompt, leaving ZIP files alone. If you change your mind later, you can restore this feature by typing "regsvr32 zipfldr.dll."

9. **ClearType**: Protect your window to the soul

Windows XP supports "ClearType" — Microsoft's anti-aliasing font display technology — but by default, this display effect is not enabled. This feature is worth trying, especially if years of using computers have made your eyes extremely tired and your vision has started to decline. To enable this effect, right-click on the desktop with the mouse, select "Properties," "Appearance," "Effects," choose "ClearType" from the second drop-down menu, and turn it on. This will help achieve the best display results on laptops. If you want to use "ClearType" in the Windows XP welcome login screen, set the value of the registry key "HKEY_USERS/.DEFAULT/ControlPanel/Desktop/FontSmoothingType" to 2.

10. **Remote Assistance**: No longer limited by internal networks

You can help a friend using Network Address Translation (NAT) on a home network through Windows XP's remote assistance feature, though it's not automatic. Have your friend send you a remote assistance invitation, then edit this text. Under RCTICKET properties, you'll see an NAT IP address, possibly something like "192.168.1.10." Replace this internal address with your friend's real IP address — they can check it at www.whatismyip.com — and ensure that port 3389 on their firewall is open. Then you can directly access the problematic computer.

11. **Multi-user operation**: No need to log out the current user

You can run a program under another user identity without logging out the existing user. Right-click on the icon of the executable program, select "Run As...", and then input the username and corresponding password you wish to use. This setting will only apply to this instance of program execution. When you need to install a program as a system administrator, this trick will be very useful. However, please note that you can run multiple programs under different user identities simultaneously within the system, which might be interesting but could produce unforeseen effects on the system.

12. **Instant Start Auto Update**: Eliminate your waiting troubles

Windows XP will stubbornly prompt you to check for automatic updates, register a Microsoft Passport, use Windows Messenger, etc. Although these interruptions will gradually disappear after some time, if you feel like you'll go crazy before then, run "Regedit" and go to

"HKEY_CURRENT_USER/Software/Microsoft/Windows/CurrentVersion/Explorer/Advanced," create a "DWORD" value named "EnableBalloonTips," and set its value to 0.

13. **Auto Login**: No need to enter your password every time

You can choose to automatically log into the Windows XP desktop without entering a username and password each time. In the "Start Menu," select "Run," and input the command "controluserpasswords2," which will open the user account management program. In the "Users" tab, find the option "Users must enter a username and password to use this computer," uncheck this option, and save the settings. An auto-login setup dialog box will appear; input the username and password you wish to use for automatic login.

14. **Housekeeper**: Automatically delete temporary files

Internet Explorer 6 can automatically delete temporary files, but you need to make the relevant settings. Open the web browser, select "Tools/Internet Options...", choose "Advanced," go to the "Security" area, and select the option "Empty Temporary Internet Files folder when browser is closed."

15. **Network Status Always in Control**: Experience the free network indicator light

Windows XP comes with a free network activity indicator because you cannot see the LEDs on your network card. Right-click on the "My Network Places" icon on the desktop, select Properties. Then right-click on the icon for your LAN connection or dial-up connection, select Properties, and check the option "Show icon in notification area when connected." Select Save Settings. Then you will see a small network icon in the taskbar at the bottom right corner of your desktop, which will flash when your network communication is active.

16. **No More Delays**: Create a quick Start Menu

Sometimes, the pop-up speed of the Windows XP Start Menu feels overly leisurely, but you can speed up this process. Simply go to the registry key "HKEY_CURRENT_USER/ControlPanel/Desktop/MenuShowDelay," and change the default setting of 400 to a more satisfying number, such as 0.

17. **Batch Rename Without Tools**

You can rename multiple files simultaneously in a Windows window. All you need to do is highlight all the files you want to rename in the window, then right-click on one of them and rename it. The other selected files will also be renamed with this name, with different numbers added in parentheses at the end to distinguish them. Similarly, in a folder, you can rearrange icons by viewing them "By Name" or "By Type."

18. **Listen and Watch**: Customize your Media Player

Windows Media Player can display the related album cover image while playing CD music — if it finds the image online when you copy music from your CD to your computer. If it doesn’t find it, or if you already have many ready-made WMP music files, you can place your existing album cover images in the same folder as these musics. Just name this image "Folder Name.jpg," and Windows Media Player will recognize the image and display it as the album cover when playing these musics.

19. **Windows Key**: Different combinations serve different purposes

"Windows key + Break key" displays the System Properties dialog box; "Windows key + D key" shows the computer desktop; "Windows key + Tab key" switches between taskbar icons. (The Windows key refers to the key on the keyboard with the Windows XP logo.)

20. **Avoid Foolish Operations**: XP acts as a reviewer

Windows XP understands that ordinary users often don't know what they're doing. Therefore, it won't let you perform truly foolish operations, such as deleting the Windows system directory (unless you spend several hours convincing it that you really want to).
