Baidu PC input method exposure

by geekzhang on 2010-09-06 16:20:29

In the early morning of September 6, Baidu Chinese input method PC version (hereinafter referred to as Baidu Chinese input method) was quietly exposed. In this test version, the interface design is similar to Baidu Japanese input method, and the search function is built into the input box. Some users find that this input method is incompatible with Google Chinese input method.

In previous reports, we have revealed that at the end of August, early September, Baidu will release three client products, which will include Baidu Chinese input method. And the latest information shows that the product is very close to launch.

Our newly obtained Baidu Chinese input method client shows that the installation package size of this product is 14.2MB, the kernel version number is 1.0.0.72, and the product name is Baidu Chinese input Method. According to sources, the above input method products have been tested inside Baidu, and the test scope was further expanded about a week ago.

In the installation agreement, which was signed in August this year, Baidu said it was a free input software.

This version of the client is an early Alpha release. Its interface style and Settings are very similar to the Baidu Japanese input method introduced last year. However, in addition to the normal pinyin input functions, Baidu Japanese input method skin, emoji, handwritten input and other functions do not appear in this input method.

It is worth noting that users can search directly through the input box of Baidu Chinese input method while entering text. It is said that the future Baidu Chinese input method will also provide more cloud search services. At the moment, the websites related to this product, shurufa.baidu.com and ime.baidu.com, are not available to the public.

Some users who participated in the test told us that after installing this version of Baidu Chinese input method, there was an incompatibility with Google Chinese input method, and the phenomenon was that Google Chinese input method could not be used normally.

On November 25 last year, Baidu officially launched Baidu mobile input method. During the period, Wang Xiao, director of Baidu client software Division, told us that Baidu input method products were not first launched on the PC side because the mobile phone side has a large development space. Baidu employees also privately revealed that this strategy is also to avoid the relatively mature and competitive PC market in the early stage.

However, due to the relatively strong platform portability of the input method, it is widely speculated that Baidu will launch the PC input method next. At the time, Wang Xiao responded to the speculation by saying that Baidu had no definite ideas and plans.

Guo Goji, the former chief strategy officer of Google China, has publicly analyzed that in recent years, the reason why we are crazy for the input method is "because the user conversion cost of the input method is low, the stickability is high, and it is easy to be used as a medium to promote other products or content." According to Guo, Google missed the opportunity to develop the input method in 2005 due to a lack of personnel.

Baidu Japanese input method "Baidu Type" on December 17 last year quietly launched and began public testing, the software installation package size of about 39MB. In March this year, Baidu's official website publicly recruited input method research and development engineers. As of press time, we have not yet learned the specific release time of Baidu Chinese input method.
