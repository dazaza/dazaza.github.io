Wubi input method, input method, another main battlefield of the Internet?

by mennei on 2009-12-07 03:26:31

What are the most basic applications of a network, including the Internet and mobile networks? Not only networks, but also terminals such as desktops, laptops, and mobile phones, should have some basic functions. It's not a browser, not instant messaging (IM), not antivirus software, not office software, but rather an input method. Whether or not your computer is connected to the Internet, whether your mobile phone is just used for sending text messages or for surfing the web, you need an input method. The input method is the most fundamental application of the network era, but it has also been consistently overlooked. Many hardware and software products, as well as content services based on the network era, have shaped many successful enterprises, but it seems that only the input method has always been a supporting role, watching the changes in the network world without ever dominating it.

Speaking of Chinese input methods we've encountered, the earliest one should be Wubi Input Method. It was really difficult. I memorized the character root table in high school when I was using Wubi, but fortunately, I eventually came into contact with phonetic input methods. The earliest phonetic input method was probably Microsoft Pinyin, followed by full pinyin, and eventually Intelligent ABC. Chinese phonetic input methods have continuously strived for improvement in intelligence and user experience. Among phonetic input methods, the one that left a deep impression on me was Ziguang. However, there might have been issues with the technical team because Ziguang often had bugs. Subsequently, Sogou Input Method quietly took over the lower-right corner of many people's computers. If input methods were previously either bundled products like those from Microsoft or the result of small teams, then Sogou Input Method marked the beginning of serious efforts by major players in the Chinese Internet industry to cultivate this market. Afterwards, Google, Tencent, and even Baidu, which recently acquired a mobile input method company, all began entering the input method market. This might become another fiercely contested battleground for Internet giants in the future. Did Google's input method completely fail against Sogou Input Method? There was also a lot of hype around the recent lawsuit between Sogou and Tencent regarding input methods. For the Internet world, resolving disputes through legal means may become the mainstream in the future, and this might also be the best way to resolve competition. Sohu's early strategic planning can be seen in its current success: According to a survey report by iResearch, 43.7% of users who use Sogou Input Method come from friend recommendations. This far exceeds the proportion of users gained through promotion on new media or traditional media. At the same time, Sogou Input Method has reached over 100 million daily active users. In 2009, nearly 80% of users used Sogou Input Method, making it the most commonly used input method software, with a user share exceeding 50%, far ahead of other input method products and ranked first. Data also shows that 61.4% of users are willing to recommend Sogou Input Method to friends, a proportion significantly higher than other input method software.

In the field of input methods, which requires technological accumulation, direct competition relies on technological leadership and more on product service. The lead of Sogou Input Method, I think, is the result of multiple interactions. If we set aside the accumulation of technology and the continuous R&D innovation of Sogou, I believe it's still interaction that made Sogou Input Method successful. Only in the Internet era could a successful Chinese phonetic input method like Sogou Input Method appear. In the future, Sogou Input Method and its competitors will engage in fierce competition, but undoubtedly, Sogou Input Method has already taken the lead in market competition - Interaction, intelligence, Internetization, cloud comfort, and mobile input may become key words in the competition of input methods in the next few years. The excitement of input methods is just beginning.
