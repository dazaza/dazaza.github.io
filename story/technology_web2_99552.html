UserNameCheck: Check if the username has already been registered.

by hooooooo on 2008-10-29 16:16:41

A username represents an individual's virtual identity on the internet, so in most cases, we prefer to use the same username when registering on different websites. For some people, a username has become as important as a domain name.

UserNameCheck can help users check if a particular username has already been registered. Currently, it can check the username registration status across 68 websites.

UserNameCheck uses Ajax technology to dynamically display query results. Websites where the username is already registered will be marked with a red "taken" label and linked to the user's profile page. Websites where the username is still available will be marked with a green "available" label and linked to the website's registration page.
