PHP is a server-side scripting language used for creating dynamic web pages. Similar to ASP and ColdFusion, users can mix PHP with HTML to write web pages. When a visitor accesses the page, the server processes the PHP commands first, then sends the processed results along with the HTML content to the visitor's browser. However, unlike ASP or ColdFusion, PHP is an open-source program with excellent cross-platform compatibility. Users can run PHP on Windows NT systems and many versions of Unix, and it can be run as either an Apache server built-in module or as a CGI program.

In addition to precisely controlling the display content of web pages, users can also use PHP to send HTTP headers. Users can set cookies, manage user authentication, and redirect visitors to other pages through PHP. PHP has very strong database support and can access almost all popular database systems currently in use. Moreover, PHP can integrate with multiple external libraries to provide users with more practical functions, such as generating PDF files.

Users can directly input PHP command codes into web pages without requiring any special development environment. In web pages, all PHP code is placed between ``. Additionally, users can choose forms like ` `. The PHP engine automatically identifies and processes all code within the PHP delimiters.

The syntax structure of the PHP scripting language is very similar to that of C and Perl languages. Users do not need to declare variables before using them. Creating arrays in PHP is also very simple. PHP also has basic object-oriented components that greatly facilitate users in effectively organizing and encapsulating their own code.

#### C. Arrays

When users create arrays in PHP, they can place array indices (including regular indices or associative indices) inside square brackets. For example:

```php

$fruit[0] = 'banana';

$fruit[1] = 'apple';

$favorites['animal'] = 'tiger';

$favorites['sports'] = 'basketball';

```

If users assign values to an array without specifying the index, PHP will automatically append the object to the end of the array. For example, the following code produces the same result for the `$fruit` array:

```php

$fruit[] = 'banana';

$fruit[] = 'apple';

```

Similarly, in PHP, users can create multi-dimensional arrays as needed. For example:

```php

$people['David']['shirt'] = 'blue';

$people['David']['car'] = 'red';

$people['Adam']['shirt'] = 'white';

$people['Adam']['car'] = 'silver';

```

In PHP, users can also quickly create arrays using the `array()` function. For example:

```php

$fruit = array('banana', 'apple');

$favorites = array('animal' => 'tiger', 'sports' => 'basketball');

```

Or use the `array()` function to create multi-dimensional arrays:

```php

$people = array(

'David' => array('shirt' => 'blue', 'car' => 'red'),

'Adam' => array('shirt' => 'white', 'car' => 'silver')

);

```

Additionally, PHP provides the built-in function `count()` to calculate the number of elements in an array. For example:

```php

$fruit = array('banana', 'apple');

print count($fruit);

```

The displayed result is `2`.
