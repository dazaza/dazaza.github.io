Vestwell 为其基于 API 的技术筹集了 7000 万美元 C 系列，该技术帮助雇主，尤其是小企业，管理员工退休计划

2021-07-23 14:51:05

Vestwell 是一家总部位于纽约的用于工作场所储蓄和投资计划的 API 技术平台，在 C 系列融资中筹集了 7000 万美元。 Wells Fargo Strategic Capital 和 Fin Venture Capital 共同领投，高盛、摩根士丹利、宏利、Point72、Nationwide Ventures、Allianz Life Ventures、Northwestern Mutual、FinTech Collective、Greenspring Associates、Primary Venture Partners、Teamworthy Ventures 跟投、F-Prime Capital、Industry Ventures 和 Commerce Ventures。该公司打算利用这笔资金进一步扩大团队、平台功能，并为其数千名合作伙伴和客户提供服务。在创始人兼首席执行官 Aaron Schumm 的领导下，Vestwell 为工作场所储蓄和投资计划（例如 401(k) 和 403(b) 退休计划）提供了一个基于云的数字记录保存平台。该公司基于 API 的技术旨在支持金融服务和金融科技公司的多元化生态系统，从注册投资顾问和薪资提供商到银行、经纪交易商、信用合作社、保险公司、资产管理公司等。通过这些合作伙伴关系，Vestwell 使高质量的工作场所储蓄计划变得可及且负担得起，特别是在服务不足的小型企业市场。 Vestwell 作为一家小型计划提供商，为包括 Morgan Stanley 和 Voya 在内的数千家家喻户晓的计划提供支持，同时与 Toast、QuickBooks Payroll、Gusto、Rippling 和 Paylocity 等薪资提供商进行整合和合作。
