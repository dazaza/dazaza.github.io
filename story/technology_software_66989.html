Sogou Pinyin Input Method 3.3 official version is released!

by yinyin on 2008-05-05 18:22:59

The official version 3.3, the 15th edition of Sogou Pinyin Input Method is finally online. Thanks to all for your support of Sogou Input Method!

We have added many new functions in Version 3.3, mainly including:

Website Prompt: Users can be prompted with websites related to the Chinese characters they input. For example, if you type "Sohu", it will prompt the website www.sohu.com. The website prompt function not only helps you quickly input websites but also provides prompts and guidance for some unclear websites; Mouse word deletion and submission of wrong words and new words: This maximizes user participation in vocabulary library construction. Users can report and delete wrong words and new words in the vocabulary library;

Personalized Settings Wizard: This function can guide users to set up a few commonly used functions and personalized configurations during installation, making a public input method exclusively yours, showcasing your personality;

Smart Help System: By analyzing your personal input habits, it intelligently displays quick help related to the functions of the input method, such as stroke input, v-mode input, etc., providing you with timely tips and assistance;

Contextual Word Frequency Adjustment: After the Chinese characters are displayed on the screen, contextual word frequency can still be adjusted, further improving the accuracy of input;

Random Skin Change Function: Users just need to click on [Random Official Website Skin] in the right-click menu of the input method to change skins and get the most recommended popular skins from the skin website with one click.

In terms of functional improvements, the Version 3.3 input method integrates a skin editor, optimizes part of the keypress process, and optimizes the system vocabulary, intelligent word grouping, and wrong pronunciation prompts.

For bugs reported by users of Version 3.2, we have made repairs, mainly fixing compatibility issues with Adobe cs3 series software, such as Dreamweaver, Flash; at the same time, we have fixed bugs that cause Vista status bar crashes under certain conditions.
