Note to mobile phone users: Microsoft will charge for Windows Live Messenger on the move

by turbo0612 on 2008-01-11 15:31:07

Check it out, switch to Feixin or QQ.

Recently, many mobile Windows Live Messenger users overseas have posted on various communities reflecting that after logging in through a mobile terminal, they were informed that their Windows Live Messenger would enter a 30-day trial phase from that day forward. After the trial period ends, users can pay to purchase one month of usage rights.

In response, many users expressed doubts because many instant messaging software applications on the market today increase installation volume by not only offering free registration and use but also providing a large number of additional functions to enrich user experience. However, Microsoft's decision to start charging for the mobile version of Windows Live Messenger is truly baffling.

According to the situation reflected by users, I found that the fee structure for the mobile version of Windows Live Messenger varies depending on the network and device. For example, I am using China Mobile's communication network, a Nokia N73, with the Chinese version of Windows Messenger installed, and after logging in, no payment information was prompted.

Additionally, all users currently reporting fee prompts are using devices based on the Symbian mobile operating system. Moreover, other Windows Live client functions have not been affected, such as email synchronization and contact information synchronization.

In response, Microsoft customer service personnel stated that starting this week, Microsoft launched a "direct-to-consumer paid service" for Nokia S60 mobile terminal devices in the UK and Sweden. Microsoft indicated that although it is considering implementing similar measures for the Windows Mobile mobile operating system in the future, it has not yet been implemented. However, Microsoft admitted that due to technical issues, many Windows Mobile users received trial prompt messages. Windows Mobile users should not receive any 30-day trial prompts, and Microsoft is currently working to resolve this issue as soon as possible.
