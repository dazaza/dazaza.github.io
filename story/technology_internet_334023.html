Microsoft IE9 interface preview, and it will definitely not support XP

Sina Technology News, early on March 17th Beijing time, according to foreign media reports, Microsoft began offering the download of the first developer preview version of the IE9 browser on March 16 (Sina download address).

The IE9 platform preview does not include the IE9 user interface; it only provides basic elements of the browser, especially Microsoft's latest JavaScript engine codenamed Chakra, as well as a homepage with a list of test websites. This version of IE9 does not have buttons such as "back," nor does it have any security features. It only has the basic IE9 rendering engine and early developer tools.

Microsoft officials will showcase the IE9 developer preview at the Mix10 keynote speech on Tuesday morning and discuss Microsoft's future plans for supporting standards like HTML5, CSS3, and SVG2. Last week, Dean Hachamovitch, General Manager of the Microsoft IE team, said: "We love HTML5, and we want this technology to be used."

Microsoft officials stated that as Microsoft increasingly supports standards such as HTML5, CSS3, and SVG2, Microsoft's ACID3 score will increase. At the "Professional Developers Conference" in November last year, Microsoft officials showcased an early version of IE9, which scored 32 on ACID3. The currently downloadable version has already increased its score to 55.

Before providing the IE9 beta version for download, Microsoft will release multiple preview versions. The Microsoft IE team promised to release updates every eight weeks and interact with web developers through the "Microsoft Connect" feedback system. Microsoft has not disclosed when the beta version of IE9 will be released. Industry insiders speculate that Microsoft will launch the IE9 beta before the release of Windows 8, which may come out in 2011.

Microsoft's goal for IE9 is to make the browser faster, at least as compatible as IE8, and help web developers use the same development tools across IE9 and other non-Microsoft browsers. (By Li Hui)
