Microsoft to release Windows 8.1 for download on October 17, cheapest version to cost 988 yuan

by anonymous on 2013-10-01 21:36:29

On September 23, Microsoft China announced that starting from 7:00 PM Beijing time on October 17, Chinese users will be able to download Windows 8.1 through the Windows Store. The regular edition of Windows 8.1 will be priced at RMB 988, while the Windows 8.1 Professional Edition will cost RMB 1988.

Microsoft stated that in the Chinese market, consumers might confuse the various channels offering Windows products and license types. After communicating with many consumers, since the launch of Windows 8, Microsoft has simplified the distribution model for Windows in China, no longer providing physical media versions of Windows products.

Currently, there are two ways to obtain Windows 8.1: purchasing a new device pre-installed with Windows 8.1 by computer manufacturers, or downloading Windows 8.1 from the Microsoft Windows Store via electronic software download.

A notable change in Windows 8.1 is that Microsoft will provide "full installation software" during the download process, meaning it does not require an older version of Windows to be installed on the device before reinstallation. In contrast, the currently available copies of Windows 8 on the market are "upgrade versions," which must be installed on eligible older versions of Windows.

Microsoft revealed that if you purchase a device with the Windows 8.1 system later this year, you will be able to buy the Windows 8.1 Professional Upgrade Pack for RMB 1088, which includes all features provided in the Professional Edition as well as the Windows Media Center. The Windows 8.1 Professional Edition, however, allows you to purchase the Windows Media Center for only RMB 88.

It was reported that in May 2012, Microsoft announced a Windows 8 promotional upgrade plan: consumers who purchased computers pre-installed with genuine Windows 7 between June 2, 2012, and January 31, 2013, could upgrade to Windows 8 Professional Edition for just RMB 98. After February 1, 2013, the price of Windows 8 returned to RMB 988 for the regular edition and RMB 1988 for the professional edition.
