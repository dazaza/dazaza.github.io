About how to install and run the Android system on a Windows Mobile phone

by dream2012 on 2012-02-24 13:26:40

Since its launch in 2008, the Android system has become the world's third largest smartphone platform in just two short years. The competition among smartphone platforms is becoming increasingly fierce, and both users and the market are focusing their attention on iOS and Android smartphones. Windows Mobile phones, which once played a significant role in the smartphone arena, are now rarely mentioned. Even Microsoft itself is going all out to prepare for the launch of the Windows Phone 7 system, and it is only a matter of time before they abandon the Windows Mobile platform.

If you have an idle Windows Mobile phone or if you're tired of the aging Windows Mobile system, you might want to try the long-desired Android system on your phone. No problem, developers from the XDA forum have provided a very simple and reliable method: by using the Xdandroid project, just follow the steps below, and you can transform your Windows Mobile phone into a vibrant Android phone.

1. Pre-installation notes:

Currently, this method only supports the following HTC-branded Windows Mobile phones:

- HTC HD (Blackstone)

- HTC Touch Diamond (GSM and CDMA versions)

- HTC Touch Pro (GSM and CDMA versions)

- HTC Diamond 2

- HTC Touch Pro2

The Android system will be installed on the SD card of the WM phone, without affecting the internal storage of the phone. Therefore, there's no need to worry about losing your system data or bricking your phone.

The Android system installed and run via this method may not be as perfect as other native Android phones; some functions such as Bluetooth and camera may not work properly. However, the XDA forum continues to release new versions of Xdandroid, supporting most of the features and characteristics of the Android system. Currently, the latest version supported by Xdandroid is the Android 2.1 Eclair system.

2. Download related files:

Download the HTC Xdandroid related files from the following address: http://htcandroid.xland.cz/XDANDROID.21.6.10.AOSP.7z. You can also download other editions of Xdandroid for HTC WM phones from this address: http://htcandroid.xland.cz/

Place the downloaded Xdandroid file on the SD card of your Windows Mobile phone.

3. Xdandroid installation steps:

Use the file browser on your Windows Mobile phone to find the installation file you just placed on the SD card, double-click the file to start the installation.

The system will prompt you for the installation location; please choose to install it on the SD card rather than the internal storage.

After the installation is complete, go to the program menu on your Windows Mobile phone, find the Android icon, click to launch the Android system. The system will create the necessary file system for Android settings and application storage, unpack the system kernel modules, configure default settings, and calibrate the screen, so it will take about 5 minutes to complete the startup process. However, the next time you start it, you can quickly enter the Android system.

After the first Android system startup, it will prompt you to perform initial setup according to the wizard. Follow the steps, and it will be easy to set up. Then, you can add a Google account to use Android Market and Google Sync services, and then you can start your Android journey. Congratulations!

That's right, it's that simple. Now you can parade around with your originally intended-to-be-discarded Windows Mobile phone pretending it's a brand-new Android phone.
