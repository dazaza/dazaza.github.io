据称Windows XP源代码是在网上泄露的

2020-09-25 13:42:52

据称，Windows XP SP1和其他版本的操作系统的源代码今天在网上泄露。

一位声称在过去两个月里花了两个月的时间编译泄露的Windows源代码的人今天发布了他们整个收藏的43 GB的Torrent。

这条洪流中包含所谓的Windows XP和Windows Server 2003源代码，以及该操作系统更旧版本的分类。

Torrent还包括一个媒体文件夹，其中包含关于比尔·盖茨的阴谋论视频的离奇集合。

除了Torrent，一个更小的2.9 GB 7zip文件也在网上发布，其中只包含Windows XP和Windows Server2003的源代码。

泄密者表示，Windows XP源代码多年来一直在黑客之间私下传递，但今天首次公开发布。

Bleepingcomputer已经看到源代码，但我们无法确认它是否是Windows XP或Windows Server 2003的实际源代码。

我们已联系微软确认泄密消息来源是否合法，但尚未收到回复。

虽然Windows XP是在近20年前发布的，但如果现代版本的Windows中仍在使用任何代码，则此泄漏可能是一个潜在的安全风险。

源代码是人类可读的指令，它告诉计算机程序应该如何运行。然后将该源代码编译成可由计算机执行的二进制可执行格式。

对Windows进行反向工程以查找错误总是可能的，但有了源代码，就更容易详细了解Windows操作系统的内部工作原理。

如果在Windows XP源代码中发现可攻击的错误，并且该代码仍在Windows 10中使用，则威胁制造者可以在现代版本的操作系统中攻击该错误。

然而，对于大多数人来说，这次泄露只会让他们得以窥探Windows的历史。
