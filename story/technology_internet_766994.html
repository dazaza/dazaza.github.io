Xiaomi removes 360 products: see the logic behind 360's fight

by anonymous on 2013-10-02 11:05:36

On the morning of September 27, which was yesterday, Sina Tech reported that Xiaomi App Store had completely delisted all 360 products. The reason was that a large number of netizens reported that the 360 Mobile Assistant suggested users uninstall apps such as Xiaomi App Store and Baidu Maps without giving any reason. Below is a screenshot of the news. Shockingly, 82% of users chose to uninstall the Xiaomi Store app!

Regarding this incident, my first reaction was: why did Xiaomi, which has always been so compliant, offend someone again? But after some thought, I realized that Xiaomi didn't offend anyone. However, its "MIUI" ROM is the entry point of all entry points. Xiaomi's "MIUI system" remains the most popular Android ROM in China. This ROM not only dominates Xiaomi phones but also a large number of other Android phones. A ROM is not simply about doing security on top of the system; it replaces the entire system. From a design principle perspective, apps can only call underlying API interfaces. Therefore, security apps can never bypass the system to do things at the bottom layer. This is the boundary that the "general security" mentioned by Old Zhou is bound to fail to cross. Of course, 360 could also create a ROM, but the cost would be too high, and the chances of success would be extremely small. 360’s failure with special supply machines shows how difficult this path is.

Friends who have used the stable version of "MIUI V5" must know that the "MIUI V5" system has already excellently completed everything that security software can do: clearing memory, blocking blacklisted SMS, blocking blacklisted calls, cloud-sharing blacklists, controlling 3G data usage, controlling app privacy permissions, controlling app SMS-sending permissions, controlling app calling permissions, controlling app internet access permissions, cloud storage for photos and contacts, anti-theft protection, software management, power management... What does this mean? It means that MIUI has already done everything that security software can do, making any security software unnecessary! "MIUI V5" has already taken the bread and butter of security apps from the bottom ROM!

Moreover, the stable version of "MIUI V5" made an absolutely brilliant decision: closing ROOT permissions. As previously explained, since "MIUI V5" has already done everything that security software can do, and security software needs ROOT permissions to exercise phone security management, if security software cannot obtain ROOT permissions, it is like a clever housewife having no rice to cook. Even with excellent skills, there is no place for them to use their abilities. In my personal experience, after using "MIUI V5", my user habits have changed, and I haven’t used any security software anymore. I believe this is the purpose of MIUI V5: improving the production of the underlying ROM to replace all security apps.

The ROM is the first entry point for security. The competition between security apps and the underlying ROM for entry points is destined to end with the failure of security apps. At least on MIUI ROM, it can already be declared that not only 360, but all security apps are dead!

Speaking of this, don't forget one thing. What does 360 mainly do? Security! If 360 Antivirus, this core app, is no longer needed on the phone, it is equivalent to uprooting it. So this time, 360 started targeting Xiaomi's distribution platform, "Xiaomi App Market." The so-called "uninstall pre-installed software" helping users eliminate "pesky ads" is just playing the moral card. How many truly pre-installed apps are there? Users will naturally delete apps they dislike. The real purpose of 360 is to eliminate competitors. Although Xiaomi has completely dominated the advantages on the mobile side, 360 has the advantage on the PC side, counterattacking from the PC side to the mobile side. But this move may not stand firm morally.

We saw when 360 fought with Sogou, Sogou itself also had bundled behavior with users. Everyone was ruthless, dirty tactics were used during the promotion of browsers and search engines, unclear and unexplained, mutual mudslinging was fine. But this time, the ROM is different. Xiaomi's "MIUI" ROM is deeply loved by users and is installed on phones. Moreover, everyone has a consensus when brushing ROMs: the ROM's built-in app store is not considered bundling, and the app store is the most basic necessity. "MIUI" ROM is transparent and uses no dirty tactics, so splashing such mud on it will probably not convince the public.

Let's review 360's battle history:

- Doing search, fighting with Baidu and Sogou.

- Making special supply phones, fighting with Xiaomi.

- Making browsers, fighting with Sogou.

- Doing mobile security, fighting with Xiaomi.

- Making mobile browsers, fighting with Tencent and Baidu.

- Doing distribution, fighting with Baidu and Xiaomi.

We see the characteristics of 360's fights:

First, to gain attention, 360 is always proactive. When it causes a siege, 360's goal is achieved. 360 can then play the role of a victim under attack to gain sympathy.

Second, "interests". Every fight 360 likes to talk about user interests, but in fact, it is for its own interests. Talking about protecting user interests is very superficial; user interests are actually just 360's shield. And user interests are naturally objects of harm.

Third, weak. 360 always plays the image of a weak challenger to giants to gain media sympathy, but in fact, 360 has always been a monopolistic barbarian. Through the channel of 360 Antivirus, it bundles huge traffic and is ruthless towards opponents.

Uninstalling pre-installed software is indeed a good thing. What 360 should do is help users uninstall the software they want to uninstall, rather than raising the banner of righteousness again to eliminate opponents. Using percentages to guide users to uninstall apps is unnecessary. Mobile apps are not PC software; this is a place that emphasizes user experience more. Many apps are legitimate products from large companies that want to stay on users' phones long-term. No app dares to easily misbehave on the limited space of a mobile phone. So, please, 360, put away such designs, return the choice to the users, and let competition be fairer.
