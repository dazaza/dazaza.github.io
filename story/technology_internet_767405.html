Windows Phone Blue: Microsoft's Next-Gen Tablet System?

by anonymous on 2013-10-15 21:00:25

Although Microsoft has not yet released the latest Windows Phone operating system, rumors about it continue to circulate. The biggest focus is whether the next generation of the Windows Phone operating system will support 7 to 10-inch tablets? Recently, there have been rumors that this may indeed become possible.

According to Paul Thurrott, editor at windowsitpro.com and a well-known IT commentator, the next generation of the Windows Phone operating system might be able to support devices with 10-inch screens.

Windows Phone Blue, which ultimately was not officially named Windows Phone 8.1, will be Microsoft's first major upgrade to the Windows Phone operating system since the release of Windows Phone 8 in the fall of 2012. We learned about the existence of Windows Phone Blue months ago. There are reports that Windows Phone Blue will be made available to hardware manufacturers in the spring of 2014.

Thurrott also stated that according to an unnamed source, the Windows Phone Blue update will include some new UI and underlying design changes. In fact, as early as February this year, we heard that WP Blue might help integrate Windows by bringing Windows 8, Windows RT, and Windows Phone 8 closer together around the NT core, programming interfaces, and user interface. Thurrott's source also mentioned an interesting statistic (though we've never heard Microsoft publicly disclose it): currently, only 33% of Windows Phone 8’s API overlaps with Windows RT. The ideal goal is for WP Blue to achieve a 77% reuse rate when it launches.

According to Thurrott’s source, the ultimate goal is to enable developers to create one app that can run simultaneously on both Windows RT and Windows Phone. This would easily accommodate a unified Windows Store model—something Microsoft executives have privately promised for the next generation of Windows.

Additionally, the source informed Thurrott that Microsoft plans to eliminate the back button on Windows phones. This is something we hadn’t heard before. I’m curious how Microsoft plans to provide backward compatibility for users who, like us, have already purchased Windows Phones with a back button. Also, we hope Microsoft moves the search key to the left side of the phone because its current placement on the right is prone to accidental presses.

However, what interests us most among these rumors is that Windows Phone Blue might expand support to 7 to 10-inch tablets. So far, Microsoft has prohibited OEMs from installing the Windows Phone system on tablets of these sizes. If OEMs want to run Windows on devices of these sizes, they must use either Windows RT or Windows 8. Previously, we reported that the upcoming Windows Phone 8 GDR3 update might support 5 to 6-inch phablets, such as the rumored Nokia "Bandit" Lumia 1520 phablet.

Terry Myerson, Microsoft’s new head of the operating systems division, recently publicly stated, “As smartphones increasingly merge with tablets, we can expect to see more ARM-based tablets, including Windows ARM tablets.”

I think this means that Microsoft might consider abandoning Windows RT in favor of using a unified operating system across all product lines. The final name could be either Windows Phone OS or simply Windows, which is an interesting question. Of course, whatever it's called, we can be sure that the new operating system will run on both ARM-based phones and ARM-based tablets.
