Tool setting operation on a large CNC lathe for CNC machining

by wanjuncnc1 on 2012-03-09 16:50:13

Operation of tool setting on large CNC machining CNC lathe Large CNC machining CNC lathe is also called CNC lathe, namely computer numerical control lathe, which is the most used and widely covered type of CNC machine tool in China at present, accounting for about 25% of the total number of CNC machine tools. CNC machine tools are mechatronic products integrating mechanical, electrical, hydraulic, pneumatic, microelectronics and information technologies. It is a working mother machine in mechanical manufacturing equipment with advantages of high precision, high efficiency, high automation and high flexibility. There are three methods of tool setting on CNC lathes: trial cutting tool setting method, mechanical tool setting method and optical tool setting method.

1. Tool setting operation of reference tool: a) use the outer circle turning tool to cut the end face of the workpiece, and input the Z coordinate of the tool tip position into the CNC system. b) use the outer circle turning tool to cut the outer circle of the workpiece, measure the diameter of the outer circle of the workpiece, and input the measured value of the diameter of the outer circle of the workpiece into the CNC system, i.e., the X coordinate of the tool tip position.

2. Trial cutting tool setting method Assuming that the tool post changes to a cutting tool at the position of the outer circle tool, although the tool post does not move and the coordinate position of the tool does not change, the tips of the two tools are not in the same position. If the tip position error generated after changing the tool is not eliminated, it will inevitably cause cutting processing errors after changing the tool.

3. General tool setting operation Use the tip of the cutting tool to aim at the intersection of the workpiece end face and side generatrix, and input the Z coordinate and X coordinate of the cutting tool tip position into the CNC system. In this way, the CNC system records different coordinate values of the two tool tips in the same position, calculates the deviation of the tip positions of the general tool and the reference tool after changing the tool, and eliminates the tip position deviation after changing the tool through the tool position deviation compensation of the CNC system.
