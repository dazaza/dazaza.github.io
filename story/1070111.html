Shopify 允许商家直接通过他们的店面销售 NFT

2021-07-28 01:10:27

周一，NBA 的芝加哥公牛队通过在 Shopify 上开设了一家在线商店，推出了其首个 NFT——包括 NBA 总冠军戒指的数字艺术作品。公牛队的球迷现在可以使用信用卡或借记卡直接在球队的在线商店购买数字艺术品，而不必去 NFT 市场。 Shopify 的商家服务副总裁 Kaz Nejatian 表示，在推出 NFT 的第一天，NBA 球队在短短 90 秒内就售罄了 NFT。 “你以前可以用信用卡购买 NFT，但老实说，现在 Shopify 之外的 NFT 购买体验对任何人来说都不是很好，”他告诉 TechCrunch，“这就是我们决定做这项工作的原因。商家和买家不应该参加加密课程来购买他们关心的东西。”这也是为了让消费者有更多的选择来购买 NFT——尤其是那些不精通加密货币的人。该公司表示，通过让商家可以直接通过他们的 Shopify 店面销售 NFT，它为想要销售 NFT 的商家创造了机会。 Nejatian 说，由于 Shopify 支持多个区块链，他们最终将能够根据他们的产品和客户群选择他们想要销售的区块链。 “相比之下，如果商家想在 NFT 市场上销售，他们需要根据该市场支持的区块链进行选择，”他补充道。例如，芝加哥公牛队为其 NFT 选择了 Flow 区块链。但总的来说，Shopify 商家今天可以从 Flow 和 Ethereum 中进行选择，但很快“Shopify 上的其他区块链将有更多选择，”Nejatian 表示。此举还受到商家要求销售 NFT 的需求以及为创作者和艺术家提供另一个专业发展平台的愿望所推动。

“许多创作者已经看到向他们的粉丝出售 NFT 的价值，但我们正在为他们自己和他们的买家消除一些摩擦，让他们能够更好地通过他们的作品和他们与观众的联系获利，”他补充道。 “我们正在开辟一个世界，让他们的粉丝感受到与他们品牌的有意义的联系，并且 NFT 越来越成为我们在线购买和销售方式的一部分。”
