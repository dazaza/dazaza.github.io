The Firefox extension that changes the browser ID is User Agent Switcher

by fisio on 2007-06-26 23:48:13

Some web designers will prompt you to "please use X browser for best results", and even some web pages can be completely different under different browsers; Other blogs have been nice enough to hide Firefox promotion ads from Firefox users - all because the browser sends its own identity and operating system information to the site. User Agent Switcher, used to disguise the identity of the browser and operating system.
