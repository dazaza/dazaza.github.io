@115 Cloud Storage: The latest creation from 115 Cloud Storage, #115AttachmentPro#, will undergo limited testing. This tool has been developed for webmasters to manage website attachments. Industry Dynamics:

@Sync Control: [Alibaba Cloud to launch new version of YunLiao] According to insiders from Alibaba Cloud, the new version of "YunLiao," which supports video calls under WiFi conditions, is currently under development.
