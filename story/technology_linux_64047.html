Ubuntu 8.04 LTS RC Edition ISO Download

by xloong on 2008-04-19 17:56:23

Nightfire: There are only 5 days left until the official release of Ubuntu 8.04 LTS, and the scheduled Ubuntu 8.04 LTS RC was finally released after a 1-day delay. If you don't want to re-download and install, you can refer to the Ubuntu 8.04 LTS RC Upgrade Manual. And most importantly, you can now request a free copy of Ubuntu 8.04 LTS. As a desktop version of Linux, Ubuntu is simply amazing - whether it's the interface or functionality, both are exceptionally outstanding.

The originally planned release of Ubuntu 8.04 RC on the 17th of April had to be postponed by one day due to some small issues during the build process.

All versions, including Kubuntu (KDE3/4), Xubuntu, Edubuntu, and the server edition, are now available. Friends who wish to perform a fresh installation of Ubuntu can now download the ISO images for installation.

As the second Long Term Support (LTS) version in the Ubuntu series, the community and developers have put in a lot of effort.

The continuously improving Ubuntu desktop environment not only enhances hardware support (with a rewritten "Hardware Drivers" tool), but the newly introduced PulseAudio also boosts multimedia capabilities (volume adjustments are made per application). Additionally, the inclusion of Firefox 3, integration of Compiz Fusion 0.7.4, combined with the latest GNOME 2.22, brings an unprecedented Ubuntu desktop experience.

On the server side, Ubuntu continues to progress towards becoming increasingly professional. Besides adopting KVM virtualization technology, Ubuntu has also enabled AppArmor for more applications, improving protection against kernel memory attacks. These two measures significantly enhance Ubuntu's security.

Main new features:

* GNOME 2.22.1

* Linux kernel 2.6.24

* Xorg 7.3

* Firefox 3 Beta 5

* KVM virtualization technology.

* Inkscape 0.46, enabling native PDF support in Ubuntu.

* Added applets such as world clock functionality.

* Replaced Serpentine with Brasero CD/DVD burning software.

* Introduced ufw, an easier-to-configure firewall program.

* Enhanced memory protection against rootkits and other malicious attack codes.
