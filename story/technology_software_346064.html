New product features

Nearly every component and technology in Adobe CS5 introduces a number of new technologies and features, which are briefly summarized as follows:

1. Photoshop CS5 Extened introduces improved Edge detection technology "Truer Edge" (more realistic edge), which can get better results in a shorter time, as well as "Content-aware Fill" (Content Aware Fill), The ability to remove an element from an image and instantly intelligently fill in the pixels as they should be.

2, Photoshop CS5, Premiere Pro CS5, After Effects CS5 native support 64-bit technology, can more smoothly handle high-resolution objects.

3. Flash Professional CS5 adds a new "Text Layout Framework" to provide professional-grade typesetting and printing effects, such as close-lining, linking, spacing, line spacing, multiple columns, and so on.

4. Illustrator CS5 adds a new stroke option that allows you to create strokes of variable width and precisely adjust the stroke width anywhere.

5. The Adobe Mercury Playback Engine in Premiere Pro CS5 supports NVIDIA GPU hardware acceleration to open objects faster, adjust HD sequences in real time, and play complex projects without rendering.

6, After Effects CS5 added a new "Roto Brush" (rotating brush), which can remove foreground elements in a very short time.

7, Dreamweaver CS5 supports popular content management systems, such as Drupal, Joomla! WordPress provides an accurate and dynamic view of Web content within the application.
