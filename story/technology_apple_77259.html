Preliminary Analysis of the Features of the iPhone 3G

The iPhone 3G may seem similar to the first generation iPhone: surfing the web with Safari, making calls, taking notes, iPod, YouTube, widgets, watching videos, including the highly popular Multi-Touch technology; there are no significant differences. If you haven’t used a first-generation grey-market iPhone, we will again introduce its pros and cons.

Dialing Interface and Caller ID Picture

The iPhone 3G only has one button, so how do you make a call? Actually, there is a phone icon on the main screen, press it and then select the number keypad to make a call. After connecting, the six shortcut keys are very convenient. When receiving a call, a large image will appear allowing you to reject the call. However, if your contacts are synchronized from Outlook to the phone, the caller ID picture will be small. It is recommended to change the image directly on the iPhone 3G for a proper display of the caller ID picture.
