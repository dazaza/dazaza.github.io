Ubuntu 21.04就在这里

2021-04-23 15:26:08

2021年4月22日：今天，Canonical发布的Ubuntu 21.04带有原生Microsoft Active Directory集成，Wayland图形默认情况下，以及扑波应用程序开发SDK。另外，Canonical和Microsoft宣布在Ubuntu上对Microsoft SQL Server的性能优化和联合支持。

“Ubuntu上的本机Active Directory集成和认证的Microsoft SQL Server是我们企业客户的首要任务。” Caronical首席执行官Mark Shuttleworth表示。 “对于开发人员和创新者来说，Ubuntu 21.04提供了Wayland并扑振，以进行更光滑的图形和清洁，美观，设计的跨平台开发。”

Ubuntu从桌面到公共云和微服务的首选企业工程平台，拥有各种公司采用者，在金融，汽车，科技和政府部门。

“规范的桌面解决方案是具有成本效益，易于管理的易于管理，并且简单地与企业系统集成。对于我们100,000名警察，Ubuntu是用户友好的，可靠的，并且在没有额外的第三方解决方案的情况下提供必要的隐私和安全性。 Ubuntu是我们的首选操作系统“法国宪兵国法国宪兵·德姆蒙特 - 上校德里奥尔德·德姆蒙特。

从此发布的企业绩效和可扩展性工作已被推回Ubuntu 20.04.2 LTS，以支持Microsoft SQL Server。数据库管理系统（DBMS）及其命令行界面（CLI）现在可以在Azure上的优化Ubuntu图像上提供，提供具有十年安全维护的生产级，高可用的数据库平台。

Canonical和Microsoft将为Ubuntu与Microsoft SQL Server部署的Ubuntu进行集成支持，用于关键任务工作负载。

联合解决方案具有性能增强功能，如XFS文件系统上的强制单元访问（FUA），用于数据耐用性。持久性存储器（PMEM）提供高性能数据存储，而无需任何额外的配置。最后，整个平台高度可用，由Corosync和Pacemaker支持，以确保弹性。

Ubuntu计算机可以在安装中加入Active Directory（AD）域以获取中央配置。广告管理员现在可以管理Ubuntu工作站，这简化了公司政策的符合性。

Ubuntu 21.04增加了从AD域控制器配置系统设置的能力。使用组策略客户端，系统管理员可以在所有连接的客户端上指定安全策略，例如密码策略和用户访问控制以及桌面环境设置，例如登录屏幕，背景和喜爱的应用程序。

Ubuntu 21.04默认使用HAYLAND，安全中断的大幅前进。 Firefox，Obs工作室和许多用电子建造的应用程序，并自动地利用WAYLAND，用于更顺畅的图形和更好的分数缩放。

浮动SDK Snap Build Integration可以轻松发布您的多平台浮动应用程序，以便按数百万Linux桌面用户进行单击安装。

“规范是扑腾的辉煌贡献者，使Linux桌面支持在扑腾和开放机会，为Linux带来高质量的颤动应用程序。我们期待着我们持续的伙伴关系在使App开发人员的最佳选择，无论他们的目标是哪个平台，“共享Chris卖，产品经理，扑振。

21.04包括重新设计的黑暗主题，雅鲁，导航和新文件图标中的可访问性改进。 Yaru由社区维护，由Canonical支持。
