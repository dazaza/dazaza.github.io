亚马逊计划到2025年通过免费的云计算培训帮助2900万人提高其技术技能

2020-12-12 05:10:29

我们将通过现有的AWS设计程序提供培训机会，并开发新课程以满足各种日程安排和学习目标。

多年以来，我一直在与企业和政府领导人讨论如何共同努力缩小技术技能差距。在这些对话中，我们都同意普及知识并使所有人（不论其背景，学历或社会地位如何）有机会建立技术技能的重要性。因此，今天我在re：Invent 2020上宣布，到2025年，亚马逊网络服务（AWS）将通过免费的云计算技能培训帮助全球2900万人提高其技术技能。

在今天的re：Invent会议中，我很荣幸能与世界经济论坛（WEF）的创始人兼执行主席克劳斯·施瓦布（Klaus Schwab）教授一起讨论未来的劳动力以及COVID-19大流行如何影响需求技能。施瓦布教授和我强烈同意，弥合技能差距将需要私营和公共部门的有意识的，持续的努力。对于世界各地的商业和政治领袖而言，世界经济论坛将劳动力培训需求列为首要任务的工作堪称典范，并且在COVID-19大流行之后，这项工作比以往任何时候都更为关键。

为了继续支持未来的员工队伍，我们投入了数亿美元，为200多个国家和地区的各行各业和各种知识的人们提供免费的云计算技能培训。我们将通过现有的AWS设计程序提供培训机会，并开发新课程以满足各种日程安排和学习目标。培训范围从旨在帮助个人更新其技术技能的自定进度的在线课程到强化的提高技能计划，可以在技术行业带来新的工作。

建立我们的图书馆，其中包含500多个免费课程，互动实验室和虚拟的全天培训课程。希望按自己的进度学习云技术的个人可以在AWS培训和认证以及AWS Educate中获得两个强大的资源。这些计划专为寻求基础云计算技能的个人而设计，一直到经验丰富的IT专业人员寻求最新技术，这些计划提供了500多种免费的按需在线课程，互动实验室和虚拟日-长期的培训课程以及基于工作的学习路径，并以多种语言提供了许多免费的讲师指导的网络研讨会。 2020年，AWS培训和认证除了已提供的数百种免费课程外，还推出了50项新的数字课程，并将继续增加更多内容和新的学习方式。

继续投资免费培训以帮助个人获得AWS认证。我们将继续投资免费的数字培训和考试准备课程，以帮助人们为AWS认证做准备，该认证显示了与AWS一起工作的技术专长。培训内容包括新的Twitch直播的AWS Power Hour：Cloud Practitioner系列，内容涵盖云计算基础知识，并为个人准备参加AWS认证的Cloud Practitioner考试。

扩展AWS re / Start计划，以帮助代表性不足的社区的人们过渡到技术工作。该计划是一门为期12周的全日制高影响力课程，旨在为失业或未充分就业的个人做好云计算职业的准备，并将90％以上的毕业生与工作面试机会联系起来。 AWS re / Start在12个国家/地区的25个城市中运营，随着该计划的不断发展，我们预计2021年城市数量将增加一倍。 AWS re / Start通过与澳大利亚土著人拥有的Goanna解决方案，英国的Generation，荷兰的TechGrounds，加拿大的青年就业服务以及Per Scholas的组织合作，建立了各种入门级受过训练的人才的渠道以及美国新学院学院（New College Institute）这个计划已经在积极改变人们的生活。学到更多。

试行新的培训计划。我们正在试行培训计划，例如为期两天的AWS光纤熔接证书计划。光纤电缆是现代世界的骨干，承载互联网，电视和电话数据，它们由细小的玻璃管组成，这使维修和测试它们成为专门的工作。接受了此免费计划的学生将学习如何安装和维修光纤，还可以参加职业交流会，使参加者与当地的雇主见面。另一个计划是亚马逊的机器学习大学，它是一门免费的加速课程，旨在为人们提供学习和应用机器学习概念来解决业务问题的机会。

这只是我们为帮助世界各地的个人学习新技能并开拓新的技术职业机会所做的工作的快照。我们为全球2900万人提供技能培训的计划是基于我们去年作出的投资7亿美元培训10万名亚马逊员工的承诺而建立的。抓住这次培训机会的员工中有Caleb Jarrett。他在美国空军职业生涯结束后加入了该计划。凭借很少的IT知识，他很快就了解了AWS云的基础知识，获得了AWS认证的Cloud Practitioner认证，并通过在Amazon的沉浸式在职培训获得了技术经验。 Jarrett于今年初完成了为期18个月的计划，现在在AWS销售团队中全职工作。这是他对自己的经历的评价：“参与该计划改变了生活。它帮助我在短时间内到达了我想要成为专业人士和个人的地方。从军中出来后，我就有动力为自己的未来做出有意义的改变，而亚马逊则以机会迎接了这一动机。”

Rowenda Sabajo是另一位受益于亚马逊培训以实现新职业机会的人。 Sabajo，去年加入了AWS re / Start Amsterdam队列。在阿姆斯特丹史基浦机场（Schipol Airport）监督地面处理服务15年后，她已准备好转行，并希望过渡到IT部门。通过全职的AWS re / Start计划，她完成了三个月的密集型云培训，包括基于场景的学习，动手实验和课程工作，现在她是荷兰任仕达集团的IT专家。在反思该计划时，她说：“加入AWS re / Start是我做过的最激动人心的事情之一。了解AWS提供的所有高级技术和服务非常吸引人。”

亚马逊专注于建立对我们运营所在的社区具有持久积极影响的创新计划，而设计STEM和技能培训计划对于这种方法至关重要。我们打算继续增加技能培训的途径，为任何想进一步提高云技能的人提供实现这一目标的工具。开始使用并详细了解我们的技能提升工作。
