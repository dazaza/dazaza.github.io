Huawei applies for two patents on anti-cheating technology for online games - Age1983

by hexiaoyao on 2008-08-21 17:19:09

The telecommunications equipment manufacturing giant Huawei was recently exposed to have applied for two online game patents in 2007, involving technology for preventing online game cheats. The industry speculated that Huawei might share the online game market by charging patent fees.

The names of these two online game patent applications are similar: "A method, system, server and terminal device for preventing online game cheats" and "A method, system and device for preventing online game cheat software", but their specific contents are different, involving two key points of preventing cheating programs: identifying cheats and encrypting data communications between client and server.

The main use of the patent application "A method, system and device for preventing online game cheat software": how to determine whether players use cheating programs in games.

This patent quantifies the performance of players in the game in various data forms, calculates a comprehensive weighted average value, and then compares this value with the comprehensive weighted average value of normal game task performance preset by the system. If the contrast is too large, it means that the player uses a cheat, and then interrupts the player using the cheat.

The main use of the second patent "A method, system, server and terminal device for preventing online game cheats" is to encrypt the transmitted data between the client and the server at different levels, so as to block the analysis of package data by cheat programs at the lowest cost.

The working principle of many cheat programs is to analyze the transmitted data between the client and the server, and then modify it to help players cheat. Therefore, one of the ways to deal with cheats is to encrypt the transmitted data between the client and the server, but this method faces a dilemma. If the encryption algorithm is too complex, it will increase the server operation load, at this time, the operator faces a difficult choice between rising operating costs or poor user experience due to slow system speed. If the encryption program is too simple, it will lead to the packet being easily cracked, which is tantamount to a token gesture.

How to prevent online game data from being cracked during transmission between the server end and the client end with the highest cost-performance ratio, Huawei's patent idea is to classify the transmitted data packets, choose encryption algorithms according to different importance, so as to achieve the purpose of preventing encrypted data from being cracked without affecting operating costs and user experience too much.

According to the investigation, Huawei has not conducted substantive examination on these two patent applications so far, therefore, the technologies involved in these two patents are not patents yet, and using them now does not infringe upon Huawei's rights. But if Huawei applies for review and passes, it may sell to online game operators and thereby share the big cake of online games.
