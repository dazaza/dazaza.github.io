A Detailed Guide to Updating the Chinese Version of Vista to SP2 {Repost}

The method of modifying the registry to update SP2 that CB provided yesterday only applies to English versions of the system. Unfortunately, users with the Chinese version of Vista could only sigh in frustration.

Today, a viable SP2 update method has appeared on Yuanjing, and I have successfully updated my system. Below, I will share this process with everyone.

Note: My system is the so-called 28-in-1 OEM Flagship Edition cracked Vista SP1 system.

1. Run the update.bat script file (already provided as an attachment).

2. Install the English language pack! (No need to change the default system language.) Install the KB955430 patch (after installing this patch, you should be able to see SP2).

3. Modify the default system installation language in the registry (please back up the original registry item before modification for restoration later; the attachment already provides it).

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Nls\Language]

"Default"="0409"

"InstallLanguage"="0409"

4. Delete the following registry items (remember to export and save them beforehand, and import them again after all processes are completed; the attachment already provides them).

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\MUI\UILanguages\zh-CN] Remove the zh-CN item.

5. Search for the latest updates again, and SP2 will finally appear (this search process may take slightly longer, so please be patient).

6. Install the SP2 update, which is approximately 290MB and installed in three stages. Before restarting at the end of the installation, revert the registry values in steps 3 and 4 (run the two .reg files).

7. Restart the computer, and the installation is complete! You may see the words "Evaluation Copy" in the lower-right corner of the desktop (it's not guaranteed to appear; it didn't appear on mine).

By following these steps, you can successfully update your Chinese version of Vista to SP2.
