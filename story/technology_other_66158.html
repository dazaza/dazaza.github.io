如何制作 Windows XP 的 USB 启动盘

by @comsharp on 2008-04-30 09:13:06

TechRepublic 的 Bill Detwiler 制作了一个视频教程，关于如何制作 Windows XP 下的 USB 启动盘，你需要一个可启动的U盘，你的电脑的 BIOS 要支持 USB 启动，同时，你需要下载一个叫做 PE Builder 的免费程序，最后，我们需要 Windows Server 2003 Service Pack 1 以便从中取两个文件，当然，你还需要拥有 Windows XP 的安装光盘。
