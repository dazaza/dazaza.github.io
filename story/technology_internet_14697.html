ToToLook V 0.2.0.5 new version released, supports real-time screenshot of playback画面!

by dajiaoban on 2007-03-13 23:10:01

ToToLook is a free software that supports BT video download and playback simultaneously, providing the latest popular American TV series such as Prison Break and 24 for you to watch while downloading. The new version V 0.2.0.5 allows users to take real-time screenshots of the current scene during movie playback! This way, users can capture their favorite movie scenes at any time!

---------------------------------------------------------------------------------------

Here are the updates in the new version:

- Added: Real-time video playback screenshot function (the screenshot will be automatically saved on the user's desktop, screenshot shortcut key: Ctl+P);

- Added: Support for multi-language operating systems; just install the Chinese font library to run ToToLook smoothly;

- Improved: Optimized client download/upload transmission efficiency;

- Improved: Optimized resource search (ensuring the latest search results);

- Improved: Optimized running speed;

- Improved: Task list display method, progress bar improvements, etc.

When playing a video, right-click on the screen, select "Take Screenshot" from the menu, and it will capture the current playback screen and save it on your desktop!
