Essential Skills for Understanding and Using the Android Operating System (Part IV)

by dream2012 on 2012-02-21 17:43:38

The Android operating system might seem complicated and confusing to many novice users who are just getting started. Here, we have compiled some essential tips for understanding and using the Android operating system, hoping that they will be helpful. Please note that these tips are mainly targeted at the most popular Android 2.1 system, but most of them also apply to older or newer Android systems.

31. Quick picture deletion: Every time you delete a picture in the phone's album app, a confirmation box appears. At this point, pressing the Menu key allows you to delete all pictures without confirmation, saving a lot of time.

32. Quick input of special symbols: This is a trick that most people don't notice, but if you are a heavy user of mobile text input, this function will be very useful. When using the official Android on-screen keyboard, long-pressing any character brings up the corresponding special symbol for that character. For example, long-pressing S inputs @, and long-pressing D inputs #. This trick will significantly improve your mobile input speed.

33. More Android emoticons: The above trick also applies to Android emoticon input. Long-pressing one type of Android smiley face will bring up other smiley faces from different systems.

34. What do the emoticons others send me mean? If you're old enough to not understand what different smiley faces represent, you can press the Menu key in the input interface, choose to insert an emoticon, and then see all Android emoticons along with their corresponding explanations.

35. Adjust Android spell-check: If you find the spell-check function inconvenient while using the Android keyboard, you can turn it off in Settings > Language and Keyboard > Android Keyboard.

36. Install an Android file manager: File management is one of the biggest functional gaps in the Android system. Although you can use Android system tools to delete pictures or music, downloading a file management software from the Android Market will make managing files on your Android phone as efficient as on a PC. Currently, the best file management software for the Android system is Astro, which we recommend downloading and installing.

37. Manage your Android battery usage: You may suddenly find that your Android phone's power consumption has sharply increased, yet you're unsure where the problem lies. In fact, the Android system already collects detailed statistics on battery usage. You can view the proportion of battery usage by different Android functional modules in the phone settings menu > About Phone > Battery Usage. This way, you can figure out which program is causing the issue and decide whether to delete it.

38. Access keyboard settings from the Android keyboard: The native Android keyboard input method provides a quick function. Long-pressing the 123 key brings up the Android input method's setting options.

39. Quickly finish typing: During the typing process, double-tapping the space bar will automatically stop typing and even add a period.

40. Clear phone input history: When you want to completely clear the phone data, you often overlook this step. Don't forget to clear all input history in Phone Settings > Language and Keyboard > User Dictionary.
