苹果为开发者发布开源的“Password Manager Resources”项目

2020-06-06 02:42:10

苹果公司今天宣布，它正在启动一个为密码管理器开发者设计的开源项目。目标是让开发人员更容易“创建与流行网站兼容的强密码”。

苹果的iCloud钥匙链平台已经能够在创建账户或更改密码时生成强密码。新的Password Manager Resources开源项目旨在将其扩展到Password Manager应用程序。

Password Manager Resources开源项目允许您集成iCloud Keychain Password Manager使用的特定于网站的要求，以生成强的、唯一的密码。该项目还包含已知的共享登录系统的网站集合，指向用户更改密码的网站页面的链接，等等。

苹果在GitHub上推出了新的密码管理器资源，让所有人都可以使用。该项目背后的团队解释说，这个想法是为了让“密码管理器的创建者可以在资源上进行协作，为用户更好地进行密码管理。”到目前为止，可用的信息由数据或“怪胎”和代码组成。

具有共享凭据后端的网站：已知使用相同凭据后端的网站组，可用于增强登录网站的建议凭据。

更改密码URL：为了推动强密码的采用，能够将用户直接带到网站的更改密码页面是很有用的。

通过共享资源，所有密码管理器都可以通过比任何单个密码管理器获得相同效果所需的工作更少的工作来提高其质量。

通过公开记录特定于网站的行为，密码管理器可以激励网站使用标准或新兴标准来提高它们与密码管理器的兼容性；在列表上被叫出来一点都不好玩！

通过提高密码管理器的质量，我们提高了用户对他们的信任度，这对每个人都是有利的。

你可以在苹果的GitHub页面上查看完整的密码管理器资源集合。
