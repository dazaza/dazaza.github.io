Rising Kaka Internet Security Assistant 5.0 Released

by xloong on 2008-01-22 23:30:51

After turning on my computer today, the Rising CCKK upgrade program popped up. I thought it would be a quick update of just a few files, but later I realized that today's upgrade was particularly slow and there were many files to update. This made me think about the Rising CCKK 5.0 beta version that came out a few days ago. Was today's upgrade moving to the official 5.0 version? After the upgrade was completed, I opened the Rising CCKK Internet Security Assistant and sure enough, it had upgraded to version 5.0 with significant changes in its interface.

However, as soon as I opened Rising CCKK, it automatically started scanning, and the speed wasn't very fast. I'm not sure if this can be turned off; I couldn't find where to set it. Aside from some of the usual functions, there weren't many noticeable improvements. I only use the adware scanner and startup item manager anyway. Although msconfig can also manage startup items, it feels cumbersome and isn't as intuitive as CCKK's feature. Maybe it's just habit. Let's take a look at the new features introduced on the Rising CCKK official website:

New Features:

- One-click Solution: Easily solve three major mainstream security issues - system vulnerabilities, rogue software, and popular trojans.

- Online Diagnosis: Automatically report suspicious files to the Rising Anti-Virus Lab.

- File Clinic: Query online if active system processes are safe.

The easiest operation for the safest results: In CCKK Internet Security Assistant 5.0, we proudly introduce the one-click solution feature, allowing users to easily fix system vulnerabilities, remove malicious rogue software, and scan for popular trojans all with one click.

Usage Method: When you open the main interface of Rising CCKK Internet Security Assistant, after the automatic scan is complete, simply click the "One-click Solution" button.

Popular trojans pose serious threats, especially to the safety of your system, game, and other important accounts. The anti-trojan function in CCKK Internet Security Assistant 5.0 ensures the security of your system account.

Usage Method: When you open the Internet Security Assistant, under the 'Antivirus' tab's functional page, you can use the 'Scan for Popular Trojans' feature. You can choose quick scan, full disk scan, or custom scan.

When you connect to the network, do you feel that some connections are suspicious? The view network connection feature in CCKK Internet Security Assistant 5.0 makes dangerous connections visible. It also provides functions to report and comment on suspicious network connections. (Note: If your comment is the first for a certain file, your comment will be displayed on the CCKK software client for reference by 50 million CCKK Internet Security Assistant users.)

Usage Method: When you open the Internet Security Assistant, under the 'Firewall' tab's functional page, click on 'View Network Connections' to check.

The "Account Safe" is one of the biggest technical highlights of Rising Antivirus Software 2008. It is a new technology application mode derived from the "Active Defense" technology. After installation, without any settings from the user, Rising "Account Safe" will automatically protect more than a hundred popular software programs.

Usage Method: When you open the Internet Security Assistant, under the 'Privacy Protection' tab's functional page, click on 'Account Safe' to learn more.

CCKK Internet Assistant can disable or clean plugins in the system and effectively prevent malicious plugins from being downloaded automatically. This way, you no longer have to endure the troubles brought by plugins.

Usage Method: Open the Internet Security Assistant, under the 'Advanced Functions' tab's functional page, click on 'Plugin Management and Uninstallation'.

CCKK Knows is a network platform that helps users solve security problems. Users can search for solutions to problems here, and Rising experts will resolve difficult questions within it.

Here, users can conveniently ask questions or answer questions and have the chance to become a CCKK Star.

Usage Method: Open the Internet Security Assistant, under the 'Diagnosis Assistance' tab's functional page, click on 'CCKK Knows'.
