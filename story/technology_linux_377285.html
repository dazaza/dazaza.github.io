U盘上的操作系统 Lucid Puppy Linux 5.2 发布

by ownlinux on 2011-01-07 09:43:18

Lucid Puppy Linux 5.2 是 Puppy Linux 的又一个分支，它是基于 Ubuntu 10.04 LTS (Lucid Lynx) 的版本，正是如此 Lucid Puppy Linux 5.2 可以兼容绝大多数 Ubuntu 中的二进制软件包，为 Lucid Puppy Linux 5.2 提供了更加方便的软件安装支持。Lucid Puppy Linux 5.2 是 Lucdi Puppy Linux 5.1.1 的升级版本。

Lucid Puppy Linux 5.2 是一个全功能的小型Linux发行版。它是同前不久U盘上的操作系统 Wary Puppy Linux 5.0 发布中的 Wary Puppy Linux 5.0 同属 Puppy 这个大家庭。虽然它只有几十MB的大小，但它自带了大量的描扫仪、打印机、数码相机的驱动以及音乐录制/编辑/转换软件。由于 Puppy Linux 比较小巧，资源占用相当少，把它用在一些比较古董级电脑上的效率比较高，让老电脑也可以通过 Puppy Linux 来发挥余热，更不用担心自己隐私被收集的问题。
