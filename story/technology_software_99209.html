Common Issues and Solutions for Windows Vista

(Note: The following content is for reference when needed. Some functions have already been set up in the system included on this disc, so do not set them again.)

1. How to turn off UAC?

Control Panel → User Accounts → Turn User Account Control on or off → Uncheck "Use User Account Control (UAC) to help protect your computer."

2. How to turn off system backup and restore?

Control Panel → Backup and Restore Center → Create a restore point or change settings → System Protection → Automatic Restore Points → Uncheck restore points on selected disks.

3. Why does the hard drive go crazy at a fixed time?

This is due to Vista automatically running the disk defragmentation schedule. To turn it off, go through Start Menu → Accessories → System Tools → Disk Defragmenter → Uncheck "Run on a schedule (recommended)."

4. How to prevent the computer from going into hibernation?

Right-click desktop → Personalize → Screen Saver → Change Power Settings → Change when the computer sleeps → Set sleep time to Never.

5. The power button on the Start Menu defaults to "Hibernate," how to change it to "Shut Down"?

Control Panel → Power Options → Selected "Preferred Plan" → Change plan settings → Change advanced power settings. In the popup window, find "Power buttons and lid" → "Start menu power button" → Setting → Shutdown, confirm to complete the setting.

6. Every time I start, the security center prompts me, how to turn off this prompt?

Control Panel → Security Center → Change the way Security Center notifies me → Select Do not notify me but show icon.

7. How to display the Run command in the non-traditional Start Menu?

Taskbar and Start Menu Properties → Start Menu → Customize → Check "Run command."

8. How to cancel default sharing every time I start?

Use Vista Optimization Master, or import the following content into the registry and restart (premise is to turn off UAC).

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesLanmanserverParameters]

AutoShareWks=dword00000000

[HKEY_LOCAL_MACHINESYSTEMCurrentControlSetControlLsa]

restrictanonymous=dword00000001

9. How to achieve automatic login?

Start → Run → Input "rundll32 netplwiz.dll,UsersRunDll" command to open the account window, select the account to automatically log in, deselect "To use this computer, users must enter username and password," input the account's password (premise is to turn off UAC). Using Vista Optimization Master from Vista Home makes it simpler.

10. How to change media files to play with players other than WMP?

Select media file → Right-click menu → Open With → Choose Default Program → Select playback program → Check "Always use the selected program to open this kind of file."

11. Why can't I visit other computers in the local area network?

Approach from the following aspects:

Local Connection → Right-click Properties → Uncheck Internet Protocol Version 6 to shorten authentication time.

Check if the machine's workgroup name is consistent.

Check if the machine's IP is on the same subnet, e.g., 192.168.1.X.

Set Network and Sharing Center → Network Connections → Custom → Location Type to Private, reducing protection.

12. How to visit other computers in the local area network after enabling ICF?

Cancel blocking all incoming connections in Windows Firewall and check "File and Printer Sharing" in "Exceptions."

13. How to log in with the Administrator account?

Computer → Manage → Local Users and Groups → Users → Administrator Account → Properties → Uncheck Disabled option for the current account → Properties → Check Disabled option.

14. When opening Control Panel, it reports an error "Windows Explorer has stopped working," how to fix it?

Hardware driver conflict, possibly related to sound card drivers. Try reinstalling the sound card driver.

15. How to quickly copy file, folder paths?

Press Shift key, right-click the file, folder, or shortcut you want to copy the path of, in the popup menu, there is an option "Copy Path (A)," click it to copy the file or folder path.

16. How to view system license information?

Run slmgr.vbs -dlv.

17. What is the shortcut key for reverse selection of files?

ALT+E pops up the menu, then press I.

18. How to change the size of icons on the desktop?

Ctrl+mouse wheel or right-click the desktop → View → Classic Icons.

19. How to switch programs using 3D?

In the case of Aero appearance enabled, use Win (Window Key)+Tab to switch programs using 3D, Win (Window Key)+Ctrl+Tab keeps the 3D window temporarily on the desktop.

20. How to set up proxy in IE browser?

Open Internet Options in the IE toolbar → Connections → Select your dial-up connection in Dial-up and Virtual Private Network Settings → Set. In the "Settings" page, check "Use a proxy server for this connection" and fill in the proxy server address and port in the "Address" bar, then click "OK."

21. How to view the PID value of a process?

In Task Manager → View → Select Columns.

22. How to clear the history of recently opened documents when exiting the system?

Run gpedit.msc, User Configuration → Administrative Templates → Start Menu and Taskbar → Clear recent document history when exiting the system, set to Enabled.

23. Screen tearing when capturing screenshots

You will often find that Vista displays fine itself, but whether using the Print Screen key or other screenshot tools, part of the resulting image will tear, sometimes as a line, sometimes as a block. This is mainly due to the incomplete graphics driver provided by Vista. The solution is to download and install the latest graphics driver, which usually resolves the issue smoothly.

24. Recovering the disappeared "Run" dialog box

By default, the familiar "Run" command is missing from the bottom of Vista's Start Menu, which can be inconvenient for some advanced operations. You can bring back the "Run" dialog box through two methods:

(1) Use the "Win+R" key combination to open the "Run" dialog box, a trick we've already used in the fourth question.

(2) Right-click on the Start Menu, select "Properties," enter the Start Menu properties settings interface, click the "Customize" button at the bottom, find "Display Run" in the list box and check it, then confirm and exit. You'll find the "Run" command reappears at the bottom of the Start Menu.

25. Unable to Boot

Being unable to boot is a fairly common and severe fault. Previously under XP, users had to manually repair this kind of fault. However, under Vista, you just need to boot the machine from the disc into the recovery console and enter the fixboot command. Vista will automatically detect boot faults and repair them using the BCDMD command. This is a great help for ordinary users and deserves commendation.

26. Why does QQ always disconnect under Vista?

Under Vista, QQ frequently disconnects but quickly returns to normal. This is because the network card is defaulting to energy-saving mode under Vista. If you want it to continue saving energy, you can ignore this phenomenon as it does not affect the normal use of QQ. If you need to handle it, simply remove the checkmark for "Allow the computer to turn off this device to save power" in Device Manager → Network Adapters → Properties → Power Management.
