1 Non-closed linear tree planting problems mainly fall into the following three situations:

⑴ If trees are planted at both ends of a non-closed line, then:

Number of trees = Number of segments + 1 = Total length ÷ Tree spacing + 1

Total length = Tree spacing × (Number of trees - 1)

Tree spacing = Total length ÷ (Number of trees - 1)

⑵ If trees are planted at one end of a non-closed line and not at the other, then:

Number of trees = Number of segments = Total length ÷ Tree spacing

Total length = Tree spacing × Number of trees

Tree spacing = Total length ÷ Number of trees

⑶ If trees are not planted at either end of a non-closed line, then:

Number of trees = Number of segments - 1 = Total length ÷ Tree spacing - 1

Total length = Tree spacing × (Number of trees + 1)

Tree spacing = Total length ÷ (Number of trees + 1)

2 Closed-line tree planting problems have the following numerical relationships:

Number of trees = Number of segments = Total length ÷ Tree spacing

Total length = Tree spacing × Number of trees

Tree spacing = Total length ÷ Number of trees
