Comprehensive Personalized Homepages - The Persistence of Bugs

by zzymale on 2008-09-02 11:05:18

I previously wrote an article titled "Quickly Increase Your Feed Subscribers," which mentioned "personalized homepages." To put it simply, this is a service that embeds a feed into a webpage. There are many personalized homepages available, and here's a summary. The ones with a light green background have been tested and can use the method mentioned above, while the others remain untested.

### 1. Domestic Personalized Homepages

1. **Shiwo**: A new generation of internet portal websites. Unlike traditional portals you often visit, "Shiwo" not only offers news, videos, blogs, forums, BT downloads, novels, original content, software downloads, financial market conditions, and a series of unprecedentedly rich contents, but also allows users to set and define every aspect of their "Shiwo" personalized portal, including page style, color, layout, and combination. Each content module can be set individually, meaning every "Shiwo" user has a unique "Shiwo" portal with all the content they like and need!

2. **GuGu**: Convenient subscription to news, podcasts (Podcast), and other RSS aggregated information. The latest information will be right on your homepage.

3. **YaKe**: Free browsing pages, customization, and management of your own RSS subscriptions; import and export OPML files; free customization and management of blog, podcast, and other Web2.0 content; free browsing, customization, and management of pictures, music, and videos; access to a large number of excellent RSS feeds; direct search for RSS and customize it to your personalized homepage.

4. **HuoChai**: Provides personal portals.

5. **WoYong**: Everyone has their own internet. By customizing your own personal portal page, you can easily obtain various internet information and services in one place, satisfying most internet application needs.

6. **WanDou**: Fully customizable personalized homepage according to your needs, including one-click account login, network bookmarks, notepads, URL navigation, practical tools, etc., and supports multi-page customization.

7. **YaWa**: A personalized aggregation platform based on the Web3.0 concept (another startup playing with concepts ——||). It uses search-customized aggregation modules and innovative WidgetRank aggregation technology and algorithms to help users automatically and effectively analyze, capture, and aggregate any blog, forum, website, etc., RSS addresses, and present them in multiple reading formats according to the user's wishes. You can freely create new modules and pages, drag and drop, edit module content and styles, etc., via mouse. At the same time, YaWa will automatically save the user's personalized network platform and generate a unique second-level domain name.

### 2. Foreign Personalized Homepages

1. **Pageflakes**: An Ajax-based personalized homepage providing personalized and community-oriented personal homepage services. It integrates RSS readers, emails, address books, work logs, searches, Flickr, YouTube, Odeo, etc.

2. **Netvibes**: A third-party personal information portal built using AJAX technology. It supports creating Tab pages and provides many personalized services, making it a great lightweight personal information aggregation platform.

3. **Webwag**: Another personalized homepage system built using Ajax technology. However, it currently does not support Chinese, although its interface is quite aesthetically pleasing.

4. **24eyes**: It resembles a personalized RSS reader with default channels such as Business, Sports, Technology, etc. You can delete or add channels you like, create new pages, and it features good sharing functions.

5. **CosmoPod**: Offers both free and premium accounts. The free version provides 1GB of storage space and offers various online applications like Web Browser, text processing, file transfer (with a single file limit of 2MB).

6. **Eskobo**: A personalized homepage where you can add different feeds and customize background colors. However, it offers relatively fewer personalized functions.

7. **Google Start**: Google's personalized homepage.

8. **Inform**: A personalized homepage similar to an RSS reader. The page layout feels nice, and the navigation bar also feels pretty good.

9. **ItsAStart**: A personalized homepage service offering various widgets, allowing you to import RSS and easily change the background and text background color.

10. **LinkedFeed**: Similar to Pageflakes, it supports creating Tab pages, offers various widgets, and provides personalized community services with over sixteen thousand feeds available for subscription.

11. **Microsoft Live**: Microsoft’s Live personalized homepage.

12. **Protopage**: Using AJAX technology, Protopage is easy to operate. Sticky notes and link panels on the page can be added or deleted at will. The difference is that the latter makes it easier to input links in bulk, while the former allows direct HTML code input after enabling "Treat as HTML," meaning you can embed your original blog.

13. **Start**: MSN's Web2.0 personalized homepage, with a very clean interface.

14. **SuprGlu**: Similar to an RSS aggregator site, it can gather related content from delicious, flickr, blogger, etc.

15. **My Yahoo**: Yahoo's personalized homepage.

16. **Schmedley**: Currently still in the Alpha version, many functions are not yet perfect, such as the customization function being insufficiently strong. However, according to Schmedley's introduction, its customization function will be strengthened in future versions.
