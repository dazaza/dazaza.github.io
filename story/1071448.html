谷歌浏览器不再显示安全网站指示器

2021-08-05 21:32:51

谷歌浏览器将不再显示您访问的网站是否安全，仅在您访问不安全的网站时显示。多年来，谷歌一直在齐心协力推动网站使用HTTPS，以提供更安全的浏览体验。为了进一步推动 Web 开发人员在其网站上仅使用 HTTPS，Google 引入了该协议作为排名因素。那些没有托管安全网站的网站在他们的 Google 搜索结果排名中可能会受到轻微影响。根据谷歌透明度报告的“网络上的 HTTPS 加密”，它似乎已经起作用了，目前谷歌 Chrome 浏览器中超过 90% 的浏览器连接都使用 HTTPS 连接。目前，当您访问安全站点时，谷歌浏览器会显示一个锁定的小图标，表示您与该站点的通信已加密，如下所示。由于大多数网站通信现在都是安全的，因此 Google 正在测试一项新功能，该功能可以移除安全网站的锁定图标。此功能可在 Chrome 93 Beta 和 Chrome 94 Canary 版本中通过启用“Omnibox Updated connection security indicator”标志进行测试。启用此功能后，谷歌浏览器将仅在站点不安全时显示安全指示器，如下所示。

对于希望继续使用 HTTPS 安全指标的企业，谷歌为 Chrome 93 添加了一项名为“LockIconInAddressBarEnabled”的企业策略，可用于再次启用地址栏上的锁定图标。对于那些想要测试禁用 Chrome 安全指示器功能的人，您可以使用这些说明在 Chrome Beta 或 Chrome Canary 中启用它。当显示“多功能框更新连接安全指示器”标志时，单击“默认”并选择“已启用”。 Google 将不再向您显示站点是否安全，并且仅在您访问不安全站点时显示指示符。
