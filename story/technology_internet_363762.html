Non-advertisement post - Recommend the patent search function of Baiteng website

Baiteng website is a non-profit and free patent search site. "5i" stands for "I love", and "patent" stands for patent. After several upgrades, its functions are now stable and convenient. I would like to briefly explain some of the search techniques that I know and share them with you.

1. Interface

After the redesign, 5ipatent has several interfaces as follows:

1) Google-like interface - my favorite interface

Website: http://so.5ipatent.com

This interface is similar to Google, where you can input keywords, patent numbers, applicants, inventors, classification numbers, etc., directly in the search box without needing to use table searches or write logical expressions. This reflects the goal of the website: to make patent searches simpler and more practical.

2) SIPO (State Intellectual Property Office)-like interface - Table Search

Website: http://so.5ipatent.com/AdvancedSearch.aspx

This is a table search interface that basically supports the table search functions of the State Intellectual Property Office. Below the table, there is a detailed introduction to the table search function, which is more comprehensive than that of the State Intellectual Property Office.

The above are the interfaces I commonly use.

2. Functions

1) Search Function

The search function can be described by an internet buzzword: very good, very powerful, hehe. It's the best one I've seen.

As mentioned earlier, different items such as keywords, patent numbers, applicants, inventors, classification numbers, etc., can be used as conditions. Baiteng website adopts some Google habits, so if you're familiar with Google, you'll find 5ipatent easy to use. Here are a few Google-like habits:

A: The total number of search results and the search time are displayed at the top of the search list;

B: There are 10 search results per page;

C: The page numbers are listed below;

D: The search keywords are retained in the search box.

E: The format of the search results.

On the search results page, there is a toolbar on the right side that allows further narrowing down the scope, i.e., secondary filtering.

Special Note: When searching by application number, please pay attention to the check digit, i.e., the digit after the decimal point. 5ipatent currently recognizes searches without the digits after the decimal point or with the decimal point included. If you do not include the decimal point but write out the check digit, it will result in an error. For example, if "a.x" represents an application number, where "a" is the number and "x" is the check digit, you can search using "a.x" or "a", but if you search using "ax", it will result in an error.

2) Browse Function

It realizes the full-text browsing function and ensures the ability to flip pages forward and backward. It also includes zooming, full-screen, dual-screen browsing, etc.

3) Save Function

It realizes the full-text saving function, but the files are single-page files, with each file being stored in a folder. The folder is named in the format "Patent Number-Invention Name-Application/Authorization Text". The files within the folder are individual .tif files, consistent with the naming convention of the State Intellectual Property Office. A new progress bar for saving has been added.

3. Expectations

After using it for a month, I now basically use Baiteng for my searches because it's fast. Moreover, the update time is noted on the webpage, and I feel it is synchronized with the State Intellectual Property Office.

Most importantly, I think it should learn from Google and Baidu and open up a user login function. This way, both old and new functions can be preserved, allowing users to choose. For example, regarding the browse function, I prefer the original function. Regarding the input function, I prefer to save a separate PDF file. Features like the collection function can be developed into more service functions. Of course, some future fee-based functions can be considered, as long as they aren't too expensive.

In short, I think Baiteng's patent search is the best patent search I've encountered. Even Google's patent search isn't as good. As for Baidu, I look down on it in this aspect. In all other aspects, including video search and image search, it competes with Google, but it remains indifferent when it comes to patents. Google only handles US patents, which we can't help, although its other services, including Google Desktop, are excellent except for insufficient support for Chinese.
