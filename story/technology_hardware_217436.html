60 Computer Usage Tips

by cqe on 2009-08-03 17:38:36

1. If you have multiple windows open at the same time and want to close them, you can hold down the shift key and then click on the close icon in the upper right corner of the window.

2. Before saving a webpage, you can press the "ESC" key (or work offline) before saving, which will save it quickly.

3. To listen to a CD on your computer without any playback software, connect the speaker wire directly to the headphone jack on the optical drive, insert the CD, and press the play button on the optical drive to listen directly. Listening this way does not consume system resources. (If your computer is broken and cannot start or something, and cannot be repaired temporarily, do not waste resources. Use the power inside to power the optical drive, and it can temporarily serve as a CD player.)

4. In MSN, when sending messages, press enter. If you want to create a new line without sending the message, press shift+enter or ctrl+enter.

5. You can debug short HTML code in the browser's address bar. Method: Type about:abc in the address bar and press enter, and you will see the effect.

6. Windows shortcuts:

win+m shows the desktop

win+pause system properties

Quick restart: Press shift before pressing OK (not applicable for 2k, xp)

Thorough delete: shift+del

Prevent auto-run of CDs: Press shift

Ctrl+Esc: Equivalent to "Start" or WIN key

Ctrl+Home: Moves the cursor to the beginning of the text editing area (Home alone moves to the beginning of the line)

Ctrl+End: Moves the cursor to the end of the text editing area (End alone moves to the end of the line)

Alt+F4: Closes the current window (if you click on the desktop and press it, it shuts down the computer)

F2: Rename

windows+e Explorer

windows+r Run

windows+f Find

windows+u Shut down system

windows+d Minimize all windows, press Win+D again to return to the minimized windows

windows+m Minimize all windows, but pressing it again will not return to the minimized windows

Shift+F10: Opens the right-click menu of the selected item

Hold CTRL+SHIFT while dragging files: Creates a shortcut

8. Restart shortcut:

(2). Type rundll.exe user.exe,exitwindowsexec

(3). Type or choose your desired name - Press

9. RUN -> cmd or command

You will see the DOS window. Have you noticed that small icon in the upper left corner?

Clicking this icon will show a drop-down menu, especially some functions under Edit, which are very convenient.

10. Con, nul are reserved names in the MS system, they cannot be used as filenames! To check if your web space is on the M$ system, try creating a folder named con.

11. Want to make a list of files in one of your folders? What software should you use? None, just type "tree /f > index.txt" in the command line. Open index.txt and you'll see everything clearly.

12. To save a page with no borders, use CTRL+N to open a new window.

13. If there are many files in a folder and you want to quickly find the file you're looking for, select any file first, then press the first letter of the file you want on the keyboard.

14. A simple method for QQ stealth login:

When surfing the internet in an internet cafe, if you log in directly from the registration wizard, QQ will log in online. However, sometimes you don't want to appear. How to achieve stealth login?

Log in with a useless QQ account first, set the status to stealth, then select stealth login in the system settings, confirm, then run the registration wizard on this QQ to log in to the QQ you want. You'll find that QQ is now stealth mode.

15. Method to change IP in Win98 without restarting:

1. After changing the IP, select cancel.

2. Disable the network card with the changed IP in the device manager.

3. Enable the network card.

Okay, the IP you changed can now be used.

16. In 2000 and XP, running "cmd", entering the DOS window, copying and pasting are both done with the right mouse button.

17. Do not put bookmarks on the system drive to avoid forgetting to back them up. After each system installation, use "Super Magic Rabbit" to specify the path.

18. For parts of the system you don't understand, usually look more at the help that comes with Windows, don't blindly ask others to solve it, maybe help is the fastest and most complete way. (The same applies to other software.)

19. WinXP has a built-in batch renaming function, just select a bunch of files, select rename, then modify the first file, after modification, the other files will also be automatically modified.

20. When entering a URL in the address bar while surfing the internet, the system will record it. Although it's convenient for later use, if it's a public machine and you don't want others to know where you've been, use "CTRL + O". A "Open" dialog box will pop up, enter the URL in the address bar within it, and it won't be recorded.

21. IE shortcuts:

Ctrl+W Close window

F4 Open the dropdown selection of URLs in the address bar

F6 or ALT+D Select the address bar

Spacebar scrolls down a page, Shift+Spacebar scrolls up a page

22. If you use Foxmail and the email path is not in the default location, you can modify account.cfg.

23. On NTFS partitions, there is a "Summary" feature in the file properties, which is very useful, so you won't forget what some software installation programs are!

24. Input javescript:alert(document.lastModified) in the IE address bar to get the update date of the webpage.

25. When using network sharing, try to use "Run", \ip address shared filename, instead of browsing through "My Network Places", it's slow and often incomplete (especially in 2000, xp), and pay attention to the appropriate use of $

26. In Win98, to quickly make a boot disk, just copy all the files in the windows/command/edb directory to a clean floppy disk, and you can make a Win98 boot disk.

27. If PQ partition fails and unrecognized partitions appear, find PTEDIT32.EXE in the PQ installation directory UTILITY, which can restore the original partition to its original format.

In the resource manager, select the folder, use the * on the numeric keypad, and it will quickly list all subfolders existing in this folder.

28. Method for QQ to add itself: Add yourself in the blacklist, then drag yourself to "My Friends" and you're done, then run the registration wizard.

29. If the page you are browsing uses JavaScript to disable the right mouse button, here's how to solve it:

1. As "xiayupei"'s method, hold down the left mouse button first, then move the mouse to the target, click the right mouse button, and the shortcut menu will appear after one or two seconds.

2. Single-click the right mouse button (do not release the right mouse button), move the mouse to the warning window, click the left mouse button, close the window, then move the mouse back to the target, release the right mouse button, and the shortcut menu will appear.

30. RealPlay multi-track playback

Select multiple tracks, then drag them to the playback address bar of RealPlay, then find the ram (which will be generated automatically), copy the content inside multiple times, and you can replay the songs repeatedly.

31. Under Windows, files can have only extensions

Create a new text document and save it as .txt without adding anything before it, and it will work, but you cannot rename it to such a filename! You must use "Save As"!

32. True blank page in IE:

Right-click on the IE shortcut --- Properties --- Select "Target", the information here is: "crogram FilesInterne ExplorerEXPLORER.exe", add " -nohome" after it. Note that there should be a space before -nohome.

Quickly set the homepage: Drag the "e" webpage icon in the IE address bar directly onto the "Home" button icon on the toolbar.

Refresh the registry without restarting: Press Ctrl+Alt+Del, highlight "Explore" in the popped-up Windows task list, single-click "End Task", the shutdown screen appears, click "No", wait a moment, an error message pops up, click "End Task", and the Windows Explorer will reload along with the new registry!

33. Quick start technique for RealOnePlayer: Search for netid.smi and getmedia.ini in the Real folder, rename them to betid_bak.smi and getmedia_bak.ini respectively, then try opening it again, isn't it much faster?

34. Skip the startup screen

Press ESC during startup, or simply modify msdos.sys by adding logo=0 to the options paragraph.

40. Speed up floppy drive transfer speed

When storing a lot of data on a floppy disk, it can be quite annoying to wait. We cannot modify the mechanical structure of the floppy drive, but we can increase the data transfer speed by modifying the system registry. The specific method is as follows:

Open the system registry editor, find

"HEKY-LOCAL-MACHINESystemCurrentControlsetServicesClassFCD00", in its right window blank area, right-click and create a new "DWORD" value, name it "Fore-Fifo", set the value to "0". Finally, close the registry editor, restart the computer, and everything is okay!

46. Usage of the letter "v" in Smart ABC input method:

① In Smart ABC input, V + number (1~9) can input various characters, graphics, numbers, etc., try it and you'll find endless wonders, but I guess most people have already used it?

② In Smart ABC input, during the process of inputting pinyin ("standard" or "double strike" mode), if you need to input English, you don't have to switch to English mode. Just input "v" as a marker, followed by the English you want to input, and press the space bar.

47. In Office series represented by Word, micro-adjustment of non-character elements such as graphics, text boxes: Take Word as an example, inserting non-character elements such as graphics or text boxes into the page, during page layout, you are often disturbed by Word's inherent limitations, making it difficult to achieve satisfactory results. During use, I have found the following experience to share with you:

① When adjusting the position of a certain picture (or other element, hereafter referred to as a picture), use the mouse or arrow keys to move it to a rough position, hold down the Ctrl key, and then use the arrow keys to move it. You will find that the distance moved each time is much smaller than before, and under Word's required precision, it should fully meet your requirements.

② When adjusting the size of a certain picture (or other element, hereafter referred to as a picture), if you don't use the mouse, the changes in size are discontinuous. Hold down the Alt key and adjust it, and now its size can be adjusted arbitrarily.

48. Has anyone encountered the phenomenon of a black screen on the computer? After pressing ESC, are there still many black shadows left on the desktop? Especially the taskbar below... Without using the mouse, it turns black... Press "Start"---"Shut Down Computer"---"Cancel", and it will recover.

49. In 2000 or Xp, hard drives are shared by default, even if you turn off sharing, it will still be shared next time you start.

To prevent workstations within the domain from stealing your personal work results, try creating a batch file and placing it in all users' startup (or create a shortcut inside). Content as follows: net share c$ /delete

net share d$ /delete

net share e$ /delete

50. Remove the default sharing of Windows 2000

Modify the registry: For server versions: In the registry editor (regedit.exe), sequentially find "HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServiceslanmanserverparameters", then create a new "double-byte value" under it, name it "AutoShareServer", and set its value to "0". Then restart the server. For professional versions: Same as above, just change AutoShareServer to AutoShareWks

51. Double-click the speaker on the taskbar, if you feel that the pop-up volume control panel occupies too much desktop space or cannot be fully displayed, press Ctrl+S and it will display in mini mode, to restore press Ctrl+S again.

57. Our workplace relies heavily on computers (computers in different departments are connected in a network), most employees do not know how to use email, and the leadership requires using network resources to distribute notices, etc. In this situation, I use the following method (simple, easy to implement, cost-free):

1. Establish a shared folder (read-only) on any computer (as a mail server), name the folder "XX" (meaning mailbox).

2. Create a shortcut on other computers' desktops (make sure to map the network resource, pointing to the shared folder of the mail server), name the shortcut "Public Mailbox".

3. Place all "Word" files (notices) in this folder.

4. Users just need to click this shortcut to see all the notices.

60. There are three display modes when using a laptop with a projector:

1. The laptop screen has display, but the projector does not.

2. Both the laptop screen and the projector have display.

3. The projector has display, but the laptop screen does not.

The switching method is to press the Fn key plus F3 (for Compaq machines, that is, plus the key with a small monitor icon)!
