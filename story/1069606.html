Venmo 变得更加私密——但它仍然不完全安全

2021-07-25 13:56:12

流行的移动支付服务 Venmo 重新设计了其应用程序。这通常是您可以放心忽略的新闻，但此公告值得仔细研究。除了进行一些导航调整和添加新的购买保护之外，PayPal 拥有的平台最终关闭了其全球社交源，该应用程序在其中发布了来自世界各地的人们的交易。这是解决应用程序世界中最突出的隐私问题之一的重要一步，但工作尚未完成。多年来，Venmo 的全球动态一直是对完全陌生的人的财务习惯进行窥探的洞察的字体。提要不显示给定交易的金额，但包括名称和注释表情符号和喜欢。点按姓名即可进入该用户的个人资料，一个有进取心的忙碌人士（或更糟）可以很快建立一个小档案，记录该人的朋友、他们的爱好以及他们滑入信息流中的任何其他内容——也许没有意识到该信息的公开程度。在写这些段落的时间里，亲戚们互相报销费城人队的门票，有人支付了“液体黄金😍”，不止一组室友平分了他们的网费。多年来，Venmo 交易和其他用户数据的可见性一直受到隐私和消费者权益倡导者的批评。数字权利组织电子前沿基金会的激进主义主管 Gennie Gebhart 说：“这种对社会支付应用程序的这种奇怪的企业位、这种企业 DNA 的承诺是一个巨大的负担。” “这不是一场等待发生的灾难，而是一场已经发生在这么多人身上的灾难。”最近和最引人注目的公开可能出错的例子发生在 5 月，当时 Buzzfeed 的一组记者找到了乔·拜登总统的 Venmo 帐户，以及他的家人和密友的帐户，只需在应用程序中搜索即可。他们花了10分钟。当时，即使您的交易记录被锁定，您的好友列表也是任何人都可以找到的公平游戏。对于围绕通常敏感的汇款和收款业务而构建的应用程序而言，这似乎有点不明智。然而，在 Buzzfeed 报告发布两周后，Venmo 添加了新的隐私控制，让您首次将应用程序上的联系人列表设为私有。删除全局提要扩展了这项工作，使窥探完全陌生的人变得越来越困难。很快，该应用程序的社交元素将仅限于您的 Venmo 联系人所做的事情。该公司在宣布重新设计的博客文章中表示：“这一变化使客户能够与最重要的人联系并分享有意义的时刻和体验。”虽然这当然算作进步，但隐私倡导者认为这还远远不够。 “Venmo 终于意识到，在金融应用程序上进行最大程度的宣传是一个糟糕的主意，”专注于互联网开放性和可访问性的非营利组织 Mozilla 基金会的高级活动家 Kaili Lambe 说。 “然而，从一开始我们就一直呼吁 Venmo 默认是私有的，因为如此多的 Venmo 用户实际上并不知道他们的交易是公开的。”

Venmo 的一位发言人表示，该公司目前没有计划考虑将这些交易默认设为私有。这意味着用户仍然需要竭尽全力确保他们的每个点对点交易不会向全世界广播。很难看出维持现状的好处。 “你会想到很多非常敏感的用例，”Gebhart 说。 “你想到治疗师，你想到性工作者。你想想美国总统。想象一下这些默认设置可能会出现严重错误并对真实的人造成真正伤害的地方，并不需要很大的想象力。” Venmo 公开默认立场的影响已经超出了拜登账户的发现。 2018 年，隐私倡导者和设计师 Hang Do Thi Duc 使用 Venmo 的公共 API 整理了平台上近 2.08 亿笔交易，仅根据他们在应用程序中的活动，拼凑出五名用户的详细画像。第二年，程序员 Dan Salmon 编写了一个 20 行的 Python 脚本，让他在几周内获得了数百万的 Venmo 付款。此后，Venmo 对您通过公共 API 访问交易数据的速度进行了限制，但 Salmon 表示，该公司做得还不够。 “Venmo 基本上有一个我可以连接到交易数据的消防水管，”他说。 “现在这被切断了，交易仍然存在；只需要多走几步就可以得到它们。”他说构建一个新的抓取工具需要大约一个小时的工作。 “在 Venmo，我们会定期评估我们的技术协议，作为我们对平台安全和不断改善客户的 Venmo 体验的承诺的一部分。抓取 Venmo 违反了我们的服务条款，我们积极努力限制和阻止违反这些政策的活动，”Venmo 发言人 Jaymie Sinlao 在一封电子邮件声明中写道。 “我们继续为获得批准的开发人员提供对我们现有 API 的选择性访问，以继续在 Venmo 平台上进行创新和构建。” Venmo 远不是唯一一个让您选择退出共享而不是主动寻求共享的应用程序。但是因为它的用例完全是金融的，所以赌注要高得多，而且其用户的假设可能会放错地方。 Venmo 并没有让用户特别容易弄清楚他们正在分享什么或没有分享什么。 2018 年，它与联邦贸易委员会达成和解，部分原因是其令人困惑的隐私设置。 “有趣的是，人们非常惊讶地发现金融服务应用程序默认是公开的，”Mozilla 基金会的兰贝说。 “即使是使用 Venmo 多年的人也可能不知道他们的设置是公开的。”

为确保您不会继续前进，请前往“设置”>“隐私”并选择“私人”。然后点击过去的交易，然后点击全部更改为私人以追溯锁定。在此期间，请继续点击好友列表，然后点击私人并关闭出现在其他用户的好友列表中。否则，您将与您认识的每个人以及许多您不认识的人分享您的信用卡购买的数字等价物。或者考虑使用 Square 的 Cash App 之类的东西，默认情况下是私有的。对于 Venmo 及其用户而言，失去全球供稿是迈向隐私的重要一步。希望还有更多步骤。
