Kaspersky Internet Security is a complete solution that protects your computer from almost all major threats on the internet, including viruses, phishing, hacker attacks, adware, spam, and spyware. The features of Kaspersky Internet Security v8 are as follows:

- Firewall performance has increased, using multiple stealth modes and adding many new contents;

- The antivirus system uses DNA-based detection methods, which can scan and remove 100% of viruses;

- The active defense system adds macro protection and group policy protection systems, further enhancing system security;

- To promote legitimate use, Kaspersky 8.0 eliminates the activation mode via authorization files, only allowing activation through activation codes. Corporate users will have an activation code that can activate multiple computers;

- The parental control system will be fully implemented in Kaspersky 8.0, achieving 80% anti-porn interception with a powerful database of inappropriate content always running in the background;

- The monitoring level of Kaspersky has been improved, with 15 monitoring items present in the task manager, providing real-time surveillance and feedback for various file types such as Bxt, txt, exe, td, key, doc, ppt and their opening methods;

- The spyware protection system adds bank key protection and VISA monitoring, and can warn against Nimda virus, Viking virus, and other popular network viruses;

- The scanner adds Rootkit and Trojan scanners;

- A new skin-changing system allows free customization of the appearance of the Kaspersky antivirus software;

- Added user interface enlargement options, giving advanced users more freedom.
