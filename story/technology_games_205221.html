A Detailed Explanation of the Order of Immortal Martial Arts and Cultivation Manuals

by lk0gameh on 2009-07-18 21:32:33

After passing the 89-level cultivation, one can learn the following skills: True Lion, True Lingfeng, True Tiger Strike, True Cunli, True Yi Jin, True Yi Sui, True Dragon Manifestation.

So, how should the order of learning be arranged? Which skill should be learned first and which later? What is the most scientific way to learn them? This is a question worth discussing.

In my personal opinion, it should be as follows:

1. Learn True Yi Sui first

Learning True Yi Sui first means a qualitative improvement in physical defense. True Yi Sui can increase physical defense by 250%, much more substantial than True Golden Bell. With this scripture, leveling up and fighting physical monsters becomes much easier. It's a guarantee for leveling up and should definitely be learned first.

PS: After learning this scripture, learning True Golden Bell will only add a small amount of physical defense.

2. True Lion

Some people say that True Lion only reduces ten points of spirit energy and isn't very useful, far less effective than the 100% hit rate of Mad Lion. Actually, it's not like that. Don't underestimate the reduction of these ten points of spirit energy. During leveling up, with True Lion, you'll find that your spirit energy lasts longer. As for the hit rate, I haven't seen many misses with True Lion; it's basically infallible and highly reliable. Moreover, when dueling or PKing against magic users, having True Lion means you can stun them first with the same amount of accumulated spirit energy. True Lion is very useful for both leveling up and PKing. Therefore, I strongly recommend that the second book for Xianwu players should be True Lion.

3. True Golden Bell

For Xianwu players who already have True Yi Sui, learning True Golden Bell is actually for dungeons and teammates. Because with True Yi Sui, learning True Golden Bell doesn't significantly improve physical defense. However, True Golden Bell is much more practical for teammates than the regular Golden Bell, absolutely a good skill. With True Golden Bell, you have an advantage in dungeons, and everyone will especially welcome you. Since the new dungeons from Chapter Two opened, the drop rate of True Golden Bell has been decreasing, and high-level players are refreshing dungeons, thus neglecting the level 99 book. Having True Golden Bell makes you more popular and competitive in dungeons. Additionally, with True Golden Bell, when you activate the magic defense scripture Yi Jin, your physical defense won't drop too much, making it practical! I recommend learning True Golden Bell as the third book.

4. True Yi Jin

A good skill to enhance magic defense, especially when paired with the 90-level military rank equipment for WX (Wuxia). Without MaoMao, you can handle small-scale group magic attacks on monsters, and magical attacks won't hurt as much. During PK, you won't fear mages and other magic-based classes as much. There's no need to elaborate further; once you use it, you'll know it's really good. I recommend learning this as the fourth book.
