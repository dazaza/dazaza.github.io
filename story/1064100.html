Microsoft的Windows Virtual Desktop现在是Azure Virtual Desktop

2021-06-08 01:20:20

“当我们第一次使用Windows Virtual Desktop时，世界上一年半前的Windows Virtual Desktop，Microsoft Azure Virtual Desktop的总经理Kam Vedbrat表示，世界上是一个非常不同的地方。 “要钝，我们看着这项服务，我们在建造的东西，我们正在建造它，相当不同。当时没有人知道这种全球大流行将发生，并且它会导致世界各地的许多组织以及数百万人必须基本上离开办公室和工作 - 服务将发挥的角色在实现很多的地方。“

虽然原始想法是帮助企业将虚拟桌面环境从他们的数据中心移动到云端，但大流行为Windows Azure虚拟桌面带来了一个新的用例。它现在托管虚拟学校实验室的任何东西到传统的远程企业用例。这些新用户也有一些不同的需求和来自这些用户的专业知识最初是为了，所以在今天的名称变化之上，该公司也在推出一组新功能，应该使新用户更容易开始使用使用Azure Virtual Desktop。

其中是新的快速入车体验，即将在公共预览中发布。 “我们看到的一块反馈是，由于许多组织正在寻找Azure虚拟桌面，以便启用混合工作的新方案，他们希望快速获取这些环境，以便快速地了解他们的工作方式，他们的应用程序如何表现如何，如何考虑应用程序组和主机池以及一些新概念，“vedbrat解释说。理想情况下，它现在应该只需点击几下即可从Azure Portal设置完整的虚拟桌面环境。

Azure Virtual Desktop中的新增功能也支持管理Microsoft Endpoint Manager，Microsoft对设备管理的统一服务的Microsoft Endpoint Manager管理多次会话虚拟机（VM）。此标记第一时间Endpoint Manager能够处理多次会话VM，这些VM是Azure虚拟桌面的最大销售点之一，因为它允许业务在运行云中运行Windows 10 Enterprise的同一台计算机上托管多个用户。

此外，Azure Virtual Desktop现在还为Azure Active Directory提供增强的支持，除了新的每用户访问定价选项（除了在Azure基础架构上运行的成本），允许用户将应用程序传递给外部用户。 Microsoft认为，这将允许软件供应商作为SaaS解决方案将应用程序提供。

至于名称更改，vedbrat争辩说，虽然Windows显然是经验的核心，但许多服务的用户也关心底层的Azure基础架构，例如存储或网络。 “他们看看他们正在创造的更广泛的环境 - 那个窗口的遗产，他们在云中创造了 - 他们认为这是一个更大的东西，他们看起来很多蔚蓝。所以我们觉得在这一点上有正确的事情，为了解决我们的客户正在接受的更广泛的观点，就是看看新名称，“他解释说。

我认为Windows虚拟桌面解释了核心概念，但没有人指责我是营销天才。
