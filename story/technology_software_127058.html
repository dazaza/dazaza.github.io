Enhanced Task Manager (Powerful Process Termination Program)

by youxiruanjian88 on 2009-02-27 08:46:35

【Basic Introduction】

This is a tool similar to an enhanced version of the Task Manager, which implements all the basic and practical functions of the Windows Task Manager and expands/strengthens these functions. It can be used as an auxiliary tool for detecting Trojans and viruses.

【Software Functions】

1: Process Management

Processes or files that are not part of the system are displayed in red, while blue indicates a system program where other DLL modules have been inserted into the process (for example: Explorer.exe). Double-clicking on a process will list the modules and detailed information contained within the process. You can terminate all processes, including system processes, but it is not recommended to terminate any processes before the first svchost.exe, as doing so may cause the system to restart or make it impossible to shut down.

2: Service Management

Non-system services are displayed in red. Clicking on the list header allows sorting, making it easier to quickly find newly added system services. For services started with svchost.exe, the file path shown is the path of the service file itself, not the path of svchost.exe. Deleting a service removes the service registry key (no backup is provided before deletion. Therefore, if you decide to delete, you must confirm yourself whether you really want to do this!).

3: Startup Management

You can view information about many hidden startup programs!

4: File Monitoring

It can monitor changes to files, generally used for Trojan removal and monitoring streaming media on web pages.

5: Other Notes:

Alt+Q is the program hotkey; if this hotkey fails, it can be changed to Alt+~! When you're not using this program, please minimize the window to conveniently monitor your computer in the background. There are still many more functions waiting for you to explore..... :-)
