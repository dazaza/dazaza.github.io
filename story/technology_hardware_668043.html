1. Main Board Test

Startup Method: Open the front door of Sub Unit No. 1 (IP) and press the Main Board Test (MAIN TEST) button for a 10-player fishing machine to enter manual test mode.

End Method: During testing, press the MAIN TEST button or select the PE item in the test menu and press the red lion button to exit.

Test Items: The project number will be displayed on the Coin-Out (GREDIT) LED. Press the coin-out button to select the test item (any one of the six sub-units' coin-out buttons can be used).

P1: RAM/ROM (Read/Write Memory/Read-Only Memory) Test - Press the red lion button to start the test. The results will be shown on the Sub Unit Ratio LED.

P2: Micro-switch Test - All current micro-switch settings will be displayed on the ratio LED.

P3: Light Emitting LED Test Mode A - All light emitting LEDs will light up in the order of green, red, orange. Press the red lion button to pause temporarily (PAUSE).

P4: Light Emitting LED Test Mode B - Orange LEDs will light up sequentially in a counterclockwise direction. Press the red lion button to pause temporarily (PAUSE).

P5: Reed Switch Test - The normal operation of reed switches will be displayed on 11 LEDs. Red Lion Button: Motor ON/OFF.

P6: Motor Test - Red Lion Button: DC motor ON/OFF (drives outer circle animals). Green Lion Button: Stepper motor ON/OFF (drives needle).

- Press the red panda button to move clockwise. Press the green panda button to move counterclockwise. Adjust until the needle tip points exactly at the center of the light emitting LED. After adjustment, press the green lion button to rotate the needle and check its action.

P7: Sound Test - Press the red lion button to emit different sounds in sequence. Press the green lion button to repeat the last sound.

P8: Counter (COIN COUNTER) Test - Token IN/OUT counters will each jump 5 times in sequence. Press the red lion button to pause temporarily (PAUSE).

P9: Communication Test - Press the red lion button to conduct the test. Results will be displayed on the ratio LED.

PE: End Test - Press the red lion button to end the test mode.

2. Sub Board Test

Startup Method: Open the front door of the sub-unit and press the TEST button to enter manual test mode for that sub-unit. Each of the six sub-units has its own test button. During main board testing and coin-out device operation, the sub-board test button does not function.

End Method: During testing, press the TEST button or select TE from the test items and press the red lion button.

Account Checking: Press the sub-board test button once; the total score position (point distribution area) shows B1 to view the sub-unit account. Press the yellow rabbit button again to display B2 for the main unit account.

First Row: 300 000 000 - Point Distribution (Total Out)

Second Row: 200 000 000 - Total Score (Total Bet)

Third Row: 100 000 000 - Total Out (Total Win)

Fourth Row: 000 000 - Point Distribution (Total In)

Note: B1 refers to the sub-unit account; B2 refers to the main unit account.

Other item numbers will be displayed on the CREDIT LED. Press the yellow coin-out button to proceed to the next item.

T1: RAM/ROM (Read/Write Memory/Read-Only Memory) Test - Press the red lion button to start the test. Results will be displayed on the ratio LED.

T2: Micro-switch Test - All current micro-switch settings will be displayed on the LED.

T3: LED Test - All LEDs will sequentially display the digits 0-9. Press the red lion button to pause temporarily (PAUSE).

T4: LED Running Light Test - LEDs will display in an 8-stroke form from left to right.

T5: I/O (Input/Output) Test - Press any button, and the bulb inside that button will light up. The red coin-out button will not light up.

T6: Sound Test - Press the red lion button to emit different sounds in sequence. Press the green lion button to repeat the last sound.

T7: Coin-Out Device Test - Press the red lion button to eject tokens, ejecting 10 tokens. Press the green lion button to eject all tokens.

T8: Counter (COIN COUNTER) Test - Token IN/OUT counters will each jump 5 times in sequence. Press the red lion button to pause temporarily (PAUSE).

TE: End Test - Display program version.

3. Main Unit Related Fault Information

Faults related to the main board or central part will be displayed on the 1P-6P sub-unit display panel (CREDIT LED).

EA: RAM BAD - Bad RAM (Read/Write Memory). If possible, purchase the same type and specification RAM locally for replacement; otherwise, return to the original factory for repair.

EB: ROM BAD - Bad ROM (Read-Only Memory). If possible, purchase the same type and specification ROM locally for replacement; otherwise, return to the original factory for repair.

EC: BACKUP RAM BAD (Data Memory) - Bad data memory RAM. Please delete the originally set BACKUP MEMORY.

ED: DC Motor Fault - Fault with the DC motor, reed switch, or wiring contact.

EE: Stepper Motor Fault - Fault with the stepper motor or NC optical sensor, or wiring contact.

EH: POWER DOWN - Sudden power outage during game operation. Open the front door of 1P and press the MAIN TEST button.

4. Sub Unit Related Fault Information

Faults occurring in each sub-unit will be separately displayed on each sub-unit's display panel (CREDIT LED).

E1: RAM BAD - Bad RAM (Read/Write Memory). If possible, purchase the same type and specification RAM locally for replacement; otherwise, return to the original factory for repair.

E2: ROM BAD - Bad ROM (Read-Only Memory). If possible, purchase the same type and specification ROM locally for replacement; otherwise, return to the original factory for repair.

E3: BACKUP RAM BAD (Data Memory) - Bad data memory RAM. Please delete the initially set BACKUP MEMORY.

E4: Insufficient Tokens - Turn off the sub-unit power and replenish tokens.

E5: Token Jam - Remove jammed tokens and restart the power supply.

E6: Coin-Out Device Fault - Excessive token ejection or fault with the coin-out device itself. Restart the power switch to resolve.

E7: Communication Fault - Correctly set the micro-switches (SW1-2 communication number) on the sub-board. Check the voltage of the power box.

E8: POWER DOWN - Fault caused by multiple coin-outs without action or no coins during coin-out process. Open the front door, press the TEST button, and eject excess tokens.

C1: Communication Connection Standby - Communication connector has fallen off or is not properly connected.
