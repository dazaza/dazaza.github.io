适用于Windows 10的Microsoft新PowerToys Run Launcher现已可供下载

2020-05-20 07:29:44

微软今天将为Windows10发布一款新的类似Spotlight的启动器应用。新的PowerToys Run Launcher旨在取代和现代化现有的Win+R快捷方式，包括在Windows中快速搜索应用程序和文件，以及计算器等插件，以及查找正在运行的进程的功能。

此早期版本将支持通常由内置Windows开始菜单搜索功能处理的基本搜索任务。但也有计划让它成为一个功能更强大的启动器，类似于MacOS上的Alfred，并且比苹果的Spotlight搜索功能更强大。

当前的Win+R功能是基本的，Windows高级用户使用它来启动命令提示、regedit、Powershell实例，甚至是Windows中控制面板等区域的快捷方式。这个新的PowerToys Run Launcher将支持目前Windows中Run所支持的所有命令，但微软正在与一个开源社区合作，这使得它变得更加强大。

微软一直在与WOX和WindowWalker的制造商合作，将这些项目整合到PowerToys Run中。基本功能今天就会推出，虽然是早期版本，但这里和那里都会有一些错误。

启动程序的好处意味着我们很快就会得到插件，或者能够添加定制的网络搜索、片段等等。微软现有的Windows开始菜单处理来自网络的搜索结果，但它也会强制你进入必应搜索结果，并打开Edge浏览器进行任何网络查询。

除了PowerToys Run，微软今天还发布了键盘管理器PowerToy。这是一个简单的键盘重新映射器，允许Windows10用户重新定义键盘上的按键。只要键盘管理器和PowerToys在后台运行，就会重新映射按键，并且您可以交换各个按键，甚至可以交换Windows快捷键。

键盘管理器是PowerToys最新0.18版本的一部分，可从微软的GitHub发行版获得。

PowerToys Run和键盘管理器是Microsoft正在开发的众多PowerToys中的一部分。微软去年最初带回了PowerToys，让任何人都可以为高级用户改进Windows10，第一套于9月份面世。PowerToys最早是在Windows95中推出的。这些应用程序是Windows工程师测试原型功能的快捷方式，微软很快将一些最好的应用程序打包成PowerToys捆绑包。
