Meiku Official Website New Version Released: Interpretation of 5 Major Features: Enhanced Experience, Practicality Above All (with photo)

by xdash on 2011-08-31 20:23:13

Makumo (http://mknote.com) is a permanently free personal cloud-based notepad developed by Shanda Innovation Institute, akin to Evernote overseas. You can securely store various notes and documents, accessible via web or mobile clients for editing and viewing anytime. If you're a long-time reader of Sync Control, this service should be familiar.

Last night (or rather, early this morning), Makumo underwent another significant update this year, mainly revamping and upgrading the official website and the web version's backend. After testing by XDash, the new Makumo Web version has made strides in user guidance and functional practicality, with very ideal results after the redesign. This article will interpret and recommend it to readers with text and images.

One, A Simpler and More Practical Official Website Design

(The image above: The newly redesigned Makumo official website homepage)

The new version of the official website homepage uses large areas of the main VI color - green as the background, paired with a simple and elegant page layout.

Clicking on the prominent "Download PC Client" allows direct download of the installation package to your local system. Compared to the previous design (where clicking the download button didn't immediately start the download but instead redirected to a download list page where users had to manually select what they wanted to download), this version simplifies user operations to the extreme, aiming for the fastest way to achieve the user's goal. (Honestly, the original design caused no small amount of criticism from me towards the Makumo team. This version finally fixed it, which I believe will greatly help improve the actual conversion rate of the Makumo website.)

Two, Improved Guidance for New Users and User Help

Although spurred by Apple iCloud, the industry generally looks favorably on the future of "sync," but currently, user demand still needs to be tapped, and understanding of sync tools needs further popularization (can I say that readers of Sync Control are at the forefront of the times? ^_^). This time, Makumo added a more friendly user guidance interface.

(The image above: Makumo official website newbie help interface - usage guide)

(The image above: Makumo official website newbie help interface - usage scenarios)

On the feature page, you can learn about Makumo's most basic product functions (save/sync, edit/organize, share/publish); on the usage guide page, there are illustrated interface annotations; on the usage scenario page, six common scenarios for using sync tools are listed, such as travel experiences, meeting records, web excerpts, etc.

Additionally, when first entering the newly redesigned Makumo personal backend, there will be friendly prompts for new features:

(The image above: Prompt for the new "dragging notes" function)

Three, Drag-and-Drop Operations

The new personal backend interface is cleaner and simpler, compared to the previous version where certain visual elements could easily distract attention. This improvement helps focus users on "recording" itself. It's also clearly noticeable that the new interface draws inspiration from the new Gmail, such as the "draggable" visual prompt before the note list and the upward arrow button shape above the content.

Support for drag-and-drop is a major highlight of this redesign and an embodiment of user-friendliness. In the classification list on the left, you can drag one category into another to make it a subcategory; you can also directly drag a note from the right-hand list into the left-side classification (for example, quickly setting a private note as a shared note).

(The image above: Dragging a note under a specific classification)

When setting classifications by dragging, two rules need attention: 1) Classifications that already have subcategories cannot be placed under other classifications (quite tongue-twisting); 2) Automatically created default classifications cannot be placed under other categories. These points do not receive sufficient feedback from the system during operation, relying on the user's own insight. Hopefully, subsequent versions will offer clearer prompts.

Four, Useful New Features

This Makumo Web version also adds a right-click menu (right-clicking on a classification allows direct creation of a note under that classification, among others), star marking (similar to Gmail, adding a prominent marker to a specific note), word count statistics (no longer needing to paste into Word to check word count), note view scaling, printing, optional auto-save, etc., making the overall experience smoother.

(The image above: Add star marking to notes)

(The image above: Word count statistics function)

Five, Social Sharing

For someone like XDash who is obsessed with microblogging, if there are good note materials, they would certainly want to share them with the appropriate audience in the first instance and hope that more discussions can deepen their understanding of digested content. This update allows binding to commonly used social networks (currently supports Sina Weibo, Douban, Tencent Weibo, Sohu Weibo, Kaixin Network, QQ Zone, Renren Network). When editing a shared document, simply select the desired social network from the menu.

(The image above: Share to commonly used social networks)

(The image above: The default format for sharing to Sina Weibo is title + link, which can be edited manually)

Personally, I feel that this update to Makumo Notes has reached top-tier levels domestically in terms of usability and aesthetics. Comparing it to the current version of Evernote's web interface, the former seems to better align with Chinese usage habits ——

Believe it or not, but I do. With the new floating window of the PC client, Makumo has become my preferred choice to replace the bulky Evernote.
