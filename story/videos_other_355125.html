### Operating Modes of IC Commands

#### Normal Mode

In normal mode, the acknowledgment time is optimized. The timings remain approximately the same for different configurations of the slave. To reduce acknowledgment times in normal mode, registers 0x06-0x09, 0x0C, 0x0D, 0x10-0x17, 0x50, 0x51, 0x57-0x60, and 0x7E are given only read access. Writing to these registers can only be done in setup mode.

#### Setup Mode

In this mode, all registers have both read and write access (except those that are read-only). The acknowledgment times are longer compared to normal mode. When CapSense scanning is disabled (command code 0x0A in command register 0xA0), the acknowledgment times can be improved to values similar to the normal mode of operation.

### Device Operation Modes

CapSense Express devices are configured to operate in any of the following three modes to meet different power consumption requirements:

- **Active Mode**

- **Periodic Sleep Mode**

- **Deep Sleep Mode**

#### Active Mode

In active mode, all device blocks, including the CapSense subsystem, are powered. The typical active current consumption of the device across the operating voltage range is 1.5 mA.
