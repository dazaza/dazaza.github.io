Absolutely practical good stuff! Give drive C a little slimming, and your computer will run at lightning speed (applicable to XP).

by cqe on 2009-07-26 01:53:53

For users who have been using their computers for a while, there will be some residual files accumulated in the C drive that occupy too much space. Should we delete those unnecessary files?

Deleting residual files and freeing up space can speed up our computer and avoid wasting resources. Which files should be deleted?

This section has been set to hidden; you’ve already replied, and below is the hidden content:

1. Open "My Computer" - "Tools" - "Folder Options" - "View" - check the option "Show all files and folders" - "OK".

2. Delete the contents of the following folders:

- All files under `x:\Documents and Settings\User Name\Cookies\` (keep the index file).

- All files under `x:\Documents and Settings\User Name\Local Settings\Temp\` (user temporary files).

- All files under `x:\Documents and Settings\User Name\LocalSettings\Temporary Internet Files\` (page files).

- All files under `x:\Documents and Settings\User Name\Local Settings\History\` (history records).

- All files under `x:\Documents and Settings\User Name\Recent\` (recently viewed file shortcuts).

- All files under `x:\WINDOWS\Temp\` (temporary files).

- All files under `x:\WINDOWS\ServicePackFiles` (backup files after upgrading SP1 or SP2).

- Compressed files under `x:\WINDOWS\Driver Cache\i386` (backup files for drivers).

- All files under `x:\WINDOWS\SoftwareDistribution\download`.

3. If a Windows Update upgrade has been performed on the system, delete the following files:

- Hidden files under `x:\windows\` starting with `$u...`.

4. Then defragment the disk. During the process, please exit all running programs.

5. After defragmentation, open "Start" - "Programs" - "Accessories" - "System Tools" - "System Restore" - "Create a restore point" (it's best to name the restore point with the current date).

6. Open "My Computer" - right-click the system drive - "Properties" - "Disk Cleanup" - "Other Options" - click "Clean up" under the System Restore section - select "Yes" - OK.

7. After various software and hardware installations are complete, XP rarely needs updates. Delete the system backup files: Start → Run → `sfc.exe /purgecache` (this command immediately clears the "Windows File Protection" file cache, releasing the occupied space).

8. Delete DLL files under `\windows\system32\dllcache` (reduce by 200-300 MB). These are backup DLL files. As long as you've copied the installation files, you can do this.

9. XP automatically backs up hardware drivers, but after correct driver installation, hardware changes are unlikely. You can consider deleting this backup. The file is located at `\windows\driver cache\i386`, named `driver.cab`. You can directly delete it, usually around 74 MB.

10. Delete unused input methods: For many netizens, not all input methods provided by the Windows XP system suit their use. For example, IMJP8_1 Japanese input method and IMKR6_1 Korean input method. If you don't need them, you can delete them. Input methods are located in the `\windows\ime\` folder, occupying a total of 88 MB of space.

11. After upgrading, notice that many directories like `$NtUninstallQ311889$` appear in the windows directory. Delete them all, they take up 10-30 MB each.

12. Additionally, keeping the contents of the `\windows\help` directory is a form of harm to me, haha... Delete them all!

13. Turn off System Restore: Over time, System Restore consumes a lot of hard drive space. Therefore, it is necessary to manually set it to reduce hard drive usage. Open the "System Properties" dialog box, select the "System Restore" option, and check the "Turn off System Restore on all drives" checkbox to turn off System Restore. Alternatively, you can only set restoration for the system's disk or partition. First, select the system's partition, click the "Configure" button, uncheck the "Turn off System Restore on this drive" option, and set the amount of disk space used for system restoration.

14. Hibernate function consumes a lot of hard drive space. If you rarely use it, you might as well turn it off. To turn it off: Open "Control Panel", double-click "Power Options", in the "Power Options Properties" dialog box, select the "Hibernate" tab, and uncheck the "Enable hibernation" checkbox.

15. Uninstall infrequently used components: By default, XP installs some system components, but most of these components are ones you will never use. You can uninstall them in "Add/Remove Windows Components". However, some components are hidden by default in "Add/Remove Windows Components". In this case, you can proceed as follows: Open the file `\windows\inf\sysoc.inf` with Notepad, and use the Find/Replace function to replace all occurrences of the "hide" character with nothing. This removes the hidden attribute from all components. Save the file, exit, and run "Add/Remove Programs" again. You will see many options that were previously invisible. Delete those components you don't need (remember to save as sysoc.inf, not the default sysoc.txt), such as Instant Messenger Service, Fax Service, Windows Messenger, Code Table, etc., which can free up nearly 50MB of space.

16. Clear system temporary files: Temporary system files are generally stored in two locations: one is the Temp folder under the Windows installation directory, and the other is the x:\Documents and Settings\"Username"\Local Settings\Temp folder (Y: is the system partition). Files in both locations can be directly deleted.

17. Clear Internet temporary files: Regularly deleting large amounts of Internet temporary files generated during web browsing will save a lot of hard drive space. Open IE browser, choose "Internet Options" from the "Tools" menu, select the "General" tab in the pop-up dialog box, click the "Delete Files" button under the "Internet Temporary Files" section, and in the "Delete Files" dialog box, check the "Delete all offline content" checkbox, then click the "OK" button.

18. Clear pre-read files: Although the pre-read settings in Windows XP can improve system speed, after some time of use, the number of files in the pre-read folder becomes quite large, leading to longer search times. Moreover, some applications may generate dead link files, further increasing the burden on system searches. Therefore, these pre-read files should be regularly deleted. Pre-read files are stored in the Prefetch folder under the Windows XP system folder, and all files in this folder can be deleted.

19. Compress NTFS drives, files, or folders: If your hard drive uses the NTFS file system and space is tight, you can consider enabling the NTFS compression feature. Right-click the drive to compress - "Properties" - "General" - "Compress drive to save disk space", then click "OK", and in the "Confirm Attribute Changes" dialog box, select the desired options. This can save about 20% of hard drive space. When compressing the C drive, it’s best to do so in Safe Mode for better results.

20. Turn off Dr.Watson: To turn off Dr.Watson, open the Registry Editor, find the "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\AeDebug" branch, double-click the Auto key value name, change its "Value Data" to 0, and finally press F5 to refresh and make the settings effective, thus canceling its operation. You can also type "drwtsn32" in "Start" -> "Run", or go to "Start" -> "Programs" -> "Accessories" -> "System Tools" -> "System Information" -> "Tools" -> "Dr Watson" to bring up the Dr.Watson utility, retaining only the "Dump full thread context" option. Otherwise, if a program errors out, the hard drive will read for a long time and consume a lot of space. If this situation occurred before, look for the user.dmp file and delete it to save dozens of MB of space.

21. Turn off Remote Desktop: "My Computer" -> "Properties" -> "Remote", uncheck the "Allow users to remotely connect to this computer" option in "Remote Desktop".

22. Disable ZIP support in XP: By default, Windows XP enables support for zip files, which consumes certain system resources. You can choose "Start" -> "Run", and in the "Run" dialog box, type "regsvr32 /u zipfldr.dll", confirm by pressing Enter to disable ZIP decompression support in XP, thus saving system resources.

23. Turn off error reporting: When an application encounters an error, it will pop up a window asking to send an error report. Such error reports are almost meaningless to ordinary users, so turning it off is a wise choice. In the "System Properties" dialog box, select the "Advanced" tab, click the "Error Reporting" button, in the pop-up "Error Reporting" dialog box, select the "Disable error reporting" radio button, and finally click "OK". Alternatively, you can also disable error reporting through Group Policy: Type "gpedit.msc" in "Run", run the "Group Policy Editor", expand "Computer Configuration" -> "Administrative Templates" -> "System" -> "Error Reporting", double-click the "Report Errors" setting on the right side, and in the "Properties" dialog box, select the "Disabled" radio button to disable "Report Errors".

24. Turn off unused devices: Windows XP always tries to install drivers and manage all devices on the computer, which not only slows down system startup but also consumes a lot of system resources. For this reason, you can disable less commonly used devices such as PCMCIA cards, modems, infrared devices, printer ports (LPT1), or serial ports (COM1) in Device Manager. The method is to double-click the device to be disabled, and in the "General" tab of its property dialog box, select "Do not use this device (disable)". Restart the settings to take effect, and enable them again from Device Manager when needed.

25. Periodically clean system restore points: Open Disk Cleanup, select Other Options -> Clean System Restore Points, and click Clean.

26. Uninstall unnecessary programs, no need for more explanation here.

27. Other optimizations:

a. Install application software on other hard drives (not under the system drive, which is also beneficial for reinstalling the system);

b. Move the "My Documents" folder to another partition: right-click the "My Documents" icon on the desktop, choose "Properties" -> "Move";

c. Move the IE temporary folder to another partition: open IE browser, choose "Tools" -> "Internet Options" -> "General" -> "Settings" -> "Move Folder";

d. Move virtual memory to another hard drive;

e. Point the pagefile.sys file to one location: Control Panel -> System -> Performance -> Advanced -> Virtual Memory -> Change, remember to click "Set" for the changes to take effect;

f. Right-click the "My Computer" icon on the desktop, choose "Properties" -> "Advanced - Performance Settings" -> "Advanced - Virtual Memory", adjust to 330-720. And clean periodically.
