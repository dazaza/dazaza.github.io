Microsoft: Win8 system sales exceed 100 million, Windows Blue to be released within the year

by geekzhang on 2013-05-07 16:17:46

According to a May 7 report in Beijing time, Harry McCracken, a well-known contributing writer for Time magazine, recently quoted Tami Reller, co-head of Microsoft's Windows department, revealing that the sales volume of the Windows 8 system has now exceeded 100 million copies. At the same time, Reller also clearly confirmed the news that the next generation of the Windows Blue operating system will be released within the year.The following is the main content of the article:Compared with other major tech products, it is usually difficult to sum up the actual performance of Microsoft products with just hard data. It is certain that a large number of PCs equipped with the new Windows operating system will flow into the market, but this figure cannot tell us the true voice of consumers.However, Tami Reller, co-head of Microsoft's Windows department, recently provided answers to several consumer concerns in a Q&A section on the official Microsoft blog.Reller revealed that since the official launch of the Windows 8 operating system on October 26 last year, the company has sold a total of 100 million system licenses, which means that the sales speed of Windows 8 is basically consistent with the previous Windows 7 (the previously released Windows 7 also achieved a sales volume of 100 million copies within six months).For the achievements of Windows 8, some people, considering the continuous decline in PC sales in recent years, might think that such market performance is already quite good. Meanwhile, others believe that due to the significant increase in the number of existing PCs in the market compared to 2009 when Windows 7 was launched, the sales speed of Windows 8 should have been better than its predecessor.I said a long time ago that the sales volume of Windows 8 in the first few months after its release could not explain the situation. If Windows 8 ultimately succeeds, it would mean that Microsoft's bold changes to this product were effective and successfully helped the company avoid being marginalized in the post-PC era. If Windows 8 does not achieve the expected results, it can only mean that the innovations made by Microsoft to this generation of operating systems were not recognized by consumers. But no matter what the final conclusion is, the real feedback from the outside world on this operating system will at least take six months to surface.Undoubtedly, like all other operating system platforms, Windows 8 also needs a large number of excellent applications to highlight its outstanding interface design. Reller stated that since the release of Windows 8, the number of applications in the Windows App Store has increased sixfold and surpassed the total number of iOS applications in the first year of the Apple App Store's release. Currently, the total download volume of applications in the Windows App Store has exceeded 250 million times.However, Reller has never disclosed the exact number of applications currently in the Windows App Store. But an independent website called MetroStoreScanner estimated that the number of applications in the app store should be around 68,000. If this predicted data is accurate, it would well explain why Reller has been reluctant to disclose the exact number of applications in the Windows App Store, because there were reports indicating that Microsoft hoped to have 100,000 applications within three months of the release of Windows 8.Of course, the number of applications is not the most important factor, because application quality is something Microsoft should be more concerned about.Currently, Microsoft has admitted that it is developing a Windows 8 upgrade system code-named Windows Blue. In this regard, Reller did not provide much information about the new features of Windows Blue, but he clearly told us that the system will be released this year.We believe that Microsoft is most likely to officially launch this system in the fall, because this will give the company enough time to prepare for the Christmas shopping season, so we believe that Microsoft will soon announce more information about this operating system, especially the issue of whether the Start menu and button will return, which many users are concerned about.
