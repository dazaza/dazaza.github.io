Zhengzhou floor heating thermostat TH-208 geothermal heating LCD thermostat

by tehongbiyxg on 2012-03-05 11:30:52

**English Translation:**

The Zhengzhou underfloor heating thermostat TH-208 is specifically designed for electric underfloor heating systems, hydronic underfloor heating, infrared heating films, heat pumps, and other similar systems. This thermostat uses microcomputer control technology to compare the indoor temperature with the set temperature, and through a relay, it controls the start and stop of the heating equipment to achieve constant indoor temperature and energy savings.

**Features:**

- Large LCD screen display with blue backlight and user-friendly operation interface

- The sensor of the Zhengzhou underfloor heating thermostat is encapsulated in copper, ensuring high reliability

- Suitable for floor cables, electric heating films, manifolds, and radiator heating systems. It can also be used in sweat rooms, insulation rooms, or other environments requiring temperature control.

- Power failure memory function: after power resumes, it runs according to the parameters set before the power failure

- Automatic regulation of indoor temperature, with adjustable temperature hysteresis

- Single/dual sensor options, anti-freeze function, temperature hysteresis, and temperature calibration can all be selected by users in menu mode

- High-performance imported microcontroller chip with strong anti-interference capability and FLASH program storage

- Transformer-powered, ensuring stable and reliable performance

- Relay as output control device, offering high reliability and practicality

- Clock display function with 5+2 programming capability

- Low and high-temperature protection functions (optional)

- Infrared remote control function with optional remote controller

**Technical Parameters:**

- Power supply voltage: 220~240VAC, 50/60HZ

- Timing error of the Zhengzhou underfloor heating thermostat: <0.1%

- Load current: 32A (2×16A), 16A

- Temperature control element of the Zhengzhou underfloor heating thermostat: NTC

- Temperature control range: 10℃~35℃

- Temperature measurement range of the Zhengzhou underfloor heating thermostat: 0℃~99℃

- Dimensions: Standard 86-box installation

- Shell material of the Zhengzhou underfloor heating thermostat: PC + ABS (flame-retardant)

- Wiring terminals: Maximum wire size 4mm² soft/hard wire
