4. Remove forgotten phone number binding from QQ Mobile.

In the contact information displayed in QQ profile settings, the digits in the middle of the bound phone number are replaced with * stars and cannot be viewed. To see them, it's very simple: Log in using TM mode (in the QQ menu, click 'Use TM', and if it's not installed on your computer, it will automatically prompt you to download and install it). Then click "Menu", select "Wireless QQ - Wireless QQ Guide", choose the first option "QQ Free Binding/Unbinding - Mobile Phone/Little Smart", next step, input the phone number, hit Enter.

5. Prevent QQ from running its own process TIMPlatform.exe.

Let's talk about the function of this process: It prevents logging in with two identical QQ accounts on the same computer. (Of course, this is not the purpose of disabling it.)

Here's how to do it:

Start → Run → Input dcomcnfg.exe and hit Enter → Component Services → Computers → My Computer → DCOM Config → QQMainCreator Component → Right-click and select 'Properties' → Security tab → Launch and Activation Permissions → Customize → Edit → Select "Deny" for all user permissions under the account used in "Group or User Name". This will save 2M of system memory, which is the goal. Note: This does not work on Windows9X systems.

7. Non-members can clone QQ friends for free.

First, locate your QQ installation directory, then open the folder of the QQ number you want to clone (premise: the QQ account must have been logged in before), find the file named user.db, finally copy it to the folder of the QQ number where you want to clone friends and overwrite the original file (you must close QQ during this process). When you log in to QQ, you'll see that the QQ friends have successfully been 'transplanted'. However, they haven't yet 'resurrected', and you still can't chat with them. Next is the most critical step: Open the friend manager, select the people you want to add in the friend list (press the ctrl key to select multiple people), then drag the selected friends to the strangers column, then go to the strangers column and drag them back to the friend list. Resurrection successful. From this idea, you can also use this method to retrieve lost friends from a stolen QQ account.
