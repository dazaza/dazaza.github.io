Raspberry IO 开源 —— 树莓派创新集散地

by geekzhang on 2013-08-15 12:03:21

在今年三月份的 PyCon 2013 大会中，PSF 为每个参会者提供了一个 Raspberry Pi，这是一个像信用卡一样大小的微型计算机，你可在上面使用 Python 编程语言。树莓派的强大功能和可移植性极大的吸引着一大群爱好者和学生。由此迸发出大量非常具有创新意义的思路和产品。

PSF 在提供完整完这份慷慨的礼物之后，他们已经和 Caktus 签订合约来帮助激发各种基于树莓派上的想象力。而 Raspberry IO 就是这么一个专门为树莓派和 Python 爱好者提供的网站，人们可从这个网站上学习和教授如何在树莓派上使用 Python 编程语言。

同时我们很高兴的宣布 Raspberry IO 也是一个开源的项目（Github）。我们编写了很多测试和文档来帮你使用这个网站。
