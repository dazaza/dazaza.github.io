3. Method to prevent viruses caused by auto-play from USB drives

(Thanks to netizen: 獨自探戈 for providing) Since several friends replied to me saying they don't understand many of the following steps, today I will tell everyone that some steps can be skipped because even if all are used, the ultimate goal is still to immunize against Auto-class viruses. In fact, just the third and fourth steps are enough.

When using USB drives, external hard drives, etc., in a computer, every time they are plugged into the USB port, they will automatically play, especially when external hard drives have multiple partitions, each partition needs to be manually closed, which is very troublesome! Sometimes you will find that the drive letter icon has been modified. This is due to the automatic playback function triggering the virus in the flash drive. Is there a way to prevent automatic playback from running? The virus problem caused by the automatic playback function is the most common situation encountered in recent years. While lamenting that viruses are more ferocious than tigers, I also hope everyone can prevent problems before they happen. This article introduces nine commonly used methods to turn off the automatic playback function and immunize against Auto-class viruses, which you can refer to.

Method 1: Shift key

This method has been in use since Windows 98. Just hold down the Shift key while inserting a mobile hard drive, and the automatic playback function will not be started.

Method 2: Group Policy Prevention

In the "Computer Configuration → Windows Configuration → Security Settings → Software Restriction Policies → Other Settings" of the Group Policy, right-click to add a path and filename, such as: g:\*.exe can prevent exe files from running automatically.

Method 3: Group Policy Turn Off

During the peak popularity of the Panda Burning Incense virus, someone already began to use "Group Policy" to turn off the automatic run function of mobile hard drives or flash drives. The specific method: in the "Start → Open" dialog box, type "gpedit.msc" to open the "Group Policy" window. Under the left window's "Local Computer Policy", expand "Computer Configuration → Administrative Templates → System", then in the "Settings" title under the right window, double-click "Turn off AutoPlay". In the "Settings" item, select the "Enabled" checkbox, and then click "All Drives" in the "Turn off AutoPlay" box.

Method 4: Immunization Program Method

Download the Autorun immunization program, copy the immunization program in the compressed package to the root directory of the flash drive, run the program, and it will automatically generate an undeletable autorun.ini folder in the flash drive, so that the automatic playback virus cannot invade.

Method 5: Service Turn Off Method

In "My Computer", right-click and select "Manage". In the opened "Computer Management", find "Services and Applications → Services", then in the right pane, find the "Shell Hardware Detection" service. This service provides notifications for automatic playback hardware events. Select "Stop" in the "Status", then modify the "Startup Type" to "Disabled" or "Manual".

Method 6: Disk Operation Method

This method is mainly used in the Winows XP operating system, with excellent results. The specific method: open "My Computer", right-click the drive letter where you want to disable the automatic playback function, select "Properties", in the pop-up window select the "AutoPlay" tab, here users can set different operation methods for five types of content: "Music Files", "Pictures", "Video Files", "Mixed Content", and "Music CD". Choosing "Do Not Perform Any Action" will disable the automatic run function. This method can apply to all drive devices (hard disk, flash drive, mobile hard drive, optical drive).

Method 7: Registry Method

In the "Run" box, type "regedit" to open the registry editor, expand to HKEY_CURRENT_USER → Software → Microsoft → Windows → CurrentVersion → Policies → Explorer, in the right window find "NoDriveTypeAutoRun", this key determines whether to execute the autorun function of the CD or hard disk partition.

Each bit represents a device, different devices are represented by the following values:

Device Name Bit Number Value Device represented by the following value Meaning of Device Name

DRIVE_UNKNOWN 0 1 01H Unrecognizable type device

DRIVE_NO_ROOT_DIR 1 0 02H Drive without root directory

DRIVE_REMOVABLE 2 1 04H Removable drive

DRIVE_FIXED 3 0 08H Fixed drive

DRIVE_REMOTE 4 1 10H Network drive

DRIVE_CDROM 5 0 20H CD-ROM drive

DRIVE_RAMDISK 6 0 40H RAM disk

The value "0" indicates that the device runs, "1" indicates that the device does not run. As long as the value of the non-automatic drive is changed to "1".

Method 8: Software Method

Tweak UI is a software produced by Microsoft, which can be used on any Windows operating system. Using Tweak UI software, you can control whether all drive letters have the auto-play function.

Method 9: Virus Early Warning Method

Write the same autorun.inf file in the root directory of the drive letter, and place an icon ico file in the root directory, the file content is:

[autorun]

icon=Name.ico (name is the image name)

If one day you find that the drive letter's icon has changed, you may have been infected with a virus.
