New features of the HTC Desire after the upgrade to the official Android 2.2 system

by dream2012 on 2012-02-22 14:03:56

HTC launched the official Android 2.2 update for its HTC Desire smartphone in the Asia-Pacific region via OTA on the last day of August, making the Desire one of the few Android phones currently available that runs the official 2.2 system. In addition to the Android 2.2 system, the update released by HTC also includes the latest version of the HTC Sense user interface, greatly improving the functionality and performance of the phone. However, many Desire users are unaware of the new features and improvements after the upgrade, so we will summarize them to help Desire users better use their phones.

1. Overall System Improvements:

- Faster system operation

- Enhanced WiFi and 3G signal reception

- Longer battery standby time compared to Android 2.1

- Support for ChromeToPhone function, allowing you to send web links or map links directly to your phone through Chrome or Firefox browsers

- Search function supports finding software in the Android Market

- Increased earpiece volume

- Reduced boot-up sound

- Vibration prompt when shutting down, significantly reducing shutdown time

- Screen now supports rotating to the right

- New process manager displays the names of the latest 8 running applications (previously 6) when holding the Home key

2. Newly Added Applications:

- HTC App Share

- News & Weather

- Video (for viewing recorded videos)

- Voice Search

- WLAN Hotspot (to share the phone network with other devices via WiFi)

- Flashlight (use the flash as a flashlight)

- The Album app has been renamed to "Media Library"

3. Newly Added Widgets:

- HTC Calendar widget

- FM Radio widget

- Android Market widget

4. System Settings Menu:

- New "Portable WLAN Hotspot" option and settings added under Wireless and Network settings

- New "Display" settings added

- New "Voice Input and Output" settings added

- Application Management now includes four tabs: Downloaded | Only Operational Applications | All Applications | SD Card

- New sound settings allow setting screen lock and unlock prompt tones

- New sound settings allow simultaneous setting of ringtone, media, and alarm volume under the "Volume" option in one interface

- Removed user privacy settings and backup to SD card options

5. Keyboard Input Method:

- Added Chinese handwriting input (high recognition rate)

- Keyboard supports quick language switching

- Supports batch deletion of user-defined dictionaries

6. Browser:

- Integrated Adobe Flash 10.1 support, but must select "Enable Plugins" in settings to use normally

7. Calendar:

- Integrated Facebook events and friend birthday reminders into the calendar

8. Camera/Camcorder:

- Smoother screen rotation animation when taking photos

- Video recording supports H.263, H.264, and MPEG4 encoding modes

- Video recording supports MPEG4 720p HD video recording

- 720p HD video recording does not support zoom function

- Flash can be turned on and off at any time during video recording

9. FM Radio:

- Supports RDS (Radio Data System)

10. SMS Program:

- Larger input box at the bottom

- Displays the number of characters you can still input while sending messages

- New send button

- New add attachment button

- More fluid scrolling and searching when there are many messages

11. Album Program:

- Smaller picture sharing button, all image processing options can be displayed in landscape mode

12. Gmail Program:

- Tab colors start matching the web version

- Always shows your email address in the top right corner of the screen

- If the sender's email address is in your contacts, it will display a small icon indicating this

- New buttons added to view the previous and next emails

- Can copy email content via the Menu option

13. Google Talk Program:

- Entirely new application interface

14. Android Electronic Market Program:

- Supports setting automatic app upgrade function, added one-click upgrade for all apps function

- App reviews displayed under separate tabs

- Can set any user review as: Useful, Not Useful, and Spam Content

15. Music Player:

- Automatically uses a green music symbol if the playing music does not have an album cover

- Smoother album cover transition when selecting the next song

- Higher usability of music genre function

- Faster pop-up speed when pressing the left Menu key on the music playback interface

- Optimized random play function, avoiding continuous playback of the same song previously
