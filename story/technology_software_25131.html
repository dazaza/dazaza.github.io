Windows Vista SP1 test version debuts this week

by yinyin on 2007-07-18 20:50:06

According to an e-mail Microsoft sent to developers the other day, the Windows Vista SP1 beta will be released later this week and will not be later than July 29. On that day, Microsoft will release its financial report.

This is consistent with previous reports. There was news last week that Microsoft would release the Windows Vista SP1 beta this week (starting from July 16).
