天然矿物哈克曼石使 X 射线成像的新方法成为可能

2021-07-31 04:09:22

图尔库大学的研究人员发现了一种基于天然矿物哈克曼石着色能力的 X 射线成像新方法。国际研究人员小组还发现了 hackmanite 在暴露于 X 射线时如何以及为什么会改变颜色。 Hackmanite 是一种在黑暗中发光的天然矿物。它还因其在紫外线照射下将颜色从灰白色变为粉红色或紫色的能力而闻名。在最近发表的研究中，研究了这种矿物在 X 射线照射下的着色能力。 “之前的研究已经发表了关于 hackmanite 在暴露于 X 射线时变色的方式的初步结果，但尚不清楚为什么以及如何形成这种颜色，以及该特征是否会以某种方式有用。我们是第一个尝试 X 射线的人使用由 hackmanite 制成的表面作为图像板进行成像，结果非常令人兴奋，”图尔库大学的 Docent Mika Lastusaari 说。由 Lastusaari 领导的智能光子材料研究小组在图尔库大学化学系对具有光和颜色相关特性的材料进行了开创性的研究。该小组合成地创造了hackmanites，这使他们能够通过在材料的基本结构中添加或替换原子来控制材料的特性并使其适应不同的用途。 “在这项研究中，我们在德国卡尔斯鲁厄同步加速器设施中使用产生具有可调能量的 X 射线的粒子加速器研究了四种类型的合成 hackmanite 在暴露于 X 射线时的行为。所有样品都是积极的惊喜，并且我们能够获得有关在基本结构中添加不同原子如何影响着色能力的宝贵信息，”博士生 Sami Vuori 解释说。 “我们还注意到，颜色变化的机制是通过 X 射线激发内壳电子而发生的，这与紫外线辐射不同，紫外线辐射只会导致外壳中最松散的电子参与着色，”Vuori 补充道。据研究人员称，由于其多种用途，hackmanite 可以被认为是一种奇迹材料。现在，该矿物的使用范围还将包括 X 射线成像，这已通过蚂蚁尸体的 X 射线图像得到证实。 “使用 hackmanite 进行成像的优点是它不需要昂贵的分析工具来查看图像：您可以用自己的眼睛看到图像并用相机记录它。例如，这张照片是用普通的 DLSR 相机经过 X 射线照射后。不过，它最大的特点是照片中的钢片不是一次性使用的胶片：可以借助光或热来清除图像，并且可以拍摄新物体用同一部电影成像，”伊莎贝拉·诺尔博博士解释说，她的博士论文是关于黑客的。

该研究已作为开放获取文章发表在 Advanced Optical Materials 上。更多信息：Sami Vuori 等人，使用变色哈克曼石检测 X 射线剂量：机制和应用，先进光学材料（2021 年）。 DOI：10.1002/adom.202100762
