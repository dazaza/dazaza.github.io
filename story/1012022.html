消息来源：自特朗普2018年签署秘密命令给予中情局更多自由从事此类活动以来，中情局对伊朗、俄罗斯等国进行了网络攻击

2020-07-15 21:53:27

据直接了解此事的前美国官员透露，自2018年特朗普总统签署了相当于全面授权的秘密胜利以来，中央情报局(Central Intelligence Agency)已经对伊朗和其他目标进行了一系列秘密网络行动。

这项被称为总统调查结果的秘密授权，让间谍机构在开展的行动种类和目标对象方面都有了更大的自由，取消了前几届政府实施的许多限制。这一发现使中情局更容易授权自己的秘密网络行动，而不是要求该机构获得白宫的批准。

与之前总统的调查结果专注于特定的外交政策目标或结果-比如防止伊朗成为核国家-不同，这一指令由国家安全委员会(National Security Council)推动，由中央情报局(CIA)起草，更广泛地关注一种能力：网络空间的秘密行动。

一位前美国政府官员说，这一“非常咄咄逼人”的调查结果“给了该机构非常具体的权力，让他们能够真正对少数敌对国家采取攻击性的战斗”。据另一名前官员称，这些国家包括俄罗斯、中国、伊朗和朝鲜-文件中直接提到了这些国家-但这一发现可能也适用于其他国家。“白宫想要一辆车来反击，”第二名前官员说。“这就是做这件事的方式。”

中央情报局的新权力并不是为了收集情报而进行黑客攻击。取而代之的是，他们为该机构发动攻击性网络行动铺平了道路，目的是造成破坏-比如切断电力或通过在网上倾倒文件来危及情报行动-以及破坏，类似于美国和以色列2009年的Stuxnet袭击，那次袭击摧毁了伊朗用于浓缩铀气体的离心机，用于其核计划。

这一发现使中情局更容易破坏对手的关键基础设施，如石化工厂，并参与俄罗斯黑客和维基解密(WikiLeaks)流行的那种黑客和倾倒行动，即大量被盗的文件或数据被泄露给记者或发布在互联网上。它还解放了该机构，使其可以对银行和其他金融机构等以前基本上被禁止的组织进行破坏性行动。

据前官员称，这一发现的另一个关键变化是，它降低了证据要求，这些要求限制了中情局对据信代表对手的外国情报机构的媒体组织、慈善机构、宗教机构或企业等实体，以及与这些组织有关联的个人进行秘密网络行动的能力。

一位前官员告诉雅虎新闻：“在此之前，你需要多年的信号和几十页的情报才能证明这件事实际上是政府的一个分支。”现在，“只要你能证明它隐约看起来像是慈善机构在代表那个政府工作，那么你就是好人。”

中央情报局在行使特朗普执政期间赢得的新自由方面不失时机。据这位前官员说，自从两年前签署这项调查结果以来，该机构已经进行了至少12次愿望清单上的行动。“这既有破坏性的东西--东西着火和爆炸--也有数据的公开传播：泄露或看起来像泄露的东西。”

一些中情局官员对这一新发现表示欢迎，认为这是一项必要的改革，使中情局能够更灵活地采取行动。另一位前美国官员说：“(当协议签署时)人们正在走廊里做后空翻。”

但包括一些前美国官员在内的批评人士认为，情报监督的削弱具有潜在的危险，据前官员称，这可能会产生意想不到的后果，甚至会危及人们的生命。

美国情报机构参与黑客攻击和倾倒活动也让一些前官员感到不安。一位这样的前官员表示：“我们的政府基本上正在变成该死的维基解密(WikiLeaks)，(使用)在黑暗网络上与持不同政见者、黑客和转储进行安全通信。”

中情局拒绝置评或回应雅虎新闻(Yahoo News)的一系列问题。美国国家安全委员会(National Security Council)没有回复记者的多份书面置评请求。

尽管美国中央情报局多年来一直在推动扩大其网络当局，但俄罗斯干预2016年大选导致奥巴马官员抓紧寻找新的方式来报复克里姆林宫。据前官员称，高层讨论包括建议中央情报局在网上丢弃令人尴尬的俄罗斯官员被黑客攻击的信息，以及摧毁俄罗斯的服务器。

但据前官员称，距离2016年夏末发起行动只有几天时间，情报人员被告知退出。据一位前高级国家安全官员称，这样做的决定是在奥巴马政府的最高层做出的。

在特朗普政府上台初期，情报官员曾希望总统会批准这些行动。但一位前官员表示，特朗普高级官员对报复俄罗斯干预选举不感兴趣。“这是无线电静默，”这位前官员说。“一切都烟消云散，一无所获。”

尽管立即对俄罗斯进行网络报复的计划消退，但在特朗普执政期间，关于扩大中情局网络权限的讨论继续加速。据前官员称，多年来，中情局一直受到一些情报官员认为的网络空间秘密行动繁重障碍的困扰，这些障碍甚至阻止了中情局提出许多行动。

当谈到秘密行动时，“(中情局内部)总是有两个阵营，”在中情局服务了24年的罗伯特·伊廷格(Robert Eatinger)说，他曾担任过中情局的首席律师。伊廷格回忆说，“有些人觉得自己束手无策，有些人觉得这些限制是明智和适当的。”伊廷格表示，他对特朗普签署的中情局网络调查结果一无所知，也不愿讨论他在中情局任职期间发生的具体事件。

在特朗普政府领导下的这些辩论中，主张加强网络当局的人占据了上风，这鼓励中央情报局(CIA)扩大其先前的当局，以开展更具侵略性的网络行动-特别是针对伊朗的行动。一名前情报官员表示：“特朗普希望将决策推向尽可能低的分母。”

迈克·庞皮欧(Mike Pompeo)在特朗普2017年1月任命他为中央情报局(CIA)局长后明确表示了这一点。这位前官员说，庞皮欧传达的信息是：“我们不想耽搁你们，我们想搬家，搬家，搬家。”

一位拒绝讨论美国政府具体行动或政策的现任高级情报官员称，特朗普时代对攻击性行动的兴趣是“惊人的”。这位现任官员说，中央情报局(CIA)、国家安全局(National Security Agency)和五角大楼“在过去几年里一直能够像我们应该玩的那样玩。”

2018年4月，约翰·博尔顿(John Bolton)被任命为国家安全顾问，这再次提振了那些寻求放松对网络运营限制的人。博尔顿在他最近出版的回忆录“发生的房间”中写道：“我们需要废除奥巴马时代的规则，取而代之的是一个更灵活、更快捷的决策结构。”他写道，其中一部分涉及加强美国政府在网络空间对抗“非国家行为者”和其他人的“秘密能力”。

2018年9月，博尔顿宣布特朗普签署了一项总统指令，放松了奥巴马时代管理军事网络行动的规则。尽管政府披露了这一指令的存在-被称为国家安全总统备忘录13-军事网络行动的基本交战规则仍然是秘密的。政府还对中情局的调查结果保密，这赋予了中情局新的权力。

中情局新的网络权力引发了一些官员的担忧。一位前官员表示：“特朗普上台后，纠正过头了。”这位前官员表示，过去经过国家安全委员会严格审查的秘密网络行动，在制定和执行之间有时会有长达数年的间隔，现在“从想法到批准只需几周时间”。

前官员拒绝透露中情局根据这一发现开展的网络行动的细节，但他们表示，中情局已经针对伊朗和俄罗斯进行了秘密的黑客攻击和倾倒行动。

例如，一位前官员说，中央情报局在网上发布了一家表面上独立的俄罗斯公司的信息，该公司“为俄罗斯情报机构工作”。虽然这名前官员拒绝透露更多细节，但BBC俄罗斯台(BBC Russia)在2019年7月报道称，黑客侵入了SyTech公司的网络，窃取了约7.5TB的数据。SyTech公司确实为俄罗斯国内间谍机构俄罗斯联邦安全局(FSB)工作；来自那次黑客攻击的数据被传递给了媒体机构。

在另一次令人震惊的黑客攻击和倾倒行动中，2019年3月，一个不明身份的组织在互联网聊天平台Telegram上发布了据称参与黑客行动的伊朗情报官员的姓名、地址、电话号码和照片，以及伊朗情报人员使用的黑客工具。同年11月，与伊朗伊斯兰革命卫队有关联的三家伊朗银行客户的1500万张借记卡的细节也被泄露在Telegram上。

虽然消息人士不愿透露中情局是否是伊朗入侵事件的幕后黑手，但调查结果将中情局当局的目标扩大到金融机构，如泄露银行卡数据的行动，这代表着美国网络行动的显著升级。据前几届政府官员称，在前几届政府中，财政部高级官员成功地反对泄露或抹去银行数据，因为这可能会破坏全球金融体系的稳定。一位前官员说，“中情局一直都知道这些行动是一种选择，但它们总是一座太远的桥梁。”“他们在高层被放荡了很长一段时间，但头脑冷静的人总是占上风。

根据前官员的说法，新的网络发现进一步鼓励了中央情报局针对伊朗的行动。甚至在特朗普签署指令之前，政府官员就已经在鼓励中央情报局(CIA)积极解读先前存在的与伊朗有关的秘密当局，以帮助起诉政府对德黑兰的“最大压力”行动。特朗普的高级国家安全官员以击退苏联的冷战战略为灵感，认为在其境内破坏伊朗的稳定将迫使该政权停止在国外的冒险主义，甚至可能崩溃。

前官员说，最大压力的行动包括惩罚性的经济制裁，但也涉及中央情报局(CIA)对伊朗基础设施的网络攻击。“很明显，破坏稳定是针对伊朗的计划，”一名前官员表示，特朗普政府官员急于让中央情报局(CIA)对该国境内的目标进行破坏性的网络行动。博尔顿“想要另一个工具，他想要另一个锤子。他关注的是Stuxnet，以及如何刻薄对待伊朗，所以这可能对他很有吸引力。“另一位消息人士说。

据前官员称，特朗普政府能够依靠广泛的法律权力，对伊朗采取已经记录在案的秘密行动，包括至少可以追溯到21世纪初致力于反扩散的总统裁决-换句话说，防止一个拥有核武器的伊朗。据美国前官员说，另一个长期关注伊朗的总统调查结果授权中央情报局对抗德黑兰在中东的影响力，特别是通过打击伊朗的伊斯兰革命卫队，以及支持该地区反对该政权的组织。

根据前官员的说法，这两个与伊朗相关的发现，以及新的网络发现，都没有提到政权更迭是一个明确的目标。然而，他们说，随着时间的推移，中央情报局(CIA)和其他国家安全官员对前两项伊朗调查结果的解读越来越广泛，秘密活动从狭隘地关注阻止德黑兰的核计划演变而来。一位前官员说，伊朗的调查结果受到了“典型的任务蠕变”的影响。

据前官员说，不得不不断挫败伊朗核进展的疲惫逐渐导致美国官员采取了更具侵略性的方式，开始类似于政权更迭战略。另一位前官员说，这种想法变成了“如果我们能影响政权，那么就没有炸弹”。“我们在玩语义游戏--不稳定在功能上等同于政权更迭。这是一个否认的问题。“这位前官员说。

据前官员称，虽然中情局的新权力扩大了该机构针对伊朗和其他外国对手的能力，但它们也带来了潜在的陷阱。据前官员称，中情局和五角大楼长期以来一直在争夺网络空间的权力，这些协调问题现在只会变得更加关键-特别是当美国军事特工在不知不觉中与中情局的同行发生冲突时。

一位前高级军事情报官员表示：“如果你在某人的网络上做一些事情，而你的网络上也有友好的部队，你不会想要自相残杀的。”据前官员称，即使在美国情报界，中央情报局也以保密著称。另一位前官员说，中央情报局的“去冲突很糟糕，他们没有让人们了解他们的网络行动是什么。”

一些前官员甚至担心中情局内部对网络行动的监督。一位前高级官员表示，中情局网络特工对他们的活动“并不总是透明的”。“这是个问题。有几次我很惊讶。“。

这种更加宽松的环境也可能加剧人们对中情局保护其黑客武器库能力的担忧。2017年，维基解密公布了一大批中情局黑客工具，名为“Vault 7”。中情局一份部分解密的评估报告称，这起泄密事件是“中情局历史上最大的数据丢失事件”，该评估称，中情局最高黑客部门的“可悲的松懈”安全做法导致了这一泄密事件的发生。

2015年退休的前中央情报局首席检察官伊廷格表示，对他来说，目前还不清楚新的网络发现是回到20世纪80年代中情局更随心所欲的日子，还是会走得更远。他说，不管是哪种情况，这都是一件“大事”。

根据伊廷格的说法，取消NSC对秘密行动的监督是与近代历史的重大背离。他说：“我认为情报界和军方是一样的，因为重大决策应该由文官控制--对谁开战，对谁发动攻击，对谁打一场特别的仗。”“对于像秘密行动这样敏感的活动，你会有这样的文职或非情报文职领导层，这是有道理的。”

无论如何，这些扩张的新网络权力可能会成为特朗普政府的持久遗产，巩固中情局长期以来渴望在一个关键领域发挥的更大作用，并为中情局提供它渴望的三届总统政府的权力。

“人们以为，‘嘿，乔治·W·布什会签署这份文件，’但他没有，”一位前官员说。中情局官员随后相信，“奥巴马将签署该协议。”那他就没有。“。

“然后特朗普进来了，中央情报局认为他不会签字，”这位官员回忆道。“但他做到了。”
