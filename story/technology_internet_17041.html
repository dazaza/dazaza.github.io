Practical Uses of Mobile WAP Internet Access

by foxbank on 2007-04-21 07:59:18

Mobile Internet is a mobile phone-based Internet service launched by China Mobile. It is like the INTERNET built on computers. When browsing on a mobile phone, it's not ordinary www websites but specially designed WAP sites suitable for display on mobile phones that users access. You can log in with your mobile phone and enjoy services such as news information, gaming entertainment, ringtones, and pictures anytime and anywhere. Generally, mobile companies have already activated this service for all users (Note: If you haven't subscribed to a monthly package, please be cautious about using mobile Internet because the cost can be exorbitant).

Since blogs can now be read on mobile phones, I also subscribed to the monthly service the day before yesterday. Unfortunately, it will only take effect on May 1st. But that's good too; I won't get bored on my way home, hehe!

If you feel bored during your travels during the May Day holiday, why not try WAP browsing?

This blog provides two addresses for reading BLOGS on mobile phones:

The first one is http://www.foxbank.cn/wap.asp (this one seems to be outside the monthly data package and is calculated separately. However, KUJU commented on the Moonlight Blog that when the mobile Internet settings use the 10.0.0.172 proxy for www websites, all website visits will go through WAP traffic. I haven't tried it yet, so please be aware of this point when using this URL).

The second one provided by Feed is http://wap.feedsky.com/foxbank (this is a true WAP site within the monthly data package).
