Windows 8.1 to be launched on October 17 as XP retires

by anonymous on 2013-08-20 14:49:39

Last week, Microsoft officially announced that its latest operating system, Windows 8.1, will be officially released on October 17th, Beijing time. This includes online activation of the Windows Store to provide free updates for Windows 8 users and simultaneous offline sales.

In addition, starting April 8, 2014, Microsoft will cease providing hotfixes, product updates, and security patches for Windows XP users. This means that Windows XP, which has been in operation for 12 years, is not far from "retirement."

Microsoft pushes hard for Windows 8

Recently, in order to better promote Windows 8, Microsoft again announced that it will stop supporting Windows XP 500 days later, meaning no more updates after next April. Furthermore, starting at 7 PM Beijing time on October 17th, Windows 8.1 will begin offering free updates to Windows 8 users globally through the Windows Store on Windows 8 devices.

In response to an interview with Information Times, Microsoft stated that Windows 8.1 has made improvements in many areas, such as personalization upgrades, IE 11, Bing-based search functionality, improved built-in applications (including various new apps), enhanced Windows Store experience, and users will enjoy more convenient SkyDrive cloud connectivity. Business users can use the Windows 8.1 Enterprise Preview to test Windows 8.1 deployments in their own environments.

Microsoft admitted that for individual PC users, due to reliance on old operating system habits, how many people are willing to upgrade remains unknown. At the same time, industry experts pointed out that in the domestic personal PC market, piracy has always been the biggest "variable" affecting the market share of Microsoft's new operating systems. To put it another way, on one hand, piracy, due to its low price, unintentionally acts as a "pioneer" for the popularization of Microsoft's new operating systems; on the other hand, it causes significant impact on Microsoft's sales.

The retirement of XP will have the greatest impact on domestic enterprise users

According to incomplete statistics, there are still approximately 500 million PCs running the Windows XP system. Once the system stops receiving updates, users will no longer receive patches or security vulnerability updates.

For many individual PC users, upgrading the operating system mainly involves changes in usage habits and different user experiences; but for enterprise users, it will involve multiple aspects including data migration and system equipment. Moreover, if enterprises continue to use the Windows XP operating system after Microsoft stops updating it, there will be significant security risks.

An enterprise user interviewed by the journalist pointed out that they are fully willing to accept the update of the operating system and believe that the new operating system will have a higher security coefficient. However, it will take time for enterprise technical personnel to adapt to a new operating system. In this regard, some insiders also pointed out that many enterprise users are actually quite lazy. If the new system isn't easy to learn, regardless of how good it actually is, they are unwilling to spend time learning it.

From this perspective, Microsoft's biggest enemy is itself. Its previous products, whether Windows XP or Windows 7, already have large markets, and users can use and accept them well. Now, they need to re-accept a completely different version of Windows 8. "Unless it offers an excellent user experience, otherwise, it’s really hard to say," the aforementioned person said.
