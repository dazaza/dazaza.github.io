Windows 11屏幕截图泄漏，显示新的开始菜单等

微软即将推出的Windows 11操作系统今天在线泄露。在截图中首次发布于中文网站百度之后，整个Windows 11 OS在线出现，配有新的用户界面，开始菜单和更多。

新的Windows 11用户界面和开始菜单看起来非常类似于Windows 10x中最初找到的内容。在取消此项目以支持Windows 11之前，微软一直简化了Windows for双屏设备。在视觉上，您可以在任务栏中找到最大的更改。 Microsoft在此处居中了应用程序图标，清理了纸盘区域，并包含一个新的开始按钮和菜单。

此更新的开始菜单是Windows 10中当前存在的简化版本，而没有实时瓷砖。它包括固定的应用程序，最近的文件以及快速关闭或重新启动Windows 11设备的能力。它真的比Windows 10今天存在的更简单。

Microsoft还在Windows 11中使用圆角。这些在上下文菜单中可见，以及应用程序和文件资源管理器。开始菜单本身还包括圆角。这仍然是泄露的Windows 11的早期版本，所以尚未包含所有内容。

您可能已经注意到Windows 11任务栏中的一个新图标，它在操作系统中标记为窗口小部件。谣言建议Microsoft将带回Windows小部件，此早期版本似乎包含一些。

由于这显然是Windows 11的早期版本，而OS中的小部件不会完全加载，但它们设计用于滑出并提供快速访问新闻，天气和其他Web内容。

虽然Microsoft取消了其Windows 10x操作系统，但该公司显然将与Windows 11一起使用的大型部分.Windows 10x最初为双屏设备设计，然后转向传统笔记本电脑然后被取消。 Windows 10x包括许多细化和简化的方面到Windows，大部分工作都存在于此泄露的Windows 11中。

Windows 11还包括新的Snap控件，您可以从所有应用程序上的最大化按钮访问。它们多年来在操作系统中存在的级联Windows函数的现代等同物。您可以在桌面上的部分中快速快速捕捉Windows。

不幸的是，尚未对Windows商店进行大量更新。因为这是一个早期泄露的版本，但无论微软的更新都是规划的，它可能还没有出现。

Windows Store界面在很大程度上与Windows 10中存在的内容相同，可以访问今天可用的所有相同应用程序。

据报道，微软在最近几个月的Windows新的App Store工作，谣言表明它将从今天存在的情况下变大。 Microsoft CEO Satya Nadella承诺将“为开发人员和创造者解锁更大的经济机会”，这可能是一家新商店的形式。

据报道，Microsoft批准了其Windows应用商店，允许开发人员提交任何Windows应用程序，包括浏览器，如Chrome或Firefox等浏览器。 Microsoft还可以考虑在应用程序中允许第三方商务平台，允许开发人员避免剪切Microsoft通常从应用内购买系统中获取。

在其他方面，Microsoft还包括在Windows 11中的新设置体验。它与Windows 10x，新硬件的步行用户或通过一组配置Windows的步骤升级到Windows 11非常相似。此外，此外的经验还包括一个新的Windows 11启动声音，然后在每个启动时触发触发器。

Microsoft还在提高Windows 11中的Xbox体验。新的Xbox App现在集成到Windows 11中，提供快速访问Xbox游戏通过游戏，Xbox网络的社交部分和Xbox商店。 Xbox游戏栏和Windows游戏模式所有在此早期泄露的构建中都保持与Windows 10相同。

微软一直在删除暗示它已准备好启动Windows 11.软件巨头正在持有一个特殊的Windows活动，以便于6月24日揭示其下一个OS。该事件以11AM eT开始，事件邀请包括一个窗口，该窗口创建一个阴影，其中概述看起来像第11号。Microsoft Execs也一直在削减几个月的“下一代Windows”公告，而且现在泄漏了这一泄漏该Windows 11将于本月晚些时候正式透露。
