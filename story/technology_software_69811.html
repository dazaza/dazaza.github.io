Magic Shield EQSecure is a Hips-type software. This kind of software can achieve free control over various system operations by formulating corresponding rules. Only the programs and operations on the computer that are allowed can be executed; those that are not permitted cannot be executed, thereby preventing all kinds of harmful programs from intruding and running, achieving the purpose of protecting the computer system. Its difference from antivirus software lies in: antivirus software plays a protective role by detecting and eliminating known viruses, but has no effect on newly emerged unknown viruses, showing a lagging nature; whereas Hips-type software achieves its protective function by controlling program operation actions, not relying on virus databases, and can prevent unknown viruses and other harmful programs. Especially in today's prevalence of various Trojan programs and rogue programs, traditional antivirus software seems powerless, thus the protective role of Hips-type software becomes increasingly prominent.

Magic Shield EQSecure is a domestically produced software with outstanding functions among this type of software. It currently includes three aspects: application control, registry control, and file control. Application control includes 12 aspects such as execution of applications, loading of library files, loading of drivers, access to physical memory, access to physical disks, installation of services, etc., indicating very detailed control; Registry control includes control over modifications (new creations) and deletions of registry items and values; File control includes control over the creation, opening, modification, and deletion of files and folders, with very comprehensive control.

For a virus to achieve its goal of intrusion and execution, it must first be able to copy its own file to the hard drive of the computer, then realize automatic execution through methods such as writing to the registry, and will not be intercepted when running. As long as one of these steps fails, the virus cannot succeed. The protection of Magic Shield EQSecure is carried out simultaneously at three stages, forming an integrated three-dimensional protection. In theory, as long as the rules are set comprehensively and strictly enough, it can prevent all known or unknown viruses from intruding.
