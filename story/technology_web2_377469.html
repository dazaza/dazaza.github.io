Some net disks that support direct drag-and-drop upload using HTML5 technology

by anonymous on 2011-01-08 09:33:19

With the continuous maturity of HTML5 technology, more and more applications are adopting HTML5 technology. One of the attractive functions of HTML5 is the ability to drag and drop files directly from the desktop for uploading, which is very convenient. Several overseas cloud storage services have already adopted this technology. Here, I will introduce a few of them. Of course, to use this feature, your browser must support HTML5 technology.

1. Let’s Crate

Let’s Crate is an overseas cloud storage service that uses newer technology, allowing you to quickly upload files by dragging and dropping (but it requires browser support; it's recommended to use Google Chrome). It is simple and easy to use. You don’t need to register; just open the page and drag the file you want to upload to the wooden box in the middle of the page. After completion, it will generate a file sharing link. It will also prompt you to register an account. If you don't want to register, then copy the sharing link, but it can only be used for 30 minutes, equivalent to temporary sharing. It's recommended to register. After registration, you can see the number of folders you’ve created, the total number of uploaded files, and the available space. Currently, free accounts can use up to 200MB of storage.

Website: http://www.letscrate.com/

2. Ge.tt

Ge.tt is a file storage and sharing service with a simple and elegant design, offering fast upload speeds. Its biggest feature is extremely simple and convenient file upload, storage, and sharing—just like its concise domain name, the Ge.tt page has only one prominent button for uploading files.

It provides unlimited web space. Free users can retain files for 30 days. It supports batch uploads by selecting multiple local files at once, eliminating the need to add files one by one. You can upload up to 30 files at a time. After uploading, the system will provide a super-short external link address for sharing, as well as options to share on Facebook and Twitter with one click. When your mouse hovers over the link, you can copy it to share with your friends.

In addition, you can obtain additional features by registering, such as real-time download count statistics and longer file retention times.

Website: http://www.ge.tt/

3. Min.us

Min.us offers a completely new experience for online image storage, browsing, sharing, and management. The interface is very simple. You just need to drag and drop images onto the Min.us page, and they will automatically upload. If you want to upload multiple images at once, simply select them with your mouse and drag them to Min.us. To try this feature, your browser must be Firefox 3.6+, Chrome, Safari, or IE9. Otherwise, you will be directed to a specified file upload page. After uploading, Min.us will automatically complete the process and generate an album link for easy sharing with friends. There is an "Edit" link for editing purposes and a "Viewer" link for sharing. Below each image, there is a "Link to image" for conveniently embedding the uploaded image on any website or blog.

Currently, Min.us supports the following image formats: JPEG, JPG, GIF, PNG, APNG, BMP. It allows uploading single images under 7MB, with no more than 50 images per album. Uploaded images are permanently stored.

Website: http://min.us/

4. PicsEngine

PicsEngine is an online image management and storage software, similar to the desktop version of Picasa. You can also think of it as a cloud storage service focused on image storage. PicsEngine makes photo management very convenient and easy, providing intelligent album management using tags. Its overall interface design is very cool, offering unlimited storage space and access from anywhere.

PicsEngine has a powerful and new feature: you can directly drag and drop photos from your desktop to the browser for uploading. Of course, to experience this feature, you need to install a higher version of your web browser.

Website: http://www.picsengine.com/
