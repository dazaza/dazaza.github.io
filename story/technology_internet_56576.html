A Big Collection of Classic Tricks for the Intelligent ABC Input Method!

by kkxxbb on 2008-03-24 15:14:41

I believe many netizens are loyal fans of Smart ABC, so I will introduce some practical input techniques about it for everyone, which will greatly increase your typing speed.

1. Directly Input English

If you want to type English words in the smart input method, just type "v" before the English letters you want to enter. For example, to input "teacher," simply type "vteacher."

2. Input Capitalized Numbers

To input capitalized numbers like "一", "二", "三" or "壹", "贰", "叁" in Smart ABC, use the shortcut keys "i" and "I". For example, to input "三", just type "i3"; to input "贰", type "I2" (hold "Shift + i" to input "I").

3. Input Chinese Date

In Smart ABC, quickly inputting year, month, and day is similar to other input methods, using "n", "y", "r" to separate "年", "月", "日". Just add "i" before the input. For example, to input "二○○七年一月十五日", simply type "i2007n1y15r". Using uppercase or lowercase "i" will yield different results.

4. Quickly Input Units

Smart ABC also provides quick unit input using the hotkey "i" with "i+unit abbreviation". For example, to input "厘米", just type "icm". To input "万", "千", "百", "十", etc., add "i" before their initial consonant. For example, to input "千", just type "iq".

5. Quick Input of Other Characters

Using the "v" key + number keys, combined with "page up" and "page down", can produce some special symbols. For example, typing "v1" produces 104 special symbols. You can try typing "v2" to "v9" for more unexpected symbols.

Stroke Shape Input

Smart ABC is not a pure phonetic input method but a combination of sound and shape. Therefore, if you add the stroke shape code of the first stroke of the character after entering its pinyin, you can quickly retrieve the character. The stroke codes represent: 1 horizontal, 2 vertical, 3 left-falling, 5 left hook, 6 right hook, 7 cross, 8 square frame. For example, to input the character "吴", type "wu8" to reduce the number of pages you need to flip through when searching, significantly narrowing down the search range.

Forced Memory Function

The forced memory function refers to forcibly adding frequently used strings to the user's vocabulary. If you only consider it as an extension of the word creation function, you're underestimating it. For example, to forcibly add the term "罗纳尔多" to your vocabulary, right-click on the input bar in Smart ABC, select "Define New Word" from the context menu. In the "New Word" field, input "罗纳尔多", then in the "External Code" field, input its English equivalent "ronaldo" (here we use an easy-to-remember English name as the code).

Later, you can input it by typing "uronaldo" (preceded by a "u") in the input bar, and the entry "罗纳尔多" will appear in the input bar. Press the spacebar to make it appear on the screen. Got it? Just add one extra "u".

Word-to-Character Determination

Word-to-character determination involves inputting single characters in the form of words. For example, if you want to input the character "杰", and this character is located further back, it would be troublesome to have to flip pages each time to select it. You can simplify the process using the word-to-character function. First, input "jiechu", then press the spacebar, and the word "杰出" will appear. Press the "[" key to input the character "杰", and press the "]" key to input the next character "出".

Vague Recall

During the input process, Smart ABC automatically remembers the user's input history. For terms used not long ago, you can recall them based on incomplete information using the simplest method, making it convenient to re-input previously used terms. To repeat a term just input, press the combination key Ctrl + - twice. For example, after inputting the term "天堂", press Ctrl + - twice, and the term "天堂" will be input again. This feature is very effective when the content being input is relatively simple and frequently repeated.
