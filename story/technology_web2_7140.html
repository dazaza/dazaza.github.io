Solor Stream Extractor, Let's Give It Another Try!

Previously, I've already introduced the Solor Stream Extractor, which is extremely useful. It can easily extract the real URLs of hidden media resources. However, after some time, I found that the main page was inaccessible. Later on, the site owner moved to a blog and provided some explanations. I forgot about it... but fortunately, the owner has now rebuilt Solor. Congratulations to Solor for its persistence! Let’s take a look at its features:

This tool can extract information about multimedia files in web pages, including formats such as wmv, rm, rmvb, mp3, wma, swf (newly added). Usually, when we download a multimedia file from a website, we can find the multimedia information by right-clicking --> viewing the source code. However, sometimes, some websites (such as Sina) encrypt their media download page source codes, making it impossible to directly obtain the download link. With this tool, we can still get the download link for these websites.

That said, I did notice some limitations. When extracting media information from aeeboo.com, only a Flash file was obtained. When extracting information from toodou.com, the result was: "Sorry, no media information was found on the page you were looking for. This might be due to a typo in the URL you submitted. Please correct it and try again." Similarly, when extracting information from 6rooms.com, the result was the same as with aeeboo.com.

However, most videos, especially those on major portals like Sina, are accessible. These large portals don't overly conceal file addresses; they just implement basic protections against regular users. Therefore, this tool is sufficient for most needs.

In conclusion, Solor remains a handy tool for extracting media URLs from many websites, even if it has some limitations with certain platforms.
