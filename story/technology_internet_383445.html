Ubuntu 11.04 New Edition Released, Interpreting a Beautiful User Interface

by xljgtx on 2011-03-04 22:11:51

The third Alpha pre-test version of Ubuntu 11.04 (Natty Narwhal) has been released as scheduled. In Alpha 3, Ubuntu 11.04 mainly focuses on a large number of improvements to the Unity user interface, bringing users a better experience in terms of details. In Ubuntu 11.04 Alpha 3, the Linux kernel 2.6.38 RC6 is used, the default office software has been changed to LibreOffice 3.3.1, the user interface uses Unity 3.6.0, the browser uses Mozilla Firefox 4.0 Beta 12, and components such as Transmission 2.13 and Upstart 0.9.0 are also fully included. Additionally, Ubuntu 11.04 Alpha 3 has fixed a large number of bugs.
