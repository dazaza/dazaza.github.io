Windows 搜索 4.0

by @yinyin on 2008-03-28 12:15:35

Windows 搜索 4.0 允许您执行即时搜索您的计算机。 Windows 搜索 4.0 有助于您查找和预览文档、 电子邮件、 音乐文件、 照片和计算机上其他项目。

Windows 搜索 4.0 中搜索引擎是由程序如 Microsoft Office Outlook 2007年和 Microsoft Office OneNote 2007年还使用 MicrosoftWindows 服务。 使用此搜索引擎和程序的内容索引以搜索特定程序中时获得即时结果。

Windows 搜索 4.0 包括以下改进： • 支持加密文件系统 (EFS)

• MicrosoftExchange 当您索引以联机模式, 电子并且有是没有本地缓存 (.ost) 上减少影响

• 支持索引联机委派邮箱

• 支持客户端 - 客户远程查询以共享索引位置

• 改进索引性能

• 用于 WindowsXP 快 previewer 更新

• 每用户策略设置

• Windows 软件更新的 Watson 错误
