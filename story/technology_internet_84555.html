Solutions and patch download for when fs2you cannot be opened or accessed.

by xloong on 2008-08-13 22:57:55

Remember the post I made 2 days ago about Packet Tracer v5.0 Official Edition, where I mentioned that fs2you couldn't be opened? At that time, it redirected to a page from China Unicom, so we assumed it had been moved by China Unicom. Later, I saw some news online saying that fs2you had been blocked by its DNS service provider. I don’t know why such an excellent cloud storage would be blocked - was it a misoperation?

I also found some patches online that can solve the issue of fs2you not opening or being inaccessible, and I'm sharing them for those who need it.

**Fs2You Upload/Download Patch v1.3**

This patch is used to add local resolutions for www.Fs2You.com related domains and will not cause any harm to your system.

- If you have already applied this patch, there’s no need to reapply it.

- You can close the window at any time to interrupt the patch operation.

- Fixes the error where you can access the site but cannot download.

- Written by vadera@pdahd.cn, SENG.

The patch has been updated to version 1.3, resolving issues in version 1.1 such as only being able to view but not download files.

- Updated to version 1.3, packaged as an exe program.

- Added a function to restore the hosts file.

- Recommended for everyone to reapply this patch. If you've installed an older version or another modified version, update it: first choose option 2 to clear the old version, then choose option 1 to install.

**New Features:**

- Added a feature to restore the hosts file.

- Resolves issues on some machines due to read-only permissions that prevent file modifications.

- Solves problems with other versions where manual restoration of the hosts file results in inaccurate installation detection when reinstalling.

- Compared to the modified versions, it retains 26 additional node servers, allowing web-based file uploads.

- Compared to version 1.1, it adds cache server resolution.

- After applying the patch, it restores the read-only attribute of the file, enhancing security!

**Nanopan Download:** Fs2You Upload/Download Patch v1.3

Friends who are familiar should realize that this fs2you patch is just a script to modify the hosts file. Since modifying the system's hosts file triggers a virus warning from Rising (antivirus), this is normal. I turned off Rising, applied the patch, and then checked the hosts file, which added a long list of static Fs2you-related resolutions. Subsequently, accessing http://www.fs2you.com/ worked fine. Those who need it should download it quickly~
