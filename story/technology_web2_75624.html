Web 2.0 Mini Review (8) - Image Sharing (Online Albums) | Web Travel Notes

by cpxxpc on 2008-06-26 09:06:08

Youpai [Brief Review]: A well-known domestic image sharing site, fully imitating Flickr in functionality, slogan -- come and see my photos, your photo sharing platform. [Submission]: Online upload (can set sharing permissions, albums to be included, groups, supports zip package upload), client upload (supports batch upload, screenshots and image editing, image browsing functions), browser right-click upload, album migration (Flickr, NetEase Albums, Fotolog), email and MMS upload, for more details see this link; monthly traffic 61M, supports jpg, gif, png formats. [Management]: Supports external links of various specifications of images and batch modification of images, provides short domain personal homepage (supports custom layout), user center management functions are complete and clearly laid out, supports albums (supports drag-and-drop management, relatively powerful management functions), collections (organize multiple albums into one collection, enriches the way photos are displayed), tags, archiving, groups (groups), friends, subscriptions, etc. [Features]: Blog-related services (sending images from Youpai to blogs, requires BlogAPI interface binding; sending images via email to Youpai simultaneously sends them to the bound blog); external reference code for albums (facilitates displaying images on websites and blogs); rich API extensions; comprehensive list of user-related addresses. (Currently, free users' external image links need to pass an audit first.)

Bababian [Brief Review]: A well-known domestic image sharing site, self-described as the best personal album in the Chinese language field, free albums, online albums. [Submission]: Online upload (can set sharing permissions, included collections and groups, supports bundled upload), client upload (supports batch upload), browser right-click upload (image capture upload), email upload (provides a dedicated image upload email); monthly traffic 50M. [Management]: Supports external links of various specifications of images, provides customizable domain personal homepage (supports custom templates, sidebars), user center management functions are complete and clearly laid out, supports collections (albums), keywords, groups, friends, subscriptions, etc. [Features]: Convenient acquisition of external image links; user center pages support customizable quick menus (including external links); blog-related services (sending images from Bababian to blogs, requires BlogAPI interface binding); rich API extensions and image sharing with other sites; account type settings; VIP account upgrades; list of user-related addresses.

Soupai [Brief Review]: A new and sharp image sharing site, slogan -- all about happiness; upload my photos and share with millions of photography enthusiasts. [Submission]: Online upload (can set sharing permissions, included albums), client upload (SouPai Image Management Expert, powerful features similar to Picasa, mainly includes local image management and image sharing, characteristic function -- face recognition), single file 5M; monthly traffic 69M (can increase traffic by spending points), supports jpg, gif, png, bmp formats. [Management]: Supports external links of various formats of images, provides customizable domain personal homepage (supports custom layout and background images, custom domain requires spending points), user center management functions are relatively complete, supports albums, tags, friends, subscriptions, etc. [Features]: Points system (use points to extend functions, for more details click here); external reference code for albums (facilitates displaying images on websites and blogs); relatively comprehensive list of user-related addresses. [Shortcomings]: During testing, image upload and website access speed were not ideal; image upload methods were not flexible enough; lack of API, blog, email upload and other related extension functions.

Fotolog [Brief Review]: A new and sharp image sharing site, self-positioned as an image blog, mobile blog. [Submission]: Online upload (single file 1M), client upload (green tool, supports batch upload), browser right-click upload, mobile and email upload (provides a dedicated image upload email), specific details see toolset instructions; monthly traffic 100M, total space 1G. [Management]: Supports external links of various specifications of images and online editing, provides short domain personal homepage (called mobile blog), user center management functions are relatively complete, supports collections (i.e., albums), tags, forums (groups), friends, subscriptions, etc. [Shortcomings]: Website interface appears slightly cluttered; during testing, frequent image upload failures occurred, making detailed evaluation impossible.

Paipaiyue Image Entertainment Platform [Brief Review]: Paipaiyue online network album, the network extension of the client software. [Submission]: Online batch upload, client upload (upload through the Paipaiyue client, supports local image drag-and-drop batch upload, resource manager batch-style right-click upload, one-click selective batch acquisition of online images, more functions see "My Blogging Tool (I) -- Paipaiyue Client"). [Management]: Supports external links of various specifications of images and online editing, provides short domain personal homepage, user center management functions are comprehensive and organized, supports categories, albums, forums (groups), friends, favorites, subscriptions, etc. [Features]: Flash albums, avatars, video uploads; regular users have external link click count restrictions (50,000, sufficient, watermark added for exceeding part), and do not support original image uploads (thumbnails, has little impact on small and medium-sized images), supports VIP members (membership system); although not as convenient and powerful as the Paipaiyue client, it still shows professional expertise under its simple interface as a network album, and does not fall behind professional network albums like Youpai.
