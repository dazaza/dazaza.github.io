III. Installation Instructions

1. CD Installation:

Use a burning software, select "Image Burn" mode to burn the ISO file. Before burning, please verify the accuracy of the file. Recommended burning speed is 24X!

Set the CD-ROM as the first boot device in BIOS. Select the first option to automatically install to the first partition of the hard drive. For brand-name computers with hidden partitions, it's recommended to install manually.

2. Hard Disk Installation:

If the CD has been burned, double-click the auto-play program in Windows, then select "Install WINDOWS".

If the CD hasn't been burned, use a virtual CD drive to load the ISO image in Windows, double-click the auto-play program, then select "Install WINDOWS".

Note: Please back up important data from the original system before installation. It's suggested to transfer personal files and back up hardware drivers in advance.

V. Common Issues

1. GHOST installation process breaks, showing the following prompt:

output error file to the following location: A:\ghosterr.txt

The cause is usually reading or writing data errors, which could be due to poor quality of the CD or the CD-ROM's read capability, or write errors on the hard drive.

2. Realteck HD Audio sound card front microphone and headphone interfaces cannot be used.

This is due to the new features of the latest official Realteck sound card driver, not the system. If this happens, go to "Control Panel" -> "Sound, Speech, and Audio Devices" -> "Realtek HD Audio Manager" -> "Audio I/O" -> "Jack Settings" -> check "Disable Front Panel Jack Detection".

3. After installing the system, it prompts that the genuine validation was not passed, with a star icon appearing at the bottom right.

This situation is mostly caused by incorrect system time. Correct the computer time and restart the computer, do not use any so-called cracking tools, as they cannot solve the problem.

4. What should you do if you encounter a blue screen or the system keeps restarting during installation?

Such situations are generally due to recognition errors during the installation of drivers by Windows. You can re-GHOST restore the system and press the [ESC] key when the driver selection screen appears, canceling the integration of all drivers to resolve the issue.

5. After GHOSTing, the entire hard drive becomes one partition.

This is generally due to a manual operation error in GHOST, restoring the partition image to the entire hard drive. Immediately shut down the computer and do not write any more data to the hard drive.

It is recommended to connect the hard drive to another computer as a slave drive and use relevant software to restore the partition table. If the data is important, seek help from a professional data recovery company.
