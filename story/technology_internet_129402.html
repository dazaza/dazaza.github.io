Do not disturb game time - NORTON Antivirus 2009 Game Edition

With the popularity of online games, the security problems of game players' game accounts and virtual property are becoming more and more prominent. When players choose a security software suitable for themselves, they usually hope that the security software can protect the system security while not affecting the normal game of players, because no one wants to enjoy the game in the virtual world. Suddenly a security software warning dialog box pops up or the game suddenly becomes insensitive due to an automatic security software scan. In that case, especially in the game with the enemy PK to interrupt, there will be a kind of want to hit the computer impulse, hey hey!

Norton AntiVirus Gaming Edition 2009 is Symantec's game-specific security software for gamers. Norton AntiVirus Gaming Edition 2009 (later referred to as Navg 2009) will automatically switch to "game mode" when the game player is playing, in "game mode", Navg 2009 will stop updating, warning, scanning and other activities. To ensure that the normal operation of the game is not affected. This "game mode" will be automatically activated when an application running in full-screen mode is detected, but users can also manually enable "Game Mode" through Navg 2009 from the right-click menu of the icon in the system notification area. After the "game mode" is enabled, if Navg 2009 finds a security problem, such as a virus detected, it does not display warning information and takes a direct approach to protect computer security.
