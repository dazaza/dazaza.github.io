Starting from the 20th, Microsoft implements genuine validation, netizens begin to crack the validation strategies.

by hophi on 2008-10-18 00:18:52

Starting from next Monday, users of pirated Windows XP Professional Edition may encounter a situation where the background color of their computer desktop turns black every hour... Yesterday, Microsoft China confirmed to reporters that starting from next Monday, Microsoft will simultaneously launch two genuine validation programs. After automatically updating and "residing" in user computers, these programs can reveal pirated software and repeatedly pop up warning dialog boxes. For users of pirated Windows XP, the program will also use the method of "blackening the computer desktop background color" for reminders. As early as 2006, Microsoft began implementing a genuine validation plan for different versions of Windows XP. In April this year, Microsoft also implemented genuine validation for Windows Vista.

For pirated users, it's annoying but they can't delete it

Microsoft China stated in its reply to the reporter via email yesterday that the newly launched genuine validation plan targets users of Windows XP Professional Edition and Office. Microsoft said that this is the first time genuine validation has been implemented for Chinese Office users, covering all users of Office XP, Office 2003, and Office 2007 suites.

It is understood that through Microsoft Update or automatic updates, the genuine validation program is available for computer users to choose to download and install. Once the validation program resides in the computer, after booting up, the desktop background of pirated Windows XP users will turn pure black. Although users can reset the desktop background, every hour, the desktop background will revert to black. Additionally, when logging in, pirated Windows XP users will see an interrupted login dialog box and a permanent notification and continuous reminder dialog box in the lower right corner of the screen, displaying messages such as "You may be a victim of software piracy."

The small program validating Office will also remind pirated users through pop-up dialog boxes. Thirty days after receiving these dialog box notifications, if pirated software continues to be used, visual markers will be added to the menu bars of Word, Excel, PowerPoint, and Outlook programs in the Office software.

After the validation program resides on user computers, due to the constant popping up of warning dialog boxes, pirated users are greatly annoyed. However, although pirated users can view the program using "Add or Remove Programs" in the "Control Panel," they cannot uninstall it.

Strong reactions, online exchange of cracking strategies

Regarding Microsoft's upcoming genuine validation action, netizens have shown great attention. Yesterday, on the relevant forums of major websites, "Microsoft Genuine Validation" was one of the hottest topics. Although Microsoft's new round of genuine validation has not yet started, some netizens have already begun enthusiastically discussing strategies to bypass or crack the validation on forums. The reporter noticed that some netizens expressed understanding and support for Microsoft's anti-piracy actions, while others believed that Microsoft's genuine software prices were too high and had some complaints about Microsoft's anti-piracy behavior, saying, "WPS Office from Kingsoft can be downloaded and used for free, so I might as well switch to Kingsoft." Other domestic Office software like Yongzhong Office and Tangzhou Office also entered the scope of discussion among netizens.

It is understood that Microsoft's genuine validation plan is carried out in phases, and mainland China users are no strangers to it. As early as 2006, Microsoft began implementing a genuine validation plan for different versions of Windows XP. In April this year, Microsoft also implemented genuine validation for Windows Vista. The genuine validation plan to be implemented next week has already been gradually launched in other countries and regions earlier this year.

Industry insiders revealed that Microsoft has recently intensified its anti-piracy efforts and promotion of genuine software, which could be described as "hitting with one hand and pulling with the other." Besides warning pirated users through the validation plan, Microsoft has also recently promoted the Office Home and Student Edition at a discounted price of 199 yuan (originally priced at 699 yuan), lowering the threshold for users to use genuine software and stimulating their enthusiasm to purchase genuine products.

"Blackening the computer desktop" is debatable

Microsoft stated that the black desktop background would not affect the functionality of the computer or cause shutdowns. For example, if you are writing something in Word and the desktop color suddenly turns black, it will have no impact on your operation, and running files will not be lost.

However, legal professionals expressed differing opinions on Microsoft's anti-piracy measures. Yang Lijiu, the chief lawyer of Shanghai Boyang Law Firm, said that software products are prone to piracy, and companies protecting their intellectual property through technical means is beyond reproach, such as many software programs protecting themselves by authorization or limiting installation times. However, Microsoft's genuine validation program can repeatedly turn the desktop background color of pirated users black, which, if it interferes with the user experience, seems a bit excessive.

Computer users generally set their desktop backgrounds to colors they like. But Microsoft's validation program keeps turning the desktop of pirated Windows XP users black, which Yang Lijiu sees as not only arbitrary but also troublesome for users who have to keep changing the color. "If Microsoft uses this minor inconvenience to implement some kind of 'punishment' on pirated users, it exceeds the general rights protection authority of enterprises."
