Step two: Applicable to Windows 98, Windows Me, Windows 2000, Windows XP.

Please go to the homepage of http://www.speedguide.net/.

Check if your Default Receive Window (RWIN) = 256960, whether it reaches the value of 256960.

At the bottom of the window, check Optimal settings, then click Apply changes.

Then reboot and test again with TCP/IP Analyzer.

The value of Default Receive Window (RWIN) should be 256960 (or close to it).

(If it's not 256960, this usually happens in Windows 98 because it lacks one program.
