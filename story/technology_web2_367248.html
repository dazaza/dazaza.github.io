Secure scanning of web pages online

by wodezzz on 2010-10-09 13:34:29

If you're afraid the pages you're browsing aren't safe, try this online web security service that will tell you if a particular page is safe after being scanned by more than a dozen site detection tools. Enter the website, enter the URL you want to detect in the input box on the home page, click "Scan Now" to scan, after completion, the website will display a report, if the result is CLEAN, it means security, while if the result is dangerous. The more CLEAN the safer, the less safe the vice versa. Gray UNRATED means no result, may be a website interface problem or has been blocked, can not get the result for the time being, you can click the small magnifying glass on the right to enter the website to scan separately. Of course, web security we can not completely rely on such online tools, as a reference.
