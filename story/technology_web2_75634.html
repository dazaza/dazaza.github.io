Yongshuo E Drive [Brief Review]: A well-known domestic network storage site, self-positioned as a professional data storage and exchange platform. [Upload]: Online upload (supports link URLs), single file 3M, supports exe format. [Sharing and Downloading]: Supports file (folder) sharing, download traffic limit of 20M every two hours, only supports website downloads. [Management]: Total space capacity 30M, provides a personal homepage with a dedicated domain (can set login password and visitor permissions, supports template replacement, setting announcements, friendly links, and custom areas), comprehensive and professional management backend, convenient operation interface, space supports folders (including subfolders, can set viewing passwords). [Features]: Supports paying users (can expand space, permissions, and obtain a second-level domain, details click); personal homepage supports online listing and download records, and can be personalized through the custom area; supports image slideshow previews and audio/video playback (supports external link playback). [Shortcomings]: Upload method is too单一; does not support ordinary file external links; free user capacity and single file restrictions are too rigid; lacks mechanisms for user interaction and sharing. [Note]: Free users must log in to their space at least once within two months, otherwise the space will be closed. (For a detailed review of this web disk's paying users, please see "Finding the Ideal Web Disk - Yongshuo E Drive".)

VDISK [Brief Review]: A professional online hard drive, claiming to be - China's largest professional online hard drive technology provider. [Upload]: Online upload, client upload and management (supports batch processing), single file 20M (limited to 100 files, saved for 30 days), supports exe format. [Sharing and Downloading]: Supports folder sharing (limited to 2), supports downloading via extraction codes and reference links (file reference validity 24 hours). [Management]: Total space capacity 100M, provides a personal shared homepage with a second-level domain (Storage Show, supports custom titles, template changes, setting announcements), neat management backend interface, complete functions, supports batch operations (file moving, deletion, referencing, folder sharing permissions - sharing password, anonymous access, system classification, etc.), space supports folders (subfolders, can set viewing passwords), searching, viewing friend resources. [Features]: Supports online multimedia file playback; supports anonymous extraction on the homepage; supports paying users (can expand permissions and obtain permanent external link support, etc., specific details can be found here). [Shortcomings]: Unstable uploading and downloading, occasionally unable to connect during testing (client also has this issue, and often appears frozen); recycle bin function is ineffective for free users, equivalent to a mere formality; restrictions on free users are too rigid, only suitable as temporary storage space, and short reference validity, equivalent to no external link support. (According to homepage promotion, free user permissions will soon be increased, total capacity 10G, unlimited single file size, but retention period remains 30 days, which has little significance for users requiring external links and long-term storage.)

Feng Pan [Brief Review]: A professional online hard drive, slogan - The network is storage; I have, I store, I share. [Upload]: Client upload (resembles a local hard drive partition, supports folder operations, breakpoint resume, proxy server, sharing settings, shortcuts, and other resource manager regular operations), single file 10M. [Sharing and Downloading]: Supports folder sharing (can set sharing passwords). [Management]: Total capacity 100M, provides a short domain personal homepage, complete and clearly laid out management center functions, space supports friends (Feng Friends, can conveniently share resources). [Features]: Integrated into the resource manager client program (convenient operation); complete user exchange and sharing mechanism; Jibao Basin section; rich and detailed help system; supports paying users (various types, main differences are total capacity and single file size, specific click here to check). [Shortcomings]: For a while now, registration has always been told "System debugging, Feng Pan temporarily suspends registration", don't know why, test account registration and login are also told "This Feng Pan user has been locked, cannot log in to the website", so it is impossible to conduct actual detailed tests; free user restrictions are too rigid, paying user prices are relatively high. (Currently still in the upgrade process, temporarily unavailable.)

Mike Box [Brief Review]: A feature-rich professional network storage site, slogan - Any time, any place, any file, Box, stored for you. [Upload]: Online upload (Flash, supports batch uploads, can set tags, sharing permissions, etc.), transfer accelerator upload (belongs to IE plugin, supports FTP method, faster and more stable), client upload (supports drag-and-drop batch uploads, breakpoint resume, automatic shutdown, specific click here to download), single file 10M, daily maximum upload of 5 files, does not support exe format, file validity period is 10 days if not downloaded (default file expiration period is 10 days, as long as the file is downloaded once within 10 days, it can be extended by another 10 days from the download date, and so on, achieving permanent storage). [Sharing and Downloading]: Uses extraction codes and extraction links for sharing, supports file sharing with friends, groups, systems, supports viewing shared resources from friends and groups. [Management]: Total capacity 500M, complete and clearly laid out management center functions, supports batch editing (moving, deleting, renaming), space supports folders, friends (can send invitations to increase points), groups (joining requires spending coins, creating requires reaching level) etc. [Features]: Supports anonymous upload; supports user levels and proprietary currency (coins, can expand space and permissions in hierarchical order, supports gifting, specific exchange methods click here to check), after upgrading to Mike-level user, will gain larger permissions (such as personal homepage, photo album, safe deposit box, etc., and no file size or upload quantity limits); attached network photo album substation (does not support external linking of images). [Shortcomings]: Many features (such as personal homepage) are restricted by coins and levels; free user upgrade methods are relatively singular; unstable connection during testing on Netcom lines.

Changyi Personal Storage Space [Brief Review]: A relatively feature-rich network storage site, slogan - Leading Internet storage solution. [Upload]: Online upload (supports link URLs), FTP upload, single file 80M, supports exe format. [Sharing and Downloading]: Supports file (folder) sharing, single-threaded, does not support breakpoint resume, only supports website downloads. [Management]: Total space capacity 180M (can increase space through points, up to 1G), provides a personal homepage with a dedicated domain (supports custom home page templates, site statistics, friendly links), comprehensive and professional management backend, supports batch operations (moving, deleting, sharing, adding to blog, etc.), space supports folders (subfolders), formatting, adding URL links, download statistics, etc. [Features]: Expand space and traffic through points; supports paying users (can expand permissions, space, and obtain a second-level domain, details click here); supports adding video files to blog (not tested); personal homepage supports blog posts, online photo albums, etc. [Shortcomings]: Unable to obtain FTP upload account; does not support file external links (uses dynamic IP, expires after 24 hours); free user capacity and single file restrictions are too rigid.

GBao Pan [Brief Review]: A professional online hard drive. [Upload]: Online upload (Flash method, supports batch uploads, can set tags, sharing permissions, etc.), single file 10M (permanent storage), supports exe and other formats. [Sharing and Downloading]: Supports file sharing (allows members to download, visitors have no download rights). [Management]: Total capacity 1G, provides a personal homepage with a second-level domain (supports categorized display, tag display), relatively concise and complete management center functions, supports batch operations (moving, deleting), space supports tags, friends (free users cannot save friend files), categorized search, etc. [Features]: Provides an independent download page (simple layout, supports dual-line downloads for Netcom and Telecom); image files automatically go into personal photo albums, supports mobile downloads; supports paying users (can expand space and single file size, supports permanent public external links). [Shortcomings]: Relatively singular upload method; website pages are overly concise, lack user-shared resource exchange sections (now established); blog (webpage) display plugins are invalid during testing; free user sharing permissions are strictly restricted, cannot perform external linking. (For a detailed review of this web disk's paying users, please see "Simple Yet Effective - Trial Use of GBao Pan Paying Web Disk Space.")

Mofile Network Hard Drive [Brief Review]: Mofile's subordinate network hard drive, self-positioned as - Network hard drive + Network USB drive, storage ~ sharing ~ communication. [Upload]: Regular upload, breakpoint resume upload (provides two methods, requires online installation of plugins), unlimited traffic, supports exe format. [Sharing and Downloading]: Supports folder sharing (files in this folder automatically have sharing attributes, can also manually set to not public), files only support website downloads and extraction code downloads. [Management]: Permanent space 160M (safe deposit box), provides a personal homepage with a dedicated domain (can change templates, set announcements, friendly links, etc.), management backend mimics the resource manager interface, supports batch operations (moving, deleting), space supports folders (can set passwords), friends, tags, etc. [Features]: Supports VIP paying users (details click here); supports dual-line servers for Netcom and Telecom; provides a Shared World special section for user sharing; provides auxiliary small tools such as online address books and favorites; closely integrated with Mofile videos and photo albums. [Shortcomings]: Does not support ordinary file external links; does not support batch file uploads; relatively high price for paying users; inherits Mofile's consistently chaotic sections.

91Files [Brief Review]: A simple interface, practical function network storage site, slogan - Practical, high-speed, stable, free. [Upload]: Online upload, single file 100M, unlimited traffic, supports exe format. [Sharing and Downloading]: Supports indirect external links, webpage (blog, forum) embedded codes. [Management]: Unlimited transit station capacity (files retained for 1 month), permanent space 100M, simple and practical management center functions, supports batch operations (deletion, locking). [Features]: Supports anonymous upload; supports paying users (permanent space 5G, single file 200M); provides dual-line servers; supports personalized logos on download pages. [Shortcomings]: Too singular upload method; does not support personal homepages, file categorization, tags, etc.; lacks mechanisms for user interaction and sharing; numerous and varied advertisements on file download pages.

AiCun [Brief Review]: A simple interface, practical function, and completely free network storage site. [Upload]: Online upload, single file 100M, unlimited traffic, supports exe format. [Sharing and Downloading]: Supports pseudo-external links, webpage (blog, forum) embedded codes. [Management]: Unlimited total capacity, simple and practical management center functions, supports batch operations (deletion, moving), additionally supports setting file (folder) sharing permissions, tags, sending files to the public area, space supports folders, bookmarking functions. [Features]: Anonymous upload; resource categorized sharing; management and download interfaces support image previews. [Shortcomings]: Too singular upload method; does not support personal homepages; unable to set file tags, categories, etc., during upload; tag function lacks practical value; numerous and varied advertisements on file download pages.
