Game Characteristics:

1. This machine is a super luxurious 8-person connected entertainment machine.

2. This machine integrates all the game characteristics of the animal park series and the magic hat.

3. This machine uses unique flashing lights and thunder sounds as prompts, doubling with lightning.

4. The betting multiplier for winning can flash and double or triple at random times.

5. This machine can adapt to local habits, freely converting between high and low multipliers.

6. The highest multiplier can reach up to 138 times (low multiplier) or 750 times (high multiplier).

7. This machine can remotely control the settlement of the machine's accounts, making account management both convenient and accurate.

Game Rules:

1. Sub-machine betting, single pattern, Guangzhou swing machine, maximum betting of 99 points, lowest multiplier of 2 times, highest multiplier of 46 times (low multiplier) or 250 times (high multiplier). The multiplier adopts a variable rate. Players can choose to bet on colors and animals based on the size of the multiplier or the number of color animals. If the color and animal pointed to by the pointer are the same as the color and animal that have been bet on, it is considered a win. The normal ordinary winning score is: single-grid betting score x winning color animal multiplier.

2. Lightning: During the game of this machine, there will be random lightning. When there is lightning, all the betting buttons on the sub-machines will flash like spotlights, with thunder and lightning sound double prompts. When the pointer stops on an animal, the player's winning multiplier may randomly double or triple. That is: the winning score is: single-grid betting score x the multiplier after the winning color animal doubles.

1. Host Board Testing

Startup Method: Open the front door of Sub-machine No. 1 (1P), press the Main Test (MAIN TEST) button, which will enter the manual test mode.

End Method: Press the MAIN TEST button during testing, or select the PE item in the test items, and press the red crocodile button.

Test Items: The project number will be displayed on the Credit LED, press the coin-in button to select the test item (any one of the eight sub-machines' coin-out buttons can be used).

P1: RAM/ROM (Read/Write Storage/Read-Only Storage) Test: Press the red button to start the test, and the result will be displayed on the Multiplier LED.

P2: Micro-switch Test: The status of all micro-switches currently set will be displayed on the Multiplier LED.

P3: Light Emitting LED Test Mode A: All light-emitting LEDs will light up in the order of green, red, orange, and can pause temporarily (PAUSE).

P4: Light Emitting LED Test Mode B: The orange LEDs will light up sequentially in a counterclockwise direction, and can pause temporarily (PAUSE).

P5: Reed Switch Test: Whether the operation of the reed switch is normal will be displayed on 11 LEDs, the red crocodile button, and the motor's ON/OFF.

P6: Motor Test

Red Button: DC motor ON/OFF (animal doll with outer ring)

Green Crocodile Button: Stepper motor ON/OFF (driving the pointer)

Pointer Position Adjustment: Press the red elephant button to move forward clockwise, press the green elephant button to move forward counterclockwise. Please adjust to the position where the front end of the pointer exactly lights up the LED.

After adjustment, press the green crocodile button to make the needle rotate and check its movement.

P7: Sound Test: Press the crocodile lion button to emit different sounds in sequence, press the green crocodile button to repeat emitting sounds in sequence.

P8: Counter (COIN COUNTER) Test: The token IN/OUT counters will each jump 5 times in sequence, press the red crocodile to pause temporarily (PAUSE).

P9: Communication Test: Press the red crocodile button to conduct the test, and the result will be displayed on the Multiplier LED.

PE: End of Test: Press the red crocodile button to end the test mode. Sub-machine board test.

2. Sub-machine Board Test

Startup Method: Open the front door of the sub-machine and press the TEST button to enter the manual test mode of that sub-machine. Each of the eight sub-machines has its own test button. During the main machine board test and the coin return device rotation process, the sub-machine test button does not work.

End Method: Press the TEST button during the test, or select TE from the test project and press the red crocodile button.

Test Items: The project number will be displayed on the CREDIT LED, first press the orange panda button twice, then press the red coin return button.

T1: RAM/ROM (Read/Write Memory/Write-Only Memory) Test: Press the red crocodile button to start the test, and the result will be displayed on the Multiplier LED.

T2: Micro-switch Test: The status of the currently set micro-switches will be displayed on the Multiplier LED.

T3: Digital Lamp Test: All digital lamps will display the numbers 0-9 in sequence simultaneously, and can pause temporarily (PAUSE).

T4: Digital Lamp Running Lamp Test: The digital lamps will display the numbers one by one on the LED from left to right.

T5: I/O (Input/Output) Test: Press any button, the corresponding number of the button will change from 0 to 1, detecting the connection and disconnection of the button micro-switch.

T6: Sound Test: Press the red crocodile button to emit different sounds in sequence, press the green crocodile button to repeat the sound emitted this time.

T7: Coin Return Device Test: Press the red crocodile button, tokens will be returned, 10 tokens will be returned. Press the green crocodile button, all tokens will be returned.

T8: Counter (COIN COUNTER) Test: The token IN/OUT counters will each jump 5 times in sequence, press the red crocodile to pause temporarily (PAUSE).

TE: End of Test

3. Main Machine Related Fault Information

Faults related to the main machine or central part will be displayed on the sub-machine panels (CREDIT LED) from 1P to 8P.

EA: RAM BAD - Bad RAM (Read/Write Memory), under certain conditions, you can purchase a similar type and specification RAM locally for replacement, otherwise return it to the original factory for repair.

EB: ROM BAD - Bad ROM (Read/Write Memory), under certain conditions, you can purchase a similar type and specification ROM locally for replacement, otherwise return it to the original factory for repair.

EC: BACKUP RAMBAD (Data Memory): Data memory RAM is bad, please delete the originally set BACKUP MEMOPY.

ED: DC Motor Fault - Fault with DC motor or sensor (SENSOR), or poor contact with wiring harness.

EE: Stepper Motor Fault - Fault with stepper motor or sensor (SENSOR), or poor contact with wiring harness.

EH: POWER DOWN - Power outage during gameplay, please open the front door of 1P and press the MAIN TEST button.

4. Sub-machine Fault Information

Faults occurring in each sub-machine will be displayed separately on the display panel of each sub-machine (CREDIT LED).

E1: RAM BAD - Bad RAM (Read/Write Memory), under certain conditions, you can purchase a similar type and specification RAM locally for replacement, otherwise return it to the original factory for repair.

E2: ROM BAD - Bad ROM (Read/Write Memory), under certain conditions, you can purchase a similar type and specification ROM locally for replacement, otherwise return it to the original factory for repair.

E3: BACKUP RAMBAD (Data Memory) - Bad data memory RAM, please delete the originally set BACKUP MEMOPY.

E4: Insufficient Tokens - Please turn off the sub-machine power and replenish the tokens.

E5: Token Jam - Please clear the jammed tokens and restart the power supply.

E6: Coin Return Device Fault - Too many tokens being returned or fault with the coin return device itself, please restart the power to resolve.

E7: Communication Fault - Please correctly set the micro-switches (SW1-2 communication numbers) on the sub-machine board.

E8: POWER DOWN - Fault occurs during multiple coin returns or no coins available, please open the front door and press the TEST button to return excess tokens.

C1: Communication Connection Standby - Communication connector dropped or not properly connected.
