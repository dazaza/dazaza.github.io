Samsung Galaxy S4 16GB model said to have only 8GB available

by geekzhang on 2013-05-04 13:08:29

Some users of the Samsung Galaxy S4 recently pointed out that the actual available storage space of this phone's 16GB version is only about 8GB. Samsung responded that this was due to the provision of more functions and a higher resolution display.

Users found that nearly half of the 16GB storage capacity of the Galaxy S4 was occupied by the operating system and built-in applications. After resetting to factory settings, the available storage capacity displayed for the 16GB version of the Galaxy S4 was 8.49GB.

In response to this issue, Samsung stated in a statement: In the 16GB version of the Galaxy S4, 6.85GB of capacity is occupied by the system, which is 1GB more than the Galaxy S3, in order to provide users with a higher screen resolution and more powerful functions.

It is currently unclear how much the 1080p resolution of the Galaxy S4 has affected the storage capacity. It can be certain that, in order to support a higher resolution, the graphical interface of applications will take up more storage space. Samsung also stated: In order to provide users with the ultimate mobile experience, the Galaxy S4 provides a microSD slot for expanding storage capacity.

Unlike HTC One and iPhone, the Galaxy S4 provides a microSD card slot for expanding storage capacity. However, Samsung does not allow users to install applications on the expanded memory card. This means that even if users can store more photos and videos, they still need to closely monitor how many applications have been installed. Some popular games, such as "Real Racing 3", require over 1GB of available storage space on the phone, so the 8GB storage capacity of the Galaxy S4 will soon be exhausted.

Some users expressed dissatisfaction with Samsung's approach on Facebook. One user commented: Samsung should label it as 8GB instead of 16GB. Some other users suggested that the price of this version of the Galaxy S4 should be reduced, and Samsung was suspected of false advertising. One user said: Personally, I think if you buy a phone with 16GB of storage, the available space for the user should be 16GB.

Some people believe that unnecessary built-in applications can be deleted through rooting the Android system to free up capacity. However, others argued that for a phone priced so high, rooting should not be done, as it would void the warranty or risk bricking the device.

Samsung is not the only company facing accusations of reduced product storage capacity. The 64GB version of Microsoft's Surface Pro actually has only 23GB of usable space, while the 8GB version of Nintendo's Wii U has only 3GB of actual usable space. (Weijin)
