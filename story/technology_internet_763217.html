Apache CouchDB Gets a Fresh Infusion, Integrating Distributed Technology

by anonymous on 2013-07-29 17:13:53

Cloudant, a database-as-a-service company, previously committed to merging its heavily optimized BigCouch database code into CouchDB, aiming to infuse new life into CouchDB. After months of development, the new merged version (temporarily also called BigCouch) is now available.

Apache CouchDB is a document-oriented database management system. However, since its founder Damien Katz abandoned the project in January 2012, it has become somewhat unstable. After leaving CouchDB, Damien Katz turned to Couchbase Server, a product that integrates CouchDB with Membase and Memcached technologies. However, this project is entirely unrelated to Apache, and some contributions from developers could not be directly incorporated into Apache CouchDB. Subsequently, many core team members of CouchDB also left, causing instability in the CouchDB community.

This code integration brings some characteristics of distributed databases to the CouchDB project, such as cluster management, higher concurrent access performance, and compression technology. Meanwhile, Cloudant has ceased development of the four-year-old version of the BigCouch project to fully participate in the new project.

The clustering behavior of the new version of the database is based on the concept of Amazon Dynamo. Each BigCouch node can accept requests, data is placed in partitions based on consistent hashing algorithms, and read/write operations apply the quorum protocol.

Jan Lehnardt, chairman of the Apache CouchDB Project Management Committee, stated that Cloudant's technology provides a complete strategy for replicating data across distributed systems in CouchDB. By integrating BigCouch's clustering technology, CouchDB's functionality becomes more complete. He welcomed more project developers to contribute based on this version.

The new version of CouchDB is now downloadable via the Cloudant website, and the official release will follow Apache's schedule.
