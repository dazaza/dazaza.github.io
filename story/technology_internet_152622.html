Windows 7 Logo leaked early?

by cennatian on 2009-05-23 16:02:03

First of all, on May 8 this year, Microsoft Beijing held a "Windows 7 RC initial experience" press conference for the news media, from the on-site layout and Microsoft distributed Windows 7 RC box CD, we saw the Logo of Windows 7 for the first time. However, Microsoft has not made it clear that this is the Logo that Windows 7 will adopt when it is officially released.

Two weeks ago, foreign websites exposed the design pattern of the retail box packaging of Windows 7, and we saw the same Windows 7 Logo again from the exposed Windows 7 box, which further confirmed the authenticity of this Logo.

Until yesterday (May 22), Microsoft China's official website has also shown the Windows 7 Logo, foreign IT websites have reported that the official Logo of Windows 7 was leaked in advance by Microsoft China's official website, but today this Logo seems to have been removed.

But regarding the Logo, some people think it is the top right corner of the "Windows" Logo, some people think it is just a simple "7", both sides seem to be reasonable, what do you think?
