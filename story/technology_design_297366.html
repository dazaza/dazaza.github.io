The Vehicle Communication Module (VCM) is a high-performance, ruggedized serial communications gateway designed for vehicle use. This device provides multiple vehicle serial communication interfaces to meet the requirements of all Ford Motor Company vehicles. The Ford VCM unit is housed in a magnesium enclosure with a tough, protective plastic cover for the PCMCIA expansion slot, which can be used for wireless LAN cards or memory expansion. The VCM device contains five LED status indicators to visually inform the user about the internal operations of the VCM, the vehicle, and the host device (e.g., NGS Classic's T-Handle, PC, Pocket PC/PDS, and Laptop/IDS). The VCM also has detachable cables for connecting to industry-standard high-speed host interfaces as well as to the vehicle under test. The VCM (Version V62) includes both IDS (Integrated Diagnostic Software) and PDS (Portable Diagnostic Software) tools. Diagnostic coverage includes most Ford, Lincoln/Mercury, and Mazda vehicles from 1996 MB Carsoft 7.4 Multiplexer ~ 2009 MY (produced 1995 elm327 ELM327 USB ~ 2009).

**Application Overview**

Integrated Diagnostic Software (IDS) is the next-generation diagnostic tool. IDS integrates WDS software modified to run on an off-the-shelf laptop with the Vehicle Communication Module (VCM) currently used in NGS+ and PDS, and a new Vehicle Measurement Module (VMM).

**Vehicle Diagnostic Connectors**

The VCM 16-pin DLC cable is used to connect the VCM to the vehicle. The VCM and VMM connect to the laptop through the Ethernet-USB Adapter Cable (H406/H416).

**Interface Devices**

The interface devices that are used with this application include:

- **VCM (Vehicle Communication Module):** This device provides all link-based functionality, including Data Logger, Self Test, Service Functions, Module Configuration and Programming, etc.

- **VMM (Vehicle Measurement Module):** This device provides the following functionality: Oscilloscope, autoboss v30, Digital Multi-Meter, Ignition System Test, vag com, Fuel System Test, and SGM.

Note that the application will detect the interface device connected to the laptop. Indicator icons will appear at the bottom right-hand corner of the Ford VCM IDS screen when the VCM and/or VMM are connected.
