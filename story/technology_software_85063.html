Rising releases Kaka Internet Security Assistant 6.0, offering 1 year of Rising Antivirus Software and Firewall for free upon installation.

by yinyin on 2008-08-15 19:24:57

Software Introduction: "Rising Kaka Internet Security Assistant 6.0" is a completely new anti-trojan software designed for the Internet. It has three major anti-trojan functions: "Trojan Download Interception", "Trojan Behavior Judgment and Interception", and "Automatic Online Diagnosis". Based on Rising's leading "Cloud Security" plan, it can effectively intercept, defend against, and detect various trojan viruses, and help users automatically scan and fix system and third-party software vulnerabilities, optimize the computer system, and is an essential security software for a wide range of Internet users.

New features added in Kaka Internet Security Assistant 6.0: 1. Trojan Download Interception: Based on the industry-leading anti-trojan technology, it intercepts infected computers from downloading more viruses and account-stealing trojans through the network, cutting off the channel for trojans to enter user computers, and effectively curbing the spread of malicious trojan viruses such as "Trojan Herd".

2. Trojan Behavior Judgment and Interception: Based on the powerful "Intelligent Active Defense" technology, when trojans and suspicious programs start or load, it immediately intercepts their behavior, blocking their account-stealing and other destructive behaviors. It discovers and clears trojan viruses while they are running, protecting the account security of QQ, online games, and online banking.

3. Automatic Online Diagnosis: The core function of Rising's "Cloud Security" plan. It automatically detects and extracts suspicious trojan samples from the computer and uploads them to Rising's "Trojan/Malware Automatic Analysis System" (Rs Automated Malware Analyzer, abbreviated RsAMA). Subsequently, RsAMA will provide analysis results back to the user, detecting and eliminating trojan viruses, and sharing them through the "Rising Security Database" with all other "Rising Kaka 6.0" users.

Enhanced features in Kaka Internet Security Assistant 6.0: 1. A brand-new "Vulnerability Scan and Settings" that automatically fixes operating system and third-party software vulnerabilities.

Using a newly developed vulnerability scanning engine, it intelligently detects Windows system vulnerabilities, third-party application software vulnerabilities, and related security settings, and helps users fix them. Users can also set up automatic fixes for these vulnerabilities, simplifying user operations and helping users promptly address security risks.

2. Powerful System Repair

For system settings damaged by viruses, such as the IE browser homepage being changed or frequently redirected to advertising websites, Kaka Assistant repairs the registry, system settings, and host files, restoring the computer to normal operation.

3. Robust Process and Startup Management

Helps users effectively manage drivers, startup software, and browser plugins in their computers, effectively improving the operational efficiency of user computers.

4. Advanced Toolset

Targeting experienced computer users, Kaka Internet Security Assistant 6.0 provides comprehensive practical functions: junk file cleanup, system startup item management, service management, network program management, LSP repair, file shredding, and special killing tools.

5. Seven Monitoring Systems Protect User System Security

Kaka Internet Security Assistant 6.0 has seven monitoring systems: automatic online diagnosis, USB virus immunity, automatic system vulnerability repair, trojan behavior judgment and interception, malicious website protection, IE leak prevention wall, and trojan download interception. It comprehensively protects the safety of user computers.

Other main features of Kaka Internet Security Assistant 6.0: 1. Scanning Rogue Software: Thoroughly scans and removes rogue software in the system using Rising's powerful anti-rogue software technology.

2. Detecting and Removing Popular Trojans: Completely eliminates over one million popular account-stealing trojans, ensuring user personal information security.

3. Cleaning Computer Usage Traces: Quickly helps users clean up internet traces and application software usage records, including: Windows usage traces, internet history records, audio/video playback software history traces, download software history traces, Microsoft Office history traces, and other commonly used software usage traces.
