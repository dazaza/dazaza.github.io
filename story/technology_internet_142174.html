Major improvements in the new version:

1. Support sending attachments, maximum 20M, resumable resumable

2. When installing and logging in for the first time, only a small number of recent emails are collected (no need to pay attention to old emails, which can save traffic costs).

3. Supports multiple emails for batch deletion

4. Copy and paste operations are supported

5. Support to display pictures in emails (still need to cooperate with the server, it is expected to be fully supported by the end of April)

6. Provide some option Settings

7. Automatically detect new versions for easy upgrade

8. Several BUG fixes

Installation mode

You need to download it to your mobile phone and install it. (If you have an earlier version, you will be prompted to install it in the same path, and the original data will be retained.)

What is QQ hand mail?

QQ Hand Mail is the latest mobile email client software product developed by QQ Mail, which is committed to providing stable, professional and convenient professional mobile email experience.

QQ hand mail compared with WAP mailbox, has the following characteristics:

1, better interface, better reading effect; 2, smooth operation, almost no delay waiting; 3, can be operated offline, do not worry about network disconnection; 4, save traffic, you can view traffic statistics; 5. Provide excellent experience unique to client software.

QQ hand mail compared with ordinary mobile phone mailbox software, has the following characteristics: 1, two-way synchronization with QQ mailbox, mobile phone local operation is also synchronized to the server side. Maintain consistent processing status on your phone and computer; 2, also support China Mobile CMWAP access; 3, simple operation, no complex Settings; 4, PUSHMAIL effect, automatically receive mail; 5, write email more convenient, email address automatically updated and completed; 6, can be opened all day, save electricity and traffic: data transmission with compression processing and text pre-pull technology, seamless segment pull, save traffic; 7. Secure data transmission, encrypted by public key algorithm; 8, maintain the reading effect of HTML mail, support quick viewing of picture thumbnails and document attachments.
