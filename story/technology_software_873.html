Camino 浏览器 1.0 正式版

by @geekzhang on 2006-02-20 00:21:55

年初的时候 Mozilla 推出了 beta 版本的 Camino 浏览器，近日正式1.0版已经推出，这显然是 Mac 用户遭到微软抛弃后很好的选择。该浏览器100%开源，运行于 Mac OS X 10.2 及更新版本系统之中，因为内核变化，Mac OS X 10.1以下版本只能使用0.8.4版。

　　据称新的 Camino 浏览器有很大的提升，主要表现在：

可运行在基于 PowerPC 和 Intel 的 Mac 中

更好用的 Tab Bar

包含中止/续传的下载管理

网页内容屏蔽

更多安全证书的支持

内置 Java 插件提升性能

访问历史搜索

支持 SVG、CSS2、CSS3、JavaScript1.6 等标准

　　使用 Mac 对我仍处于梦想阶段，希望有朋友试用 Camino 后把体会告诉我：）(via witao.com)
