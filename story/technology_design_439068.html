This allows them to monitor their children's specific situations, so they no longer need to worry unnecessarily about not understanding their children's daily lives. For kindergarten managers, adopting a remote monitoring system as an auxiliary tool for teaching management not only reassures parents to trust the school with their children but also demonstrates the school's management capabilities.

More importantly, the system records every moment of joy and laughter in the child's growth process. Installing a network video surveillance system in the kindergarten can not only improve the quality and efficiency of school management but also allow parents to see their children's learning and living conditions in the kindergarten through the internet, giving parents more confidence in entrusting their children to the kindergarten. III. System Design Plan

This system design is based on a network architecture video surveillance system that can utilize the existing local area network environment of the kindergarten or use ADSL broadband directly for the equipment, greatly reducing construction costs. Installing a wireless video server on the kindergarten school bus enables real-time transmission of front-end videos to the back-end monitoring center and allows easy remote video monitoring anytime and anywhere. Video materials can be recorded at any time for convenient retrieval and review. The system design fully considers scalability; when expanding the system, just add cameras and other front-end devices without affecting the normal operation of the system, realizing system expansion. Compared to traditional video surveillance, it has irreplaceable advantages.

The entire system is divided into two parts: the front-end subsystem and the monitoring center:

1. Front-end Subsystem

The front-end video acquisition system consists of wired and wireless network video servers, cameras (with pan-tilt-zoom and decoders). Among them, the wired network video server serves as the core device, responsible for uploading the video images collected by the cameras from various monitoring points after compression and encoding through the network to the monitoring center, achieving remote viewing of the video. Installing a wireless video server on the kindergarten school bus allows parents and the school to conduct real-time monitoring of the children inside the bus. At the same time, it can control classroom, dining hall, dormitory video switching, lens zooming, close-up/distant view, aperture adjustment, and control the pan-tilt movement up/down, left/right, and automatic patrol according to the commands issued by the monitoring center. Specific design includes:

(1) This system uses the wired network video server CRS4000LA and the wireless video server CRS1020C at the front end.

(2) Install PTZ cameras in the classrooms, dining halls, and dormitories of the kindergarten. Parents and the school can control the PTZ cameras via video surveillance software to achieve 24/7 monitoring of the children during school hours.

(3) Wireless vehicle-mounted video server: Set up a vehicle-mounted camera + wireless CDMA/EDGE network video server on the kindergarten school bus, transmitting the vehicle video in real-time through the CDMA/EDGE channel to the monitoring center. The system power supply uses vehicle battery power, cigarette lighter power supply, or backup lithium battery power supply methods to ensure stable operation of the mobile vehicle video surveillance system. Using a wireless channel to transmit images, the image quality reaches 8-12 frames per second, allowing timely understanding of the children's situation inside the school bus and recording the images transmitted to the monitoring center.

(4) The distance between the video server and the switch within the kindergarten campus should be within 100 meters per single Ethernet cable, depending on the layout of the kindergarten's network cables; Industrial and Commercial Bank of China, video servers connect nearby to the internet;

(5) The video server and the lens are connected via shielded coaxial cables, with a length of up to 300 meters. The video server controls the PTZ decoder through the RS485 serial port. The control line is recommended to use shielded twin-core wire, with an effective length of up to 1200 meters.

2. Monitoring Center

In the monitoring center room, configure three servers separately as the central server, storage server, and monitoring host to manage, store, and view videos from each monitoring point, as well as control the front-end cameras. Shanghai Surveillance Installation Company

(1) Install server management platform software and SQL2000 software on the central server to manage the entire system.

(2) Install storage server software on the storage server to record video materials from each monitoring point. The required hard disk space size is determined by the number of recorded videos, recording duration, retention cycle, image quality, frame rate, and scene complexity.

(3) Install monitoring software on the monitoring host to browse videos from each monitoring point and adjust the camera posture. It can also group and display the video feeds in a rotating manner.
