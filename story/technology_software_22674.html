"Random 8-digit number" virus manual removal methods and suggestions

by caisy on 2007-06-16 23:52:10

Recently, the Jiangmin Anti-virus Center has detected a virus using "image hijacking" technology rampaging on the Internet. The main characteristics of this virus are as follows: After the virus runs, it will generate an 8-character virus process composed of randomly generated numbers and letters, and it will attempt to close multiple antivirus software, firewalls, and security tool processes, making virus removal operations extremely difficult. Moreover, this virus will generate Autorun.inf and random 8-character EXE files in the root directory of each hard disk partition, allowing the virus to be activated by double-clicking the hard drive. Additionally, the virus spreads through USB drives, MP3 players, portable hard drives, and other mobile storage devices, greatly increasing its transmission speed.

The anti-virus experts at Jiangmin recommend paying attention to the following preventive measures during daily computer use:

1. Safeguard your USB drives, MP3 players, portable hard drives, and other mobile storage devices. When an external USB drive is connected to your computer, do not rush to double-click and open it. Make sure to perform a virus scan first. It is recommended to use antivirus software with USB virus immunity functions, such as KV2007's unique USB Shield technology, which can immunize against all viruses that run when a USB drive is double-clicked.

2. Install system patches, especially MS06-014 and MS07-017. Currently, most web-based malware infiltrates computers through these two vulnerabilities.

3. Keep your antivirus software virus database updated in real time, ensuring regular upgrades and scheduled virus scans.

4. Download and install software from legitimate websites to avoid installing packages bundled with Trojan or virus.

For users who have not installed antivirus software or whose antivirus software has become ineffective, it is suggested to manually remove the virus using the following methods:

1. First, download the IceSword tool online and rename it (for example, to abc.exe) to bypass the virus process blocking the tool. Then double-click to open the IceSword tool and terminate the process of an 8-character EXE file. Sometimes there may be no such process.

2. Use the file management function of IceSword to navigate to C:\Program Files\Common Files\Microsoft Shared\MSINFO\. Delete two 8-character random number files with extensions .dat and .dll respectively. Then go to %windir%\help\ and delete the same-named .hlp or .chm files, which are system help file icons.

3. Then, delete the Autorun.inf file and suspicious 8-character digital files from the root directory of each hard drive. Note: Do not directly double-click to open each hard drive partition; instead, browse using the tree directory on the left side of the Windows Explorer. Sometimes after a computer is infected, hidden files may not be visible. In this case, you can use the file management function of WinRAR to browse and delete files.

4. Use the registry management function of IceSword to expand the registry key to:

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options]

Delete the IFEO hijack item inside.

At this point, you can install or open antivirus software, upgrade it to the latest virus database, and perform a full-system virus scan.
