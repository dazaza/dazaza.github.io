APK 编辑器工作室 v1.5.0

2021-08-08 06:35:18

我很高兴地宣布 APK Editor Studio v1.5.0！距离上一个版本已经过去很长时间了，终于，包含许多新内容的重大更新发布了。最令人期待的功能之一，克隆 APK 的能力，终于来了！它允许您通过更改包名称在您的设备上安装 Android 应用程序的多个副本。目前，此功能处于实验状态，这意味着某些应用程序无法克隆或可能出现意外行为。开发这个功能需要很长时间，因为没有一致的算法来重命名反汇编的包。但是，已经有了一个良好的开端，我们希望我们的社区做出贡献。现在，您可以轻松浏览用于签署 APK 的证书列表。目前，签名查看器会验证以下签名方案： 使用特定参数运行 APK Editor Studio 以立即开始优化、签名和/或安装您的 APK。以前，如果您只是想对 APK 进行签名或优化，则必须等待解包过程完全完成。现在有类似于现有“安装外部 APK”操作的专用选项——“签署外部 APK”和“优化外部 APK”。 APK Editor Studio 现在允许您打开主窗口的多个实例。每个实例都重用了项目池和其他共享组件，因此将额外的系统负载降至最低。要打开一个新窗口，请按 Ctrl + N（在 macOS 上为 ⌘ + N）。在 Windows 和 Linux 平台上，您还可以打开多窗口模式，以防您希望在单独的窗口中默认打开后续 APK 文件。

在 Linux 环境中，应用深色主题会导致外观不一致，因为某些应用内颜色基于浅色 GUI。所以现在，APK Editor Studio 不再使用固定颜色值，而是检查系统主题并动态调整自定义应用程序颜色。修订元素：请注意，某些平台和环境不（完全）支持暗模式。例如，Windows 不会为 Win32 应用程序应用相应的系统设置，而 GNOME 环境可能需要额外配置才能使 Qt 应用程序遵循 GTK 主题。但是，我们计划在未来的版本中添加自定义的应用内深色主题。现在您可以指定第三方工具的相对路径。这对于 APK Editor Studio 可移植版本尤其重要，因为依赖绝对路径会降低应用程序的可移植性。此版本为您带来了许多新的翻译。其中一些语言是从右到左的，因此为了更好的用户体验，APK Editor Studio 会根据脚本方向正确放置 UI 元素。我们出色的社区为 APK Editor Studio 准备了 12 个新翻译。您可以在此处查看贡献者列表。谢谢！这个对 Windows 用户有用的新功能将 APK Editor Studio 操作添加到 APK 文件的上下文菜单中。它允许您直接从资源管理器中签名、优化或安装应用程序。此版本是在这些人的慷慨支持下发布的。感谢大家的贡献！
