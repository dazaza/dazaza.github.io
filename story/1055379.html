苏伊士运河的替代方案？

2021-04-01 20:07:38

1868年苏伊士运河开幕式在欧洲和亚洲之间创造了一条新的贸易路线，作为良好的希望，欧洲和亚洲的替代方案，但也有不同的路线认真考虑。

Suez和河流鸟舍河畔威廉亨利板（1847），IOR / X / 3205，印度办公室记录，英国图书馆的地图

对1798年的苏伊士的海峡斯的调查错误地显示了红海比地中海高8.5米，一个想法终于通过英国军官队长弗朗西斯·弗朗迪斯·罗登·克斯尼于1830年进行的更准确的调查.Chesney的建议然而，在欧洲州河上建立一个常驻蒸汽船服务，作为陆上陆路的一部分，将波斯湾与地中海联系起来，英国议会在确定幼牙期间的导航性方面投了20,000英镑的批准冬季。

报告推荐英国议会的蒸汽导航委员会（1834），IOR / L / MAR / C / 573，印度办公室记录，英国图书馆的幼牙探险

euphrates探险队的船员，由Chesney指挥，于1835年1月31日离开英格兰。也在船上，在需要时可以组装，是两艘铁蒸汽容器的探险将使用，名为Tigris和euphrates以纪念他们将穿越河流。 4月份抵达SoweDich [Samandać，现代土耳其]，然后在奥胡克斯陆上向Bir [Birecik，Turkey]举行，蒸汽机被组装，调查开始。

每个蒸汽船的尺寸和船员，来自Chesney的官方报告（1850），IOL.1947.c.142，印度办公室记录，英国图书馆

探险并非没有问题，包括东印度公司的最初不愿意参与计划的项目“没有[他们]参与或同意”（IOR / L / MAR / C / 573，F.29） 。奥斯曼官员拒绝了奥斯曼官员否定的重要段落或供应造成的各种延误，尽管已经从奥斯曼帝国政府获得，但英国涉嫌成为可能的俄罗斯影响力的故意阻碍主义。但是当蒸汽船陷入风暴中，最悲惨的挫折是出现在一个军官的话语中，“带来了最恐惧的速度，对我们带来了抨击”（IOR / L / MAR / C / 574，FF。 183-85）。欧余的船员能够将她固定到银行，但底格里斯被吹回河的中心，在几分钟内沉没，失去了20名男子。

探险队于1836年6月完成，皇家地理协会杂志宣布这一成功，陈述了“可以合理地寻求的一切，已经完成了”（伦敦皇家地理学会，Vol.7 （1837），第411页），虽然波斯湾的居民们写道，建立了奥胡庇氏的永久性蒸汽服务是“值得最深思的考虑”，并建议它可以加强英国已经占主导地位的方式 海湾（IOR / L / MAR / C / 574，FF。342-44）。 在19世纪40年代进行了进一步的探索，但在1854年的制剂中，开始建造苏伊士运河和地中海之间的官方陆路，而湾之际从未成为现实。
