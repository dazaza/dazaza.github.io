Tencent has recently made a lot of new moves, QQ2010 is still in Beta, and QQ concept version was born. Today, Tencent released the concept version of QQ Beta1.1 based entirely on the Windows7 operating system. The list of new features is as follows:

QQ Concept version Beta 1.1

Update Date: 2010-04-19

= = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = =

1. Personalize the login interface

Leisurely fluttering white clouds, swaying leaves in the wind, full of vitality.

2. Head motion picture frame

A variety of dynamic photo frames to add new interest to your profile picture.

3. Dynamic background

Fresh and natural bamboo leaves, vibrant neon lights, friends list is no longer monotonous.

4. Friends dynamic scrolling display

Personality signature, music status, QZone and other update information rolling display, anytime and anywhere to understand the friend's dynamics.

5. Desktop friends

Drag friends to the desktop, real-time attention to the latest status of friends, instant chat, to achieve seamless communication and interaction.

6. Friend management

Rich texture interface, friend card free drag and drop, support multi-touch, male to female ratio of friends statistics, give you a new friend management experience.

7. Multi-tab chat function

Drag and drop the avatar on the session window to merge and split the session window, making it easier to manage the session.
