Methods to log in multiple QQ accounts on a Mac (under MacOS)

by mac52ipod on 2008-05-28 21:14:22

QQ FOR MAC can only log in one QQ account on a single Apple computer. Below is a trick to log in multiple QQ accounts on one Apple computer.

It's actually very simple. To log in multiple QQ accounts on one Apple computer, you just need to place the QQ for Mac application package into different folders and run them separately.

Another method is to input the following in the Terminal: /Applications/QQ.app/Contents/MacOS/QQ &. The more times you input it, the more QQ for Mac instances you can run. This way, there's no need to copy files and occupy disk space.

The third method is to use Lump QQ and QQ for Mac simultaneously. This allows you to log in multiple QQ accounts on one Apple computer.
