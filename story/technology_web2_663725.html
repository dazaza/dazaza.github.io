Brief Introduction of Tianlan Window Screens: Invisible window screens are screens where the mesh can automatically retract. Below is an introduction to hardware invisible window screens and precautions when using them. Invisible window screens are screens where the mesh can automatically retract. They are mainly used for ventilation and mosquito prevention. The frame is tightly attached to the window frame. When needed, pull down the screen; when not needed, the screen will automatically retract into the mesh box. It doesn't occupy space and has strong sealing properties. It coordinates well with high-end home decoration. Working Principle of Roll-up Type: Through the roll-up mechanism, the mesh is stored. Opening Method: Manual, Electric, Beads Chain. Opening Direction: Vertical or Horizontal. Folding Type Working Principle: Through the pleats of the mesh (like an accordion), the mesh is stored. Opening Method: Mostly manual. Opening Direction: Vertical or Horizontal. 1. If the frame surface gets scratched, it can be cleaned with a Brite Pearl or sprayed with self-spray paint. 2. When not using the screen door/window, retract the mesh to avoid affecting the service life of the spring and the cleanliness of the mesh due to prolonged closure. 3. Avoid stepping on aluminum profiles forcefully to prevent damage. 4. The screen does not need to be removed for cleaning; pulling the screen can serve a cleaning function, or gently patting the mesh with a feather duster can also clean it. If the kitchen screen gets greasy, use a damp cloth dipped in dishwashing liquid to lightly wipe the mesh, then clean it with a damp cloth. Allow the mesh to dry before retraction. 5. For externally installed screen doors/windows, during windy or rainy weather, retract the mesh before closing the window. If the wind is too strong causing the mesh to blow out, insert the mesh back into the track before moving the screen door/window. During high winds, it is recommended to close the window partially or not use the screen at all. 6. When pulling the clamping strip, move it parallel along the track. If accidentally pulled off the track, tilt the clamping strip and place both ends back into the track, then gently move it a few times (pay attention to whether the upper and lower meshes are retracting smoothly). The screen door/window will return to normal without affecting future usage.
