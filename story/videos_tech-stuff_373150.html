Android 2.3 makes its first official appearance with Cantonese voice search capability

by geekzhang on 2010-12-03 10:12:30

Google's mobile voice search allows users to shout out search keywords anytime and anywhere, then the software can recognize the voice and start searching. Now, voice search has begun to support Cantonese.

This support is good news for users who use Cantonese, and it has also attracted the attention of global Android users and media because the engineers from Google Hong Kong used the unreleased Android 2.3 operating system during the demonstration.

This is the first official appearance of the Android 2.3 system interface. In the demonstration video, a Nexus One phone with this system installed was used, including call, browser, notification bar icons all changed to green design.

In addition, a new electronic market program appeared in this version of the system. Unlike the current one, it added relevant content on the basis of detailed information and review two tabs, so that users can find more programs in the same category when searching for the software applications they want to use.
