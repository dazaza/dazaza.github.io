Maxthon 2.0.8 Spring Festival Edition (beta) - Added download functionality

by xloong on 2008-02-01 00:22:36

Nightfire: Since the release of the news about "Maxthon partnering with Xunlei to launch a browser download engine" in November, I have been paying close attention to Maxthon's download function. Unfortunately, there hasn't been much new information until now. Finally, Maxthon 2.0.8 has introduced a high-speed multi-threaded download manager. I wonder if this is the one they mentioned before. If the speed is really good, then I can stop using Xunlei, which is bulky and consumes network resources. However, I hope that Maxthon with this feature does not occupy bandwidth like Xunlei, otherwise, Maxthon will be useless in the future.

As the Lunar New Year approaches, during this time of bidding farewell to the old and welcoming the new, Maxthon has released a new version - Maxthon 2.0.8 Spring Festival Edition (Beta). This version includes two long-awaited features: support for IE plugins and a high-speed multi-threaded download manager.

To activate IE plugins, follow these steps: Open the "Tools" menu and select "Maxthon Settings Center". In the left navigation bar of the "Maxthon Settings Center", open the "Plugin Manager". Activate the desired IE plugin under the "IE Extensions" tab.

IE plugins may cause instability in the browser, so please use this feature with caution.

To use "Maxthon Download", you need to set it up in the "Maxthon Settings Center". Specify "Maxthon Download" under "File Download". You can also configure it here. The P2P function of Maxthon Download is not enabled by default. You can enable this function in its settings interface. After enabling, the download speed can be further increased, and some dead links can be downloaded. For already downloaded or ongoing tasks, you can manage them through the "Maxthon Download" option in the "Tools" menu.

(If the "Maxthon Download" menu item does not appear, right-click on the menu bar, choose "Add or Remove Buttons" in the pop-up menu, and click "Reset" after entering.)

In addition to these two new features, this version has improved many known functions, such as adding a download function to the right-click of the sniffer, making it more convenient to capture and save.
