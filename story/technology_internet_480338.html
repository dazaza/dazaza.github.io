Retrieve WordPress post list by custom fields

by diglog on 2011-08-21 12:42:50

In WordPress, you can aggregate posts through categories or tags. If you want to aggregate posts via custom fields, that's also possible. This article will tell you how to retrieve a list of WordPress posts based on custom fields. You might use this in a WordPress custom page template.

```php

have_posts() ) : $the_query->the_post();

the_title();

the_content();

endwhile;

// Reset Post Data

wp_reset_postdata();

?>

```

Now, if you want to retrieve a list of posts based on a specific value of a custom field, just change the query parameter in the following code:

```php

$the_query = new WP_Query( 'meta_value=blue' );

```

If you want to retrieve posts based on both the custom field name and its value, for example, retrieving all posts where the custom field name is "color" and the value is "blue", you can do it like this:

```php

$the_query = new WP_Query( array( 'meta_key' => 'color', [...]

```
