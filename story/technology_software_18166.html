Software Functional Features:

1. Friendly and aesthetically pleasing cashier interface, convenient and fast operation at the front desk.

2. If the previous customer's bill is not settled, the bill can be put on hold to settle the next customer’s bill and then resumed when needed.

3. During front desk settlement, directly input the product code or mnemonic code for settlement, or press the shift key to pop up all product ordering panels, and add selected products by double-clicking.

4. Modify the discount of individual items.

5. Automatically calculate the salesperson's commission.

6. Modify the customer's consumption discount.

7. Flexible operation permission settings, detailed down to each operator.

8. Products automatically generate pinyin short codes for quick input.

9. Each product can set three different prices: retail price, promotional price, and member price.

10. Manage refund orders for settled bills.

11. Automatically calculate the commission for employees selling cards.

12. All types of reports can be exported as Excel files for editing.

13. Membership card types can be set as: stored value card, accumulated points card, discount card, upgrade card.

14. Unique introduction of powerful customizable reports, with arbitrary queries, very convenient, neat and beautiful print format.

15. Powerful member points management, points transferable to gifts or consumption.

16. Support employee photo management.

17. Membership cards can be returned after consumption is complete.

18. Lost membership cards can be reported, temporarily disabled, or replaced.

19. Cashier shift handover management, cashier amount statistics during shift change.

20. Card recharge reward function, recharge can be managed with gifted amounts, such as purchasing an 800 yuan card that can be consumed for 1000 yuan.

21. Membership cards can transfer the balance and user information from one card to another.

22. Customer password lost can be modified.

23. Automatic backup can be set when exiting the system.

24. Shortcut keys can be set to call out or minimize system windows.

25. Member birthday automatic reminder.

26. The software can be locked when the operator leaves.

27. Supports receipt printers.

28. Member personal information, historical consumption, and points records can be permanently stored in the computer system.

29. Automatic inventory alarm when stock reaches the warning quantity.

30. Automatic prompt when customer consumption is less than the minimum consumption.

31. Comprehensive employee data management and commission statistics.

32. Automatically statistics table consumption situations.

33. Query and rank member consumption by time period.

34. Query and rank product sales by time period.

35. Set product promotion time periods.

36. Automatic membership card upgrades.

37. Set deposit accumulation discounts for stored value cards.

38. Automatically statistics consumption situations for timed projects.

39. Return value operations can be performed on membership cards.

40. First-time member consumption automatic reminder function.
