### One, What Are Rogue Applications?

Simply put, rogue applications, similar to the notorious rogue software of previous years, are a small group of developers or interest groups who illegally profit by taking advantage of the open characteristics of smartphone platforms like Android. They develop or modify these applications, which once they enter a user's phone, will secretly perform unauthorized and illegal operations without permission, causing direct or indirect losses to users.

Typically, rogue applications have the following characteristics:

- Forced or bundled installation

- System resource/service hijacking

- Interference with the normal operation of other software or system functions

- Difficult to uninstall completely through normal means

Once rogue applications are installed, the most direct loss may be secret charges. Additionally, personal information privacy can also be threatened.

### Two, How Do Rogue Applications Secretly Charge Fees?

Through analysis of multiple APK installation packages of rogue applications on the Android platform, we can understand that the main process for charging fees by rogue applications involves the following steps:

- Developers implant malicious charging code into the application during development (usually at the very beginning of program startup). The final developed application is packaged into an APK and distributed through various channels.

- Users download the APK package of the application and successfully install it locally.

- Thereafter, every time the program starts, it automatically runs the malicious code. During this process, it may secretly collect the user's phone number, IMSI, ICCID, IMEI, etc., and send this information to a dedicated server (some even encrypt it to prevent detection).

- After receiving the information, the server returns instructions informing the application to execute a segment of malicious charging code, sending text messages to specified phone numbers (or secretly installing software, opening special URLs with the browser, etc.).

- After sending the text message, the charging number usually returns a charging notice. At this point, the malicious code forcibly intercepts the receipt SMS, so the user receives no notification and unknowingly pays the money.

Additionally, some rogue applications bundle additional third-party applications. Based on the promotion performance of these bundled applications, they earn commissions from the bundled parties. Typically, each successful promotion of a bundled application earns a commission of a few cents to one yuan. With the huge installation volume of bundled applications (especially those embedded in popular third-party ROMs), these rogue applications themselves also generate substantial profits.

Some rogue applications combine both methods mentioned above to make exorbitant profits, earning income from both users and bundled application parties, making them extremely malicious.

### Three, How Rogue Applications Succeed

- Unscrupulous developers/publishers implant rogue code during development to pursue short-term benefits.

- A large number of underground publishing channels for (cracked, beautified, localized, third-party) applications provide fertile ground for rogue applications.

- Inadequate review mechanisms for application uploads/releases on legitimate channels.

- Users lack professional knowledge, identification skills, and vigilance, making them easy to deceive.

- Lack of corresponding tools for detection and reporting on the market.

### Five, How to Deal with Rogue Applications

The most basic precautionary measures come from users' own vigilance. Please obtain applications through legitimate channels and avoid unreliable third parties as much as possible. For uncertain quality applications, check online reviews and compare against the rogue application blacklist. If already installed, uninstall promptly (before doing so, retain necessary evidence).
