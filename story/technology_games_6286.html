Once a game uses the "Games For Windows Logo", it means:

1. The game supports widescreen displays.

2. The game supports 64-bit Windows operating systems.

3. The game supports the Xbox 360 game controller.

4. The game can be launched in Windows Media Center.

5. Partial optimization for Windows Vista is achieved, supporting Vista features such as Games Explorer and parental controls.
