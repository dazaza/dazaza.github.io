There is a very confusing version of linux released by ubuntu -desktop. Because there is a set of servers, desktop is easily interpreted literally as desktop. And people who understand that often don't understand what the alternate is.

In fact, desktop is the name of the Live CD, that is, the Linux that is burned to run on the disc. It's a system that's already in place. You burn it to a CD, put it in the CD drive, and you can run the full linux. Of course, you can also "install" it to the hard disk after running it, and such an installation is more like a copy or restore, similar to ghost.

The proper version of the installation is alternate, which consists of a number of deb packages. The installation content can be specified in detail. Is authentic hard disk installation like. There are also some speed differences between the two versions.

desktopCD can see the basic interface and trial (liveCD functionality) before installation, alternate is aimed at more professional users who need more personalized installation.
