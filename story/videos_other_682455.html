Fairchild FL7730 8.4W Dimming LED Driver Solution

by ledlamp8 on 2012-02-28 22:33:19

The Fairchild FL7730 is a highly integrated PWM controller that provides enhanced performance for single-stage flyback converters. Its TRUECURRENT™ topology simplifies the circuit design of LED lighting applications with input voltage ranging from 80VAC to 308VAC, a starting current of 20uA, and an operating current of 5mA, providing better EMI performance. It is mainly used in LED lighting systems. This article describes the main features of the FL7730, its block diagram, typical application circuits, and reference design schematics using the FL7730 for TRIAC dimmer LED drivers at 180V-265VAC and 90-140VAC.

**FL7730MY: Single-Stage Primary-Side Regulation PWM Controller for PFC and LED Dimmable Driving**

This highly integrated PWM controller, FL7730MY, offers several features to enhance the performance of single-stage flyback converters. The proprietary TRUECURRENT™ topology enables simplified circuit design for LED lighting applications. TRIAC dimming is smoothly managed by brightness control without flicker. By using a single-stage topology with primary-side regulation, an LED lighting board can be implemented with fewer external components and minimized cost. It does not require an input bulk capacitor or feedback circuitry.

To achieve good power factor and low total harmonic distortion, constant on-time control is utilized with an external capacitor connected to the COMI pin. Precise constant-current control regulates accurate output current despite changes in input voltage and output voltage. The operating frequency is proportionally changed by the output voltage to ensure Discontinuous Conduction Mode (DCM) operation with higher efficiency and simpler design.

The FL7730MY provides protections such as open-LED, short-LED, and over-temperature protections. The current-limit level is automatically reduced to minimize output current and protect external components in a short-LED condition. The FL7730MY frequency-hopping function improves EMI performance in the oscillator. The FL7730MY controller is available in an 8-pin SOP package.

**Main Features of FL7730:**

- Compatible with Traditional TRIAC Control (No need to change existing lamp infrastructure: wall switch & wire)

- Compatible with Non-Dimming Lamp Designs

- Cost-effective solution without Input Bulk Capacitor and Feedback Circuitry

- Power Factor Correction (PFC)

- Accurate Constant-Current (CC) Control, independent of line voltage, output voltage, and magnetizing inductance variation

- Line Voltage Compensation for CC Control

- Linear Frequency Control for Better Efficiency and Simple Design

- Open-LED Protection

- Short-LED Protection

- Cycle-by-Cycle Current Limiting

- Over-Temperature Protection with Auto Restart

- Low Startup Current: 20μA

- Low Operating Current: 5mA

- Frequency Hopping for Better EMI Performance

- SOP-8 Package Available

- Application Voltage Range: 80VAC ~ 308VAC

An LED has become a promising light source for replacing conventional lighting systems, such as fluorescent and incandescent lights. Especially in the conventional TRIAC dimmer infrastructure, there has been much research into developing an LED bulb compatible with TRIAC dimmers. Because the incandescent light source consumes a hundred watts with a short lifetime, an LED bulb can be an excellent substitute with considerably less power dissipation and a longer life.

The biggest recent issue with TRIAC-dimmable LED bulbs is dimmer compatibility. The conventional TRIAC dimmer was originally designed to handle hundreds of watts induced by incandescent bulbs. An LED bulb consuming less than 20W should interact with those dimmers composed of high-power devices. If the interaction between the dimmer and LED bulb is not stabilized, visible flicker is perceptible. To manage the interaction without flicker, some requirements for dimmer operation need to be considered.

TRIAC dimmers need latching current at firing and holding current during TRIAC turn-on after firing. If these two currents are not met, the TRIAC dimmer misfires and the LED light flickers. The TRIAC dimmer blocks the input line at the beginning of the line cycle, then connects the input line and LED bulb after firing. The TRIAC dimmer turns off if the latching or holding current flowing through the dimmer is inadequate. The latching and holding currents vary depending on the dimmer model. The typical range of latching and holding currents is around 5 ~ 50mA. These operating requirements do not cause problems when using incandescent bulbs due to their high power consumption. An LED bulb with less than 20W output power cannot maintain this amount of current over the whole line cycle.

This application note provides a practical guideline for TRIAC-dimmable LED bulb board design. Passive and active bleeder design guides detail how to maintain latching and holding current without visible flicker. Active damper design improves efficiency by minimizing the count of external components. The input filter design section covers the effect of filter components on PF, THD, and EMI.
