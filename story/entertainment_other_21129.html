GreenBrowser

by shooter on 2007-06-01 20:05:55

In the current multi-page browser field, the most user-friendly browsers are undoubtedly MyIE2 and GreenBrowser. Many people have even published articles comparing the merits of these two software programs. In fact, it's hard to reach a definitive conclusion on this issue because many modern multi-page browsers are developed based on the original MyIE source code, including MyIE2 and GreenBrowser. Therefore, there is often a sense of familiarity when using such software. In this article, we will introduce several operation techniques of GreenBrowser (hereinafter referred to as GB), which are characterized by low resource consumption and powerful functionality.

1. A page manager that can be turned on and off easily. When GB starts, the "Page Manager" will pop up, allowing us to conveniently choose to open pages visited last time, closed pages from the previous session, or currently closed pages. Compared to MyIE2, which can only load pages from an abnormal shutdown after startup, GB offers more flexibility. However, we can also press Ctrl+M at any time in GB to bring it up.

2. Customizable automatic classification saving. Have you used the Universal Browser? Many browsers can only save web resources to a specified directory, but GB allows you to right-click on the tab of the current page, select the "Automatic Classification Save" command, and then choose to store it in a category preset by the program or in a self-created category directory. You can also save according to the time automatically. This feature is somewhat similar to FlashGet's file download management function.

3. Extremely convenient one-click image saving. Normally, to save an image from a webpage, you need to right-click the image and select the "Save Image As" command, then set the storage path, which is quite troublesome. In GB, however, just drag the image to be saved to the right with the mouse, and it will be stored on the hard drive. The images are saved in the website folder under GB. If you want to change the image's saving path, you can click on "Tools → GreenBrowser Options → Save" in the menu bar and customize the saving path in the "Image Save" text box. If you drag the image link to the right, it will open the link in a new window instead of saving the image.

4. Flexible and quick page search. When we need to quickly find the location of a sentence on a webpage, we can select a keyword and drag it to the left with the mouse, then choose "Page Search" from the popup menu. If you cannot quickly locate the keyword on the webpage, you can directly input it into the search bar and press Ctrl + Enter to search for the keyword from top to bottom on the page. To go back to the previous keyword, press Shift+Enter.

5. Customizable status bar display format. By default, the data information displayed on the GB status bar is very limited. If you want to expand its content or change the display position, click on "Tools → GreenBrowser Options Window → Status Bar" label and check the status bar display format in the right view. Here, you can customize it according to the annotation information below. For example, if I want to display the local IP address on the status bar, I just need to input "K_" and the IP address will be displayed.

6. Disable automatic webpage refresh. Some websites or forums add codes that make webpages automatically refresh to increase traffic or ensure visitors get the latest information. Although this does not harm the system itself, it still wastes some system resources, so we should disable it. Right-click on the tag of such a webpage and select the "Disable Page Auto Refresh" command to disable these codes.

7. Personalized group download control schemes. In GB, we can set different download control schemes for each group, including objects like images, sounds, and videos. It is somewhat similar to Firefox's quick groups. Click on the "Group → Set Group" command in the menu bar, then in the popup dialog box, select a pre-set group sub-item and click the "Edit" button below. In the popup dialog box, check the "Use Specific Download Control Scheme" checkbox, and then set as needed to categorize the opened webpages.
