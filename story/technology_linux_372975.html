Running QQ2010 with Wine

by anonymous on 2010-12-02 03:24:22

Since I fell in love with Ubuntu, I've been troubled by the QQ issue. I've tried the official Linux version of QQ, WebQQ, and many other modified versions of QQ, but none of them satisfied me. Now, I've finally found a way to say goodbye to logging into QQ via a server: running QQ2010 with Wine, which supports common features like screenshots and Weibo.

First, let me explain what Wine is (experts can ignore this):

Wine is not a Windows emulator; it's software that uses a compatibility mode to call DLLs to run Windows programs. Wine isn't perfect; as a compatibility layer, it may cause program crashes, high CPU usage, low efficiency, and some hardware-related operations (like online banking) cannot be completed. Using Wine is legal unless you use unauthorized Windows DLL files. Wine is under rapid development; Google Picasa for Linux is a program built on Wine, and Disney’s contribution to Wine development has made it possible to run Photoshop effectively. Note that Wine only supports 32-bit systems and does not support DOS or other 16-bit programs.

via: Ubuntu Chinese Forum (http://forum.ubuntu.org.cn/viewtopic.php?f=121t=174310)

Below are the steps to run QQ2010 with Wine on Ubuntu. If you need it, feel free to try:

1. Upgrade Wine

a. Add PPA source

System Management -> Software Sources -> Other Software -> Add

ppa:ubuntu-wine/ppa

b. Run the following commands in the terminal to upgrade Wine (official reference):

sudo apt-get update

sudo apt-get dist-upgrade

2. Download and install Winetricks (Wine library installation assistant)

a. Download:

wget http://www.kegel.com/wine/winetricks

b. Install libraries:

sh winetricks msxml3 gdiplus riched20 riched30 ie6 vcrun6 vcrun2005sp1 flash

3. Install SimSun font (SimSun.ttf), modify the Wine registry to resolve character encoding issues.

a. Install SimSun:

Copy SimSun.ttf from a Windows system (path: c:/windows/Fonts/) to /home/username/.wine/dosdevices/c:/windows/Fonts/

b. Modify Wine registry:

Enter "regedit" in the terminal (the interface is the same as Windows)

Open [HKEY_LOCAL_MACHINE\Software\Microsoft\WindowsNT\CurrentVersion\FontSubstitutes]

MSShellDlg=SimSun

MSShellDlg2=SimSun

Tahoma=SimSun (manually create string)

4. Download QQ and launch it.
