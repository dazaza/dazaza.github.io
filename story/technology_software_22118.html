Firefox 3 Alpha 5 发布 添加Place功能

by @yinyin on 2007-06-09 23:04:05

Mozilla发布了Firefox 3 Alpha 5(产品代号:Gran Paradiso).

Firefox 3 Alpha 5 加入了以下新的特性

# 基于”Places”的标签系统.

# 新的错误报告系统“Breakpad“

# 基于Javascript的密码管理器.

# Support for Growl notification under Mac OS X.

# Mac OS X版加入了对原生控件的支持.

# Gecko 1.9 的漏洞修复.

“Places”是用于存储和管理标签,历史记录等和网页相关信息的系统.它是基于SQLite数据库系统的,稳定性和速度都会较firefox2.0有显著的改进.Firefox 3 Alpha 5是第一个加入了基于”place”标签系统的版本.
