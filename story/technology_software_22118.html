Firefox 3 Alpha 5 release adds Place feature

by yinyin on 2007-06-09 23:04:05

Mozilla has released Firefox 3 Alpha 5(product code :Gran Paradiso).

Firefox 3 Alpha 5 adds the following new features

# Places-based labeling system.

# New bug reporting system "Breakpad"

# Javascript based password manager.

# Support for Growl notification under Mac OS X.

# Mac OS X version adds support for native controls.

Bug fix for # Gecko 1.9.

"Places" is a system for storing and managing tags, history, and other information about web pages. It is based on SQLite database system, stability and speed will be significantly improved over firefox2.0. Firefox 3 Alpha 5 was the first version to incorporate a "place" based tag system.
