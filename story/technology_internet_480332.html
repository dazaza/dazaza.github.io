Webmaster Legion, a niche webmaster query tool

by diglog on 2011-08-21 12:42:50

A small station I tinkered with is called "Webmaster Corps," offering a few relatively basic webmaster query tools. In terms of functionality, I plan to gradually optimize it in the future. Now, the second version has been released, improving the tool navigation so that you only need to input once to query everywhere. Below is a brief introduction to each small tool:

1. **Search Engine Indexing**: Provides Alexa overall ranking, PageRank value, Sogou rating, and Chinese website rankings. It also offers simple search engine page indexing volume and backlink quantity queries.

2. **IP Geolocation**: Queries the corresponding geographical location based on the given domain name or IP address and provides a map for reference. Some data may have certain deviations, so it is for reference only.

3. **Alexa Ranking**: Offers various queries such as Alexa overall ranking information, ranking change trends, daily average traffic statistics, and sub-site visit distribution situations.

4. **Domain Whois**: Simply provides domain Whois information queries. The speed is a bit slow, but we are still working on resolving this issue.

5. **PageRank Value**: Queries the Google PageRank value corresponding to the domain and provides PageRank value invocation icons in various image sizes.

Next steps: Baidu weight, keyword mining, reverse IP domain lookup, and filing queries. There seems to be much more to do, but that's all for now.
