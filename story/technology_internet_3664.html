Exchange Server 2007 开始公开测试

by Wangtam on 2006-07-24 06:45:03

微软今天如期公开发布了电子邮件服务器软件 Exchange Server 2007 的第二个测试版，同时发布的还有用于 Exchange Server 的电子邮件安全防护产品 ForeFront Security。去年年底的 Exchange Server 2007 Beta 1 仅面向 1400 个内部测试人员和合作伙伴，而最新的 Beta 2 提供公开下载和 DVD 光盘定购。Exchange Server 2007 Beta 2 支持 10 种语言，提供新的管理工具、统一消息服务、更好的邮件可靠性等，但仅有64位版本，最终版亦将如此，而不再像 Beta 1 那样同时支持32位和64位。...
