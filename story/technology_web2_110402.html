wordpress plugin for mobile management: WP-T-WAP

by wsgeqd on 2008-12-10 16:13:14

Sometimes we don't update our wordpress blog very much when we are on a business trip, studying, and not in front of the computer. By chance, we thought of trying WordPress mobile plug-in, and found WP-T-WAP in Wopus Chinese community, which is developed for WordPress and used to realize mobile browsing and management functions of blogs.

1. Browse, add and delete articles through mobile phone

2. Browse, post, delete and approve comments via mobile phone

Author: TangGaowei.com

1. Installation

Into the plugins directory (plugins/wp - t - wap / *. *), and then start the plug-in can be in a similar "http://www.mysite.com/wap" or "http://www.mysite.com/wap/index.php" address access. (Note: You must have access to the wap directory at the root of the WP site; otherwise, the installation will fail.)

Step 2 Update

Overwrite the old WP-T-WAP plug-in directory with the new WP-T-WAP plug-in directory, then disable the WP-T-WAP plug-in, and then enable the WP-T-WAP plug-in to complete the update.

Step 3 Set up

"General Settings" in the background of WP website -> In WAP Plug-in (slightly different in WP 2.5), you can set Display Mode, Custom Title, and Version Information.

4. Bind the domain name

Bind the domain name directly to the wap directory in the WP site root directory. For example, if the WP site root is /usr/http/www, you should bind the wap.mysite.com domain name to the /usr/http/www/wap directory. (Generally bind the domain name to the directory in the "virtual host" management interface of the website first, and then add the domain name resolution to the "Domain name management")

Test on UCWEB mobile browser, as shown in the figure: the effect is not bad.
