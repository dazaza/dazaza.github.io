Download: Google Chrome 0.3.154.9 Beta

by yinyin on 2008-10-31 18:34:40

According to the latest update notes of Google Chrome, the latest version of Chrome 0.3.154.9 has modified the download behavior, when the download file is an executable file, the browser will change the download file name.

The update includes improved download methods, user interface, plugin upgrades, JavaScript and Web API upgrades, and addresses some application crash issues.

Among them, R3285 fix: change the download method of the knowable files, when these files are downloaded, they will be saved as unconfirmed_*.download files, and only when the user agrees to use the download file, the file will be converted to the real file name. Unconfirmed downloads will be deleted when the browser exits.

Interface improvement

Fixed in R3177, R3207, when starting the browser, the active connection will be listed and password management will be enabled.

R3180, Add function to add anonymous access to the browser.

R3189, fixed a new bookmark error that opened the editor when the edit button was clicked.

R3209, changes Google Chrome's terms of service regarding the use of open source software.

plug-in

R3211, fixed an issue where the video stopped after 1 second

R3272, which allows plug-ins to send forms, such as Adobe Reader.

R3276, better support for Windows Media Player

R3327, let the plugins run at a normal priority so that they don't cause the browser to stop responding.

JavaScript/web API

R3198, optimized content detector

R3231, fixed a plugin that could cause web content to disappear

R3269, change the minimum timer, set SetTimeout to 4 milliseconds.

Fixed crash issues

R3172, fix crash when using spell check

R3278, fixed browser crash when using Windows Vista Tablet PC operating system
