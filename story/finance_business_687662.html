BMW Multi Tool Latest OBD2 CAS1-3+ Key Programmer

by sbauto2012 on 2012-03-02 17:10:03

BMW Multi Tool V1.2 is a comprehensive tool for BMW vehicles, consisting of four main components: hardware device, dongle, programmer, and software. Before running the software, you must connect the dongle and hardware device to your computer. This tool is more powerful than AK300!!!

**Description:**

1. **OBD II CONNECTOR:** Connects the tool to the vehicle’s Data Link Connector (DLC).

2. **Dongle:** USB dongle.

3. **USB CONNECTOR:** Connects the tool to the PC/Laptop via a USB cable.

4. **Programmer:** Programs keys.

5. **Programmer Female Pin:** Connects the programmer to the device.

6. **Programmer Male Pin:** Connects the programmer to the device.

**Specifications:**

1. Operating Temperature: -20 to 70°C (-4 to 158°F)

2. Storage Temperature: -40 to 85°C (-40 to 185°F)

3. Diagnostic Interface: 16 PIN

4. USB Interface: USB 2.0

5. Power: DC 5V - 12V

6. Consumption: 1W

7. Dimensions: Length 95mm, Width 49mm, Height 28mm

**Function:**

For BMW MULTI TOOL supported functions include:

1. Car access system identification

2. Read/Write CAS EEPROM (OBDII)

3. Read/Write KM stored in CAS system (OBDII)

4. Read/Write synchronization codes stored in CAS system (OBDII)

5. Synchronize DME and CAS (OBDII)

6. Synchronize CAS and ELV (OBDII)

7. Prepare dealer key for CAS1/CAS2/CAS3/CAS+ (supports the latest CAS3 version) (OBDII)

8. Read/Restore CAS KEY information (OBDII)

9. Disable/Enable key with working key (OBDII)

10. Clear DTC and Shadow (OBDII)

11. Prepare dealer key with CAS EEPROM dump

12. Unlock dealer key prepared with our software using CAS EEPROM dump or KEY information

13. Change KM in CAS and instrument dashboard with file dump

14. With a key programmer, you can program dealer keys directly.

**Pay attention to the function: Help->Device update.**

You can get the latest software version and firmware version when you have internet access.

1. **Menu:**

- **Tool:** You can choose the language here, supporting "English" and "Chinese Simplified". Restart the software to apply your settings.

- **File Make Key:** You can prepare dealer keys with EWS and CAS EEPROM dump.

- **Help:** Device update. You can check your device and dongle ID version and update it.

2. **Protocol Type:**

Now BMT supports three types of protocols: KWP-CAN BUS 100Kbps, KWP-CAN BUS 500Kbps, KWP-2000. The "Auto Detect" function can help you choose the right protocol. After choosing the protocol, press "Connect". If connected successfully, all CAS info will display in CAS Identification, including CAS number, VIN, production date, etc.

3. **EEPROM Function:** Read EEPROM, Write EEPROM, Save File, Load File.

1). You should back up the CAS EEPROM before preparing dealer keys. After successfully reading the EEPROM, please save it as a backup file. This step is very important.

2). Now, you can't read the whole EEPROM for the CAS3 system. But the main section is contained and should be saved. When you read the CAS EEPROM, it will give tips (Press OK to continue).

4. **Odometer:** Supports reading and changing KM stored in the CAS system. Does not support changing KM in the instrument cluster. If you want to change the KM display in the instrument, you should change the KM in the CAS system first. Then modify the KM in the instrument manually.

5. **Synchronization Codes:**

1). CAS3 code-DME, CAS3code-EGS: Read and write DME and EGS codes stored in the CAS3 system.

2). Synchronize DME and CAS: This function will be useful while preparing dealer keys or any other operation that causes the engine not to start.

3). Synchronize CAS and ELV: This function helps clear wheel errors.

6. **Key Learn Function Introduction:**

1). Get Key Info: With this function, you can get the key cutting, remote frequency, and key data.

2). Save Key Info: Save the read key info.

3). Load Key Info: Load the saved key info. To see the key data and prepare for writing key info.

4). Write Key Info: Write the loaded key info into the CAS system.

5). Make Dealer Key: You can prepare dealer keys after getting key info successfully. Also, you should choose the key position.

6). Add Key: Add the new dealer key into the CAS system. Some CAS systems need this step to start the engine.

7). Enable Key: Insert the current key into the ignition switch, press "Enable Key". The key after the current key will be enabled.

8). Disable Key: Insert the current key into the ignition switch, press "Disable Key". The key after the current key will be disabled.

9). Clear DTC: Before preparing dealer keys and after it, you can use this function to clear DTC.

10). Clear Shadow: Before preparing dealer keys and after it, you can use this function to clear Shadow.

7. **File Make Key:**

1). Choose CAS or EWS type.

2). Load the chosen EEPROM file, and you will get the make dealer key window.

3). Choose the key position and press the "Make Dealer Key" button. Follow the given tips to prepare the dealer key. After succeeding, you can save the new EEPROM dump file.

- For EWS1, EWS2, EWS3, EWS4, CAS1, CAS2, CAS3 types, the prepared dealer key can start the engine directly.

- For some CA3+ types, you need to write back the new dump file into the car.

4). CAS3 encrypt data, sbb key programmer, only OBDII prepare make key is supported for CAS3 encrypt data.
