[Product Description]

Wiper blades are vehicle consumables. If there is an issue with them, it can affect the driver's line of sight. For safety, we need to check them regularly and replace them when necessary. This product can only be used on wipers that use a U-hook and cannot be used on wipers that use screws for fixation.

[Warm Reminder]

Please carefully measure the length of your car's original wiper blades and choose the appropriate size based on your measurement. Please note the required size in your order comments.

[Usage Method Diagram]

Step One: Remove the original factory-installed wiper... You must press the clip underneath! Then pull it out...

Step Two: Detach the original wiper from one side, but be extremely careful not to let the wiper arm snap back and damage the windshield...

Step Three: Slightly lift the front end of the middle clip to make installation easier...

Step Four: After inserting the middle clip, pull it tight! Once you hear a click, it is securely in place!

Step Five: Some models have different wiper sizes for the driver's side and passenger side. Generally, the driver's side is longer, and the passenger side is shorter. Be sure not to install them backward. If both wipers are the same size, then it doesn't matter which side you install first...

Step Six: Install the passenger-side wiper! Ensure the direction is correct...

Step Seven: Tighten it the same way until you hear a click...

Finally, check the installation results! Make sure they are not installed backward! The slanted surface should face forward so that the wiper can easily utilize airflow to brush upward...
