How to install APK files on an Android phone

by dream2012 on 2012-02-24 13:55:33

Although Android phone users can find almost all the applications you need to download and install in the electronic market (Android Market), you may still encounter installation files with the suffix APK on forums and Android application download websites. APK stands for Android Package, similar to the sis files of Nokia's S60 system. APK files are the dedicated installation file format for mobile devices on the Android platform. Essentially, APK files are a variant of JAR (Java ARchive) files. There are several reasons why we might use APK files to install Android phone applications:

1. Android developers who want to upload their applications to the Android electronic market must pay a $25 registration fee and go through complex operations such as activating Google Checkout and binding a credit card, which is not very friendly to domestic developers. Therefore, some developers distribute their Android programs only in APK file format.

2. Mainland China Android users cannot purchase paid programs in the electronic market, but many foreign users who have purchased Android paid programs export the APK installation files of these programs using technical means and share them on forums or related websites for other users to download, allowing us to experience more powerful paid applications on our Android phones.

3. Android smartphones, like PCs, sometimes need to reinstall the system (flash the ROM). Before reinstallation, you can back up all installed programs as APK files using software, and then reinstall these APK files after the system is installed to restore the previous applications. This method is convenient and saves network traffic costs.

* Installing APK files onto an Android phone via the electronic market

Use the following methods to install downloaded APK files onto your phone:

1. Open the Android phone settings menu, go to application settings, check the "Unknown Sources" option, then enter developer settings, and check the "USB Debugging" option.

2. Copy the APK installation file to the Android phone's memory card, not necessarily the root directory.

3. Download an APK installation software from the Android electronic market, there are many such types of software available, such as this Apk Manager.

4. Open the Apk Manager software (other APK installation software operates similarly), the program will automatically scan and display the APK files on the phone's memory card.

5. Click on the APK file to install the application.

If you think it's too troublesome to copy the APK file to the memory card using a USB cable each time, or if you happen not to have a USB cable nearby, you can also send the APK file as an attachment to your Gmail mailbox, then open the Gmail app on your phone and click on the attachment to download it. After downloading, you can install it using the tAttachApkInstaller program (available for download in the electronic market, needs to be installed in advance).

* Installing APK files onto an Android phone via Android SDK

If your phone temporarily cannot access the network and cannot connect to the Android electronic market to download applications, you can also use the Android SDK (Android Software Development Kit) to install APK files. However, the operation is much more complicated than the methods mentioned above and may not be suitable for Android beginners. It is still very useful in certain situations, especially for developers.

1. Download and install the Google Android SDK program and the Android USB driver program. The download links are as follows: http://code.google.com/android/intro/installing.html http://dl.google.com/android/android_usb_windows.zip

2. Similarly, open the Android phone settings menu, go to application settings, and check the "Unknown Sources" option. Then, in the SD card and phone memory settings in the phone settings, select "Unmount SD card" to disable this function.

3. Next, you can proceed to the installation phase. Open the Windows command prompt and input the following command: #adb install path/file.apk. Of course, you need to replace path with the full path of the directory where the APK file to be installed is located, and replace file with the APK filename.

4. Done! Your application has been successfully installed on the Android phone, and you can now open the program and use it.
