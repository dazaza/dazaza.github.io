Corel VideoStudio 9 Green Mini Edition

by rkcms on 2010-05-17 09:49:48

The official version of Corel VideoStudio 9 is 660MB,

but this green version is only 68MB and has all the functions. There are no missing useful features! Only some optional functions have been removed!

After downloading, unzip the file, find the "install" folder (this step is mandatory), double-click all the self-extracting files inside, then run "vstudio.exe".

Using Corel VideoStudio 9 to add titles, subtitles, and transitions to videos is effortless, very user-friendly, simple to learn, and basically doesn't require additional tutorials. After editing, click "Share", then click "Create Video" to finish.

**Notes:**

1. Based on the official Simplified Chinese edition, it basically has all the functions required for video editing. However, some functions have not been tested, such as video capture and DV rewinding.

2. All sample audio/video files (occupying 140MB) and all templates (occupying 360MB) have been deleted, so the wizard function cannot be used.

3. SmartSound is not installed, so the SmartSound audio function cannot be used.

4. Since the default library file path in the software cannot be modified, it would cause every file to need its links re-modified, which is troublesome. Therefore, these library files have been deleted outright and can be manually added when needed. (See Usage Item 2 for the method.)

5. FrameServer has already been integrated, allowing direct output of high-quality MPEG files using TMPGEnc. (See Usage Item 3 for the method.)

6. The Hollywood FX transition plugin has already been integrated, enabling direct use of Hollywood FX Gold v4.58 transition effects (must be Hollywood FX Gold v4.58, other versions will not work). (See Usage Item 4 for the method.)

**Usage:**

1. Find the "install" folder, double-click all the self-extracting files inside to initialize, then run "vstudio.exe" to start Corel VideoStudio 9. Execute "@UNinstall.cmd" to uninstall perfectly.

2. On the first use, the video, audio, decoration (objects, borders), and Flash animation categories will be empty (see Note 4 for the reason). Add the media files you need to edit for audio and video. The original objects, borders, and Flash animation files from the official version are located in the "Samples" directory under the installation directory. I have categorized them and renamed the directories in Chinese. Please add them manually when needed.

3. Using FrameServer: First, go to the FrameServer directory under the installation directory and execute "@InstFS.cmd". Then, when creating a video file, choose "Custom", and in the save file type, select "DebugMode FrameServer Files (*.avi)", then save. Finally, run TMPGEnc, open the saved AVI file (only a few KB), set the relevant parameters, and you're done.

4. Using the Hollywood FX transition plugin: First, install Hollywood FX Gold v4.58 (must be the 4.58 Gold Edition), then under the transition menu in Corel VideoStudio 9, select "Hollywood FX". At this point, there will only be one "Gold" effect under the project. Drag and drop the "Gold" effect to the place where you need a transition, then click "Customize" on the left control panel to bring up the Hollywood settings panel. After setting the effect, confirm to add it to the transition.
