GitHub: The Innovation Hub on the Internet

by anonymous on 2013-07-29 15:07:58

　　The Internet’s Innovation Hub

　　GitHub has created a social network for programmers that gets things done without bosses, emails, or meetings. This reflects how online collaboration can reduce the importance of physical technology hubs.

　　The Next Silicon Valley

　　Every city dreams of becoming the next tech center, nurturing future startups.

　　From an indicator standpoint, San Francisco-based startup GitHub appears to have all the hallmarks of the next big social network. With a rapidly growing user base of 3.6 million and a valuation of $750 million after raising $100 million last year—at least on paper.

　　However, GitHub is not a social hangout, nor a place to share photos. It's a website where software developers store, share, and update their personal programming projects.

　　GitHub CEO Tom Preston-Werner points out that what makes this social network special is that it's built around creating something valuable.

　　This combination of practicality and sociability makes GitHub a hub for software innovation. People from all over the world (78% outside the U.S.) come here to test and refine their application ideas (mobile or web server software).

Wharton Assistant Professor Ethan

Mollick views platforms like GitHub and Kickstarter as a new type of technology platform that allows innovation to occur free from traditional geographical constraints or established hierarchies.

More and more innovation elements are being virtualized and delivered through the internet. For example, GitHub virtualizes software development; Kickstarter virtualizes funding; AngelList virtualizes venture capital; Science

Exchange virtualizes laboratory research; Coursera virtualizes higher education; CodeAcademy virtualizes programming education; Mechanical

Turk (Amazon) virtualizes gig work.

　　GitHub's guiding principle is to break down any barriers that prevent anyone from contributing to others' projects. There are no emails, no traditional management. Participation in anything here does not require permission from others.

　　Founded in 2008, GitHub is now profitable, with 10,000 new users added daily. Its latest business model involves renting out a version of GitHub for companies to use internally. Marissa Mayer mentioned in her first company-wide memo after becoming Yahoo CEO last year that GitHub would be used as one of the tools to correct Yahoo's suffocating bureaucracy.

　　GitHub's most important feature is the pull request. Through it, someone can propose changes to another person's project code, and the project owner can easily see and review these suggestions. If approved, it only takes a click of a mouse to merge them into the project, or initiate discussions about these changes. If someone's pull request isn't accepted, they can "fork" the project, creating a parallel version on GitHub to incorporate their own ideas.

　　San Francisco's office is GitHub's only physical location in the real world, housing one-third of its 176 employees (the rest work from home, cafes, or rented desks). The company doesn't have fixed working hours; some sleep until noon, while others work late into the night. Some start at sunrise and finish by 3 PM. The entire company has only one fixed title: Preston-Werner as CEO. Others have ever-changing titles like "Bad Guy Catcher," "Open Wizard," among many others.

　　GitHub plays an important role in open-source software development, and the company also uses it to maintain and expand its services. Although Preston-Werner sets the overall goals for such projects, he leaves the implementation details entirely to the employees. GitHub's employee teams form, develop, shrink, and dissolve according to the company's needs, then everyone moves on to find new tasks.

　　Meetings are considered extremely wasteful. At GitHub, thanks to pull requests, there are fewer occasions requiring meetings. Preston-Werner even believes they will never need managers.

　　Preston-Werner hopes his philosophy will spread so that GitHub can accommodate more types of work. GitHub has already launched graphic design features for designers. Some journalists, scholars, and even the White House have started experimenting with writing articles, conducting research, and collaborating on policy documents on GitHub. Preston-Werner says, "We started with software, but our vision is much broader than that."
